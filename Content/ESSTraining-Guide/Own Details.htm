Own Details

This function enables you to view or maintain the following employee record details depending on access:

  • Postal address
  • Date of Birth
  • Sort Code
  • Account Number

Navigate to the Employee Self Service module and click on Own Details as shown below

On the ‘Main’ tab you can view your own HR details or if access provided amend these details using the open button

Once changes made press the save button in the bottom right hand corner
