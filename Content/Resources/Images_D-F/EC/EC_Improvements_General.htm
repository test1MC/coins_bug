Screen and Processing Changes

Order Confirmation Validation

The order confirmation routines in both and now use the same validation, so they report any errors consistently.

eCommerce Workbenches in OA

A ! is now displayed in a new column on eCommerce workbenches if the supplier does not exist or if a file cannot be found. A tooltip shows the error.

Background Processes

If any of the parameters that are used by eCommerce background processes are changed, the next time the background process runs it will read the new value of the parameter. The process does not need to be stopped and started first.

BASDA/ Mapping Codes

A new screen, 3rd Party Invoice/ Codes, allows you to set up a list of the codes that are used in the BASDA/ Mapping function.
