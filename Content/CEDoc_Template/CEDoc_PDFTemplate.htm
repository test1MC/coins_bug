Table Style = COINSBasic.css is the style you can use to create tables for printed output.

  1. From the Insert tab, click the Table button.
  2. Select the General tab.
  3. Select COINSBasic.css in the Table Style field, as shown below. Click OK.

Address Snippets

You have a choice of address snippets to use:

  • If you know you will be generating a document for one region, only insert the snippet for that region.
  • If you might need to generate a document for more than one region, use the snippet with region conditions.

    • Snippet: COINS Address - US.flsnp (US COINS Address) [no conditions; add where needed]

     

    Snippet: COINS Address - UK.flsnp (UK COINS Address) [no conditions; add where needed]

     

    Snippet: COINS Addresses US&UK with Conditions.flsnp (US&UK combined with conditions; added to title page and can be used wherever needed]
