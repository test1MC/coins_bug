Installation Instructions

  1. Use Internet Explorer to go to https://secure.gateway.gov.uk/submission. The following page will be displayed.

  2. Click the Padlock Icon so the following is displayed:

  3. Click  View certificates.

  4. Click the Details tab.

    5. If Copy to File is greyed out, add the following to your trusted sites.

    (In Internet Options, Security, Trusted Sites)
    https://secure.gateway.gov.uk/submission

  5. Click Copy to File and then click Next. The following screen will be displayed; leave the format as the default.

  6. When you click Next you will have the option to Browse.

  7. Click Browse and save the file to your desktop or a suitable area on your “C” drive or network.  The example below saves the file on a desktop.

    8. You will need to name the file; name it newcert2016 .

  8. Click Next.
  9. Then click Finish.

  10. Log into as root, as only the root user can import the certificate.
  11. FTP the file from your desktop (or wherever you have saved the file) into the /tmp directory on the server.
  12. Check the setting of the JAVAHOME parameter for Payroll or Subcontract Ledger, and make a note of this. Please note this version should be java1.6 or higher.
  13. Change directory (cd) to the location of jre as in the JAVAHOME parameter.
  14. Run the following command to install the certificate:

    15. ./bin/keytool –import –keystore ./lib/security/cacerts/ -alias govtalk2 –file /tmp/newcert2016.cer

  15. You will be asked for the Keystore password. The password is changeit.

  16. At the Trust this certificate? prompt, enter Y.

  17. If you get an error message stating it already exists run the following command to remove the existing one and then re-run the command above :

    18. ./bin/keytool –delete –keystore ./lib/security/cacerts/ -alias govtalk2
