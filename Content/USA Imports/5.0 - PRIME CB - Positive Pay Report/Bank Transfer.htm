Bank Transfer

Module: (11.07) [Cash Transaction Entry]

  1. Click on > Cash Transactions > Cash Transaction Entry.Cash Transaction Entry

  2. Click .Cash Transaction Entry

  3. Enter all the required details
    • : Period in which transaction will be Posted.
    • Transaction Date: Date the transaction is being entered
    • Effective Date: Date the transaction actually occurred
    • Manual/Print:  Manual
    • Payment/Receipt: Payment
    • Allocate: Always . This allows allocation to an invoice.
    • Inter-Company: box for Inter-Company Cash Transaction
    • Bank Code: Select Bank  Account that the Cash will be transferred From
    • Description: Brief Description of purpose of the transaction
  4. Click .Cash Transaction Entry – Update Batch

  5. Click .Cash Transaction Entry – Update Batch

  6. Enter all the required details
    • Reference:  A reference to identify the transaction; for example, the deposit reference.
    • Number:  A Number must be entered. For the purpose of a Bank Transfer, it is recommended to use a common default number that is not part of the an actual Number sequence.
    • Type: Payment
    • Date: The date the Transfer is being made
    • Transaction Date: The transaction date of the Transfer that will be used when matching to the Bank Statement
    • T/Type: PAY-Payment
    • Ledger: BK
    • Account: Bank to which the transfer will be deposited
    • Payer/Payee: Account that the Transfer is being withdrawn from
    • Narrative: Description of the Transaction/Transfer
    • Currency: USD-US Dollar
    • Amount: Amount of Transfer
  7. Click .
  8. The CB Transaction Entry – Details opens.Cash Transaction Entry – Details

  9. Click .Analysis Lines

  10. Enter the tax code and click .Bank Transfer

  11. Click .CB Transaction Entry – Update Batch

  12. Click .CB Transaction Entry – Batches

  13. Select the batch and click on the Post apply action, to post the batch.Posting the Batch

    The batch gets posted.

    Posted Batch

  14. When the batch is posted, three reports are generated and can be viewed from the Report Status.Posted Batch

  15. To open the reports click on the hyperlink of the report in Report Status.Posted Batch
