Bank Reconciliation

Module:

This document will show how to properly reconcile a bank statement.

Prior to step 1, the bank statement is going to have to be in the correct Excel format.

Bank Statement Conversion File


There are four fields that are necessary.

  • Number- The reference number.
  • Payer/Payee- Who the was addressed to.
  • Amount- Dollar amount Amount on the .
  • Type of Transaction- Typically, PAY(payment) or REC(receipt), User Defined

Before uploading delete the column header row.

If this is the first bank reconciliation, the opening balance will need to be entered.

  1. Go to > Setup Maintenance > Bank Accounts. Selecting Bank


  2. Open the line and then open the Statement Type drop down menu. Select Manual. Save the line. Then go to > Bank Reconciliation> Manually Enter Bank Statement and hyperlink into the correct statement.Selecting the Statement


  3. Click and adjust beginning balance.Adjusting Opening Balance


  4. Go to > Bank Reconciliation> Import Bank Statement and enter the appropriate information.Import Bank Statement


    • Bank- Select the bank that the statement is from.
    • Date- The statement date.
    • Reference- Reference to identify the statement.
    • Opening Balance- This field is not editable, unless this is the first bank reconciliation, in which the opening balance needs to be entered.
  5. Click .

    The bank statement will upload and a report will generate.

    Load Status


  6. Go to > Bank Reconciliation > Auto Bank Reconciliation and select the bank.Auto Bank Reconciliation Bank Selection


  7. Click .

    A report will generate and show what was reconciled and what wasn’t.

    Auto Bank Rec


        Both items weren’t able to be reconciled.

  8. Now go to > Bank Reconciliation> Bank Reconciliation to manually reconcile. Select the correct bank and click .Bank Reconciliation


  9. Hyperlink into the transaction that should be reconciled.Matching Transactions


  10. Select the matching transaction and Reconcile Selected Items using the action bar.
  11. Use the filter to help search for the matching transaction. Multiple items can be matched with one statement item.
