Positive Pay Report

Module:

This will generate the Positive Pay file that will need to be sent to the bank. This step will be taken after are printed.

  1. Go to > Reports > Positive Pay Report and enter selection criteria.
    • Enter a meaningful File Name. If the default file name is used, each time the Positive Pay Report is generated, a file with the same name will be created and will overwrite the file in COINS.
    • When all selection criteria have been entered, click .
    Figure 1: Positive Pay Report Selection

  2. To view the report, click the Report Status icon. Figure 2: Report Status

  3. To view the report in pdf format, click the US Positive Pay Report hyperlink.Figure 3: US Positive Pay Report hyperlink

  4. The pdf version appears as follows:Figure 4: US Positive Pay Report

  5. A .csv file is also generated and can be found on the Report Status Page. Click the My Files tab.Figure 5: My Files Tab

  6. Click the .csv file. This action downloads a .csv file with the same name to the system. Figure 6: .csv file

  7. Click .Figure 7: Save file

  8. Click .Figure 8: Open

  9. Choose Excel as the format and if desired, save and store this file on your local PC.Figure 9:  Output in Excel format
