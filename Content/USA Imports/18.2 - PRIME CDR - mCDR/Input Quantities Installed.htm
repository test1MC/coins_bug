Input Quantities Installed via mCDR

Module: Job Status/mCDR

This document demonstrates how to input quantities/units installed during mobile crew daily report (mCDR) entry, including the necessary setup required.

 

Setup

  1. In Cost Code Maintenance of the appropriate Job, the cost code(s) to input quantities installed with need to have a value entered for Units (Figure 1).

Figure 1: Cost Code Needs Units Specified

 

 

 

 

 

 

 

 

 

 

 

 

 

 

  1. Next, go to Job Status > Setup > Configuration > JC Parameters (Figure 2).
  1. CDRQTYIN (Display Quantity Input Fields) needs to be set to Y to enable entry of quantities installed on crew daily reports.
  1. ENTZBUD and EXCQBUD are parameters related to quantities installed to keep in mind that are system-wide and not exclusive to Crew Daily Reports.
  • ENTZBUD (Enter Qty Installed for cost codes with no budgeted qty?) is an optional parameter that can be set. If set to N, it will not allow entry of quantities installed on cost codes without a budgeted quantity. If set to Y, quantities installed may still be entered on cost codes without a budgeted quantity.
  • EXCQBUD (Allow user to exceed budgeted qty when entering Qty Installed) determines whether to allow users to exceed the budgeted quantity when entering quantities installed.

Figure 2: Job Status > Setup > Configuration > JC Parameters

 

 

 

 

 

 

 

 

 

 

 

mCDR Entry

  1. Inside the mCDR app, select the appropriate job card, or add a new one if necessary (Figure 3).

Figure 3: Tap into Job Card

 

  1. Once inside the job card, tap the + symbol and select Add Report (Figure 4).

Figure 4: Add Report

  1. Enter the appropriate information for Report Settings and Save (Figure 5).

Figure 5: Report Settings, Save

 

  1. Select the newly created daily report (Figure 6).

Figure 6: New Crew Daily Report

  1. Inside of the daily report, in the Labor entry screen, select the + symbol and Add Multiple Cost Codes (Figure 7).

Figure 7: Add Multiple Cost Codes

 

  1. Select the appropriate cost code(s) and Save (Figure 8).

Figure 8: Select Cost Code(s), Save

  1. To input quantities installed, tap the section with the Unit listed below the cost code, and enter in the number/amount (Figure 9).

Figure 9: Enter Quantities Installed Amount

 

  1. When finished, tap the triangle in the upper, right corner of the screen and select Review and Submit (Figure 10).

Figure 10: Tap Review and Submit

  1. Click Next at the Review and Submit summary screen (Figure 11).

Figure 11: Tap Next

 

  1. Optionally, enter notes for the day (Figure 12).
  1. Enter the signature for the foreman (Figure 12).
  1. Tap the paper airplane symbol in the bottom, right corner of the screen to submit the crew daily report (Figure 12).

Figure 12: Notes, Signature, Tap Paper Airplane Symbol

  1. The daily report will now show as Submitted (Figure 13).

Figure 13: Crew Daily Report is “Submitted”

 

Review and Approval

  1. The quantities installed entered on the cost codes for the job will not be official until the daily report is “approved.”
  1. Go to Job Status > Daily Reports > Review and Approval (Figure 14).
  1. There is a Qty column on the workbench that displays the quantities installed amount for each daily report (Figure 14).
  1. Select the appropriate daily report, select Approve Current Daily Reports from the drop-down on the bottom action bar, and click the arrow to “Apply Action” (Figure 14).
  1. Alternately, click the date hyperlink of the crew daily report to review the cost codes/quantities installed in detail in the Cost Codes tab, before approving (Figures 14 and 15).

Figure 14: Approve from Workbench

Figure 15: Approve from inside Crew Daily Report

 

  1. Make sure the Action is Approve and move the daily report over to the Selected section (Figure 16).
  1. Click Next in the bottom, right corner of the screen (Figure 16).

Figure 16: Approve, Click Next

 

 

  1. Next, hover over the printer icon and click on Approve Current Daily Reports (Figure 17).

Figure 17: Click into “Approve Current Daily Reports” Report

 

  1. The report should read that the quantities installed were approved (Figure 18).

Figure 18: “Approve Current Daily Reports” Report, Qty Installed Approved

 

  1. The quantities entered on approved crew daily reports will show up on reports and inquiries for the job.
