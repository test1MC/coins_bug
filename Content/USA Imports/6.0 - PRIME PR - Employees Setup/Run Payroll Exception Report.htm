Run Payroll Exception Report

Module: Payroll

The information on this report helps to determine if time has been missed for an employee during timecard entry, before payroll is processed and posted. This report should be run before Payroll Computations are processed each payroll period. Run this report after running (and fixing any errors) in the Missing Timecard Report (see Run Missing Timecard Report).

  1. In the Payroll Frequency, go to Payroll > [Payroll Frequency] > Reports > Payroll Exception Report.
  2. The Period Type must be selected, all other filters are optional. If desired, filter on the optional criteria in the Employee Selection tab, or change the Output Options tab.
  3. Click to run the report.Find and Run Report

    Selection Criteria

  4. To view the report go to the icon in the top left and click the hyperlink to enter the report.View Report

  5. Check and fix any errors found in the report. Run the report as many times as needed until no errors appear.
