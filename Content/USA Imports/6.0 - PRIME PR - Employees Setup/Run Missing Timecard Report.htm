Run Missing Timecard Report

Module: Payroll

The Missing Timecard Report allows you to list all employees who are active for the period and do not have a current timesheet on record. After inputting all of the timesheets for the frequency period, run this report to find any errors that may be in the data, go back to fix the errors and then re-run the report.

  1. Inside of the selected Payroll Frequency, Go to Reports > Missing Timecard Report. Enter any desired filter criteria or any changes to the output options from the tabs here.
  2. Click to run the report.Find and Run Report

  3. To view the report go to the icon in the top left and click the hyperlink to enter the report.View Report

  4. Check for and add any missing timecards found in the report (or verify the timecards will not be run for this payroll). Run the report as many times as needed until no errors appear. Find and Fix Errors
