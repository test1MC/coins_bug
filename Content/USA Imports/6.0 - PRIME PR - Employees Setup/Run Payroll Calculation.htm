Run Payroll Calculation

Module: Payroll

Once all Employees and timesheets have been added and the reports run to verify the data is correct, then run the Payroll Calculation.  This will give an “OK” in the report, if no errors are found. If errors are found, reset the payroll, fix the errors, and re-run the payroll calculation. If there are only a few errors, it is advised to reset ONLY the employees with errors.

  1. Under Payroll > [Payroll Frequency] > Calculation.  Click to run the Calculation for the Period.  In this example, we are processing Period 4.Run Calculation Report

  2. Click the icon located next to your username on the top right, and select the hyperlink to view the report.Enter Report

    This will display the PDF to see if the Calculation went through correctly or not. 

    View Report

  3. After reviewing these reports, return to Payroll > [Period Frequency] > Input > Timesheets. On the Net Detail tab, review the net totals, burden, etc. and see where the calculations may be off.Fix Errors

  4. Check and fix any errors found in the report. Run the report as many times as needed until no errors appear.
