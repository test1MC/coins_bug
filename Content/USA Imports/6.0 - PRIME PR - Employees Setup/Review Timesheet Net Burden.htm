Review Timesheet/Net/Burden Detail

Module: Payroll

This is a spot check after running the Payroll Calculation. When reviewing large amounts of data it is useful to use the Excel export function at the bottom of the screen to the data in Excel format.

  1. Go to Payroll > [Payroll Frequency] > Period Maintenance. Enter into the Period Maintenance screen, which allows you to set up weekly payroll periods, then click the period hyperlink, to review the data.Figure 1: Period Maintenance Screen

  2. Go to the Timesheet Detail tab and review the employee’s timesheet details.Figure 2: Review Timesheet

  3. On the Net Detail tab, review the net amount details for an employee.Figure 3: Review Net

  4. On the Burden Detail tab, an employee's detailed burden information can be viewed.Figure 4: Review Burden

  5. After reviewing the data, correct any errors that are found. Remember: Reset the payroll to make changes to the employee. Then recalculate and review the employee again for errors.
