Employee Setup

Module: Payroll

Enter all employees receiving a paycheck into the system here. Employee Maintenance allows you to maintain and calculate payroll information for employees.

  1. Go to Payroll > Employee Maintenance. Add a new Employee to the system.Employee Maintenance

  2. This will open the General Tab. Enter all pertinent employee information here. All mandatory fields have been highlighted in red. General Tab

  3. Navigate to the Profile Tab. The Pay Interval used should match the company Pay Frequency. You need to create a payroll in the frequencies you will be utiling in order for the drop down to populate.  If a title is added, the EEO may or may not default in per your setup. Profile Tab

  4. Navigate to the Tax Authority Details Tab. The exemptions, withholdings and adjustments for the Federal, State and Local level will vary by location and employee. State Default also holds the employee’s WC code, this is a mandatory field. Tax Authority Detail

  5. Navigate to the Payment Tab. Set the employee payment preferences here. Payment Tab

  6. Navigate to the Standard Rates Tab. Set any Standard Rates here.Standard Rates

  7. Navigate to the Other Tab. Enter the employee’s home address and additional info here. When finished, click SAVE.Other Tab

  8. Once the employee is saved, the entry will now appear in the Employee Maintenance list. It is now necessary to click on the newly created employee hyperlink to set up the employee's Earnings Categories and Pay Calculations.  Click the Employee Link

  9. There are now additional Tabs that will bring up new areas where data can be entered. By clicking each tab, additional tabs may appear below. The Employee Details Tab and the sub-tabs have been previously set up.General Tab

  10. To set up an earnings category, click on the earnings category tab. Click . Select the earnings category by clicking . This will bring up all of the earnings categories to choose from.  Choose a category, enter an effective date, (Cost method and Analysis are optional) Enter a pay rate (if hourly the overtime and double time will auto calculate), and finally click the box, Primary, for the main pay rate.  ing this will set the earnings category and rate that will automatically default in when entering a timecard and be used to calculate annual salary in HR. You can define costing for all earnings categories, i.e. VA, SI, PTO. Note: If using the memo earnings category UNPAID do not associate with a rate. Earnings Categories Tab

  11. Next, on the State Details Tab, this will all default in from the Tax Authority Tab, double check that a Work Comp code has been previously set up for all employees.State Detail Tab

  12. On the Pay Calculations Tab, set up calculations that will not be generated in HR, such as garnishment, uniforms, child support, etc. Also set lifetime, annual or monthly maximums for loans, garnishment, etc. Set Item basis for Percent Calculations and Item for Max if necessary.

    • Excluded Earnings Categories will default from Pay Calc file, and modify if necessary.

    • If Pay Calc is EFT related, set up Account & Routing Number in Employee > Payment > Additional Bank Details. Setup Remittance information if Invoices will be created when payroll posts.Pay Calculation Tab

  13. If you are not using the HR module, then this data can be completed here in the PR module. Information entered here will be pulled over to HR.HR Detail Tab

  14. This tab will hold employees pay history, once employee has been paid. Pay History Tab

  15. The Employee record has now been created. If there is an HR module it can be now be set up in the Employee Workbench.
