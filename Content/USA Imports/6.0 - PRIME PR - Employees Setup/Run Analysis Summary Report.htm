Run Analysis Summary Report (Costing Workbench)

Module: Payroll

On the Costing Workbench, use the drop down arrow and change the action bar to Analysis Summary Report to run this report. The Batch Analysis Summary Report provides a detail breakdown of all costs associated with a Payroll run. Costs are displayed by Analysis , cost code, or WBS, and by General Ledger. This information is provided prior to completing the payroll process and allows you to correct any data that may be in error prior to posting.

  1. Go to the Costing Workbench on Payroll > [Payroll Frequency] > Costing Workbench. Select Analysis Summary Report from the drop down and use to run the report for the open period. Click when prompted to proceed.Figure 1: Run Report

    Figure 2: Continue

  2. These reports will allow you to verify Employee's time and costing of jobs.  You can clearly see all Union reports and make sure that people's rates (regular vs. overtime) were calculated correctly.Figure 2: Review the Report

  3. Again, if errors are found, reset the payroll. Fix the errors. Recalculate the payroll and run the Analysis Summary Report again to check if the errors have been fixed.
