Run Edit Summary Report (Costing Workbench)

Module: Payroll

On the Costing Workbench, use the drop down arrow and change the action bar to Edit Summary Report to run this important report. The Batch Edit Summary Report provides a summary view of all components of Net Pay and Burdened Gross Pay.

 

  1. Go to the Costing Workbench under Payroll > [Payroll Frequency] > Costing Workbench. Click the Edit Summary Report and then click . Click to run the report.Edit Summary Report

    Continue

  2. This report will allow you to verify Employee's time and costing of jobs.  You can clearly see all Union reports and make sure that employee's rates (regular vs. overtime) were calculated correctly.Review the Report

  3. If errors are found, reset the payroll. Fix the errors. Recalculate the payroll and run the Edit Summary Report again to check if the errors have been fixed.
