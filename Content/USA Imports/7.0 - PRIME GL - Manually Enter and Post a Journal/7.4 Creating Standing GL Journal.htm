Creating Standing GL Journal

Module: General Ledger (11.06) [Standing Journal]

This document will demonstrate how to setup automatic transactions used for regular entries such as monthly payments.

  1. To setup a standing journal, use General Ledger > Standing Journals > Standing Journal Setup.  Standing Journal Setup

  2. Click .
  3. Fill out the Journal information.
    • Journal Number: The code to identify the standing journal.
    • Journal Description: The description of the journal.
    • T/Type: Transaction Type, is a code that allows you to identify different categories of transaction. These codes will vary greatly from one company to another. See your Configuration > Transaction Type for details.
    • Fixed: If the debit and credit amounts will stay the same every month, the Standing Journal can be designated as Fixed. However, you cannot add or amend the detail lines for fixed standing journals.  Only the original standing journal configuration can be changed.
    • Reversed: If the journal is flagged as Reversed, at posting, a transaction of the opposite amount will be generated for the following month.
    • Debit: Debit amount that will be posted.
    • Credit: Credit amount that will be posted.
    • Next Post: Refers to the in which this journal begins. This updates automatically each time the journal is posted. This field will show amber if the next period is not the current GL period.
    • Start: The first the batch will be created
    • End: The last period the journal batch will be created.
    Standing Journals - Add

  4. Click .Standing Journals

  5. Click on the journal code hyperlink, this will give you the Journal Detail screen for your newly created journal.Standing Journal 10

  6. Click .
  7. Enter the required details:
    • Change the first selection toggle to match the type of account to which the journal is being entered.
    • A (Asset), G (GL), T (Tax), W (Workshop), C (Contract)
    • The Analysis is the GL account to which the entry is posted
    • Enter the debit amount in Value.
    Standing Journal 10 – Debit Entry

  8. Click .
  9. Add the matching opposite credit entry so the batch balances.Standing Journal 10 – Credit Entry

  10. Once the batch balances, click .Standing Journal Entry

  11. To post a previously created standing journal, go to

    General Ledger > Standing Journals > Enter Standing Journal.

  12. Select the journal that is ready for posting and select ready to post. Then apply the action.Enter Standing Journal

  13. The Ready to Postbox gets ed for the selected journal.Enter Standing Journal – Ready to Post

  14. Go to General Ledger > Standing Journals > Post Standing Journals.
  15. Enter the required fields:
    • From – To: The code to identify the standing journal.
    • Period: The end date of the . The period is automatically filled with the current .
    Post Standing Journals

  16. Click .
  17. Click  the icon to check the status of the Post Standing Journals action.Post Standing Journals

  18. To verify that your standing journal entry has posted correctly, go to

    General Ledger > Standing Journals > Standing Journal Status Report.

  19. Select your standing journal in the From and To boxes, or the exact Match box.
  20. Click .Standing Journal Status Report

  21. Go to Report Status to view the report.Report Status
