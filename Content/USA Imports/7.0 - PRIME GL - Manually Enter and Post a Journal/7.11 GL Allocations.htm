GL Allocations

Module: General Ledger

This document illustrates how to set up GL allocations.

  1. Go to General Ledger > Allocations > Allocation Types
  2. Click .
  3. Choose a type of Allocation Method from the drop-down menu

    The Allocation Methods are:

    • GF = Fixed GL Allocations.
    • GV = Uses allocations based on GL account balances.
    • JC = Costs based on cost codes for specific jobs.
    • JV = Allow allocation of GL/JC amounts to jobs.
  4. Click (For this example, we are going to choose GF method)GL- Allocation Type

  5. On the Allocation Types – Add tab
    • Enter in a unique, user-defined code for the allocation type.
    • Enter in a Description of the allocation type (Optional).
    • Group: The group to which this allocation type belongs.
    • Override Credit Account: The account to be credited when a journal is processed (If GF, GV or JV allocation method is being used it is optional and the accounts added in the GL allocation in the main tab will be used).
    • Select the GL accounts using the fields Starting Code, Ending Code or Matches Code. And then enter the percentage in the % to Allocate to be credited from the accounts selected.
    • Click .
