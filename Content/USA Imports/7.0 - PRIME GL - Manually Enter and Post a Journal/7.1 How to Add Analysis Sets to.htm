How to Add Analysis Sets to the GL for Reporting

Module: General Ledger

The purpose of this how to is to link analysis sets to the GL to make reporting easier.

  1. Go to General Ledger > Chart of Accounts > Chart of Accounts Setup Maintenance > Account Analysis Sets.Analysis Sets


  2. Click on the Type drop down screen located at the bottom of the page. Select Income Statement and hit Enter.
  3. Click the button to export the data to an Excel file. Export Income Statement

  4. Then open by clicking on the file in the bottom left of the screen.
  5. Go to General Ledger > Chart of Accounts > Account MaintenanceExport Account Maintenance

  6. Bulk open the Account Maintenance file. To do this first click the button then click . Matching Analysis Sets with GL Accounts


  7. Copy and paste the Analysis Set code onto the exported Account Maintenance file. Make sure the code is appropriate for the description of the account. Paste the code under the Income Statement column (column O). Do this for all the GL Accounts.

    A code can be distributed to multiple GL accounts

  8. After all GL accounts have a code assigned, copy the Account Maintenance file back into . To do this, copy the file, go to and paste it into the box. Then click . Save


  9. Alternatively, the same process can be done on an single account by going to General Ledger> Chart of Accounts> Account Maintenance. Open the account you want to add an analysis code to. Open GL Account

    Assign a Code

  10. Under the Account Analysis tab, select the appropriate code and click .
