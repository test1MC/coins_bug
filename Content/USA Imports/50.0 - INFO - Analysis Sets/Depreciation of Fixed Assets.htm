Depreciation of Fixed Assets

Module: Fixed Assets

This document describes how depreciation of fixed assets works in COINS.

For further reference, see the how-to documents for “Adding an Asset” and “Running Depreciation.”

 

  1. How an asset depreciates is based on certain factors specified on the Asset:
  • Initial WDV (written down value): The Book Value that depreciates from
  • Start Date: When the asset starts depreciating
  • Depreciation Method: How the asset depreciates
  • Minimum Book Value:  The value an asset depreciates to (the salvage value)
  1. We will demonstrate by adding an asset.
  1. If not using the AP Reconciliation for capital purchases - : When adding an asset, the Minimum Book Value is entered in the Purchase tab (Figure 1). (if using capital purchase reconciliation – please refer to the How To Document for Capital Purchase (Fixed Asset) Invoice Reconciliation
