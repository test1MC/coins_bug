Formulas for “At Completion” Methods in OA

Module: Forecasting, Job Status

 

In Job Status, Standard Cost Types can be defined that will be the basis for cost codes created when new jobs are initially set up. The Forecasting tab of the these Standard Cost Types include choosing a method by which COINS OA will define the Calculated Cost at Completion (CCAC) and the Estimated Cost at Completion (ECAC).

Each of the various methods are calculated differently, as described below:

 

Calculated Cost at Completion:

 

Greater Cost - The Greater in value of these two:

  • Total Commitment = Actual Costs + Outstanding Commitments + Accrued Costs
  • Forecast at Completion

Budgeted Hourly Cost:

  • Actual Costs + (Budgeted Costs/Budgeted Hours) X (Forecast Hours – Hours to Date)
  • Which is more simply: Actual Costs + (Budgeted $/HR X Forecast Hours to Complete)

Actual Hourly Costs:

  • Actual Costs + (Actual Costs/Hours to Date) X (Forecast Hours – Hours to Date)
  • Which is more simply: Actual Costs + (Actual $/HR X Forecast Hours to Complete)

Budgeted Unit Cost:

  • Actual Costs + (Budgeted Costs/Budgeted Units) X (Forecast Units – Units to Date)
  • Which is more simply: Actual Costs + (Budgeted $/Unit X Forecast Units to Complete)

 

Actual Unit Costs:

  • Actual Costs + (Actual Costs/Units to Date) X (Forecast Units – Units to Date)
  • Which is more simply: Actual Costs + (Actual $/Unit X Forecast Units to Complete) 

 

Estimated Cost at Completion

 

Forecast:

  • Forecast at Completion

 

Calculated Cost at Completion:

  • Calculated Cost at Completion (See Above)

 

Greater Cost - The Greater in value of these two:

  • Calculated Cost at Completion (See Above)
  • Forecast at Completion

 

Forecast and Overages:

  • If Actual Cost is greater than the Forecast at Completion:
  • Calculated Cost at Completion (See Above)
  • Otherwise:
  • Forecast at Completion

 

 

Forecast and Unexpected Costs:

  • If both the Budgeted Costs AND the Forecast at Completion is Zero:
  • Calculated Cost at Completion (See Above)
  • Otherwise:
  • Forecast at Completion

 

Calculated Cost at Completion and Adjustments:

  • If the Forecast Adjustment is Zero:
  • Calculated Cost at Completion (See Above)
  • Otherwise:
  • Forecast at Completion

 

Calculated Cost at Completion, Adjustments, and Overages:

  • If the Forecast Adjustment is Zero OR the Actual Costs are Greater than the Forecast at Completion:
  • Calculated Cost at Completion (See Above)
  • Otherwise:
  • Forecast at Completion
