Custom Keyboard Shortcuts

Module: System [11.08]

This document demonstrates how to set up custom keyboard shortcuts for pages in COINS.

There are some pre-defined keyboard shortcuts for the most common actions in COINS OA, like Ctrl+Shift+S for saving the record. The problem is that some browsers disallow intercepting some shortcuts – for example, the standard Ctrl+Shift+N shortcut (Add/New) does not work in current versions of Google Chrome. Keyboard Shortcuts Configuration allows you to configure the list of COINS OA shortcuts to avoid browser-specific problems, and to define alternative shortcuts.

 

  1. Go to System > System Setup > Keyboard Shortcuts Configuration (Figure 1).
  1. Click Open at the bottom of the screen.

Figure 1: Click Open in System > Keyboard Shortcuts Configuration

 

 

 

 

 

 

 

 

 

 

  1. To replace a Shortcut, click into the field and press Delete on your keyboard (Figure 2).
  1. Then, enter the desired Shortcut by pressing the buttons on your keyboard in the desired sequence (Figure 2).
  1. Utilize an Alternative Shortcut field to add an additional shortcut for an action (Figure 2).
  1. Note: Only shortcuts with Ctrl, Alt, and/or Shift modifiers are allowed.
  1. Note: An error message will display if a shortcut that is already being used for another action is entered.
  1. To restore keyboard shortcut defaults, click the Restore Defaults button (Figure 2).
  1. Also, see the Toggle Help (side frame help) for reference.
  1. Then, click Save in the bottom, right corner of the screen (Figure 2).

Figure 2: Edit/Add Keyboard Shortcuts, Click Save

 

  1. For changes to take effect, you may need to refresh the page and clear your browser cache.
