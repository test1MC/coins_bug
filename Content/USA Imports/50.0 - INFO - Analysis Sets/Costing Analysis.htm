Costing Analysis

Module: Accounts Payable

This Information sheet will explain the types of Costing Analysis used on invoices.

Figure 1: Costing Analysis


 

 

 

 

 

 

 

 

There are four types of Costing Analysis:

Job- Selecting job will cost the invoice to a job. In the next field a job must be selected. For example, if ordering material for a job this option will be used. The Analysis String will be Job.CostCode.CostCategory

Figure 2: Job Analysis String


 

 

 

 

 

 

 

 

 

 

 

 

 

Department- This option could be selected if the invoice is getting costed to a GL Account. For example, an invoice for office supplies can be costed to a department within the company. A division will have to be selectred in the adjacent field if deaprtmenrt is selected. The Analysis String will be Division.Center.Natural.(GL Account)

Figure 3: Department Analysis String


 

 

 

 

 

 

 

Asset- Selecting asset is when the invoice gets costed to an asset. For example, adding a dump-body to a vechicle. The Analysis String will be Asset.StandardAssetCostType.

Figure 4: Asset Analysis String


 

 

 

 

 

 

Workshop- Selecting Workshop will require an equipment service ticket. This is typically used for repairs to an asset. **Note** Not every company will have a workshop department. The Analysis string will be WorkshopTicket.WorkshopCostType

 

 

 

Figure 5: Workshop Analysis String
