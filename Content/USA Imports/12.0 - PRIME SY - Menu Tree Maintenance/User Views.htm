User Views

Module: System

User views allow different users to see different lists of companies and projects. For example, COINS may hold information about thousands of companies, but staff at a local office may only need access to companies they actually deal with. Staff at a different local office may need to see a different list of companies, and staff at a regional office may need to see a larger list that includes all the companies visible to the local offices. When turned on, Paramater CI/AUTOVIEW will automatically create these user views.

Setting Up User Views

Note:  User Views can be seen in two places – inside a User’s Record and on the Maintain User Views Menu Function.

Option 1 – User Maintenance

  1. Within a User Record, click on the User Views tab. Here you can see the User Views that have been auto created when the Parameter CI/AUTOVIEW is set to yes. If it is set to no or you would like to change the view for a user, go to Step 4 below.
