Analysis Set Creation

Module: Job Status (11.07) [Setup]

  1. Go to Job Status > Setup > Configuration > JC Configuration.

Figure 1: JC Configuration

 

  1. To add the new Analysis Set, select the Job Analysis Sets tab and click Open located on the bottom of the page.

Figure 2: Job Analysis Sets

 

  1. Add the new Analysis Set, click the Save button.

Figure 3: Update

 

  1. Next, define the options for what the analysis set can be set to under Job Status > Setup > Setup Maintenance > Job Analysis Sets.

Figure 4: Filter

 

  1. The selector at the bottom of the page will have options for all of the different analysis sets. Choose the analysis set we want to create options for (Building Type), then click the Add button located at the bottom center of your page.  Add all the options you want to have available:

Figure 4: Job Analysis Sets

 

  1. Click the Save button to the left after entering each line. 

 

  1. If you want to make this analysis set optional, then you need to add a blank line.

Figure 4: Job Analysis Sets
