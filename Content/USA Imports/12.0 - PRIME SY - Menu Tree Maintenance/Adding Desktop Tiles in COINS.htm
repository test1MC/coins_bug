Adding Desktop Tiles in COINS OA

Module: System (11.07) [Home Page]

 

  1. To add tiles, use the Magnifying Glass to search and choose any function you like. In the example below. For example, let us choose the Job WIP Adjustment Inquiry in the Job Status module.

Figure 1: Home Page

 

 

  1. In the chosen function (in our example, WIP), go to the dropdown arrow at the top corner beside the user silhouette. 

Figure 2: History icon

 

  1. Hover the mouse over the History, to get a list of 10 functions/pages/reports

Figure 3: List of Functions/Pages/Reports

 

  1. Click on the Add to Desktop icon on any of the functions on the list.

Figure 4: Add to Desktop

 

  1. This brings up the Add to Desktop window.

Figure 5: Add to Desktop window

 

 

  1. The “Add To” drop down menu will let you choose in which Tab/Section this tile belongs.

Figure 6: Add To

 

  1. The Subtitle field will remain grayed-out unless override is checked, refer above. If it is, you can add a custom Subtitle to the tile. Otherwise, the tile will pull in the module name as the Subtitle.

Figure 7: Override and Subtitle

 

 

  1. Select how you would like the report to open.

 

  1. Select the background and the text (foreground) color.

 

  1. The icon will also auto-populate based on the function, or you can choose your own.

 

  1. Select the fields that the report should have.

Figure 8: Open In, Background/Foreground Colors, Icon, Selected Fields

 

 

  1. When all details are entered, click on Next.

Figure 9: Saving the customized Desktop tab

 

  1. You can move on to the next tile if you like, but to see the changes you’ll have to click F5 to Refresh. If the refresh doesn’t take you to the desktop, click Show/Hide Desktop.

 

  1. You can see below that the tile was added to PM Home Page, in the selected colors, with the selected title, overridden subtitle and icon.

Figure 10: WIP Report Desktop tile

 

  1. When we click on the WIP Desktop tile, the WIP Report opens in a separate window. This is because we had selected Open In Window option.

Figure 11: WIP Report selection criteria page
