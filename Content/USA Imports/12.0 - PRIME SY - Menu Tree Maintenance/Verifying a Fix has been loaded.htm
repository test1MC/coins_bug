Verifying a Fix has been loaded

 

System: System Administration (V11.07)

 

  1. Go to System > System Administration

 

  1. Click the “Fixes Inquiry Tab”:

 


Figure 1: Fixes Inquiry Tab

 

3.   This will populate a list of all fixes that have been loaded to date. You can also search to see if a fix has been loaded by using the search function drop down menu in the lower right hand corner, and by entering in the number of the fix.
