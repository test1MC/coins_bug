Changing Environment Color

Module: System

Environments in COINS can have different color palettes to help users differentiate them. This document demonstrates how to add/edit a color palette and how to apply it to an environment.

 

  1. Go to System > System Setup > Color Palettes (Figure 1). Color Palettes are created and maintained here.
  1. Click the Open folder in the appropriate row to edit the color scheme (Figure 1).
  1. Alternately, create a new palette by clicking Add at the bottom of the screen (Figure 1).

Figure 1: Add or Edit a Color Palette

 

  1. Enter a Palette name and Description (Figure 2).
  1. Select the desired/appropriate colors for the background and text (Figure 2).
  1. Click the floppy disk to Save (Figure 2).

Figure 2: Edit the Palette

  1. Next, go to System > System Setup > SY Parameters (Figure 3).
  1. Open the PALETTE parameter.
  1. Change the Value to the appropriate Palette (Figure 3).
  1. Click the floppy disk to Save (Figure 3).

Figure 3: SY Parameter PALETTE

 

  1. If you have multiple companies, you can have different color schemes for different companies, to differentiate easily.
  1. To have a company-specific color palette, select the “PALETTE” parameter and click on the arrow for the Create Company Specific option (Figure 4).

Figure 4: Create Company Specific Color Palette

  1. To see the changes, refresh the page (Figure 5).
  • Sometimes the browser cache needs to be cleared before the changes appear.

Figure 5: Refresh the Page
