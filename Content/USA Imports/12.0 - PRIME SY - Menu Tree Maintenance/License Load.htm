License Load

Module: System [License Maintenance]

This document shows you how to use the Load License function in OA System > Module/License Maintenance to load license changes provided by COINS in the form of a CSV file.

  1. In COINS OA, go to System > System Setup > Module/License Maintenance.

Figure 1: Module/License Maintenance

 

  1. Click the Load License tab to reach the Load License screen

Figure 2: Load License

 

 

  1. Click the Choose File button to find and select the CSV file containing the license details.

Figure 3: Select License File

 

Figure 4: Choose correct CSV file

 

 

  1. Once you have chosen the correct file click the next button at the bottom right hand corner of the page.
  • COINS first validates the license. If the license is invalid, an appropriate message is displayed and the license is not saved.
  • If the license validates correctly, the license details are updated.

 

Figure 5: Load License File
