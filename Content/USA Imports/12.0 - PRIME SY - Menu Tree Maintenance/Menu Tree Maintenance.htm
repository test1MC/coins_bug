Menu Tree Maintenance

Module: System

This document demonstrates the process of changing the menu tree in a COINS environment - to add, hide, or vertically move modules.

 

  1. Go to System > System Setup > Function Maintenance (Figure 1).
  1. Use the filter to search for US-WMAIN and click the hyperlink (Figure 1).

Figure 1: Click “US-WMAIN” in System > Function Maintenance

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

  1. Click into the Menu Items tab (Figure 2).
  1. The functions listed here represent the modules displayed on the menu tree, depending on the functional security of the user (see the arrows in Figure 2).
  1. Standard COINS function codes begin with "%W". In this example, “%WFM” represents the Facilities Management module folder for the all the functions under it (Figure 2).
  1. To remove a module from the menu tree, select the appropriate row and click Delete (Figure 2).

Figure 2: Remove a Module from the Menu Tree

 

  1. After the browser page is refreshed, the module folder is now missing from the menu tree (Figure 3).

Figure 3: “Facilities Management” Absent from Menu Tree

  1. To add/include a module in the menu tree, click Add at the bottom of the screen (Figure 4).
  • Alternately, to add a module at a specific spot in the menu tree, select the appropriate row to add the module above, then click Insert.
  1. Enter the function (module) and click the floppy disk to Save (Figure 4).

Figure 4: Add a Module to the Menu Tree

 

  1. Refresh your browser for the changes in the menu tree to update.
  1. The Facilities Management module has now been added back to the menu tree (Figure 5).
  1. Because it was “added,” not “inserted,” it will display at the bottom of the menu tree (Figure 5).

Figure 5: “Facilities Management” Present in the Menu Tree

  1. To adjust the placement of modules vertically in the menu tree, select the appropriate row(s) and click Up or Down (Figure 6).

Figure 6: Select Row(s), Move Modules Up or Down

 

  1. Refresh the browser to see the changes to the menu tree.
  1. Now the Facilities Management module is back in its original place, with all the modules alphabetical order (Figure 7).

Figure 7: “Facilities Management” Back in Original Spot
