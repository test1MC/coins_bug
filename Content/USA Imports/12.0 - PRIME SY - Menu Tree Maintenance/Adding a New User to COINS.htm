Adding a New User to COINS

Module: System

This document will explain how to add a new user to COINS. Users in the COINS system can perform varying amounts of actions, depending on the security access they are granted – which can be specified. This document covers how to enter in that user’s information and how to add users into security groups.

 

  1. Go to System > User Maintenance > Users (Figure 1).
  1. Click Add at the bottom of the screen (Figure 1).

Figure 1: Add a New User

 

  1. This will bring up a blank Add User screen. Enter the User (username used to log in) and a display Name (Figure 2).

Figure 2: Enter User and Name

  1. In the Details tab, select the checkbox for Named User (Figure 3).
  • Named User means that a user will be able to log into COINS. There is also a finite amount of licensed “Named Users” that can be designated.
  1. Enter in a Password for the user (Figure 3).
  1. Optionally, enter an Expires date to force the user to change their password by a certain date, such as the first time that they sign in (Figure 3).
  1. Optionally, enter in an Email address for the user (Figure 3).
  1. Select the appropriate Prime Group, the first security group for the user (Figure 3). Other users who belong to this group will be able to see this user’s reports, post their batches, etc.
  1. Select AMERICAN for Language (Figure 3).
  1. Select the appropriate Prime Company, the company the user will connect to when they log into COINS (Figure 3).
  1. Enter the list of Companies this user can access (Figure 3). If a “*” is entered, the user can access all companies.
  1. Select the appropriate Security level (Figure 3). In tandem with the Prime Group, the security level determines whether users can modify each other’s batches, reports, and print requests. A higher number means higher access, and vice versa.

Figure 3: Enter the User’s Information

 

 

 

  1. Before adding a new user to COINS, decide which group (or groups) they will belong to. If an appropriate group does not exist, add a new group. Click on the Groups tab.
  1. In the Groups tab, select the appropriate security groups to add to the user, and click the arrow to move them to the Selected section (Figure 4).
  1. Then, click Save in the bottom right hand corner of the screen to save the user (Figure 4).
