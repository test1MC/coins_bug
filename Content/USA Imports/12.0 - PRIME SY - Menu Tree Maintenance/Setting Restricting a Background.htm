Setting/Restricting a Background Image for All Users

Module: System

 

Setting the Background

  1. Go to System > User Maintenance > Users

 

  1. Select all users for which you want to change the desktop image and click Multi

Figure 1: User Selection

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

3.    Desktop images is the field that we will use to change the background image. To             change this, click the check box and select the image that you want all the selected          users to have. Then click Save.

Figure 2: Image Selection

 

4.     All of your selected users will now have this image as their desktop. This does not

        prevent users from changing this via the preferences tab. If you want to restrict

        access to this tab, continue reading.

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

Adding Your Own Images

1.First find the image that you would like to set as your background image and save it somewhere on your computer. Then go to: System > System Setup > Background Images.

 

2.On the “Put File” tab, click the Choose File button and select your photo. Click Next.

 

Figure 3: Choose File

 

3.You should now see your file on the Background Images tab.

Figure 4: Background Images tab

 

4.Once your photo is on this tab it is now on the desktop image drop down.

Figure 5: Desktop Image Drop Down

 

NOTE:

If you are a cloud-hosted client, you will be able to pick your picture from this list, but it is not going to show up as a desktop background yet. It will just be a white background. Cloud clients run on a tow server system. Right now, the picture is on the wrong server. At this point, you have two options; you can contact COINS tech support and they can run the script that will transfer the pictures onto the other server or you can wait until the next day. This script runs automatically every night, so your picture will be ready to use the next day.

 

 

 

 

 

 

 

 

 

 

 

 

Hiding the Desktop Image Field

 

1.The easiest way to keep users from changing the imaged would be through field access maintenance.

 

2.Go to System > System Setup > Field Access Maintenance.

 

3.Click Add and enter the field name and set access to read only or hidden.

 

 

Figure 6: Field Access Maintenance

 

4.Putting company 0 sets this access to all users in all companies. Then click the Save Icon.

 

5.If down the road you want to change the picture, you will have to come back to this screen and set access to update then you will be able to change it.
