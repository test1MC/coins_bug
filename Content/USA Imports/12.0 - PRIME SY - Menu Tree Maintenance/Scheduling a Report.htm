Scheduling a Report

Module: System

Every report in COINS has the ability to be scheduled.

  1. On any report, go to the Output Options tab. Every report has the same options on this page regardless of what is available on the other tabs.

Figure 1: Output Options

 

  1. Click the Scheduled button
  • It may also be useful to select the email checkbox. This will automatically send the report via email as scheduled. This allows the recipient to view the report without logging into COINS.
  • The email checkbox is not required for scheduled reports and non-scheduled reports can also be emailed in the exact same way.
  1. Click the NEXT button.

Figure 2: Schedule and/or Email

 

  1. Enter the time of day you would like the report to be scheduled.
  • Enter start and end dates in the From and To fields if desired.
  • Add a description of the report if desired.
  1. Click on the lookup button next to Schedule to view a list of standard intervals

Figure 3: Schedule Details

 

  1. Click on the Hand Icon to choose an existing standard interval

Figure 4: Choose Standard Interval

 

  1. To create a new interval, select the Type and then click on the ADD button.

Figure 5: Add Standard Interval

 

  1. Fill in the required information
  • This screen will vary depending on which Type is chosen.
  1. Click the Save icon when finished.

 

Figure 6: Add Interval Details

 

  1. Once all fields are completed, click SAVE.

Figure 7: Save Scheduled Report

 

  1. The report now appears in Report Status > Scheduler tab.
  1. To modify or view the scheduled report, click on the Open icon.
  1. To run the report on demand, click the Run Now icon.

Figure 8: View/Modify and Run Now

 

  1. NOTE:  If you are running a report based on a fiscal period, a Fiscal Period option will appear on the Scheduler Update screen.
  • To run the report for the current fiscal period, choose This Fiscal Period.
  • Choose Leave to always run the report on the period you originally defined. Using this option, the period you are reporting on will not change as time progresses.

Figure 9: Fiscal Period
