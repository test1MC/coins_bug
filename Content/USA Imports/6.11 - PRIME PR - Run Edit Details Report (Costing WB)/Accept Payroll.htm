Accept Payroll

Module: Payroll

After all of the reports have been run and checked, the payroll can be accepted. Please be sure to have fixed any errors and have re-run the reports and verified it is correct before this step. Accept Payroll lets you accept the [payroll frequency] pay calculation for each employee; they are ready to be paid. You can either accept individual employees to be paid or a number of employees to be paid.

  1. After all the reports look good and Payroll is hitting the accounts correctly, Go to Payroll > [Payroll Frequency] > Accept Payroll.  Click to run the payroll.  Accept Payroll

  2. View the report by going to the icon in the top right.View Report

  3. Check and see if the report found in Report Status has given you the okay to Post your Batch. If not, find the error, fix it and repeat these steps until that is the case.Fix Any Errors
