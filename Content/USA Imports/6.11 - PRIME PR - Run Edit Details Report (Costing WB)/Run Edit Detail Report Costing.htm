Run Edit Detail Report (Costing Workbench)

Module: Payroll

The Batch Edit Detail Report provides a detailed view of all components of Net Pay and Burdened Gross Pay.

  1. Go to the Costing Workbench on Payroll > [Payroll Frequency] > Costing Workbench. Select Edit Detail Report and use to run the report. Edit Detail Report

  2. Click to run the report.Continue

  3. These reports will allow you to verify Employee's time and costing of .  You can clearly see all Union reports and make sure that people's rates (regular vs. overtime) were calculated correctly.Review the Report

  4. If errors are found, reset the payroll. Fix the errors. Recalculate the payroll and run the Edit Details Report again to check if the errors have been fixed.
