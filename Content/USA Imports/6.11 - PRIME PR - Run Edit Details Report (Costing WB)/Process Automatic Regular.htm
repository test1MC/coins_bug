Process Automatic / Regular Hours Overtime

Module: Payroll

This is only used when the company rule for overtime hours is anything over 40 regular hours. Process Automatic Regular Hour Overtime allows you to calculate overtime hours based on a regular hour limit. The overtime is calculated on the current unposted payroll period.

  1. Go to Payroll > [Frequency Period] > Processing > Process Automatic Regular Hours Overtime.
  2. Select the type of processing you wish to perform.
    • Process Overtime - Processes the automatic regular overtime hours.
    • Reset Hours - Resets hours to their original state for any that have had overtime processed.
    Run Report

  3. View the report. Be sure to check if the output status gives an OK, if not go back and correct any errors found. Review Status
