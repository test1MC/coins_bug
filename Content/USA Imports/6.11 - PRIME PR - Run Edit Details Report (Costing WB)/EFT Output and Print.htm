Output and Print

Module: Payroll

Output and Print allows you to output transactions onto the medium for those employees who are paid by , direct deposit.

  1. After the Payroll has been accepted. Go to Payroll > [Payroll Frequency] > EFT Output and Print. Click to run the report. EFT Output & Print

  2. Staying in the Report Viewer, under Report Status check the report that was generated for any errors.Review Report

  3. After reviewing the report go to Report Status and Select My Files, the last tab. Find the Output that has been processed by looking for the date and time of processing on the right hand side. You can view the file by clicking on the hyperlink.Review EFT Payslips

  4. Once the report is run, it is important to remember the .txt file which is created to save and send to the bank for all employee direct deposits.  This file will contain all information needed to direct the funds to all employee accounts by the bank.
    • Click the icon to bring up the Report Status page, and go to the My Files tab.
    • Be sure to check you have to correct file by looking at the date modified.
    • Give this file to the bank for the Electronic Funds Transfer () payment to the employees.
