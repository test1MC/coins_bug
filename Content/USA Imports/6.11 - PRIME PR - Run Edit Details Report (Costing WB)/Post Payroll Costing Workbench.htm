Post Payroll (Costing Workbench)

Module: Payroll

DO NOT PERFORM THIS STEP until 100% confident that all payroll calculations are correct. Make sure all reports have been run, validated, and accepted without errors before this finalation process is run.

  1. Go to the Costing Workbench under Payroll > [Payroll Frequency] > Costing Workbench to post the batch.  Make sure that the correct Period has been selected.  Using the dropdown arrow, select Post Batches and use to perform the action.  Figure 1: Post Payment Batch

  2. Posting reports are available for review in Report Status as well after the batch has been posted. Review all posting reports generated.
  3. This weekly (or other frequency) Payroll has now been successfully processed.Figure 2: View Posting Report
