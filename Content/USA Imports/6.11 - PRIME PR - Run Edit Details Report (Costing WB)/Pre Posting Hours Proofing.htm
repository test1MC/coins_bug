Pre Posting Hours Proofing Report

Module: Payroll

Pre Posting Hours Proofing Report shows the hours (Regular, Overtime, Double Time) and rates for an employee or (s) by day of the week for unposted time cards. It is useful for all users who enter time cards daily.

  1. Go to Payroll > [Payroll Frequency] > Reports > Pre Posting Hours Proofing Report to run the report. Under the Main Tab there is the ability to sort by or Employee. Changing this will bring up different filters. Go to Report

  2. If desired, you can filter by Employee by going to the Employee Selection tab and choosing the available criteria here. Otherwise simply click to run the report.Optional Filter

  3. The report is now generated and can be viewed by going to icon in the top right of the screen and selecting the report.Locate Report

  4. Check for Errors. Fix any problems that are found, by resetting, fixing and recalculating the employees.Check for Errors
