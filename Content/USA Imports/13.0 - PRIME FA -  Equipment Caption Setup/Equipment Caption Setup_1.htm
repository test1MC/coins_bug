Equipment Caption Setup

Module: Fixed Assets

The purpose of this document is to outline how to add date triggers and update the existing assets through the configurable asset load

  1. Go to Fixed Assets>Setup>Maintenance>Asset Trigger Definitions.

Figure 1: Asset Trigger Definition


 

 

 

 

 

 

 

 

 

  1. Click ADD.
  1. Fill in the appropriate fields.

Figure 2: Add Trigger Definition

 

 

 

 

 

 

 

 

 

 

  • Trigger- the name of the trigger being added
  • Description- brief description of the trigger
  • Data Type- the type of data the trigger represents. In this case, it is Date
  • Format- the progress display format to be used for the trigger
  • Trigger Type- this combined with the Value determines how the expiration value for the trigger is reset
  • Value- the reset value, which determines what the expires field on the trigger is reset to. The entry in Trigger Type governs how the trigger is reset.
  • Main Trigger- this is used if the new trigger being added is a secondary trigger
  • Validation- how COINS responds when the trigger is triggered. The options are:

None- no warning or error are given

Warning- a warning message appears but gives the user the option to continue anyway (select the asset)

Error- an error message appears and does not allow the user to continue (asset cannot be selected)

 

  1. Click SAVE.
  1. Go to Fixed Assets>Setup>Maintenance>Caption Maintenance

 

Figure 3: Select/Add a Caption


 

 

 

 

 

 

 

 

  1. Select a caption that correctly corresponds to the trigger that was just added. If there is no option that corresponds accurately to the newly entered trigger ADD a new one.

 

 

 

Figure 4: Select/Add a Caption


 

 

 

 

 

 

 

 

 

  • Code- Enter a code to identify the caption
  • Caption- the text that will be used as the prompt form the caption
  • Group- the caption group to be associated with this caption
  • Freq- how often the caption needs to be updated
  1. Click SAVE.
  1. Click the hyperlink for the appropriate caption.
