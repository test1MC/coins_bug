Equipment Caption Setup

Module: Fixed Assets

This document will walk through the basic process of adding a new Caption. These user-defined fields can be logical, decimal, integer, character, or date-based, and used to contain usage information about the Asset Groups to which they are assigned. They can be set to require scheduled updating.

 

  1. To create the Caption, begin by navigating to Fixed Assets > Setup > Maintenance > Caption Maintenance, and click Add.

Figure 1: Clicking Add on the Caption Maintenance Workbench

 

  1. Enter the Caption information:
  • Add a Caption Code and Description
  • Select the “Caption Group” which will be associated with this caption. These groups are defined on the Maintain Caption Groups Workbench, and are an easy way to organize captions.
  • The caption groups also defines whether captions that belong to the group can be updated or displayed at points in the rental life-cycle, and whether they can be linked to triggers. The U/W column on the Caption Group Workbench has these options:
  • blank = Captions can only be updated on the FA Asset Workbench.
  • Update = Captions in the group can be updated at various stages in the rental life-cycle (specified on the individual caption), and can be used to update triggers linked to them.
  • Warning = Captions in the group can be displayed at various stages in the rental life-cycle.
  • Choosing a Caption Group which is marked as Update or Warning will allow for additional tabs on the Caption, once it’s saved. See step #4.
  • Frequency controls how often the caption needs to be updated. This is used for reporting on due dates. Enter the numerical value, and the unit of time measure from the drop-down menu.
  • Type and Format define the kind of information to capture, and how it will be presented
  • Character = x(8) format (for example: abc123)
  • Date = 99/99/99 format (for example: 12/08/17)
  • Decimal = ->>,>>9.99 format (for example: 123.45)
  • Integer = ->,>>>,>>9 format (for example: 345)
  • Logical = Y/N format (for example: Y or N)
  • Checking the History box produces a link on the caption value field that provides a view into the history of the changes of the value. If this box is not checked, the history of changes will not be saved. The best practice is to check the History box for any caption that will change over time, such as Mileage, Hours Used, or a service maintenance caption, like Oil Changes.
  1. When all details are filled out, click Save. The system will assume another Caption is to be added immediately and will produce another blank Caption Details screen to be filled. If this is unnecessary, simply hit Undo.

Figure 2: Completed Caption Details

 

  1. Once saved, and back on the Caption Maintenance Workbench, click the hyperlink of the newly created Caption, and select the Functions Tab. This tab, and the Triggers tab, will only be visible given the Caption Group settings. See Step #2.
  1. The Functions tab simply defines at which stage of the rental cycle to capture values.

Figure 3: The Functions Tab of the new Caption

 

  1. If applicable, the Trigger Tab is used when associating the caption. To associate the new Caption to the appropriate Trigger, refer to the Link Captions to Triggers document.
  1. To add the new Caption to an Asset Group, navigate to Fixed Assets > Setup > Maintenance > Asset Groups, and click the hyperlink of the appropriate group.

Figure 4: Selecting the Hyperlink on the Asset Group

 

  1. In the group’s Caption Tab, use the Group filter at the bottom of the page to select the appropriate Caption Group to which the caption was assigned in Step #2.

Figure 4: Using Group Filter on Caption Tab of Asset Group Record

  1. After filtering for the right Caption Group, click the Add button.
  1. Type in the Caption Code, or use the lookup to choose it, to add the Caption to this Asset Group. A default caption value can be added here, but is not required.
  1. Hit Save when complete.

Figure 5: Typing Caption Code

  1. The system will assume another Caption is to be added immediately and will produce another blank line to be filled. If this is unnecessary, simply hit Undo.
  1. Adding the new caption here will allow it to be inherited to any new Asset added to this asset group.
  1. To update the caption on already existing assets, the configurable load process will be used. Please refer to the Equipment Mileage Update document for a good example.
  1. While on the Asset Record, another Caption-related field to consider is the “Caption Used for Service Meter,” found on the Details Tab

Figure 6: “Caption Used for Service Meter” field on Details Tab of Asset Group

 

  1. This field defines the caption that holds the meter value upon which service cycles will be based for the group. The caption needs to be a numeric data type and must be set up to retain history, as the history is used to work out historical rate of change used in calculating the expected next service.
  • This will only need to be set if overriding Equipment parameter SERVMETR, which is the default caption to use for service cycles.
