Configurable Asset Load

Module: Fixed Assets

The asset load will be used to update existing assets in the system.

  1. Open excel and create a new file.

The file format will look like the following:


Figure 1: File Format

 

After the file is filled out delete the first three rows.

  1. Example: Asset Trigger

Column A Row 1 à faa_code

Column A Row 2 to XX à the asset number to be updated

Column B Row 1à RW_faa_ftv^NameofTrigger|expires

Column B Row 2 to XX à Date the trigger expires

Figure 2: Excel Load Format


 

 

 

 

 

 

Save the document as a csv file (comma delimited).

**Note** This is not the only data that can be loaded using the configurable asset load. A wide range of data can be loaded.

 

  1. Go to Fixed Assets>Administration>Fixed Asset Load.

Figure 3: FA Asset Load


 

 

 

 

 

 

 

 

Figure 4: Asset Load Header(Validate)


 

 

 

 

 

 

 

 

 

  1. Select the excel file that was just created
  • Upload Mode- select validate
  • Delimiter- the character used to separate entries in the file (select comma)
  • Configurable- whether to use the configurable field format (check the box)
  • Mode- select the asset load to perform (select Update existing assets)
  • Purchase- the purchase post method (select Not posted)

Click Next.

  1. If there are no errors, rerun it but switch Validate to Load.
