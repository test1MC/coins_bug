Running Depreciation

Module: Fixed Assets

Steps 1-3 are setup. If depreciation is already set up, skip to step 4.

  1. Go to Fixed Assets>Setup>Configuration>FA Configuration.

Figure 1: Book Details


 

 

 

 

 

 

 

 

 

  1. On the Book Details Tab verify that the GL accounts are setup correctly

 

  1. Go to the Defaults Tab verify that Next Depreciation is set to the current fiscal period.

Figure 2: Defaults


 

 

 

 

 

 

 

 

 

  1. If the selected fiscal year is incorrect, click OPEN, and change to the correct year.

Figure 3: Fiscal Year Adjustment


 

 

 

 

 

 

 

 

 

  1. Click SAVE.
  1. Go to Fixed Assets>FA Asset Workbench.

Figure 4: Select Asset

 

 

 

 

 

 

 

 

 

 

 

 

 

 

  1. Hyperlink onto the appropriate asset and go to the Details Tab> Depreciation Tab


Figure 5: Depreciation Tab

 

 

 

 

 

 

 

 

 

 

  1. Click OPEN, and set depreciation preferences


Figure 6: Depreciation Preferences

 

 

 

 

 

 

 

 

 

 

  1. Click SAVE.
  • The setup in steps 1-3 will only need to be done once. To simply run depreciation, go to step 4.

 

 

 

 

 

  1. Go to Fixed Assets>Input>Depreciation>Calculate Depreciation.


Figure 7: Calculate Depreciation

 

 

 

 

 

 

 

 

 

 

  1. Click NEXT, and a report will be generated

 


Figure 8: Depreciation Report

 

 

 

 

 

 

 

 

 

 

  1. The report shows that the Steam Roller was depreciated.

 

  1. This can also be seen by clicking on the Calculated Depreciation Workbench which is right below the Calculate Depreciation function on the tree menu.


Figure 9: Calculated Depreciation Workbench

 

 

 

 

 

 

 

 

 

 

  1. Once the depreciation is verified correct go to Fixed Assets>Input>Posting>Post Depreciation Batch


Figure 10: Posting Depreciation Batch

 

 

 

 

 

 

 

 

 

 

  1. Select the appropriate batch and post using the action bar.
