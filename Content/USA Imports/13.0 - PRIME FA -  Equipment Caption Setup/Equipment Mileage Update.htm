Equipment Mileage Update

Module: Fixed Assets

Updating Asset fields can easily be accomplished using the Configurable Fixed Asset Load, and that process will be explained and outlined here, as it is used for updating an Asset’s Mileage field. If the caption does not currently exist, see the Equipment Caption Setup document to learn how to create a new one

 

  1. The process begins with finding the field name to be updated.
  1. Navigate to the page in question, here it’s Fixed Assets > FA Asset Workbench > Details Tab > Captions Sub-Tab.
  1. Use keyboard the shortcut to activate the Debug mode: [Ctrl+Shift+C], and hover over the blue “I” circle next to the appropriate field to read the field name.

Figure 1: Mouse Hover over Blue Circle next to Mileage Field on Asset Record

 

  1. This field reads as “RW_faa_caption|N^80”
  1. Using this field, and the filed name for the Asset Code itself (faa_code), wil be used to create a CSV file. This file will be uploaded into OA as a configurable load.
  1. In Excel, create a new spreadsheet and enter the following:
  • Row 1: field labels for the Asset Code and the Mileage
  • Faa_code
  • RW_faa_caption|N^80
  • Row 2 and onward: the actual Asset Code and the actual Mileage (example below)

Figure 2: Spreadsheet of Configurable Asset Load Format

 

  1. After adding all of the assets to the spreadsheet with the appropriate dates, save the file as a CSV (Comma Delimited).
  1. Navigate to Fixed Assets > Administration > Fixed Asset Load, and choose the file created in the previous step.
  1. Choose Validate as the Upload Mode, with the following settings:

Figure 3: Configurable Load Validation Settings

  1. Click “Next”, and then click OK on the pop-up warning to acknowledge that one file is being validated.
  1. After the process completes, check the report printer for the Fixed Asset Load report that was created. Make sure there are no errors, and all lines were counted and checked successfully.

Figure 4: Asset Load Validation Report

  1. When everything is confirmed good, repeat Step #9 (Figure 3) but change the Upload Mode to Load.
  1. Back on the Asset record, confirm the Mileage field has updated as expected.

Figure 5: Updated Mileage field on Asset Record

 

  1. Use the hyperlink on the mileage to see the update history.

Figure 6: Asset Record Mileage History
