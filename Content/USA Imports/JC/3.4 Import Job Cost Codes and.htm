Import Cost Codes and Budget

Module: Status

There are several methods of importing a Budget in . The suggested method, which will be described in this document, is Importing by Default Category/Year.

  1. This process begins with a -provided template. This will be a Budget-type specific template, modified in Excel, which will be saved as a Comma Separated Value (.csv) file before being uploaded into .
  2. Using the template headers as a guide, enter the number, cost code, beginning , and budget amount for each . Add new lines for each cost code that will exist on the . Do NOT include commas anywhere in the .csv file.
    • Note: Be aware that Excel will remove leading 0s when cells are not formatted as text. For example, if “00” is used as the Revenue code, and is typed into Excel, it will automatically switch the number to “0” - which will cause the data to be incorrect when uploaded into .
    • To reformat the Excel cells to text:
      1. Highlight the cells needed to be changed to text, or just select all (in the top left corner of the spreadsheet).
      2. Click the “Format” drop-down on the “Home” tab in the ribbon.
      3. Select “Format Cells” from the drop-down.
      4. In the “Number” tab, select “Text” and click “OK.”Reformat Cells to Text

  3. When all the lines have been added, delete the header line.Delete the Header Line

  4. Then, save the file as a .csv.Save the File as a .CSV

  5. In the tree menu, go to Status > Administration > Import Budget/Memo Data > Import Budget by Default Category/Year > Import.
  6. Then, click Choose File.Import Budget by Default Category/Year

  7. Select the .csv file created in step 4.Select .CSV File

  8. Choose “Comma” as the Delimiter option, to make the formatting of the .csv file correct.
  9. Choose “Initial Budget” from the drop-down as the Memo Type.Delimiter and Memo Type

  10. Fill in the correct .
  11. Create Cost Codes must be ed only if creating new cost codes.
  12. Create Sections must be ed only if using sections and new ones are needed to be created.
  13. Then, click at the bottom, right corner of the screen., boxes, and Click “Next”

  14. When the pop-up appears, click OK to confirm that one file will be uploaded.Click “OK” in the Warning Pop-Up

  15. At the Report Status workbench, it will say “Complete” if the file is imported successfully.
    • If it says “Complete (with errors),” click on the (Excel) symbol in that row. That will download an Excel spreadsheet that can be opened. It will list the errors in the upload.Report Status Workbench

  16. Next, check the budget and cost codes in the Workspace for Import Budget by Default Category/Year. This can be done by searching in Menu Search or going to Status > Administration > Import Budget/Memo Data > Import Budget By Default Category/Year > Workspace through the tree menu.Menu Search for “Workspace...”

  17. The Workspace for Import Budget displays a temporary area showing all the records that have been imported. This page will let the user know of any errors or duplicates from the imported budget data.
  18. When all the lines have been confirmed to be accurate, click Validate All Records from the Options menu.Click “Validate All Records”

  19. The Validation Notes should change to “OK” (as shown below).
  20. If a cost code is not already there, a message will appear in the Validation Notes column, stating the cost code will be created automatically (Cost code 11 in the screenshot below)
  21. Then, click Post All Records from the Options menu.Validation Notes Read “OK”/Click “Post All Records”

  22. If a warning pop-up appears, click OK in the Warning pop-up to allow overwriting budget data.Click “OK” in the Warning Pop-Up

  23. The lines have been imported, validated, and posted, so the Workspace is now blank. The next step is to freeze this budget, which allows comparison as the progresses.
  24. Next, navigate to the Freeze Initial Budget function. This can be done by using Menu Search or by going to Status > Budget Set Up > Freeze Initial Budget through the tree menu.Navigate to the “Freeze Initial Budget” Function

  25. Choose the appropriate from the lookup, or type in the Number if known.Choose or Type in Number

  26. Then, click at the bottom, right corner of the screen.Click Next

  27. Move the mouse pointer over the Printer icon at the top of the screen to view Reports, then click on the Freeze Initial Budget report.Click on “Freeze Initial Budget” Report

  28. The “Freeze Initial Budget” report can then be viewed. This report shows the budgets, cost codes, and amounts, as well as the cost, revenue, quantity, and hour totals within the .“Freeze Initial Budget” Report
