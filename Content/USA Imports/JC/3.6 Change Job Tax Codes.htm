Change Tax Codes

Module: Status

This document demonstrates how to change tax codes on a and how to change the taxability status of certain cost categories. tax codes may need to be changed if the original codes were wrong or if a new section was added incorrectly. Certain categories are either taxable or exempt, based on location and several other factors. The process of changing tax codes also differs depending on if the uses WBS (Work Breakdown Structure) codes or not.

Updating Tax Codes on a

  1. Go to Status > .
  2. Click in the row of the to edit.Open a

  3. Under the Sales Tax tab, this example was set up as tax exempt. All the boxes under Taxable are uned, meaning that none of the cost categories are treated as taxable. For this example, even if any of the boxes were ed, it would still use the NY-EX (exempt) tax code for those cost categories. is Tax Exempt

  4. In the Tax Codes box, change the tax code for the . This can be done by typing in the tax code or finding it via the lookup (…). Then, click in the bottom, right corner of the screen.
    • For this example, the is being changed to use the tax code: NY-ALB.
    • The tax code is set up based on the number and state. The steps are the same, regardless of what code is used. When the tax code is changed from NY-EX to NY-ALB, the boxes on the right in the Taxable section will be filled accordingly, based on the tax code setup.
    Change the Tax Code, then Click Save
