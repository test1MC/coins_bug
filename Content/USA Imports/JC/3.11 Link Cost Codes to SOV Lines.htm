Link Cost Codes to Lines

Module: Status

This document demonstrates how to link cost codes to cost based lines. For information on how to set up , see the how-to document for “Set Up a ().”

  1. Go to Status > .
  2. Click the hyperlink of the appropriate .Click Hyperlink

  3. Go to Cost Code Maintenance (which may include WBS) and click Cost Codes.Click Cost Codes

  4. Click in the appropriate line to open a cost code. Field will be used for this example.Click Open in Appropriate Line

    • Alternatively, click Bulk, then on the bottom action bar to open, edit, and update all the cost codes in an Excel spreadsheet.
    • Multi Update can also be used by selecting all appropriate rows, and clicking on the bottom action bar. This will allow the users to apply identical changes to all the rows selected.
  5. There will be a Billing Item field on the cost code, which should match to the appropriate cost-based billing line.
  6. Clicking will find all of the lines that exist on the for this .Click the Lookup for the Billing Item Field

  7. Select the appropriate Line by clicking .Select the Appropriate Line

    • Make sure the line selected is a Cost Based line. To check this, go to > (appropriate selected working ) > > Workbench in a new tab or window, and check the Row Type of the line.

  8. Once the correct line is selected, the item will fill in on the cost code screen.
    • Note: The Row ID is used to identify which line this cost code links to.
  9. Make sure that the Include in Cost-Based Based Billing box is selected.
  10. Click in the bottom, right corner of the page.Select “Include in Cost-Based Billing” box, Click Save

  11. Repeat these steps as appropriate/required for each cost code.
  12. Any code left out of this process will not be included in any cost-based billing for this .
