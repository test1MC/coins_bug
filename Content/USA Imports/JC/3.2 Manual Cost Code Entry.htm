Manual Cost Code Entry

Module: Status

This task can also be performed with an upload/import file, but for manual cost code entry this tutorial will be used.

  1. Go to Status > . Click on the Number hyperlink. Screen

  2. Hold the mouse over Cost Code Maintenance and click on Cost Codes.Cost Codes Screen

  3. From the Cost Codes tab, you will see every code that is currently assigned to the . Here click to create a new cost code. Alternately there is the option to Delete or Close/Deactivate a code to remove from a .Add a Cost Code

  4. Begin by either typing in the code number into the first field OR click to open the Cost Code Lookup window, then select a Cost Code by clicking .Select the Code to Add

  5. Upon entering the Cost Code, additional fields will autofill. If these fields do not fill in, make sure that the code you are using is a valid cost code.Entry Screen

  6. Check that the appropriate GL Accounts are on the G/L Accounts tab. Click . Check G/L Accounts

  7. Note:  After saving the Cost Code Maintenance screen will reset. The previously entered code has been saved. The system assumes you will add another cost code. Simply click when you've finished adding codes.Finished Code
