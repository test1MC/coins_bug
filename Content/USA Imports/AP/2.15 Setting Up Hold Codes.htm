Setting Up Hold Codes

Module:

This document will demonstrate how to set up hold codes when processing invoices. This useful when workflow is not set up. The system can automatically put the invoice on hold, then taken off hold by using the approval workbenches.

  1. Go to > Setup Maintenance > Financial Hold Codes Maintenance.  These are the three hold codes available to be put on invoices.Financial Hold Code Maintenance


  2. To add a hold code, click on add and enter in a code and a description. Adding a Hold Code


    Then save the line.

  3. can default to a hold code when an invoice is entered.   There are three parameters that must be set. DEFCODEF, USEFINH, and DEFFINH. DEFCODEF is where the value of the hold code that should default gets entered. Go to > Configuration > Parameters.Default Hold Code


    Search DEFCODEF and open the parameter. Enter the hold code in the value line and save the line. (INVAP in this example) The code that was entered will default when an invoice is entered.

    Then search for ESEFINH. Open the line switch the setting to YES and save the line. This will allow the use of financial hold codes.

    Use Financial Hold Codes


    Search for the third parameter DEFFINH. Open the line and switch the value to YES. Save the line.

    Invoice on Financial Hold Code as Default


  4. Now enter an invoice, for instruction on how to do this refer to Entering an Invoice Without a PO.

    Post the invoice and go to > Payments > Select/Deselect Invoices(Manual).

    INVAP Hold Status


    Select the filter at the bottom of the screen to ALL and click on the filter icon. This will show all the invoices. Look at the status of the invoice that was just entered. It will show the Hold Code that was set in step 3. When the line is selected and the Pay Invoices action is selected an error will pop up.

    Error Message


  5. To remove the hold, go to > Invoices > Approve Invoices by Supplier.Approving Invoices

    Find the invoice, select it and apply the Approve Invoices action.

    **Note** Invoices can also be approved by , Entry Batch, and Costing Batch.

  6. To see who approved the invoice, go to > > Invoices . Invoice


    Hyperlink on to the Internal Reference number and go to the Hold History tab. This screen shows who put the hold on and who approved the invoice (took the hold off).

    Hold History


If nothing appears under the Hold History tab, go to > Configuration > Audit Configuration (Record Level). Open the ap_invoice line and make sure the following fields are in the audit fields text box.

ain_hold,ain_hcode,ain_hreason,ain_hold2,ain_hcode2,ain_hreason2

Audit Configuration


Save the line.
