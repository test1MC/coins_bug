Posting and s from Invoices

Module:

This will complete the /PO process.

Note: If Electronic Funds Transfers BACS payments and are present in the batch, both must be printed prior to posting.

To Post or s

  1. Go to > Payments > Payment Batches. Select batches to be posted by ing the box to the left of each batch number. To select more than one batch at a time, hold control key while selecting batches. Select Post Payment Batches from bottom function list and click the action arrow.Post Payment Batch

  2. A message box saying Task submitted successfully will appear. Check the Reports Icon in the top right and you can see the Posting Reports for the PL, GL and CB. The Status will have changed to Posted. You have then completed this tutorial.
    • Note: If posting – Double click on the printer icon in the top right and Go to Report Status - My Files to retrieve the file and then send this to the bank for verification.
