Changing Supplier Addresses in the Central Repository

Module:

This step is needed if a company’s mailing or remittance address needs to be changed.

  1. Go to Central Repository > Companies.

    Once a supplier has already been pulled into the from the Central Repository, it gets multiple address “types” and it is necessary to change the “remit to” address to change what appears in the .

    Select Companies in the CR

  2. Click the link to the Main Office.Select Main Office

  3. Go to the Addresses tab.Addresses Tab

  4. You will have multiple address types created as you pull a company into multiple modules. (The types available depend on how your system is set up.) For example, there might be a "MAILING" address where PO's pull addresses from, and a "REMIT" address for the address (that's the address you need to update to change the supplier account's address).

    customers might also have an "ARINV" a “SLINV” (Invoice address/bill to) and an "ARSTMT" a “SLSMT” (statement address).

  5. If changing an address, click to open the address for editing.Open Address to be edited

  6. Make any changes you wish, then click to finish.Edit and Save the Address
