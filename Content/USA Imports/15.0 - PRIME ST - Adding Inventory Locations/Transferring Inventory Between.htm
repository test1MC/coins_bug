Transferring Inventory Between Locations

Module: Inventory

When Inventory is transferred from one location to another this is how that can be shown in the COINS system. The following steps will show how to move inventory from one location to another.

  1. Go to Inventory > Location Inventory Transfer. Make sure the Depot on the bottom is the Location that needs to be transferred from

Figure 1: Location Inventory Transfer

  1. Check the row of the inventory item that needs to be transferred, and on the action bar select Create Transfer Batch

Figure 2: Create Transfer Batch

  1. This will bring up the Create Transfer Batch Screen, in the Receiving Location box select the location that inventory is being transferred to. From this screen select “Go to Transfer Batch” to go to the next screen and transfer the batch. Or select “Stay on Location Transfer” which will go back to the previously screen to select another inventory item to transfer

Figure 3: Choosing Receiving Location

  1. Select the next button on the bottom right hand corner of the screen

Figure 4: Go to Next Page

  1. This brings up the “Transactions” tab. Note that the inventory shows ___ units (this is the whole balance of the item I-CP-3 at the North Warehouse). To change this value, click on the folder open button to the left of the line

Figure 5: Transactions Tab

  1. Change the quantity on the screen above and select save in the bottom right corner.

Figure 6: Changing Quantity

  1. Once back on the Inventory Transfer Entry – Transactions note that the quantity has been changed. Click on the back arrow in the browser to return to the post batch screen

Figure 7: Go Back to Post Batch

  1. Select Post in the action bar and Post the batch

Figure 8: Posting Batch

  1. Go to Inventory > Input > Input Inventory Transfer. This is where it can be verified that the batch was posted

Figure 9: Verifying Posted Batch

  1. Finally go to the report viewer and look at the Inventory – Inventory Transfer Report which is shown below. Here it shows the From and To location and it can be verified that everything is correct

Figure 10: Inventory Transfer Report
