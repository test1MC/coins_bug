Adding Inventory Items

Module: Inventory

After the Inventory Location has been creating and the Inventory Group was added the Inventory Items can now be added. These are what will be used when technicians select Inventory for Service Orders.

  1. Go to Inventory > Inventory Items on the tree menu

Figure 1: Inventory Items

  1. Click on the green “ADD” button at the bottom of the page

Figure 2: Add Button

  1. This will bring up the adding Inventory Items screen. The first screen is Item Details, fill out all pertinent information on this screen.
  1. Commodity Code: refer to the how-to-document titled “Commodity Code and Inventory Links”  for more information
  1. Inventory Group: refer to the how-to-document titled “Adding Inventory Groups” for more information
  1. Cost Type: the cost type to charge by default when issuing inventory to a job
  1. Unit Cost: the price per item
  1. Per, Unit of Quantity: the unit to use when counting, measuring, or ordering this item

Figure 3: Inventory Items Details

Figure 3: Inventory Items Details [Continued]

  1. If your system is configured to use “Analysis Sets” for inventory items, then you will have a tab called “Analysis Sets”—on this tab, you must select a valid option on every drop-down field you see on the screen (the number of drop-downs, the name of each drop-down, and the options available in your system may not match what is shown here)

Figure 4: Inventory Items Analysis Sets

  1. Click on the Save button in the bottom right hand corner of the screen

Figure 5: Save Button

  1. Going back to the Inventory Items screen the new Inventory Item will be there

Figure 6: Newly Created Inventory Item

  1. If any additional edits are needed click either the folder open button to the left of the record, or highlight the record and click the “OPEN” button on the bottom of the screen

Figure 7: Multiple Open Buttons
