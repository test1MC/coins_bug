Dispatching Inventory

Module: Equipment

When there is inventory at a warehouse it may need to be moved to a Job Site or be used with a Job. In order to do this the Equipment Manager Module is used with the steps below. Before using this function, make sure the user ID is set to the depot which is done in Equipment Manager > Setup > Equipment Users

  1. Go to Equipment Manager > Fast Dispatch. This will pull up a screen as shown below

Figure 1: Caption Title

  1. Select the Depot that the inventory is being taken from. Chose where the inventory is being dispatched to, in this example it is a job. When this is chosen Address, City, State, Zip Code, and Dispatch Type will automatically fill in

Figure 1: Caption Title

  1. Write any comments that need to be entered and choose the Dispatch Date which will automatically default to today. Choose the next Action as Create Dispatch

Figure 1: Caption Title

  1. Finally choose the item that are being dispatched and how many are being dispatched. If there is a person responsible for these items put that in the Responsible section

Figure 1: Caption Title

  1. Finally hit the NEXT button in the bottom right of the screen

Figure 1: Caption Title

Figure 1: Caption Title
