Manually Record Visit Details, Mobile Tech, Payroll Timecards

Module: Service Management

If a visit was done on mobile but a technician forgot to input their time then this document will show how to record these visit details.

  1. On the Schedule and Dispatch Board, right click on the visit, then select visit details.

Figure 1: Schedule and Dispatch Board

 

  1. It will open up the summary for the visit you wish to enter or verify details for. The tabs on the top of the Service Order allow you to manually enter or verify visit details. The Summary tab looks at all details. Assets are any assets being serviced on the visit, tasking are any service the technician needs to perform on the asset. Material will be any materials purchased for the Service Order or manually entered by the Technician. Purchase Order items are anything related to a Purchase Order tied to the Service Order. This will only display if purchase order is taken out by office prior to technician closing the call. Miscellaneous tab will be for any chargeable items such as a parking fee. Equipment is any chargeable equipment on the order. Work done allows a resolution on the order and confirms the work is done. Additional work is for the technician to enter some opportunities (or recommendations) for additional work in the future.

Figure 2: Service Order Details

 

  1. Any information that needs to be added or updated should follow the next steps, Open up the Labor Tab. Select the green ADD button to record labor.

Figure 3: Service Order Details Labor

 

  1. Add Labor needed and press the save button.

Figure 4: Adding Labor Service Order

 

  1. Open the Materials tab and select the green add button if additional materials are needed.

Figure 5: Service Order Details Material

 

 

  1. Enter the Material code along with quality and price. The billing method will default from the allowable work on file. While the billing method can be changed, it will default back to the allowable work set billing method automatically. Fill out the cost source (in this case invoice) and material type (commodity of inventory). Then click save.

Figure 6: Adding Material Service Order

 

  1. Click on the work done tab then click open if anything needs to be added.

Figure 7: Service Order Details Work Done

 

  1. Enter Resolution code with any comments and check the all work done box if applicable, then click save.

Figure 8: Editing Work Done Tab

 

  1. Open up the payroll stages tab.

Figure 9: Service Order Details Payroll

  1. Select the Visit Labor Approved for Payroll line and click open folder on the left: Check the done box and hit save.

Figure 10: Editing Payroll Stages Tab

 

  1. The system will now show that the visit has been approved for payroll. The 4th sequence line is Visit Labor Transferred to Payroll. This will usually be done automatically by transferring SE visit to payroll. However, if any changes are made to the visit/ or the payroll is entered manually, you must go in and open the line “Visit Labor Transferred to Payroll” and check the done box. Otherwise, the next time visits are transferred to payroll, this line will be transferred, and could result in a double payment.

Figure 11: Payroll Ready for Billing
