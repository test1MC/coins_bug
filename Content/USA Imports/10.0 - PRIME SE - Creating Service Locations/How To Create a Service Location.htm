How-To Create a (Service) Location

Module: Service Management

This document is showing how to create a Service Location. Service Locations need to be created in order to create Service Location Customers, which need to be created in order to enter in Service Orders.

 

  1. Go to Service Management > Locations Select the green add button on the bottom of the screen.

Figure 1: Starting a New Location

 

  1. Enter all the information for the location.

Figure 2: Opening Screen of New Location

 

  1. Location Name will be used when creating service orders. Address name will appear on Service Reports and Invoices, it default from the Location Name when adding a new location. Address is the physical address.

Figure 3: Zoomed in first section of Location

 

  1. Location Level is Hierarchy of the account, it will either be, I- Independent, P-Parent, S-Subsidiary or T-Template. In most cases, I-Independent will be sufficient. Location type is the code to be used to distinguish the location type (ie. Hospital, Hotel, Construction Site ect.) Inventory Location can be used if this location will always take from the same inventory location.

Figure 4: Zoomed in second section of Location

 

 

  1. Open up the Directions/Access Tab, here you can enter any special directions or access comments if applicable. This will be visible to the technician on their mobile IPADS.

Figure 5: Zoomed in second section of Location

 

  1. Open up the PR Info Tab.

Figure 6: Zoomed in second section of Location

 

 

 

 

  1. Click the Save button on the bottom right hand corner of the screen.

Figure 7: Zoomed in second section of Location
