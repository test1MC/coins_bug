Add Location Customer Type of Work

Module: Service Management

After creating a Service Customer for a Location the next step is to create the type of work. This is what type of Service Orders can be created, for example Job or Non-Agreement.

  1. Using the tree navigation to the left of the screen go to Service Management>Locations and Hyperlink into your location as shown below.

Figure 1: Service Management Location Screen

 

  1. Now click into the Customer Account that had been created. This is shown in the screenshot below.

Figure 2: Customer Location Screen

 

  1. Once hyperlinked into the customer account there will be three tabs along the top “Customer Details”, “Non Agreement Allowable Work”, and Comments. Click into “Non Agreement Allowable Work”.

Figure 3: Allowable Work Screen

 

  1. The top of the screen shows the section “Allowable Work” click the ellipse and select the type of work you would like to make allowable.

Figure 4: First Section of Allowable Work

 

  1. The first Tab is Main. In general Info it is mandatory to enter in “Bill To”, “Description”, “Service Departments”, and “Cost type Set”.

Figure 5: Allowable Work General Information

 

Bill To selects the type of billing, choices are C-Customer, J-Job, and N-None

From and To sets the dates this allowable work will be valid for. Usually this is left blank until the allowable work is dormant.

The following will prefill based on the type of work set up:

Description – The description for this type of allowable work.

Service Departments – Enter code used for service department, this should auto generate after picking the allowable work.

Auto-generate Section – Should the jobs section auto generate when creating a service order. This is always checked.

Costing Job – The costing job from the job Status Module.

Cost Type Classes – Enter the cost type that will be assigned when work is selected cost types are set up in Job Status > Setup > Setup Maintenance > Cost Type Classes.

Cost Type Set – Each Service Order detail line is associated with an Allowable Work record which holds reference to a Cost Type Set. The Cost Type Set contains a list of Cost Type Groups that determine the Billing Type and the default Billing Method for the service order detail line. Service Management > Setup > Company Setup > Cost Type Sets.

Override Cost Type Set – Whether to override the preset cost type sets. This is usually left unchecked.

  1. The second Tab is Billings and Sales Tax

Figure 6: Allowable Work Billing and Sales Tax

 

The billing rates are used to establish the labor billing rate table. Typically, clients will use a standard rate for non-contract customers and a preferred rate for agreement customers.

Labor Price Group – STND is typical

Material Price Group – STND is typical

Inventory Price Level – 1 is typical

Miscellaneous Price Group – STND is typical

Subcontract Price Group – CM115 or preferred markup is typical

Unit Price Billing Price Group – usually blank

Labor Line Description Source – This is what will display on the Service Invoice under Labor: Options are T-Technician name or B-Technician Billing Class Description

Minimum Bill Amount – Minimum amount it is possible to charge

Requires On-Site Payment – Yes or No

Require Serviced Asset – Yes or No

Retention % - The amount you want to withhold for payment

Response Type – How the service call needs to be acted upon

The default discounted percentage for the associated type of commodity

Figure 7: Allowable Work Default Discounts

 

The SE Revenue tax defaults are used for which lines will be taxable on Service Invoices

Figure 8: Allowable work SE Revenue Tax Defaults

Revenue Taxable Tax Code – Tax code for the revenue

Revenue Exempt Tax Code – Use non-taxable code for billing/processing

Revenue Taxable – Check off box for all associated categories that are taxable

The JS Section Tax Defaults are for whether or not you will pay taxes on material orders, such as a purchase order

Figure 9: Allowable Work JS Section Tax Defaults

Taxable Tax Code – Select the appropriate tax code for the job

Taxable – Select the associated categories that are taxable
