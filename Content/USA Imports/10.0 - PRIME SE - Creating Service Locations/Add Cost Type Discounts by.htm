Add Cost Type Discounts by Location

Module: Service Management

If you have certain customers where you give discounts for certain types of work but not others this is the way to discount certain types of work.

  1. Under Service Management, open Locations

Figure 1: Service Management Locations

 

  1. Click the blue hyperlink on the location

Figure 2: Close up of Location

 

  1. Select the blue customer account hyperlink under Locations > Customers

Figure 3: Location Customer

 

  1. Under Locations > Customers > Nonagreement Allowable Work, select the folder open next to the allowable work

Figure 4: Nonagreement Allowable Work

 

  1. Under the main tab, check the override cost type set box and hit save

Figure 5: Allowable Work Main Screen

  1. This will bring up the nonagreement allowable work page, now click on the blue hyperlink of the allowable work

Figure 6: Nonagreement Allowable Work

 

  1. Now a cost type groups tab on the allowable work details page will appear. Select the folder open on any of the cost type groups that need a discount applied

Figure 7: Allowable Work Cost Type Groups

 

  1. Enter in the Discount type and hit save

Figure 8: Discount Percentage

 

  1. Updating multiple cost type at once is allowed. To do this select multiple cost types and select concurrent on the bottom of the screen

Figure 9: How to Use Concurrent Open

 

  1. All of the fields will open together allowing multiple changes at once, then hit save

Figure 10: Applying Multiple Discounts

 

  1. Multiple saved fields are pictured below

Figure 11: Finished Cost Type Discounts
