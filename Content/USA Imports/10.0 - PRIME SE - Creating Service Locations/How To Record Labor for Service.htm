How-To Record Labor for Service Order

Module: Service Management

Once a Service Order has been issued and performed by the Service Technician, if it is not done on a mobile device, it will need to have the labor recorded. This is done by the dispatcher through the S&D board, or on the visit tab in Service Management.

  1. Once a Service Order has been completed by a technician, it will be greyed out.

Figure 1: Schedule and Dispatch Board

 

 

 

 

 

 

 

 

 

 

  1. Right click on the visit and select visit details.

Figure 2: Viewing a Service Order on S&D Board

 

  1. Select the labor tab.

Figure 3: Labor Tab on Service Order

 

  1. View the labor the technician put on the service order. Verify they are correct and make any changes if necessary.

Figure 4: Check the Labor from the Technician

 

  1. Towards the bottom there is an ADD, COPY and DELETE Record buttons. These will be for any changes that were entered. If there is no labor set up on the Service Order manually add it. Select the ADD button on the button of the page.

Figure 5: Buttons for Editing Labor

 

 

 

  1. Type of work, date, and technician billing class will auto fill along with the earning category and billing method.

Asset, union details, trade and skills can be added but are not mandatory.

Figure 6: Adding the Labor to the Service Order

 

  1. Add the total hours worked, designating whether it will be regular, overtime or double time coverage.

Figure 7: Entering Labor Hours

 

 

 

 

 

  1. Add a comment for tracking, however this is not mandatory.

Figure 8: Comment on Labor Hours

 

  1. Hit Save

Figure 9: Save Button

 

  1. This will bring up add additional labor, if there is no more labor to add hit the undo button.

Figure 10: Undo Button

 

 

 

 

 

 

  1. This will bring up the labor summary tab.

Figure 11: Labor Summary Tab

  1. This visit can be added through Service Management. First open the service order, and go to the visits tab.

Select the visit.

Figure 12: Visits Tab on Service Order

 

 

 

 

 

 

  1. Select the ADD button at the button of the page.

Figure 13: Adding Labor through Service Order Screen

 

  1. Add Labor

Hit save.

When done adding labor, hit undo, and this will bring back the labor tab.

Figure 14: Adding Labor Through Service Order Screen

 

 

  1. If this is correct, all labor has been added.

Figure 15: Completed Labor Tab

 

  1. If using detail lines for labor, go to the Service Management Module and select Detail Lines.

Figure 16: Detail Lines on Service Order

 

 

  1. Towards the bottom of the screen, select generate from visit details, this will generate the labor previously entered.

Figure 17: Generate From Visit Details

 

  1. Select the open folder to see the details of the charge.

Figure 1: Details of Labor Added by Visit Details
