Entering Time into the ESS Portal

Module: Employee Self Service Portal (11.07) [ES]

Please be aware that the ESS module needs to be configured properly before using this document as a reference.

  1. From a user’s perspective, she will have received an automatically generated email from COINS after being added as an ESS Portal user. The user will then follow the directions in the email.

Figure 1: Automatically Generated Email

 

  1. Following the link supplied in the email, the new portal user will be directed to login using the supplied user name and password. The user will be prompted to change the password. Once this is done the user will be logged into the ESS Portal.

Figure 2: Login Screen

 

  1. The user is now logged into the portal and, depending on the configuration and the employee, something like the screen below will appear to the user.

Figure 3: ESS Portal Screen

 

  1. A user will be presented READ ONLY INFORMATION for all of the portal tiles, with the exception being My Timesheets Input. This is the main screen where a user will be interacting with. Click on the tile.

Figure 3: ESS Portal Screen

 

  1. Depending on how the ESS module is configured in OA will determine what appears to the portal user. This should be considered similar to an open payroll period for this user to enter time to, this is maintain in OA. Drill into the hyperlink.

 

Figure 4: My Timesheet Entry

 

  1. The portal user will now be able to add time. To do so hit the ADD button.

Figure 5: Adding Time and Jobs

 

  1. The user is now able to enter a Pay Type, Job, and amount of hours for each day. One line will be used for one job. For example, an employee works Monday, Wednesday and Friday on one job and Tuesday and Thursday on another job, two lines will be needed to enter the correct time.

Figure 6: Adding Time and Jobs

 

 

  1. Use the ellipses to select a job, or if the GL account number is known simply enter that directly in and it will begin to auto-fill.

Figure 7: Costing to a Job

 

  1. By clicking the ellipses, a new selection screen will appear. This first screen in the job selection. Click the arrow to select the job.

Figure 8: Costing to a Job

 

  1. Next select the cost code. Hit the arrow.

Figure 8: Costing to a Job

 

  1. Chose a category. Hit the finger.
    NOTE** A user will normally not have as many options as shown in the images to cost time to.

 

  1. Enter the time and Save the line for that job/cost code.

 

  1. Continue this until the week has been fully accounted for. Once the weeks’ worth of time has been entered, the time needs to be submitted. To go back one screen, hit the NEXT button in the bottom right of the screen.

 

 

  1. Select the checkbox for the week that time has been entered to and using the Submit action button, hit Submit.

 

 

  1. From the Portal User’s perspective the submission is now complete. An approver or reviewer is now needed to finalize and verify the timecards are correct, then time can be brought over into Payroll.
