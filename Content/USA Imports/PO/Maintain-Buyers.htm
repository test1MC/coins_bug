Maintain Buyers

To enable a user to raise a Purchase Order, that user must be a Buyer.

  1. Go to Procurement > Setup > People Information System > Buyers.
  2. Select Figure 1: Add Buyer

  3. Enter the user name under Buyer’s Code and name in Buyer’s Name.
  4. Select . This newly created account will be visible with the other Buyers.Figure 2: Add Buyer Info
