Trade Code Setup

Module: Procurement

This document demonstrates how to set up Trade Codes. Trade Codes are purchasing codes that will be used on the Subcontract orders.

  1. Begin by going to Procurement > Commodity, Trade, Workbench > Trades. Click .Add a trade Code

  2. Enter desired Trade Code and Description. Click Header if you wish to break up the code list. You can group the detailed codes (individual commodities, trades, etc.) under Headings (which can be used when sourcing Trades). Click Retired if the code is retired.Add Code and Description Name

  3. Click the drop down lists Units of Quantity and Units of Price to choose a relevant measurement. These are mandatory fields. If desired, enter an Extended Description in the field. Tip: It is recommend to choose "EA-Each" for both quantity and price has been the standard used for the codes that already exist in the system.Enter Quantity and Price

  4. Click the tab to open the entry page.
  5. Enter all appropriate Construction and Service Cost Types from the drop down menus, as well as any Cost Category information. Service is the that will be used if this item is ordered to a service order (the "NAB" code is used as a catch-all for most costs on service orders. Click the Save button. Enter and Category

  6. The newly-created Trade Code will now be on the Trade Code Workbench.Finished Trade Code
