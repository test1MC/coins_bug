Review Reports

Module: Payroll

After running the Calculations, the resulting output needs to be reviewed in three places; the Calculations Report that was generated, the Net Detail Tab and the Burden Detail Tab. This important information needs to be verified. If any of the data is incorrect, you will need to Reset the checks and run the calculations step again.

  1. Running the Calculation will produce a report. Click the icon in the top left to view. The net pay to the employee will appear here. View Report

    Review Report

  2. Go to the Net Detail Tab, and review this information. The Manual Calculator Net Detail screen allows you to see a Gross to Net view of all calculated pay. Use this screen to either export the net information to Microsoft Excel or to print the information, which can then be provided to off-site locations if a needs to be cut at a site. This also offers the ability to view components of pay, such as Localities, to view the detail behind the tax, such as Taxable Wages and amounts withheld by the jurisdiction(s).Review Net Detail

  3. Go to the Burden Detail Tab. Review this information and again, verify it is correct. The Manual Calculator Burden Detail screen allows you to see a Gross to Burden view of calculated pay. It displays cost data associated with payroll transactions. This also offers the abilities to display the detail behind a cost, such as Pay Fringes and Union Fringes.Review Burden Detail

  4. If any of this information is incorrect, proceed to the Reset tutorial. If the are reset, then the Calculations process will need to be run again as well as this process of reviewing the reports. If everything is correct, proceed to the Run Edit Report Tutorial.
