Reset for Selected Employees

Module: Payroll

Reset for Selected Employees resets the employee by restoring to-date and associated figures to the position they were in immediately before calculations were run. This allows you to add, update and delete timecards for these . This process is queued. Only employee that have been calculated can be reset.

  1. If any of the information is incorrect from reviewing the previous calculations and reports, then it is necessary to reset . Select the box next to the timesheets to reset and select the action bar. Reset Selected Employee(s)

  2. This will bring up the Output Options Screen, click to proceed. The options here allow you to choose how the printed output from the procedure will be produced; for example by emailing the report to an address. Output Options

  3. This will also produce the Reset report. This report can be viewed by selecting the icon in the top right of the screen. Verify that the Status reads as OK and the have been reset. Verify Report

  4. It is then possible to verify the Employee is reset by viewing the box, it will now be uned.Employee not Calculated

  5. Once the reset has been performed, any corrections to the employee can now be made.
  6. After the corrections have been made, the employee will need to be recalculated.
