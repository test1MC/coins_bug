Enter Timecards with the Manual Calculator

Module: Payroll

Enter all timecards for the employee(s) to be paid. Note: Users now have the ability to pull future and recurring timecards into a Manual Calculator period. The Manual Calculator input screen allows you to enter employee timesheet information for manual calculator periods. These timesheets are not part of the regular payroll run.

  1. From the same location Payroll > [Payroll Frequency] > Input > Manual Calculator. Click on the hyperlink of the Period where the timecards are to be added.Enter Future Period

  2. This will pull up the Timesheet Input tab. Add new timecards here, and/or pull in any future or recurring timecards by selecting the employee(s) to be paid. Save the line.Timesheet Input

  3. This launches the Time Cards and Overrides tabs. Enter the hours or salary for the employee. Save the line.Time Card Input

  4. On the Overrides tab, make changes needed. If an employee’s has been calculated, but manual changes need to be made to taxes, fringes, or deductions you will need to reset the timecard. After overrides have been entered, the needs to be calculated in order to achieve the correct net.
  5. Save the line. A warning message will appear to notify you that the overrides will be spread to all applicable time cards during the calculation process; click OK. Repeat this process for as many employees as necessary. Overrides Input
