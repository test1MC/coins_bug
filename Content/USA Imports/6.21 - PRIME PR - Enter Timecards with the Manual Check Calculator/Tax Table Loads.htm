Tax Table Loads

Module: Payroll

Two factors determine whether an update is needed: 1) the date of the Tax Tables files themselves, and 2) the date of the latest change to the jurisdictions where you work. For Calculations to be run correctly in Payroll, the latest tax tables will need to be loaded.

  1. Find the Tax Tables page on the COINS Client Site, and download the extractor program. If there are any problems with a username or password, please contact support at SupportHelpdesk@coins-global.com or 518-242-72000. Caption Title

  2. Extract the two COINS “.p” programs and remember in which directory they are located.Extraction of Tax Tables

  3. This process will need to be completed for every COINS database in which is being maintained. This does not need to be done in every each Kco – or individual company. In COINS OA, go to Payroll > Global Setup > US Specifics > Tax Table Load.Tax Table Load

  4. With File Location on PC, use the Choose File button to find the tax table. tx_tables File

  5. Make sure to keep the Upload Mode at Validate, and click .Uploading the file

  6. A warning appears that one file is set to load. Click OK.Pop up Message box

  7. Check the Report Status to see if the Validation found any errors.Report Status

  8. Best possible outcome will be the Successfully Checked lines match the total lines of Data.Tax Table Load Validation pdf

  9. Once the file has been successfully Validated, choose the same file and change the Upload Mode to Load.Tax Table Load

  10. Check the report for errors. Tax Table Load pdf

  11. This process needs to be repeated for the Tax Calculation Load as well. That’s found in Payroll > Global Setup > US Specifics > Tax Calculation Load.Tax Table Load pdf

  12. Make sure to choose the “calc rules” file from the appropriate PC directory: tx_calcrules file

  13. Repeat the Validate, error check, and load process for the Tax Calculation load as well.
  14. When both files have been validated, and loaded successfully, go to Payroll > Company Setup > Tax Authority Configuration and click on the details of the most recent Federal.Tax Authority Configuration

  15. Verify that the Tax Tables Release Dates are the most recent ones.Tax Authority Configuration
