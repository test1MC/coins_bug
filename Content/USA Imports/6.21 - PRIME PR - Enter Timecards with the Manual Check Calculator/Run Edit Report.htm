Run Edit Report

Module: Payroll

Running the Edit Report for the selected will give a detailed view of the calculations for the layoff that are being processed.

  1. On the Timesheet Input screen, select all of the employees to be paid by clicking the box next to their name and selecting Edit Report. Next, choose .  Run Edit Report

  2. Continue through the Output Options screen to view the reports by clicking .Output Options

  3. Click the icon and select the report to view. View Report

  4. Review the information from the report to verify all data is correct. Review Report for Errors

  5. If there are any errors with the calculations, it is necessary to Reset the Payroll, make any changes, rerun the calculation, and then review the reports and run this edit report again to make sure that the changes have been corrected.
