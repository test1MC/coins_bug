Run Calculations on Future Timecards

Module: Payroll

The Payroll Calculation for Selected Employees runs the calculations for individual employees from within Timesheet Entry.

  1. Run the Calculations from the Timesheet Input tab screen. From the Future Period that was created, from the input screen, select the employees to be calculated and using the action bar, choose Run Calculations.Run Calculations

  2. This will bring up the Output Options Screen, simply click to proceed. The options here allow you to choose how the output from the procedure will be produced; for example by emailing the report to an address. Output Options Screen

  3. This will mark the employee as Calculated. This is displayed to the right of the employee. Also, a Payroll Calculation report record is produced. Review this report in the next step.Calc’d
