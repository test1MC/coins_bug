New York State Family Paid Leave

Module: Payroll [v11.07]

This must be setup as a Pay Calc because employees can ‘opt out’ of this benefit/deduction.

  1. Create a Pay Item that will act as the holding item for the calculation to run. Do this by going to Payroll > Global Setup > Pay Items.

    This pay item will be linked to a Memo Pay Calc that will be used to hold the calculation to expand the decimal place for New York State Family Paid Leave. This item has no costing and no effects. This should be linked to the pay item for the actual deduction (which will be 10% of this calculation) in effect giving a three decimal number (.126%) that will be the actual deduction.

    In this example Pay Item 20 is used, but this will vary. Do not use a Pay Item that is already in use. This Pay Item will act as the holding item for the calculation.

    Add a “Holding” Pay Item

    When creating the Pay Item,

    • On the main tab:
      • Set the item a number
      • Set the Type to S-Sub-Subtotal
      • Set Years History to 7
      • Set Year end to clear
      • Set Payslip to none
      • Set it to have no effectsHolding Pay Item Setup

    • On the Costing Tab, verify that the costing is set to none. Holding Pay Item Setup

  2. Then Add a new Pay Calculation to use this Pay Item. Go to Payroll > Global Setup > US Specifics > Pay Calculations.
    • Set this up as a Memo Calc
    • Set the percent to 1.26%. (Do not set to .126% here as it will be rounded in the system this will be resolved later)
    • Be sure to use Pay Item that has been created in the previous step. Create a “Holding” Pay Calc

    • If this benefit does not run on Bonuses, Auto Allowance, Holiday pay, etc. be sure to add them to the Excluded Earnings Category.Setup Pay Calc Excluded Earnings

  3. Create a second Pay Item. Go back to Payroll > Global Setup > Pay Items. Again it must have an unused number, in this example pay item 26 is used.

    Since this is pre-tax, and should have the same effects as child support, therefore if desired, simply copy a child support Pay Item, give it a new number and rename it. 

    Add a “Calculating” Pay Item

    When creating the Pay Item,

    • On the main tab:
      • Set the item a number
      • Set the Type to S-Sub-Subtotal
      • Set Years History to 7
      • Set Year end to clear
      • Set Payslip to Deduct
      • Set it to have the same effects as a child support. The example below may vary from client to client. Pay Item Setup

    • On the Costing Tab:
      • Set the costing to Credit
      • Set the Cost method to GL
      • Enter a default G/L accountPay Item Setup

  4. Now create the Pay Calc and set it up as a Deduction for the New York State Family Paid Leave. Go back to Payroll > Global Setup > US Specifics > Pay Calculations. It is to be set up as a 10% deduction, because taking 10% of 1.26% will move the decimal place from what is on the Pay Item in the Memo Calc, which will now give the correct amount of .126%. Add a “Calculating” Pay Calculation

    • On the Main Tab
      • Link to the second Pay Item (26 in this example)
      • Set the Rate to 10%
      • Set Percent Basis to Item Current Amount
      • Set Percent Basis Item to first Pay Item (20 in this example)
      • Set Flat Amount Spread Basis to Total Gross
      • Set Max Amount Per Period to 1.65
      Pay Calc Setup

  5. Now set the order for the calculations to be performed in. Go to Payroll > Global Setup > Item Sequences. Here there should be a sequence that is used for employee deductions other than garnishments. Add both of the Pay Items to the list. In the sequence, be sure that the Memo Calc’s Pay Item precedes the NYS-PFL. Set Item Sequence

  6. Add both of these pay calculations to an employee. This will now calculate the correct amount for NYS FPL.
