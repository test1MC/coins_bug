Cancellations Workbench Configuration

To use the Cancellations Workbench you need to set up the following:

  • Cancellation Workbench events (use Cancellation Workbench Events).
  • Plot cancellation letters
  • PX cancellation letters
  • The following lookup tables:
    • Reasons for Cancellation allows you to set up and maintain a list of codes that can be used to identify why a purchaser decided to cancel the purchase of a property.

      For example:

      • MORT = Mortgage Problems
      • FIN = Financial Problems
      • CHANGE = Changed Mind

      This information will then be available to users for reporting, and when completing the relevant details in the Cancellations Workbench.

    • Refund Terms allows you to set up and maintain a list of the terms that will be applied to the if the sale is cancelled.
  • Cancellations Captions
  • You can also link plot or PX captions and cancellation captions, so that when a sale is cancelled, caption information that was recorded with the sale is retained in the cancellation captions.  Use Link Plot & Cancellation Captions and Link PX & Cancellation Captions.
