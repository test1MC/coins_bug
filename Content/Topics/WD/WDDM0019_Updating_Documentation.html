Updating Documentation

You can update the documentation, to make sure you have the most recently released documentation, regardless of which release of you are using.

Updating documentation

  1. Go to the website and log in to the client area.

    If the link above did not open the OA Documentation Update page:

    1. Click Documentation.

    2. Then click OA Documentation Update.

  2. On the OA Documentation Update page, click the images.tar link and download the images file to your PC.
  3. Right-click one of the documentation links, choose Save As, and download the documentation file to your PC.  The OAWebDoc.Dat file contains all the current documentation; the other files only contain documentation for specified modules.

  4. Use FTP (or similar) to transfer the images.tar file and the .Dat files to your server.
  5. In , run syloadwh.p.  Pass the full path and file name of the .Dat file as a parameter.  For example:
          syloadwh.p ("/home/user1/WDSC.Dat")
    You will need to run this separately for each .Dat file you download. 

    The updated documentation is then available from .
