Configurable Cover Sheets

You can configure documents to print cover sheets for  recorded in Project Management.

Define the layout of the document using Document Designer.  

cover sheets are made up of the following sections:

 

Header Information about the .
Body One record for each reason.
Comment Header Column headings.
Comment Body Comment lines (field: tra_notes).
Item Header Column headings.
Item Body One record for each item. 
Footer Additional information about the , page numbers, etc.

To configure cover sheets:

  1. Go to Document Designer.
  2. Use the document type Print . 
  3. On the Extras tab, enter names for the comments and  items sections.
  4. See Configurable Documents for general information on how to configure documents for printing.
