SC Ledger Basics - Subcontract Lookup

Wherever you need to enter a subcontract order number at the Order Number prompt, you can use the Order Number lookup icon to select the code.

You can search for the subcontract by:

  • order number
  • subcontractor

An I in the right-hand column indicates an inactive subcontract.

An M in the left-hand column indicates a matrix subcontract (that is, a plot-based subcontract as used in the Valuations and Payments Module).

Within the lookup, you can display the subcontract details frame.
