Adding Charges to a Pending

You may need to add lines for delivery, collection and fuel charges to a request.  You can only do this after you have assigned the component to which the charges relate.

To add a charge for a component:

  1. On the Process Pending screen, select the component line.
  2. Select Add Extra Line from the Choose Action menu and click .

    adds a new line with a symbol in the Date column.  The new line is associated with the line you selected, and has the same line number.

  3. Click .
  4. In the Product column, enter the code for the charge.
  5. Enter the Charge Price and the Price Confirmed? box.
  6. Click .

NOTE

You can also use this method to add further asset groups or items to the request; for example, if you know that an additional item will be needed to go with a particular piece of equipment on the request.

Use Charges to set up the possible charges.  If the charge is set to be charged "ON-RETURN" (for example, a collection charge), the accrual calculation won't accrue for the charge until the item has been returned.
