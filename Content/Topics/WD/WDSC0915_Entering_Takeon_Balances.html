Creating a Subcontract Record - Entering Takeon Balances

  • After you have entered the subcontract record, asks if you want to Enter takeon balances.

    If you are creating a record for a new subcontract, enter N.

  • If you are creating a new subcontract record for an existing subcontract (usually this only needs to be done when is first set up), you can take on cumulative to date balances. Enter Y.

displays the Subcontract Takeon Balances transaction input screen (identical to the Certificate entry frame — see Subcontractor Certificates - Entering Subcontractor Certificates), with transaction number 000. Enter the Cert Date, Val'n Date, and Due Date (these are for your information only). Enter the cumulative values to date for this subcontract. This will not generate any accounting transactions, and you do not need to post it.

If you need to update fields that aren't normally available as input fields (for example Income Tax), go to Subcontract Certificate Definition and change the Up (Update) flag for that line to Y (Remember to change it back afterwards).
