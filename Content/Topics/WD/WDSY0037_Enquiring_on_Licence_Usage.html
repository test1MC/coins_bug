You can use the following to on module and usage:

  • Module shows the number of users using each of the modules.
  • Current Users shows information about the users who are currently using , and what they are doing.  This includes users of , GUI applications and .

    You can delete inactive WEB sessions to release the being used.  If you delete an active session, that user will be forced to log in again.

    It is not possible to delete a session via this ; the record will be locked and not accessible.

    Do not delete any GUI sessions as this may have implications on the GUI application being used.

  • Named Users shows the current Named Users and additional options.
