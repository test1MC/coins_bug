Adding Items to a Book

You can add existing items (topics or books) to a book menu; for example, if you want to move items from one book to another.

NOTE

You should not leave the same item on more than one menu, as the navigation buttons and the highlight of the current topic in the tree view will no longer work correctly.  If you are moving items to a new book, remember to delete them from the original book.

The Topics Not On A Menu filter in the List of Topics lists any topics that have been removed from books, or have been added without adding to a book. 

To add items to a book:

  1. Go to Topic Maintenance.
  2. Using the tree view, browse to the book that you want to add the topic or book to.
  3. Using the Add Type drop-down, select Existing Item and either:
    • Click to add the existing item at the bottom of the list.
    • an existing item and click to add the existing item above the selected item.
