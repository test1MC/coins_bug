Hold Codes

Certificates can be marked as on hold for either commercial or financial reasons. If a certificate is on hold, the Select/Deselect Certificates (Batch) procedure will not select them for payment.  When you mark a certificate as on hold, you specify a hold code that gives the reason the certificate is held. So before you can hold certificates, you must set up hold codes.

To set up hold codes

  1. Go to either Subcontract Commercial Hold Codes or Subcontract Financial Hold Codes.
  2. In the Code column, enter codes of up to 4 characters.
  3. In the Description column, enter a text to explain the hold code.
