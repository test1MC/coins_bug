Menus

In , menus are displayed using a tree view on the left-hand side of the window. Each menu item corresponds to a function (see Functions). The functions on a menu can ether be:

  • "Function-type" functions, which run programs. displays these with an icon to indicate what type of program they run (for example: maintenance, report, processing). The function category determines which icon uses.
  • "Menu-type" functions, which display further menu items. displays these with a menu folder icon.

Each module has its own menu, which the user chooses using the module selector at the top of the menu frame. The entries in the selector are all menu functions on the main menu %WMAIN.

You can control which functions each user has access to. See How Function Access Works.

NOTE

Menu nesting is limited to 10 levels.  This is to prevent processing problems if a menu calls itself.
