Buy-in Events

There are a number of things that happen during the buy-in process that you may want to record.  You can record the following using events:

  • Property inspections
  • Property survey
  • Exchange of contracts
  • Completion of the buy-in
  • Meter readings

Buy-in events configuration

Use Part Exchange Workbench Events to configure buy-in events.  Use these system event codes:

  • INSEPECT for inspection events
  • SURVEY for survey events
  • BUYEXCH for the exchange of contracts
  • BUYIN for the completion event
  • METBUY for the meter readings on buy-in
