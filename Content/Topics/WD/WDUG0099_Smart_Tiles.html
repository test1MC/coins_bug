Smart Tiles

Smart tiles show values (such as the number of unposted batches or the value of payments received).  There are several standard functions that are set up to show values.  In addition your system administrator can set up calculations to show values on specific functions.

To set up a tile to show values:

When you are maintaining the details of the tile, if the tile function has been set up to display values, the Values Available list will be populated:

Select the value or values you want to show on the tile, and save the tile.

The way the values are presented depends on the size of the tile and the number of values:

  • On Small tiles, only the values are shown, on a single line, with tooltips that explain what each value represents.
  • On other tiles, if there is only one value it is shown as a large number with a label below it to explain what the value represents.
  • On other tiles that show more than one value, these are displayed as smaller numbers with labels to the left.
