Printing in OA

Each module has functions that generate reports. A number of other functions generate printable output, such as , invoices, and so on.

The separate documentation for each module describes the content of the reports and printed documents for that module. The way reports and documents are sent to a printer is described here, since it is the same for all modules.

By default, produces printable output in a PDF file. You can also choose to print the output directly to a printer. For reports, also generates a CSV version and an XML version.
