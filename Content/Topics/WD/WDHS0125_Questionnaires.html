Questionnaires

Throughout House Sales you can record the responses of and purchasers to a number of questionnaires.  Questionnaires are set up in advance.

You can use Questionnaire Analysis Report to produce an analysis of the responses.

To record responses to a questionnaire:

  1. From the relevant record summary, select the Questionnaire tab (usually on the More > tab).
  2. Click to add a new set of questionnaire responses.
  3. Select the questionnaire from the list, and click .

    4. displays the questions from the questionnaire.

  4. Fill in the responses.

    5. Some questions may be optional; others may be required (you must enter a response). When you try to save the responses, if there are required questions you have not entered a response for, will display an error message telling you which question you need to answer.

  5. Click to save the responses.
