Batch Configuration

Batch Configuration is used to define the standard characteristics for each type of batch transaction. The frame shows only those batch types that apply to .

To configure batches in :

  1. Go to Batch Configuration.

    displays the Batch Configuration frame (similar to the one shown, but without data):

The batch types available in are:

Batch type code Used for
PLAPPLY apply payments batches
PLCANCHQ batches
PLCHQ batches
PLCOST costing batches
PLINV invoices
PLMAN manual payments

The batch numbers for PLCHQ, PLMAN and PLCANCHQ are controlled by the batch types CBTR and CBCAN, and are shared with other ledgers that enter cash.

The control totals fields on batch headers, are identified by the following numbers:

  1. Gross Value (for Credit/Debit/Invoice transactions) or Cash Value (for Cash transactions)
  2. Value
  3. Net Value
  4. Number of Transactions

See Batch Input for further details about batches.
