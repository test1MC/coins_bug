Sales Progression Setup

The Progression tab on the Plot Summary of the Sales Workbench shows on a single screen the steps that a plot sale must go through, the expected dates for each step, and if the step has happened, the date that it happened.  To set up the list of steps, use Plot Progression.

To configure the sales progression screen:

  1. Go to Plot Progression.
  2. For each step you want on the Sales Progression screen, enter a line record. fills in the line number by default; you can change it if you want to insert a line between two existing lines.
  3. Each line can show either an actual date or an expected date or both. Specify where you want to get the each date from.

    4. Actual dates can be one of the following:

    • The date when a Sales Workbench Event occured. In the Source field, select Event Date, and in the Event field enter or select the code of the event. If the event is a construction stage, you also need to enter the code for the stage.
    • Manually entered by the user.  In the Source field, select Manual Input.
    • The exchange deadline date or the unconditional exchange date. In the Source field, select the appropriate code.

    Expected dates can be one of the following:

    • Manually entered by the user. In the Source field, select Manual Input. To have enter a default date (which the user can update) based on the number of days after the reservation date, enter the number of days in the Reserve Offset field.
    • Any of various key dates (exchange deadline date, expected build completion date, expected completion date, expected exchange date, expected occupation date, original expected completion date, original expected exchange date). In the Source field, select the appropriate code. To use the date as a default and allow the user to update it, the Update field.
    • A date from a House Sales forecast. In the Source field, select Forecast Date, then enter the code for the forecast and the event.
  4. If you do not want the line to show a date in one column, leave the corresponding Source field blank.
