Seeing Which Parameters Are Used

You can show which parameters a program has used, and what they are set to.  This may be useful when debugging.

To show debug information:

  1. Press Ctrl+Shift+C.

    displays the COINSInfo frame at the bottom of the screen.  This shows information about the program, such as:

    • the parameters that the program has referred to
    • internal functions that the program has used
  2. Click Parameters and Variables to show the list of parameters used.

To hide the debug frame, press Ctrl+Shift+C again.

The help frame also shows information about parameters.
