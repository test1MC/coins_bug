Configuration Checklist

The following is a checklist of things you need to do within to implement the electronic submission of PAYE in-year movement returns (assuming that you are already submitting end-of year returns electronically).

Done? Action Comment
Install Fixes (see Install Fixes) This can be done in advance.
Confirm that you have already been submitting end-of-year PAYE returns and monthly CIS returns electronically. If so, no action required at this step.  If not, please contact for advice.
Set the start date for in-year online submissions (see Configuration) If you have more than 50 employees, this needs to be no later than 6th May 2009.
Grant user security access (see Grant User Security Access)
