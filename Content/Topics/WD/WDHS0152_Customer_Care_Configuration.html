Configuration

To set the parameters for , use House Sales Module Parameters (also available on the Setup menu), and filter on parameters. See House Sales Parameters.

To use the Workbench, you must set up the following:

  • Issue Priority Codes allows you to set up the codes you will use to classify issues by priority, and to set the target response time for issues.
  • Issue Status Codes allows you to set up the codes you will use to track the status of issues.
  • Issue Category Codes allows you to set up the codes you will use to categore issues for reporting.
  • Appliance Types allows you to set up the codes you will use to categore appliances on calls, for reporting.
  • Call Types allows you to set up the codes you will use to identify different types of call. 
  • Task Reasons

  • Subcontractor Contact allows you to record contact details for each subcontractor.  These are used when printing documents in , instead of the details in the Subcontract Ledger, and when sending texts and emails to the subcontractor.

  • Contact allows you to record a contact name and fax number for each .  This is used when printing documents in , instead of the details in .

  • Task Instructions allows you to define a library or standard text for task instructions.  Users can insert this text quickly in the task instructions.
  • Customer Care Contact Types
  • Issue Locations
  • Standard Issues
  • Call documents and task documents for printing full details of calls and tasks. Use Document Designer with document types Call Print (HS) and Task Print (HS).
