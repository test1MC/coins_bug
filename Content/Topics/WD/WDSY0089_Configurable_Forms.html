Configurable Forms for Faxing

When faxing a form, you are not able to select the font size and orientation at run time.  To ensure that the printed form from provides VSI-Fax with the correct font information, it is necessary to insert an additional line into each configurable form which is to be faxed, to specify the font size to use.  For example, to configure the form to print/fax using the font 132p, enter:

Type Line Field Format Column
HEAD 1 "$<132p>" x(7) 1
