Component Resources

Component Resources allow you to produce a detailed estimate for a single item on an order.  You can enter a "summary" item against which you can enter a detailed breakdown of items.  The values etc on the summary item will be automatically updated from the component resources entered.

This is used for supply and fix orders.  A typical example is bathrooms where the SC order will be a single item to supply and fix a bathroom set, so we need a single item to raise the correct SC order.  But within the we want to break down this item into its components, such as the , materials which would include bath, shower, toilet, taps, etc.

When adding a item, the Add Type drop-down allows you to choose "Resource Line" or "Component Resource".

When you update a component item, the ordering details (package and ), payment stage, estimating section, ready to use and discount fields are copied to any associated component items.

items based on component resources can only be ordered as a unit and will produce a single order line.  (By contrast, items from intelligent resources will appear as individual items and can be ordered separately.)
