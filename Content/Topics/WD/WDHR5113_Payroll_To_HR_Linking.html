Payroll To HR Linking

Linking from Payroll to HR can be via the Payroll Requests Workbench in HR or it can be direct.

Payroll to HR linking via the Payroll Requests Workbench is very similar to HR To Payroll Linking Via A Workbench, just in the opposite direction. The Payroll Requests Workbench screen is the same as the HR Requests Workbench, except that it also displays the and period type for the linked Payroll employee and shows the HR ID instead of the employee number.

Direct Payroll to HR linking can be used to effectively bypass the workbench stage, though if you are using HR to Payroll workbench linking it is not recommended to use direct Payroll to HR linking for the same type of value as this could lead to changes made in HR (with an associated request waiting on the HR Requests Workbench) being wiped out (which will delete the request) before anyone in Payroll has noticed the change.

For benefits, Payroll changes will only link to HR if the HR person already has an appropriate benefit (that is, it will not add a new one).
