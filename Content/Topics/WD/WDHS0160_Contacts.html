Contacts

Contacts are people you record who have some connection with a plot for purposes. There are different types of contacts; these could include the original purchaser, subsequent owner, tenants, or property agents. The types available will be set up by your system administrator.

When you print a letter or other document that refers to the current owner, looks first for the most recent contact of type 'subsequent purchaser'; if there isn't one, it uses the original purchaser's details. Similarly, if the letter refers to the current occupier, looks first for the most recent contact of type 'tenant' before looking for the original occupier.

Using the Contacts tab in {Customer Care} Workbench, you can find a plot by searching for the contacts for the plot.

To add a contact:

  1. Go to the plot summary page.
  2. Click the Contacts tab.
  3. Using the Add Type drop-down at the bottom of the page, select the type of contact you want to add and click .
  4. Enter the contact's and first name, and click (next to the Contact Details field) to choose the contact from the People Information database, or add them if they don't exist.
  5. Click to save the contact details.

You can record up to four contacts of each type on a plot. If you want to identify one of the contacts as the person to contact first, set them as the main contact.  You can have one main contact for each contact type.
