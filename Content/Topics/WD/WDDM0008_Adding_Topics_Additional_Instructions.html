Adding Topics - Additional Instructions

See Also links

As well as entering links to other topics in the body of the text, you can add "see also" lists of topics after each section.  Enter a comma-separated list of the topic function codes in the See Also field on the Links tab.

NOTE

It might be a good idea to write topics to use the See Also lists instead of links in the text, wherever possible.  If a user's function access permissions don't give them access to a topic, links to that topic in the text are disabled; See Also links to that topic are hidden.

Parameters

If there are parameters that are relevant to the topic, you can display the parameter help text in the configuration section of the topic. Enter a comma-separated list of the parameter codes in the Parameters field on the Links tab of the Configuration section; for example PL/STOPBLNK,PL/STOPDUP.

NOTE

The Parameters field does not become available on a new Configuration section until after you save it. Save the section first, then edit it to add the parameters.

Pasting from Word

While it is possible to copy and paste text from Word directly into the HTML editor, we strongly recommend you do not do this, because Word creates very bloated HTML with lots of additional tags. To copy and paste text from an existing document, we therefore suggest using one of the following methods:

  • Paste the text first into a text editor (such as Notepad), copy and paste from there, and use the HTML editor to apply any formatting.
  • Use PureText, which is a tool that can be downloaded to your computer. PureText allows you to copy text from a web page or a document and then paste it as simple text into another application (such as HTML editor) without getting all the formatting from the original source. Once you have downloaded PureText.exe, you will see a 'PT' tray icon appear near the clock on your task bar. Simply select and copy the text, click on the 'PT' icon, and paste the text into the HTML editor, where you can then apply any formatting. For more information and to download this tool, go to http://www.stevemiller.net/puretext/.
