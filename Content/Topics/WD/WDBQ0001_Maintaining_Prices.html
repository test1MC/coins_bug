- Maintaining Prices

The Prices Workbench allows you to maintain a master list of prices that can be applied to items.

Import Prices allows you to import prices from a file.

To add prices manually:

  1. Go to Prices Workbench.
  2. Click .
  3. In the drop down:
    • To set up a price that applies for all , select Global Prices (at the top of the list).
    • To set up a price for a single only, select that .
  4. Select the Resource using the lookup.
  5. Fill in the remaining fields as required.
  6. To set up a price, fill in the , Office, Discount and Agreement fields.
  7. Click .
