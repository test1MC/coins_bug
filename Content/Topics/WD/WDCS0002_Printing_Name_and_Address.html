Printing the Customer's Name and Address

To print the customer's name and address in the header

  1. Add header lines containing the customer's name field rcm_name (the example below also prints the text Tax Invoice Number and the tax invoice number on the same line):

  2. Add lines containing the first line of the customer's address (field rcm_addr__1).
  3. Add further lines containing the fields

    rcm_addr__2, rcm_addr__3, rcm_addr__4 and the (rcm_pcode).

    Note that the first column of each row is blank; this is to position the address below the customer name.
