Change Default Codes

There are several types of records in which include default codes. Any of these which are currently set to a standard rated code will need to be updated to the appropriate new code.

Many of these records may be blank or may be set to a code which is not affected by the change. However, it would be difficult to identify any records which may need updating using standard operational functionality. For this reason, a utility program is available to overcome this.

The records which may need to be updated are:

  • FM
  • Series
  • Customers
  • Subcontractors
  • Subcontracts
  • Subcontract order headers in Procurement -- the utility updates these if you update subcontracts
  • VAP Subcontract Payment Stages
  • VAP Standard Payment Stages

For subcontracts, subcontractors and payment stages, you can choose to update the codes on different billing types separately:

  • Items
  • VAP matrix subcontracts - the utility will update lines on subcontract matrices

To run the Default Code Update utility:

  1. Go to Update Default Codes.
  2. Select an old standard rate code (you will need to do this for each old code in turn).
  3. Select the new standard rate code you want to replace it with.
  4. the record types you want to update; you can select more than one type of record.
  5. If you selected Subcontractor, enter a comma-separated list of the billing types (A for Authenticated Receipt, I for Invoicing, S for Self-Billing) you want to update with the new code.
  6. On the Output Options tab, choose how you want the report to be printed.
  7. Click .

    updates any records of the type(s) you selected, and produces a report to show you what has changed.

  8. Repeat the process for the other codes (and other record types) as necessary.

Manual update

You may also need to change the default codes on:

  • Sundry Charges
  • Expense Items

Since you are only likely to have a few of each of these, the Default Code update utility does not process these records. You will need to change the codes manually.

Configuration

Configuration specifies default codes for internal and external . If these are currently set to a standard rated code, you will need to update them with the appropriate new code.
