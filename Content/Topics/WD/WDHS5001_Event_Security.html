Event Security

Many of the workbenches have events associated with them. Events allow you to record things that happen relating to the workbench, such as receiving a telephone call, selling a plot or completing a part-exchange. You can set permissions that control which users can run which events. For example, you might only want to allow certain users to cancel a sale.

To set event security:

  1. Go to the Event Security function for the appropriate type of event.
  2. Select the user or group for which you want to set the security.
  3. In the Events to Maintain field, enter a can-do list of the event codes you want the user (or group) to be able to run.
