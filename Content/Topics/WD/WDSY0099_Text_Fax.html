Sending a Text Fax From

Send Text Fax allows a user to send a simple text-based fax to any recipient from a standard menu.

For administrative uses, this option can be used to determine that and VSI-Fax can communicate and send a fax from the host server.

To send a text fax:

  1. Go to Send Text Fax.

    displays the following screen:

    defaults in User information and name and fax from the current logged user and selected .

  2. Fill in the "To"details:
    Name:The fax will marked for the attention of this person.
    Company:The destination company.
    Fax Num:The recipient's fax number (or email address if VSI-Fax is configured for Send Mail).
    Subject:A description of the fax contents.

    Once you have entered the Subject, prompts for any additional text you wish to include in the fax.

  3. Fill in the remaining details: 
    Res:Standard or Fine (Std or Fine)
    Delay:The amount of time (in whole hours) you wish to delay the sending of the fax ? for example if you wish to send faxes overnight.
    Cover:Whether to fax a cover sheet with this fax. (For a text fax this should always be yes, although when faxing from other areas of you may not want to send a cover sheet.)
  4. To accept the fax details press F1.

    prompts you to confirm you wish to send the fax.

If the fax is sent successfully, will return a four digit Fax Request ID to the bottom left of the screen; if the fax fails, and/or VSI-Fax will return an error - if this happens you should report this to your System Administrator.
