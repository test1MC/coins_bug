Changing Buttons and Other Images

Buttons and other images on screens are held in the /images/ directory of your installation.

You can change the images on the screen by copying the image files to a new location and changing them.

To set up a user-defined images directory:

  1. Underneath the webserver base document directory there will be a coins directory. Create a new directory underneath this, and create a directory called images underneath that; for example: coins/bespoke/images.
  2. Copy all the contents of the Standard images directory into this user images directory. 
  3. Set the SY/WEBIMGS parameter to the name of the parent directory of the images directory.  For example: bespoke.

    NOTE

    If you also have user-defined directories for scripts and styles, set SY/WEBBASE to the parent directory of the images, scripts and styles directories, and set SY/WEBIMGS to $WEBBASE.
  4. You can then modify the images in the user directory, or add your own.
  5. If you want different images for each , create different directories under the webserver base document directory (for example: coins/company1/images, coins/company2/images) and set the SY/WEBIMGS parameter in each to point to the releavant directory.

You can place logos and background images specific to reports anywhere underneath the webserver base directory.  Use the SY/WEBLOGO parameter to specify the logo.  Do not place these files anywhere underneath $BASE/app.
