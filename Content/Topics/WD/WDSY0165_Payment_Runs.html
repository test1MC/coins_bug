Multi-Currency Payment Runs

In , if you are making payments in more than one currency, you must create separate payment batches for each currency.

To process payments:

  1. Run the Select/Deselect procedure to select the transactions to be paid. If you use the automatic (batch) selection, you must create separate payment batches for each currency.
  2. Freeze the payment batch(es). If you use sector-based ledgers or have selected transactions of more than one currency, displays a list of the batches, showing the currency. Select the batch you want to freeze.
  3. Make the payments. If you have frozen more than one batch - for example batches in different currencies - you must select the batch you want to process. If you are printing , make sure your stationery matches the bank account you have selected.

You can only process automatic payments in different currencies if you have bank accounts set up in the relevant currencies. If you do not have an account in the relevant currency, pay the transactions using , and allocate payments to the transactions manually.
