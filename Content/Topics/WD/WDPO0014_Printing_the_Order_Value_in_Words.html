Printing the Order Value in Words

provides special fields that allow you to print the value of the order in words. These fields use the appropriate wording for the currency (for example, Dollars and Cents or Pounds and Pence).

To print the order value in words

On the purchase order document, add one of the following fields:

  • RO_poh_amount_words - The total order amount in words
  • RO_poh_famount_words - The total order amount in the order currency, in words
  • RO_Order_FAmount_words - The total order amount in the order currency including charges and tax, in words

The following fields are also available for printing the value of an order line in words:

  • RO_pol_amount_words - The order line amount in words
  • RO_pol_famount_words - The order line amount in the order currency, in words
  • RO_Line_Ftotal_words - The order line amount in the order currency including charges and tax, in words

Note that the wording for currency units and subunits needs to be set correctly in Currencies.

If the order value is zero, the text "THIS ORDER HAS A VALUE OF ZERO" is returned.
