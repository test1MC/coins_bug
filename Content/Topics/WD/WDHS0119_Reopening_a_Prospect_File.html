Reopening a File

If a file has been closed, and the again shows interest in purchasing a property, you can reopen the file.

To reopen a file:

  1. Find the .

  2. From the summary, click Options and choose the Reopen File event.

    (The Reopen File event may have a different name. It will only appear if you have permission to run it.)

  3. Enter any comments you want to make about reopening the file, then click .

    4. The File Status changes to Open.

Reopening a file - configuration

You need to set up a event with the system event code REOPEN, and use Event Security to control which users can run the event.
