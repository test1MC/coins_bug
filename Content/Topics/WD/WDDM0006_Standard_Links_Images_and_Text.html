Inserting Standard Links, Images and Text

The Documentation Module allows you to insert:

  • Standard images that are shipped with the Documentation Module.
  • Images of buttons used in the interface.
  • Links to other topics in the Documentation Module.
  • Glossary entries.  These are shown as highlighted links; when a user clicks the link, the glossary definition is displayed.
  • Standard text.  This is for blocks of text that are used in several different places.
  • Function descriptions from the side-frame help.
  • function names.  When a user hovers over the name, a tooltip shows the function code.  If the user clicks the name, this runs the Function {Enquiry} which shows where on the menus the function can be found.

The Documentation Module uses tokens to represent these standard items.  There is a lookup for all except the function names.
