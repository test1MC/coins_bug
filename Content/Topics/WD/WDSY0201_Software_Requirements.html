PC Software Requirements for OA

To use all the features of you will need a PC with the following software installed:

  • Internet Explorer 11, Microsoft Edge with Windows 10 Anniversary Update, or the latest version of Google Chrome, with the following permissions:

    • Allow Session Cookies. On versions of IE that we are familiar with, session cookies appear to always be allowed.
    • Allow load/run ActiveX controls (this is only required for House Sales mail merge and drag and drop document upload). To allow ActiveX controls, set the security settings for the relevant zone.
    • Allow execution of JavaScript, enable the "Active Scripting" security setting for the relevant zone.
    • Security settings for the URL set to Local Intranet or Trusted Site (see Making a Local Intranet Site).
    • Pop-up blockers turned off (to use the 'bulk export' functionality).
  • MS Word
  • MS Excel
  • Acrobat Reader (to view PDF reports)
  • Flash (only required to view old-style charts and instruments)

Performance can also be affected by network settings, for instance some companies "scan" all internal FTP traffic or scan for certain objects such as PDFs. This can slow the transfer.
