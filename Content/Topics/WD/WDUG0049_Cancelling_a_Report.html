Cancelling a Report

If you request a report by mistake and don't want to generate it, you can delete or cancel the request.

To cancel a print request:

  1. Go to Report Status. (For information about running Report Status see Report Status).
  2. the print request you want to cancel.
  3. Do one of the following:
    • If has not yet started to generate the report (status Waiting), click .
    • If is generating the report (status Generating), select Cancel Report Request from the Choose Action menu and click .

This stops the report at the earliest possible point. If has not yet begun to generate the report, it is not generated.

If the report is stuck at a lower level of generation, the status remains at Aborting; someone must disconnect the report generator. See How To Stop a Background Report.
