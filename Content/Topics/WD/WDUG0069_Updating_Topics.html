Updating Wiki Topics

If you have permission to write to the Wiki, you can update topics, whether they were entered by yourself or by another user.

NOTE

You may have permisson to write to some Wikis, but only to read others.

When you change a topic, makes a new copy of the topic, so users can see earlier versions of topics and compare them to see the changes you have made.

To update a Wiki topic:

  1. View the topic and click .
  2. Do one of the following:
    • If you are making significant changes to the topic, the Major box - when you save the topic will give it a new version number (for example, 2, 3, or 4).
    • If the changes you are making are minor, clear the Major box - when you save the topic, will give it a change number (for example, 1.01 or 1.02).
    • If you were the last person to update the topic, and you do not want to create a new version of the topic, but just want to make changes (such as spelling corrections), clear the New Version box.
  3. Make your changes to the text.
  4. Click .
