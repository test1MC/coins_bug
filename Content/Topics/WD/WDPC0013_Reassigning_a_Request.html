Reassigning a Request

If you have permission, you can reassign a request to another user.  This allows the new user to process the request.

To reassign a request:

  1. On the Pending screen, select any line on the request.
  2. Choose Re-assign Owner from the Choose Action list, and press .
  3. From the drop-down list, select the new owner for the request.
  4. Click .

    reassigns all lines on the request to the new owner. The link in the Txn No column is enabled for the new user.
