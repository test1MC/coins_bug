Documentation – Inserting Links

You can use the button to insert a hyperlink to an external website.  If you want to link to a topic within the documentation system, you need to use a topic link token.

To insert a link to another topic:

  • In the text, type $O$function_code$O$.  For example: $O$%WDHS0124$O$.  This gets replaced with a link to the topic, using the context description from the function.
  • To use text for the link that is different from the function description, type $O$function_code|alternative text$O$.  For example: $O$%WDHS0103|Find the prospect$O$.
  • You can also use the lookup: select Topic.
