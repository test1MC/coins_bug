Reserved Styles

There are a number of special formats you can apply to text.  These are:

Name Usage
Field Field labels. Example: Account No.
Field Name Name of a database field. Example: avm_num
Key Keyboard keys like Ctrl, or on-screen text buttons like OK.
Module module names.  Example: .
Parameter parameter names.  Example: PL/AUTOREF.
Screen Type Text displayed on the screen. Example: Loading. Please wait.
User Input Text that the user types. Example: myfile.doc
User Select Option that the user selects from a menu or drop down. Example: Apply Changes

 

To apply a reserved style:

  1. Highlight the text you want to apply the style to.
  2. Click Styles drop down (the first one).
  3. Choose the name of the style from the drop down list.

Reserved styles - configuration

The reserved styles are set up in the style sheet. 

If you want to use additional styles, edit the docuser.css style sheet (in the /styles/ directory of the installation).  For example, to add a red, underlined style called ulred, you would enter:

SPAN.ulred{
    color:red;
    text-decoration:underline;
}

in the style sheet. 

To get the styles to appear in the drop down, add the style name to HTML Span.
