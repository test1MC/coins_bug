Format O (HSBC Hexagon (New Format))

Fields in HSBC Hexagon (new format) statement files are standard CSV format (comma-separated, quote delimited). The fields are as follows:

  • currency
  • /account
  • transaction date,
  • value,
  • description,
  • transaction type

If the transaction type is 'CHQ' then the number comes from the description field. The and account are contained in character positions 1-6 and 8-15 respectively.
