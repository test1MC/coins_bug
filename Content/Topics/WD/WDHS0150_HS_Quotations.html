Configurable House Sales Quotations

allows you to configure the format used for printing quotations in order to print to your business requirements and use pre-printed stationery.

You define the layout of quotations using Document Designer. See Configurable Documents for general information on how to configure documents for printing.

If you want to produce quotations with different layouts, you need to set up a separate configurable document for each different type of quotation.

To define the layout for printing quotations:

  1. Use the document type Quotation.
  2. When the quotation is printed, you can configure to include an event automatically in the sales history.  Enter the event code and any comments.
  3. The body of the quotation consists of sections using the following hard-coded names:
    • EXTRA - to print the that have been ordered on the plot.
    • EXTRABRK - to print the totals for .
    • EXTOPT - to print a combined list of and . 
    • EXTOPTBRK - to print the totals for and .
