Report Writer - Grouping Report Data

Either when you set up the report initially, or afterwards, you can specify how the information will be grouped and sorted, and which subtotals will be shown.

To group the report data:

  1. While you are updating the report, click the Group tab.
  2. In the Group By column, select the field you want to group the report by. Records on which the value of this field are the same will be grouped together and sorted in the order you specify, and you can show totals for the group.
  3. In the Label column, enter a label to show on the headers and footers (before and after each group).

    NOTE

    Your system administrator may set up more detailed group headers (using Default Report Labels), which show, for example, a code and a description for each group.

  4. Choose whether to sort the group in ascending or descending order.
  5. Choose whether or not to display group headers (a label and the value of the group by field) before each group, and group footers (a label and the value of the group field, plus the totals of any totalled columns).
  6. If you want the report to start on a new page when the value of one of the fields changes (for example, to print information about different on separate pages), use Page Level to select the field.
  7. If you want to produce a summared report based on details that you do not want to show, use Summary Level to select the field to summare by.

     

    For example, if you want to produce a report on costs by section, you could produce a cost code report, but summared by section. This would show the cost code totals for each section, but would not show the individual costs for each cost code.
