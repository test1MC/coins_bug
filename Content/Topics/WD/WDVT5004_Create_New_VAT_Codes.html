Create New Codes

If a change in rates involves new rates that you have not used before, you need to set up new codes to use the new rates.

If you already have old codes with the new rates (for example, if the change in rates involves reverting to rates that were in force previously), you do not need to create new codes.  If these codes already exist, ignore this step and proceed to the next step Unretire Old Codes.

Two new codes will normally be required:

  • New Standard Rate Purchase
  • New Standard Rate Sales

and in Ireland:

  • New Reverse Charge Standard Rate Purchase
  • New Reverse Charge Standard Rate Sales

and for house builders:

  • New Non-Recoverable Standard Rate Purchase

These codes can safely be set up in advance so that they are ready to use when the rates change.

To set up new codes:

  1. Go to Codes.
  2. Click to add a new code.
  3. For each of the new codes, enter a code and description. For example:
    • P11 New Standard Rate Purchase
    • RS11 New Standard Rate Reverse Charge Sales

      NOTE

      1. You may wish to use a naming convention for the new codes that ensures they display alongside the existing codes. For example, if the existing standard rate purchase code is P1, using a new code of P11 should ensure that the new code is displayed after the existing one.
      2. You may also wish to change the descriptions on the existing standard rate codes to “Old Standard Rate…”.
  4. Complete the remaining fields, taking note of the following points:
    • Rate: Enter the new rate.
    • Inactive: If you set up the codes in advance, you may wish to  this; however you must remember to make the codes active when the rate changes.
  5. Save the record.
