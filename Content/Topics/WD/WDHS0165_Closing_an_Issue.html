Closing an Issue

When all tasks allocated to an issue have been closed, closes the issue automatically.

You can also close an issue manually.

To close a issue manually:

  1. Open the issue summary.
  2. Click the History tab.
  3. Using the Add Type drop-down, choose the event for changing the status of an issue, and click .
  4. Select the Issue Closed status and enter any other details.
