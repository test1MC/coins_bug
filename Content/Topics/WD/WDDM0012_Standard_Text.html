Standard Documentation Text

You can use standard text for blocks of text that will be the same for several different topics or help nuggets (for example a caution about changing parameters on every parameter maintenance page), or for text that may change depending on the user's set up (especially country-specific wording).

To insert standard text:

  • In the text, type $T$text_token$T$.  For example: $T$audit_process$T$. 
  • You can also use the lookup: select Standard Text.
