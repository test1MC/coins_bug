Entering Your Own Help

If you have created your own functions, you will probably want to enter side-frame help for them.

To enter online help:

  1. Go to Help Maintenance.
  2. Click to create a new help nugget.
  3. If the nugget is for:
    • a function, set the Type to Function and enter the function code in the Function field.
    • a field, set the Type to Field and enter the field code in the Field field.
