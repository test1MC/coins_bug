Report Writer End-User Training Documentation

End-user Training documentation covers all aspects of writing a report.  Documentation and training exercises are provided on the client site for easy access to all users.

  • Sourcing, researching, and finding the data needed for a report
  • Building a report; selecting the query, defining the layout, filtering and sorting the data, adding fields.
  • Enhancing reports using totalling methods; sorting and summarising data.
  • Using calculations to manipulate the data.
  • Displaying the information graphically using standard conditional formatting and charts.
