Project Management Specification Templates

Project Specifications are a list of section codes and descriptions; you can relate Project Management items to sections in the Project Specification. 

You can create specification templates which are available for all projects in Project Management.  These can be copied into individual projects.

To set up specification templates:

  1. Go to Specification Template.
  2. Create templates with a Code and a Description. 

  3. You can then add sections and descriptions to the template specification.
