Creating a PCard Batch

When all the transactions have been marked as "reviewed" and any errors corrected, you can create a batch of PCard transactions.  When this batch has been posted, creates a batch of fully paid and costed invoices.

To create a PCard transaction batch:

  1. Go to PCards Workbench - Administrator Mode.
  2. From the Options menu, choose Create PCard Batch - Select .

To post PCard batches:

  1. Go to PCard Transactions.
  2. the PCard batches you want to post.
  3. From the Choose Action menu, select Post Batches and click .

    For each PCard batch, automatically creates and posts two batches:

    • a batch of invoices (Batch Type PLINV).
    • a PCard payment batch (Batch Type PLPPAY) which marks the transactions as paid.

Configuration

Use Batch Configuration to configure the following batch types:

  • PLPCARD (Purchase card transactions)
  • PLPPAY (Purchase card payments)
