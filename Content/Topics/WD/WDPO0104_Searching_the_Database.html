Searching the Database

You can use the procurement system to find appropriate offices, for example, offices which supply the commodities or services you require, or which work in particular areas.

To prepare a mailing list by searching the database:

  4. In the Area section, select the code for the geographical area you want to search. By default, inserts the area defined on the project.

    The area on the project is defined in the Location field on the New Project screen.
