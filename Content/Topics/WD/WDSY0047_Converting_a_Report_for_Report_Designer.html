Converting a Report for Report Designer

Report Writer allows you to write fairly straightforward reports. For more complex reporting requirements, provides a Report Designer. Additional features in Report Designer include the following:

  • Dynamic conditions allow you to exclude records based on calculations
  • Generated record sets
  • Advanced layout and formatting
  • You can include subreports within the body of the main report
  • 10 levels of sorting
  • Analysis reports (summaries by a chosen field, at the end of the report)
  • Flexible design for headers and footers
  • Configurable input forms

If you want to use Report Designer to enhance a report you've written in Report Writer , you can create a copy of the report in Report Designer format, and modify that.

NOTE

You cannot use Report Writer to modify reports written or modified using Report Designer, and you cannot convert them into Report Writer format. However, your original Report Writer reports are still available.

To convert a report for Report Designer:

  1. Go to Report Writer.
  2. In the Report column, click the link for the report you want to convert.
  3. From the Options menu, select Convert to Report Designer.
  4. Give the new report a code (and a different title if you want), and click the [Forward] button.
