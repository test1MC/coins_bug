Unscheduled Tasks

When you add a task, you can leave it unscheduled, and schedule it later.  A task is unscheduled if:

  • It does not have a date and time against it.
  • It has a date and time, but has been assigned to the "TBA"  (if you are using this feature). 

To schedule an unscheduled task:

  1. Go to {Customer Care} Workbench.
  2. Click the Unscheduled tab.

    This shows tasks that have no date and time, or that have been assigned to the TBA .

  3. Click the next to the task you want to schedule.

    NOTE

    If you are using the TBA feature, make sure that the TBA Staff filter on the scheduling screen is set to Show. The scheduling screen opens with the TBA column at the left, and the day the TBA task has been scheduled on at the top.

    See Scheduling a Task for more information about the scheduling screen.

The TBA

When adding a task, you cannot save it without assigning it to an .  If you want to be able to add tasks and assign the operative later, you can set up a dummy "TBA" (to be arranged) operative.  Tasks assigned to this operative will appear on the Unscheduled Tasks tab.

To set up a TBA :

Using Sales Staff, add an record (for example, called TBA) and the Scheduling TBA box.

A user record with the same code must exist.
