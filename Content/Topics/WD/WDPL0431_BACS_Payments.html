Once you have created and frozen the payment batch, you can print the and generate payments. 

If you print remittances separately from processing the payments, you must print the remittances after you have processed the payments.

To process payments:

  1. Go to Create .Create – Select Batch

  1. In the Batch Number column, click the link for the batch that contains the payments you want to process.
  2. Select a configurable document.  (You need to do this even though you are not printing .)Select Check Form

  3. On the Main tab it will show you the number of items in the payment

  4. On the Processing tab:
    • Select the bank account from which the payments will be made.
    • Verify the date to appear in the file; this defaults to today’s date.
    Create the

  5. Select any Output Options. 
  6. Click .

    displays a message informing you of the number of and/or payments that will be included in the payment run. To continue, click OK. Otherwise, click Cancel to return to the selection fields.

    Depending on the output options you selected, either displays a "Task completed successfully" message to tell you that the print request has been sent, or displays a "Generating" page, and then displays the printable output PDF file).
  7. Next, print the remittances; use Print Remittances.Print EFT

  8. You can print a Payment List to check that the payments are correct (use Payment List).
  9. Finally, you need to post the batch.
