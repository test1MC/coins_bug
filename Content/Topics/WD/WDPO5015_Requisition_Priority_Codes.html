Requisition Priority Codes

Requisition priority codes indicate the relative priority of requisitions. You can arrange the Requisition Workbench display by priority code.

To set up priority codes:

  1. Go to Requisition Priority Codes.
  2. Fill in the fields as follows:
    Requisition Priority Code A single-character code for the priority of the requisition. Note that it is the sequence number, not the code itself, which determines the priority.
    Description Text to show the meaning of the code.
    Sequence The relative priority of the code. Codes with a lower sequence have a higher priority.
