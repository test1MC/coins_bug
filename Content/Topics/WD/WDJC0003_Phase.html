Phase

The phase code provides a way of dividing large into smaller units. Phases are optional -- some may be divided into phases while others are not, and those that use phases can use different ones.

The phase code can be up to two characters long.

You define whether a uses phases in , and define the phase codes for a using Phases (on the Options menu for the within ).

NOTE

We do not advise you to use phases; they are provided in for compatibility with earlier versions, but are not fully supported in all areas of .  We recommend using analysis sets to summare (see Using Analysis Sets to Manage Phased ).
