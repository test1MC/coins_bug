Deleting Wiki Content

You can only delete content that you have entered.

You can delete a version of a topic that you created.  If you want to completely remove a topic that has several versions, you would have to delete each version (or use Browse Topics if you have access to it).

You can delete an attachment that you have added, but only if no other topics contain links to it.  So for example if an image file is included on another topic, you need to remove the reference to the image on that topic before you can delete the image attachment from the topic it's attached to.

To delete a topic:

  1. Click at the bottom of the topic.
  2. At the "Delete this topic version" prompt, click OK.

    If a previous version of the topic exists, displays that.  If no other versions exist, displays the parent topic.

To delete an attachment or comment:

  1. Browse to the topic the attachment or comment is attached to.
  2. Click the link to the attachment or comment at the bottom of the topic.
