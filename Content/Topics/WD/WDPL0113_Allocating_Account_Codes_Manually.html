Allocating Account Codes Manually

If you are not using the auto-numbering feature of , when you create a new record (by selecting the company and office from the Company Information Workbench), you need to provide the account code yourself.

To allocate an account code manually:

  1. Create a new record (see Creating a New Account).
  2. Enter a new account code in the field.

(It is advisable to use consistent codes, in other words, the same number of characters and/or digits, so that they are easy to remember).

NOTE

If you want subcontract on invoices to be treated correctly, you must make sure that the codes for of subcontract in are the same as the subcontractor codes for those in Subcontract Ledger. The Subcontractor details are maintained in the Subcontractors Ledger.

If the same account code also exists in other , these accounts can be linked together so that any changes made to the account details in one , will automatically be updated in the accounts that exist in other .
