Format K (Bank of Scotland CHOBS)

This is an ASCII CSV format. The file to load is the Sterling Account Transactions (the file is denoted with an 'S.TRN' extension). The file line entries are listed below in order:

  • Date (yyyymmdd format)
  • Value
  • Transaction Type
  • Description (for , this will be the number)
  • Ignored field 1
  • Ignored field 2
  • Account
