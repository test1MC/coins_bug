Field

Description
Status A code to identify the action item status.

Open Whether this status is open or closed.
Dormant Whether this group code is dormant.

Set up a list of Status Codes for the each of the menu items.  Ensure that each list has at least one closed status, that is, one status where the Open option is not selected.
