Notes Re IRmark

Since 6th April 2009, the electronic messages sent to HMRC for PAYE in-year movements have needed to include a security feature known as the IRmark.  This has been a requirement for UK subcontractor CIS returns since April 2007. All UK clients who also pay UK subcontractors and file CIS monthly returns electronically should, therefore, already have the IRmark technology working on their server. However, before doing in-year filing for the first time, any client who is not also filing CIS returns should log a support call with the communication so that the necessary arrangements can be made.

As yet, there is no requirement to include the IRmark in end-of-year PAYE returns.

Incorporation of the IRmark into the in-year returns has required a enhancement delivered as a software fix ? see Install Fixes.
