Setting Up to Use Price Functionality

Price functionality in allows you to build schedules of items and/or work that you want to request quotations for, select companies to send the to, and record and compare the quotations you receive.  You can do this (separately) for projects that are at both and stage. For projects at stage, you can raise orders based on the winning quotation.

The names of the events are configured using Material Package Captions. 

Spreadsheets

You can send in the form of a spreadsheet, which the can fill in and return to you.  You can then load the details of the quotation into for comparison with other quotations.  This feature requires the following:

  • The office to which you want to send the spreadsheet must have a contact that has a category of , and a valid email address.
  • The following PI parameters must be set to Y to enable the Schedule and Comparison tabs for each type of package:
    Material packagesUSESCHMA
    packagesUSESCHPL
    Subcontract packagesUSESCHSC
  • There must be XML templates for the spreadsheet in $BASE/ var/ spreadsheets/ templates/.  
  • The following PI parameters must be filled in with the filename of the spreadsheet templates:
      stage stage
    Material TMPLTMAT TMPLCMAT
    TMPLTPLT TMPLCPLT 
    Subcontract TMPLTSUB TMPLCSUB 

Mail Merge

Use Merge File Maintenance to set up the letter against function %WPIBCTG1, and use the Method ctg-rsp.letterSent so that records the date when the is sent.  See Setting Up a Generic Mail Merge.  You can use fields from the following tables: pi_project, pi_package, ci_company, ci_office.

Workflow

You can set up a workflow to control the process of selecting a winner and raising an order, in which case instead of a Set to Winner option, users will see a Set to Recommended option, which triggers the workflow.  This feature requires the following:

  • The PI parameter PIWFLOW must be set to Y.
  • The following PI parameters must be filled in with the name of the workflow templates to use:
      from requisitions from schedules
    Material  PIWFEMAPIWFMA
     PIWFEPLPIWFPL 
    Subcontract  PIWFESCPIWFSC 
    Overhead  PIWFEOV 
     PIWFEST 
    Workshop Job  PIWFEWS 
    Asset  PIWFEAS
