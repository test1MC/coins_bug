Page Banner

 

The page banner appears at the top of the page.

It includes the following:

  • Selector
  • Global Search
  • Desktop Button and Minimed Frames
  • Report Status
  • Account Name and Image
  • Outstanding Actions indicator

 

Selector

A selector in the header area of the desktop allows you to change to a different (if you have access to more than one ).

Global Search

The Global Search allows you to find records in by typing part of the name or description, or other information (such as part of an address). It also allows you to find and run functions.

For more information, see Global Search.

 

Desktop Button and Minimed Frames

The Desktop button allows you to switch between a function in the main area and your Desktop. If you have minimed frames, an indicator shows how many there are; the minimed frames show when you move the mouse pointer over the Desktop icon, and you can close, restore or maxime them. For more information about the Desktop, see Desktop; for more information about frames, see Frames.

 

Report Status

The Report Status icon shows two indicators: 

  • The number of your reports that are currently generating is shown in a red circle.
  • The number of your unread reports is shown in a circle of the theme .

If you move the mouse pointer over the icon, a popup lists the reports that are generating.

Click the icon to open Report Status. For more information about Report Status, see Report Status.

 

Account Name and Image

The page header shows your user name (or user ID) and an image to represent your account.

It also gives access to a menu of other options.

See also: Changing Your Account Image and Display Name.

 

Outstanding Actions Indicator

If you have any outstanding activities assigned to you, an indicator on your profile picture shows how many: . If you move the mouse pointer over the profile picture, then over Activity Workbench on the menu, a popup menu shows the activities.

For more information about the Activity Workbench, see Activity Workbench.
