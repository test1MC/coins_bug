Browse Reports

Many browse screens allow you to produce a report based on the contents of the browse.  By default the report contains the fields on the browse, but you can select other fields to include on the report. 

This also produces a spreadsheet that contains the same data.  Since generates this using a background generator, it will not time out if there are a large number of records, and may be more convenient than exporting directly from the browse.

To produce a report:

  1. Use the filter on the browse to identify the records you want to report on.

    The report includes all the records in the filtered record set.

  2. Click .

    displays a selection screen.  By default, the Selected box contains the fields that are shown on the browse.

    Browse Report Field Selection

  3. In the Selected: box, add, remove and re-order fields.
  4. Select the output options.
  5. Click .

Once the report has generated, you can use Report Status to view the report or open the spreadsheet.
