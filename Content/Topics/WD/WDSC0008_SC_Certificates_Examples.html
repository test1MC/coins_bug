Configurable Certificates - Examples

Remember that you can use the lookup when adding lines to see which fields are available.

To print the subcontractor's name and address in the header:

  1. Add a header line containing the subcontractor's name field RO_sbm_actualName(the example below also prints the subcontract order number on the same line): 

  2. Add a second line containing the first line of the subcontractor's address (field RO_sbm_actualAdd__1):

  3. Add further lines containing the fields RO_sbm_actualAdd__2, RO_sbm_actualAdd__3, RO_sbm_actualAdd__4 and the (RO_sbm_actualPcode).

To print column headings:

  1. As the last row of the header form, add header lines with a separate field for each column heading. Use the same widths as the fields they correspond to.

    This prints column headings as shown below:

Description Total Certified Previously Certified Currently Certified

To print the body of the certificate:

Suppose that in the body of the certificate, you want detail lines showing the current valuation, previous valuation and this period values for each of the following: gross certified, retention, discount, tax, and payment.

You do not need to define a separate line for each of the detail lines you want to print.

  1. Add a body line containing the fields cdf_desc (the line description), scd_cum (cumulative value), scd_prev (previous value) and scd_tp (this period value):

  2. To enter the Rate percentage on the line, use the field cdf_rate.

    These fields will be printed for each line specified in the parameter list. In this example, the parameter list would look something like:

    Body IDs: GRC,RET,DIS,TAX,VAT,PAY

    depending on the Item IDs you have used.

To use detail amounts in the header or footer:

Suppose you have used the item IDs PAY and RET for payment and retention, and you want to show the 'this period' payment amount in the footer.

  1. Add a footer line containing the field RO_sbp_scd_tp^PAY .

    In this case, you need to give the Item ID (PAY), since the header and footer are not covered by the parameter list.
  2. As another example, to include the cumulative retention, enter the field RO_sbp_scd_cum^RET.
