Wiki

The Wiki is a tool that you can use to enter and maintain shared information.  Any user of at your company can have access to the Wiki, either to read or to enter and update information.  You can use the Wiki as a way of distributing information, as a knowledge base that everyone can add to, or as a discussion forum.

The Wiki can be divided up into a number of different areas, known as "webs", and each user can have access to different combinations of webs.  So, for example, you might have one web for general company information which everyone can read but only selected people can contribute to, one web for social events that everyone can read and contribute to, and separate webs for individual departments that only members of that department can read or contribute to.

A Wiki topic can contain images, comments and attachments.  It can have links to other Wiki topics, to topics in the documentation, and to external websites.  It can also have links that run functions, so you could use it to build a tutorial or a customed menu system.  

Setting up Wiki

To set up Wiki, you first need to provide names for the individual webs that make up the Wiki.  Use Wiki Webs.

To grant a user access to the Wiki:

  1. Open the user record in Users.
  2. Click the Wiki tab.   

    This lists the Wiki webs you set up in Wiki Webs.  (If you haven't set up any webs, the Wiki tab does not appear.)

  3. For each web you want the user to be able to read, the Wiki Read box.
  4. For each web you want the user to be able to contribute to, the Wiki Write box.
  5. Save the user record.
