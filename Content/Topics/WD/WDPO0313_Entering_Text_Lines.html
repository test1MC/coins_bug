Entering Text Lines

When entering an order or a requisition, you can add a text line.

To add a text line in the Order Lines or Requisition Lines frame:

  1. Using the Add Type drop-down at the bottom of the frame, choose Text.

  2. Click to add a new text line.
  3. Enter the text in the Description field.
  4. To include a standard phrase in the text frame, click and select the phrase code.  ( adds the phrase after any existing text, but you can use the mouse to select and move it within the Description field.)
  5. Click to save the line.

Configuration

Phrases are maintained using Standard Phrases.
