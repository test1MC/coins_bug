Links in a Wiki

Links allow readers to move between topics.  To add a link, you need to know the WikiWords that identifies the topic you want to link to.

There are different types of link. To enter a link...

  • ... to another Wiki topic, put double square brackets around the WikiWord for the topic you want to link to.  For example: [[ProjectsThamesBridge]].  This will appear as a link like this: ProjectsThamesBridge.

    To enter different text for a link, add the text in square brackets as follows: [[ProjectThamesBridge][A bridge over the Thames]].  This will appear as a link like this: A bridge over the Thames.

    • NOTE

    If there is no topic with the WikiWord you entered, will display the WikiWord (or the alternative text if you entered any), followed by (?).  This is a link which users can click to create that topic.
  • ... to a Wiki topic in a different Wiki Web, add the Web name and a before the WikiWord.  For example: [[Social.XmasParty]] will link to XmasParty in the Social Wiki Web.
  • ... that will run a function, enter the function code with $F$ on either side, inside double square brackets.  For example, [[$F$%WGL5100BGAN$F$]].  displays this as a link, using the function name (for example: General Ledger Dig Deeper); when a user clicks the link, the function loads in the same window, replacing the Wiki topic. 
  • ... that will display a topic from the online documentation, enter the documentation topic code with $O$ on either side, inside double square brackets.  For example, [[$O$%WDUG0066$O$]].  displays this as a link, using the topic title (for example: Wiki); when a user clicks the link, the documentation topic opens in a documentation window. 
  • ... to an external website, enter the web address with $W$ on either side, inside double square brackets.  You can use alternative text for the link; for example: [[$W$www.bbc.co.uk/news$W$][News from the BBC]].  When a user clicks the link, the external web page opens in a new window or tab.
