Import CSV File Layout for Mark-Ups

Load Markups allows you to import mark ups from a CSV file.

Mark-ups are a way of adding a premium to an incurred cost which can then be re-charged to a customer.  Use Markup Rates in FM Set up to maintain mark ups manually.

The Import CSV file layout for mark-ups is found below.
