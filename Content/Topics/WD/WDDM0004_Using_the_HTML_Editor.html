Using the HTML Editor

Various text fields in , such as email message text on report output or mail merge, use an HTML editor.  This provides a number of familiar formatting and layout options.

HTML editor options:

Text formatting

  • To format text as bold, italic or underlined: highlight the text and click , or . See also Reserved Styles.
  • To change the or background of text: highlight the text, click the arrow next to or , then choose the .
  • To change the font or size of text: highlight the text and choose the font or font size using the Font Family or Font Size selectors.
  • To use more advanced formatting, highlight the text and click .

Paragraph formatting

  • To change the alignment of a paragraph: highlight the paragraph and click , , or .
  • To convert a paragraph or paragraphs to numbered or bulleted lists: highlight the paragraph or paragraphs and click or . To convert the list back to a normal paragraph, highlight it and click the button again. See also HTML Editor - Lists.
  • To increase or decrease the left hand margin of a paragraph: highlight the paragraph and click or .
  • To change a paragraph into a heading: highlight the paragraph and choose the heading level using the Heading selector.

Tables

  • To insert a table: click , enter the number of columns and rows, and click Insert. inserts a blank table. Click the cells to type the content of the table. You can also use the following buttons, which are enabled when you click a table cell:
    • to add a row above or below the current row.
    • to add a column to the left or right of the current column.
    • or to delete the current row or column.
    • to change the properties of the table.
    • to change the properties of the current row.
    • to change the properties of the current cell.
    • to merge table cells (select more than one).
    • to split merged table cells.

Inserting

  • To insert an image, right-click where you want to add the image and choose from the menu, then enter the URL of the image and click Insert.
  • To insert a link to a web page or external document: select the text you want to act as the link, click , and enter the URL. If you don't select any text, the link text will be the same as the URL. See also Inserting Links.
  • To insert a horizontal line: click .
  • To insert standard images, text, glossary entries or links to other documentation topics: click ; see Inserting Standard Links, Images and Text.
  • To insert a block of user-defined HTML formatted text: place the cursor where you want to insert the text, click and use the lookup. Use HTML Snippet to define the text you can insert.

The HTML editor may not allow you to achieve exactly the results you want. If you want to edit the HTML source directly, click to open the source in a new window. To save changes to the HTML source, click Update.
