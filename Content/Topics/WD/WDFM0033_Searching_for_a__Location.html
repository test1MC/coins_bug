Searching for a Location

When a caller contacts you with a new call, you need to search for their and location. You can search for a location using any of a number of details, such as the , address, or geographical area. Alternatively, you can select a and open the location details from there.

If the location is not already set up on the , you can add a new location.

To search for a location:

  1. Go to Locations.
  2. Use the standard search or the advanced filter to search for the location.
  3. Click the link in the Location column to open the Location Workbench.

To search for a location on a contract:

  1. Go to {Jc Contracts}.
  2. Use the standard search to search for the contract.
  3. Click the link in the column to open the contract details.
  4. Click the Locations tab.
  5. Click the link in the Code column to open the Location Workbench.

From the Location Workbench, you can amend the details of the location if necessary.
