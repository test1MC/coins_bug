Negative Returns

HMRC will not accept entries on the CIS 300 for a subcontractor with a negative net value.  You must notify HMRC separately.

If the net position for a subcontractor after running the Process Month End Return is negative (for example if you have a payment for which you made a submission in the previous month, and have not made sufficient payments since to cover the difference), excludes the entry from the submission. 

In this case you should notify HMRC of the negative return(s) (using the CIS helpline). 

NOTE

When contacting HMRC to inform them, do not use the expression "negative return".  Instead, say that you want to make an amendment to the previous return, as the negative amounts represent cancellations or adjustments to previously returned figures.

You can then indicate in that you have notified HMRC.

The Monthly Status Report indicates the number of negative returns, and the number for which you have not notified HMRC (that is, you have not indicated in that you have notified HMRC).
