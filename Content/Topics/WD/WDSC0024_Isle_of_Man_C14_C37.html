Isle of Man C14/C37

The SCC14C37 is an Isle of Man (only) tax document which produces a C14 Subcontractor?s Tax Deduction Card (to be given to the individual subcontractor), listing their payments and deductions over the year (effectively a summary of the C11). At the end of the print is a final page with the information required for the C37 to be returned to the Tax Authorities.

To configure an SCC14C37 document:

  1. Go to Document Designer in the System Module. 
  2. Use the SC Annual C14/C37 (IoM) document type.
  3. See Configurable Documents for information on how to define configurable documents.
  4. The following tables are available:
  5. The SCC14C37 has the following tables available to it: sc_mast, sc_sc60, co_config.
  6. The SCC14C37 has the following forms:
    • The BODY form provides information from sc_mast and the client's information from co_config.  The required figures RO_TaxCertno, RO_LabourTaxYear, RO_MaterialsTaxYear, RO_TaxDeductedTaxYear which are all populated by calculations in the print program itself.
    • At the end of the document is the Contractor's Annual Return Form (by default C37).  This provides information about the client's company using the co_config table and the required statutory declaration wording and space for a signature.  There are number of RO fields: RO_C14Count, RO_TotalLabourTaxYear, RO_TotalMaterialsTaxYear, RO_TotalTaxDeductedTaxYear, which present the total annual values summarised from the SCC11.  These are all populated by calculations in the print program itself rather than being available as RO fields for a particular table.

      The field RO_TaxYears (on sc_mast) provides the current tax year as YYYY/YYYY.

    • The FOOT and CFOOT forms show a page count using the standard tokens.
