Selecting a Potential Prospect

When adding a prospect, if there are people in the database that may be the same as the person you want to add, displays a list.

Do one of the following

  • If the person you want to add is shown in the list as a prospect, click the link in the column to display the details of the prospect. You can update these details if appropriate.
  • If the person you want to add is shown in the list, but is not a prospect, click the link in the column to add the person as a prospect. displays the Add Prospect screen with the name and address details already filled in.
  • If the person you want to record is not shown, click ADD to add a new prospect record. displays the Add Prospect screen but the address details are blank. Click to add a new person. When you save the new person details, displays the Add Prospect screen.
