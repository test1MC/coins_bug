Defining Standard Subcontractors Certificates

The Subcontractor Certificate defines the method of calculation used to pay subcontractors, and the layout of the input screen for subcontractor certificates.

The standard certificate definition is used as a "template" for the payment calculations for each new subcontract. If necessary, you can modify the definition for individual subcontracts using Subcontract Certificate Definition (see Creating a Subcontract Record - Changing Certificate Details for the Subcontract).

The first step in defining a standard certificate definition is to know which fields you want on the input screen. These depend on the layout of the subcontractor certificate you use. For each field, you also need to decide whether the value is cumulative or this period, and whether or not the field is available for input.

NOTE

It is very important to make sure that the standard Subcontractor Certificates are defined correctly, in particular that the entries in General Ledgerand Status Ledgerare as required. If you make any changes to the certificate definition, ensure that test transactions are entered and checked before using the certificate definition for live work. If you are in any doubt, contact Construction Industry Solutions for assistance.
