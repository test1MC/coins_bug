Adding Topics to a Wiki

If you have permission to write to a Wiki, you can add new topics to it. 

NOTE

You may have permisson to write to some Wikis, but only to read others.

You can also add topics that have been requested or suggested previously.  These are topics for which a link has been created, but do not yet exist; they are identified by a (?) link on a topic.

To add a new Wiki topic:

  1. Click at the bottom of the topic.

    The new topic will become a child of the current topic.

  2. In the first field, enter a title for the topic.  This is what will appear at the top of the topic.
  3. In the second field, enter a WikiWord as the name of the topic.  This is what other users will need to enter if they create a link to this topic.
  4. Leave the Major box ed, to create version 1.0 of the topic.  (You can clear the box, but this will create the new topic with a version number of 0.01.)
  5. Enter the text of the topic.
  6. Click .

To add a topic that has already been suggested:

  1. On the topic where the link to the non-existent topic has been added, click the (?) next to the link.

    opens a new, empty topic with the WikiWord already filled in.  The parent of the new topic will be the topic from which you followed the link.

  2. Enter a title for the topic in the first field.
  3. Edit and save the topic as above.
