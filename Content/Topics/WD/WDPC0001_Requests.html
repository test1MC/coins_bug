Requests

Requests allows you to create requests (these can be either requisitions for internal users, or orders for external customers - however, currently only internal requests are supported) from a .  The uses an "add to basket" approach similar to many Internet shopping sites.

Running Requests

How you start Requests depends on how it has been set up at your company, but usually you will either have a shortcut or an Internet Explorer "ites" link.  After you have clicked the link or the shortcut, you need to log in to using your normal User ID and password. 

Creating a request

The basic procedure for creating a request is as follows:

  1. Find the items you want to request by:
    • Using the search,
    • Digging down through the groups, or
    • Copying a previous request.
  2. Add the item or items to your basket.
  3. When you have added all the items you want to request, go to the Checkout and specify who and where the request is for.

Setting up shortcuts for Requests

The URL for the shortcut is as follows:

<base>/wocoins.p?TopMenu=%WPCX&kco=n

where

<base> = value of SY/WEBROOT parameter + / + value of SY/WEBSCRPT parameter.  For example: http://123.45.99.9/cgi-bin/live.cgi

n = The number (kco) of the for which you want to run requests.  You may have to set up separate shortcuts if you have the running in different or environments.
