Format L (Northern Bank)

The first two lines in a Northern Bank statement file are header lines, which Load Bank Statement ignores. The remaining lines each contain a single string with data at specific character positions. The character positions of the fields that Load Bank Statement reads are as follows:

Characters Field
1-6
7-14 Account number
16-17 Transaction type
36-46 Transaction value (in pence)
47-64 Description
65-82 If the transaction type is 03, the transaction is a , with the number right-aligned.
102-106 Date in Julian format (YYDDD where DDD is the number of the day in the year)
