Using Attachments When Faxing From and VSI-Fax

When printing certain documents from such as Purchase Orders, the original document may be printed onto headed or pre-printed stationery which has information such as company terms and conditions printed on the reverse. To include information such as this in a fax sent from VSI-Fax, it is necessary to create the attachment and upload into VSI-Fax, and then configure the datafile to reference the attachment.

To create an attachment simply create the document in the standard way (for example, using MS Word) and follow the instructions for creating the .prn file of the document and converting it to a TIFF file.

Once you have created the TIFF file it must be uploaded into VSI-Fax using the following command: (NOTE: it is crucial that you do this as the VSI-Fax Administrator so the attachment is accessible by all users of VSI-Fax) :

$ vupload -d "Description of Attachment" attachment tc.tif

You can check the attachment has been successfully uploaded by running the following command:

$ vupload -l attachment

The VSI-Fax server will return a list of the TIFF files that have been updloaded as attachments.

To allow to reference this attachment you must ensure that a copy of the TIFF file exists in the overlay directory and also that it included in the datafile. With an example of an attachment tc.tif - an entry must be made into the dat file of:

tc.tif,,$
