Ground Rent and Service Charges Configuration

House Sales allows you to bill purchasers for automated repetitive charges (ground rent, service charges) and additional manual charges.

  1. Ensure that the HSCHARGE batch type is set up correctly in Batch Configuration.  (If the batch type is not shown, open and save Module Configuration to create the batch type automatically.)
  2. Set the House Sales Parameters.  There are separate parameters for rents, for service charges and for other charges.  For optimum functionality all these parameters should be set.
  3. Use Ground Rent Billing Periods and Service Charge Billing Periods to create standard tables of billing periods for rents and for service charges. If these vary by then set them up on each : the standard list then becomes the default for with no billing periods of their own.
  4. Use Transaction Categories to configure one or more House Sales Financials transaction categories for rents and/or service charges.  Each transaction category will be linked to a General Ledger control account for the Debtor Balance on that category: transaction categories should not normally share a control account.
  5. Use Journal Maintenance to configure a new House Sales Journal for rents and one for service charges.  The configuration of these journals will determine the General Ledger and/or Status postings to be made for rents and/or service charges. 
    • Set the Usage to Plots.
    • Set up two journal lines: the first one should have a line type of Addition and post to the appropriate GL revenue account; the second should have a line type of Deduction and post to the ground rent debtor account.
    • The Item ID of the first line of the journal needs to be identified in the appropriate parameter for the type of journal: GRITEMID, SCITEMID or OCITEMID.
    • For ground rent or service charges the journal line should be set up with the same category as defined in the parameter GRCAT or SCCAT.  For other charges the category on the journal line will be overwritten with the category from the charge.
    • Within the Description box on each journal line you can use tokens to automatically pull information into the text.

  6. Use Letter Configuration to configure invoices, statements and any other charge letters required (for example, overdue rent/service charge reminders).
    • Use the letter type Charge.
    • Charge letters can be configured to generate an event in the History on the Sales Workbench.
    • To configure an invoice document so that will automatially generate a unique invoice number, the Invoice Reference box (on the Parameters tab).

Plots must be set up correctly before legal completion, and the annual rent and/or service charge must be set up on each plot.
