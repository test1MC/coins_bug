Global Search

 

The Global Search feature allows you to search for records in the database (for example, you might search for a by typing a word or words from the name, or for a by typing the ).  It also allows you to find and run functions.  The search is configurable: your system administrator can choose which tables, and which fields on those tables, to include in the search.

To use Global Search:

  1. Press Ctrl+Shift+? (or click the button on the page banner).

    The Global Search screen appears.

  2. Type a word or words to search for. 

    As you type, displays matching records, grouped according to the category of record.

    The results will match words that begin with your search words. 

    • You can type the words you want to search for in any order.  For example:

      report balance or balance report or bal rep
      manchester const       or manch construction

    • To search for an exact match, enclose it in quotation marks.  For example, to search for names that contain the whole word "sub", but not "subcontract", type "sub".
    • You can also use | (pipe) to represent "or", and & to represent "and". For example:

      (John|Sally) Smith       will search for John Smith and Sally Smith.

    • You can enter the module code to help identify functions with similar names (for example, posting report CS will show only posting reports).

    If the total number of matches is less than 500, the results show all matches.  If the total number of matches is 500 or more, only the first 10 results from each category are shown, and if there are more than 10 results in a category, a more... link takes you to a full-screen browse of those records. 

  3. You can filter the search to show only records that belong to the categories you select.

    remembers your filter settings and the text you searched for, and uses them the next time you open Global Search (until you log out).
  4. If you press the Up or Down arrow keys, or move the mouse pointer over an item, a detail frame to the right will be displayed with more detail about that record.

  5. Click the link on an item (or press Return on a highlighted item) to go directly to that item.  Depending on how your system is configured, the page may open in your current window, in a new window or in a frame.   (Alternatively you can right-click to choose to open it in a new tab or new window.)
