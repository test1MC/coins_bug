Subcontract Ledger Set-Up

Before you can use Subcontract Ledger, or whenever you set up a new , you must run SC Configuration. 

  1. Make sure you are connected to the correct .
  2. Go to SC Configuration.

    displays a screen like the one below:

  3. Fill in the fields on the screen (refer to the side frame help)

The next tab shows Defaults - these can be left blank.

The next tab in SC Configuration is Analysis Set Maintenance. Define the analysis sets for Subcontract Ledger. (Analysis sets are explained in the section on Analysis Sets in the System Guide.)

If you want, you can leave setting up analysis sets till later. To set up Analysis sets later, you will need to run SC Configuration again.

To set up codes for the analysis sets, use Analysis Sets.
