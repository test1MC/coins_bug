Journals

House Sales journals are used for moving values between different analysis codes when the sale of a plot or part-exchange is completed.  You can set up journals for:

  • Plot sales
  • Part-exchange sales
  • Plot sale cancellations
  • Part-exchange cancellations
  • Pre-release reservations

Journals are entered in the appropriate workbench.  You may then need to approve (or authore) the journals, and create a batch of journals to post them.

Journals - configuration

Journal Item ID's allows you to set up the identifiers for journal items.  These allow you to refer to lines in different journals that represent the same thing (but may have different line numbers).

Journal Maintenance allows you to set up templates for different types of House Sales journal.
