Other Set-Ups

  • SC Batch Configuration allows you to define the standard characteristics for Subcontract Ledger batches.

    The batch types in Subcontractor Ledger are:

    Batch type code

    Used for

    SCAPPLY

    Subcontractor apply payments batches

    SCAUTH

    Subcontractor authenticated receipt batches

    SCCANCHQ

    batches

    SCCERT

    Subcontractor certificate batches

    SCCHQ

    Subcontractor batches

    SCMAN

    Subcontractor manual

    The batch numbers for SCMAN, SCCHQ and SCCANCHQ are controlled by the batch types CBTR and CBCAN, and are shared with other ledgers that process cash.

  • If the subcontractor works in more than one trade, use this tab to enter the codes for the trades they work in.

    NOTE

    This information is now largely superseded by the Procurement module.

  • If some subcontractors are paid on a timecard basis, you need to define the possible types of activity. Timecards can be used for entering any type of measured work (for example, s of work completed), not just for "hours worked" .
  • Transaction Types allows you to set up Subcontract Ledger transaction types.
  • Subcontract Types allows you to define the subcontract type codes. Whenever a new subcontract record is created, the subcontract is given a code to identify the type of subcontract, for example: only, nominated, or direct.

    For example:

    D Direct
    L Only
    N Nominated

  • Trade Codes allows you to set up codes for subcontractor trades.
  • SC Periods Maintenance allows you to open or close Subcontract Ledger periods.
