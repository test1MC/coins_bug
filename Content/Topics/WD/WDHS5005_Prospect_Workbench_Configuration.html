Workbench Configuration

To use the Workbench, you need to set up the following:

  • Lookup Tables:

    • Market Mix Codes (see Market Mix)
    • Source of codes (and optionally Source of Categories and Source of Areas) (see Source of )
    • Reasons for Buying/Move (see Reasons for Buying)
    • Purchaser/Prospect Status (see Purchase Status)
    • Action Types and Access Maintenance for the Activity Workbench
  • Events (for recording contact with a , see Events) and Event Security (for restricting access to events - see Event Security)
  • Letters for mailshots and letters to individual (see Configuring Mail Merge Letters)

You may also wish to set up:

  • Captions (User-Defined Data)
  • Questionnaires (see Questionnaire Set Up)
