Captions - User-Defined Fields

Captions are fields that you can configure that allow users to record additional information about a record. This is intended for details for which does not already provide standard fields. Many key records in allow you to create captions.  Caption fields are organed into caption groups. If captions have been configured for a record, they are presented as an extra tab when users update the record.

There are two slightly different types of caption, which differ mainly in the way they are configured: "House Sales" style captions and "Company Information" captions.

To configure House Sales style captions

House Sales style captions use a single menu item for maintaining caption groups and captions.

  1. Go to the caption maintenance function (usually on the module's Set Up menu).
  2. First you need to create a caption group: click to add a new group.
  3. Enter the group name. numbers the groups automatically.
  4. Click .

    displays the Caption Group Details screen, where you set up the captions in the group. You can add up to ten captions in each group.

  5. For each caption you want to add, enter:
    • A label - this is the prompt that the user will see. (The Captions tab on the record will only show the captions you give labels to.)
    • A type - the type of data that is required.
    • A default value for the field. This is optional, except for List type captions, for which you must enter a comma-separated list of the values you want users to select from.
  6. Click to save your changes.

To configure Company Information style captions

Company Information style captions use separate menu items for setting up caption groups and captions.

  1. Go to the caption group maintenance function.
  2. Click to add a new group.
  3. Enter a code and description for the caption group.
  4. Click .
  5. Go to the caption maintenance function to set up the captions.
  6. Click to add a new caption.
  7. For each caption, enter:
    • A code and label for the caption.
    • The group this caption belongs to.
    • The type of data that is required.  If you want users to choose from a drop-down list of values, use the type "Character".
    • The format to use for displaying the caption.
    • Optionally, for non-character captions, a default value.
  8. Click .
  9. If the caption is of type "Character" and you want to set up a drop-down list of values, click the Allowable Values tab and create entries for each value you want in the list. If you want, you can then go back and choose one of these as the default value for the caption.

    If you do not set up allowable values, users will be able to enter any text.
