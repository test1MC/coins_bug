Groups

The is organed into hierarchical groups, which can help you to find specific products in the by presenting similar things together.

To find a product using groups:

  1. In the web , click .

    This shows a list of all the top-level hierarchical groups.

  2. You can restrict the list by entering text in the Search field. This will only show items within the group you are currently looking at. For example, if you are looking at the above group list and enter Power, the list will only show 06 Power Generation and 07 Powered Access.
  3. Click the Code or the image to show the contents of the group. The group may contain other, more specific, groups or may contain a list of products. If the results contain only products (that is, no groups), the list shows more information about each product, including a guide price and a full description.

  4. The links above the results list show the parent groups above the group you are looking at.  You can click one of these links to display that group. 

Setting up groups

To set up groups, you need first to set up the group hierarchy using Hierarchy.  You need to give each group a code; we suggest you give the codes a structure to make them easier to manage: for example, a two-digit code for the top level groups, a three-digit code for the next level and so on.  However, the codes can contain letters as well as digits, and it is up to you what structure to use.

Then you need to assign products ( groups or items) to groups in the hierarchy, using Assign Hierarchy.
