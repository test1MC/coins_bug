Table Access Security

You can control which users have access to which database tables when using an ODBC connection. By default, users do not have access to any tables. You can grant users or user groups access to tables using User Function Access and Group Function Access. Access to each table depends on a function name of the form %Dtable-name. For example, if you grant a user access to the function %Djc_job, they will have access to the table jc_job.

These permissions also control whether the user has access to queries in Report Writer, and to the Database .
