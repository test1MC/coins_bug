Financials

House Sales Financials covers three general areas of processing:

  • Recording cash transactions associated with the sale or cancellation of plots and part-exchanges.  These transactions are linked to the financial ledgers.
  • Processing journals which arise when the sale of a plot or part exchange is completed.
  • Processing ground rent and service charges.

Financials - configuration

Use Batch Configuration (in) to set up House Sales batches.

Transaction Types allows you to set up and maintain the various types of financial transactions that will be handled by the House Sales "module".

Transaction Categories allows you to set up the codes that identify different categories of transaction within House Sales Financials. These specify which General Ledger control accounts the transactions are posted to, and other settings.

Each category can apply either to plots or part exchange properties or both.
