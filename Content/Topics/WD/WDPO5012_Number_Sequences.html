Number Sequences

You can change the next sequence number for any of the purchase order numbers, subcontract order numbers, requisition numbers or subcontract instruction numbers. You would normally only need to use this if you are moving from another system and want to use the same sequence number formats.

To change sequence numbers:

  1. Go to the relevant "PO Sequences" function.

    NOTE

    The function parameters to the procedure determine which group of sequence numbers you can maintain. 

    This is a standard line-based maintenance procedure. Each line shows a different prefix and the next sequence number for that prefix.


  2. To change a sequence number, open the line and enter the new number.

NOTE

The Next Number is the next number that will put into a temporary storage area. This acts as a 'pool' of available sequence numbers. uses this pool to ensure that sequence numbers are not lost if, for example, a user creates an order and then deletes it. tops up this pool when the number of available sequence numbers drops below a pre-defined level. Thus the next sequence number may not be the same as the number shown here, because it is taken from the pool.
