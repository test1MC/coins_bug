Bank Reconciliation

Before you run Bank Reconciliation, you must enter a statement, either manually (see Entering Statements Manually) or automatically (see Loading Statements Automatically). You can report on unreconciled statement items or transactions using the Bank Statement Report or Unreconciled Report.

There are two ways to run bank reconciliation:

  • Manually using Bank Reconciliation: For each transaction on your bank statement, you select the corresponding transaction and include it in the batch. See Manual Reconciliation.
  • Automatically using Auto Bank Reconciliation: runs a process that matches on your statement with transactions with the same number and amount, or which have a corresponding statement transaction type that is configured to reconcile automatically. Any other transactions remain unreconciled; you will need to reconcile them manually.

You can combine both methods; for example, reconcile automatically, then match any un-reconciled items manually.

Configuration

Use the Statement Transaction Types tab in Bank Accounts to set up the transaction types used on statements for each bank account.  You can mark specific transaction types to be reconciled automatically (in which case during auto reconciliation, creates and General Ledger journals that correspond to transactions of this type).
