Company Search

Subcontractor Search and Search allow you to search for subcontractors and according to a wide range of search criteria.  For example you can search for companies:

  • that provide specific goods or services
  • that work in a particular area
  • that have achieved performance rating scores above a given level
  • that you have given a particular status code to

The search is available on the menus, but also within the Projects (when sourcing companies for a package - see Searching the Database) and within the SRM Workbench.
