End-Of-Year Submissions - Payroll Parameters

The Payroll parameters listed below are used with online end-of-year submissions.

Parameter Purpose Your notes (likely settings)*
EGOVCOS A can do list of numbers that you wish to do online returns for. This parameter can be used to exclude if you have copied  - as the tax reference would be the same.  
EGOVRDIR The root directory for storing XML files. Note all users who will send the Internet returns will require permissions to read and write from this directory.  
JAVAHOME The UNIX directory for java has been installed in. /opt/java1.4
NIPREFIX Additional valid NI prefixes. This is to be used if part of the IR issues invalid NI numbers, which they later decide to make temporarily valid. Typically encountered with from outside the UK.  
SENDER Either 'Employer' or 'Agent'. This will be Employer unless you have registered as an agent to send multiple scheme submissions. Employer
TAXDELAY The day and month to apply the new tax bands and rates. Normally 19/5. See the section on Annual Tax & NI Rates for further information. ENSURE THAT THE PREVIOUS YEAR DATE IS TAKEN OUT OF THIS FIELD BEFORE RUNNING YOUR FIRST PAYROLL OF THE YEAR. DON'T FORGET TO ENTER THE DATE WHEN THE NEW TAX CODES WILL TAKE EFFECT.  

*Use this column to note down the settings which you will be using - based on your payroll configuration.

All the parameters listed below are used to nominate a pay item that holds a particular value. If you have used different items for different pay frequencies then you can enter a comma separated list of item numbers in the order weekly, monthly, fortnightly and four-weekly. Note if the SPP-ITEM is not configured then will assume SMP paid to a man is SPP.

If SPP-ITEM is not configured then will treat values in the SMP-REC item as SPP recovered when the employee is a man.

You can leave all of the Recovery parameters blank and manually input the figures if you wish.

Parameter Purpose Your notes (likely settings)*
SAP-ITEM Statutory Adoption Pay  
SMP-ITEM Statutory Maternity Pay  
SPP-ITEM Statutory Paternity Pay  
SAP-REC SAP Recovered  
SMP-REC SMP Recovered  
SPP-REC SPP Recovered  

We have found that adjustments for the new P60/P45 laser forms has proven tricky and dependent on the printer used. The following parameters are designed to let you make the P60 fit on the page. They allow the print to be shifted sideways or up and down by a number of characters to cater for different margin settings.

Examples
3|mylaser=4 ? means the default is 3 characters (or lines) and for 'mylaser' it is 4 characters (or lines)
