What Happens If I Need to Change the Payments in a Batch?

If for any reason you need to make changes to a frozen payment batch (for example, to add or delete last minute payments, or if you print a cheque by mistake or do not want to issue a cheque you have printed), you can still do so. These changes must be made before the payment batch is posted.

To add or delete payments after freezing the batch:

  1. Go to Payment Batches. 

    displays a list of payment batches.

    NOTE

    If your system is set to run multi-currencies and/or sector based ledgers, then these will determine whether users are able to create multiple (unposted) payment batches. allows for only one payment batch, per currency, per sector. 
  2. From the list, select the payment batch or batches that you want to unfreeze.
  3. Choose Unfreeze Payment Batches from the list.

    4. You have now unfrozen (re-opened) the payment batch. 

  4. Now you can delete individual payments from the batch, or make the necessary changes to the batch using the selection procedures (see Selecting Invoices for Payment).

    NOTE

    If you have already posted the payments you will need to use the Input - Cancel Payment procedure (see What Happens If I Have Posted an Incorrect Cheque?).
  5. You need to freeze the batch again before you can process the payments.
