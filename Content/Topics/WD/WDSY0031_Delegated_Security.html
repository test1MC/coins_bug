Delegated Security

You can delegate to other users (for example, regional or departmental administrators) the setting up of users, and the granting of security and access permissions. 

Those users will then be able to maintain the user records and function access security for any of the users they 'own', but will only be able to grant access if they themselves already have access.

If you are using delegated security there is an additional filter on the Users browse to allow you to filter by the Maintained By field. This will allow you to identify a block of users that are managed by an individual.

Delegated security - configuration

If you are using delegated security, enter the user ID of the owner against each user they are responsible for, using the Maintained By field on each user record
