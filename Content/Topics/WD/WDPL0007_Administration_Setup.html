PCard Administration Setup

Each PCard belongs to a single scheme.  Typically, a contractor will have only one PCard scheme in place but it is perfectly possible to have more than one scheme.  

  1. Use PCard Schemes to set up the scheme(s).
  2. When you have set up a scheme, click to set up the control accounts for each that is going to be affected by PCard transactions belonging to the scheme.
