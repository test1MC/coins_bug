Updating VAP Budgets

VAP Standard House Type Budget

You can update the Standard House Type Budget in VAP with figures calculated from the Standard House Type .  This also updates the VAP  Budget at Plot level from the , ensuring that the updates are applied consistently.

You can also update the budget for selected payment stages, from the Payment Stages summary.

VAP House Type Budget

You can update the House Type Budget in VAP with figures calculated from the House Type .  (When the VAP budget is updated from the module then a copy of the relevant records will be created with this field populated with the VAP budget that was updated.  This allows a snapshot of the to be taken to support the updated VAP budget values.)

To update the VAP standard house type budget:

  1. Go to Standard House Type Workbench.
  2. Choose the house type whose budget you want to update.
  3. Either:
    • On the Option menu, select Update Standard House Type VAP Budget and specify the you want to update. (This runs a background report process)
    • From the Payment Stage Matrix, select the payment stages you want to update and choose the Update VAP Budget action.
  4. From the Budget Type list, select the budget you want to update.

To update the VAP house type budget:

  1. Go to House Type Workbench.
  2. Choose the whose budget you want to update.
  3. Either:
    • On the Option menu, select Update VAP Budget and specify the and house types you want to update. (This runs a background report process)
    • From the Payment Stage Matrix, select the payment stages and house types you want to update and choose the Update VAP Budget action.
  4. From the Budget Type list, select the budget you want to update.
