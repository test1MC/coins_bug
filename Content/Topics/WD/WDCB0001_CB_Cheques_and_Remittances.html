Configurable and Remittances

lets you configure the format for printing and remittances.  You can print and remittances separately, or as a combined document.  You can also configure a document to print multiple on one sheet.

Define the layout of and remittances using Document Designer. See Configurable Documents for general information on how to configure documents for printing.

To define a document for and/or remittances:

  1. Use the document type "/Remittances (CB)".
  2. In the Layout Type field, select:
    • to define the layout for ;
    • Combined to define the layout for combined and remittances;
    • Remittance to define the layout for remittances.
  3. To configure a combined /remittance document, configure the body form to print the remittance lines. The can be either at the top or at the bottom of the page. If the remittance lines take up more than one page, you will need to print a "VOID" form on the portion of the stationery on all pages after the first one. Configure a form for this and enter its name in the VOID Form field, then, depending on whether the is at the top or bottom of the stationery, do one of the following:
    • If the is at the top: in the Location field, select Top of Page. In the Continuation Header Form field, enter the name of the VOID form. Configure the Header Form to print the details.
    • If the is at the bottom: in the Location field, select Bottom of Page. Configure a form to print the details, and enter that form name in both the Footer Form and Continuation Footer Form fields. then replaces the footer with the VOID form after the first page.
  4. To print one per page, leave the Multiple per Page?box blank, and configure the header or footer form to print the details. To print multiple per page, select Multiple per Page?, and configure the body form to print the details.

    NOTE

    Each document needs to have a BODY form (even if it doesn't print anything).
