Format of GRN Numbers

gives each delivery (GRN) a unique reference number. These GRN numbers are always generated automatically by , but you can control the format of the numbers.

The format of GRN numbers is determined by the PO parameters GRNDELIM and GRNFORM. The number is made up of the code, followed by the delimiter specified in GRNDELIM, followed by a sequence number as determined by GRNFORM. For overhead orders, the number is made up of the delimiter and the sequence number only.

For example, if GRNDELIM is / and GRNFORM is GR9999, the first GRN number for A100 will be A100/GR0001. The twentieth GRN for an overhead order will be /GR0020.

Note that users may be more likely to use the 's delivery ticket number to identify the delivery.
