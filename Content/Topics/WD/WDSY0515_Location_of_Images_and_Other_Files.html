Location of Images and Other Files

Standard files are deployed by COINSAdmin into directories under $BASE/app/xcode/db on site.  By default will look for files in these directories.

Standard installation

In a standard installation, for each environment, the Unix variable $BASE points to the environment's installation directory.  For each environment, the following directories exist:

$BASE/app/xcode/db/images - holds the screen images and the logos.

$BASE/app/xcode/db/style - holds the style sheets.

$BASE/app/xcode/db/scripts - holds the scripts.

$BASE/app/xcode/db/docimage - holds images for the online documentation.

You should never place anything that is user-defined into these directories. It is likely that COINSAdmin will override and/or delete these files.  If you want to add your own images, styles or scripts, or modify our standard ones, you need to create user-defined directories and point COINS to those.

Base document directory

There will also be a webserver base document directory.  The location of this directory can vary from server to server. On an HP machine it would be /opt/hpws/apache/htdocs. (If you cannot find it on a particular server, do a Unix find for index.html.) Below this will be a coins directory, and below that, directories corresponding to each environment.  Below each environment directory will be directories for images, styles and scripts that are linked to the directories below $BASE/app.

Parameters will be set as follows:

  • SY/WEBBASE would be blank.
  • SY/WEBIMGS would be $WEBBASE
  • SY/WEBSTYLE would be $WEBBASE
  • SY/WEBLOGO would be blank.
  • SY/WEBROOT specifies the server name.
