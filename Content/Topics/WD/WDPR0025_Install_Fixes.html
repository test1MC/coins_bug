Install Fixes

Before using to submit P45 and P46 in-year returns to HMRC from 6th April 2009 onwards, you will need to ensure that the appropriate update for your release of has been loaded and deployed.

Releases from v10.18 and later include the updates necessary for in-year returns.

For releases 10.15, 10.16 and 10.17 the update was included in the 0901 patch ? if this patch has not been deployed and you are doing in-year filing for the first time, you need to install the appropriate fix from the list below:

ReleaseFix
V10.17 47010
V10.16 47009
V10.15 47008

As with all new software fixes, we recommend that you install and test this in a non-live environment first.
