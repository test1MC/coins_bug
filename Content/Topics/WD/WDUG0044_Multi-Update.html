Multi-Update

Some functions have a multi-update button. This allows you to choose several records and update certain fields on those records with the same value, all at once. Not all fields are necessarily available on a multi-update screen; it depends on how the screen was designed.

Configuring a multi-update form

To add multi-update to a browse, you need to configure a multi-update form for the page. The multi-update form specifies which fields are included in the multi-update.
