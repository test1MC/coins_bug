Earlier Year Update

If any amendments have been made to the year-to-date values submitted for a prior tax year, these amendments can be sent in a further final FPS return for the previous year up until 19th April of the following tax year. From 20th April onwards however any amendments must be sent using an Earlier Year Update (EYU) return instead.

To submit an earlier year update:

NOTE
