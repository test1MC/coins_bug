Retire Old Codes

You may wish to make the old rates inactive, to prevent them being used by mistake. Note that this should only be done once all invoices dated on or prior to the rate change have been processed.

To retire codes:

  1. Go to Codes.
  2. Click against the code you want to retire.
  3. Click the Inactive box so that it shows a .
  4. Click to save the new code.

At this point, if you use for and/or Subcontract Ledger transaction input, you may also wish to remove the old codes from the CS and SC parameters WEBVAT (see Check Parameter Settings for details).
