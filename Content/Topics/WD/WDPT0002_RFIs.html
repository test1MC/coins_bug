Configurable Documents

You can configure documents to print details of (Requests for Information) recorded in Project Management. can be used to track questions, and responses to those questions, as they relate to a project.

Define the layout of the document using Document Designer.

documents are made up of the following sections:

Header Information about the . 
Body One record for each journal entry.
Copies To Header Column headings.
Copies To One record for each recipient.
Question Header Heading for the question.
Question Body Details of the question.
Answer Header Heading for the answer.
Answer Body Details of the answer.
Footer Further information about the , space for signatures, etc.

To configure documents:

  1. Go to Document Designer.
  2. Use the document type Print . 
  3. On the Extras tab, enter names for the copies to, question and answer sections (as shown in the table above) that you want to use.
  4. See Configurable Documents for general information on how to configure documents for printing.
