Additional Information

Using terminology tokens

If you edit copies of existing topics, you may see terms contained within curly brackets "\". These are terminology tokens, which allow terms used in one language (such as British English) to be substituted with terms in another language (such as American English). You do not need to be concerned with these tokens and may leave them as is. However, if you would like a further explanation of what terminology tokens are and how they are used, please refer to Customing Using Terminology on the OA Online Documentation Menu.

Adding images

You may need to insert standard images into your documentation. For this to work correctly, these images must be located on a network drive or other location accessible to anyone using your system. You can then add these images by:

  • Using the "I" or "J" tokens; or
  • Using the Image button.

Please consult with your System Administrator or Support Representative for more information about adding images.

Adding web page references

You may find it necessary to refer users to a web page (in other words, URL address) for more information, such as to download software or forms, view previously recorded webinars, etc. This can be done simply by typing the URL address (or copying and pasting it from the source) directly into the text. The example below shows the result of doing this.

EXAMPLE:

For more information about , go to http://www.coins-global.com.

Removing/deleting menu books and topics

If you need to remove/delete a menu book or topic that you have created for online documentation, follow the steps outlined under the "To remove a user-defined topic" section in the OA Online Documentation Menu under Modifying Existing Topics.
