Creating a New Menu

A menu is a type of function.  To create a new menu, you need to create a new function, then add menu items to it.

To create a new menu:

  1. Go to Function Maintenance.
  2. Add a new function. 
  3. In the Type field, select Menu.
  4. In the Category field, select Menu.
  5. Save the function record.

To add functions to a menu:

  1. On the Function Maintenance browse screen, click the Menu link for the new function (in the Type column).

    displays the Menu Item Maintenance screen.

  2. Click and enter the code for the first function you want to appear on the menu.

    NOTE

    The function's context description will be shown on the menu.  If the standard context description from the function is not appropriate on the menu you are creating, you can override it using the Context field.  For example, the context description for Payments is "Payments"; on a Payment menu you might want to change this to read "PL Payments".
