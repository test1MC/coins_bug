Electronic Submission

Under this method, you can print a draft monthly return for checking as with the manual submission method.  You send the final monthly return electronically from to HMRC.  also receives and records the acceptance or rejection of the return by HMRC.  This method relies on successful electronic communication between your server and HMRC.  In case of communication problems, you can use the manual submission method as a fallback.

NOTE

The person who submits the return must make certain statutory declarations (see Statutory Declarations).

To submit a month-end return:

  1. If you have more than one in a scheme, log in to the submission for that scheme.
  2. Go to Monthly Return Submissions.

    The submission line has a status of "Incomplete".
  3. Click to open the submission record.

  4. Fill in the statutory declarations (see Statutory Declarations).
  5. If the password field is blank you need to enter your Government Gateway password for CIS submissions. fills in the password if it has been entered in the SCUK/PASSWORD parameter.
  6. Click .
    The submission line now has a status of "Committed".
  7. Select the submission line.
  8. Use the Choose Action selector to select Submit, then click
