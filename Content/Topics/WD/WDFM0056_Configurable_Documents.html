Proforma (or )

Proforma (FM) (or ) (document type FMPROF). The Header is the Invoice Header table and the BODY is the Invoice Line.

Proforma is used when using simple measured invoicing, and when using /certification at item level. See FM Configurable (FMPROF) Documents.

Invoice

Invoice (FM) (document type FMINV) is used to print FM invoice documents if Proforma (FM) is used for .

On the Invoice documents there are many extras; generally for subtotalling/grouping by and by Invoice Class. See FM Invoice Configurable Documents.

Daily Tasks

Daily Tasks (document type FMTASKDY) is used to print a document for a , showing all their tasks per day. It has the (sw_personnel) record in the header and then FM tasks (sw_task) in the BODY.

This has an extras body form for printing information about hazards defined on the location.

Letter

Print Letter - FM Stage (document type FMLETTE): Originally intended as a letter print, this can be used for many things. It can print the and location details in the Header. Details of schedule of rates items can be printed in the BODY.

Quote

Quote (document type FMQUOT) is a configurable print of a quote within FM. In the header you can print information from the , location and the quote header (fm_qheader). In the body you can print details of the quote line (fm_qline) and the associated schedule of rates item (sw_jobsor).

Tasks

Task Print (FM) is a configurable print of an FM task.  You can also configure a more complex document that prints extended text for hazards, task description, task instructions, and task notes. See FM Task Print.
