Monthly Status Report

The Monthly Status Report allows you to check that you are ready to run the month-end process, and to check at the end of the process that you have completed the process satisfactorily.

Before running the month-end process, the main reason for running the Monthly Status Report is to show if there is any unallocated cash or unposted batches.  After running the month-end process, the main reason is to check the submission status and show any negative submissions.

You can run the report for all , groups of , or the current only.

For each , the report shows:

  • Whether the month-end process is "Not Started", "Incomplete" or "Complete".
  • Whether the submissions are Complete (HMRC has accepted the submission), or how many are Incomplete or Failed.
  • Whether all monthly statements of deductions have been printed ("Complete") or how many have not been printed.
  • Whether there are any negative returns, and how many you have not flagged as notified.  If so, you will need to notify HRMC separately.  See Negative Returns.
  • Whether there are any unallocated subcontractor payments, and how many.  If so you will be unable to run the month-end process until the cash has been allocated.
  • Whether there are any unposted payment batches, and how many.  If so you will be unable to run the month-end process until the batches have been posted.

The detail section lists any negative values, unallocated cash and unposted payment batches.
