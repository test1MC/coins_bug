Options and Actions

For each type of item (such as , , ), there are a number of things you can do (such as send an email or print a document).  These vary for each type of item.  They are available as a list of Actions on the relevant browse screen (where you can select one or more items you want to run the action for).  The same list is also available as an Options menu when you are looking at the detail of an individual item; this means you do not have to go back to the browse screen to run the action.

Sending Emails

Each type of item has a standard email template, which is used as the default text when you send an email.  The email template can contain tokens for database fields, such as the name or code of the item, which then fills in with the relevant data. You can edit the email text before you send it. 

If there are any names on the Copies To list for the item, these are available as boxes on the email; you can choose who to send to as well as enter other email addresses manually.

If there are any documents associated with the item, boxes allow you to choose which documents to attach to the email.

You can also send emails containing documents that relate to the project as a whole, using the Options menu from the main Communication Management screen.

Letters

Some types of items allow you to generate letters.  These are based on standard templates and are opened as RTF documents in Word.  The letter templates can contain tokens for database fields, such as the addressee's name and address; then replaces these tokens with the relevant data when the letter is created (like a mail merge letter).

Letters you generate are available on the "Linked" tab of the item, and also on the "Communications > Letters" tab.

Letters can have a reference code which generates automatically based on the format you specify.
