Selecting What and How to Print

When you run a report, or choose to generate a printable document, displays one or more selection tabs, similar to the one below, which you use to select what you want to print:

Selection Example

For example, you may simply need to specify the dates you want to report on, or you may need to specify which information to include, how much detail you want, and how you want the information arranged on the report. These report options are the selection criteria for the report.

At the end of a report, may (depending on a parameter setting or the report setting) print the selection criteria you used to define the report.

Selecting what and how to print - configuration

You can suppress the report criteria, record type and sort information from the spreadsheet output if required: set noInfo=Y on the report function parameters to suppress these additional data columns from the output.
