Adding an Issue to a Call

You can add an issue to an existing call. For example, if you become aware of a separate issue that you want to log with the call.

To add a new issue to a call:

  1. Open the call Detail page.
  2. Click the Issues tab.
  3. Using the Add Type drop-down at the bottom of the page, choose whether to use Quick Add or Standard Add, then click .

    (Quick Add creates a task for the issue at the same time.)

  4. Enter details of the issue. You can insert standard issue descriptions by clicking next to the Full Description field and choosing the appropriate text. This also selects the appropriate category.
  5. If you are using Quick Add:
    1. Enter details of the task and who it is assigned to. You can insert standard task descriptions by clicking next to the Task Instructions field and choosing the appropriate text.

    asks if you want to log another issue.

  6. If you want to log another issue as part of the same call, click OK; otherwise click Cancel.
