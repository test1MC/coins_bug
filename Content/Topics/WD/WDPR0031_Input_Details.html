Details Which Need To Be Directly Input to Forms

Details of the previous employment for s on form P45(3) will always need to be input on the form as these are not recorded elsewhere on Payroll (except the previous cumulative pay and tax figures).

Details of which statement has been ed by the employee on form P46 (A, B, C or none) will always need to be input on the form.

For s, all details on form P45(1) can be completed from normal payroll processing; it should not be necessary to input any details directly on the form.
