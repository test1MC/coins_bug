Controlling Access to Side-Frame Help

side-frame help includes information about the purpose and usage of screens and fields, as well as information relating to parameters and configuration.  By default this information is available to all users of , but not all of it is relevant to certain types of user.  If necessary you can set up so that specified users will not see the parameter and configuration information in the side-frame help.  You can also remove access to the side-frame completely for some users.

Side-Frame Help with and without Configuration details

Access to Help

To deny access to the configuration information in side-frame help:

  • Set the user or group access for the function %WSYD003 to No.

NOTE

The SY parameter HELPPARM, which was used previously to specify groups who did not see the parameter lists in the help, has been removed.

To deny access to the side-frame help:

  • Set the user or group access for the function %WSYD002 to No.
