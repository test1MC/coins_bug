Report Writer - Filtering Report Data

Either when you set up the report initially, or afterwards, you can specify filters to control which information is included in the report.

To filter the report data:

  1. While you are updating the report, click the Filter tab.
  2. Choose the first field you want to evaluate to decide whether to include the record on the report.
  3. Choose how you want to evaluate the field (for example, whether to include records where the field equals something, or doesn't equal something).
  4. Type the value you want to compare the field with.

    For example, to include only in London, select Location from the Filter list, select Equals from the comparison list, and type London in the value field.

    If you choose Matches from the comparison list, you can use a can-do list in the value field. So for example to include only cost codes that end with 99, select Cost Code from the Filter list, select Matches from the comparison list, and type *99 in the Value field.

  5. Repeat the process for other fields you want to filter on. Note that the filters are 'AND' filters; so for example if you specify where Location equals London, and where equals John Smith, the report will only include in London that are managed by John Smith.
