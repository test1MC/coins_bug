Selected/Not Selected Report

Selected/Not Selected Report allows you to produce a report that shows details of outstanding transactions selected or not selected for payment. The report is broken down and totalled by and shows the balance for each transaction, the total amount still payable and the amounts, if any, that have been selected to be paid. The report indicates whether transactions were selected via Batch Selection or via Manual Selection. If there are any unallocated amounts that exist for , these are shown to the right of the report. Details of any transactions on hold that require approval before being paid are also shown.

If the 's account balance would become negative if the selected invoices were paid, will display the message: "Neg Balance" to the right of the report. If there is a negative balance existing on a related S/C account, then displays the message: "Warning: Negative Balance on S/C Account" next to the appropriate details.

An aged analysis of the payment run is shown at the end of the report.

You may run the selected/not selected report at any time during the payment run process.
