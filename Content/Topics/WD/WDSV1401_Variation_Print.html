Documents

Using configurable documents, you can produce variation (for client-related changes) and subcontractor variation documents. These can be simple letters of confirmation or details of the changes requested. Both documents can show a variation header, with variation items in the body.

Define the layout of configurable documents using Document Designer in the System Module. See Configurable Documents for information on how to define configurable documents.

  • Use the document type Variation Print (CM) for client-related variations. Fields from the following tables are available:
    • sv_vari - Client Variations
    • sv_job - Contract
    • jc_job - Contract
    • cs_series - Certificate Series
    • ar_cust - Customer
    • sv_item - Bill Items
    • sv_measure - Bill Measure
    • co_config - Configuration
  • Use the document type SC Variation Print (CM) for subcontractor variations. Fields from the following tables are available:
    • sv_varisc - SC Variations
    • sv_job - Contract
    • jc_job - Contract
    • sc_mast - Subcontractor
    • sc_subcon - Subcontract
    • po_hdr - Order Header
    • sv_item - Bill Items
    • sv_measure - Bill Measure
    • co_config - Configuration
