Search

The search allows you to search for products by typing part of their code or description.

To search the :

  1. Click the button to display the Search screen.
  2. Type any part of the description or code of the product ( group or item) you are looking for. You do not need to enter wildcards. You can enter more than one word, in any order. You can also enter parts of words. 

    NOTE

    Whenever you are on the Search screen, the search looks at the whole .
    displays the search results. The results show any groups, or any individual products, whose code or description contain the word you entered. If you entered more than one word, the search results will only show products or groups that contain all the words, but they could be in either the code or description. 

  3. You can use the Sort By selector to change the order in which the results are displayed.
  4. In the results list:
    • If an entry in the results list is a group, the tooltip shows "Groups"; you can click the link to show the group's contents.
    • If an entry in the results list is a group or item (that is, a product you can request), the tooltip shows "Add to Basket"; you can click the link to add the entry to your basket (see Adding Items to the Basket).
