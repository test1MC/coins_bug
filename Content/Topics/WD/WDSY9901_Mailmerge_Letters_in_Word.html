Word Mail Merge Letters

uses Word mail merge features to generate some types of letter (for example, prospect letters and mail shots in House Sales).

To set up a mail merge letter in Word 2003/2007/2010:

This assumes that you already have a data file in a suitable format (such as a CSV file) generated from .

  1. Create a new document in Word.
  2. Bring up the Mail Merge task pane.
    • In Word 2003, from the Tools menu, select Letters and Mailings then select Mail Merge.
    • In Word 2007 or Word 2010, on the Mailings tab, in the Start Mail Merge group, click Start Mail Merge then select Step by Step Mail Merge Wizard.
  3. In step 1 (Select document type), choose the type of mail merge document you want to create (for example, Letters).
  4. In step 3 (Select Recipients), choose Use an existing list, and browse for your data file.
  5. In step 4 (Write your letter), to insert data fields in the document, click the (Insert Merge Field) button and select the field you want to include.

To set up a mail merge letter in older versions of Word:

  1. Create a new document in Word.
  2. Select the Mailmerge option to bring up the Mail Merge Helper.
  3. Click Create and choose the type of mail merge document you want to create.
  4. Click Get Data, select Open Data Source, and browse for your data file.
  5. To insert data fields in the document, click the Insert Merge Field button and select the field you want to include.
