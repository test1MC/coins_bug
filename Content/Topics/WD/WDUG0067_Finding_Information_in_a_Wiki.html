Finding Information in a Wiki

A Wiki site is divided in to a number of webs; each one representing an area of collaboration.  Each web is made up of a collection of hyperlinked pages (or 'topics').

  • To browse a Wiki web, just click one of the highlighted links.
  • You can navigate to different webs using the combo in the upper right corner of each topic.  Individual topics are identified using topic names called WikiWords: these usually consist of two or more words with initial capital letters and no spaces. If you know the WikiWord for a topic you can jump straight to it using the jump to field in the upper right corner of each topic. The Web home page of each Web is the WebHome topic. 
  • The "breadcrumbs" at the top of each topic show you the topics above the one you're looking at; you can click any of the links to go to the relevant topic.

  • You can also search for topics using the search page.  Click the search button at the bottom of the page.

To search a Wiki:

  1. Click the button.
  2. In the search page, the boxes for the Wiki Webs you want to look in.  For example, if you are only interested in information from a "Projects" Wiki, just the box next to Projects.
  3. In the Words field, enter the word or words you want to search for.

    will find topics that contain all the words you enter, in any order.  For example, if you search for project future bridge, will find topics that contain the words "bridge" AND "future" AND "project", but not topics that only contain "future" and "project" but not "bridge".

    Also, will only find the whole word, so if you search for bridge, will not find topics that only contain words like "bridges" and "bridgebuilding", but not the word "bridge".  However, the case of the word is not important: would find topics containing "bridge", "Bridge" or "BRIDGE".

  4. the Live box if you only want to search the most recent version of each topic.  If you want to include previous versions of topics in the search, clear the Live box.
  5. If you want to search for topics according to when they were updated, enter a range of dates.  For example, you could leave the Words field blank, and enter a date to find any topics that have been modified since that date. 
  6. Click Search.
