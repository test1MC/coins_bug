External Topics

You can include external documents in the Documentation menu tree.  External topics could be Word or PDF documents, HTML pages, or even websites.

To create an external topic:

  1. Go to Topic Maintenance.
  2. Using the tree view, browse to the book that you want to add the new topic to (we'll call this the parent book).
  3. Using the Add Type drop-down, select New external topic and either:
    • Click to add the topic at the bottom of the list.
    • an existing item and click to add the topic above the selected item.
  4. Enter a code for the topic.  This can be any code you choose, but do not begin it with % or +.
  5. The module defaults from the module of the parent book; you can select a different one if you wish.  The module shows on the Search results.
  6. Enter a Menu Description for the topic; this shows on the menu. 
  7. In the Function Parameters field, enter the URL of the document, file or web page.
  8. Select a Function Category; we recommend you select a Function Category that identifies the type of topic.  For example: PDF, WORD.
