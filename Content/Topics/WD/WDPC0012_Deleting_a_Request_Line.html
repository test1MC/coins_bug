Deleting a Request Line

You can delete individual lines from a request.  You might do this, for example, if you have fulfilled most of the request but there is a line that you are unable to fulfil.

To delete a request line:

  1. On the Pending screen, select the line or lines you want to delete.
  2. From the Choose Action list, select Delete Pending and click .
  3. Enter the reason for cancelling the request line(s).
  4. Click .

The Deleted Pending screen shows details of any deleted request lines.
