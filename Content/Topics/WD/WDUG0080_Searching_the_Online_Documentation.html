Searching the Online Documentation

You can search all of the Online Documentation for topics that contain a particular word or words.

To search the documentation:

  1. In the Search field (above the title area of the documentation window), type the word or words you want to look for.
  2. To search only the topic titles, Search in Titles Only. This often returns fewer results, which may be more directly relevant. If Search in Titles Only is uned, the search looks in the titles and body of the topic.
  3. Click the button.

    NOTE

    1. The search finds any topics that contain all the words you enter. For example, if you search for contract security, lists all documentation topics that contain both "contract" and "security", in any order. It does not find topics that contain only one of these words.
    2. The search also finds words with different endings. For example, if you search for contract, also lists topics that contain the word "contracts" or "contracting". (However, it does not find words that have the search text at the end or in the middle1. For example, searching for contract does not find "subcontract" or "subcontractor".)

  4. displays a list of the matching topics, showing the first few lines of text.

     

     

    NOTE

    By default, the topics are sorted in alphabetical order. If you want to group the topics by module (this may make it easier to find the topic you want), Sort by Module before you click .
  5. To display a topic, click the link in the search results.
  6. The search text remains in the Search field, so if the topic shown is not the one you want, click to display the results again.
  1. Actually the search will find strings in the middle of words if you enter more than one search term and at least one of those search terms is found at the beginning of a word. So although a search for contract will not find subcontract, a search for contract certificate will find topics that contain subcontract and certificate.
