Amending Menus Using the '+' Technique

One method to amend a menu is to create a copy of the standard menu, which you can then tailor to suit your requirements (for example, by adding reports or giving different names to functions). The menu is named the same as the standard menu, except that it has a + at the start rather than a % (for example +WSC8 instead of %WSC8).

To create a '+' menu:

  1. Go to Function Maintenance.
  2. the menu you wish to change.
  3. Click . 
  4. Replace the % in the function code with a +. 
  5. Click .

Note that having created a menu with an apparently different name you do not need to change the calling menu. Although the calling menu may say %WSC8, recognes that a tailored version of this menu exists and displays +WSC8 instead.

Having created the new menu, changes can be applied using Function Maintenance.

Acceptable changes would include the addition of new functions or menus or the deletion of functions or menus. However functions should not be changed - see next section.

One of the main advantages of the '+' technique is that the Function Access is the same as for the standard '%' menu.
