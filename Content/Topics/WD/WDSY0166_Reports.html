Multi-Currency Reports

Some standard reports provide options for which currency they will report in, while others do not. Where there is no provision for foreign currency, the entries are always in the base currency of the .

Posting reports print in the base currency of the , and then repeat the report in any alternative base currencies in use. If one of these is also the base currency for a that section would be a duplicate, therefore in this particular situation the alternative base section of any posting report is suppressed.  There is an option on posting reports to suppress the alternative base currency sections and report only in the base currency of the .

The following reports allow you to report on foreign currency.  Unless indicated otherwise, they have the option to report in base currency or alternative base currency; some also give you the option to report in document currency.
