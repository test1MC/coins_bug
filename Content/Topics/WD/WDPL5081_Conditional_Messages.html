Conditional Messages

allows you to configure a set of conditional messages relevant to of subcontract . You can print these messages in the header and/or footer of and/or remittances if the relevant condition is met. For example, you can print a warning if a subcontractor's insurance is due to expire or has expired.

To use conditional messages:

  1. Enter the field names RO_sbm_letterCondMsg__1 to RO_sbm_letterCondMsg__10. If you use several different RO_sbm_letterCondMsg fields on different lines, the message will print on the line for which it is defined.
  2. To print the conditional messages on consecutive lines, enter the condition numbers of all the messages you want to print on each line. For example, if you wanted to print any of messages 1,2,5 and 6, you would enter four lines, each with the field RO_sbm_letterCondMsg^1256. Then if only conditions 1 and 6 apply, automatically selects those messages and prints them on the first two lines.

For details of these and other conditional messages used in , refer to the Subcontractors Ledger Guide.
