Actions

Actions allow you to set up reminders of tasks or appointments that relate to the workbench you are using. You can set a reminder for yourself, or for another House Sales user. For example, in the Workbench you can set up an action to remind yourself to phone a , or to remind a colleague to send a brochure to a .

You can also create follow-up actions when you run an event (see Events).

The action will appear on the Actions tab for the appropriate record in the workbench, and also on your Activity Workbench (or the Activity Workbench for the user you assigned the action to).

To set up an action:

  1. From the Summary page for the record, click the Actions tab.
  2. Click to create a new action.
  3. Either:
    • If the action is an appointment, click Appointment and enter the date and time of the appointment.
    • If the action is a task, click Task, and enter the date by which the task is to be completed.
  4. Select the type of action, and the priority.
  5. By default the User will be yourself; this means the action is for you. If you want to assign the action to another user, enter or select that person's user ID.
  6. If you want the action to be visible only to the person it is assigned to, the Private? box.
  7. Enter text to describe the appointment or task.
  8. Click to save the action.
