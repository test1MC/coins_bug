Purchase Order Numbers

Depending on your order numbering policy, you can control whether and under what circumstances generates order numbers.  This topic explains how to set the parameters that control how generates the numbers.  For information on the format of purchase order numbers, see Format of Order Numbers.

Generating Numbers for Purchase Orders

How do you want to generate purchase order numbers?

  • Generate numbers automatically, but allow users to enter their own purchase order numbers, as an alternative to the automatic sequencing:

    POALLWMN ðY,

    POAUTOGN ðY

  • Generate numbers automatically and not allow users to enter their own numbers:

    POALLWMN ðN,

    POAUTOGN ðY

  • Users must enter purchase order numbers manually:

    POALLWMN ðY,

    POAUTOGN ðN

Generating Numbers for Template Orders

If generates purchase order numbers automatically, how do you want to treat template order numbers?

  • Allow users to enter numbers for template orders manually? (This allows users to enter meaningful codes for templates):

    POAUTOGN ðY,

    POTMPGEN ðN

  • to generate both purchase order and template order numbers:

    POAUTOGN ðY,

    POTMPGEN ðY

Generating Numbers for Change Orders

How do you want to generate change order numbers?

  • Generate numbers automatically, but allow users to enter their own change order numbers, as an alternative to the automatic sequencing:

    CHGALWMN ðY,

    CHGAUTOG ðY

  • Generate change order numbers automatically and not allow users to enter their own numbers:

    CHGALWMN ðN,

    CHGAUTOG ðY

  • Users must enter change order numbers manually:

    CHGALWMN ðY,

    CHGAUTOG ðN
