Controlling Access to Online Documentation

You may decide that the online documentation is not relevant to all users.  You can either prevent individual users or groups of users from accessing specific topics within the online documentation, or prevent them from using the online documentation at all.

Access to Online Documentation

To deny access to specific topics in the online documentation:

  • Set the user or group access for the documentation functions to No.

NOTE

Online documentation functions begin %WD followed by the module abbreviation.  Topics have a function category of HPG and books have a function category of HBK.

To deny access to all of the online documentation:

  • Set the user or group access for the function %WSYD001 to No.

The button will not be available to these users.  If the users have access to the side-frame help but not the online documentation, hyperlinks from the side-frame help to documentation topics are disabled.
