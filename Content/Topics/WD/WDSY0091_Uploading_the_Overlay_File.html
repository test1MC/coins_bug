Uploading the Overlay File to the VSI-Fax Server

Once the TIFF overlay has been created, you need to upload this to the VSI-Fax server as follows:

$ vupload -d "Purchase Order Overlay" overlay order.tif

NOTE

It is crucial that you do this as the VSI-Fax Administrator so the overlay is accessible by all users of VSI-Fax.

You can also use -u (update) or -x (delete) with vupload to overwrite or delete files:

$ vupload -u overlay order.tif
$ vupload -x overlay order.tif

You can check the overlay has been successfully uploaded by running the following command:

$ vupload -l overlay

The VSI-Fax server will return a list of the TIFF files that have been updloaded as overlays.
