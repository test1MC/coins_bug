Format R (Co-Op Bank)

The first 4 lines are header lines which the load program ignores. The remaining lines each contain a single character string with data in fixed positions.

Characters   Field
1-6
7-14 Account number
16-17 Transaction type
36-46 Amount
65-82 Description
If transaction type = 11 this contains the number
101-106 Transaction date (YYMMDD)
