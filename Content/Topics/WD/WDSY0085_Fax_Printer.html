Setting Up a Fax Printer

To ensure that the file produced by to be sent to VSI-Fax is formatted in the same way as if the document was to be produced by a printer, it is necessary to configure a printer record for use solely by the faxing functionality. VSI-Fax will use this printer for all formatting (such as codepages and column settings). This printer record must not be a real printer - it is for the sole use of VSI-Fax (NOTE: printing to it will not produce a fax).

      Printer: FAXHP             Description: Fax Printer
     O/S Name:                        Device: /dev/null
       Banner: N                   Code Page:
        Group:
      Command:
   Initiale: INIT
        Reset: RESET
   Default to: #HPLJ4
     Owner(s): *

With the following printer escape sequences:

Code Description
132 132 Columns
158 158 Columns
198 198 Columns
220 220 Columns
BOLD Bold On
BOLDOFF Bold Off
SLVOFF SLAVE printing Off
SLVON SLAVE printing On

Parameters

The System parameter DEFLASER must be set to the code of the "fax" printer; in this example we are using FAXHP.
