Import Resources File Format

This file format is used by the Import Resources (CSV) option in Projects. The file must be a CSV format file. The first row contains headings and will be ignored. The remaining rows contain resource data with the following columns.

Column
Number 		Description Data Type
1 Major Type (one of: MATERIAL,TRADE,PLANT,LABOUR) Char
2 Bill Ref (only used if Major Type = TRADE) Char
3 Type (C=Commodity/​Trade Code, S=/​Luckins Code) Char
4 Code (that is, Commodity/​Trade/​/​ Code or Item/Luckins Code) Char
5 Description Char
6 Quantity Dec
7 Unit Char
8 Rate (Internal) Dec
9 WBS Code (currently ignored) Char
