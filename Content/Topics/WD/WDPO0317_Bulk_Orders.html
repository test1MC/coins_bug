Bulk Orders

A bulk order is an order for an approximate quantity — such as for readymix — which will typically be delivered in several deliveries over a period of time.

NOTE

A bulk order is one with quantity class E. Order types are defined using Purchase Order Types.

When printed, the order has wording such as Approx printed below the quantity. Unlike normal orders, it is not marked as complete when invoices for the ordered amount have been received; you must complete it manually (using Material Orders).

Entering a bulk order is very similar to entering a material order. The differences are:

  • In the Type field, enter the code for bulk orders (for example, B).
  • You do not need to enter a due date.
