Configurable Documents

You can configure documents to print cover sheets for  and packages recorded in Project Management.

Define the layout of the document using Document Designer.  

  • cover sheets are made up of a standard header, body and footer.  The body section does not repeat, but can be used to show remark notes.  (Remarks are associated with revisions to the ).
  • package cover sheets are made up of a standard header, body and footer, with the body showing one record for each in the package.

To configure cover sheets:

  1. Go to Document Designer.
  2. Use the document type:
    • Print for cover sheets for individual .
    • Print Packages for cover sheets for packages.
  3. See Configurable Documents for general information on how to configure documents for printing.
