Returning a Request to Site

If you are unable to fulfil a request, or if there are problems with the request, you might discuss with the person who made the request and decide to return it to them.  The request will then appear on that person's Previous Requests list with a comment from you, and they can create a new request by copying and amending details from the returned request.

NOTE

You can only return a request if all lines are "Awaiting attention"; that is, if no-one has begun to process the request.

To return a request:

  1. From the Pending screen, select any line on the request.
  2. Choose Return Entire Requisition/Order to Originator from the Choose Action list, and press .
  3. Enter the reason for returning the request.  This will appear on the requester's Previous Requests list, along with your name and the date you returned the request.
