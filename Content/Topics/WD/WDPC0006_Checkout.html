– Checkout

Checkout creates a request that contains the items in your basket.

To create the request:

  1. From your basket, click Checkout.

    displays the Checkout Basket screen.

  2. In the field, enter the code of the for which you are raising the request.  By default, fills in the details from the previous request.  You can click the button to select the using the standard lookup.
  3. In the Charge field, enter a code to indicate where this request will be charged to.  This is currently only memorandum text.
  4. In the Site, Address and fields, enter details of the site to which the products should be delivered.  By default fills in the normal site address from the .
  5. If you have access to the module and want to assign the immediately, the Process on Checkout box.  After the basket has been processed and the request is created, opens the Process Pending screen in a new window.  See Processing Pending .
  6. Click OK.

    creates the request and displays an alert box that shows the request number:

    The request number format is: < code>/999999.

  7. Click OK.

    Depending on how is set up, this request will either be approved and go straight to the , or be put into a list of requests awaiting approval.
