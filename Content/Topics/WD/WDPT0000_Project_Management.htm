Project Management

Project Management provides the tools that construction project managers need to control project-related communications and proactively manage change orders.

 

Control and Administer Contracts

The Project Management Module gathers the tools needed for project management and administration into a unified project level menu.

The Project Management Module enables project managers (PMs) to quickly enter, transmit, and manage Requests for Information (RFIs), meetings minutes, daily reports, and other project communications.

External files, including spreadsheets and scanned documents, can be associated with any project.

 

Manage Project Documents

Project managers can produce, distribute, search for and retrieve project communications of all kinds without leaving Project Management:

• Email.

• Meeting minutes.

• RFIs.

• Drawings and drawing sets.

• Issues that need to be tracked.

• Daily reports.

The Communication Workbench helps PMs find documents quickly and makes it easier to focus on open tasks, with coding for overdue items. Its timeline view gives the PM quick access to communication or task information at a given point in time. Different types of documents often relate to the same project issue, so the Linked tab brings them together - making it easier to manage and research requests for information, replies, design changes, and other project issues.
