Synchroning Activity Workbench and Project Management

If Project Management items are to be synchroned with the Activity Workbench, you need to create Activity Workbench Types.

Activity Workbench Types can be maintained elsewhere in but it is best to use the maintenance option in Project Management, as then the Types will be always created for 0, where Project Management are created.

To set up Activity Workbench types:

  • Go to Activity Workbench Types.

To configure the synchronation between the Activity Workbench and Project Management:

  1. Go to Configuration.
  2. Click the Activity Workbench tab.
  3. For the given Project Management Item types, specify the corresponding Activity Workbench Type that should be created. If the Activity Type field is left blank, no synchronation is done.

  4. In Closed Status, select the status the item should be set to when an item is closed in the Activity Workbench (or closed in Outlook if the Activity Workbench is being synchroned with Microsoft Exchange). If this field is left blank, the status of the Project Management item is not updated when the Activity Workbench item is closed.
  5. On this tab also select the journal type to use for emails imported from Microsoft Exchange.
