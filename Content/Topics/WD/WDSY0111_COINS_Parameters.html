Report Servers - Parameters

There are several Print Manager parameters that need to be configured:

Parameter Typical Value Notes

PM/SPOOLDIR

$BASE/var/spool

Use $BASE to allow for different absolute paths on different physical servers.

PM/TIMEOUT

60

Report agent will automatically shut down if no reports found for this many minutes.

Can be over-ridden by a parameter to the report agent start-up script.

PM/GENPROCS

5

Maximum number of report agents to start.

Ignored if a report agent family name is specified as a parameter to the report agent start-up script.

Can be over-ridden by a start-up parameter to the report agent script.
