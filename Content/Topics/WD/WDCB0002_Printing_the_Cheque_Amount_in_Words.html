Printing the Amount in Words

provides special fields that allow you to print the amount of the in words, either as individual words for each digit, or as a line of text that represents the amount.  The 'line of text' versions use the appropriate wording for the currency (for example, Dollars and Cents or Pounds and Pence).

To print the amount in words:

  • To print a line with the following format:

    ONE ZERO NINE SEVEN FIVE THREE 64

    On documents, add a line containing the field RO_cbd_digitword__n where n is the position (1 = units, 2 = tens, 3 = hundreds, and so on). Repeat this field for each position. Use RO_cdb_pence for the pence/cents amount in figures:

    The following table shows the fields to use on other documents:
    ModuleWords FieldPence/Cents
    RO_acs_digitWord__nRO_acs_pence
    Subcontract LedgerRO_scs_digitWord__n  RO_scs_pence 
    PayrollRO_prk_AmountWord__n  RO_prk_AmountPence 
  • To print the payee and the amount in "full text"; for example:

    Apex Industrial Ltd. ***************
    ONE HUNDRED AND TWENTY THREE S 99 ONLY ****************
    *************************************************************

    On documents, add consecutive lines containing the fields RO_cbd_chqnarr*, RO_cbd_digitLine1 and RO_cbd_digitLine2.

    The following table shows the fields to use on other documents.

    ModulePayee  Text Line  
     RO_acs_jointPayeen* RO_acs_famountWords   
    Subcontract Ledger RO_scs_jointPayeen*  RO_scs_amtWords__n 
    Payroll   RO_prk_AmountWords__n 

Configuration

Use Currencies to specify the currency name (such as Dollars or Pounds) and the currency sub unit (such as Cents or Pence) to print on .
