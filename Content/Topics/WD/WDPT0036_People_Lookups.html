People Lookups

Throughout the Communication Management workbench you need to record who items are received from, who they are sent to, or who is responsible. People involved with a project include subcontractors, and client staff (all of whom are external contacts) as well as your own employees (personnel). 

To make it easier to look up and enter contacts and personnel, you can use any of the tabs that may appear when you do a people lookup:

  • Project Team ? This tab lets you set up a project team that combines selected personnel and contacts into a single list. In fields that require you to enter a person, you can use the lookup to select people from the project team.
  • Personnel ? This tab allows you to select personnel (which are set up in Personnel).
  • Contacts ? This tab allows you to select contacts (which are set up in the Contacts Workbench).
  • Distribution Lists ? This tab lets you select distribution lists. See Copies To Tab.
  • New Contact ? This tab lets you add a new contact to send copies to; you can use this tab to:
    • Select an existing contact. To speed up entry, instead of using the lookup you can type the first few letters of the person's and press Tab or Return; searches the project team, then the personnel list, then the contacts list, and fills in the first name it finds that matches what you enter.
    • Create a new contact.
