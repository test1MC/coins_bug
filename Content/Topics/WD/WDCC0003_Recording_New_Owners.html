Recording New Owners

When a plot is resold, you need to record details of the new owners of the plot.

Using an event to record a new owner will remove the current owner contact record or records. Depending on how is configured, it may also cause the warranty to expire.

To record new owners for a plot:

  1. In Workbench, open the plot summary.
  2. From the Options menu, select the event that corresponds to RESALE (for example, it may be called New Owner or Plot Resold).

  1.   displays the event tab.

  1. Fill in the name(s) of the new owner(s), and click to either enter the details of a new purchaser, or select an existing purchaser.
  2. If the Trigger Warranty box is available for update, the box if reselling the plot causes the warranty to expire.
  3. Click .

The Contacts tab shows the new owner(s).

Configuration

You need to set up a General Event based on the RESALE system event code.
