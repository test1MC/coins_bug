Configurable

You can configure documents to print Project Management showing details of weather conditions, which trades worked on site, and other customable information.

Define the layout of the document using Document Designer.  

include a number of customable fields; these can be included on the printout.  documents are made up of the following sections:

Header Information about the . 
Body One record for each entry.
Custom Header 1 Heading for the first section of custom information.
Custom Body 1  Custom information.
Custom Header 2 Heading for the second section of custom information.
Custom Body 2 Custom information.
Footer Further information about the , page number, etc.

To configure a document:

  1. Go to Document Designer.
  2. Use the document type Print . 
  3. On the Extras tab, enter names for each of the custom sections (as shown in the table above) that you want to use.
  4. See Configurable Documents for general information on how to configure documents for printing.
