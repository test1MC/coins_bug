Isle of Man C20

The SCC20 produces a C20A report (that is, a substitute report if that is agreed by the specific contractor with the Isle of Man authorities). The C20A report provides details of active subcontractors the start of a tax year (6th April).

To configure an SCC20 document:

  1. Go to Document Designer in the System Module. 
  2. Use the SC Annual C20 (IoM) document type.
  3. See Configurable Documents for information on how to define configurable documents.
  4. The following tables are available: sc_mast, co_analtype, sc_return, co_config
  5. The SCC20 has the following forms:
    • The BODY form shows data from sc_mast records (for those subcontractors who have been paid in the specified tax year) including RO_BuiltAddress1, RO_TaxRefNum, RO_TaxCertno, RO_UsesOtherLabour, RO_BuiltAddress2.
    • The HEAD and CHEAD have column heading text with the HEAD having RO_TaxYears.
    • The Confirmation Form (by default CONF) is used to provide information from the client's company using co_config and has space for a signature.
    • THE CFOOT provides the name of the form, typically Substitute C20, and a page count using the standard tokens.
