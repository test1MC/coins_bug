Examples of Schedules

Report Schedules allow you to run reports (and other processes) at regular intervals. There are three basic types of schedule: daily, weekly and monthly. These give you a lot of flexibility in setting up schedules that suit your needs.

Use Standard Schedules to set up the schedules.

Below are some examples.

When to run the report How to do it
Every 10 days.
  1. Use a Daily Schedule.
  2. Set the Every field to 10.
Every Monday and every Thursday.
  1. Use a Weekly Schedule.
  2. Set the Every field to 1.
  3. Monday and Thursday.
Every four weeks on a Friday.
  1. Use a Weekly Schedule.
  2. Set the Every field to 4.
  3. Friday.
Every three months on the 15th of the month
  1. Use a Monthly Schedule.
  2. Set Occurs to Day 15.
  3. the relevant months (for example, March, June, September and December).
The last day of every month.
  1. Use a Monthly Schedule.
  2. Set Occurs to Day 31.

    (31 represents the last day of the month, regardless of the actual number of days in the month.)

  3. all the months fields.
The first weekday of the month.
  1. Use a Monthly Schedule.
  2. Set Occurs to First Weekday.
  3. all the months fields.
The last Friday of the quarter.
  1. Use a Monthly Schedule.
  2. Set Occurs to Last Friday.
  3. the relevant months (for example, March, June, September and December).
The second Wednesday of every month.
  1. Use a Monthly Schedule.
  2. Set Occurs to First Wednesday.
  3. all the months fields.
  4. Set the Offset field to 7.
