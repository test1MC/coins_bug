Requisition Descriptions

requires you to enter a description when you enter a requisition.  You can use Purchase Requisition Description Maintenance to set up a list of common descriptions which you can select from the lookup in the Description field.

Set up separate lists for purchase requisitions and subcontract instructions.

To set up requisition descriptions:

  1. Go to Purchase Requisition Description Maintenance.
  2. This is a standard line maintenance procedure. Enter the text of a description on each line.
