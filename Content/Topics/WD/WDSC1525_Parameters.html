eRCT Parameters

The following standard Subcontract Ledger Parameters need to be set:

Parameter

Description

Action

DEFTAXSC

Default Subcontractor Tax Scheme.

Under IRE2, this should be set to U or V.

SBSTAX

Show in/out tax scheme on subcontract.

Set this to Y if you have some subcontractors who have some subcontracts that are not Relevant Contracts. This makes extra fields available in Subcontract Maintenance.

Set this to N if, for all subcontractors who are not exempt, every subcontract is a relevant contract.

TAXATPAY

Recalculate PAYE at Payment.

Set this to Y.

TAXRSELR

Recalculate Tax on Certificates Selected/Not Selected Report

We advise you set this to Y.

TAXRULES

Country Tax Rules.

Set this to IRE2.

 

In addition, the following Irish Subcontractor Tax Parameters need to be set:

Parameter Description Action

AUTOCRCN

RCT Auto create contract notification.

We advise you to set this so that creates notification records when orders are committed for payment.

LOCNSRCE

RCT Default location of work field.

Set this to the field on the JC record (or a related field) you want to use as the default location of work on new contract notifications.

LTAXRATE

Standard (Lower) Tax Rate (%).

Set this to 20.

PRINNAME

Principal's Main Name.

Set this to the name as held on Revenue records.

REGNO

Tax Registration Number.

Set this to the Principal Contractor's tax reference as held on Revenue records.

RETFREQ

RCT Return Frequency. 

Set this to the frequency (monthly or quarterly) as advised by Revenue.

RPTQUEUE

RCT Submission Report Queue.

If you are using report servers, set this to a queue on the main server, where the digital certificate is loaded.  Otherwise, leave blank. 

STOPEXP

Certify or Invoice against Expired Contract.

How you set this will depend on your internal processes. For example, you might set it to Warning if one person enters certificates and someone else deals with contract notification.

STOPUNPO

Un-Notified Mat/Plant Orders with Labour.

We recommend you set this to Warning.

STOPUNRG

Certify or Invoice against Un-Notified Contract.

How you set this will depend on your internal processes. For example, you might set it to Warning if one person enters certificates and someone else deals with contract notification.

TAXRATE

The higher rate of tax, which will be deducted from unregistered and non-compliant subcontractors.

Set this to 35.
