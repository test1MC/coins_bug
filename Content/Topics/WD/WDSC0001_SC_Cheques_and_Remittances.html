Configurable Subcontractor and Remittances

allows you to design and create various layouts for printing and remittances onto your own company pre-printed stationery.

Define the layout of and remittances using Document Designer in the System Module. See Configurable Documents for information on how to define configurable documents.

To define a document for and/or remittances:

  1. Use the document type s/Remittances (SC).
  2. In the Layout Type field, select:
    • to define the layout for ;
    • Remittance to define the layout for a remittance;
    • Combined to define the layout for a combined and remittance.
  3. You can set up the following forms:
    • header/footer - In the header form, put information to appear on the top of the remittance advice, such as the subcontractor's name and address. In the footer, put any additional information that is needed. Put the details that will be printed on the in the header or footer (see below). Header and footer use tables co_config, sc_mast, sc_check.
    • BODY - In the body form, put a single line containing details of the certificates paid. Body forms use tables sc_pay, sc_certificate, sc_subcon, jc_job.
    • TCARD - To print the timecard details for a payment, use form type TCARD. TCARD forms use table sc_timecard.
    • VAP - To print the VAP details for a payment, use form type VAP. VAP forms use tables vp_subcon, vp_subcc, vp_certdet, vp_unitdef, jc_stndcc, vp_wbsdef, vp_site.
    • DAYWORK - To print the dayworks details for a payment, use form type DAYWORK. DAYWORK forms use table vp_daywork.
  4. To print the timecard, VAP or dayworks lines before the body line to which they relate, the Details First?box.
  5. To configure a combined /remittance document, configure the body form to print the remittance lines. The can be either in the header or in the footer. If the remittance lines take up more than one page, you will need to print a 'VOID' form on the portion of the stationery on all pages after the first one. Configure a form called VOID, then, depending on whether the is at the top or bottom of the stationery, do one of the following:
    • If the is at the top: in the Location field, select Top of Page. In the Continuation Header Form field, enter VOID. Configure the Header Form to print the details.
    • If the is at the bottom: in the Location field, select Bottom of Page. Configure a form to print the details, and enter that form name in both the Footer Form and Continuation Footer Form fields. then replaces the footer with the VOID form after the first page.

The following fields are available:

To print details from the certificate:

RO_scd_tp^<CERTID>

      for the amount on the current certificate (where

<CERTID>

    is the Item ID from the certificate definition).

To print running totals:

RO_tot_amt, RO_tot_vat, RO_tot_paye, RO_tot_net, RO_tot_disc, RO_tot_tpByID^<CERTID>, RO_tot_cumByID^<CERTID>

      (where

<CERTID>

    is the Item ID from the certificate definition).
