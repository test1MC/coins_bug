Menu Icons

 

The menu tree uses the following icons:

Image Code Meaning
HOM A module home page.
ADM An administration function.  For example, functions to purge or delete transactions.
CFG A configuration function.
ENQ An .
FRM A function that produces a printable output, such as a document.
INP An input function.
MEN A menu.  Click the icon to expand the menu. 
MNT A maintenance function.
PRO A processing function.
REP A report function.
WBN A workbench.

Menu icons - configuration

Each function has a function category, which determines the icon used on the menu (see Function Categories). You can choose the of the icon for each category.

The images used for the menu icons are in the /images/ directory of the installation (see Location of Images and Other Files).
