Asset Captions

Captions are fields that allow you to record information about an asset that would not normally be held in the system as standard, and for which fields are not provided.

Captions on their own are for information only; you can and report on them, but there is no processing associated with them.  However, you can link asset captions to triggers, so that when you update a caption, updates the triggers linked to it.  This means can issue warnings, or stop users selecting the asset, when certain conditions have been met (for example, when a service or insurance renewal is due); see Linking Triggers to Captions.

To set up captions:

  1. Set up caption groups (using Caption Maintenance to set up Captions.
    • The Group and Type you select on the caption govern the additional tabs available after you click .
    • For character captions, you can update a list of allowable values. If you want to allow any value (in other words, to make the caption a free-text field) leave the list blank.
    • For captions that belong to groups configured as Update or Warn, specify which functions allow you to update the caption: Dispatch, Return, Workshop , Timesheet, Daily Timesheet, Picking List Workbench.
    • For captions that belong to groups configured as Update, add the triggers.

      NOTE

      Workshop already allows update of an Odometer field. If you want to use the odometer reading to update a trigger, use a caption for odometer and ignore (or hide) the existing Odometer field.
