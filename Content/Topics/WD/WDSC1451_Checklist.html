CIS Setup and Configuration Checklist

The following is a list of things you need to do to set up to operate the CIS 2007 scheme.

  1. For testing prior to the scheme

    In a non-live environment:

    • Set the SCUK date parameters (CIS2DATE, CIS2TRAE, CIS2TRAS) to an earlier date (for example, 1 year early) (see Changing Parameters)
    • Set the other parameters as for live running (see Changing Parameters)
    • Run UK CIS Scheme Configuration to check that schemes have been configured correctly (see Validating the Configuration).
    • Optionally configure a statement of deduction for testing (see Statement of Deduction)
  2. Basic processing within
    • Set the parameters that are relevant for basic CIS operation (see Changing Parameters)
    • Configure a statement of deduction (see Statement of Deduction)
    • Run UK CIS Scheme Configuration to check that schemes have been configured correctly (see Validating the Configuration).
    • Set security levels (see Gateway Password and following topics)
  3. Electronic submission of monthly returns
    • Set the additional parameters required for electronic communication (see Changing Parameters)
    • Set up the electronic communications infrastructure (not covered in this version of the documentation)
  4. Electronic subcontractor verification
    • OA is required for electronic verification.
    • Set the additional parameters required for electronic communication (see Changing Parameters)
    • Set up the electronic communications infrastructure (not covered in this version of the documentation)
