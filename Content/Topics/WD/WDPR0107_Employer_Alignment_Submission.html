Employer Alignment Submission

HMRC will issue each employer with a date per PAYE scheme from which they should start doing RTI submissions. On or after this date most employers will need to make an Employer Alignment Submission (EAS) for the PAYE scheme. The EAS is a one-off submission for the PAYE scheme, which will include all employees employed in the current tax year (including leavers, and s who have not yet been paid but have had an In Year Movements form sent already).

It will only be possible to create one live EAS per PAYE scheme and only when no live RTI submissions have yet been made for that PAYE scheme.

The HMRC requirement to do an initial alignment submission is only for the transition period from the old PAYE returns to RTI and will not apply after October 2013 at all. Until the end of November 2013 will require there to be an EAS (or first FPS - see First FPS) before enabling the other RTI features, to allow a little slack for teething problems getting a first submission sent out. From December 2013 EAS submissions will not be allowed at all.

Making an Employer Alignment Submission

  1. Make sure all employee information is correct.
  2. On the HMRC Workbench, click the link for the PAYE scheme.
  3. On the RTI tab, select the Return Type "EAS - Employer Alignment Submission" and click .
  4. Set the Type to Live (or to Test when testing).

  6. Select the submission record.
