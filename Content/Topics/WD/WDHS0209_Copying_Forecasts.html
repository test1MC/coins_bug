Copying Construction Forecasts

You can copy one contstruction forecast to another from the Forecast Matrix screen.  For example, when you have completed the initial forecast, you can copy it to a baseline forecast which compares with the current forecast.

You can also use Copy Daily/Weekly Forecast from the menu to copy sales and construction forecast data for multiple .

To copy a forecast in Construction Workbench:

  1. In Construction Workbench, choose the Copy Daily/Weekly Forecast option.
  2. Select the forecast to which you want to copy the forecast you're working on.
  3. Set the output options.
  4. Click .

    5. If the forecast you selected is not locked, copies the construction forecast data from the forecast you're working on to the forecast you selected, overwriting any existing data.

Forecast Setup

Use Plot Forecast Types to create the forecast types.  These are the names for the different forecasts you want to record.  Using the parameters PRWBFC and PRWBLFC, specify which of these types you want to use for the default forecast and the baseline forecast.

Locking Forecasts

If you copy data from one forecast to another, any existing data in the forecast you copy to will be overwritten.  If you want to keep a forecast, you need to lock it to prevent it being overwritten.  Use Lock/unlock Forecasts to lock or unlock sales forecasts and/or construction forecasts.
