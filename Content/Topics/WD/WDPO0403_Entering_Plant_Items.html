order lines - configuration

The layout of the Details frame depends on the Style, which is maintained using Styles in the Technical Information System.

Holiday Calendar (-in ) allows you to set up holidays (or 'charge free days'). These are days for which charges do not apply, such as public holidays. These are 'global' holidays which apply by default to all customers and assets. The Style specifies whether holiday charges apply.
