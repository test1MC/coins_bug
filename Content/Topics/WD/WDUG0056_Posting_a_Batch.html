Posting a Batch

Posting a batch commits the transaction details to the database and updates the ledgers.

To post a batch

  1. From the browse screen for the batch, select the batches you want to post. You can use the filters to help you identify the batches.
  2. Choose Post from the Choose Action list.

     

    NOTE

    Alternatively you can use the Choose Action list on the batch summary screen to post an individual batch.

     

    When you try to post the batch, checks that any batch control totals you entered agree with the actual totals for the batch. If not, will not let you post the batch, unless that batch type can be forced.

     

    also checks that all the transactions can be posted. You will not be able to post the batch if, for example, the costing for any of the transactions has not been completed. The Status field on the transaction will show details of any problems, and will display an error message when you try to post the batch.

You can see the results of the posting in Report Status. Click the 'Post xxxx batches' link.

Once you have posted the batch, if you want  to change the financial details you will need to use journals.
