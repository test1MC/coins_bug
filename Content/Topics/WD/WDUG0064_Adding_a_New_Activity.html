Adding a New Activity

 

You can add a new task or appointment for yourself, or assign it to another user.

To add a new task or appointment:

  1. Go to the Activity Workbench.
  2. Use the Add Type drop-down to choose task or appointment.
  3. Click .
  4. Fill in the details of the activity:
    • Select the type of activity.
    • To assign the activity to another user, use the lookup in the User field.
    • If you are importing activities into Outlook®, and want want Outlook® to issue a reminder for the activity, Reminder? and enter the date and time of the reminder (for a task) or the number of hours and minutes before an appointment.
