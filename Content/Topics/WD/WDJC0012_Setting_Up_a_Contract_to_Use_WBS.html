Setting Up a to Use Work Breakdown

The decision to use work breakdown analysis can be made on a -by- basis, though most businesses have one way of working - either using work breakdown, or not using it. 

To set up a to use work breakdown:

  1. When setting up the record using , click the Cost & WBS Codes tab and the Use WBS Codesbox.

    This box may be restricted so that only certain users can change it.

  2. If you are using WBS codes that are structured in the same way as your section and activity codes (for example, if the third and fourth characters in the WBS code correspond to an activity code), use the Section Length, Section Start Position, Activity Length and Activity Start Position fields to specify the length and starting position of the section and activity components of the WBS code.  This allows to work out the section and activity codes from the WBS code and assign the WBS code to the correct section and activity.  For example, a might have sections A1, A2, ... B1, B2,.. and activities DE (Design), EX (Excavate), FE (Fencing) and so on. So the WBS code for a particular excavation activity in section B1 might be: B1EX1402. 

    When you enter a new WBS code, combines the short descriptions for the section and activity and uses this as the default description for the WBS code.

    If you are not using both sections and activities, leave the relevant fields blank.
