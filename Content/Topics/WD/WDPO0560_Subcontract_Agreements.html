Subcontract Agreements

lets you keep track of the progress of subcontract agreements, and print reminder letters.

keeps track of the progress of subcontract agreements through the various approval and signatory stages by using a list of "events". Events are “things that have happened to the agreement”. Each event also shows when it happened, and who did it.

There are two types of event: automatic and manual.

  • Automatic events are events that generates as part of its processing. Procedures which deal with an agreement (for example, committing an order, accepting or rejecting an agreement, sending reminder letters) add automatic events to the list for that agreement.
  • Manual events allow you to record stages in the progress of an agreement which are not related to automatic processing; for example, telephone conversations.

Recording an Agreement

When you enter an order, the Agreement Requiredbox allows you to specify that an agreement is required.  If this is ed, the Agreements tab is available, which allows you to record the details of the agreement.  

Agreement Administration

SC Agreements Workbench allows you to track and manage the progress of agreements.  

The dates for certain key events are shown on the screen, and choose actions allow you to record these key events for one or more selected agreements:

  • Sending the agreement to the subcontractor.
  • Receiving the agreement from the subcontractor.
  • Accepting/agreeing the agreement.
  • Rejecting the agreement.
  • Setting the date.
  • Signing of the agreement by a director.

 

If you reject an agreement, sets the Date Sent and Date Received fields to blank. If a rejected agreement is re-sent, sets the Date Rejected field to blank. 

If an agreement has been sent but is not received back from the subcontractor within a specified number of days, the Date Received column shows amber.  If it is still not received after a further specified period, the column shows red.

If an agreement has been received from the subcontractor but not agreed within a specified number of days, the Date Agreed column shows amber.  If it is still not agreed after a further specified period, the column shows red.

You can filter on when reminder letters are due, and if an appropriate Mail Merge has been set up, you can send reminder letters to subcontractors and record the dates using an automatic action.

All the events and their dates are shown on the Agreement Events screen (click the link on the Agreements browse to show the Agreement Events screen for an agreement).  You can also use this to add events manually, to record other stages in the process.
