Printing the Documentation

You can print individual topics from the Online Documentation, or you can print all the topics in a documentation "book". A book () is a menu folder for documentation topics.

You can print all the topics in the documentation book for a module (for example, Procurement ), or just the topics in one of the books within that (for example, Goods Received Notes).

To print a single topic:

  1. Display the topic you want to print.
  2. Make sure that any hidden text or images you want to print are displayed.
  3. Click the button (above the title area of the documentation window).
  4. Select the Windows printer you want to print to.

To print a book of topics:

  1. Click Documentation Print (in the documentation menu).
  2. At the Start Menu prompt, enter the function code of the documentation book (or menu) you want to print. You can use the lookup button to select it.
  3. Some books include submenus that show the descriptions from the side-frame help for each function on the menu. If you want to include these in the print, the Include Menu Options box.
  4. Click .

    displays a print-formatted version of the documentation topics in the book you selected. This includes a table of contents, and any hidden text or images. 

    then displays the Windows print box. Depending on the number of topics involved, there may be a delay before the box appears.

  5. Select the Windows printer you want to print to.

To create a Word document from the documentation:

  1. Generate the print-formatted version of the documentation topics by following steps 1-3 above.
  2. When the Windows print box appears, click Cancel.
  3. Select the text in the documentation window by clicking and dragging (do not use Select All as the result seems to cause problems when you paste into Word).
  4. Right-click the documentation window and click Copy.
  5. Open a new Word document and paste the documentation into it.

    The results appear as a set of nested tables.

  6. In Word, select the whole of the outer table, and from the Table menu select Convert, then select Table to Text.
