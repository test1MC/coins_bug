Submitting a Return

When the Return details have been checked, the Return can be submitted to HMRC.  For UK Returns this can be done electronically using .  If the return is submitted outside , it should be marked as manually submitted.

To submit a Return electronically:

  1. Go to Returns Workbench.
  2. On the Returns tab, select the Return you want to submit.
  3. Choose the Submit Return Electronically action.  

    This shows the figures that will be returned.

  4. Click .

    Any adjustments to the Return or other input on this screen will be saved.

  5. Review the Return figures before submission to HMRC.

  6. Click .

    submits the Return electronically to the Government Gateway.

NOTE

Once a Return has been submitted to HMRC, it cannot be cancelled within (except as noted under Cancelling a Return).  Note that there is no facility for to send an amendment or cancellation of a submitted Return to HMRC.  If a Return which has been submitted to HMRC needs to be amended or cancelled, you will need to contact HMRC.

 

Submitting the Return manually

If the Return is submitted outside of , mark the Return in as "submitted manually", so it is clear in the workbench that the Return has been submitted. 

To mark a Return as Manually Submitted:

  1. Go to Returns Workbench.
  2. On the Returns tab, select the Return you want to submit.
  3. Choose the Mark Return as submitted manually action.

    This updates the submission status from "Unsent" to "Manually Submitted".

NOTE

Currently only supports electronic submission of UK Returns.  Submissions to other jurisdictions, including Ireland, should be marked as manually submitted.

does not support electronic submissions for Returns under the annual accounting scheme as this requires additional entries, and very few clients would qualify for this scheme.
