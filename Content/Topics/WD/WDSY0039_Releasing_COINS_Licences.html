Logging Out and Releasing s

Logging out of in the normal way releases the .

To release a being used by the user must either :

  • Log out using the Log Out icon
  • Log out via the Menu

NOTE

When a is being used by , will monitor the time at which the was last used. By default, after one hour of inactivity, this will be released and the user will be prompted to log back in should they try to perform any action within the session.

The Menu System Parameter WEBLGOUT can be set to any value above 60 (value in minutes) to extend this default if required. The minimum, however, will always be 60 minutes.

If a user closes their session using the Windows Close () button, the will remain in use for one hour until it is released by the system as above. 

NOTE

Timed Logout settings in Menu System Configuration and Users are only used for sessions.
