Batch Control Totals

Batch control totals act as a check that the transactions have been entered correctly.

For each batch, you can type batch control totals before entering the batch. When you have finished entering the batch, will check the batch control totals against the actual totals for those fields. If they are not the same, someone needs to take action: for example, to enter more transactions, correct transaction details, or change the batch totals.

will only display a control total if the batch type has been configured to show the total. If a total field is not shown, then it is excluded when checking to see if a batch is balanced.

If the total is shown, then it is disabled for input unless any of the following are true:

  • You are adding a new batch.
  • You are a member of the ROOT user group or you are logged in as sysadmin.
  • It is not your batch, and you have permission to post the batch.
  • is configured to allow users to change their own batch totals
