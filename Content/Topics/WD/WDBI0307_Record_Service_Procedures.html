Record Service Procedures

Every table in the database has a RSP (Record Service Procedure). Each of these RSPs provides the OA Reporting tool with the business logic required to extract the appropriate data from the database.

In the Database you can see the RSP under the Table Code and its Label. The RSPs have a naming convention - {table-id}-rsp.p.

Where the Table ID is as shown in the Database (you may also hear this referred to as the Table Acronym or TLA).
