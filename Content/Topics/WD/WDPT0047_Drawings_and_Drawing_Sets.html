Drawings and Drawing Sets

You can store drawings against the project.  You can record each separate revision of the drawing, and store the drawing file for each revision.

Drawing sets allow you to group together and maintain related drawings for easy review, distribution, and revision.

You can:

  • Email drawing documents
  • Send a (relating to the drawing)
  • Print a drawing set (this prints a cover sheet that lists the drawings)
  • Email drawing set documents

There is a report ? Drawing Log ? which shows details of each drawing, and lists any items linked to it.

Configuration

Use Drawing Discipline to set up a list of drawing disciplines for users to choose from.
