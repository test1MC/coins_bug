Additional Characters in Searches

You can use the following additional characters in the Search field of a simple filter (or in the Matches field of an advanced filter).

  • To search for any single character, use . (). For example:
    C1.0  tells to show any items beginning with C1 and ending with 0, with any single character in the third position.
    ...tells to show any items with exactly three characters.
  • To search for any number of characters, use * (asterisk). For example:
    C1* tells to show any items beginning with C1.
    C1*0  tells to show any items beginning with C1 and ending with 0, with any number of characters in between.
    *BE*  tells to show any items that contain BE.
  • To search for a list of items, use , (comma). For example:
    A1000,A1500,B1224  tells to show only the items A1000, A1500 and B1224.
    A1000,A1500,B* tells to show the items A1000 and A1500, and any item that begins with B.

    NOTE

    To search for a blank field, enter a single comma.  This is interpreted as a list containing two blank entries.

    To search for a non-blank field, enter .* (a  and an asterisk).  

  • To exclude specific items, enter an exclamation mark (!) before the items you want to exclude and follow them with ,* (comma and asterisk)enter a caret (^)  before the items you want to exclude. For example:
    ^A1000!A1000,*tells to show all items except A1000.
    ^A1000,A1500!A1000,!A1500,*tells to show all items except A1000 and A1500.
    !A15*,A*tells to show all items beginning with A, except those that begin A15.
  • To search for a range of items, enter \\ and use - (dash) as a range separator. For example:
    \\A50-B50 tells to show items with codes in the range A50 to B50
    \\^A50-B50 tells to show all items except those between A50 and B50.
  • If the values contain a dash, you can enter this with a backslash (\) or by enclosing the string in quotes.
    e.g. "A-000"-"A-999" would select a range of codes A-000,A-001,A-002....A-999.
