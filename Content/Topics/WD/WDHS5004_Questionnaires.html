Questionnaire Set Up

Questionnaires are available from several of the workbenches, and allow you to record responses to various questions. You can then produce a statistical analysis of the information that has been entered; use the Questionnaire Analysis report.

To set up a new questionnaire:

  1. Go to Questionnaires.
  2. Click to enter a new questionnaire.
  3. Enter a code and description for the questionnaire, and select which workbench(es) it can be used on.
  4. Click .

    displays the Questions tab.

  5. For each question you want to include in the questionnaire, add a new record.  Refer to the side-frame help.
  6. If the response type of the question is LIST, after you have saved the question record, click the link in the Number column to fill in the possible responses that the user can choose from.

Once you have set up the questionnaire, users can select it while they are viewing a record in one of the workbenches, by clicking the Questionnaires tab and choosing the questionnaire from the drop-down selector.
