Recording Cash for a Part Exchange

When you sell a part-exchange property, you need to record the receipt of the cash and allocate it to the correct property.

To record cash received for a part exchange:

  1. In the module, go to Transaction Entry.
  2. Create a new batch (or click a link in the batch column to add the transaction to an existing batch).
  3. Add a new transaction.
  4. Fill in the transaction details, paying attention to the following:
    • If the Type drop-down is available, select Receipt. (If the Type drop-down is not available, the type is specified on the batch header.)
    • In the T/Type field, select the appropriate transaction type.
    • In the Ledger field, select HS (or the item that corresponds to House Sales on your system).
    • In the Account field, type PX then use the lookup to select the part exchange property, then the category for the transaction (for example, completion monies).
    • fills in the Payer/Payee field with the name of the purchaser of the part-exchange property.
  5. Save the transaction.
