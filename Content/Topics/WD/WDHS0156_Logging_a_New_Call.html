Logging a New Call

When you log a call, you record one or more issues that have been raised as a result of the call.

You then add tasks to the call. Tasks can be associated with one or more issues (for example, a task might be a visit to the property to assess the status of several issues).

There are two ways to add calls:

  • Standard Add allows you to record details of the call and one or more issues. You must then record the tasks separately and assign them to the appropriate issues.
  • Quick Add allows you to record a call, an issue and an associated task at the same time. Quick Add records the key information for calls and issues; Standard Add records more details.

To log a new call:

  1. Open the summary for the plot.
  2. Click the Calls tab.
