Schedules of Rates

FM allows you to store multiple schedules of rates against each major category on a .

You can have different types of schedules. For example:

  • You can have different material, , and schedules, with different fields.
  • You can have different schedules to allow for rate changes over time, allowing you to retain a history of changes and prepare new schedules in advance.

You can prepare the schedules in a spreadsheet and load them into . There is also a maintenance function within . 

To set up schedules of rates types:

  1. Use the parameter SORFIELD to define names for the additional fields you want to use. For example: Length|Width|Miles|.
  2. Use SOR Types to set up the different schedule types and which standard and additional fields you want to display or update on each type. For example, if you do not use invoice classes or , you can hide these fields.
  3. If you are using schedule types to record rate changes over time, set the current schedule type.
  4. You can change the current schedule type for individual :
    1. Open the in .
    2. Choose Setup from the Options menu.
    3. On the Details tab, change the Current SOR Type.

To load a schedule of rates:

  1. Use Load SOR Items to import a schedule to a major category.
