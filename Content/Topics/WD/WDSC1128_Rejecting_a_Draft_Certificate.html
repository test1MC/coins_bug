Rejecting a Draft Certificate

Any draft certificates that are rejected will be marked as status 0 - "attention required", and will appear back on the draft certificate entry screen so they can be revised.  They will disappear from the Authoration screen and cannot be recalled.

To reject a draft certificate:

  1. Click the link in the Txn column.
  2. Select Reject from the Options menu.
  3. A comments box is displayed (30 characters).  If you cancel from here () then the rejection does not proceed. If you proceed, the rejected certificate disappears from view. Rejecting the certificate sets the status to 0 ("Attention required").
  4. Click .

If there are any later draft certificates for the same order, these will be rejected also. asks you if you wish to proceed if this is the case.
