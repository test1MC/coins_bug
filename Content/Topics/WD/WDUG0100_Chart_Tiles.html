Chart Tiles

Chart tiles present information as a chart.

Each of these provides a single series chart specially designed as a desktop tile; you can configure the foreground and background .

See List of Chart Tiles.

To configure a chart tile:

  1. Go to the Desktop Item Maintenance screen for the tab you want to add the chart to.
  2. Click  to add a new record.
  3. In the Function field, enter the function code of the chart.  For example, %WCB5100CCBD is the code for the “Cash Receipts Last Seven Days” chart.
  4. In the Title field, enter the title you want to be displayed on the chart.
  5. In the Size field, select an appropriate size (not Half or Full).
  6. In the Open In field, select the option for the how you want the linked function (in this example, ) to open.
  7. Change any other settings, such as the or subtitle.
  8. Click .
  9. Click the  logo on the Desktop to refresh it.

    The chart now shows on the Desktop:

If the chart function needs to collate a large amount of data, it may take a while before the chart appears.  Charts are not refreshed automatically (as smart tiles are); click the  button to update the chart content.
