In the Middle East it is a normal business practice that for substantial purchases, house sales and tenancy agreements a series of Post Dated (PDCs) is handed over.  For example, for twelve monthly payments a company would send twelve separate with dates in the future. 

Configuration

To enable the feature, set the parameter FUTRPR to Y.

In PL parameters, specify a hold code for : set FUTRPYHC to the hold code that will give to invoices that have assigned to them.  Make sure that PL/USEFINH parameter is set to Y as this is required to be able to use financial hold.

There is a Print action on the browses that allows you to print a for one or more future payments.  Configure a new document in Document Designer using the Future Payments (CB) document type.
