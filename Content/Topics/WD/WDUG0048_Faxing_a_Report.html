Faxing a Report

If is set up to allow it, you can fax the output. the Fax box, and fill in the other fax fields.

If you want to review the fax before it is sent, the Review box. See Reviewing Messages. For example, you may choose not to review fax or email output if you are running a delayed or scheduled report.

After the document or report has generated, either puts it directly into the fax queue, where it is converted to an image file and sent, or (if the review feature is enabled - see Reviewing Messages) puts it on a queue for you to review.

Faxing printed output - configuration
