File Uploads

Various routines including spreadsheets and importing PO Lines allow you to upload files from a PC or network drive. In order to do this you need to set the upload script and directory.

Upload script and directory

  1. Set the GZIPATH parameter - gunzip and gzip are normally installed in /usr/contrib/bin/ or /usr/bin/ (find / -name gz*). SY Parameters

  2. Ensure the system parameters UPLOADS and UPSCRIPT are set correctly. The Unix uploads directory and .plx should have been setup as part of the technical build. SY Parameters – UPLOADS
