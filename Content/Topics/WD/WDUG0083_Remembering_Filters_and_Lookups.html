Remembering Filters and Lookups

By default, "remembers" any filter settings you enter on a screen, so that the next time you go to that screen, the same filter is applied.  For example, if you run Subcontractor Maintenance and filter to show only subcontractors with codes that begin D, when you next run Subcontractor Maintenance it will still show only subcontractors with codes that begin D.

In the same way, "remembers" if you have clicked the ("Show All") or ("Total") buttons, and any records that you select remain selected.

You can change the way remembers filter settings, so that, for example, filters are never remembered, or are only remembered within the same day.

To change how filters are remembered:

  1. Move the mouse pointer over your profile picture, and on the user menu click to open the User Preferences screen.
  2. Click .

    You can change the way remembers filter settings on browse screens, and on lookups, separately.
  3. Set the drop-down in the Recall Filters and the Recall Filters in Lookups fields as follows:
    • Default - To use the default settings for your system.
    • Full - To always remember the filter settings.
    • Session - To remember the filter settings during a session, but clear them when you log out.
    • Day - To remember the filter settings during the same day only.
    • None - To never remember the filter settings.
  4. Click .  

Remembering filters and lookups

If page sections are changed, such that the stored data becomes inconsistent with the current page definition, it can sometimes result in either blank pages, errors, or missing bits of HTML. If you add helpMode=clearDefault to the URL, then before each page starts the defaults record will be deleted (removing all filters and sort orders) and reset as though it were the first time in the system. This continues on each page you visit/revisit until you remove it from the URL.

Parameters

The parameters PREFFILT and PREFLKUP define the default "recall" .
