Template Orders

A template order is an order with partial information filled in, that you can copy and use as the basis for creating orders (this does not apply to ). For example, a template order might contain a set of standard items you need to order often (perhaps with the and/or omitted).

To create a template order:

  1. Go to Material Orders and add a new order.
  2. In the Type field, select the code for Template order types. 
  3. Fill in the information you want the template order to contain.

You do not need to commit template orders.

To base an order on a template:

  1. Go to Material Orders.
  2. If you want, you can filter the list to show template orders only: in the Search: field select Order Type, and enter the code for Template Orders (for example: T).
  3. Select the template you want to use.
  4. Click [Copy].
  5. Fill in the rest of the order in the usual way.

    NOTE

    You can use any order as a "template" by copying it; it does not have to be a template-type order.
