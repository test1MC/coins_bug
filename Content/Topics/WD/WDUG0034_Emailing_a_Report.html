Emailing a Report

If is set up to allow it, you can email the output to anyone with an email address.

On the Output Options tab:

  1. the Email box.
  2. Fill in the email address or addresses and the subject of the email.  The subject defaults from the title of the document or report, but you can change it.

  3. The document or report (and the associated spreadsheet, if you ed Spreadsheet) will be included as an attachment to the email.  By default the file names are based on a sequence number for the module, but you can rename the file using the Attachments Name field.
  4. If you want to include text in the body of the email, type this in the Message box.  To enter a formatted message, the HTML box; you can then use the HTML editor to format the message.
  5. If you want to review the email before it is sent, the Review box.  See Reviewing Messages.  For example, you may choose not to review fax or email output if you are running a delayed or scheduled report.
  6. Click to run the report and send the email. 

includes the PDF file of the document or report as an attachment to the email.  For reports, also includes the CSV file in the email.

Emailing a report - configuration
