GL Reporter – Columns

The columns control what information is shown on each line.  You can select from a number of different data items, and specify which row types each one is shown on. Information displayed will typically include descriptions of the rows and balances for account lines and total lines.

When adding a column, first select the type of item you want the column to contain. 

The Label is generally for your reference when configuring the report, to explain what the column shows; however, for Literal columns it is the text to print on the report.

Column widths

In the Col field, specify the starting position for the column.  The starting position is (approximately) the number of characters across the page.  Note that if the format of a column is wider than the difference between the two Col fields, the following columns will not be aligned.
