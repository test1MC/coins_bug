Installing Tax Tables

 

  ***NOTE: THIS IS A US SPECIFIC TOPIC.***  

Once a US has been configured in OA, Tax Tables for processing US Payroll need to be maintained. The procedure below demonstrates how this can be done.

How to Install Tax Tables in OA:

A. Download Tax Tables from Client

  1. Go to http://support.coins-global.com/usa/download/softwareExchange.php ( USA Client Area login required).
  2. Click Download Tax Tables.
  3. Save file to PC and unzip.
  4. Be sure to download the correct versions of Linux or Unix (Note: Win does not apply to OA) or there will be errors in View Report in Report Status.

B. Load Tax Tables

  1. Log into OA.
  2. Go to Tax Table Load.
  3. If files are on your Local PC, choose Local.
  4. Browse to your Local File Path for the files called tx_tables.d.
  5. Deselect Validate Only to run the Load program, and select Validate Only to check for errors.
  6. Select Output.
  7. Whether Validate Only was selected or not, you will be able to view the results of the load in REPORT STATUS.
  8. If Validate Only was selected, be sure to run the LOAD program with Validate Only deselected.

C. Load Calculation Rules

  1. Log into OA.
  2. Choose Tax Calculation Load.
  3. If files are on your Local PC, choose Local.
  4. Browse to your Local File Path for the files called tx_calcrules.d.
  5. Deselect Validate Only to run the Load program, select Validate Only to check for errors.
  6. Select Output.
  7. Whether Validate Only was selected or not, you will be able to view the results of the load in REPORT STATUS.
  8. If Validate Only was selected, be sure to run the Load program with Validate Only deselected.
