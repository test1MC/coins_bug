Returns

The recommended process for creating and submitting a UK Return using is as follows:

1. Check you are ready to create a Return

1.1. Ensure all transactions are posted

1.2. Run the Reconciliation Report

1.3. Review any unreturned transactions

2. Create the Return

3. Check the Return

3.1. Review any unreturned transactions

3.2. Review the transactions included in the Return

3.3. Review the Return figures

3.4. Run the Reconciliation Report

3.5. Input any manual adjustments

4. Submit the Return

4.1 Submit the Return electronically or

4.2 Submit the Return manually

5. Post the Return

6. Check the Return processing is complete

 

See also Cancelling a Return
