Actions

Actions allow you to set up reminders of tasks or appointments that relate to a property, a call or an issue. You can set a reminder for yourself, or for another user. For example, you could set up an action to remind yourself to phone a purchaser, or to remind a colleague to follow up on a call.  

You can also create follow-up actions when you run an event (see Events).

The action will appear on the Actions tab for the property, call or issue, and also on your Activity Workbench (or the Activity Workbench for the user you assigned the action to).

To set up an action:

  1. From the Summary page for the plot, click the Actions tab.

  1. In the Add Dropdown, select the type of Action to be added

  1. Click to create a new action.
  2. Do one of the following:
    • If the action is an appointment, click Appointment and enter the date and time of the appointment.
    • If the action is a task, click Task, and enter the date by which the task is to be completed.
  3. Select the type of action and the priority.
  4. By default, the User will be yourself; this means the action is for you. If you want to assign the action to another user, enter or select that person's user ID.
  5. If you want the action to be visible only to the person it is assigned to, the Private? box.
  6. Enter text to describe the appointment or task.
  7. Click to save the action.
