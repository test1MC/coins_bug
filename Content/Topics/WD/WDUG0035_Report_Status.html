Report Status

Report Status lists all recent reports, as well as other print requests and background processes.  You can filter to show only your reports, print requests and processes, or those generated by all users for which you have security access (depending on your permissions, the "all users" option may not be available).  You can use it to view previous reports, and see the progress of current report requests.

Unread print requests are shown with a theme-ed circle next to them.

If you have print requests that are not yet complete, the Report Status icon in the page header shows how many: .  If you have 10 or more outstanding requests, the button shows !.  If you move the mouse pointer over the icon, a popup menu shows any unread print requests.

Report Status also shows saved reports (see Saved Report Options) and scheduled reports (see Scheduling a Report).

To view the Report Status:

Either:

  • Choose Report Status from the menu, or
  • Click the icon in the page header.  This opens Report Status in a frame.

Several row actions are available:

  • Mark as Read allows you to mark print requests as "read" without opening or reading them.  They will then not be highlighted in bold, and will not appear on the Reports popup menu.

  • Mark as Unread allows you to mark print requests as "unread".  They will be shown in bold and will appear on the Reports popup menu.

  • Cancel Report Request allows you to cancel a request (see Cancelling a Report).

  • Store Document allows you to store the output of a print request using Document Management.

  • Urgent Report Request allows you to put reports onto an urgent report queue; see Urgent Reports and Report Queues.

  • Archive Report allows you to keep the output of a print request; it will be available from the Archived Reports tab.

  • Reprint allows you to select one or more reports and print them as a single PDF document.
  • Email Report allows you to send the files associated with the selected report(s) as attachments to an email.
