PL Configuration

Initial configuration

You need to run PL Configuration for each .

This includes:

  • G/L Details - set up various General Ledger control accounts and how to post to them.
  • Defaults - set up the default bank account, payment due and discount terms, and the periods for calculating outstanding payments.
  • Analysis sets - set up the PL analysis sets.  Then use Analysis Sets to set up the valid codes for each analysis set.

Open or close using Periods Maintenance.
