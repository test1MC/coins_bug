Activities

 

Activities represent a type of work on a , such as road-works, earthworks or excavation. They are typically standard - that is, they are the same for all . However, individual can use their own list of -specific activities if they involve activities that are not in the standard list. Standard activity codes cannot be used for -specific activities, and -specific activity codes cannot be used for standard activities; different can use the same codes as each other for -specific activities, but they will not necessarily mean the same thing.

Activity classes

Activity classes allow you to control which sections and activities can be combined.  Each activity can be assigned to an activity class. For each section, you can specify which activity classes are allowed to combine with it. In order for an activity to be valid on a section, the Class on the activity must be in the Activity Class list on the section (or the Activity Class must be *).
