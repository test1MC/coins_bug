Prices Import File

You can import prices into the Prices Workbench, using Import Prices.

The prices import file consists of a CSV file with the following columns:

  • Resource
  • Supplier's Resource Code
  • Supplier's Resource Description
  • Category
  • Gross Price
  • Per
  • Status
  • Valid From
  • To
  • Agreement
  • Notes
  • Office
  • Discount1
  • Discount2
  • Discount3
  • Discount4
  • Discount5
