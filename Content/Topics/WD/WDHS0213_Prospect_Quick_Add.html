Quick Add

The quick add provides a quick way to enter details of a and to check whether their details already exist in the database.  If they do already exist you can add to the existing details.

To search for and add a :

  1. On the Workbench, go to the Quick Add tab. 
  2. Type in the details you know for the contact.  If you want to record a second contact, the Is there a Second Contact? box and type the details of both contacts.

    As you type, searches the database for potential matches, and displays them below the search screen.  

    The list of possible matches is sorted by their score; this is based on how many of the fields you entered match the information on the People Workbench records.

    If any of the people in the list are already recorded as a in this , the column for that person has a View link that will take you directly to the record. 
  3. At this point, you can:
    • Use the View link to open an existing record.
    • Use the Quick Add screen to add information to an existing record, if one (or more) of the people in the list is the contact you want.
    • Use the Quick Add screen to create a new record. 
  4. If you are going to create a new record, enter the details that relate to the :

    NOTE

    Depending on how is configured, not all of the fields will be available.
  5. Use the Duplicate? field to specify how the potential duplicate People Workbench records relate to the main and second contact details you entered.  This will then be used to determine what People Workbench and records will create and/or update. 
    • By default, the Duplicate? field is set to Not a Duplicate. This indicates that the person is not the one you were searching for.  
    • Select Duplicate of Main Contact Above or Duplicate of Second Contact Above if the person in the duplicate list is the same as one of the contacts you entered. You can only select one duplicate of main contact and one duplicate of second contact.
    • Select Different Contact for Same if the person in the duplicates list is to be included as a .

    • Duplicate of Main Contact Above
      • If the person is not a record and the second contact is also not a , creates a new record and attaches the person to the new record.
      • If the person is not a record and the second contact is a , attaches the main contact person to the second contact's record.
      • If the person is already a , uses any details missing on the person record that have been supplied on the quick add screen to update the person record.
    • Duplicate of Second Contact Above
      • If the person is not a record and the main contact is also not a , creates a new record and attaches the person to the new record.
      • If the person is not a record and the main contact is a , attaches the second contact person to the main contact's record.
      • If the person is already a , uses any details missing on the person record that have been supplied on the quick add screen to update the person record.
    • Different Contact for Same
      • If the person is not a record, creates a new record and attaches the person to the new record along with the main and second contacts.
