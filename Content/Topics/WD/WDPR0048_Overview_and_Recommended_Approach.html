Overview and Recommended Approach

Once you have completed the setup, you are ready to operate the End of Year Workbench. This workbench controls the activities and the flow of data to and from the Government website.

Step 1 - Preliminary Tests

We recommend that you perform a test submission as soon as possible after April 5th (see the three types of return that can be used in $O$WDHTEOY0421$O$). Also, if you are going to be using the new laser P60s, we recommend that you test the setup on the intended printer so that you can adjust the P60SSIDE and P60STOP parameters.

Step 2 - Live Submission (for each Scheme)

For live submission, follow the flow outlined below:

  1. Select Scheme.
  2. and update employee forms.
  3. Complete the P35 summary.
  4. Complete the return (this is still internal; in other words, nothing has been submitted).
  5. Submit the return.
  6. Print the P60s.
