VSI-Fax Configuration for Faxing and Emailing

Once VSI-Fax has been installed on the host server there are various areas within the system which must also be configured to allow successful faxing from .

Depending on which types of document you wish to fax, and from which modules, the nature of the configuration may change. These topics are designed to give an overview of the system parameters and central configuration.

You can on fax requests using Fax Request History.

Configuration

  1. Set up a report queue for faxing, using Report Queues.
  2. Create Report Agent families.
  3. If using email, link to the mail SMTP server.  The SMTP server needs to authore relaying of SMTP mail traffic from the server. 
    • For , VSI-Fax needs to be configured with the mail server details.
    • For , the SY/MAILHOST and SY/MAILPORT parameters point to the mail server.
  4. The CO/FAXSEND parameter must be set to the full path of the VFX command /opt/vsifax/bin/vfx .
  5. The Print Manager parameter GENBACK must include the VSIFAX command: VSIFAX=/opt/vsifax .
