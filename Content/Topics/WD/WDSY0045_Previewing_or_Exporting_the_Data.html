Report Writer - Previewing or Exporting the Data

When you are writing the report, the Data tab shows the data that the report will contain.

If the report requires selection criteria, the Input tab allows you to select which data to include, in the same way as when you run the report. When you click the [Apply Filter] button, refreshes the Data tab with the data you specified.

On the Data tab, you can also:

  • Show totals (for the fields marked to show totals): click the [Total] button.
  • Export the data: click the [Export] button
