Selecting Records

You can select several records from a browse screen, then carry out the same action on all of them (for example: delete them, or apply an action from the Choose Action menu).

NOTE

Some choose actions only work  on a single record, and will display an error if you try to run the action for more than one record.

To select records

  • To select a single record, click the record.
    Selecting a Single Record
  • To select several adjacent records, click the first record you want to select, hold down Shift, and click the last record you want to select.
    Selecting Adjacent Records
  • To select several separate records, hold down Ctrl and click each record.
    Selecting Adjacent Records
  • To select all the records in a browse, hold down Shift and click (at the top left corner of the browse).
  • To see how many records you have selected, move the mouse pointer over the button; the "Show Selected" menu item ”Rows selected” tooltip shows the number of records selected.
  • To show only the records you have selected, either click the button, or move the mouse pointer over the buttonclick the button, and click the "Show Selected" menu item. The of the button are reversed to indicate that only the selected records are shown.
  • If you have selected several records and want to un-select one, hold down Ctrl and click the record you want to un-select.
