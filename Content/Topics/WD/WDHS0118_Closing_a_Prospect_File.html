Closing a File

If a is no longer interested in purchasing a property, you can close the file. If a file is closed, there will be certain events you can no longer run for the record.

To close a file:

  1. Find the .
  2. On the Actions tab, open any outstanding actions relating to the prosptect and complete them.
  3. On the Required tab, open any the showed an interest in, and the No longer interested in this development? box.
  4. From the summary, click Options and choose the Close File event.
  5. Enter any comments you want to make about closing the file, then click .

    6. The File Status changes to Closed.

You can also use Close Inactive or Bulk Close to close several files in one go.

Closing a file - configuration

You need to set up a event with the system event code CLOSE, and use Event Security to control which users can run the event.
