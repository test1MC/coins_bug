Adding User Text to Documentation

User text allows you to add notes and comments to existing topics.  You can also add links to external documents (such as Word documents) or to web pages.  The user text is formatted like this:

This paragraph is formatted as user text.

User text is visible to all users on your system whenever they view the topic.  The text appears below the section you add it to.

To add user text to a topic:

Either

  1. Go to Topic Maintenance.
  2. Find the topic you want to add text to and click the link in the Code column.
  3. Click next to the section you want to add the text to.
  4. Click the User tab.
  5. Enter the user text.
  6. Click .

or

  1. In the Address box at the top of the documentation window, type &helpMode=edit at the end of the URL, and press Return to refresh the documentation window.

    The documentation window now has editing buttons.

  2. Click next to the section you want to add the text to.
  3. Click the User tab.
  4. Enter the user text.
  5. Click .

Formatting user text

You can control the formatting of user text by changing the user.css cascading style sheet file. 

Enter formatting declarations for div.usertext.  For example:

div.usertext{
    background-image: url(/images/mylogo.gif);
    background-repeat:no-repeat;
    background-color: #FFFFAA;
    font-family:"Times New Roman";
    padding: 2px;
}
