Personnel Tables

The following tables should be populated for use in Personnel:

  • Titles allows you to set up personal titles.
  • Job Group allows you to set up the job groups and the job roles that belong to them. For example, you might use job groups to represent departments.

    Click the link to add or update job roles within a job group.

  • Qualifications allows you to set up codes to indicate employees' qualifications.
  • Membership allows you to set up codes to indicate employees' professional memberships.
  • Skills allows you to set up codes to indicate employees' skills.
  • Experience Categories allows you to set up codes to indicate employees' experience.
