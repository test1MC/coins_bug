Subcontractor Papers - Analysis Sets

The final frame in the Subcontractor Maintenance series shows Analysis Sets if they have been defined in configuration. You can change the Analysis Set information later by selecting Analysis from the pop up menu.

For each analysis set, enter the code for this subcontractor, or use drop down list to select one.

If no analysis sets have been defined then this frame does not appear.

See the section on Analysis Sets in the System Guide for more information about Analysis sets.
