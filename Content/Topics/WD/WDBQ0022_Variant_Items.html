Variant Items

Variants require changes to the standard : either adding or removing items.

To add items for a variant:

  1. From one of the item browses, click to add a new item.
  2. In the Variant field, select the variant on which the item is needed.
  3. Complete the remaining details of the item as normal and click .

    The new item is added to the list of items; the Variant column shows the variant it belongs to.

To remove an item for a variant:

  1. From one of the item browses, select the standard item you do not want on the variant.
  2. From the Action list, select Create Variant Omit Item and click .
  3. Select the variant from which you want to omit the item.
  4. Click .

    The list of items shows a new line, which is a copy of the standard item with the quantity and value reversed; this effectively cancels out the standard line. The Variant column shows which variant it belongs to.
