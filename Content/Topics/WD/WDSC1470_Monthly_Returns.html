Monthly Returns

The procedure for electronic monthly returns involves two stages:

  • processing the month-end return using Process Month End Return
  • submitting the month end return electronically using the Submit action menu item (Submit - part of Monthly Return Submissions)

Because the person who submits the return must confirm that the information is correct and declare that they have:

  • considered the employment status of each subcontractor and
  • verified all new subcontractors with HMRC,

you may wish to grant access to the Submit function accordingly.  You may also wish to require users to enter the Government Gateway password to confirm these declarations; to do this, leave the SCUK/PASSWORD parameter blank (see Gateway Password).
