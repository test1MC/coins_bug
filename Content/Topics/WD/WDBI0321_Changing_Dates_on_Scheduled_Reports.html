Changing Dates on Scheduled Reports

If you have a saved or scheduled report that includes selections for dates or periods, you can choose how you want to handle the dates when the report is run.  For each date, you can use the date you entered, or any of a list of date offsets (dates that are calculated relative to the run date).

To change the dates for a saved or scheduled report:

  1. On the Report Status Saved or Scheduled tab, click the button next to the report whose dates you want to change.
  2. For each date choose one of the following:
    • Leave to use the date you entered (this is shown in brackets after the date field).
    • One of the other options to use the appropriate date offset.  The date offsets are set up separately using Date Offset.
