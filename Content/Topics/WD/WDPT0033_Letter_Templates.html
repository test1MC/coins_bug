Letter Templates in Project Management

Letter templates represent the standard text for letters related to a particular item, such as , meeting minutes or .

To create a letter template:

    1. Create a document in Word. At the points where you want to insert data from , type the name of the database field, enclosed in braces. For example:

 

Construction Industry Solutions Ltd.

Building

11 St Laurence Way

Slough

SL1 2EA

{pt_recipient.ro_ptr_name}
{pt_recipient.ro_ptr_company}
{pt_recipient.ro_ptr_add__1}
{pt_recipient.ro_ptr_add__2}

{pt_recipient.ro_ptr_add__3}
{pt_recipient.ro_ptr_add__4}

 

{ptg_date}

 

Dear {pt_recipient.ro_ptr_salutation},

 

 

  1. Save the document as an RTF file.

    NOTE

    If you used Word to create the document, after you have saved it as a .rtf file open the .rtf file in WordPad and save it. This removes unwanted hidden characters that can cause the merge to fail.
  2. Go to Letter Templates. 
  3. Use the selector at the bottom of the browser to select the template type.
  4. Enter a code and description for the letter template.
  5. the template line and use the Upload Letter Template File action to upload the RTF file you created.

NOTE

You can create more than one template for each type; when the user runs the Generate Letter action, they can choose which template to use.
