Report Writer - Producing the Report

To run an existing report:

  1. Go to Report Writer.
  2. Find the report you want to produce and click the link in the >Report column.
  3. From the Options menu, select Run Report.
  4. Choose how you want to produce the report:
    • If the report requires selection criteria, you can select which data to include on the report. For example, if there is a selection tab, you can choose which to report on, according to the different fields that are available on the tab.
    • Choose the type of output you want. For example, you can choose to display the report on your screen as soon as it is ready (foreground). You can choose to email the report to someone, or to have it generated at a later date or time.
  5. Click the [Forward] button.

You can also run the report at any time while you are writing it by selecting Run Report from the Options menu.
