Web Set Up

The Web uses a hierarchical arrangement of groups to help users search for products.  You need to set up the hierarchy of groups, and assign products ( groups and items) to the most detailed groups.

Each group or product can have a large image and a smaller thumbnail image associated with it.  The large images of products are shown on the Add To Basket and Update Basket screens; the thumbnails are shown on results lists.  If no thumbnail is available, uses the large image (resized appropriately).  You can set up a default image that is used if a product doesn't have an image.

You can associate documents, such as specifications or safety instructions, with groups and products; these are available for users to view, both on the results lists and on the Add To Basket and Update Basket screens.

To create the groups:

  1. Go to Hierarchy.
  2. Add a new record for each group.

    We suggest you use a structure for the group codes; for example, two characters for the first level, three characters for the second level, and so on. 

  3. In the Group field, enter the code for the parent of the group you are adding.  For groups at the highest level, leave this blank.
  4. Specify the filenames for the image (currently this is only used if the thumbnail is not specified) and the thumbnail (shown on results lists), and enter a full description (shown when viewing details of the group).
  5. Save the record.

To add products to groups:

  1. Go to Assign Hierarchy.
  2. Use the Component Type selector to choose whether to add assets or , and click .
  3. In the Group field, enter the code for the group that the product belongs to. 
  4. Specify the filenames for the image (shown when viewing details of the product) and the thumbnail (shown on results lists), and enter a full description (shown when viewing details of the product).
  5. Save the record.

To associate documents with groups or products:

  • In Hierarchy or Assign Hierarchy, click next to the group or product with which you want to associate the document.  See Attaching a Document.
