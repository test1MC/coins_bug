Configurable House Sales Statements

allows you to configure the format used for printing statements in order to print to your business requirements and use pre-printed stationery.

You define the layout of statements using Document Designer. See Configurable Documents for general information on how to configure documents for printing.

You need to set up a configurable document for each different type of statement you want to print (for example: internal and external statements).

To define the layout for printing statements:

  1. Use the document type /Options Statement.
  2. When the statement is printed, you can configure to include an event automatically in the sales history.  Enter the event code and any comments.
  3. The statement has four BODY-like sections and four corresponding subfooter sections.  Set up a form for each of these, using the following hard-coded names:
    • EXTRA - to print the that have been ordered on the plot.
    • EXTRABRK - to print the totals for .
    • OPTION - to print the that have been ordered on the plot
    • OPTIONBRK - to print the totals for .
    • INCENT - to print the incentives that have been ordered on the plot.
    • INCENTBRK - to print the totals for incentives.
    • CASH - to print the details of payments made.
    • CASHBRK - to print the totals for payments.
