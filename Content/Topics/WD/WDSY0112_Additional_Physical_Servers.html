Additional Physical Servers

Each additional physical server needs the following:

  • Progress Client Networking
  • FOP
  • Printers (for direct printing)
  • Access to the application on the physical server
  • Scripts to start a report agent and run FOP

For ease of maintenance it is strongly recommended that the application be shared from the main server via NFS or similar file-sharing technology. This will ensure that any updates or fixes are automatically made available to additional physical servers. The additional physical servers need to be able to write to the spool directory on the main server.
