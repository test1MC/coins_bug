Configurable Accident Forms

The Accident Form document type allows you to configure documents for printing accident forms.

Define the layout of accident forms using Document Designer in the System Module. See Configurable Documents for information on how to define configurable documents.

To configure accident forms:

  1. Use the document type Accident Forms (H&S).
  2. Configure the BODY form to print details of any witness records. This uses the table pp_orgtype.
  3. Configure an additional form (give it a name on the Extras tab) to print any images linked to the accident record using Document Storage. GIF, TIFF and JPG files will be printed, PDF will be appended. Use the field RO_ExtrFile^Ext.

Accident forms use the following tables:
hr_incident - the accident record
pp_accidents - association of the accident record and injured person with injury details
pi_people - H&S record of the person on site
pp_orgtype - witnesses (this is the body record)
