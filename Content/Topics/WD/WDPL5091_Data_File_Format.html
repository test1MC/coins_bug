Electronic Invoice Data File Format

 

The following file format is used by Load Electronic Invoices.

Fields must be separated by spaces. Alphanumeric fields must be enclosed in quotation marks (" "). Any blank alphanumeric field must be ""; blank numeric fields must be set to 0; blank date fields must be set to ?.

Header Line

Each invoice must begin with an invoice header line. The header line must be preceded by a line containing only the letter H.

The invoice header line must contain the following fields:

  • Invoice date
  • Invoice received date (if PL/ENTERREC = Y)
  • Gross amount
  • Source ledger code
  • Transaction type
  • amount
  • Due date
  • Discount date
  • Discount
  • Description
  • Hold code, reason
  • P O number
  • Distribution type (R/P/C)
  • , Department, Sector
  • Currency Code
  • Internal reference

If CO/TAXTYPE=SALES, an additional line containing the following header fields is also required:

  • Merchandise Amount
  • Freight Amount
  • Sales Tax Code
  • Sales Tax Amount
  • Consult PL Hold Code parameters (Y or N: if this is Y, the parameters HLDREGIN, DEFHOLD, DEFHCODE, DEFFINH and DEFCODEF will be consulted and the relevant hold codes set).

Cost Distribution, Distribution, Invoice Lines and Intrastat Lines 

Cost distribution lines, distribution lines, invoice lines and optional Intrastat distribution lines then follow in any order.

Invoice cost distribution lines must be preceded by a line containing only the letter D, and contain the fields:

  • Amount
  • Analysis type (1 - , 2 - , 3 - General Ledger, 5 - Workshop)
  • Analysis (for example, /cost code/category, GL account)
  • Summary (YES/NO)
  • Description
  • PO Number  
  • Sales Tax code
  • Quantity
  • Units

 

Distribution lines must be preceded by a line containing only the letter V, and contain the fields:

  • code
  • amount
  • Goods amount
  • Summary (YES/NO)

 

Invoice lines must be preceded by a line containing only the letter L, and contain the fields:

  • Part code
  • Part description
  • Quantity
  • UOQ
  • Price
  • Per (UOM)
  • Discount
  • Amount
  • reference
  • Delivery date
  • rate

 

Intrastat distribution lines must be preceded by a line containing only the letter I, and contain the fields:

  • Mass
  • Value
  • Commodity code
  • Summary (YES/NO)
  • Date
  • Transaction nature
  • Delivery terms
  • Transport method

 

Each invoice must end with a line containing only the letter E.
