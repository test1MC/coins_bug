Monthly Status Report

In COINSplus this is %SC754 (sc754.p).

Function parameters:

SDTYPES <taxation status> (for checking whether Statements of Deduction have been printed):
G = Gross
N = Net at lower rate
H = Net at higher rate
For example, if you are not issuing statements for gross payments, use SDTYPES=N,H
GROUP Default list of groups (as set up in Company Group Maintenance).  To run this for the current (kco) by default, omit this function parameter. 

(This is not yet implemented in OA.)
