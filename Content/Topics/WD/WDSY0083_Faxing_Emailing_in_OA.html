Faxing and Emailing in OA

All output options screens have a Fax option which uses VSI-Fax to change the PDF to a TIFF image file and fax that.

You can also select Email to send the document to an email address.

See Selecting the Output Options.

In addition, you can send PL and SC remittances, SC certificates and SC statements of deduction to the recipient automatically.

Configuration

/ SC Ledger and Subcontractor set-up: Default Print method can be set to allow for VSI-Fax to fax remittances via fax or email.
