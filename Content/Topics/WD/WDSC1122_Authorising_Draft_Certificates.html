In level 2 authoration, the draft certificates that have been 'Released' in the entry screen are shown (their status is marked as 'To Authore 2').  In level 3 authoration, the draft certificates that have been passed by level 2 authoration are shown (their status is marked as 'To Authore 3').  A certificate passed by level 3 authoration is marked as 'Approved' and is ready to be batched and posted. These will only appear on the authoration screen if the Authored selector is set to Include, so they can be recalled to level 3, if necessary.

NOTE

If the NOTIERS parameter is set to 2 then the level 3 authoration is skipped, and a draft certificate authored at level 2 will go straight to 'Approved' (that is, "awaiting batch creation").  The same thing happens if you has both authority levels set.

Any draft certificates that are rejected will be marked as status 'Attention required' and will appear back on the entry/amend screen so they can be revised. They will disappear from the authoration workbench and can't be recalled.

An authored draft certificate can be recalled to a particular level, as long as it hasn't been further authored or rejected.

To authore a certificate or certificates:

 

Delegated Authorities can be established where more than one person is involved in the creation and approval of a Subcontract Draft Certificate.  Once the Certificate has been released it can be reviewed before authorisation.

A Certificate will not appear as a cost until it has been posted by the Finance Department.  Therefore the Certificate must be authorised to allow posting to costs and then payment when due.

This section will demonstrate how to review and authorise a Subcontract Draft Certificate.

Select Draft Certificates > Authorisation

 

‘%’  Remember - The Draft Certificate Authorisation workbench is unique to the person logged in

    Select the TXN hyperlink for the Certificate

    Select all Tabs and review the certificate details.

    Approval/Rejection and Hold options can be actioned from this screen.  Select the Options drop down list.

    Approval and Rejection will present a comments box for you to add any information regarding your decision. 

    Once entered, click . This authores each of the selected records if the certificate is the earliest one for the order at the 'Awaiting Level 2 Authoration' level, otherwise gives an error.

‘%’  Note - that you are only able to reject a certificate from the Options drop down menu and not from the Authorisation Workbench Actions menu

    Approvals may also be actioned directly from the Draft Certificate Authorisation Workbench.

‘%’  Note - that just as in printing certificates you are able to select and authorise multiple certificates in one action

    Select the box next to the Certificate(s)

    From the Actions drop down list select Authorise Draft Certificates , and Apply Action

As noted above, only approval is available when using this method, rejection must be done within the Draft Certificate Enquiry Options menu
