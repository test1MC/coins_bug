Customing Help Text

provides a screen that shows help text for each function and the fields on that function. In addition to the text that is provided with , you can add your own help text for functions and individual fields. This allows you to enhance the help to address the needs of your users, or to provide help for user-defined fields such as analysis sets. This "user text" is displayed with a different ed background, to differentiate it from the standard help text (you can modify the appearance of the user text). User text will not be overwritten when you upgrade .

To provide your own help text for a field or function

  1. In the Address box at the top of the documentation window, type &helpMode=edit at the end of the URL, and press Return to refresh the window.
  2. Move the mouse pointer over your profile picture and in the user menu, click Toggle Help.

    The help frame now has editing buttons.

  3. Click next to the nugget you want to add the text to.
  4. Click the User tab.
  5. Enter the user text.
  6. Click .

To custome help using Help Maintenance

  1. Identify the help nugget you want to change.
  2. Go to Help Maintenance.
  3. Search for the help record for the field or function.

    NOTE

    If both the Field and the Function are specified, the text is function-specific; that is, it only appears for that field on that function.
  4. Open the record.
  5. Enter your text in the fields on the User Text tab:
    • The User Text field is shown with the text that is normally shown.
    • The Additional Text field is shown when the user clicks the More... link.

      NOTE

      Consider giving display-only access to the standard help text fields (syw_ttext1, syw_ttext2 etc), and only allowing users to update the user text fields (syw_ttext3 and syw_vtext3).
  6. Save the help record.

To modify the way the user help text is displayed

  1. Edit the cascading style sheet file docuser.css.
  2. Add a style declaration for div.usertext.

For example, to change the background to pale yellow, you could enter

div.usertext{background-color:#ffffdd}
