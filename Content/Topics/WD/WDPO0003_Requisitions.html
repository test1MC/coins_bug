Configurable Purchase Requisition Documents

allows you to configure the format used for printing purchase requisitions in order to print to your business requirements and use pre-printed stationery.

You define the layout of purchase orders using Document Designer. See Configurable Documents for general information on how to configure documents for printing.

You need to set up a configurable document for each different type of requisition you want to print (for example: material requisition, requisition, requisition).

To define the layout for printing purchase requisitions:

  1. Use the document type "Purchase Requisition".
  2. Configure BODY forms to print the details of requisition lines.
  3. There are special forms for printing text lines and clauses. Set up:
    • a BODYTEXT form to print text lines. Use the field RW_rql_FullDescription for the text.
    • a BODYINDENT form to print indented clauses. Use the field rql_desc__1 for the clause text.
