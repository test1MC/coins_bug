Functions

In a function can represent different things:

  • A menu.
  • A page.  Usually this corresponds to a page with the same code as the function.
  • A feature within a page, such as a button, option or tab folder, or a specific bit of functionality.

Each of these can be combined with function permissions to control whether a user can access the corresponding feature.  For example, a user might have access to the reports menu, but only have access to some of the reports on that menu.  Or a user might have access to the function, but not be allowed to delete , so they would not have access to the function for the delete button on that page.
