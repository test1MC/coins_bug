Uncommitting a Purchase Order

You can only uncommit a purchase order (for example, if you need to cancel a committed order) as long as it has not had goods received against it. You cannot uncommit an order if you have committed a cancellation order for it.

To uncommit a purchase order:

  1. Go to the order entry screen.
  2. Select the order number (or orders) you want to uncommit.
  3. From the Choose Action list, choose Uncommit Purchase Orders.

    produces a report that lists the orders that have been uncommitted.
