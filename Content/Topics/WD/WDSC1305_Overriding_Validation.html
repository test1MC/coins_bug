Overriding Validation for a Subcontractor

Each  has standard validation rules set up for it.  These determine the actions for which checks the  details, and whether issues a warning or prevents you entering the transaction.

You can override the validation for an individual subcontractor.

To override validation for a subcontractor:

  1. Go to the  workbench.
  2. From the Options menu, select - Overrides.
  3. Open the record for the subcontractor and policy combination you want to change.
  4. Make the changes to the validation settings.
  5. Click .

  1. Go to the workbench.
