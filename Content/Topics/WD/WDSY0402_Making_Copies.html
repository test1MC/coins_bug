Making Copies of Functions

If you want to make changes to a standard function (one with a code that begins %), you can make a copy of the function (with a code that begins +).  Whenever a user selects the standard function, the corresponding "+" copy will be run instead. 

You can also copy standard pages, reports and menus in the same way. So if you make a "+" copy of a standard page and change it (for example to add a field), when a user selects the standard function that calls the page, will show the amended page.

NOTE

As an alternative procedure, you can create a menu that is specific to your needs and consisting of certain functions relevant to that menu without making + versions of functions. This is discussed separately in Adding to Standard Menus.

Do not change standard (%) functions, pages or reports.

NOTE

If any element on a page has a 'plus' version that overrides the standard % version, displays a small + after the function description in the page banner. 

The 'plus' version could be a function, tab, menu, page, form, etc.

To copy a standard function:

  1. Go to Function Maintenance.
  2. the function you want to copy.
  3. Click .
  4. Replace the % in the function code with +.
  5. Click
You can then make changes to the "+" copy.

Using the standard function

If you add &helpMode=stdpage to the URL, will run the standard % function rather than the amended + version.  You may need to do this before raising a support call, to check that the issue can be reproduced on the standard pages or reports.
