GL Account Structure

As part of General Ledger configuration, you need to set up the structure of the accounts.

To define the structure of General Ledger accounts:

  1. In Configuration, click the Account Structure tab.
  2. In the Name row, enter the names of up to four elements.
  3. Under each element name, in the Length row, enter the number of characters in the element.

     

    The total of the lengths plus the separating s must not be greater than 19.
  4. Under each element name, in the Integer row:
    • If the element is to consist of digits only, the box.
    • If the element can contain any alphanumeric character, leave the box blank.
  5. In the Category field, enter the number of the element (1,2,3 or 4) which is to be linked to costing categories.
  6. In the Type field, enter the number of the element (1,2,3 or 4) which is to hold the General Ledger account type (Asset, Liability, Capital, Expenditure or Revenue).
  7. In the Fixed at field enter the number of the element that accounts are fixed at. If you have configured General Ledger to generate accounts automatically, you can use fixed elements to prevent creating unwanted accounts. See Fixed element maintenance.
  8. Click .

This procedure defines the structure for General Ledger account codes and the names of the elements.

When you have finished setting up the account structure, you need to define the valid components and codes for each element. For example, an element called Division might contain four components: North division, South division, East division and West Division. To define the components of the elements, use GL Elements and Fixed Elements.

You can then generate the accounts themselves. See GL Accounts.
