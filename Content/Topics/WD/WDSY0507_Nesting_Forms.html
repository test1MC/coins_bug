Nesting Configurable Document Forms

 

You can include one form in another. This allows you to: 

  • Apply formatting (for example, borders and shading) to several fields together.
  • Define blocks of fields that you can arrange relative to each other.
  • Define blocks of fields or text that you can use more than once (for example, on both a header and a continuation header).
