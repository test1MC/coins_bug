Costing Distribution

It is now possible to create detailed costing distribution from the order lines. There are a number of pre-conditions:

  • Parameter PO/SCWBS must be set to L (WBS at line level)
  • Parameter SC/DISTBD must be set to Y
  • Parameter SC/IDORIG must be set (enter ID to create breakdown created from order lines) and optionally parameter SC/IDVAR.
  • Certificate definition lines with the ID specified in SC/IDORIG above must have breakdown box ed.
  • This Payment value on the breakdown item is non-zero.
  • No distribution lines exist for the certificate.

A distribution line will be created for each non-zero breakdown item which has been generated from the purchase order (for example, where the Item column isn't blank). The costing code will be taken from the order line and the amount will be the This Payment amount.

There may be differences between the total of the distribution created from order lines and the certificate costed value (cost line 1), such as for additional breakdown items, discounts, retention, etc. If so, another distribution record is created for the difference. This is costed to the first default costing on the subcontract. Certificate definitions allow for 2 separate costed amounts (and cost codes) so it is possible to structure the definition appropriately.
