Adding User-Defined Text to an Existing Topic

provides a screen that shows help text for each function and the fields on that function. In addition to the text that is provided with , you can add your own help text for functions and individual fields. This allows you to enhance the help to address the needs of your users, or to provide help for user-defined fields such as analysis sets. This "user text" is displayed with a different ed background, to differentiate it from the standard help text (you can modify the appearance of the user text). User text will not be overwritten when you upgrade .

For instructions on how to provide your own help text for a field or function, please refer to Customing Help Text on the OA Online Documentation Menu.
