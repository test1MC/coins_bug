Format C (Bank of Ireland)

The first two lines in a Bank of Ireland statement file are header lines, which Load Bank Statement ignores. The remaining lines each contain a single string with data at specific character positions. The character positions of the fields that Load Bank Statement reads are as follows:

1-6
7-14 Account number
16-17 Transaction type
36-46 Transaction value (in pence)
65-82 Description
If the transaction type is 02, the transaction is a , with the number in the right-most positions of the description field.
102-106 Date in Julian format (YYDDD where DDD is the number of the day in the year)

Lines with transaction types U1,u7, U8, U9 44 and 54 are memorandum lines which Load Bank Statement ignores.
