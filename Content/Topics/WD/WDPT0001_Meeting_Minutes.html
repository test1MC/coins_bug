Configurable Meeting Minutes

You can configure a document to print the minutes of a meeting recorded in Project Management.

Define the layout of the document using Document Designer.

A meeting minutes document is made up of the following sections:

 

Header Information about the meeting, organer, etc. 
Next Meeting Header Column headings.
Next Meeting One record for each next meeting (usually there will only be one).
Attendees Header Column headings.
Attendees One record for each meeting attendee.
Copies To Header Column headings.
Copies To One record for each recipient.
Body Header Column headings.
Body One record for each meeting item.
Footer Further information about the meeting, space for signatures, etc.

To configure meeting minutes:

  1. Go to Document Designer.
  2. Use the document type Meeting Minutes (PT). 
  3. On the Extras tab, enter names for each of the sections (as shown in the table above) that you want to use.
  4. See Configurable Documents for general information on how to configure documents for printing.
