Using Overlays When Faxing From COINSplus

Reports and other documents printed from often use overlays to print fixed details such as logos and headings.  These overlay files can't be passed to VSI-Fax in the same way as to a printer.  Instead, VSI-Fax uses a graphic TIFF file for the overlay.  This needs to be placed into the overlay directory on the VSI-Fax server, together with a ".dat" file to reference to use the correct TIFF file.

There are several ares of configuration which must be completed to ensure that both and VSI-Fax have the necessary information to attach the correct overlay to the fax.
