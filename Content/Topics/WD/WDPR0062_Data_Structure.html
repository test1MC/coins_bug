Data Structure

A may have employees. Each employee belongs to a Tax Scheme. There can be multiple Tax Schemes in one , or more than one can use the same Tax Scheme.

An End Of Year Return will be for a single Tax Scheme and Tax Year. For a Scheme and Year, there can be more than one Return, as you can have test and live returns.

The data on the employee form is a snapshot of data held elsewhere. We hold this snapshot for performance reasons (it is quicker) and to allow historical reporting.

Each Return will be validated and may have Validation Errors, which are problems that detects in your data. A Return may have a number of Submission Attempts, each of which will have its own Protocol Events and possibly Submission Errors (errors reported by the Gateway).
