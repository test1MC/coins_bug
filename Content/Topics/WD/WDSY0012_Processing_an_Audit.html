Processing an Audit

Processing an audit (also called "creating an audit") assigns the next audit number to a set of audit records (that is, records of changes). You should process audits regularly. When you generate audit reports, you can choose to report on an individual audit.

To process an audit:

  1. From the Audit - Processing menu in the relevant module, select the option for the table for which you want to process an audit.
