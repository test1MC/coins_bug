Setting Internet Explorer to Allow ActiveX Controls

Web uses ActiveX controls for displaying CUI reports, and for displaying the table of contents in the on-line documentation.

To allow ActiveX controls:

  1. In Internet Explorer, select Internet Options from the Tools menu.
  2. Click the Security tab.
  3. Click the Local Intranet zone (or the zone you have assigned the URL and Website to).
  4. Click Custom Level.
  5. In the Security Settings :
    • If you are using Windows XP:
      1. Scroll down to Download unsigned ActiveX controls and click Prompt.
      2. Scroll to Initiale and script ActiveX controls not marked as safe and click Prompt.
      3. Scroll down to Run ActiveX controls and plug-ins and click Enable
    • If you are using other versions of Windows:
      1. Scroll down to Run ActiveX controls and plug-ins and click Enable

  6. Click OK.
  7. At the warning prompt, click Yes to confirm that you want to change the security settings for this zone.
  8. Click OK to close the Internet Options .

If you are using Windows XP, when you first try to run the online documentation, Windows will display a warning.

To install the ActiveX and display the Online Documentation, click Install.
