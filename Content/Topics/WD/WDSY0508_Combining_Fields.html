Combining Configurable Document Fields

You can append one data field to another, using a calculation. For example, you might want to print all the address fields on a single line, or print the number and name as if they were a single field.

To enter a calculation:

  1. Add a new line.
  2. Leave the Field field blank.
  3. Enter the calculation in the Calculation field. Put a $ before each part of a calculation that involves text strings; use the full table and fieldname.

    4. For example:
         $a=jc_job.job_num; $b=" (" + jc_job.job_name + ")"; $a + b
    will display the contract number, followed by the contract name in brackets.
