Entering Journals

Journals are used for transferring money between different analyses.  They are available in each of the main workbenches. 

To enter a journal:

  1. Select the type of journal you want to enter using the Add Type drop-down, and click .

    displays a header screen, which also shows the lines that the journal will contain. 

  2. Update the header if necessary and click .

    displays the journal lines for you to update. 

  3. Update the journal lines.  Note that the journal must balance; debits must equal credits.
  4. Click .

    displays the journal browse. 

  5. If you have finished entering the journal details, select the journal and choose Complete Journals from the action list.

After you have completed the journal, it may need to be approved (authored).  This can be done either from the same workbench or using Approve Journals, by selecting the journal and choosing Approve Journals from the Choose Action list.

Then use Journal Batches to create a batch that contains the outstanding journals, and post the batch.
