Forecast Snapshots

In order to run a number of the management reports, allows you to save the current forecast information as a snapshot.  This will normally be done once a month.  The information is then available to the reports.

To process a snapshot:

  1. Go to Snapshot Processing.
  2. Select the appropriate snapshot type.
  3. Enter a list of the inspection types (for example: white card, yellow card) that you want to include in the snapshot.
  4. If you want, use the Selection tab to choose which information you want to include.
  5. Click .
  6. If Snapshot Processing has already been run for the snapshot you selected, displays a warning:

    • To replace any information already in the snapshot with the current information, click OK.

      produces the snapshot, and also generates a report that confirms the snapshot has been produced.

    • To leave the snapshot as it is, click Cancel.
