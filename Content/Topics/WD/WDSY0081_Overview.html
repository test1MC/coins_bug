Faxing and Emailing Overview

  • In , you can send purchase orders, PL remittances, SC remittances and certificates, and Expenses remittances as faxes or emails, depending on the recipient's preferences. 
  • In you can send any print output to a fax number or email address.  You can also fax or email PL and SC remittances, SC certificates and SC statements of deduction to the recipient automatically - again, taking account of the recipient's preferences.  An optional review stage allows users to preview and reject individual fax or email messages.

For faxing, uses a third-party program, VSI-Fax.  also uses VSI-Fax for sending emails.

How it works

FAXING

  • In , VSI-Fax converts the data, merging it with overlay images if appropriate, to generate a TIFF file which it passes to the fax device.
  • In , the report generator produces a PDF file which it passes to VSI-Fax.  VSI-Fax then converts this to a TIFF file which it passes to the fax device.

  • In , VSI-Fax converts the data, merging it with overlay images if appropriate, to generate a TIFF file.  It then uses its sendmail option to pass this to the SMTP server.
  • does not use VSI-Fax for emailing.  The report generator generates a PDF file which is passed directly to the SMTP server.
