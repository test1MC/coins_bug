Customing the Documentation

The standard documentation that provides is generic and may not be suitable for your needs.  You can modify the documentation in different ways.

  • You can add user text to each of the sections.  This is highlighted and shown below the appropriate section.  Use this if you want to add notes or comments to the standard text that is already there.
  • You can create a user-defined copy of a topic, and change the text.  When a user selects the topic, they will see the modified version of the topic.
  • You can write your own topics, and add them to your own menus, or to user-defined copies of standard menus.
