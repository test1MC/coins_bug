Cost Categories

The cost category allows you to costs further into up to six categories, such as , materials, and so on. Each cost category can be any combination of up to two letters or numbers. The cost categories apply across the whole .

Cost category configuration

You define the cost categories as part of the JC Configuration procedure, and define default categories for standard in Standard .  If you do not use all six cost categories initially, you can add them at any time. If you need to change or delete a cost category code which has had transactions entered against it, you must consult Construction Industry Solutions for advice.
