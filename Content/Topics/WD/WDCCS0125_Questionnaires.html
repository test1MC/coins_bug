Questionnaires

Questionnaires in allow you to record the responses of purchasers to questionnaires you have set up in advance.

You can use Questionnaire Analysis Report to produce an analysis of the responses.

To record responses to a questionnaire:

  1. Open the plot record in {Customer Care} Workbench.

  1. Click the More > tab then the Questionnaire tab.

  1. Click to add a new set of questionnaire responses.

  1. Select the questionnaire from the list, and click .

    2. displays the questions from the questionnaire.

  1. Fill in the responses.

    2. Some questions may be optional; others may be required (you must enter a response). When you try to save the responses, if there are required questions you have not entered a response for, will display an error message telling you which questions you need to answer.

  2. Click to save the responses.

If you need to change the responses, you can go back to the Questionnaire tab and edit the responses; click .
