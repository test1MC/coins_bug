Desktop

The Desktop shows important information at a glance, and allows you to run functions.  The functions you use most can be grouped into sections on the screen, and a function search allows you to search for other functions.  

Desktop

The desktop is configurable; if you have the necessary permissions you can choose what to include on your desktop.  

NOTE

The Desktop and the button are only available if a desktop has been set up for you.  If no desktop is configured, you will be able to choose a homepage.

The sections below explain more about how to use the Desktop

Tiles

The main area of the desktop contains customable tiles, each of which runs a function.  The tiles are arranged in sections which you can show or hide with a mouse click, and the sections can be grouped into tabs, which you can select using a link in the header area of the desktop.

Some tiles are "smart tiles" which show summary information about the function (such as financial values or the number of items requiring attention).

 

On small smart tiles that show values, a tooltip explains what each number represents; move the mouse pointer over the number to show the tooltip. For more information, see Smart Tiles.

Some tiles can show information as a chart:

Click to refresh the chart.  For more information, see Chart Tiles.

Showing the Desktop 

If a Desktop has been set up for you (and if you don't have a user home page set up), you see the Desktop when you log in to .  If you run a function in the main area and want to show the Desktop again, click the Show/Hide Desktop button (), or press Ctrl+Shift+!.

Running a Function

To run a function from the desktop, click its tile.  The function runs in one of three different ways, depending on how the tile has been configured:

  • The desktop is hidden and the function screen is displayed in the main area.
  • The function opens in a new tab.
  • The function opens in a frame.
