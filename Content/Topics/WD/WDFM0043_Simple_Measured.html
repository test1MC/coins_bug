Simple Measured Without SOR

Simple measured use an application and certification process. Generate and print a pro-forma with the amount you are applying for, which you send to the client. You then confirm the value to be invoiced.

Simple measured can either use schedule of rates items or not.

To process a simple measured invoice:

    1. Enter the application value:
      1. In the Workbench, click the button.

        The Financial Summary tab shows the financial information for your reference.

      2. Enter the value you are applying for.
      3. Click .
    2. Use Generation to generate the pro-forma invoice.
    3. Use Print to print the pro-forma.

      Once printed, this needs to be sent to the client.

    4. When the client responds, enter or confirm the actual value:
      1. In the Workbench, click the button. On the Actual Value tab, click the link in the Certify column for the pro-forma you want to confirm.

        displays the Confirm Value to be Invoiced - Update screen.

    5. Do one of the following:
      • If the client has certified the valuation, select Fully Certified as the Type. If the client has approved a different value, enter that value.
      • If the client has rejected the valuation, select Cancel as the Type.
      • If there are further pro-formas to be entered for the , select WIP as the Type.
