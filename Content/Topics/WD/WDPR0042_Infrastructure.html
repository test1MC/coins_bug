Infrastructure

The exact requirements will vary slightly from server to server and environment to environment, but in broad terms the requirements are as follows:

  • Your network must be connected to the internet (that is, you can connect to the internet from a PC on your network)
  • Your server must:
    • Have Java 1.3 (+ security add-ins) or Java 1.4 installed. If you do not, installation of this will require a system re-boot.
    • Have DNS installed
    • Have a Digital certificate installed to access the Government Gateway

uses Java to make the secure connection from your server to the Government Gateway, via the Internet. In setting up the connection, the Government Gateway issues a Digital Certificate (DC), which verifies it is the Government Gateway. Java has to recogne this certificate as a "trusted certificate" for things to proceed.

Without going in to the technicalities, you do not need to trust the actual certificate, but you must have the appropriate root certificate of the Certificate Authority (CA) to verify the DC. The CA in the case of the Government Gateway is Entrust (web site http://www.entrust.net ).
