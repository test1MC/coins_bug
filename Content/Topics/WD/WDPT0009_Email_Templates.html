Email Templates in Project Management

Email Templates represent the standard text to be emailed for a particular source document, such as an . You need to create Email Templates for each type of document.

To set up email templates:

  • Go to Email Templates. 

  • Use the Selector at the bottom of the browser to select the Template type.
  • Click
  • Create email templates for each of the actions.

    • You can use field names in the Subject and Message fields; these are replaced with the appropriate data when the email is created. For example, if you use {tra_date} on a email, this is replaced by the date of the you are sending.
    • You can use the HTML editor to format the message text.

     

  • Specify the template name in the corresponding parameter so they are used automatically by .
