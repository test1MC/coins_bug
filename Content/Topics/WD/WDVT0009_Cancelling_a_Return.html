Cancelling a Return

If a Return within is incorrect and needs to be created again, it can be cancelled subject to the validation rules below:

  • If the posting status of the Return is 'posted' and/or the submission status is 'complete', the Return can only be cancelled by the sysadmin user on input of the day password). If the Return is 'posted' it can be unposted using:
    • Returns Workbench, Returns tab, Unpost Return to GL action or
    • Returns Workbench, any tab, Cancel Return option

    If the Return has not been submitted, this will allow it to be cancelled. If you need to cancel a Return which has been submitted, please contact the support desk.

  • If the Return does not have posted or complete status, users can cancel the Return subject to permissions set in the CB/VATIDCAN parameter.

Note that there is no facility for to send an amendment or cancellation of a submitted Return to HMRC. If a Return which has been submitted to HMRC needs to be amended or cancelled, you will need to contact HMRC.
