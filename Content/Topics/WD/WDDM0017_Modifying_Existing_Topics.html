Modifying Existing Topics

If you want to change an existing documentation topic, you can create a user-defined copy of the topic and modify that.  When a user selects the topic, they will see the modified version of the topic.

NOTE

If the standard documentation is changed in a future release, remember that your user-defined copy will not reflect these changes.

To create a user-defined copy of a topic:

  1. Go to Topic Maintenance.
  2. Using the tree view, browse to the book that contains the topic you want to change.
