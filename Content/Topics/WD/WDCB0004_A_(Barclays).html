Format A (Barclays)

Barclays statement files contain header (or summary) lines and memo lines (which are the transaction detail lines). Load Bank Statement only reads the memo lines. The format of memo lines is as follows:

  • ignored field,
  • ignored field,
  • account number,
  • MEMO,
  • bank sorting code (with separators),
  • currency,
  • transaction date,
  • ignored field,
  • transaction value,
  • bank transaction type,
  • number (if relevant),
  • ignored field,
  • description

The fields are separated by commas.

Barclays statement files include information for several accounts in a single file. Load Bank Statement only reads the information relevant to the bank and account you are processing.
