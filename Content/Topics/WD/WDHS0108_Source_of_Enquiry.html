Source of

Source of codes allow you to identify where a purchaser or first heard about the or part-exchange.

For example:

  • Localpap = Local Paper
  • TV = Television

This information will then be available to users when completing the relevant details in other House Sales procedures.

The source of can be a single level (one set of codes), or you can group the sources of into categories and areas. For example, categories could include "national press" and "local press"; the "national press" category could include as areas "The Times" and "The Guardian"; the area "The Times" could include "Times - Weekdays", "Saturday Times", "Sunday Times".

Set up

Use Source of Activities to set up the source of activities. 

If you want to use categories and areas, set these up using Source of Categories and Source of Areas.
