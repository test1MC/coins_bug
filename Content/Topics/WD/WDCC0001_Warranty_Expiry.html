Warranty

Plots for which the warranty has expired are marked as inactive.

You can filter out contacts on inactive plots from the {Customer Care} Workbench, and if a plot's warranty has expired, the plot summary screen shows a message: Note: Out of Warranty.

To mark plots with expired warranties as inactive, you need to run Process Warranty {Expiry} periodically.

Warranty - configuration

You need to set the warranty period and the basis for calculating when the warranty expires.
