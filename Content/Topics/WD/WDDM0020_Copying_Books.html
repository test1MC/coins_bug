Copying Standard Documentation Books

Online Documentation topics are arranged in books; these use the same menu system as the rest of , with books being the equivalent of menus, and topics the equivalent of functions.  You can create you own documentation books using Topic Maintenance; however, if you want to modify an existing standard documentation book, you need to create a user-defined copy of it first.

To create a user-defined copy of a documentation book:

  1. Go to Topic Maintenance.
  2. Using the tree view, browse to the book that contains the book menu you want to change.
  3. the book you want to change.
  4. Select Create + version from the action menu and click .

    In the farthest right-hand column, displays a against the book, to show that a user-defined copy of the book exists.

  5. Right-click the Topic Maintenance tree view and choose Refresh.  This updates the tree view and allows you to edit the user-defined copy of the book.
  6. Browse to the book you want to change.

    Notice the reminder "You are editing a + copy of a standard menu" above the list of topics.  You can add new or existing topics to the book menu, or remove topics; you cannot, however, edit the standard topics themselves - you need to create user-defined copies of them.
