Isle of Man C11A

The SCC11A (see also Isle of Man C11) is an Isle of Man (only) Tax Document.  It can be used to produce a substitute C11A form (that is, a substitute if that is agreed by the specific contractor with the Isle of Man authorities).   The C11A provides details of subcontractor's payments and tax deducted and provided a report to the Tax Authorities of the contractor's payments to subcontractors and tax deductions over a period.  Although the C11A is intended to be provided once a week, but since a contractor may have a specific agreement allowing them to produce it for a longer period, a weekly run is not enforced.

To configure an SCC11A document:

  1. Go to Document Designer in the System Module. 
  2. Use the SC Monthly C11A (IoM) document type.
  3. See Configurable Documents for information on how to define configurable documents.
  4. The following tables are available: sc_mast, sc_sc60, co_config.
  5. The SCC11A has the following forms: 
    • The BODY form is not currently used on this document. 
    • The heart of the document is the Taxes Per Period Form (by default PERIODTAX) which shows a number of RO fields: RO_PeriodNum, RO_Date, RO_C11Unique, RO_Taxable, RO_Cost, RO_Deduct, RO_PeriodTaxStr.  These are all populated by calculations in the print program itself rather than being available as RO fields for a particular table. These fields are summaries of the values from the sc_sc60 for the subcontractor for each particular period requested for the print (period being tax periods ending on the 5th of the month). 
    • The Total Taxes per Period (by default PERTOTAL) provides a total for all subcontractors on the report by tax period.  These are displayed using the RO fields: RO_LabourTotal, RO_MaterialTotal, RO_TaxTotal, RO_PeriodTaxTotal.
    • The Form Name Form (by default FORMNAME) simply provides the name of the Form as agreed with the Tax Authorities (typically Substitute C11A).
    • Totals Form (by default TOTAL).
    • The FOOT and CFOOT forms have a page count using the standard tokens.
