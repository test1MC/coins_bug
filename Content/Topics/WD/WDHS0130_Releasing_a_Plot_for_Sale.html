Releasing a Plot for Sale

 

Before you can sell a plot, it must be released for sale.

To release a plot:

Click the Options button and choose the event for releasing the plot.

Fill in the release details (such as the release price and any discount or incentive amounts), and choose whether to create a follow-up action (see Actions).

 

Field

Description
Release Date The date on which the event happened.
Release Price The gross release price
Level 1 DI Discount/Incentive value. The type of information in this field is specific to the event being run.
Net Release The net release price for the plot.
Reservation Deposit The reservation deposit required for this plot.
Show On Release List  
Comments Enter any comments about the event.
Follow Up? Select if you wish to enter a follow-up action for this event

Click .

If no standard or options have been set up against the plot, and if and options are supposed to be set at plot level, will warn you.

To return to the plot summary an add or options, click Cancel; to release the plot, click OK.
