Applying Price Changes In

If you update the price of resources, you can automatically re-apply resources to house type .

Changes to the master prices can be applied to items, standard house type bill items and house type bill items.

You can apply prices:

  • Using an Option in Prices Workbench which either runs a report process or applies the prices.
  • From the matrix views on the House Type Workbench, Surveyors Plot Workbench or Buyers Workbench, using selected rows (payment stages, packages, estimating sections or resources) for all or selected house types.

To update prices from the Price Workbench:

  1. On Prices Workbench, move the mouse pointer over Options.
  2. Select BQ Apply Prices to Standard House Types or BQ Apply Prices to House Types.

  1. You can produce a report (to assess the effect of changing prices of resources that have been used on a house bill), or commit the price changes.
  2. Specify the resources you want to report on or update; you can select anything from a single resource through to multiple resources or all resources.
  3. On the Select Prices tab, specify the type of prices you want to report on or apply.
  4. On the Select Bills tab, specify the bills you want to report on or apply the prices to.

    NOTE

    If you select Processing mode, this could, potentially, change prices across all the standard and non-standard house type bills!  You should make sure that you have made the correct selections.
