Order Import File Format

The following import format is used by Import Purchase Orders. The file must be in CSV format.

The first line of the file contains the field descriptions, which are for the user's benefit; the second line contains the field names (as below) and is required for the import. The following lines contain the order data to be imported.

The file can contain multiple orders, each with multiple lines. Each new order (and the first line of that order) is identified by a line with Y in the first column; subsequent lines on that order have N in the first column.

Column
Number		 Description Field Comments
1 New Purchase Order? .new

This field cannot be blank.

Y = The line is a new order.

N = The line relates to the previous order (identified by Y in the .new column)

Values in fields related to the order header are ignored on lines where this is N.
