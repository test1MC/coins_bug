Project Management Item Numbering

Item numbering is controlled by various parameters.

The table below gives the parameters which control automatic numbering for each item along with some sample values and the format produced by these values.

Item Parameter Description Sample Value Sample Format
Drawing Sets DRTAUTO Drawing Set Reference code to be automatically generated Y SET0001
DRTCODE Characters to precede Drawing Set Reference code SET
DRTFORM Drawing Set Reference code Format 9999
Drawings DRWAUTO Drawing Reference code to be automatically generated Y DRW0001
DRWCODE Characters to precede Drawing Reference code DRW
DRWFORM Drawing Reference code Format 9999
ISSAUTO Reference code to be automatically generated Y ISS0001
ISSCODE Characters to precede Reference code ISS
ISSFORM Reference code Format 9999
Letters LTRAUTO Letter Reference code to be automatically generated Y LET0001
  LTRCODE Characters to precede Letter Reference code LET
LTRFORM Letter Reference code Format 9999
RFIAUTO Reference code to be automatically generated Y 0001
RFICODE Characters to precede Reference code
RFIFORM Reference code Format 9999
Items SNGAUTO Reference code to be automatically generated Y PUN0001
SNGCODE Characters to precede Reference code PUN
SNGFORM Reference code Format 9999
Packages SPKAUTO Package Reference code to be automatically generated Y PAC0001
SPKCODE Characters to precede Package Reference code PAC
SPKFORM Package Reference code Format 9999
SUBAUTO Reference code to be automatically generated Y SUB0001
SUBCODE Characters to precede Reference code SUB
SUBFORM Reference code Format 9999
TRAAUTO Reference code to be automatically generated Y TRA0001
TRACODE Characters to precede Reference code TRA
TRAFORM Reference code Format 9999
Items TSKAUTO Reference code to be automatically generated Y TAS0001
TSKCODE Characters to precede Reference code TAS
TSKFORM Reference code Format 9999
