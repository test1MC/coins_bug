Mail Merge

Some browse screens allow you to produce a mail merge file from records that you select. These screens have a merge button in the button bar.

Depending on what has been set up for the screen, this means you can:

  • Merge data with a Word document (for example, to produce a personaled letter).
  • Send personaled email messages.
  • Create a spreadsheet (CSV) file that you can use as a mail merge data source.

To create a generic mail merge:

  1. the records you want to include in the merge.
  2. Click Merge.

    displays a list of the records you selected.

  3. You can review the list and remove records by deselecting them.
  4. Choose a mail merge document from the Choose Merge drop down menu.
  5. Click Apply Action.

    A window appears containing:

    • Your selected records merged into a Word document, or
    • The emails you have sent, or
    • A spreadsheet with the records you selected.
  6. If appropriate, you can save the Word document or spreadsheet.
