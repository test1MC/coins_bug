Examples of Schedules

Report Schedules allow you to run reports (and other processes) at regular intervals. There are three basic types of schedule: daily, weekly and monthly. These give you a lot of flexibility in setting up schedules that suit your needs.

Use Standard Schedule to set up the schedules.

Below are some examples.

When to run the report

How to do it

Every 10 days.   Use a Daily Schedule. Set the Every field to 10.

Every Monday and every Thursday. 

 Use a Weekly Schedule. Set the Every field to 1. Monday and Thursday.
Every four weeks on a Friday.   Use a Weekly Schedule. Set the Every field to 4. Friday  
Every three months on the 15th of the month Use a Monthly Schedule. Set Occurs to Day Set Occurs to Day 15. the relevant months (for example, March, June, September and December).
The last day of every month. Use a Monthly Schedule. Set Occurs to Day 31. (31 represents the last day of the month, regardless of the actual number of days in the month.) all the months fields.
The first weekday of the month. Use a Monthly Schedule. Set Occurs to First Weekday. all the months fields.
The last Friday of the quarter Use a Monthly Schedule. Set Occurs to Last Friday. the relevant months (for example, March, June, September and December).
The second Wednesday of every month. Use a Monthly Schedule. Set Occurs to First Wednesday. all the months fields. Set the Offset field to 7.
