Format J (Lloyds Bank)

There is no separator in the file format, that is, fixed position fields.

In a line:

Characters

Field

1 to 6

7 to 14

Account

16 to 17

Transaction type

36 to 46

Value in pence

47 to 82

Description (for , the number is in the right-most positions of characters 65-82)

102 to 103

Year in yy format

104 to 106

Day of the year (that is Feb 1st = 032)

When using LloydsLink to export statements for bank reconciliation, export the statement in APACS format - this matches format J.
