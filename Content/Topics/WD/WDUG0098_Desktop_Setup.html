Setting Up Your Desktop

If a blank Desktop has been set up for you, there is a link to the Desktop Setup Wizard at the top of the screen.

The Desktop Setup Wizard allows you to set up the layout of your Desktop (which tabs and sections it contains) and which functions to add to it.

NOTE

You can also run the Desktop Setup Wizard from a link on the User Preferences screen.

To set up your Desktop:

The wizard consists of a sequence of screens:

  1. On the first screen, choose an existing Desktop to copy. This is used as the 'starting point' for your Desktop. Desktop Wizard

    • If you select an existing Desktop, creates a copy of that Desktop, which you can then modify (without affecting the original).
    • If you select the Desktop that has your own user ID, deletes that Desktop, and creates a new, empty desktop for you.
    • If you leave this blank, creates an empty desktop for you.
  2. On the Sections screen, name the tabs you want on the desktop, and the sections you want on each tab.  Enter at least one section for each tab. Sections Screen

  3. On the History screen, select the functions to add to the Desktop. The browse shows a list of functions. History Screen
