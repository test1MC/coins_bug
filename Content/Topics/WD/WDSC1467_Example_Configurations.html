Example Configurations

Examples of working form/document configurations are available to download from the Client Area of the website.  You can then import these into your environment and amend them to suit your requirements or use them as guidelines for configuring your own.  We recommend that you import them into a non-live environment first.

To import a configuration file

  1. Open the file.
  2. Select the entire contents and copy them.
  3. Go to Import Document Definition under the Report Setup menu).
  4. Paste the contents of the file into the Report Data field.
