Putting Invoices for an Order On Hold

You can put all invoices for an order on hold.

To put invoices for an order on hold:

  1. While you are entering the order, click the Hold tab.
  2. Set either (or both) the Financial Hold Code or Commercial Hold Code field, if available, to the hold code you want to use for invoices raised against the order. 

    If you select a hold code, all invoices raised for the order will inherit that hold code, and the person who enters the invoice will not be allowed to change the code.  The invoices can still be approved using standard approval functions.
