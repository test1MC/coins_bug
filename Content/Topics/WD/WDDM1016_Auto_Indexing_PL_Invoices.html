Auto Indexing PL Invoices

Auto indexing (using Index from Pool and Index from Batch) allows you to index scanned PL Invoice batches and automatically create PL invoice records.  creates an invoice batch in , with an invoice record for each document you index. The invoice batch has a status of incomplete.

The scanned document images are displayed in a separate window, allowing you to enter the indexing details in the window. If you have two monitors, you can display the document image on one and the input screen on the other.

When you save the indexing details for one invoice, you move on to index the next invoice, and automatically displays the corresponding invoice in the image window.

To auto-index invoices:

  1. Go to Index from Pool or Index from Batch.
  2. Click the link for the pool or batch you want to index.
  3. Select the folder where you want to store the indexed documents.
  4. Click the link for PL invoices.

    opens the first invoice in a new window.  will create a new invoice batch in (the batch number is shown at the top of the indexing screen, and the batch description will be 'OA DM Batch').

  5. Leave the Auto Index box ed.
  6. Enter a description for the invoice.
  7. Click .

    displays the next image and prompts for the description of the next invoice.

  8. If the new image is a continuation of the previous invoice, un- the New Invoice field; will attach the image to the previous invoice document. If you New Invoice, creates a new invoice record and a corresponding new invoice document.
  9. Continue entering indexing details for each image in the pool or batch.

    When you have indexed all the images in the pool or batch, it disappears from the list.

    10. NOTE

    If you leave any of the documents un-indexed, you can select the same pool or batch for indexing later, but will create a new batch.

To complete the invoice details:

  1. Go to Enter Invoices.
  2. Select the invoice batch. 

    When you open each invoice record, displays the corresponding image.

  3. Fill in the invoice details for each invoice.
