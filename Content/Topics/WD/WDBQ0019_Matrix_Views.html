Matrix Views

 

 

The matrix views (on the House Type Workbench, Surveyors Plot Workbench or Buyers Workbench) allow you to view and update data according to various breakdowns.  For example:

  • House types by payment stage
  • Plots by material package

They present the information in rows and columns (similar to a spreadsheet), with the columns in each case being the main entity on the workbench (for example, plots or house types), and the rows being the components of the breakdown (for example, package or payment stage). 

The matrix views allow you to:

  • View by value or by status.
  • Apply actions (for example, to update budgets or change a status) to whole rows or to selected elements within the matrix. 
  • Dig down to view or update the items that make up each element.
