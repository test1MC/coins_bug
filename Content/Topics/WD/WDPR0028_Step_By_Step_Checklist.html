Step By Step Checklist for Creating and Submitting an In-Year Return

Eight steps are required to create and submit an in-year return from . The following checklist can be copied and used each time to guide you through the process. This process needs to be followed for each tax scheme for which returns are to be made.

Done Step Action Comment
1 Ensure that all necessary Payroll input for s and s has been completed. will automatically add these to the next incomplete return.
2 Select the return to work with (see Selecting the Return to Work With).
3 Review the return information for each employee on the In Year Movements workbench (see Form Details).
4 Input the additional information required on forms on the In Year Movements workbench as required (see Details Which Need To Be Directly Input to Forms).
5 Flag the return as Complete (see Completing the Return and Resolving Validation Errors).
6 Resolve any validation errors if necessary (see Completing the Return and Resolving Validation Errors).
7 Submit the return (see Submitting the Return).
8 Check that submission has been successful (see Submitting the Return).

The In Year Movements workbench function (%PR738) can be found on the Payroll Global menu.
