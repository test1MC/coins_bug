More Than One in a Scheme

In the SC/LINKCOIR parameter enter a list of the numbers you want to treat as one scheme.

Enter a list of linked numbers, and separate each group of with a pipe character.

For example:
    1,2,3|5,6|7

links 1,2 and 3 together, and 5 and 6 together.  7 is not linked.  If a is not listed, it will not be linked. 

This allows you to link for reporting and submission separately from the CO/LINKCOMP list of linked , which are linked for maintenance.  If LINKCOIR is blank, uses the list from LINKCOMP.

Payments will take the contractor's details from the parameter settings in the main .  The main in each case is the first in the comma-separated list.  The main is also referred to as the submission .

NOTE

It is important to ensure that LINKCOIR and LINKCOMP are NOT set as -specific parameters in any of the involved.
