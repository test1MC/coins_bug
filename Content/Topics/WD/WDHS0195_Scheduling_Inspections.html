Scheduling Inspections

You can update the forecast inspection dates for an individual plot.

To enter inspection forecasts:

  1. Go to Inspection Workbench.
  2. Choose the , then the plot you want to enter (or update) the forecasts for.

    displays the Plot Summary.

  3. Click the Forecast Dates tab.

    This shows any inspection types that are relevant for the plot.

  4. Update the forecast dates using the standard maintenance buttons.

    If the forecast dates for any of the other inspection types are set up to default from the inspection type you updated, updates the forecast dates for those inspection types.

To enter inspection bookings:

  1. Go to Inspection Workbench.
  2. Choose the , then the plot you want to enter the inspection booking for.

    displays the Plot Summary.

  3. Click the Inspections Booked tab.
  4. Click to add a new inspection record.
  5. Select the type of inspection, and the Booked status (or the status that corresponds to booked).
  6. Fill in the other details and click .

Inspection forecast configuration

Use Inspection Types to set up the types of inspection you require, and which types of plot they relate to (according to the sales classification). By setting an offset, you can have update the forecast dates from when either the forecast date for the previous inspection changes, or the previous inspection is passed.
