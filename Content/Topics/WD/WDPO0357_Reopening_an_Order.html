Reopening an Order

If you have closed an order (that is, matched invoices to it) and then need to enter more goods against that order (for example, you received fewer goods than you ordered but closed the order anyway, then received the remaining goods), you can re-open the order.

To reopen an order:

  1. Go to the order entry screen.
  2. Select the order to be reopened.
  3. From the Choose Action list, choose Re-open Purchase Orders.

You can now enter further goods received against this order.
