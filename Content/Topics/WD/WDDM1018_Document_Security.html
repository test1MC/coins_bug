Document Security

You can restrict access to documents so that users can only see documents that belong to the categories assigned to them.

To set up document security:

  1. Use Document Group to set up groups.
  2. Set up which groups each document category belongs to:
    1. Go to Category Maintenance.
    2. Click the link in the Description column.
    3. Use the Document Groups tab to add groups to the category.
  3. Use Document Group Security to specify which document groups each user is allowed to access.
