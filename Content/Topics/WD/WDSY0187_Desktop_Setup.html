Setting Up Desktop Features

The Desktop gives users a customable screen from which they can run a selection of functions. This page explains how to set up various features that are available on users' desktops.

Tile Values

Some functions have been enhanced to allow you to show certain values on the tile. You can use Desktop Tile Values to set up calculations that will provide additional values. For more information, see Desktop Tile Values. 

Tile Options

You can set up alternative for the links on Desktop tiles (for example, to open a specific tab or a named filter). These are then available as options that the user can select when configuring the tile. Use Desktop Tile Options to set up these options.

Global Search

Global Search allows users to search for records in the database (for example, suppliers, customers, subcontractors or contracts), and also to find and run functions. The search is configurable, so you can choose which tables, and which fields on those tables, to include in the search. See Global Search Configuration.

Desktop Images

The Desktop features a background image (which each user can choose from a list of available images) and an account image (such as a photograph of the user). Two separate functions allow you to load the images to your system:

  • Background Images allows you to load images that can be used as the desktop background. Individual users can then select one of these to use on their desktop. The file name you give to the file is used in the Background Image selection list in Users. The images are saved in the $BASE/ custimage/bg/ directory. Standard images, provided in the $BASE/ images/bg/ directory, are also available to users.
  • User Images allows you to load the images that are used as the account picture on each user's desktop. The image files must be JPG format files, and should be approximately 60px square. (You can use larger images but the browser will resize them, which does not always produce good results.) You must name the files <userid>.jpg (for example, for a user whose user ID is JSMITH, the file would be jsmith.jpg. The images are saved in the $BASE/ custimage/ sysuser/ directory.

    If there is no file for a user, the account image will be the image from the user’s HR record if one exists, otherwise it will be a silhouette image.
