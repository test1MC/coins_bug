Loading Statements Automatically

You can load bank statements electronically using a data file supplied by your bank, for automatic or manual reconciliation. To load statements automatically, the account must be set up to accept statement files. See Loading Bank Statements for details of the formats for each bank.

To load a bank statement:

  1. Go to Auto Bank Statement Load.
  2. Select your bank.
  3. Enter either:
    • The date of the oldest transaction on the statement (if you are behind on your statement loading).
    • Yesterday's date (if you are working on an ongoing basis).
  4. Enter a reference to allow you to identify the statement.
  5. Browse for the statement file on your PC.
  6. Click .

loads the data from the input file.

Report Status shows the result of the statement file load.

You can use Bank Statements to amend the statement.

Loading statements automatically - configuration

The Statement Type field in Bank Accounts determines which statement file format the account is set up to accept.
