SC Self-Billing Agreement

The Subcontractor Self-Billing Agreement document format allows you to configure a document which is intended to constitute and detail the agreement between the client and a subcontractor as regards the issuing and accepting of self-billed invoices. 

Use Subcontract Self Billing Agreement to print these documents.  They are printed after the selection of a and particular subcontractors who have subcontracts for that . 

To configure a self-billing agreement document:

  1. Go to Document Designer in the System Module. 
  2. Use the SC Self-Billing Agreement document type.
  3. See Configurable Documents for information on how to define configurable documents.
    • The following tables are available:
      • jc_job -
      • sc_mast - Subcontractor
      • co_config -  Configuration
    • All information should be in the header or footer; the BODY is not currently used on this document. The HEAD typically contains data about the client from co_config and about the subcontractor from sc_mast (and could contain information about the specific from jc_job); for example: name, number, name and number.  The FOOT contains provision for the signatures of both parties.
