Changing Lookups

provides the option of having a lookup (activated by clicking or pressing Ctrl+Shift+L) on each field that has a table of validated data. Normally, the same field in different situations will have the same lookup. If a lookup has been specified for a field, then wherever that field occurs, the lookup will be available. also automatically makes the lookup available for "report selection" versions of the field. For example, if a lookup exists for avm_num, then report selections that use the field "RS_avm_num" will also have that lookup. It is possible to modify the lookup, or to specify a different lookup in particular situations.

To change a lookup:

  1. Go to Lookup Maintenance.
  2. Add or modify the record for the field you want to provide the lookup for (in the example above it would be avm_num).
  3. To provide different lookups for the same field on different functions, enter separate records with the function code in the Function field. If the Function field is blank, uses this as the default lookup for the field.
  4. Enter or change the lookup function. This is the function to call to provide the lookup data. This could be a standard browse or function, or one specially written for the lookup. The lookup is %WPL5100BAVM.
  5. If necessary, enter the parameters for the lookup.

    NOTE

    The lookup settings are issued as part of standard data.  Be aware that if you make changes to the setttings for standard (%) functions, these will be overwritten when is upgraded.
