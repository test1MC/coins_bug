Sending Notifications Using the Notifications Report

The Notifications Report allows you to send email notifications and produce mail merge documents for auto- notifications.  Notification records will have been created automatically by , or added manually; see Auto- Notifications.

There are different versions of the report, one for each pay frequency.  The report could be scheduled or run manually.  Employee selection is available, and employee security is applied.  You can use My Files to access the documents it creates.

Alternatively, you can use the mail merge feature on the Notifications browse to send emails and produce mail merge documents.

To produce notifications:

  1. Select the notification type and merge file(s) to use. 
  2. In the Stamp Date Sent As field, assign a date to all the notifications that are processed.  If left blank no date will be assigned.  
  3. Use the Employee Selection tab to restrict the employees; for example if you want to send different notification messages on the same notification type. Notification Report Selection Screen

The report content gives details of who was included and the method of notification employed.
