Types of Documentation Represented in OA

You should be aware that different types of online documentation may be represented for menu books and topics on the OA Online Documentation Menu. These include:

  • Existing (standard) - A menu book or topic that is already established in the OA online documentation and is intended for all users. Its function code is preceded by a %.
  • Plus Copy (non-standard) - An existing menu book or topic in the OA online documentation that is copied and modified by users. It appears in place of the existing menu book or topic and is available to all users on the same system. It is called a "plus" menu book or "plus" topic because the function code that identifies it is identical to the existing function, except it is usually preceded by a + (plus) instead of a %.
  • User-defined (new) - A menu book or topic that is created by a user. It also encompasses user-defined text added to existing topics and appearing in addition to existing text. It is not a copy of an existing menu book or topic. User-defined documentation is available to all users on the same system.

NOTE

Documentation added through the creation of plus copy menu books and topics will not be overridden when a fix or new OA software release is uploaded. Also, the topics appear only in the environment where they are created.
