Creating a Plus Copy Menu Book

Follow the steps below to create a plus copy of an existing menu book. An example of how the treeview changes is shown below, with modifications indicated in orange text.

Before Change After Change
%MenuBook1 %MenuBook1 (Existing Menu Book 1)
%MenuBook2 +MenuBook2 (Plus Copy of Menu Book 2)

How to Create a Plus Copy Menu Book:

  1. In the System module, go to Topic Maintenance. Select a menu book that you want to copy; for example, select "How to Setup Report Servers and Queues" under Online Documentation > System Guide > Printing (see Figure 1).

Figure 1: Selected Menu Book

  1. Select Create + version at the bottom of the screen (see Figure 2) and click Apply Action .

Figure 2: Create + Version

NOTE

A + appears at the end of the row to show that you have created a plus copy of this menu book (see Figure 3).

Figure 3: Plus (+) Sign

  1. Refresh the treeview by right-clicking it and choosing Refresh.
