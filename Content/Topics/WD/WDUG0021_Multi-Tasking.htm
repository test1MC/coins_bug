Multi-Tasking

If you are in the middle of one activity and need to do something else -- for example, if you are creating a supplier record and want to enquire on the payment history of a related supplier, or if you are entering a batch of transactions and want to print a report -- you should open up a new window.

When you have finished the new activity, you can return to your original activity and continue.

To start up a new window

On the current tab of your browser, right-click the tab and select Duplicate

Another tab will open in which will allow you to navigate and use without affecting the screen you have open in the first tab.

 

Using this method does not create another session for the user (therefore has no impact on the number of in use). It only allows the user to multi-task within their current session. Logging out in any tab will close the session for all tabs.
Duplicate tabs may be closed without logging out - the first tab will remain logged in.
