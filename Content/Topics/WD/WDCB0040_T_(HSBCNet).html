Format T (HSBCNet)

Standard CSV format.

The first field – Account number/Sort Code – is a combined field consisting of bank sort code (6 digits) followed by account number (8 digits) separated by a hyphen. For example: 123456-01234567

1st Field Account Number/Sort Code (combined)
2nd Field Transaction Date
3rd Field Currency
4th Field Amount
5th Field Description
6th Field Transaction Type
