Inspection Management Configuration

You need to set up the reporting calendar, which determines the end dates for each of the reporting periods.

  • Use Reporting Calendar Types to set up a calendar type.
  • Use Reporting Calendar to set up the periods and period-end dates for the calendar type.

You need to set up the status codes that will be used to identify whether an inspection has been booked and the results of inspections, and the to use on the management reports for each status.  Use Inspection Status Codes.

Use Inspection Types to set up the different types of inspection that are required.  Each type of inspection can be set up to be a given number of working days after the previous type.  Specify which forecast dates are required for each type, and which plot sales classifications the inspection applies to.  For example, this allows you to set up Blue Card inspections that are forecast for 10 working days after the forecast date of White Card inspections, and only apply to 'housing association' plots.

Specify which users can update blank and existing forecast dates, using the parameters HS/UBLNKLST and HS/UXSTLST.
