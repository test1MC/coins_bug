Unretire Old Codes

If you have previously retired codes that use the rates that are due to come into force, you will need to un-retire them.  If you previously changed the description of these codes to "Old Standard Rate" or similar, then you will also need to update the description as appropriate.

You may also wish to update the description of the codes you currently use to "Old Standard Rate" or similar. For clarity it may be advisable to ensure that the % rate is included in the description.

To un-retire codes:

  1. Go to Codes.
  2. Click against the code you want to un-retire.
  3. Edit the description if necessary.
  4. Click the Inactive  box so that it no longer shows a .
  5. Click to save the update.
