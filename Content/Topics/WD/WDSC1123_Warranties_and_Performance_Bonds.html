Subcontractors Certificates - Warranties and Performance Bonds

Against each subcontract, you can record whether warranties and performance bonds are required and have been received.

If either of these are required (set to R), but not yet received, it will not be possible to approve a payment certificate. If not required the flag is set to N. If required and received it is set to Y.

An error is given if any of these checks fails, and the draft certificate can't be authored.

On Inactive orders, only retention release is permitted.
