Home Pages

 

There are two kinds of home page in .

  • Module home pages, which are shown whenever you change to a different module.
  • The user home page, which is a function that is shown when you log in to .  Each user can choose their own user home page.

Module Home Pages

Each module menu has a home page, which shows useful links and information that is relevant to the module. 

Information shown on the home pages may include:

  • Module text.  For example, general text to explain the scope and purpose of the module.
  • Environment text.  To identify the environment you are using (for example, Live or Copy of Live).
  • Message board.  Messages can alert you to planned downtime, upgrades, input deadlines, new procedures, configuration updates, etc.
  • Key functions and documentation for the module.  Shortcuts to selected functions and documentation topics within the module (or related modules).
  • User-specific ite functions.  You can choose the functions to include on your view of the home page. These ites are available on every home page.
  • RSS feeds.  You can choose which news headlines you want to see.  The home pages only show the three most recent feeds from each channel.

User Home Pages

You can choose which function shows when you first log in.

To set your user home page:

  1. Go to the function you want to set as your home page.
  2. Move the mouse pointer over your profile picture and click  Set Home Page.
  3. Click OK to confirm your choice.

Next time you log in, will display this function.  To change your home page, just set it again as above.

To clear your user home page:

  1. Move the mouse pointer over your profile picture and click User Preferences.
