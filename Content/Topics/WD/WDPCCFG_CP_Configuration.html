CP Configuration

To set up Cash Profiles, you need to:

  • Use the GL parameter CPJRNLS to identify which General Ledger transaction types are Cash Profiles transactions.
  • Use the CP parameter CPROFILE to turn on Cash Profiles.
  • Set up global () interest rates and, if necessary, interest rates for individual .

You may also need to enter or create initial balances for existing by using Cash Profile Recalculation and Entering Take-on Balances.

If you want to post cash journals from Payroll, you must also set the PR parameter CPROFILE to Y.
