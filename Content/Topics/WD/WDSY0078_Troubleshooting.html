Troubleshooting Internet Explorer Settings

Below are some suggested solutions to problems involving Internet Explorer settings.

Lookup window not resizing

Problem: lookup windows open in a fixed-size window, which means that you are unable to use the advanced filter.

Solution:

  1. From the Internet Explorer Tools menu, select Internet Options.
  2. On the Security tab, click the zone to which the OA site belongs (we recommend that this is the Trusted Zone).
  3. Click Custom Level.
  4. Scroll to the Miscellaneous heading.
  5. Ensure that "Allow websites to open windows without address or status bars" is set to "Enable".
  6. Click Yes to confirm you want to change the settings for the zone.
  7. Click OK again to close the box.

Bulk update not working

Problem: When you click the button, the spreadsheet with the selected records does not open.

Solution:

  1. From the Internet Explorer Tools menu, select Internet Options.
  2. On the Security tab, click the zone to which the OA site belongs (we recommend that this is the Trusted Zone).
  3. Click Custom Level.
  4. Scroll to the Downloads heading.
  5. Ensure that "Automatic prompting for file downloads" is set to "Enable".
  6. Click Yes to confirm you want to change the settings for the zone.
  7. Click OK again to close the box.

Internet Explorer secondary windows too small

Problem: Secondary windows (such as the documentation window or windows you open using right-click + "Open in New Window") are not the size you want.  This is often because IE has "remembered" the size of the last window you closed.

Solution:

  1. Close all instances of Internet Explorer except for one.
  2. Right-click a link in the page and select: "Open in New Window".
  3. Close the first browser window using (upper right corner).
