Configurable Drawing Set Cover Sheets

You can configure documents to print cover sheets for drawing sets recorded in Project Management.

Define the layout of the document using Document Designer. 

Drawing set cover sheets are made up of a standard header, body and footer, with the body showing one record for each drawing in the set.

To configure drawing set cover sheets:

  1. Go to Document Designer.
  2. Use the document type Print Drawing Set.
  3. See Configurable Documents for general information on how to configure documents for printing.
