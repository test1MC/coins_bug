USA Payroll EEO Report

 

The USA Payroll EEO report is printed using the Payroll Document (PRDOC) document type.  This report is required to be in a specific format and is not intended to be configured by users.  It gathers data from Payrolls that have been processed in , summares this data into a temporary table, then generates the document from the temporary table.

Configuration details:

The BODY is a summary of Payroll data by trade and skill level, taken from the sy_temp table.

The following extra fields are available:

  • RO_coadd1, RO_coadd2, RO_coadd3 - These are the address lines formatted to USA Address standards.
  • RO_FedID - Federal Tax ID Number.
  • RO_StartPer - Payroll period starting date the report is run for.
  • RO_EndPer - Payroll period ending date the report is run for.
