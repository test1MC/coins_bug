Adding a User-Defined Menu Book to a Plus Copy Menu Book

Follow the steps below to add a user-defined menu book to a plus copy menu book. An example of how the treeview changes is shown below, with modifications indicated in orange text.

Before Change After Change
%MenuBook1 %MenuBook1 (Existing Menu Book 1)
%MenuBook2 +MenuBook2 (Plus Copy of Menu Book 2)
          %Topic1           %Topic1 (Existing Topic 1)
          %Topic2           %Topic2 (Existing Topic 2)
          UserBook1 (User-defined Menu Book)

How to Add a User-defined Menu Book to a Plus Copy Menu Book:

  1. In the treeview, highlight the plus copy menu book to which you wish to add a user-defined menu book; for example, highlight "How to Setup Report Servers and Queues" (see Figure 1).

Figure 1: Highlighted Plus Copy Menu Book

  1. The message "You are editing a + copy of a standard menu" appears at the top of the screen (see Figure 2).

Figure 2: Plus Copy Message

  1. Select New menu book from the drop down menu at the bottom of the screen, then click Add  to add a new user-defined menu book to the plus copy menu book; for example, add "UserBook 1" to the plus copy of the "How to Setup Report Servers and Queues" menu book (see Figure 3).

Figure 3: Add New Menu Book

  1. When the next screen appears, enter a function name in the Code field; for example, enter UserBook1 (see Figure 4).

NOTE

Do not begin the code with a + or % sign.

Figure 4: Code and Menu Description fields

  1. Enter a Menu Description for this new user-defined menu book; for example, enter User Book 1 (see Figure 4).
  2. Click Save  to save the user-defined menu book.
  3. The new user-defined menu book should appear on the menu list (see Figure 5). You can now add new user-defined topics to this menu book, as discussed in Adding a User-Defined Topic to a Plus Copy or User-Defined Menu Book.

Figure 5: New Menu Book on List

  1. Refresh the treeview by right-clicking it and choosing Refresh.
