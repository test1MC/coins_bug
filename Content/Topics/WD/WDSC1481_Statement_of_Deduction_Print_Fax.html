Statement of Deduction Print/Fax

Print:

In COINSplus this is %SC753 (sc753.p).

In OA this is Print Statements of Deduction (scf7531.p).

Program: wou005
Parameters: FOPprog=scf7531

FAX:

In COINSplus this is %SC753A (sc753.p).

Function parameters:

FORM <form code>
* = Select at runtime
Code of form to use.

TAXTYPES

<taxation status> (for checking which payments to include on Statements of Deduction):

G = Gross

N = Net at lower rate

H = Net at higher rate

For example, if you are not issuing statements for gross payments, use SDTYPES=N,H
MODE

<run type>

PRINT = Generate printed document.

FAX = Generate and submit fax.

In OA, faxing is an output option on Print Statements of Deduction (scf7531.p).
