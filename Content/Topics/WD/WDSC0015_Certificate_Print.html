SC Certificate Breakdown - Certificate Print

The OA Certificate Print allows printing of breakdown items. The breakdown items are printed as an appendix at the end of the certificate (one appendix for each detail line with a breakdown).

Setup is done in the OA Document Designer. There are a number of fields for the appendices:

Field Notes
Appendix IDs List of IDs for certificate detail lines with breakdown items. An appendix will be printed for each of these.
Appendix Header Forms List of form codes, one for each ID in the Appendix IDs list.
Appendix Body Forms List of form codes, one for each ID in the Appendix IDs list.
Appendix Footer Forms List of form codes, one for each ID in the Appendix IDs list.
Appendix Rows List of rows (how many body records can fit on a page), one for each ID in the Appendix IDs list.

In the example below, two appendices will be printed at the end of the certificate. The first one is for detail line with ID GROSS. This will have a header form of HEADGROSS, 36 rows of body form BODYGROSS and a footer form of FOOTGROSS.

Each appendix has a header form, a body form, and a footer form. The following tables are available for use in each of these forms:

Form Tables
Header sc_certificate, sc_subcon, sc_mast, jc_job, co_config, sc_defn,
sc_defndet, sc_certdet
Body As above, plus sc_certitem, po_item, po_line, po_hdr
Footer Same as header
