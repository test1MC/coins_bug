If a Currency Joins the Euro

if a currency that you work with joins the Euro, you will need to:

  • Specify the fixed exchange rate between that currency and the Euro.
  • Convert /customer accounts.

Specifying the Exchange Rate

The exchange rate between the Euro and other currencies indenting to join the single European currency is fixed on the joining date. has a table to hold these exchange rates.

If a currency is intending to join the single European currency, enter that currency in this table. After the joining date, uses these rates and uses the triangulation rules for conversion.

To set up Euro exchange rates:

  1. Go to EMU Currency Conversion Rates.
  2. Enter a line for the currency, including the date that the currency will join the Euro and the conversion rate to convert one Euro to this currency.

    NOTE

    Once you have set up a currency in EMU Currency Conversion Rates, you no longer need to maintain the exchange rate for that currency - you only need to maintain the exchange rate for the Euro against currencies outside the Eurozone, using Currency Exchange.

Converting Accounts

If one of your or customers changes their accounting currency (for example from a European currency to Euros), you will need to set up a new account for them. All the outstanding transactions processed up to the point where you to set up a new account will be processed in the 'old' currency; all transactions processed after that time will be processed from the new account, in Euros.

The instructions below refer to changing a ; the same procedure applies to subcontractors and customers.

To change a from one currency to another:

  1. Identify the old account number.
  2. Go into Companies. Click the Company link. On the Offices tab, open the relevant office record.
  3. On the Accounts tab, enter a new Account number.
  4. Save the office record.
  5. Go to .
  6. Add a new account for the office. The details should be the same as on the old account, except that the Currency code should be set to the new currency.

If you make a mistake with the currency code, you can go back and change it provided that you have not yet entered an invoice against the . Once you have entered an invoice, the Account Currency field is ed out, and you cannot change the currency code on the account.
