A Note on Field Names

Any transitional record created for CIS tax and any record created for the new scheme will have the following pairs of fields, which hold the same data:

Usage New field Old field Previous usage
Type of subcontractor sc7_business sc7_otype Other type - not used
Subcontractor's UTR sc7_utr sc7_certno CIS certificate number
HMRC Trading Name sc7_tradname sc7_holder Certificate holder’s name
Tax treatment sc7_treatment sc7_type CIS certificate type
Verification number sc7_verno sc7_ocertno NI/
Verification date sc7_verdate sc7_from Certificate valid from
However, the new fields have not been added to the sc_check and sc_certificate records ? the new data is simply stored in the "old" fields on those records.  So for example to display the subcontractor's UTR on a certificate or remittance, you would need to use the field sc7_certno.
