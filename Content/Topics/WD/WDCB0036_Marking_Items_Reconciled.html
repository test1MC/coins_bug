Marking Items As Reconciled

If there are transactions which do not appear on your bank statement, you can manually mark them as 'reconciled', which stops the transactions from appearing in the list of unreconciled transactions. Normally you would only do this when you first start using reconciliation, or if you need to enter a correction (such as for a lost ).

To mark CB items as reconciled:

  1. Go to Manually Reconcile.
  2. Select the bank account.

    displays a list showing marked and unreconciled transactions. Marked transactions are shown with Manual in the final column; unreconciled transactions are show with Unreconciled.
