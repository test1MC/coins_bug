Loading Data From Other Sources

allows you to import transaction data from other applications.

  • Use Check Electronic Invoices to check the data file before loading it.

processes the file but does not load the data. If the check runs with no errors, you will be able to load the file. If there are errors, the file may load, but will not post it.

  • Use Load Electronic Invoices to load the invoices.

    loads the data file and creates a transaction batch

    If the data loads correctly, then moves the data file to the directory specified by the system parameter DATAOLD.

NOTE

If your system is set to run Sector Processing (as determined by the GL parameter SECTOR), the sectors that the transactions are associated with will be generated when this routine is run.
