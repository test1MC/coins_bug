GRN Exception Codes

GRN exception codes allow you to record warnings and other information against Goods Received Notes. displays these codes on the invoice matching screen.

To define exception codes:

  1. Go to GRN Notes.
  2. Use the standard line-based maintenance options to add, change or delete lines.
    Code A code of up to 2 characters for the exception condition.
    DescriptionText to describe the exception code.
