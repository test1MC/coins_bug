Format E (Royal Bank of Scotland)

Fields in Royal Bank of Scotland statement files are standard CSV format (comma-separated, quote delimited). The fields are as follows:

  • ignored field,
  • ,
  • account number,
  • transaction date,
  • transaction type,
  • value,
  • sign (+ or -),
  • description,
  • narrative,
  • ignored field

Load Bank Statement combines the description and narrative fields into the description field, and combines the value and sign fields. If the transaction type is a string of zeros, the transaction is a ; Load Bank Statement enters the transaction type as CHQ and takes the number from the description field.
