Adding a Task to a Call

When you add a new task, it is added to a call. A task must belong to a call, but not necessarily to an issue. You can allocate a task to more than one issue within a call. For example, the task may be to investigate several problems that were raised as part of the same call.

To add a task:

  1. Open the call details.
  2. Click the Tasks tab.
  3. Click to add a new task.
  4. Enter the details of the task, and click to save it.
