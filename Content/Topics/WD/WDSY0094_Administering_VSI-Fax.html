Administering VSI-Fax Using

Under System Administration is the option for Fax Administration; this menu contains the following options for the basic administration of VSI-Fax.  

Purge Fax Requests

See VSI-Fax manuals for purging Fax Requests. It is not recommended to run this option from at this point. (Note: This was a function used with fax software used previous to VSI-Fax). 

Fax Status Enquiry

This will return the current Fax Server status to screen.               

Enable Fax

It is not recommended to run this option from at this point. (Note: This was a function used with fax software used previous to VSI-Fax). 

Disable Fax

It is not recommended to run this option from at this point. (Note: This was a function used with fax software used previous to VSI-Fax).

Modify / Cancel Requests

Although previously used to modify fax requests, this option is now only an . See below for configuration of the Menu Parameters associated with the function.

Start Fax Scheduler

Starts the VSI-Fax Server.

Stop Fax Scheduler

Stops the VSI-Fax Server.

Send Text Fax

Allows a user to send a simple text based fax to any recipient from a standard menu.  See Sending a Text Fax From .
