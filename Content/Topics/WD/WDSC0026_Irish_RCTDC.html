Irish RCTDC Documents

The RCTDC is an Irish tax document.  It can be used to produce a substitute form, which must be completed and signed by the contractor and issued to the subcontractor with payment.  The form relates to subcontractor payments made through both the Subcontract Ledger and .

NOTE

Irish Revenue has stated that computer substitute forms must "be reproduced with the same details and printed on paper of the same or similar quality as that used by the company on their headed notepaper" and a sample must be submitted for approval.  The company name and logo should be reproduced at the top or bottom of the form.

RCTDC documents can be printed using the Print RCTDCs action on the following screens: Payment Batches, Manual Payment Entry, Payment Batches, Manual Payments.  It will not be possible to print RCTDCs if there are insufficient RCTDC numbers available.   Printing RCTDCs will assign a number to the and increment the next number in the block.

To configure an Irish RCTDC document:

  1. Go to Document Designer in the System Module. 
  2. Use the RCTDC (SC) document type.  provides a standard template RCTDC document, %SCRCTDC, which you can use as a starting point. 
  3. See Configurable Documents for information on how to define configurable documents.
  4. The following tables are available: sc_mast, sc_voucher, co_config.
  5. All the configuration for the RCTDC can be done in the HEAD form.   

Configuration

  1. Set SCIRE/RCTDCPRC to Y.
  2. Set up the tax groupings for using Tax Groups.  These should be kept in sync with the listed in SCIRE/LINKTAX.
  3. Record the RCTDC numbers from the Revenue using RCTDC Number Blocks.   Set the parameter SCIRE/RCTDCWRN to the level at which you want to receive warnings.
  4. Fields to record whether the subcontractor is an individual or a partnership, and the date of birth or the partnership reference number, are available in Subcontractor Maintenance.
  5. The parameter SCIRE/REGNO must also be populated in each with the tax registration number for that , as this is printed on the RCTDC document.
