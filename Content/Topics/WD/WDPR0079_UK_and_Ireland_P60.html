Configurable UK and Ireland P60 Documents

The document type P60 (PR) is used to print the UK and Ireland P60.

Use Print P60s to print the P60s.

To configure a P60:

  1. Go to Document Designer in the System Module.
  2. Use the P60 (PR) document type.
  3. See Configurable Documents for information on how to define configurable documents.
  4. The P60 document does not use a BODY form; set the Body Rows and Continuation Body Rows fields to 0, and set up all the fields in the HEAD form.
  5. Fields from the Employee table are available. For UK you can also use fields from the sy_table for the tax reference. For Ireland the co_config table is also available.
  6. Additionally there are a number of special fields:
    • RS_kyear__1 - the tax year.
    • RO_Gross - string version of the gross pay for USC/income levy rounded to the nearest Euro (post mid-year rate change for 2009). Ireland only.
    • RO_Levy - string version of the USC/income levy in cents (post mid-year rate change for 2009). Ireland only.
    • RO_GrossPreMay - string version of the gross pay for UCS/income levy rounded to the nearest Euro prior to the mid-year rate change (for 2009 only). Ireland only.
    • RO_LevyPreMay - string version of the income levy in cents prior to the mid-year rate change (for 2009 only). Ireland only.
    • RO_FirstName - the employee's first name only (Employee.fname may contain a middle name too).
    • RO_StartDate - string version of the employee's in "DDMMYY" format where this date is after 1st Jan in the tax year. Ireland only.
    • RO_LeftDate - string version of the employee's in "DDMMYY" format where the is in the tax year and after the 1st Jan. Ireland only.
    • RO_EmployerReg - the .
    • RO_EmployerName - the employer name. Ireland only.
    • RO_EmployerEmail - the employer's contact email address for P60 enquiries.
    • RO_PrintDate - string version of today's date in "DDMMYY" format.
    • RO_Week53 - "X" if this is a 53 week year; blank otherwise.
    • RO_TaxFlag - blank for UK or "D" (or "J7" pre 2011) for Ireland if there was a net tax deduction in this employment; "R" (or "H9" for Ireland pre 2011) if there was a net tax refund in this employment.
    • RO_USCFlag - "D" if there was a net USC deduction in this employment; "R" if there was a net USC refund in this employment. Ireland only.
    • P60^<field> - string version of any field from the P60 field list below. Decimal fields will have two decimal places. bDirector will be "D" if TRUE, blank if FALSE. bRefund and bRebate__* will be "R" if TRUE, blank if FALSE. bWeek1Basis will be "W" if true, blank if FALSE. Date fields will be in the format "DDMMYYYY" for UK and "DDMMYY" for Ireland. jStart and jStartDate will be blank if not in the tax year.
    • P60Date^<field> - any date field from the P60 field list below as a date so the format can be set on the form. jStart and jStartDate will be blank if not in the tax year.
    • P60Integer^<field> - any integer field from the P60 field list below as an integer so the format can be set on the form.
    • P60Decimal^<field> - any decimal field from the P60 field list below as a decimal so the format can be set on the form.
    • P60Logical^<field> - any logical field from the P60 field list below as a logical so the format can be set on the form.
