Financial Summary

Financial Summary Enquiry allows you to define the information that is shown on the Financial Summary tabs in Sales Workbench and Part Exchange Workbench.

To configure a summary financial :

Currently you need to set this up in
  1. Enter a code and a title for the .
  2. For each line on the , fill in the following fields on the Detail frame.
    ItemUse the lookup to select the item of financial information you want to show.
    DisplayEnter Y to display the item on the ; enter N to hide the item (for example if you want to use it in a calculation).
    ReverseEnter Y to display the reverse value of the item (for example, to show costs as negative values).
    LabelEnter the label that will be shown on the .
  3. For certain items, you need to fill in other fields:
    Item IDTo show the value from a HS Journal (using the JOURNAL item), use the lookup or enter the item ID to identify the journal.
    CategoryTo show the value of a cash transaction (using the CASH item), use the lookup or enter the transaction category.
    CalculationTo show the result of a calculation (using the CALC item), enter the formula for the calculation.
    CaptionTo show the value of a caption, use the lookup to select the caption group and the caption.
    If the item represents plot costs or revenue, you can enter a range of to include.
    RangeIf the item represents plot costs or revenue, you can enter a mask.
    CategoriesIf the item represents plot costs or revenue, you can enter a range of cost categories to include.
    RangeIf the item represents plot costs or revenue, you can enter a cost category mask.
  4. To improve the layout of the summary, you can enter blank lines (using the BLANK item) and underlines (using the UNDL item).

Financial Summary - Configuration

You need to add &summary=summary-code to the parameters for the page function %WHSSVWBFIN (Plot Summary - Financials Option). The default summary is PLOTSUM.
