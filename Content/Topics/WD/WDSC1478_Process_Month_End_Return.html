Process Month End Return

In COINSplus this is %SC750 (sc750.p).

In OA this is Process Month End Return (scr750.p).

Function parameters:

MODE <run type>
MEND = create submission records (COINSplus only)
CHKSUBBIES <Check for unallocated payments>
N = allows you to run it even if unallocated payments exist. Provided for testing purposes only. Not recommended for live running.
1STPERIOD <earliest period date>
This overrides CIS2DATE. Allows for testing and where customer does not start on 05/05/07. Payments after this date will be included.
