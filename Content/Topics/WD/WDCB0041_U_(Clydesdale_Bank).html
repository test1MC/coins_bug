Format U (Clydesdale Bank)

Standard CSV format

1st Field: Date.  Format: characters x(10) "yyyy/mm/dd"

2nd Field: Ignored field

3rd Field: /Account Number - Combined field x(14). First 6 positions for the ; next 8 for the account number

4th-6th Field: Ignored fields

7th Field: Description

8th Field: Value

9th Field: Narrative.  If Description = "", the narrative contains the number.
