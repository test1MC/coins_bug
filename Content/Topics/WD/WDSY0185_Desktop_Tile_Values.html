Desktop Tile Values

Smart tiles show values (such as the number of unposted batches or the value of payments received).  There are several standard functions that are set up to show values (see Smart Tiles).  In addition you can set up your own calculations to show values on specific functions.

To set up a function to show additional values:

  1. Go to Desktop Tile Values.
  2. Add a new record.
  3. In the Info Code field:
    • If you want the values to be available to an individual function, enter FUNCTION.<MainArea> where <MainArea> is the code of the function. 
    • If you want the values to be available as additional values on functions that are already set up to show values, enter the info code for those functions.  For example, batch input functions have the info code cosval.batches; if you enter cosval.batches, your new value will be available on all the batch input functions that already use that info code.  You can find the info code by looking at the parameters of the function.

    • If you leave Info Code blank, the new value will be available on all functions.
  4. In the Info ID field, enter a code to identify the value.  The combination of Info Code and Info ID must be unique.
  5. In the Description field, enter a description for the value; this is used as the label on the tile (and as the option that the user can select when configuring the tile).
  6. If you want to use a specific configuration table (other than co_config, sysparam and menuparm, which are available by default), enter it in the Configuration Tables field.
  7. Enter the calculation that will return the value.
  8. Save the record.
