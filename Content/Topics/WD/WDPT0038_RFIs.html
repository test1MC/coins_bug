There is a report ?  Log ? which shows details of each with the question and answer, and lists any items linked to the .

Log tab

The log allows you to track the activity related to a .  generates some log entries automatically when:

  • The is first entered.
  • The question is changed.
  • The answer is first entered.
  • The answer is changed.
  • The is closed.

You can add other log entries manually.

Note that you can add, change or delete entries in the log.  It is not an audit trail (although Auditing can be enabled to produce an audit trail for the log).
