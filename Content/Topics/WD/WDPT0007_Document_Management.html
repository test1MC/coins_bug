Enabling Document Management for Project Management

Project Management uses Document Manager to allow you to attach documents (such as project drawings) to items. You need to enable Document Management links for each of the Project Management tables.

To enable Document Management links:

Go to Document Management Links (in the System module).

Enter the following values:


Table 

Base URL
pt_action_item coins
pt_closeout coins
pt_daily  coins
pt_drawrev  coins
pt_issue  coins
pt_letter  coins
pt_meeting  coins
pt_rfi  coins
pt_submittal  coins
pt_tasklist  coins
pt_transmittal  coins

 

Also, enable Document Management for pi_project if this has not been done already.

If the facility to store documents against Project Management items is to be used, the amount of disk space required for these documents needs to be considered as part of any deployment.
