Sample eInvoice Messages

Sample messages designed to test the validity of a message from a new are marked by a in the Sample column in the workbench. (The Sample column and sample messages are only visible if the user is an administrator for the workbench, and the filter is set to All Messages or Sample Messages.)

Sample messages are validated and behave exactly the same as regular 'live' messages but can never be passed through to the ledger, the Create action is not available for these messages.

NOTE

This functionality is only available for XML messages coming from the ETC Hub.
