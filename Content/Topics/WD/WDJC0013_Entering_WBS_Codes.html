Setting Up WBS Codes

There are several ways you can set up the WBS codes on a :

  • Import them from third-party project planning software.
  • Create them automatically when a user types them in.
  • Set them up manually.
  • Generate them automatically based on the existing section and activity codes.

When you create a new WBS code, takes account of whether the activity and section combination is valid (see Activities).

To import WBS codes for a :

  1. Go to Import WBS Codes.
  2. Enter the code and the name and location of the WBS import file.
  3. Click to load the WBS codes.

To set up WBS codes manually:

  1. Go to .
  2. Click the link for the in the field.
  3. From the Options menu, choose WBS Codes.
  4. Use the standard maintenance features to add the WBS codes.

To generate WBS codes automatically:

  1. Run Auto Generate WBS Codes.
  2. Specify the or , and a list of section and activity codes for which you want to generate WBS codes.

Creating WBS codes automatically

You can configure to create WBS codes automatically if a code that does not previously exist is typed in. The first few characters will assumed to be the Section and Activity (depending on the configurationand provided a valid Section and Activity exist.  Set JC/AUTOCWBS to Y to enable this.

WBS import file structure

The import file containing WBS codes must have the following format. Each line finishes with a carriage return.  Fields are separated by spaces, and character fields are also enclosed in quotes.

For each work breakdown code, there is:

  • a line containing fields for: work breakdown code, description, section, activity, 0 (to indicate a detail line)
  • followed by none, one or more lines for each cost code, containing fields for: cost code, budget (decimal)
  • followed by a line containing a single period.

 

For example:

"05.20.5001" 5000.00
"05.20.5010" 125.50
"05.20.5013" 9548
