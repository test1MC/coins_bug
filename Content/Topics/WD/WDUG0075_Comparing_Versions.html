Comparing Versions of Wiki Topics

Whenever you update a Wiki topic (unless you clear the New Version box), creates a new version of the topic.  The previous versions are saved until the person who created them deletes them.

You can compare different versions of topics to see what changes have been made.

To compare versions:

  1. Click the Version link at the top of the topic.

    displays a list of versions of the topic.  You can click any of the links to view the different versions.

    NOTE

    The link in the Abstract column displays that version of the topic.  The link in the Version column displays a "raw" version of the topic (showing the [[ ]] marks) so it can be copied.
  2. Click the radio buttons against the two versions you want to compare.
  3. Click the above the radio button column.

    displays the topic with text that has been deleted shown with a line through, and additions shown with an underline.
