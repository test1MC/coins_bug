Modifying the Side Frame Help

The Side-Frame Help gives information about the functions and fields on each screen.  Although we aim to give as much information as possible in the side frame, you may want to add to or change the information we provide.  There are a number of things you can do:

  • Add your own text to individual nuggets.  This is shown in addition to the standard help text for the function or field.
  • Use a language translation to write different text that replaces the standard help.
  • Provide help for your own functions.
  • Control which users see the parameter information in side-frame help (see HELPPARM, below).
