CIS Scheme - Gateway Password

Electronic submissions to the HMRC website (for subcontractor verification and month end returns) require you to supply a password.  The password is issued by the Government Gateway.  You can choose to store the password in a parameter and include it automatically in submissions, or you can choose to require users to enter the password each time they make a submission.  If you store the password in a parameter, you rely on function security to prevent misuse; if you require users to enter it, you rely on the appropriate users knowing the password.

Set the SCUK parameter PASSWORD as follows:

  • Fill in the password to have include the password in submissions automatically.
  • Leave the parameter blank to require users to enter the password for each submission.

If you are using different passwords for different submission , set the parameters as -specific parameters.
