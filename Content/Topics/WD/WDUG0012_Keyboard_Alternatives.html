Keyboard Shortcuts

If you prefer to use the keyboard instead of the mouse, there are several key combinations (sometimes called keyboard shortcuts) available that you can use. These are:

 Button Description  Shortcut Keys
  Apply Filter Ctrl +Shift +A
Show/Hide Advanced Filter Ctrl+Shift+F
   Show or hide debug information about the screen  Ctrl +Shift +C
Ctrl+Shift+click ( logo in the header)
Lookup Ctrl +Shift +^
Add/New Ctrl +Shift +N (Not available in Chrome)
Open Ctrl +Shift +O
Refresh page Ctrl +Shift +R
Save Ctrl +Shift +S
Undo Ctrl +Shift +U
Go to next stage in the process Ctrl +Shift +X
  Go to the next block of fields (on an input form). Ctrl +Shift +Return
  Add a new line (in a text field). Ctrl +Return
Show or hide the Desktop. Ctrl+Shift +!
Open Global Search. Ctrl+Shift+?

NOTE

With the exception of the "add" shortcut, all shortcuts are now available in both IE and Chrome.

You can configure the keyboard shortcuts and set up alternatives; see 1 , Keyboard Shortcuts ConfigurationOverview.

The button tooltips show what the keyboard shortcuts are.

The following standard browser keys are also available:

Alt+Left Back
Alt+Right Forward
Ctrl+A Select All
Ctrl+C Copy
Ctrl+F Find text on a page
Ctrl+N New window
Ctrl+P Print
Ctrl+V Paste
Ctrl+X Cut
Esc Stop the page from loading
F5 Refresh (whole page)
F11 Full Screen
