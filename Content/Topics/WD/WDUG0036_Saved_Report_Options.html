Saved Report Options

If you frequently generate reports using the same report options, you can save those options, then call them up each time you want to generate the report.  When you run a report function, if there are saved options for that report, a separate tab allows you to select them.

To save report options:

  1. Define the report options for the report in the usual way.
  2. On the Output Options tab, select Saved.
  3. You may want to change the report title to identify this saved report.
  4. Click .

    displays Report Status with the Saved tab selected. This shows the saved report selections you can use (including the one you just saved).

To change the options on a saved report:

  1. Click the link in the Description column.
  2. Make your changes to the selection criteria or output options.
  3. Click .
  4. returns you to Report Status, and does not generate the report.
  5. To change the name of a saved report, click , type a new name in the Description field, then click .

To use previously saved report options:

  1. Go to the report you want to run, or go to Report Status.
  2. Click the Saved Reports tab.
  3. Click next to the saved report you want to run.
  4. displays the selection tabs for the report.
  5. If you want, you can change the selection criteria or output options for the report (for this run only) now.
  6. Click to run the report.
