Construction Workbench

 

Construction Workbench Overview Video

The Construction Workbench brings together information from and plot level to provide a single workbench for planning, forecasting and tracking progress of construction on a plot and construction stage basis.

The Construction Workbench:

  • Consolidates functionality from other workbenches in one place.
  • Provides additional methods of updating the actual construction progress:
    • Progress sheet.
    • Confirm forecasts as actual.
  • Allows you to do a build release up to a particular construction stage, for single plots or a range of plots.  You can validate actual progress against the build release.
  • Provides a Gantt chart style report.  This can be configured with different for each construction stage.  This reduces the need to use external project-planning tools.
  • Provides relevant information from the Sales Workbench for reference.
  • Allows you to compare current construction forecast against a baseline.
  • Allows you to apply construction s backwards from a projected completion date, as well as forwards from a construction start date.
  • Allows you to update s with actuals and then apply them forwards from that point.
  • Allows you to delay a stage and apply the onwards from there, creating a gap in the plan.
