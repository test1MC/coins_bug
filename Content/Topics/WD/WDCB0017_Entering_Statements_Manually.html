Entering Statements Manually

You can enter bank statements manually, for automatic or manual reconciliation, if the bank is set up for manual entry. You can also change statements that have been loaded automatically.

To enter a bank statement:

  1. Go to Bank Statements.
  2. Add a new statement.
  3. Fill in the fields for the statement header.
  4. Click the Transactions tab.

    For each statement entry, fill in the fields as required.

  5. When you have finished entering transactions, click .
  6. If the total value of the transactions equals the balance movement (the difference between the opening and closing balances), marks the statement as complete.

    If the total value of the transactions is different from the balance movement, warns you.

    • To leave the statement as it is, choose Yes. (You need to complete the statement before you can reconcile it.)
    • To go back and correct any mistakes, choose No.

NOTE

If you want to enter a statement with a zero value transaction, for reporting purposes, enter a closing balance that equals the opening balance; you can then enter a statement line with a value of zero.

Entering statements manually - configuration

You can only enter bank statements manually if the Statement Type field for the bank in Bank Accounts is set to Manual Entry.
