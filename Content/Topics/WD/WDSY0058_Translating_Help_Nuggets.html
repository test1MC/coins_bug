Translating Help Nuggets

One way to display different text for the side-frame help nuggets is to use a translation.

If you have entered a translation for a nugget, and the user is set up to use that language, the help will show the translated text.  For nuggets that you haven't entered translations for, the help will show the standard text.

To translate a help nugget:

  1. Identify the nugget you want to translate.
  2. Go to Translations.
  3. Click the link for the language you want to use (or click to add a new language).
  4. Click the Documentation tab.
  5. Open the record for the nugget you want to translate.
  6. Enter the translation.

    Change only the text; do not modify any of the HTML tags.  You may find it easier to copy the standard text (shown alongside), paste it into the editing field, and modify that.

  7. Save the record.

Translating help - configuration

You need to set the language on each user's user record to the language you want them to use.  You can use the Multi-Update button on Users to change the language for several users at once.
