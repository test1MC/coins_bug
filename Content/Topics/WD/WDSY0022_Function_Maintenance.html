Function Maintenance

provides a standard list of functions, with names beginning with a percent sign "%". This standard list covers most combinations of programs and parameters for released software, but you may want to add new functions if the combination you want is not catered for, or if you want to create new menus.

If you do create new functions, choose function names that do not begin with a percent sign (%).  Any functions that begin with % will be overwritten when we release new software. For the same reason, instead of changing existing functions or page sections (for example to change the name or the screen layout), you set up new functions which correspond to the standard ones. See Making Copies of Functions.

We recommend that you use a simple convention to distinguish your version of the function you have changed, for example put an X in front (please do not use punctuation characters).  For example, your version of %WSC1980FSBP (Print Certificates) could be XWSC1980FSBP.
