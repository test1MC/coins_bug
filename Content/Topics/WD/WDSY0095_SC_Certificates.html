Faxing Subcontract Certificates (Remittances) in COINSplus

SC certificates can be automatically emailed, faxed or printed according to default print method of the subcontractor.

Configuration

To fax subcontract certificates (using procedure sc198.p) - once the appropriate overlay configuration is complete, the fax option is enabled using the following Menu Parameters as defined below :

1 Print Form Name...      (Mandatory)
2 Printer Form
3 Billing Type...  (Optional) ASI or * or BLANK
4 Auto Fax ...  (Optional) 'Y' to only filter FAX numbers  
'N' to only filter Subcontractors without FAX numbers
'*' to print all
5 Internal use entry: Leave BLANK
6 FAX (Optional)  'FAX' to output to FAX or BLANK
7 Last Cert Only Default (Optional) LAST or BLANK
8 Job/Subcontract Selection (Optional) FRONT or BLANK
9 Override Order to be Job or S/C (Optional) J,S, or BLANK
10 Default Print (Optional) DEFPRN or BLANK
11 Fax Options (Optional) 'Y' override default fax options for each Subcontractor
'N' Keep default fax options
12 FAX Delay Default number of hours to delay fax
