Configuring Your Desktop

If a desktop has been set up for you and if you have the necessary permissions, you can maintain the content and appearance of your desktop.  You can change:

  • The background image.
  • The tabs and sections.
  • The tiles on a section.
  • Which section a tile is on.

And you can add a tile from the History menu.

NOTE

If you make changes to your Desktop, they will be visible next time you log in or when you refresh the Desktop.  To refresh the Desktop, click the logo in the top left corner.  If you refresh the Desktop, you will lose any open frames.

To change the background image:

  1. Move the mouse pointer over your name or user image.
  2. In the user menu, click to display User Preferences.
  3. Click the Preferences tab.
  4. Click .
  5. In the Desktop Image field, select the image you want from the drop-down list.
  6. Click .

To add tabs and sections:

  1. Move the mouse pointer over your name or user image.
  2. In the user menu, click to display User Preferences.
  3. Click the Desktop tab.

To move or delete tabs or sections:

  1. Move the mouse pointer over your name or user image.
  2. In the user menu, click to display User Preferences.
  3. Click the Desktop tab.
  4. Click the link on the Tabs label to rearrange the tabs, or the link on one of the tab names to rearrange the sections on that tab.

    opens the Desktop Tab Maintenance screen, or the Desktop Section Maintenance screen, in a new frame.

  5. Use the and buttons to rearrange the tabs or sections. You can also delete tabs or sections, or add standard menus.
  6. Click .

    4. Click the link on the name of the sections whose tiles you want to modify.

    or:

For each tile, you can specify the following:

Title

The description to display on the tile. If blank, the tile shows the standard context description.

Override

Whether to override the subtitle of the tile.  If this is not ed, the module the function belongs to is used as the subtitle.

Subtitle

The subtitle to display on the tile.  If blank (and if Override is  ed), the subtitle is blank.

Size

The size of the tile.  Refer to the side-frame help for details.

Open In

How to open the function: in the main area (replacing the desktop), in a new window (or tab, depending on the of your browser), or in a frame on top of the Desktop.

Foreground 

The of the text on the tile. You can use the picker to choose a , or you can enter a hexadecimal code directly.

If this is blank, the text is black.

Background 

The of the tile background. As with the Foreground, you can use the picker or enter a hexadecimal code. If this is blank, the tile background is white.

Icon

The icon to display on the tile.  If this is blank, the default icon, which is based on the category of the function, is shown on the tile. 

Options

Additional options for the page that opens when the tile is clicked.  These may be set up by your system administrator and will depend on the function that the tile runs.

Values 

If the function has been written to allow data to be displayed on the tile, this allows you to select which data to show.

To move tiles between sections:

  1. Move the mouse pointer over your name or user image.
  2. In the user menu, click to display User Preferences.
  3. Click the Desktop tab.
  4. Click the link on the name of the you want to move the tile from.

    displays the Desktop Item Maintenance screen for that section.

  5. Select the rows for the tiles you want to move.
  6. Select the Move Item row action and click .
  7. Select the section you want to move the tile or tiles to.
  8. Click .

    NOTE

    You can't move whole sections or tabs, but you can achieve the same effect by moving all the tiles from one section or tab to another, and renaming the section or tab .

To add a tile from the History menu:

  1. Move the mouse pointer over the icon.
  2. On the History menu, click next to the function you want to add to your desktop.

    displays an "Add to Desktop" box.

  3. Select the section you want to add the function to.
  4. In the other fields, choose how you want the tile to be displayed (for example, the title, and size).
  5. Click .
