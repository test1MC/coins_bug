Split Payments

Sometimes payments to need to be separated either for or individual invoices.  You can control payments to such that payment by , by or by transaction are possible.  You can set up defaults on  records, which will default down to the invoice record where it will also be updateable.

To process split payments:

  1. In , on the Payment Method tab, set the Payment Type field as follows:

    = payment for each  ? this will be default value and also the value used if the feature is suppressed.

     = Separate payments for each .

    Separate = Separate payments for each transaction (invoice).

  2. When you enter an invoice, you can specify how to pay that invoice using the Payment Type field on the invoice record.
  3. When you freeze the payment batch, creates one payment record for each invoice or as required by the payment type set on the invoice.
