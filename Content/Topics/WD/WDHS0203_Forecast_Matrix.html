Construction Workbench - Forecast Matrix

Forecast Matrix  allows you to maintain the construction forecast (the anticipated construction dates or week numbers) for each construction stage on each plot. 

Click to update the forecast.

Use the Options to change between weekly and daily forecasting.

To create or update a forecast:

  1. Go to Construction Workbench and click the Forecast Matrix tab.
  2. Click to update the forecast.
  3. Make changes to the forecast dates. There are various ways to do this; see below.

    A red flash in a date field means that you have changed the date but have not yet saved the updates.

    A green background means that the date is when the actual date, not a forecast date; you cannot update these.

  4. Click to save the forecast.

To update a forecast based on the standard construction s:

    1. On each plot, enter the forecast dates for the first stage (or the first incomplete stage) and/or the last stage.
    2. Do one of the following:
      • To calculate the forecast for all plots: from the Matrix action list, choose Apply Standard and click .
      • To calculate the forecast for all plots from the end date backwards: from the Matrix action list, choose Apply from End and click .
      • To calculate the forecast for selected plots: select the plots and from the Row action list, choose Apply Standard , then click .
      • To calculate the forecast for selected plots from the end date backwards: select the plots and from the Row action list, choose Apply from End, then click .

Actions

The actions you can apply to elements include:

  • Reset Element: Undo for selected cells.
  • Update Values: Updates/imposes a single date for all selected cells.
  • Delay Stage/Residual: Delays the selected cells and all later stages by a single number of days (entered once).

the elements you want to change, then use the Element: Choose Action list to choose the action, and click .

 

The actions you can apply to rows include:

  • Reset Cells: Undo for all cells in the row.
  • Copy between Events: Copies a date (with or without an offset) from one event to another.
  • Offset Forecast: All incomplete events are delayed by the offset.
  • Delay Start: Delays all events by a number of days.
  • Update Event Date: Simply updates a single event date.
  • Apply Residual : Applies the standard to events from and including the selected one.
  • Apply Standard : Using the date of the first incomplete event, the standard is applied to all incomplete events.
  • Apply from End: Applies the standard back from the last event to all incomplete events.  This calculates the forecast dates by taking the date of the last stage which is valid on the construction for each selected plot and subtracting the days and weeks for each stage cumulatively from that date.  If that last stage does not currently have a forecast date for a plot then no forecast stages for the plot will be updated.

The actions you can apply to the entire matrix include:

  • Reset Matrix: Undo for all cells in the matrix.
  • Copy between Events: Copies dates from one event to another for all plots with an optional offset.
  • Offset Forecast: Delays the first incomplete events by a number of days for all plots and re-applies the 
  • Delay Start: Delays all incomplete events by a number of days.
  • Re-sequence Plots: Re-sequences plots based on the start date of the first events.
  • Update with Actuals: Updates all events with actuals from the sheet.
  • Update with Actuals/Program: Updates all events with actuals from the sheet and re-applies the from latest completed event.
  • Apply Standard : Applies the standard from the earliest incomplete event for each plot.
  • Apply from End: Applies the standard from the last event backwards for each plot.
