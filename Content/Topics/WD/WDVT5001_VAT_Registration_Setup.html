Registration Setup

In order to use the Returns Workbench, Registrations must be configured for all - and where appropriate, sectors. 

WARNING

You cannot start using Returns Workbench in some and not others.

Whether  Returns Workbench is in use depends on whether there is at least one entry in Registration; if so, the old options for creating and managing returns are no longer available on the menus.

To set up registrations:

  1. Using Registration, set up a record for each registration (each entity that is registered with the tax authority and that will make returns).

    For or sectors that are not registered, set up a separate "dummy" registration record with the Regd field uned.

  2. Using Global Company Configuration, indicate which registration each  or sector belongs to.  The browse screen shows all ; each must be linked to one of the registration records (or if sectors are in use, each sector must be linked to a registration record).  The Registration Number field is -coded to indicate any registration numbers than need updating. 
    • To change the registration number for a , update the record in Global Company Configuration.
    • To change the registration number(s) for sectors within a , click the link in the Sector Reg? column to open Sector Details Maintenance, and update the sector record.  You can leave the registration number on individual sectors blank if it is the same as the that the sector belongs to.
