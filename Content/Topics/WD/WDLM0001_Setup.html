Lettings Financials Set Up

This topic details the items that need to be setup to operate the lettings financials. The two main items to configure are the control accounts and transaction categories.

  • Module Configuration allows you to specify the control accounts used in Lettings Management.
  • Transaction Categories allows you to set up the categories of debit/credit notes required. For example: for rent, tax, management fees, admin feeds and other charges.

    Against each category are various options. The key thing is the account to post to; for rent this is the rent suspense account.

Parameters

Various LM parameters are required for Financials.

In addition:

  • To enter cash for lettings, the CB parameter CBLEDGER will need to be amended to include the lettings module (14th entry).
  • The GL parameters ANALOPT and ANALDESC will need to be amended to allow for lettings recharges to be costed in GL, PL and SC ledgers.
