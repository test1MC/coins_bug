Format N (Bank Of Ireland (No Header))

There are no header lines contained in a BOI statement file. The lines each contain a single string with data at specific character positions. The character positions of the fields that Load Bank Statement reads are as follows:

Characters Field
1-6
7-14 Account number
16-17 Transaction type
36-46 Transaction value (in pence)
65-82 Description
If the transaction type is 02, the transaction is a , with the number in the right-most character positions.
102-106 Date in Julian format (YYDDD where DDD is the number of the day in the year)

Transaction types U1, U7, U8, U9, 44 and 54 are ignored.
