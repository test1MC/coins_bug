Alternative Base Currency

There are up to six values available for each item held in :

The six fields are :

1 - Document currency (for example invoice; the original transaction value)
2 - Base currency for the
3 - Alternative base currency 1
4 - Alternative base currency 2
5 - Alternative base currency 3
6 - Account currency (for example, account in PL, currency in JC)

See Basic Principles - How Multi-Currency Works.

Three values (document, base and account) are calculated for all relevant transactions. The other three, which can be configured if required, are known as alternative base currencies.

If you are going to be doing significant reporting in currencies other than base, you can configure alternative base currencies. This means that each item entered will be converted to the value for those currencies on reports.

If, for example, you process all your transactions in GBP, but are owned by a European owner who you have to prepare reports for in Euros, you could define alternative base 1 as Euros, and all transactions on entry will have Alt-Base 1 calculated and held in Euros. This then means reports can be defined, in Euros, to read this value easily, instead of having to do some substantial arithmetic during production of the report.

If you have a number of different working in a range of their own local currencies, you may find it convenient to set up your most commonly-used currencies as your various alternative base currencies, because you can then more easily define cross-company reports etc which add together the various values without needing to perform any exchange calculations. For example, if you have a number of whose base is GBP, and others with base EUR, you might set up alternative base currencies of GBP, EUR and USD in all . This would allow you to define consolidating reports in all three of these currencies which add values from all the in the system.

Posting reports print in the base currency of the , and then repeat the report in any alternative base currencies in use.  If one of these is also the base currency for a that section would be a duplicate, therefore in this particular situation the alternative base section of any posting report is suppressed.

Configuring Alternative Base Currencies

Use Alternative Base Currencies to enable multi-base currency and define up to three alternative base currencies. will then process all historical transactions and store the alternative base values - this may take a long time.
