Dates

You can record the dates on which key events in the lifecycle occur (and if appropriate, the value associated with that date, such as the award).

You can view milestones using Milestones or Timeline.

To record a key date:

  1. Go to the Details screen.
  2. From the Dates tab, select Key Dates.
  3. Click to enter a new date.
  4. Select the Key Date Type from the drop-down list. The dates available depend on the type.
  5. Enter the date or select from the lookup.  If appropriate, enter the value associated with the date.

  6. Click .

    If the date is set up as a milestone, it will also appear on the Milestones screen.
