Meetings

Using the Communication Management workbench, you can set up meetings, invite attendees and record who attended the meeting, set up lists of meeting items and automatically carry forward recurring items and unfinished actions to the following meeting.

You can use the Meeting Types function to set up different types of meeting.  Meeting types can be set up with default items which are added to new meetings when they are created.

There are three different types of meeting items:

  • Single items.  These are items that only relate to one meeting.
  • Recurring items.  These are similar but distinct items that occur at every related meeting; if a recurring item is added to a meeting?s item list, an identical item is also added to the item list of the following meeting when that is created.
  • Actions - things that need to be done.  These are items that are carried over from one meeting to the next until they are complete.  If an action is not marked as closed, it is added to the item list of the following meeting when that is created.

You can:

  • Print and/or email the meeting minutes
  • Generate a letter relating to the meeting
  • Create a follow-on meeting

You can attach documents to a meeting, and to individual meeting actions (but not to single or recurring meeting items).

To create the next meeting:

  1. Use the option (or the browse action) Create Next Meeting.

    When you create the next meeting, marks the current meeting as having happened.   This restricts the changes you can make to the meeting record. 

    You should usually create the new meeting after you have entered all the notes about the meeting, but before you print the meeting minutes. 

  2. works out the default date of the next meeting from the meeting frequency, but you can change the date and time.  The meeting number of the next meeting will be one more than the current meeting number.
    • Recurring items will be created for the new meeting, and given a number based on the new meeting number.
    • Any open Actions will be copied to the new meeting, and will be given the same number as on the current meeting.
    • Entries on the Attendees and Copies To tabs will be copied to the new meeting.

Configuration

Use Meeting Types to set up the meeting types and the action item template for each type.  See Project Management Item Types.
