Reports

The following reports are available:

  • Certificates Selected/Not Selected Report displays I if the test fails.

    For configured insurances that require validation, it checks whether the insurer and policy number are entered, and whether the date is set or is within the lead time to (for example, 1 month to ) if PAYINS <> '''. Report description

    Certificates Selected/Not Selected Report allows you to produce a report that shows details of outstanding certificates that have been selected for payment.The report is sorted and totalled by subcontractor (and can optionally be sorted and totalled by ). It shows the amount of each payment and whether the certificates were selected via Batch Selection or Manual Selection. Details of any certificates that are on hold and require approval before being paid are also shown. 

    If the subcontractor's account balance would become negative if the selected invoices were paid, displays the message "Neg Balance" to the right of the report line.

  • Defaulters Report displays which types have failed.

    For configured insurances that require validation, it checks whether the insurer and policy number are entered, and whether the date is set or is within the lead time to (for example, 1 month to ) if PAYINS <> '''. Report description 

    Defaulters Report allows you to produce a report that shows subcontractors that are in default for any of various reasons you select. If there are authenticated receipts outstanding, the report lists them.
