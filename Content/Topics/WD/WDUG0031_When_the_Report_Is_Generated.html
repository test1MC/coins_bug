Choosing When the Report Is Generated

You can choose when to generate the output:

  • Foreground - generates the document or report (and displays a message): 

    When it is complete, displays the document or report in a new window. You can view it again later from Report Status.

    NOTE

    Foreground printing is identical to background printing except that the screen is disabled until the print is completed. .
  • Background - generates the document or report in the background - you can continue with your work. When the document or report is ready, a message pops up (at the bottom right of your screen), with a link you can use to go to Report Status and view the document or report.

  • Delayed - generates the document or report when the date and time you specify are reached. If the data changes (for example, if more transactions have been added), the output reflects these changes. Go to Report Status to view the document or report.
  • Saved - saves the selection criteria which you used to select the report, and you can run the report later, using those selections, by going to Report Status. See Saved Report Options.
  • Scheduled - You can schedule a report to run automatically at regular intervals (for example, every day, or on the 15th of each month, or every second Wednesday). See Scheduling a Report.
