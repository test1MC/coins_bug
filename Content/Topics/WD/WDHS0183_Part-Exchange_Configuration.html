Part-Exchange Configuration

To use Part Exchange Workbench, you need to set up the following:

  • Part Exchange Workbench Events
  • PX Buyin Progression
  • PX Sale Out Progression
  • Letter Configuration (to set up PX letters)
  • Offer Calculation

There are several routines that are also required for the Sales Workbench; see Sales Workbench Configuration.
