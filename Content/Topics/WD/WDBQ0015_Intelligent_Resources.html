Intelligent Resources

An intelligent resource is a composite of two or more resources, which can be added to a in a single action. 

For example an intelligent resource could be created to represent all the components per square of a cavity wall, including bricks, blocks, mortar, wall ties, , etc. The user could then choose to add a number of square of cavity wall to the which will then calculate the quantity of each resource to be included.

Each intelligent resource has a code, a description and a unit of measure.  This is linked to a list of other resources that make up the intelligent resource.

You can also create -specific intelligent resources, which can be either modified copies of standard intelligent resources, or intelligent resources you set up for a single  only.

You can nest intelligent resources; that is, you can build up an intelligent resource from other intelligent resources.

You can specify other intelligent resources that should not be used on the same plot.  prevents you adding an intelligent resource to a plot if a conflicting intelligent resource already exists on that plot.

When you use an intelligent resource on a (either an actual or a template), the associated resources are added to the ; if you modify or delete any of the resources separately, this breaks the link with the intelligent resource; changes to the intelligent resource are then not reflected in the item lines, and you cannot add the intelligent resource to plots or house types.

To create an intelligent resource:

  • Use Intelligent Resources to create new intelligent resources and add resources to them.
  • Use Intelligent Resources (an option in House Type Workbench and Surveyors Plot Workbench) to set up -specific intelligent resources.
