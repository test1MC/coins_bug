Analysis Sets

Reports and browse screens give you various ways to sort and filter the records you're interested in.  For example, you can sort and filter records by account code, Registration Number, , and so on.  But these standard ways of sorting and filtering may not be adequate for your business needs. 

Analysis sets give you a way to set up additional fields on a record, which allow you to filter reports and , and arrange the information  in a sequence other than the standard sequence. Using analysis sets, and depending on the characteristics of the reporting program, you can:

  • Select which records to report on.
  • Change the order records are listed in a report.
  • Calculate subtotals by analysis set.

You need to set up the analysis sets for each module, to suit the needs of your business. You can do this by following the steps below.

  1. Add an Analysis Set (includes Code, Description (Field Label), Start/End positions) to module configuration. See Defining Analysis Sets and Populating Analysis Sets Automatically for details on how to do this.
  2. Add Analysis Set Value Codes (includes Code and Description) for the analysis set to the appropriate analysis set maintenance procedure. See Setting Up Codes for Analysis Sets for details on how to do this.
  3. If desired, add User Text to the side-frame help of the Description (Field Label) added in Step 1. This User Text will not be overridden when you upgrade . See Customing Help Text for details on how to do this.

If you cannot produce the analysis you want using analysis sets, there are a number of alternatives which you should discuss with Construction Industry Solutions.
