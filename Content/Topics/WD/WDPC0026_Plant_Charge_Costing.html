Charge Costing

When posting charges for or items, can determine the and category for the costs.  If you want to create costed invoices, set the parameter INTCOST to Y.

By default, uses the process below to determine the costing when creating an invoice.  The codes in square brackets are used in parameters to modify the (see Parameters below).

  • If not, and if the invoice is for an internal client , looks for the and category specified on the following, in order:

    • Group code in client . [C2]
    • Item code in client . [C3]
    • parameter JCCCCAT in client . [C6]
  • If the item was , takes the costing from the original item that was requested on the sales order, or if the item was added in Pending looks for a unique item that is mapped to the commodity code on the dispatch.

    By default, does not consult the PO commodity or the item code for , but you can instruct to look at these by including these steps using !C15 - PO commodity and !C19 - Item code in the parameters.
  • If none of the above apply (or is unable to determine the costing from the above), the following are then consulted in order:

Parameters

  • INTXCCMC for charges
  • INTXCCMK for kits
