Setting Up A Standard House Type

Standard house type allow you to create "templates" which can then form the basis of the House Type and Plot s if required.

To set up a standard house type :

  1. Go to Standard House Type Workbench.

    This shows a list of standard house types.

  2. Click the link for a house type.
  3. Click the House Type Items tab.
  4. To copy items from an existing :
    1. On the Options tab, select Copy Items.
    2. Choose whether to copy from a standard house type or a -specific house type.
    3. If you are copying from the -specific house type, enter or select the code.
    4. Enter or select the house type.

    6. Click to copy the items.
