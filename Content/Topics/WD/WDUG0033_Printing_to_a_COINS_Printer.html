Printing to a Printer

If is set up to allow it, you can print the output directly to a printer, without (necessarily) viewing it. the Print box, and choose a printer and number of copies. prints the document or report as soon as it has generated.

In , a printer name refers to a specific set up on a specific printer. Different printer names may refer to the same printer, set up in a different way.

Printing a report - configuration

Set the SY/PDFTOPS parameter to point to the path and command for the acroread command to allow the PDF file generated by to be converted to a PostScript® file, which can then be sent to a printer.

NOTE

This option only prints to printers which accept PostScript files.
