s

A is something that needs to be done on the project; it is assigned to an individual.

s, which are groups of that need to be done on the project, can be managed and added by using the s tab in Communication Management. You can synchronise with the Activity Workbench (and therefore with Microsoft Exchange).

templates allow you to define standard lists of that are often performed as part of a project, such as a project start-up or health and safety inspection.  Using a template means the tasks do not have be entered on each project.  You can copy these template lists into the project.

Configuration

  • To set up codes and names for the s on the project, use Project Setup >  Lists.
  • To set up template s, use Global Setup > Templates > List Template.
  • To copy from a template into one of the project's s, use Project Setup > Populate from Template.
