Cost Invoices

If you did not cost the invoice(s) at the time of invoice entry, you can cost them at a later time. allows you to cost multiple invoices for various at the same time. Every group (batch) of invoices that you cost at any one time will be automatically allocated a costing batch reference number by .

To confirm the costing analysis for a "Registered" or a "Provisionally" costed invoice:

  1. Go to Input - Cost Invoices
  2. Enter a new costing batch.

    NOTE

    If you delete a costing batch, any transactions that were costed in the deleted batch will remain on the register and will have to be costed again in another batch. 
  3. Specify the invoice you want to cost. Either:
    • Enter the code in the Account field, and then enter the internal reference number of the invoice in the Internal Reference field.
    • Leave the Account field blank and enter the reference number in the Internal Reference field. displays the Costing Distribution tab.

      NOTE

      The Internal Reference lookup list includes only those transactions that have not yet been paid. Also shown is the distribution type (Registered, Costed or Provisional) so that you can easily identify and select the transaction(s) that still need costing. will not allow you to select a transaction if it has already been costed. 
  4. To cost the invoice, enter a new costing record (see Distributing Invoice Costs).
  5. When you have finished costing invoices, you need to post the batch.

Once the batch has been posted, the costed transactions can be selected for payment. (See Selecting Invoices for Payment).
