Frames

Some functions, when run from the Desktop, open in a frame, which you can resize, maxime or minime.

Function in a Frame

The of the frame's title bar matches the of the tile from which it was run.

Frame Buttons

The following buttons are available in the title bar:

 

Close the frame.

 

Maxime the frame (so it fills the whole window).

Minime the frame (so it is hidden, but you can make it visible again).

Restore the frame to its default size (if it has been minimed or maximed).

Refresh the contents of the frame.

Show the side-frame help for the frame.

Resizing and Miniming Frames

You can resize a frame by dragging the handle in the bottom-right corner: . remembers the size of the frame; the next time you run the frame, it opens with the new size.

If you minime a frame, it moves to the "frames menu" in the page header. A number on the Desktop icon shows how many minimed frames you have.

To show the frames menu, move the mouse pointer over the Desktop icon; this shows the minimed functions.

Click the restore button or double-click anywhere on a minimed frame to expand it again. Click the maxime button on a frame to maxime it.

Use the and buttons in the user menu to minime or restore all open frames at once.
