Select/Deselect Invoices (Manual)

You can use Select/Deselect Invoices (Manual) to select individual invoices for payment. For selecting multiple invoices, see Selecting Invoices for Payment (Batch).

  1. Go to > Payments > Select/Deselect Invoices (Manual) to select individual invoices for payment.

    A list of invoices will be displayed.

  2. If you run multi-currency, ensure the page selector is set to the correct currency for the invoices and apply the filter. Invoices already in a payment batch will show a status of Sel'd for Payment

  3. Before selecting a transaction, you can check various details about it, by clicking the link in the Internal Reference column.

  4. Select or deselect individual invoices as follows:
    • To pay the full amount of a transaction, select the transaction(s), choose Pay Invoices from the list, and click . completes the Amount Paid and Discount Taken fields for you. The invoices will be added to the batch.
    • To part-pay a transaction, open the line using . The Amount Paid and Discount Taken fields become input fields. Override the full amounts by entering the amount of the transaction that you want to pay at this time, and click . The balance will remain on the System for payment in a later run.

    • To deselect a transaction from being paid, select the transaction, choose Delete Payments from the list, and click . The invoices will be removed from the batch.

    You can also select credit transactions, which will adjust the amount of the payment accordingly.  For further details about making payments using credit transactions, see Credit Notes.

NOTE

If you select an invoice containing a element, may not allow payment if the subcontractor does not meet certain requirements (see the SC Ledger Basics - Subcontractor Warning Messages for details). 

You can run the Selected/Not Selected Report report to check all the invoices selected for .

When you have finished selecting invoices, the next step will be to freeze (close) the batch of payments so that no other transactions and payments can be added to it (see Freezing the Payment Batch).

The payments you have selected will be processed on the next payment run (see Printing and Remittance Advices).

To make it easier for users to identify the correct invoices during payment selection, you can configure different columns to displayed on the PL manual payment selection screens.  In Page Field Sets, using the function %WPL1110BAPY, add a field set, selecting appropriate columns to display on the manual payment selection screen.
