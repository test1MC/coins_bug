Changing the Tax Rate Parameter

A new basic rate of deduction (20%) came into force when the CIS scheme was introduced on 6th April 2007. You need to set the SC parameter PAYE% to 20. You need to do this after you have completed processing the last payment run before 6th April 2007, and before you start processing the first payment run after 6th April 2007.

We strongly recommend you have the SC parameter TAXATPAY set to Y. (If you have TAXATPAY set to N, you will have to reverse out any unpaid transactions before you change the PAYE% parameter, as they must be paid at the new rate.)

Assuming TAXATPAY is set to Y, the point at which tax is finally calculated depends on the SC parameter TAXSTAGE. This can be set to either FREEZE ( makes the payment calculation when the payment batch is frozen) or PAY ( makes the payment calculation when the payment is actually produced).

NOTE

The parameter for the higher tax rate, HPAYE%, is only used under the 2007 scheme, so you can set this any time before the scheme starts.

You must also ensure that the tax line in subcontractor certificate definition is NOT set to Calc. Cumulative; if yours is, contact for advice. If the tax line is set to Calc. Cumulative, the new deduction rate will be applied to cumulative payments.
