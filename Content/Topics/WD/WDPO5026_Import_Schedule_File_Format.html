Import Schedule File Format

This file format is used by the Import Schedule from CSV File option from within a package in Projects. The file must be a CSV format file. The first row contains headings and will be ignored. The remaining rows contain resource data with the following columns.

Column
Number 		Description Data Type
1 Bill Reference (only used if Major Type = TRADE) Char
2 Line Type (C=Commodity/​Trade/ Code, S=/​Luckins Code), T = Text Char
3 Code (that is, Commodity/​Trade/​/​ Code or Item/Luckins Code) Char

4

Y/N

Logical
5 Description Char
6 Quantity Dec
7 Unit Char
8 Rate (Internal) Dec
9 Per (if not supplied, this is set to the same as Unit (column 7) Char
