WikiWords

A WikiWord consists of two or more words with initial capitals, run together. WikiWords are topic names. A Wiki topic name always has a fixed format: two or more words with initial capitals, run together. Like the name of the current topic: WikiWord. When you type the name of a topic in double square brackets, you create a link to that topic. You type [[WebHome]] and on saving the topic this becomes Web Home.  See Links in a Wiki. 

WikiWord linking is easy to use:

  • You don't have to know the full path to where the topic is stored - you just type the name
  • You don't need to write HTML
  • Without HTML, the topic text is easier to read when editing
  • Easy linking leads to interesting texts with links placed in context

WikiWords are styled like this because:

  • It makes Wiki hyperlinks instantly recognable
  • It leads to interesting Wiki topics
  • It avoids the need to fiddle with HTML tags
  • It avoids over-general topics because at least two words are required

Good examples of WikiWords

  • WikiWord
  • GoodStyle
  • VersionFiveDotThree
  • ReleaseVersion5dot3
  • AVeryLongWikiTopicNameIsAlsoPossible : wherever an uppercase or lowercase letter is allowed, a group of letters of the same case is allowed
  • YearTwoThousand

Bad examples of WikiWords:

  • Web: Name without the uppercase letter(s), lowercase letter(s), uppercase letter(s) sequence
  • Know-How: Name with dashes in between

Variations in linking

When you write the name of a topic, it becomes a link. There are more ways

  • To write a custom link label, use bracket notation: [[WebHome][the home page]] - this becomes: the home page
  • To link to a topic in another web, write: [[Main.WebHome]] - this becomes: Main.WebHome

Hints

  • Insert WikiWords wherever you can. Rich linking helps to make a Wiki successful.
  • Be specific. All topics in a web share one name space. For example, instead of FunctionSpec write BreadSlicerFunctionSpec because other projects might also have a functional spec topic.
  • Create topics with singular names. Plural WikiWords can be linked to the singular topic, that is, the link WikiWords links to the topic WikiWord.
  • Turn acronyms into WikiWords, that is, take FaqIndex for a "FAQ index" topic.
