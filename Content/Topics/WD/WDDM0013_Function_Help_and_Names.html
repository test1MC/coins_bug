Function Help and Names

If there is help text available for a function (as shown in the side-frame help for that function), you can include that text in a documentation topic. 

You can also insert a token to display the name of a function.

To insert help text for a function:

  • In the text, type $S$function_code$S$.  For example: $S$%WPO101ABPOH$S$.

    NOTE

    The help text must already exist.  Enter help text using Help Maintenance.
  • You can also use the lookup: select Side Help.

To insert a function name:

  • In the text, type $F$function_code$F$.  For example: $F$%WPO101ABPOH$F$.
