BOQ Resource Import File

Using Import Resources you can import resources from a file.

The Resources Import file is a CSV file containing the following columns:

  • Resource
  • House Type Dependent
  • Desc
  • Substitute
  • Quantity Unit
  • Override Description
  • Default Size
  • Size Unit
  • Retired
  • Material Wastage
  • Default
  • Payment Stage
  • Estimating Section
  • Default Material Package
  • Default Subcontract Package
  • Default Package
  • Analysis 1
  • Analysis 2
  • Analysis 3
  • Analysis 4

Then the following group of columns, repeated for each of the cost categories (up to six):

  • Cat
  • Valid
  • Default Price
  • Price Unit
  • Override
  • Status
  • Type
  • Commodity
