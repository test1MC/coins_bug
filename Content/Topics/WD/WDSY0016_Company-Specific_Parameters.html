-Specific Parameters

You can set parameters differently for different . For example, if you wanted to generate batch references automatically for most , but for 04 you want to use manual batch references, you would set the SY parameter AUTOREF as a -specific parameter in 04.

To set a -specific parameter:

  1. Switch to the in which you want to set the -specific parameter.
  2. Go to the relevant parameter maintenance function.
  3. the parameter you want to set and choose Create Specific from the Choose Action? selector.

    The column has a to show that this parameter applies to this only. If you connect to a different , the does not appear and the parameter has its global value.

  4. Open the parameter and change it to the setting you want to use in this (see Parameter Maintenance).

NOTE

You can make auditing settings -specific in the same way.

To see which parameters are -specific:

  • The Filter drop-down on each parameter maintenance screen has a ' specific Parameters' option.  Select that and click to show only those parameters that are set to be -specific.
  • Parameter has a Specific filter that shows which parameters have -specific settings.

  2. Choose Delete Specific from the Choose Action? selector.
