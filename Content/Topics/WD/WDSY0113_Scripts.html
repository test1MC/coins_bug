The report agent is a Progress batch session running pm99b.p. There are a number of configuration options which can be passed in to pm99b.p using the Progress -param start-up parameter. These are passed in as a pipe-separated list of name/value pairs, for example:

-param ?TIMEOUT=120|SERVER=OTHER?

The options are:

Option Notes

TIMEOUT

Over-rides the PM parameter TIMEOUT.

GENPROCS

Over-rides the PM parameter GENPROCS. Also over-rides the maximum report processes specified for a report agent family (if SERVER argument provided).

SERVER

Report agent family name. If specified then the report agent family must be defined within . The report agent family definition will be used to determine the maximum number of report processes (unless GENPROCS is also passed as a start-up parameter) and the report queues this agent will process.
