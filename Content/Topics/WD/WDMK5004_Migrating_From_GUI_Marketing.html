Migrating From GUI

If you are an existing user of GUI you need to do the following to migrate to OA .

  • Convert companies

    MK Companies Conversion allows you to convert the format of companies from the format used in GUI to the format used in . 

    This only changes the way the data is stored, and is needed for performance reasons.  You only need to run this once.

  • Convert dates

    You first need to set up the date types you will use:

    Date Setup allows you to set up the key dates that you want to record for .

    Each (Lead, , etc.) has its own list of key dates.

    Then you need to convert existing dates to the new format:

    Date Conversion allows you to convert dates used on existing in GUI to the format used in .
