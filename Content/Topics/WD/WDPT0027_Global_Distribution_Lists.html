Global Distribution Lists

Distribution lists allow you to specify groups of recipients for emails and letters (on the Copies To tab for an item).  You can create global distribution lists for use by all projects (for example, a list of company ).  Distribution lists can contain both personnel and contacts.

To create a global distribution list:

  1. Go to Distribution Lists.
  2. Create a new list using a code and description.
  3. Click the link in the Distribution List column to maintain the list.

    The lookup allows you to search for Personnel and Contacts.
