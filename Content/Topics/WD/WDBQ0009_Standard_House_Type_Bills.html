Standard House Type s

Standard house type are "template" which will then form the basis of  House Type Bills and Plot . 

You can create by importing from an external file, copying from another (standard house type,  house type or plot), or entering manually. 

The Standard House Type Workbench gives you a number of different views of the : by /payment stage, by estimating section, by package, or by item lines.  Each view allows you to add and maintain the resources.

You can add text lines to the , to add comments or to build in structure.

There is an option to print out the house type .

You can group items in various ways:

  • Payment Stages

    and payment stages are a means of grouping items for cost reporting. Each can be broken down into one or more payment stages, which typically relate to the sequence of construction.   You can build up the by adding resources to payment stages.

  • Packages

    Packages provide a way to break down the into manageable groups of items.  Packages can be selected for update and ordering.  The package names and codes are set up in Procurement, but the resources that make up the packages can be different on each house type or plot.

    When you add resource to a , you can select the package it belongs to (resources can have a default package associated with them), or you can maintain packages on the and add resources to them.

    On the Package Browse, you can control the order the items appear within a package, to help in the ordering process.

  • Estimating Sections

    Estimating sections allow the estimator to summarise and maintain house type independently as different sections within the same .  This will allow the estimator to build up alternative estimates and structure the bill to their own requirements which may well be different from the surveyors' and buyers' views of the world. 

Updating VAP Standard House Type Budget

You can update the Standard House Type Budget in VAP with figures calculated from the Standard House Type Bill.  This also updates the VAP  Budget at Plot level from the BOQ, ensuring that the updates are applied consistently.

You can also update the budget for selected payment stages, from the Payment Stages summary.
