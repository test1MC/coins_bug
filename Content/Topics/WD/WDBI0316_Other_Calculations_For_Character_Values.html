Other Calculations For Character Values

ENTRY$(3,"A.B.C",".") would return "C"
RIGHT$("ABCDE",2) would return "DE"
LEFT$("ABCDE",2) would return "AB"
SUBSTRING$("ABCDE",3,2) would return "CD"
LENGTH$("ABC") would return 3
REPLACE$("ABCDA","A","X") would return "XBCDX"
CAPS$("aBc",1) would return "ABC"
CAPS$("aBc",0) would return "abc"
TRIM$("ABCDA","AD") would return "BC", that is, trims all chars in second parameter from both ends.
RIGHT-TRIM$("ABCDA","A") would return "ABCD"
INDEX("ABCDA","A") would return 1
RIGHT-INDEX("ABCDA","A") would return 5
NUM-ENTRIES("A,B,C,D",",") would return 4
LOOKUP("B","A,B,C",",") would return 2
