Procurement

Procurement via Packages - Order from Separate Suppliers (24906)

The Comparison tab in Project Resources has been updated to use the new-style matrix features. A new "Move Items to New Package" option (on the right-click menu on the Comparison tab) allows you to create a new package using selected items (so that those items can be ordered from a different supplier). See OA-CE-PO026 for details.

Supplier-Specific Styles (25190)

To make it easier to choose the correct style on a order, styles can now be linked to a supplier company record in the Company Information Workbench, so that when an order is raised, the styles dropdown is limited to only the styles provided by the supplier. See OA-CE-PC010 for details.

Purchase Orders - Locking the Description on PO Lines (25197)

It is now possible to prevent users from updating the commodity description on PO lines, allowing meaningful analysis of purchases when using a detailed set of commodity codes, or when using the file as a true file or as a product list on material orders. An additional description field can be made available for specific commodities as required. See OA-CE-PO022 for details.

PO Sales Tax Default from (25210)

When Item is added to either a Standard Order or a Single Form Order, the system now checks the new PO parameter STTAXDEF. If it is set to "I", and the selected Item has a non-blank Sales Tax Code, then the Sales Tax Code is defaulted. In all other circumstances, the Sales Tax Code is defaulted from the Location. See OA-CE-PO023 for details.

Requisition from e- (25264)

The ability to raise requisitions has been added to the e. See OA-CE-PO024 for details.

PO Notifications (25343)

A new PO Notification Sender routine has been added, which allows committed orders to be emailed automatically to PO administrators, suppliers/subcontractors, and internal users. See OA-CE-PO025 for details.
