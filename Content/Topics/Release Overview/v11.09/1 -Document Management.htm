Document Management

Simple DM Standard Workflow (25244)

New functionality has been added that allows emails to be triggered when uploading, checking out, checking in or deleting a document in DM. See OA-CE-DM007 for details.

Scanned Batch Workbench (25245)

A new Batch Control workbench provides additional functionality to manipulate Document Management batches and pages within a batch. See OA-CE-DM008 for details.

Improved Searching (25258)

A new DM Global Search is available. This is based on the existing Global Search (using Solr), but is specifically for Document Management documents. See OA-CE-DM009 for details.
