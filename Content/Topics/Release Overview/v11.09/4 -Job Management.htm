Job Management

JC Owner Budget Status (24927)

OA users are now provided with billing values based on Estimates and Owner costs (not based on current rates). This includes Planned Costs, Owner Costs, and Costs.

SC AUS Subcontract Statutory Declaration (24565) (AUS)

It is now possible for a Contractor to know whether a Statutory Declaration has been received from the Subcontractor, which indicates that the Subcontractor has paid all suppliers, employees, etc. This functionality has been added for the AUS Region to the Subcontract Ledger and modules. See OA-CE-SC010 for details.

SV Zero Margin Status (25077)

A new Zero Margin option is available in the Status Maintenance function. It gives users a convenient way to recalculate the pricing by applying values to result in a Zero Margin on a . See OA-CE-SV045 for details.

CV Model Specific Items and Styles (25113)

Users can now maintain Items and Styles at the Model level (having Model specific sets of Items and Styles). Any changes made in a Model will no longer have influence on other Models. See OA-CE-CV023 for details.

CV BPF Forecasting Contingency Costs (25120)

BPF Forecasting based on usage did not provide users with the ability to account for Contingency Costs on Cost Codes when using standard features. The module has therefore been enhanced so that you can now define a list of Breakdown Types on Item properties, set the Type values on Breakdown Items, and combine values of an Item by these Types. See OA-CE-CV026 for details.

SV Risk & Opportunities / Split (25151)

The Risk & Opportunities has been enhanced to allow for the assessment of Risk and Opportunity not only on Margin, but also on and on a single record. As a result, Final and can be updated to account for any Risk and Opportunity (not just its impact on the Final Margin). See OA-CE-SV044 for details.

CV Subcontract Package Views (25123)

New Items, Browse, and Forecast views for Subcontract Packages have been introduced in the module. See OA-CE-CV025 for details.

CV by Cost Code/Category (25243)

The functionality that allows you to collect and enter data for Cost Code items has been expanded to include Cost Code/Category Items. Category split values of Cost Code/Category Items are now available at Cost Code, , and levels. See OA-CE-CV027 for details.

CV Spread Performance (25252)

The performance of the Spread screen has been improved. Some changes include:

  • Reducing the number of items recalculated through better dependency checks.
  • Shifting the dependency calculation to a background queue.

See OA-CE-CV024 for details.

CS Certificate Defaulting and Validation (25326)

and modules have been changed to allow a process of automatically defaulting and validating Certificate numbers.

CS Adjusted CS Certification Status (25327)

When a CS Certification has been adjusted, users can now see indicators of that adjustment via Adjustment flags and columns within . See OA-CE-CS004 for details.

JC Summary Changes (25335) (US)

The Summary has been updated for the US with more consistent and calculations for Future Revenue (Revenue Backlog). It also now provides users with the ability to review the Revenue Backlog on Cost Based that are not using budgets. See OA-CE-JC036 for details.

JC Summary - Data Mart (25353) (US)

The Summary only uses LIVE data, which caused performance issues when larger, more complicated column sets were used on the browse. A new Summary - Data Mart has therefore been introduced to Status to resolve this issue. It uses stored data using data marts and dataset definitions. It is only available for the US region and if there are any populated data marts. See OA-CE-JC039 for details.

JC Quick Budget Entry (Phase II) (25355)

Enhancements have been made to the Budget Entry Workbench to allow for the input of At Complete and To Complete values. This functionality also lets you set the Budget Input Method to Qty (Hours) on more than one category when the Cost Code/Category version of the Workbench is used. See OA-CE-JC038 for details.

JC Listing Report (25356)

A standard Listing Report is now available in Status. It lists all Subsidiary and their values for each selected. Also, a change was made in the master Listing report to allow for a Standard WIP option. See OA-CE-JC037 for details.

Site Overhead Management (25063)

New functionality has been introduced to manage the detail of site overhead valuations and cost-to-complete calculations. For home building customers, additional features allow integration with VAP Budgets, and other specific home building functionality within . See OA-CE-JC034 for details.
