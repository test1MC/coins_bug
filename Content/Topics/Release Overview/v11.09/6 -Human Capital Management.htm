Human Capital Management

PR US Print W-2 for Multiple Localities (25070) (US)

Additional functionality has been added to OA so that employees can print one W-2 for all of the Localities where they worked per State. This W-2 shows a total value for all of the Locality tax paid for that State. The employee ends up with a single Locality W-2 for each State in which they worked. This applies to the US Region only. See OA-CE-PR111 for details.

HR Function Specific Lookup Codes (25073)

The following functions have been created to provide security to Lookup Codes by function: ACA Lookup Codes, Benefit & Accrual Plans Lookup Codes, and Drug Tests Lookup Codes. See OA-CE-HR029 for details.

HR Employee Benefits for Prevailing Wage Calculations Report (25114)

The Employee Benefits for Prevailing Wage Calculations Report has been enhanced to calculate Average Hours worked and to report on the employer cost of benefits, including paid leave plans. See OA-CE-HR031 for details.

PR W-2 File Export (25119) (US)

A new W-2 File Export function has been added in OA Payroll under the Company Setup > W-2 Processing menu. It allows users to export W-2 information from OA into a .csv format so that they can view their W-2 data in a spreadsheet. This applies to the US Region only. See OA-CE-PR112 for details.

PR Employee State Details (25138) (US, CA)

OA Payroll now allows users to update an Employee's State Details if the Home Address State is changed. These changes apply to the US and Canadian Regions only. See OA-CE-PR114 for details.

PR Redirection Masking (25364) (US, CA)

PR Redirection now allows masking for the field. Also, PR Calculation now combines time card analysis with masking. These changes apply to the US and Canadian Regions only. See OA-CE-PR115 for details.

PR Certified Timecard (25122) (US)

Certified Payroll Reporting and exports that are created via LCP Certified Integration rely on Payroll time cards to contain accurate data. A new Certified Timecard has therefore been added to OA Payroll so that users can review and edit posted Payroll data for Certified Reporting. These changes apply to the US Region only.

HR New Level of ESS Approvers (25205)

ESS Approvers were previously tied only to Timesheet Input Groups. Changes have been made to also allow such Approvers to be set up in Personnel records on the HR Personnel Workbench. See OA-CE-HR028 for details.

SS Incidents/Accidents Changes (25288)

Analysis Set selection criteria are now used to provide counts and amounts for the following Health & Safety : Incidents Count by , Incidents Count by Year, Accidents Count by Company. See OA-CE-HR030 for details.

PR Missing TC Report by TS Input Group Merge Code (25289) (US, CA)

When using Timesheet (TS) Input Groups for payroll processing, OA users can now run the PR Missing Timecard Report by TS Input Group Merge Code. These changes apply to the US and Canadian Regions only. See OA-CE-PR117 for details.

PR Rates by Employee (25292) (US, CA)

OA Payroll has been enhanced for the US and Canadian Regions only to allow the defaulting of proper rates for an Employee in order to speed up processing and increase accuracy. See OA-CE-PR116 for details.

Timesheets – by Multiple Employee Defaulting Changes (25350) (US, CA)

Timesheets by Multiple has been updated for the US and Canadian Regions only to default certain information from the prior transaction when in the same session. 

GDPR Anonymation for PR and HR Records (25313)

To meet new European GDPR requirements on ‘the right to be forgotten’ and the requirement to only retain personal data for as long as it is necessary and justifiable to do so, a new routine is available which will ‘anonyme’ HR Personal and PR Employee data when required. 

Single Touch Electronic Submission (24971) (AUS)

Australian Payroll now has the ability to send an employee's pay information to the Australian Taxation Office (ATO) via a 3rd party software provider.

RTI changes for 2018 (25144) (UK)

Beginning in 2018, the Student Loan plan type will be included on the FPS RTI submission.
