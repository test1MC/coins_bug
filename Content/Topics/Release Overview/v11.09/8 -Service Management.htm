Service Management

SE Transfer Utility (25094)

Details for a Service can sometimes be assigned to the wrong . To resolve this type of error, the new Transfer utility is available in OA Service Management. It allows you to transfer all details from one to another. However, there are specific criteria necessary to successfully transfer information. See OA-CE-SE057 for details.

SE Access to Inception Date (25104)

Beginning with OA v11.07, the Inception Date was considered an Add-only field. However, the ability to access the Inception Date in Update mode has been reintroduced in v11.09. As a result, users can now change the Inception Date when certain criteria are met. See OA-CE-SE058 for details.

SE Summary Screen (25230)

A Summary screen has been added to under the Finance tab to provide users in Service Management with a more efficient way of showing costs, billings to date, commitments, and estimated gross margin prior to invoicing. See OA-CE-SE061 for details.

SE Bulk Mode Location Preferred (25231)

A new Bulk Mode Location Preferred browse has been added to the Related Functions menu of the Locations function. It allows you to set up and maintain Preferred information for multiple locations in one place. See OA-CE-SE060 for details.

SE Ready for Invoice Stage (25232)

A new Ready for Invoice Stage has been added to the list of available Stages to make it easier for users to find Orders that are ready to be invoiced. See OA-CE-SE059 for details.

SE History (25237)

OA now gives users the ability to view service billing information by , which then allows their customers to see the amount of money expended to keep their running. Changes have been made to , and a new History Report has also been created for this functionality. See OA-CE-SE065 for details.

SE Zero Dollar Billable Invoices (25242)

OA now gives users the ability to show a billable amount for repair work that fully discounts the line so that a billable invoice is not produced. New SE parameter BILAGRCH supports this zero dollar billable invoice functionality by calculating the Billable value for covered . See OA-CE-SE064 for details.

SE Renewal Renumber Toolbox (25253)

The Renewal Renumber Toolbox has been added to the SY Toolbox > Service Management menu. It can be used to adjust the renewal numbers on existing . It can also be run on that have been renewed. See OA-CE-SE062 for details.

SE Type Renumber Toolbox (25299)

The Type Renumber Toolbox has been added to the SY Toolbox > Service Management menu. It is a renumber routine that can be used by operating companies to update existing records to use a standard list of values. Users who access this function can see pending changes for all Types. See OA-CE-SE063 for details.

SE Subcontract Purchase Orders (25346)

OA users now have the ability to see Subcontract Purchase Orders within the function for purposes only.
