Home Building

HS Choices (24716)

House Sales Choices is new functionality that provides an improved way of handling customer choices and the specification of the home, compared with the existing and functionality. A migration routine allows you to convert and on existing plots to the new choices. See OA-CE-HS021 for details.

NHBC Submissions (24717) (UK)

Home builders who provide home construction warranty and insurance through NHBC need to submit data regarding their reservations, exchanges, and completions every month as part of the Insurance Distribution Directive (IDD). Changes have been made in to manage the process of making regular submissions. COINS will communicate with the NHBC interface via a new web service. See OA-CE-HS024 for details.

House Sales Kanban (25026)

A new screen has been introduced that allows you to view plots on a in a 'kanban' format, with key events along the screen in columns, and plots shown as cards under the relevant column. You can update plots by dragging them from one column to another, and can perform various actions by clicking icons on the cards. See OA-CE-HS023 for details.

Home Builder Enhancements (25172)

Several changes have been made to improve the user experience for home building clients. See OA-CE-VP006 for details.
