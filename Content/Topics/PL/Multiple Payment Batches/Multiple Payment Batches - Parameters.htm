Multiple Payment Batches - Parameters

The following new parameter in both PL and SC controls whether you can have more than one payment batch open.

Parameter

Description
CONCPAY

Whether concurrent payment batches are available.

If this is set to Y, when selecting invoices for payment automatically (using Select/Deselect Invoices (Batch)) or entering payment details for individual invoices (using Select/Deselect Invoices (Manual)), you can choose whether to use the default payment batch, use another existing batch, or create a new batch.
