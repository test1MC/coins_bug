Multiple Payment Batches - Select/Deselect PL Invoices (Batch)

Additional fields are now available on the Select/Deselect PL Invoices (Batch) screen.

Select/Deselect Invoices (Batch) Screen

Fields

Field
Description

Batch

This is only shown if the new PL parameter CONCPAY is set to Y. It is a radio set with the following options:

  • Default - Use the default batch (or create one automatically if it does not exist).
  • New - A new manual batch is created specially for the current process.
  • Pre-existing - Use a pre-existing batch.

Batch Number

The batch number that will be used. If CONCPAY is set to Y and you selected Pre-existing from the Batch field, you can enter the batch number to use; there is a lookup of incomplete and unfrozen batches.

Otherwise this field is read-only.

Batch Reference

The reference of the batch used. If the CO parameter AUTOREF = N and a new batch needs to be created, you must enter a batch reference (that is, if CONCPAY = N and no payment batch exists, or if CONCPAY = Y and you selected Default above and no payment batch exists, or if CONCPAY = Y and you selected New above).

Otherwise, this field is read-only.
