Aggregate Billing - Overview

New functionality has been introduced for aggregate supplies: output from weighbridge software can be used to create dispatch records. The dispatches are then charged using the existing prices and customer agreement functionality to create charges. The customer charges create invoices. The invoice print can include details such as the truck ID from the original load file.

A new web service is available to load dispatch files. ESB Transformation can be used to take files, transform them as required and feed in to this web service.
