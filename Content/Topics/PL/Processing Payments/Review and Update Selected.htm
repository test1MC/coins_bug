Review and Update Selected Payments

Once you have you created the payment batch, you can choose to add/remove one or more payments from that payment run.


Navigate to Select/Deselect Payments (Manual)

A list of invoices will be displayed. If you run multi-currency, ensure the page selector is set to the correct currency for the invoices and apply the filter. Invoices already in a payment batch will show a status of Sel'd for Payment

Removing an Invoice from the Payment Run

To remove an invoice from the payment run, select any invoices you do not want to pay and select Delete Payments from the Choose Action Drop Down list. Select the Apply button. The invoices will be removed from the batch.

Adding an Invoice to the Payment Run

To add an invoice to the payment run, Select any Invoices you want to pay and select Pay Invoices from the Choose Action Drop Down list. Select the Apply button. The invoices will be added to the batch

Part Paying an Invoice

To part pay an invoice, select any invoice you want to part pay and open the Line using the Open button.

Amend the Amount Paid or Discount Taken values as required.

Select the Save button to save the changes.  The invoice will only be paid to the amount specified and the balance will remain on the System for payment in a later run.
