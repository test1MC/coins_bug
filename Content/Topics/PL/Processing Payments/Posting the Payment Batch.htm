Posting the Payment Batch

The Payment Batch should only be posted once all the and Remittances have been successfully generated.  Before the batch is posted it is possible to unfreeze the batch, make changes to the payment batch, refreeze the batch and re-process the payment run (ensure you re-generate any paperwork to reflect your changes) .

Once the batch has been posted it will no longer be possible to make any corrections to the payment run and the supplier accounts will be updated with the payment details.

  1. Navigate to Payment Batches and select the batch. Post Payment Batch

  2. From the Choose Action dropdown box select Post Payment Batches and then Apply the Action.
