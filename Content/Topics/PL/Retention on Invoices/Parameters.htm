Retention on Invoices - Parameters

The following parameters control the retention on invoices functionality.

Parameter

Description
USERETPL Whether to use retention in (Yes or No). This can only be set when CO/COMPLOC in the current company is set to USA or CA.
RETDISC Whether to allow retention invoices with a discount amount to be created (Yes or No).

CALCRET

A list of additional items (charges and tax) to include when calculating retention. The list can include one or more of A1, A2, A3 (additional charges 1-3) and T (tax).

In addition, the CO parameter TAXTYPE must be set to SALES and the PL parameters REFBASE and REFFORM must be populated (to provide the format for retention-only invoices – see 1 - Retention on Invoices - Releasing Retention).
