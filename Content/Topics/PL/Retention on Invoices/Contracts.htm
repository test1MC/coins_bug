Retention on Invoices -

A new field on the record allows you to set the retention percentage.

  • On the “expanding sections” view, this is shown on the Materials & Subcontract layout of the Advanced Settings tab.

  • On the tab view, this is shown on the Configuration tab.

Field

Field
Description

Retention % ()

The retention percentage to use when entering a material order in .

If a project exists for the , this field is linked to the equivalent field on the project. Updating one field will update the other.
