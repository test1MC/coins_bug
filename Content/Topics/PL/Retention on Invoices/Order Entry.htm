Retention on Invoices - Order Entry

If the order type is set up to allow retention to be entered (see 1 - Retention on Invoices - Order Types), new retention fields are available when entering an order.

Fields

Field
Description

Maximum Retention Type

Whether the maximum retention is a value, or a percentage of the order amount.

Maximum Retention Value

The maximum retention value for the order.

Retention %

The retention percentage. This defaults from the Retention % () field on the Project Summary record for the .

If no maximum retention value is entered, it will be calculated automatically when the order is committed, based on the retention percentage and taking into account any additional charges and tax. The new PL parameter CALCRET determines whether additional charges and tax are included in the calculation.

The retention fields can only be entered or updated while the order is uncommitted.

A new column on the material orders browse shows the maximum retention.
