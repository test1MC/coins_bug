New PO Parameters

The following new parameters have been introduced:

Parameter

Description
ALWCLSST Whether to allow invoice matching against closed sections. The default is Y.
ALWCLSCC Whether to allow invoice matching against closed cost codes. The default is Y.
ALWCHOV5 Whether to allow the WBS and category to be overridden during invoice matching. The default is N.

In addition, the description of the parameter ALWCHOV4 has been updated to make it more explicit: "Allow override of and Category during Invoice Matching (Additional Charges and Extras)".
