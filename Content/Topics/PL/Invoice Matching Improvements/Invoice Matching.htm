Invoice Matching

Brief Mode

If the new PO parameter IMSTYLE is set to BRIEF, a tab selector allows you to select the new Brief mode of invoice matching.

Brief mode shows only item lines, and credit lines entered against item lines (that is, it does not show additional charges). A limited set of fields are available on the browse; concurrent update allows you to update several lines together.

Other details can be viewed using the Detail button. The Add button only allows you to add simple credit lines. Other fields, which in Detail mode are available when updating a record or viewing details, are available on column sets in Brief mode.

If the new PO parameter DEFOVVAL is set to Yes, the Override Value box on the matching lines is ed by default, and the Value Invoiced field is available to update. If DEFOVVAL is set to No, the Override Value box on the lines is uned, and the Quantity Invoiced, Invoice Price and Disc% fields are available to update.

Unmatch and Register Invoice

A new option, The Unmatch and Register Invoice , has been added to the Options menu. This option allows you to unmatch items and set the hold codes on a single screen. Choosing this option brings up the Unmatch and Register Invoice screen.

This provides three actions:

  • Unmatch my last changes - This marks as unmatched any items you matched today. This is useful if more than one person has matched invoices to an order, and you undo any changes you have made without affecting what others have done.
  • Unmatch all changes - This marks all matched items as unmatched and sets the invoice back to Registered.
  • Retain matched items - This keeps all matched items, and sets the invoice to Registered.
