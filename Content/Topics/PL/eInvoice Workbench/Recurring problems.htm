Stylesheet view not rendering OA.

The location of the stylesheet in OA is determined by the ec\ XSLLOCOA parameter , XML text will be displayed if the correct stylesheet is not available in the location that the XSLLOCOA parameter is pointing too.

The value of this parameter usually points to the document root of the web host program.

 

“/var/www/html” – Located using command “grep -i 'DocumentRoot' /etc/httpd/conf/httpd.conf”

 

The document root is usually made up of shortcuts to the application directory e.g “/u/coins/live/app/xcode/db/style” otherwise it will be its own directory and stylesheets should be present here.

The files listed below should be available in the directory listed, if not they should be copied to this directory and given 777 permissions.

Stylesheets can be copied from the Plus directory if they are not available locally.


If the stylesheet files are not available on the contractors system, the eCommerce developers will be able to provide them.
