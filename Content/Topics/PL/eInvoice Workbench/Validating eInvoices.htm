Validating eInvoices

When loading files, will validate all invoices automatically rejecting documents where configured to do so.  A previously loaded file will be re-validated when the Review, Reject or Create button is pressed.

The handling of invoices with validation errors may be configured using a non-company specific configuration program

Program ec2055.p

.  You may specify the following for each error code: if a rejection notification email should be sent; and if so, the role that should use when deciding who it should send the email to; and, thirdly, whether the invoice should be automatically removed from the workbench.
