Overriding NONPOCH and NONPOPCH

The value of the PL/NONPOCH parameter may be overridden on a Supplier specific, Office, basis. Set a value for a tag called <DefaultCostHead> in the Office’s eCommerce Company Information Configuration PLINV record.

Similarly, the value of the PL/NONPOPCH parameter may be overridden on a Supplier specific, Office, basis. Set a value for a tag called <DefaultPlantCostHead> in the Office’s eCommerce Company Information Configuration PLINV record.

A value set for the anonymous, blank, office will act as a default for all of a Supplier’s Offices.
