Validation error rules

In some cases a custom fix or update can create additional eCommerce error messages, these messages appear on the WBs when the data is in error.
If the data is in error, which uses a new error message the error below will be produced:
 +---------------------------------Exception--------------------------------------|

| Unable to validate e-PLINV: validation rule ERR085 does not exist |

|                            Display detail info?                   |

| -------------------------------------------------------------------------------------|

|                                 <Yes> <No>                        |

+-------------------------------------------------------------------------------------|

To resolve this ec2055.p should be run, hitting F1 updates the error message table with the new error. F1 again confirms the addition.
