Variation Orders

 

The workbench recognises variation order numbers in an eInvoice XML message

The EC/SWWBMEMO parameter must be set to N and the PL/NONPOWBS must be set to Y for this to work.

. The workbench will look for a / character in the <BuyersOrderNumber> tag and take the trailing number after it to be a variation number

The purchase order number prefix, PO/POORDPFX parameter includes a / character then this is taken into consideration.
