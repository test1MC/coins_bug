Auto-correction of PO Numbers

If the parameter PL/FIXPONUM is set to Y you can set rules

The rules are set in the Order Number Heuristics program ec2056.p

that will endeavour to auto-correct any typing errors the supplier may have made in the PO Number. Typical rules are shown below. Here we replace any letter O (upper or lower case) in the first six characters or eleventh to fourteenth characters of the order number with the number 0 (zero). If we find a \ as the seventh character we replace it with the / character. The wildcard character * may be a target, the eighth character will always be replaced by the letter H.

The rules are processed in numerical sequence.
