Create/Post Recurring Invoices

Create/Post Recurring Invoices allows you to create and/or post a range of recurring invoices.

Fields

Field
Description

From, To

A range of recurring invoice numbers.

Period

The posting period for the recurring invoices.

Post Batch

Whether to post the batch automatically.

If the new PL parameter RIAUTPST is set to Y, this defaults to Y, otherwise it is blank.

Only recurring invoice types with a Next Post period that is the same as the posting period and are also marked as "Ready to Post" will be created/posted.

When you click Next, a PL Invoice batch will be created for the period selected, with a description of ‘Recurring Invoices’. The batch transaction type will be taken from new PL parameter RITTYPE (for example, PRI – PL Recurring Invoice). For each recurring invoice type selected as ready to post, an invoice transaction will be created; the cost and distribution records will also be created from the template invoice. You can edit the recurring invoices in the same way as standard invoices, except for the Supplier Code and Reference fields.
