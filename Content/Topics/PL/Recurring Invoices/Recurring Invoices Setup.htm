Recurring Invoices Setup

A new function, Recurring Invoices Set-up , allows you to set up the details of recurring invoices.

Fields

Field
Description

Code

A code for the recurring invoice.

Description

A description of the recurring invoice.

Supplier

The supplier code.

Supplier Reference Base

The base for the supplier reference - this is used to automatically generate the supplier references on actual monthly invoices. The supplier reference will be the supplier reference base plus a number from 01-9999; for example, Lease1/0001.

Invoice Number of Days

The day of the month to use as the invoice date.

For example, if this is 1, the invoice date will be the 1st of the month.

If this is set to 30 or 31, the invoice date will be set to the last day of the month (including February).

Start Period

The for the first recurring invoice.

Last Period

The period for the final recurring invoice.

Next Period

The in which the next recurring invoice will be generated.

Status

The status of the recurring invoice.

Possible statuses are:

  • Template Invoice not found - no template record has been created.
  • Cost Distribution Missing – the template invoice has a Costed distribution type but there are no cost distribution details.
  • Cost Distribution Imbalance – the cost distribution value does not equal the goods amount of the template invoice.
  • Distribution Imbalance – ‘Multi-Lines ’ has been selected on the invoice header but no distribution has been entered.
  • Invoicing Complete – all monthly invoices have been posted.

When you save the details of a new recurring invoice, this creates an invoice template.

Click the link in the Code column to update the template.
