Recurring Invoices Status Report

Recurring Invoices Status is a new report which provides a status report on all recurring invoices. The report can be run from the PL Reports Menu, and allows a range of recurring invoices to be entered.

The report shows the following details of each recurring invoice type:

  • Code
  • Description
  • Supplier Code
  • Supplier Name
  • Supplier Ref Base
  • Start Period
  • End Period
  • Next Period
  • Amount
