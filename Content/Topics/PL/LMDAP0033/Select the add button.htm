Entering Invoice Details

Once you have created the batch header and saved, the Invoices screen is displayed ready for transactions to be entered.

The transaction entry screen displays any transaction entered into the current batch.  For a new batch, this screen will be empty.

Click to add a new invoice

Field

Details

Supplier Account:

Enter the Supplier Account number (if Known) or use the Lookup button to search for the Supplier accounts

PO Number

PO Number is only required for PINV transactions.  For PINN, this should be left blank.

Internal Reference:

This will be automatically generated when the invoice is saved.

Transaction Type:

This will default from the batch header, but may be changed for each invoice within the batch if required.

Analysis:

Select the type of costing analysis for this invoice and use the lookup button on the analysis filed to select the appropriate cost analysis code.  will default this from the Purchase order if specified.

Invoice Date:

Enter the date shown on the invoice.  This will auto-complete the due date and discount date.

Distribution Type

Set this to P- Provisional

Net Amount:

Enter the net amount of the invoice.  The VAT and Gross amounts will be automatically calculated.

 

Click to store the Invoice details and move onto the Provisional Costing Screen.
