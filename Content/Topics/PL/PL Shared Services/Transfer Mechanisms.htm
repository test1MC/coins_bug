Transfer Mechanisms

Whether you are considering a network capable scanning solution or the manual file transfer solution, there are two options supported by for sharing the scanned documents with .

File Transfer Protocol

This is the preferred solution for a environment. File Transfer Protocol, or FTP as it is also known, requires the configuration of a file transfer repository on the server. Any number of repositories can be created to cater for multiple companies and/or environments. Each repository is secured via a user identifier and password.

If utilising a network capable scanning solution, these repositories can be configured within the document scanners being used. End users will not be required to know the specific details of how to connect to a specific repository. User identifiers and passwords will be set up on the document scanning device and hidden from the end users.

If considering the manual file transfer solution, then a third party file transfer utility like Filezilla or WinSCP will need to be installed on the computer of any user required to transfer invoice images. Users will be required to know the particular access details of each file repository that they need to deposit invoice images in.

Network Shares

Another option is to configure network file shares within your corporate environment. These file shares can then be configured to be accessible from the server. In this case user access privileges will be controlled by the platform providing the network share. An appropriate user account must be provided to to enable the software to access the network share and manage the associated invoice images. Any number of sub-directories or network shares can be configured to act as image repositories.

If utilising a network capable scanning solution, these network shares can be configured within the document scanners being used.  End users will not be required to know the specific details of where on the network the images are being deposited. User identifiers and passwords will be set up on the document scanning device and hidden from the end users.

If considering the manual file transfer solution, then users will be required to be assigned the necessary privileges to add invoice image files to the network share. Users can then move invoice image files using the standard Windows File Explorer.
