Operation

Prior to displaying the screen below, carries out the following:

  • Is the current PL period set?  If this is not the case then the following error message is displayed: “No current PL period available for company x”.
  • Is there an unposted Shared Service PL invoice batch (PLINVSS) for the current user and the current PL financial period in the current user’s prime company?  If this is not then case then a batch is created.

The user is taken directly to the list of invoices without the need to create a batch header record.  As many PL Invoice [PLINV] batches as required; are created automatically in the companies derived from the PO Number or Cost Analysis Code.  These batches are marked specifically as Non-scanned Shared Service PLINV batches.

The list of invoices remains available for amendment and/or deletion until such time as the “Post Invoices” option is taken.
