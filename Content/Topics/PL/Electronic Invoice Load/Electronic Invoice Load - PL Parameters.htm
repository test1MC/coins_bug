Electronic Invoice Load - Parameters

 

Parameter

Description
AUTOPOST - Automatic Post Interfaced Invoice Batches This should be set to N so that Invoices loaded by the PL interface procedure will not automatically post.
VATSTOP - Inhibit Auto VAT Calc. on Invoices This should be set to Y so that does not enter the calculated VAT amount when a user enters the VAT code – The VAT amount will be on the import file
