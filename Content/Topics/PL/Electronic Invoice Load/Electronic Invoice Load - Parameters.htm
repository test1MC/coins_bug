Electronic Invoice Load - System Parameters

Parameter

Description

SY/DATADIR

The name of the directory that contains data files to be loaded. Used by GL, SL and PL load procedures. In SureStart this will be set to /tmp

SY/DATAOLD

The name of the directory to which moves data files automatically once they have been loaded. Used by GL, SL and PL load procedures. In SureStart this will be set to ./old
