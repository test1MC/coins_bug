Excel Template

allows you to import transaction data from other applications. The basic steps are:

  1. Export your data into the supplied Excel Template
  2. From the spreadsheet, create the import file in the required format
  3. Use > Invoices > Check Electronic Invoices to check the data file before loading it.
  4. processes the file but does not load the data. If the check runs with no errors, you will be able to load the file. If there are errors, the file may load, but will not post it.
  5. Use > Invoices > Load Electronic Invoices to load the invoices.
  6. loads the data file and creates a transaction batch
  7. If the data loads correctly, then moves the data file to the directory specified by the system parameter DATAOLD.
