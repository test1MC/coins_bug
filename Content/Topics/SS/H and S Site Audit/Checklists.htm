Checklists

Mobile Form Checklists allows you to set up checklists that can be used on mobile forms. These give a list of possible values to select, and provide consistency for reporting.

Mobile Form Checklists

Checklist Items

The Code value is what is actually stored so cannot be changed.

The Description can be reworded without affecting existing data.  Entries should be worded so being selected is consistently good or bad for the whole list.

The value in the True field determines whether the item is shown depending on the score entered. In Figure 1: Mobile Form Checklist Items, if the score is 2, only items C, D and E will be available.
