Health and Safety Parameters

The following new Health and Safety parameters have been introduced:

Parameter Description

CCPEOPLE

This allows you to use Customer Care people as assignees in the Health and Safety Action Monitor, instead of Central Repository personnel and contacts.

If set to Y, assignees can be Staff, Subcontractors or Suppliers.

If set to N, assignees can be Personnel or Contacts.
MFWRKFLO The workflow group that is set up for Health and Safety Audits.
TSKMBODY The text for the body of a task email. Values from the , audit and workflow can be included in the subject, using field names in { }; for example: {job_num} will be replaced by the number. If the value of TSKMBODY is a filename, the content for the email template is read from the file.
TSKMSUBJ The text for the subject of a task email. Values from the , audit and workflow can be included in the subject, using field names in { }; for example: {job_num} will be replaced by the number.
