Timecard Report by Timecard Date

A new Timecard Report by Timecard Date report function has been added to is available in COINS OA Payroll. It allows you to display Timecard data entered by day that can be run by employees.

This report is for the US and Canadian regions only.

Timecard Reports Folder

A new Timecard Reports folder is found in the PR Company Reports menu group, as shown in Timecard Reports Folder. Also, the Timecard Report and Timecard Summary by Day Report have been moved to are found on this menu.

Timecard Reports Folder

Timecard Report by Timecard Date

A new Timecard Report by Timecard Date report option is found in the Company Reports > Timecard Reports menu group, as shown above in Timecard Reports Folder. When you select this option, you can fill out the selection screen, as shown in Selection Screen. The report allows you to display Timecard data entered by day that can be run by employees.

Selection Screen

Field Definitions

Starting/Ending Timecard Date

The range of Timecard Dates to be reported on.

From/To/Matches

Section From/To/Matches

The range of and Sections to be reported on.
