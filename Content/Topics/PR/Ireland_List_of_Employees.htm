Ireland List of Employees

For Ireland as part of the steps towards implementing PAYE Modernisation in Jan 2019 the Revenue want to ensure the employee data on employers’ and on their own systems are in agreement. To help them do this they will be requiring employers to submit a new List of Employees report in 2018. To support this, new employee fields have been added, and a new report provided to produce the required data file.

Employee Maintenance

New fields have been added to the National Details tab.

Field
Description

Concurrency Stream

A sequence number used to distinguish concurrent employments.

Previous Payroll

The company and pay frequency of the previous employee record for this person.

Previous Employee

The previous employee number.

Previous Employee Transition Type
To indicate whether this was a transfer or a separate employment.
Next Payroll
The company and pay frequency of the next employee record for this person.
Next Employee
The next employee number.
Next Employee Transition Type
To indicate whether this was a transfer or a separate employment.
Employment ID
A unique ID required by the Revenue to distinguish separate employments for the same person.
Exclusion Order Start Date
The start date for an exclusion order.
Exclusion Order End Date
The end date for an exclusion order.
Proprietary Director
A tickbox to indicate whether a director is a proprietary director, which is the company’s beneficial owner or a director who can control directly or indirectly more than 15% of the company’s ordinary share capital.

NOTE

The employment history fields (Previous and Next Employee fields, Concurrency Stream and Employment ID) will be initialed automatically.

For more details, consult the side-frame help. Equivalent fields have been added in COINSplus.

Preparation

There is a small amount of data entry for customers to do before producing the file:

  1. Go through current employees who are directors and tick the new Proprietary Director tickbox where applicable.
  2. Go through current employees with an exclusion order and enter the exclusion order start and end dates where known.

ROS List of Employees

The new ROS List of Employee report has been added to the Global Reports menu.

The report selection consists of the employer registration number and the date to run the report for. The report includes all employees who were in employment on the date selected. When run this creates an XML file under My Files for submission to ROS, as well as a PDF report of the employee data for users to look at.

If this report when run produces a status of Complete(with errors), the XML file will not be created or available in My Files. Refer to the Messages section in the report, which lists details of any errors. These will need to be corrected first before recreating the report which will in turn also produce the XML file in My Files.
