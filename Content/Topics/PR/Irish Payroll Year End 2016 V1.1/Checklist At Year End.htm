Checklist:  At Year End

Once you are happy with all of the reconciliations, you will be ready to run the Year End Clearing Routine. Please continue on with steps 10 through 16.

9.  Print P60 for each current Employee.

10. Create XML P35 file for Online Submission via the EOY Workbench.

11. Run the End of Year Clearing Routine in COINSplus or Start New Period in OA once only.

12. Check the payroll has moved successfully into the new tax year 2016.

13. Amend Tax and PRSI Tables.

14. Load Employee Certificates using P2C files provided by ROS or manually update each employee.

15. Update the weekly/monthly pay periods using the Periods Maintenance procedure.

16. Amend global parameters.
