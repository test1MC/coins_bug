Using the EOY Workbench OA

If more than one tax code is set up for the relevant companies (set up against the EGOVCOS parameter), the workbench will display these. In this instance, there is a single Tax reference for both weekly and monthly payroll within the one company.

Create EOY for 2016

Go to Global Functions > ROS Workbench.

ROS Workbench

Click on the Registration number; takes you into a secondary screen within the workbench to create the XML file for online submission.

Click (Add) to add a new submission entry. Enter the Year as 2016 and click (Save). This brings up a standard report selection screen which allows you to generate the return contents using a report process.

The original status of the submission (XML creation batch) will be “incomplete”.

Amend P35 Summary Details

Clicking on the Return Number displays the Employee Forms and Summary Tabs:

Employee Forms Tab

Summary Tab:

The Summary Values tab shows

Field
Description

Taxable Benefits

This figure is updated by the payitem specified in the BIKITEMS parameter.

Medical Insurance

The number of employees for whom medical insurance was paid. This is non-updateable and shows the number of P35L forms with non-zero amounts for medical insurance. This is updated directly from the Employee P35L forms where medical insurance was paid.

Total Amount

The total amount of medical insurance paid included in the taxable benefits amount. This will be updateable when there are employees with medical insurance, but must be at least equal to the total eligible for TRS (tax relief at source) (see below).  This is updated directly from the P35L forms where medical insurance was paid and can be over written if required.

Medical Insurance - Eligible for TRS

The total amount of medical insurance paid included in the taxable benefits amount and eligible for TRS (tax relief at source). This is non-updateable when Year End Batch is at ‘Complete’ status and shows the total medical insurance amount entered on the individual Employee P35L forms.

The Total Amount figure can be amended when the Year End Batch is at ‘Incomplete’ status

Parking Levy

The Summary has the ability to show the car park levy.

Pension Details

Pension details are also displayed and can be updated manually.

Create P35 XML file

The Create file option creates the XML return file once the return has been validated without errors.  Once the file is created, refreshing the screen will show the paper clip icon.
