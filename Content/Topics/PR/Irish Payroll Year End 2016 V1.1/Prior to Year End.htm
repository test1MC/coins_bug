Modify Parameters to Enable Multi-Company P35 XML File Creation

We recommend that you populate the required parameters at this stage.

The payroll parameters function is:

In COINSplus – Global Parameters > Payroll Parameters

In OA – Global Setup > Parameters

The two parameters applicable to this process are listed below:

BIKITEMS – comma-separated list of pay items that hold Benefits In Kind amounts. (Updates Taxable Benefits on P35 Summary Screen)

EGOVCOS – comma-separated list of companies for XML file creation and submission (an ASTERISK * means ALL companies).

Note: The standard plain text report will only contain employees on a particular pay frequency. The printing of plain text P35 is purely for reconciliation purposes.
