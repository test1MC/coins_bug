Objectives

This document forms a formal Irish Payroll End-of-Year instruction manual to assist with and achieve the following:

  • Run procedures prior to year-end
  • Run procedures at year-end
  • Run procedures at start of new year

 

The document has been divided into two main areas:

  1. Checklists which can be used to mark off when each year-end step has been completed
  2. Step by Step Guide with screen shots breaking down each of the year end procedures in detail
