Run the End of Year Clearing Routine in COINSplus

NOTE

This procedure also deletes Pay Item history records when they exceed the number of years to be retained as indicated by the Years History field on each Pay Item. When all Pay Items for an employee have been deleted in this way, the employee record is also deleted. Before running this procedure, please make sure that your Pay Item History fields are set appropriately. To avoid employee deletion set at least one common Pay Item to 99 years.

Run this once only as follows:

COINSplus [Period] Payroll Administration Menu
