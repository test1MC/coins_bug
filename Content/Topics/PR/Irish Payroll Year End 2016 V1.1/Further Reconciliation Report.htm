Further Reconciliation Report Options

If required, print additional reports to investigate and correct any anomalies. Below are a few suggested reports:

Employee Pay Item Details

Run this report to work out period on period tax and total amounts for year.

COINSplus [Period] Payroll Reports Menu

Employee PRSI Details

Reconcile contribution figures using this report also. You will need to select the relevant employee and employer’s PRSI pay item codes, which are defined on the second screen of the Payroll Configuration function.

Employee Selected Items

This reports on employee pay items (to a maximum of ten). You can display department or location totals for each item and the overall total.

COINSplus [Period] Payroll Reports Menu

Employee Selected Item Report Can be configured to show period amount and year to date amount.

Figure 15: OA [Period] Payroll Selected Item Report

Costing Check for Year End Reconciliation

This report will show the difference for both year to date and period on period tax values. To run this report you need to have sufficient access to use the command line within . If you are unable to run this please contact your systems administrator.

To access the command line press the dot key on your keyboard.

Enter the following for the payroll you are working on:

For weekly payroll:

pr365.p (“0”)

Change the “0” to a “1” for monthly, to “2” for fortnightly and to “3” for four weekly.

It should be run for each pay frequency within each company as necessary.

Selection criteria are Tax Year, Check Costs Posted (Y or N) and employee selection.

The report only shows exceptions. Where it finds an exception it will display the employee number and name, as well as a detailed description of the problem.

At the end of the report it shows the total number of employees checked.

Tax and Taxable Pay check

The pay item YTD value matches the previous employment figures plus what has been paid period by period. If the figures do not match you will get detail on which period(s) has discrepancies. There are two possible tax exception messages:

If the item YTD value is out by the exact figure of the previous employment value then:

"Item X YTD is missing the P45 Previous employment figure"

Otherwise:

"Item X YTD does not equal the sum of period values in period Y".

PRSI Check

The total of the gross employee and employer contributions is compared to the combined net contributions held on the employee pay items. Again, if the figures do not match you will get detail on which period(s) has discrepancies. The exception description is:

"Employer/Employee PRSI Details record does not match Item X in period Y"
