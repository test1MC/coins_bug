Load Employee Certificates (Tax Credits and Cut off Points, USC Bands and Method of Calculation)

The Revenue Commissioners will forward by email to the Employer payroll contact, a TFA file (also known as a P2C file) which is now in a csv file format. This file is also available to download within the www.ros.ie web area.

Please note that you MUST have the recommended service packs deployed in the operating environment, to allow the loading of these files with the associated file names; see page 1.

  1. Transfer the CSV file onto the home directory or another user home directory on the UNIX server; for example: /home/coins or /usr/coins.
  2. Go to Weekly (or Monthly) Payroll > Weekly (or Monthly) Administration > Irish Routines.
  3. Go to Load Irish TFAs (load employee Tax cut-off and tax credit certificates).

    Irish Routines Menu

    prompts with the following screen:

    COINSplus Load Irish TFAs Prompt

  4. Enter the directory where the P2C file has been transferred (for example, home/coins) and enter *.csv in the File Pattern field in order to select the P2C file for upload:

    COINSplus Load Irish TFAs Prompt

  5. Press F1.  All CSV files will be listed.

    COINSplus Load Irish TFAs File Selection

  6. Validate the file by highlighting the required filename and pressing Return.

    displays the Error Status.

    COINSplus Load Irish TFAs (File Validated)

  7. If there are any errors or allowable errors press F7 to view them.  An allowable error means a warning, where data can still be loaded. For example, if a date of birth makes an employee over 65, an allowable error is detected but the file can still be loaded successfully.
  8. If there are no errors (or only allowable errors), press F1 to upload the file.
