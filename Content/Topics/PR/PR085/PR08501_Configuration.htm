Configuration

JC parameter JOBCH was added in v11.04 to allow to be configured using Specific .

Parameter

Description

JC/JOBCH

Allow specific

Example

As an example of how this parameter is used, consider the following scenario:

  • JC/JOBCH is set to Yes;
  • The option Allow Non-Standard for 62 with the description Safety is ed;
  • 62 for 000007 has non-standard description Safety NON-STANDARD.

In this case, for 000007 for 62 in the lookups and Payroll Timesheet entry functions, the Group and Group PR Redirection Info 000007 specific description Safety NON-STANDARD will appear instead of the standard description Safety .
