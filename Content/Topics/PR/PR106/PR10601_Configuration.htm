Configuration

New PR parameters have been set up in the Payroll module for use with ETP processing.

Parameter

Description
PR-ETP-MAXUNTAXED

ETP maximum untaxed amount allowed

PR-ETP-MAXAMOUNT

ETP maximum amount allowed in a year

PR-ETP-MAX-ERRTYPE

ETP maximum untaxed amount allowed and maximum amount allowed error type (warning/error)

PR-CHILDSUPPORT-PAYCALC

List of pay calculations denoting child support that should cater to the PEA (Protected Earnings Amount) requirement
