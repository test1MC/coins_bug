Pay Calculation Taxability Report

COINS OA users could not review the Taxable flags set up on all of their Pay Calculations, and there was no way to export that data (as can be done with Federal Taxation information of Payroll Calculations). A new Pay Calculation Taxability Report is found in Payroll and makes it easier for you to review the Taxable flags of Federal, State, and Locality jurisdictions set up on all Pay Calculations.

This report is for the US Region only.

Menu Item

The new Pay Calculation Taxability Report is located in Payroll > Global Reports, as shown in Menu Item.

Menu Item

Report Selection Screen

When you select Pay Calculation Taxability Report from the menu shown above, the selection criteria screen appears, as shown in Report Selection Screen.

Report Selection Screen

Field Definitions

Fringe or Deduction

Select which type of fringe benefit or deduction calculation you wish to report on.

Include Dormant Pay Calculations

Whether to include dormant Pay Calculations on the report.

Pay From/To/Matches

Select the Pay Calculation or a range of Pay Calculations you wish to report on.

Pay Group From/To/Matches

Select the Pay Calculation Group or a range of Pay Calculation Groups you wish to report on.

Report Sample

After you enter the selection criteria on the screen, a Pay Calculation Taxability Report is generated. A sample of that report is shown in Report Sample.

  • Sorting for the report is done by Pay Calculation Code, Jurisdiction (FED, FICA, FUTA, STATE, SUTA, DBL, WC, LOCAL), and Tax Authority Pattern (for STATE, LOCAL).
  • Generation of this report takes into account all US Pay Calculation records.
  • Information about Federal, State, and Local tax flags are received from the Default Nontaxable Data records associated with the master Pay Calculation record.
