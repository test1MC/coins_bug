Timesheet Input Group Administration

Group Administration Screen

A new Group Administration function (Figure 1: Group Administration Screen) is found on the Timesheet Input Groups menu for each Payroll Frequency. It allows you to set up Timesheet Input Groups as frozen/unfrozen for Timesheet Entry and Reset.

  • Only your own groups and subordinate groups can be present on the screen.
  • The browse contains two actions: Freeze Input Groups and Unfreeze Input Groups, which are detailed in Section 1.2below.
Group Administration Screen

Freeze/Unfreeze Timesheet Input Groups

During the payroll process, you may choose to limit the entry or update of time cards in a Timesheet Input Group because the totals have been verified; the back office is ready to process , etc. To facilitate this requirement, the ability to freeze/unfreeze TS Input Groups was developed as is an Administration function.

  • The functions listed below can be performed on Timesheet Input Groups that are frozen.
  • When a Timesheet Input Group is frozen, users are not allowed to add to or update the contents of the group.
  • Calculations
  • Group Validation
  • Accept Payroll
  • Process Payments
  • Print
  • Print Payslips
  • The functions listed below may only be performed on Timesheet Input Groups that are not frozen.
  • Add to or update Time Cards or Overrides
  • Automatic Regular Hour Overtime Processing
  • Blended Overtime Rate Processing
  • Days Worked Processing

The two actions found on the Group Administration function screen are explained below.

  • The Freeze Input Groups action allows you to freeze selected Timesheet Input Groups. A group is skipped if it is already frozen. The screen that appears when you select this action is shown in Figure 2: Freeze Input Groups Screen.
