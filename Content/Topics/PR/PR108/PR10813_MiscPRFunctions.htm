Miscellaneous Payroll Functions

A number of Payroll functions now support TS Input Groups. The relevant functions are listed below. To learn more about how TS Input Groups are supported for a particular function, click the link next to that function.Read this section to learn more about how TS Input Groups are supported for a particular function. Note: Information in this section only applies if the TS Input Groups are in use for the company.

  • PR Calculation
  • PR Reset
  • Accept Payroll
  • Print
  • Print Payslips
  • Output and Print
  • Change Employee Status

PR Calculation

The PR Calculation function in Payroll > Frequency ( PR Calculation Screen) now accommodates TS Input Groups as follows:

  • The Timesheet Input Groups ordered list determines which groups should be processed. It includes all of your own groups, subordinate frozen groups, and subordinate unfrozen groups for the current unposted Regular/Adjusting period.
  • A selected group cannot be processed if it has an employee in multiple Timesheet Input Groups. An error message appears when this occurs.
PR Calculation Screen

PR Reset

The PR Reset function in Payroll > Frequency ( PR Reset Screen) now accommodates TS Input Groups as follows:

  • The Timesheet Input Groups ordered list determines which groups should be processed. It includes all of your own unfrozen groups, subordinate unfrozen groups for the current unposted Regular/Adjusting period, and the blank group.
PR Reset Screen

Accept Payroll

The Accept Payroll function in Payroll > Frequency ( Accept Payroll Screen) now accommodates TS Input Groups as follows:

  • The Timesheet Input Groups ordered list determines which groups should be processed. It includes all of your own frozen or unfrozen groups, subordinate frozen groups, and unfrozen groups for the current unposted Regular/Adjusting period.
  • Only under selected groups are accepted for processing.
Accept Payroll Screen

Print

The Print function in Payroll > Frequency ( Print [%=Variables.Cheques caps%] Screen) now accommodates TS Input Groups as follows:

  • The Timesheet Input Groups ordered list determines which groups should be processed. It includes all of your own groups, subordinate frozen groups, and unfrozen groups for the current unposted Regular/Adjusting period.
  • Only under selected groups are printed.
Print Screen

Print Payslips

The Print Payslips function in Payroll > Frequency ( Print Payslips Screen) now accommodates TS Input Groups as follows:

  • The Timesheet Input Groups ordered list determines which groups should be processed. It includes all of your own groups, subordinate frozen groups, and unfrozen groups for the current unposted Regular/Adjusting period.
  • Only under selected groups are printed.
Print Payslips Screen

EFT Output and Print

The EFT Output and Print function in Payroll > Frequency ( [%=Variables.eft%] Output and Print Screen) now accommodates TS Input Groups as follows:

  • The Timesheet Input Groups ordered list determines which groups should be processed. It includes all of your own groups, subordinate frozen groups, and unfrozen groups for the current unposted Regular/Adjusting period.
  • Only under selected groups are printed. If you run Output and Print by Group, you can produce multiple output files. You can also select all groups to produce one output file.
Output and Print Screen

  • The Timesheet Input Groups ordered list determines which groups should be processed. It includes all of your own groups, subordinate frozen groups, unfrozen groups for the current unposted Regular/Adjusting period, and the blank group.
  • Only under selected groups are processed.
