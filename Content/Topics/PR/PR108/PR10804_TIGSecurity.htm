Timesheet Input Group Security

The Timesheet Input Group Security function allows you to create a comma-separated list of subordinate users to determine who can access Timesheet Input Groups (other than the owner).

Menu Option

A new Timesheet Input Group Security menu option is found on the Payroll > Company Setup > Security menu, as shown in Figure 1: Menu Option.

Menu Option

Browse Screen

When you select Timesheet Input Group Security from the menu, a browse screen appears, as shown in Figure 2: Browse Screen.

Browse Screen

Field Definitions

User ID

The ID of the COINS user.

Name

The name of the user or group with this ID.

Subordinates

This is a comma-separated list of subordinate users. You cannot be your own subordinate since the subordination should be hierarchical. However, two users could be listed as subordinates to each other, and therefore have access to each other's groups.

  • An asterisk (*) value means that all groups and timesheets are available for the user.
  • As owner, you have access to the timesheets under your own TS Input Groups and under the TS Input Groups of your subordinates.
  • A blank value means that you do not have subordinates and therefore only have access to your own groups.
