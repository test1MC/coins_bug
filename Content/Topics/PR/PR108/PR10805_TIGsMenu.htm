Timesheet Input Groups Menu

A new Timesheet Input Groups menu option is located in Payroll > [selected Frequency], as shown in Figure 1: Timesheet Input Groups Menu. The following three options are found on this menu when a company is configured to use Timesheet Input Groups:

  • Group Maintenance – see Section 1 PR Timesheet Input Groups: Timesheet Input Group Maintenance for details
  • Group Merge – see Section 1 PR Timesheet Input Groups: Timesheet Input Group Merge for details
  • Group Administration – see Section 1 PR Timesheet Input Groups: Timesheet Input Group Administration for details
