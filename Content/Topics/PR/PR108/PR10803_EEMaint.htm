Employee Maintenance

For v11.08, the following changes were made on the General tab of PR Employee Maintenance, as shown in Figure 1: General Tab Screen:

  • The label of the Timesheets Input Group field was changed to ESS Timesheets Input Group to allow for Employee Self Service.
  • A new Timesheet Input Group Merge Code field is only available if the Timesheet Input Groups are in use for the company. This field is completely optional and can be used to select a code for merging Timesheet Input Groups. It is also an additional way to group employees. The lookup for this field uses Timesheet Input Group Merge Codes Maintenance.
