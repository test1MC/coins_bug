Manual Calculator

Timesheet Input Groups are not applicable for in the Manual Calculator (MCC) function. However, several changes made to MCC apply only if Timesheet Input Groups are in use for the company. These changes are discussed below.

Convert to Manual for MCC

The Selected Employee action screen in the Convert to Manual function for MCC in Payroll > Frequency > Manual Calculator > Convert to Manual action ( MCC Convert to Manual [%=Variables.Cheques caps%] for Selected Employee Screen) now accommodates TS Input Groups.

  • A Convert to Group field determines the target Timesheet Input Group for processing. The converted are under this target group.
    • The target Timesheet Input Group does not need to be on file; if it is not on file, a new Timesheet Input Group is created.
    • Whoever runs the process becomes the owner of the newly created Timesheet Input Group.
    • If the group exists, it should be an unfrozen group associated with the entered unposted Regular, Adjusting, or Future period for the PR Frequency; otherwise, it should be not associated with any periods. This should be your own or a subordinate group.
    • This field cannot be blank.
  • A Group Description field: If the target Timesheet Input Group for the Convert Timesheets is not on file, a new Timesheet Input Group is created. The description is assigned to the new Timesheet Input Group; otherwise, this field is ignored.

Quick Add Time Cards

The Quick Add Time Cards tab in the Convert to Manual function for MCC in Payroll > Frequency > Manual Calculator ( MCC Quick Add Time Cards Screen) now accommodates TS Input Groups as outlined below.

  • A Group column, Multiple Groups box, and Frozen Group box are now available on the MCC Quick Add Time Cards tab screen.
  • The following items appear in the named filters on the screen: All Groups/My Groups/All Non-Blank Groups/Blank Group/Multiple Groups/Frozen Groups/Unfrozen Groups.
  • Only Time Card records under the blank group, your own group, and the subordinate groups are available on this screen.
  • You cannot create time cards for the Manual Calculator from Future, Regular, or Adjusting time cards if they are under a frozen group or multiple groups.
