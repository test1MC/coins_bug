Timesheet Merge Codes

A new TS Input Group Merge Codes function is now available in the Payroll > Global Setup menu, as shown in Figure 1: Timesheet Input Group Merge Codes Maintenance screen. The codes set up here can be entered in the Timesheet Input Group Merge Code field on the General tab of Employee Maintenance and then used as an additional way to group employees when merging Timesheet Input Groups.

NOTE:

This function is only available if Timesheet Input Groups are in use for the Company.
Timesheet Input Group Merge Codes Maintenance screen

Field Definitions

Code

The Timesheet Input Group merge code. It is a mandatory field. If you try to enter a blank code, the following error message appears: The Timesheet Input Group Merge Code must be entered.

Description

A short description of the Timesheet Input Group merge code.
