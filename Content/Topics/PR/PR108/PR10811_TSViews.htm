Timesheet Views

Timesheet views in Payroll now support Timesheet Input Groups. The relevant functions are explained in detail below.

Timesheet Entry Views

Timesheet Input Groups are used in Timesheet Entry views, as discussed below.

Timesheet Detail Tab

The Timesheet Detail tab in Payroll > Frequency > Input > Timesheet Entry now includes the following:

  • A Group column on the Timesheet Detail screen (Figure 2: Timesheet Detail Tab)
  • If TS Input Groups are in use, the following items appear on the screen:
    • Filters: All Groups/My Groups/All Non-Blank Groups/Blank Group/Multiple Groups/Frozen Groups/Unfrozen Groups (Figure 1: Group Filters)
Group Filters

  • A new Group Browse filter

Only time card records under the blank group, your own group, and subordinate groups are available for you on this screen.

Timesheet Detail Tab

Daily Hours Detail Tab

The Daily Hours Detail tab in Payroll > Frequency > Input > Timesheet Entry now includes the following:

  • A Group column on the Daily Hours Detail screen (Figure 4: Daily Hours Detail Tab)
  • If TS Input Groups are in use, the following items appear on the screen:
    • Filters: All Groups/My Groups/All Non-Blank Groups/Blank Group/Multiple Groups/Frozen Groups/Unfrozen Groups (see Figure 1: Group Filters above)
    • A new Exclude Recurring option under the Period Types named filter (Figure 3: Period Types Filter)
Period Types Filter

  • A new Group Browse filter

Only time card records under the blank group, your own group, and subordinate groups are available for you on this screen.

Period Maintenance Views

When TS Input Groups are in use for the company, some tabs in PR Period Maintenance correspond to tabs found in Timesheet Entry. These tabs are: Timesheet Detail, Daily Hours Detail, Net Detail, Burden Detail, Pay Calculation Detail, Overrides, and Payroll Redirection.

NOTE:

TS Input Groups are only applicable in Period Maintenance for unposted Regular/Adjusting periods and non-MCC Future periods.
