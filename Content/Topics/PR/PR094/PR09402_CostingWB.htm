Costing Workbench

Changes made in the Costing Workbench in v11.07 are shown below.

PL Analysis Report

The PL Analysis Report generated in Payroll > [payroll frequency] > Costing Workbench now includes Invoices created from Calculations.

Validate Batches

The Validate Batches action run in Payroll > [payroll frequency] > Costing Workbench now validates Invoices that are created from Calculations. Only Deductions and Fringes Not Included in Pay are processed.

Post Batches

The Post Batches action run in Payroll > [payroll frequency] > Costing Workbench now creates Invoices from Calculations. Previously, Create Invoices only retrieved this information from the Employee Cost record for Pay Calculations.
