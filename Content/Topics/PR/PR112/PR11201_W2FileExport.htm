W-2 File Export

COINS OA users needed the ability to export W-2 information from OA so that other employees could have access to view their W-2 data online. A new W-2 File Export function is therefore now available in the Payroll module to provide users with the ability to view W-2 information inside a spreadsheet.

This document describes changes made in COINS OA to provide COINS OA now provides you with the ability to export W-2 information. These changes apply This applies to the US Region only.

Menu Option

A new W-2 File Export function is found in Payroll > Company Setup > W-2 Processing, as shown in Figure 1: Menu Option.

Menu Option

Selection Screen

When you select W-2 File Export from the W-2 Processing menu, the selection screen appears, as shown in Figure 2: Selection Screen.

Selection Screen

Field Definitions

Batch From/To

Enter the range of batches you would like to include in the W-2 export file.

Output File Name

Enter the file name of the CSV spreadsheet to which the W-2 file will be exported. The name defaults in as w2file.csv but can be changed.

CSV Export File

When you click the Next button at the bottom of the selection screen, a CSV export file is generated, using the name you gave it in the Output File Name field. The generated file can be found in System > My Files (see Figure 3: My Files (in System)) or in Report Status > My Files tab (see Figure 4: My Files (in Report Status)).
