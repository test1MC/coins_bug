Overview

An employee's address change information should be reflected in the State information for both Work and Home States. These changes often affect an Employee's payroll taxes at the State and Locality level. COINS OA Payroll therefore now allows users to set up and update an Employee's State Detail information if the Home Address State is changed.

This document describes changes made in the Payroll module to allow this functionality to work correctly. This functionality applies to the US and Canadian Regions only.
