Configuration

Although there are no new parameters in COINS OA for printing W-2 Forms, some existing parameters are now being used in relevant functions, as listed below.

  • Enable Emailing Forms - Payslips use Global PR parameter DEFPRN. W-2 Forms use the Employee Print Method - Email prompt, which is checked at run time.
  • Assign Password to PDF - Payslips and W-2 Forms use the same Global PR parameter USESLIPPWD.
NOTE:
There is an option to include debugging information in the Report Status Log File, which is turned on by adding DEBUG to the value of the USESLIPPWD parameter. Under normal use, USESLIPPWD = Y, N, or YES. In debug mode, USESLIPPWD = YDEBUG, NDEBUG, or YESDEBUG, etc.
  • From Email Address - Payslips use the period specific parameter MAILFROM or the Global PR parameter MAILFROM. W-2 Forms only use the Global PR parameter MAILFROM.
  • BCC Email Address - Payslips use the period specific parameter MAILBCC or the Global PR parameter MAILBCC. W-2 Forms use only the Global PR parameter MAILBCC.
  • Email Subject - Payslips use the period specific parameter MAILSUBJ or the Global PR parameter MAILSUBJ. W-2 Forms use COINS PR W2 Form for Year + YEAR.
