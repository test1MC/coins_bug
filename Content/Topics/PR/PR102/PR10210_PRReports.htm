PR Reports

This section describes the Payroll reports using the date.

Company Reports

The following Company Reports now allow for the Date to be used:

  • Employee Earnings Detail History – All sections display the Date next to the Period Date.
  • Payroll Register – Displays the Date from the PR rather than from the PR Period.
  • Quarterly PR Returns - Form 941 - Schedule B – Displays the Date from the PR rather than from the PR Period.
  • Company Timecard Report – The Header 4 Form section of the PR Company Timecard Report includes a Number and Bank Number. It now also includes the Date.
  • Timecard Summary by Day Report - Includes the Date with the Number.

PR Frequency Reports

The following PR Frequency Reports now include the Date with the Number:

  • Timecard Report (by Employee version)
  • Edit Report (Detail version)

PR Costing Workbench Reports

The Edit Detail Report of the PR Frequency Costing Workbench now includes the Date with the Number.
