PR Frequency Timesheet Input

A new Date column is found on the Timesheet Input tab of Payroll > [PR Frequency] > Timesheets, as shown in Timesheet Input. This field is mandatory for Regular and Adjusting period types; it is disabled and blank for Future and Recurring period types.

Timesheet Input

For Regular and Adjusting period types, the process for the Date is as follows:

  1. For Computer and Adjusting , the Date is set from the period’s Date, and the field is disabled for manual changes.
    • For the Pay Method, the process remains the same throughout the payroll process.
    • For the Pay Method, the Date is set from the Print function. The Date was previously entered when printing ; the Print process was updated to change the Date.
  1. For the Manual type, the default comes from the period’s Date. This field is enabled for manual updating.
