Move Timecards to Manual Calculator

COINS OA users can now move timecards from any period (Regular, Future, or Recurring) to the Manual Calculator in the Payroll module. These timecards are in the Start position; they cannot be calculated or processed beyond entry.

This is for the USA region only.

Quick Add Time Cards Tab

A Quick Add Time Cards tab has been added to is available in the Manual Calculator (MCC) screen in Payroll > [Payroll frequency] > Input. This tab provides you with a list of all unposted Regular, Future, and Recurring timecards for the current Payroll frequency (Weekly, Monthly, Twice Monthly, 2-Weekly, 4-Weekly), as shown in Quick Add Time Cards Tab. These timecards can be selected and then added or transferred to the MCC Period.

Note:

Timecards for Adjusting are not available on this screen.
Quick Add Time Cards Tab

The Filter options at the bottom of this screen allow you to filter by All Timecards, Regular Timecards Only, or Exclude Recurring.

Create Timecard Action

The Create Timecard action at the bottom of the Quick Add Time Cards screen ( Create Timecard Action) allows you to move/copy timecards from the Regular, Future, or Recurring Period to the MCC Period.

Create Timecard Action

  • If the timecard is coming from a Regular or Future Period, it is moved to the MCC Period (that is, it is deleted from the Regular/Future Period and added to the MCC Period).

    When the Create Timecard action is run, Regular transactions that are moved still remain with employees in START mode. When a timecard is ready to be processed, the employee may no longer be in START mode.

  • If the timecard is coming from a Recurring Period, it is copied to the MCC Period (that is, not deleted from the Recurring Period).
  • The timecard transaction number is changed when moved if the Employee in the MCC Period already has a transaction number. In that case, the next available transaction number is used.
  • If any overrides are entered for a timecard, they remain with the timecard. When a timecard is added to the MCC and its transaction number is changed, the override is also changed.

Convert to Manual for Selected Employee

A new Terminate Employee box was added to is available on the Convert to Manual for Selected Employee function, as shown in Terminate Employee Fields. This function can be reached by (1) going to Payroll > [Payroll frequency] > Input > Manual Calculator, (2) clicking the Period link to open the Manual Calculator Input function, (3) selecting an Employee, and (4) applying the Convert to Manual action.

When you the Terminate Employee field, new and Reason fields become available for updating.
