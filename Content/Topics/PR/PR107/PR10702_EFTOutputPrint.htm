Output and Print

A new Use Bank Account on Timesheet box is found on the Output and Print screen. This box is ed by default (see [%=Variables.eft%] Output and Print (Box [%=Variables.tick%]ed)). If it is uned, a new From Bank Account combo box appears (see [%=Variables.eft%] Output and Print (Box un[%=Variables.tick%]ed)), which shows the Company Bank Accounts configured against PR parameter BNKONCHK.

Output and Print (Box ed)

Output and Print (Box uned)

When a Bank code from this combo is selected, all of the timesheet Bank Accounts are updated with that code. When the process is run, this Bank code is found in the From Bank Account field.

NOTE:

If the BACSFILE parameter does not have [IBANBANK] in its filename, the Use Bank Account on Timesheet and From Bank Account fields discussed above are not shown. Instead, the From Bank Account code set in the PR/CASHBANK parameter is used.
