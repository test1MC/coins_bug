Apprenticeship Levy - PAYE Scheme for Separate Employer

A new field has been added to the PAYE Scheme tab of the HMRC Workbench.

Field
Description

Separate Employer

If you use a single system to operate payrolls for an additional employer, this allows you to indicate that a PAYE scheme is for that separate employer (where the separate employer only has a single PAYE scheme). This allows the different employers to each have their own full apprenticeship levy allowance, and also their own employment allowance.
