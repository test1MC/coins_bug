Maintain Employee Pension Schemes

 

This topic details the steps required to maintain the pension scheme(s) the employee is a member of.

Select the employee record to be maintained by clicking on the employee number hyperlink.

Select the Pensions tab.

To join an employee to a new pension scheme Click and fill in the details of the appropriate fields. Or use the Open button on the line of the pension scheme you need to edit.

When Pension Auto Enrolment is correctly setup on , the auto enrolment scheme is added automatically.
Also when adding the Pension Auto-Enrolment scheme for employee, who is opting in, please consider adding the pension via Opt-In Notification on the Employee Notifications screen (the notification will automatically add the pension scheme on the employee record and populate the Joined date).
This is mandatory to display correct data in the Payment Data File Output (as well as keeping records of processes within pension auto-enrolment), as is creating an Opt-Out Notification for employees opting out of the autoenrolment scheme (the notification automatically populates the Left date on the pension scheme) opposed to just entering the Left date manually.

 

When adding a new pension the pension scheme Number and the pension Joined date are mandatory fields.

Field

Description
Number Pension scheme number. You can use the lookup option to find the appropriate pension scheme. When Pension Auto Enrolment is correctly set up, the auto-enrolment pension scheme is added automatically.
Description The pension scheme description.
Membership The employee's membership ID on this pension scheme.
Joined The date the employee joined the scheme. When creating an Opt-In Notification, the join date is automatically populated.
Left The date the employee left the scheme. When creating an Opt-Out Notification, the Left date is automatically populated.
EE Fixed If the employee’s pension contribution is a fixed amount each payroll period, enter a gross pension contribution to be deducted. The same applies to a salary sacrifice (salary exchange) contribution.
EE % If the employee’s pension contribution each payroll period is calculated based on a percentage, enter the percentage of pension contribution. The same applies to a salary sacrifice (salary exchange) contribution.
ER Fixed If the employer’s pension contribution is a fixed amount each payroll period, enter a gross pension contribution to be deducted.
ER % If the employer’s pension contribution each payroll period is calculated based on a percentage, enter the percentage of pension contribution.
AVC Additional Voluntary Contribution. If the associated pension scheme allows AVC from an employee, enter a gross additional contribution to be deducted each payroll period, as a top up to a regular pension contribution.
AVC % If the associated pension scheme allows AVC from an employee, enter the percentage for the additional contribution to be deducted each payroll period, as a top up to a regular pension contribution.

 

Any combination of the pension contributions can be entered when fields are available for update (defined by the pension scheme configuration). Keep in mind, when adding both EE Fixed and EE % values, will deduct both.

 

Click at the beginning of the line to store the entered data.

We suggest you end a membership in a pension scheme (enter Left date or create an Opt-Out Notification) every time the conditions of the membership change. [You might need to create a new pension scheme to do that – enquire about consultancy or refer to the guideHow to Configure Pensions].

 

To keep historical records, we advise against deleting pension schemes even though Delete button is available.
