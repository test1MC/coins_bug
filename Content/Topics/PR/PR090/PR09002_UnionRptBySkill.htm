Union Report By Skill

Menu Item

The new Union Report By Skill function is available on the Payroll > Company Reports menu, as shown in Menu Item. It allows you to report on Union information gathered on reports (such as the Union Calculation Report) and breaks it down by Skill Level.

Menu Item

Selection Screen

When you select Union Report By Skill from the PR Company Reports menu, a selection screen appears, as shown in Selection Screen.

Selection Screen

Field Definitions (Selected Fields)

Date From/To

Starting and Ending dates are used instead of Period and Year options.

Print Union Summary

Indicate whether you wish to print a Union Summary section on the report, which includes a summary of fringes and deductions.

Deduction Group 1-5

Options for Union and Pay Calculation Groups are available to determine which fringe/deduction columns should be printed on the report. Only a Calculation Group set with Group/Pay Type as UNION is valid for the Union/Pay Calculation Group.

Skill

This field is found on the Employee Selection tab and allows you to select Skill Levels to report on.
