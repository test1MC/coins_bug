Calculation Groups

COINS OA had no simple way to determine if a Union Calculation was designated for different purposes, such as 401k and other possible benefits. To remedy this, the Calculation Groups functionality can now be used to group calculations so that employee information can be accurately reported. Multiple calculations can exist for the same type of fringes and deductions while streamlining reporting.

Calculation Groups can be set up by going to Global Setup > US Specifics > Calculation Groups, as shown in Calculation Groups. Groups can be created here for use by any of the Group Types, such as Pay Calculation or Union Calculation.

Union Calculation Groups

Union Calculations are found on the Union Calculations tab in Payroll > Global Setup > Unions, as shown in Union Calculation Groups. They can be marked as part of a specific Group Code Type; for example, Union US-CARP is shown in the screen below with Group Code UD.

Pay Calculation Groups

Pay Calculations are found in Payroll > Global Setup > US Specifics > Pay Calculations, as shown in Pay Calculation Groups. They can be marked as part of a group from the Pay Calculation screen; for example, Union US-CARP is shown in the screen below with Calculation Group Code PUD (Pay Calc Union Dues).
