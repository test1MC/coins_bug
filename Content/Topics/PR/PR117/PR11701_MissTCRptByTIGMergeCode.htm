Field Definitions (Selected Fields)

Period Type

This field now only includes the Period Types (Regular/Adjusting and Future) of the unposted non-MCC (Manual Calculator) periods present in Payroll frequency Period Maintenance.

Periods

If the Period Type is set to Regular or Adjusting, the Periods field is hidden, and the available list of Timesheet Input Groups is populated with Regular/Adjusting groups.

If the Period Type is set to Future, the Periods field is available for the entry of a single period or a comma-separated period list. If one of the entered periods is incorrect, an error message is displayed, as shown in Figure 3: Report Error Message.
