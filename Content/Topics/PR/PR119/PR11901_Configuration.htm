ESS Start and Stop Times: Configuration

Three new parameters are available in the ES module to provide Start and Stop Time functionality.

Parameter

Description
ESSCONF

Message displayed for confirmation when user submits reporting.

PTOTYPE

Comma-separated list of ESS Pay Types included in Paid Time Off (PTO). Hours recorded by these pay types will not be checked against entered Start and Stop Time.

EMLFROM

Send From Email Address for ESS Notifications.
