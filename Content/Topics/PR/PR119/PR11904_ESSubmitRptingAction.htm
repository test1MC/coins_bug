ES Submit Reporting Action

Parameter ESSCONF allows you to display a Message for confirmation when a user submits reporting.

Submit Without Confirmation

When ESSCONF is blank, the My ESS Timesheets Summary:

  • Does not display a Signed column; only the columns below are displayed.

  • Submits timesheets without a Message to Confirm.
  • Includes the following Actions list options:
  • Submit Timesheets
  • Reject

E-Signature

When ESSCONF is not blank:

  • The Signed column is displayed, as shown below.

  • The Actions list has the following options:
  • Confirm and Submit
  • Reject - This new action allows you to perform the Reject confirmation process, as outlined below.
  1. Select Reject Submitted Timesheets.

  1. When the Warning message appears, click OK to reject reporting.

  1. If there is nothing to reject, the error below appears.

  • A Confirmation form is available, as shown below. You must the box in the Signature layout area to submit the form.
