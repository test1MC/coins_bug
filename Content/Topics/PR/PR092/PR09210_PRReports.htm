PR Frequency Timecard Report

The Timecard Report generated by Employee or for any Frequency (Weekly, Monthly, etc.) can now process Calculations as well as Prevailing Wage Calculations.

  • If a Timecard has Calculations only, the Trade and Skill columns appear as usual.
  • If a Timecard has PW Calculations, the Trade and Skill columns have appropriate PW Trade and PW Skill records, as shown in PR Timecard Report by Employee and PR Timecard Report by [%=Variables.Jc Contract caps%].

PR Frequency Edit Report

The Edit Report generated by Summary or Detail for any Frequency (Weekly, Monthly, etc.) can now process Calculations as well as Prevailing Wage Calculations.

For the Summary version of the report ( PR Summary Edit Report), text has been replaced in the Net Detail section as follows:

  • “CONTRACT FRINGES PAID TO EMP” replaced with “PREV WAGE FRINGES PAID TO EMP”
  • “CONTRACT DEDUCTIONS” replaced with “PREV WAGE DEDUCTIONS”
  • “EMPLOYER PAID CONTRACT FRINGES” replaced with “EMPLOYER PAID PREV WAGE FRINGES”

For the Detail version of the report ( PR Detail Edit Report):

  • If a Timecard has Calculations only, the Trade and Skill columns appear as usual.
  • If a Timecard has PW Calculations, the Trade and Skill columns have appropriate PW Trade and PW Skill records.
  • In the Net section of the report, the full description of Calculations has the following: Number, Trade, Calculation Number, and Calculation Description. For Prevailing Wage Calculations, the full description has Number, PW Trade, PW Calculation Number, and PW Calculation Description.

PR Company and Global Calculation Reports

Selection Screens

Calculation Type has been changed to Prevailing Wage on the selection screens of the PR Company Calculation Report ( Company Calculation Report Selection Screen) and PR Global Calculation Report ( Global Calculation Report Selection Screen).

Reports

The PR Company and Global Calculation Reports can now process Calculations as well as Prevailing Wage Calculations.

For the Company Calculation Report (see Company Calculation Report):

  • If a Timecard has Calculations only, the Group, Trade, Skill, and Employee Skill columns appear as usual.
  • If a Timecard has PW Calculations, the Group, Trade, Skill, and Employee Skill columns have appropriate PW Calculation Group, PW Trade, PW Skill, and Employee PW Skill records.

For the Global Calculation Report (see Global Calculation Report):

  • If a Timecard has Calculations only, the Trade column appears as usual.
  • If a Timecard has PW Calculations, the Trade column has the appropriate PW Trade record.

Prevailing Wage Reconciliation Report

The PR Prevailing Wage Reconciliation Report can now process Calculations as well as Prevailing Wage Calculations, as shown in Prevailing Wage Reconciliation Report. Other changes that have been made include: (1) the Calc column has been renamed as Prev Wage Calc, and (2) Default PW Trade and Default PW Skill have been added to the header of the Employee Section of the report.

  • If a Timecard has Calculations only, the Trade and Skill columns appear as usual.
  • If a Timecard has PW Calculations, the Trade and Skill columns have the appropriate PW Trade and PW Skill records.
