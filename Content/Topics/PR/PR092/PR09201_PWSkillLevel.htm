Prevailing Wage Skill Level

A new Prevailing Wage Skill Levels maintenance function is found in Payroll > Global Setup. It allows you to set up skill levels for Prevailing Wages used in conjunction with Prevailing Wage Trades and Prevailing Wage Schedules. A sample screen is shown in Prevailing Wage Skill Levels.

  • The Skill Level column cannot be blank.
  • Prevailing Wage Skills are not company specific.
