Prevailing Wage Schedules

A new Prevailing Wage Schedules maintenance function is found in Payroll > Company Setup, as shown in Prevailing Wage Schedules. It allows you to create Prevailing Wage Schedules for Payroll Calculations. Each schedule contains its own PW Calculations and Rates.

  • The Schedule Code cannot be blank.
  • The Prevailing Wage Schedules are company specific.
  • The Schedule Code has a hyperlink that directs you to the Prevailing Wage Calculations screen, as discussed in the Prevailing Wage Calculations Screen section.
Prevailing Wage Schedules

Prevailing Wage Calculations Screen

A new Prevailing Wage Calculations maintenance function is reached by clicking the Schedule Code hyperlink on the Prevailing Wages Schedules screen ( Prevailing Wage Schedules above). It appears as the first of two tabs, as shown in Prevailing Wage Calculations Screen, and allows you to set up calculations for Prevailing Wages.

Prevailing Wage Calculations Screen

Field Definitions

Trade

This is the PW Trade code. These codes are set up in Payroll > Global Setup > Prevailing Wage Trades.

Calc No.

This is the number code of the calculation used for the Prevailing Wage. It has a hyperlink that directs you to the Prevailing Wage Calculations Summary screen, as described in the Prevailing Wage Calculations Summary section Section 1.1.1 Prevailing Wage Calculations Summary.

When a calculation has new calculation rates, effective after the original date, copy the original calculation number and update the Effective Date and rates. Do NOT create another calculation or it will use both calculations when processing payroll. It is recommended not to make the original calculation dormant. It will not be accessible if a timecard is entered for the prior effective period.

Effective Date

The date that the calculation is effective.

Description

A description of what the calculation is used for.

Fringe or Deduction

Whether this is a Fringe or a Deduction calculation. The options are: Fringe Included in Pay, Fringe Not Included in Pay, Deduction.

Group Code

The PW Group Code, which can be blank or one of a valid set of codes for the Prevailing Wage. These Group Codes are set up in Payroll > Global Setup > US Specifics > Calculation Groups.

Percent

The percentage of the calculation, if applicable.

Flat Amount

The flat amount for the calculation, if applicable.

Rate per Day

The rate per day if the calculation is not a flat amount or a percentage of the gross amount.

Amount per Regular Hour
Amount per Overtime Hour
Amount per Double Time Hour

The calculations as amounts per Regular, Overtime, and Double Time Hour.

Item

The Pay Item number, 1 - 999.

Dormant

Whether the calculation is dormant.

Prevailing Wage Calculations Summary

When you click the Calc No column hyperlink on the Prevailing Wage Calculations screen, a new Prevailing Wage Calculations Summary screen appears, as shown in Prevailing Wage Calculations Summary. It also appears if you use the Add, Copy, or Update buttons on this screen.

Prevailing Wage Calculations Summary

The Prevailing Wage Calculations Summary screen contains the following 3 tabs:

  • Main
  • Participation Rules
  • Taxable Details - This tab is not available when using Add or Copy for a PW Calculation.

Prevailing Wage Rates

A new Prevailing Wage Rates maintenance function is reached by clicking the Schedule Code hyperlink on the Prevailing Wages Schedules screen ( Prevailing Wage Schedules). It appears as the second of two tabs, as shown in Prevailing Wage Rates, and allows you to set up and maintain Prevailing Wage Rates for specific PW Trade/PW Skill Level combinations for applicable Earning Categories.

These rates are used on jobs for employees who have been associated with this Prevailing Wage Schedule.

Prevailing Wage Rates

Field Definitions (Selected Fields)

Trade

The PW Trade must be entered.

Prevailing Wage Trades are set up in Payroll > Global Setup > Prevailing Wage Trades (see Section 1, Prevailing Wage Trades the Prevailing Wage Trades section).

Skill Level Pattern

The PW Skill Level of the employee must be entered. This is used with a PW Schedule and PW Trade to determine Prevailing Wage rates/calculations.

Valid PW Skill Levels are set up in Payroll > Global Setup > Prevailing Wage Skill Levels (see Section 1, Prevailing Wage Skill Level the Prevailing Wage Skill Level section).

Effective Date

The date the PW Rate becomes effective.

Description

A description of the PW Rate for the Earning Category.

Dormant

Whether this PW Rate is dormant.
