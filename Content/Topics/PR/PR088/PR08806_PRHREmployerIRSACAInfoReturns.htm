PR/HR Employer IRS ACA Information Returns

The fields listed below were added to the 1095-C table (pr_1095c). The Employer IRS ACA Information Returns function found on both the PR and HR ACA Processing menu has been updated to fill fills in these fields as an employee 1095-C record is processed and its data placed in an AIR transmission XML file. This data helps users with their AIR transmissions to the IRS.

Field

Format

Description

ahd_AIRcrbywho

CHAR x(10)

The userid of the user that created the AIR xml file containing this 1095-C’s data.

ahd_AIRcrdate

DATE 99/99/99

The date when this 1095-C’s data was placed in an AIR xml file.

ahd_AIRcrtime

CHAR x(8)

The time when this 1095-C’s data was placed in an AIR xml file.

ahd_AIR

CHAR x(2)

X, for xml files created, T, for transmitted/uploaded, A for accepted, R for rejected, C for correction, etc.
