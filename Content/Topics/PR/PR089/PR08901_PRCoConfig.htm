PR Company Configuration

Other Tab

A new field labeled Base Expense on Timecard Date is found in Payroll > Company Setup > Company Configuration > Other tab, as shown in Other Tab. It lets you determine whether the Expense Date should be based on the Timecard Date.

In addition, the layout labeled Timesheet Default on this screen has been renamed Timesheet Setup.

Other Tab

  • The Base Expense field can only be updated if there are no timecards entered for the unposted regular or adjusting period.
  • The Timecard Expense Date is only set up when the Base Expense field is ed.
  • The Timecard Expense Date calculates all timecards when the Base Expense field is ed; otherwise, it is empty for all timecards.

Significant Items Tab

A new significant item labeled Accrued Payroll for the Accrued Payroll Pay is available on the General Pay Items layout in Payroll > Company Setup > Company Configuration > Significant Items tab, as shown in Significant Items Tab. It is only available only if the Base Expense Fiscal Period on Timecard Date field is ed. The item cannot be changed if there are employees with an already calculated payroll.

In addition, the Standard Rate Contra Pay field on this screen has been relabeled as Standard Rate Contra.
