Gender Pay Reporting - Employee Maintenance

A new field has been added to the Employee Maintenance screen.

Field

Field
Description

Weekly Working Hours

This is required to calculate the hourly pay for salaried employees.

For hourly-paid employees a calculation based on a 12-week average must be used, so whilst the field is available for these employees, it is not used in calculating the gender pay gap statistics.

This field is validated against the RTI Normal Hours Worked Per Week hours band field to make sure the two are consistent. The Weekly Working Hours field links via Payroll-HR linking to the existing Hours field in HR for the same thing. The new Payroll field will initiale itself to be the same as the HR field value (where there is a linked HR record) and failing this set a best guess based on the RTI hours band.

Over time an employee’s weekly working hours may change so the current value will be stored against each period on the payslip record, as will the hourly/salary paid field. In this way the gender pay gap reports will be able to produce results.

As these changes will not be delivered in time for the first snapshot date of the 5th April 2017 any changes to this field are effectively back-dated to the period including the 5th April 2017.
