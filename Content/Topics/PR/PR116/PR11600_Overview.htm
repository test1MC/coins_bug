PR Rates by Employee: Overview

Payroll has been enhanced to can define the defaulting of pay rates for an Employee in the Employee File. The Default Rate From field in Company Setup > Company Configuration > Other tab allows COINS OA users to set the default value when adding new employees. This functionality affects all Timesheet entry functions, including Quick Adds, Transfer Timecard functions, and Load Timesheets.

These changes are for the US and Canadian regions. F urther details about changes made to provide this PR functionality are found in this document.
