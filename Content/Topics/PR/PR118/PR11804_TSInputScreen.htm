Timesheet Input Screen

The Multiple Groups column on the Timesheet Input screen in Payroll > [frequency] > Input > Timesheets can now be ed if an Employee has several under single but different groups. Previously, it could only be ed if the Employee had a under multiple groups.

For example, Figure 1: Timesheet Input Screen shows that employee 000063 has 1 under the multiple groups GR1 and GR2. Employee 000081 has 1 under a single group GR1 and 2 under a single group GR2. In both cases, the Multiple Groups column is ed.
