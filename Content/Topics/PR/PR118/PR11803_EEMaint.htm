Employee Maintenance

In Payroll > Employee Maintenance > Employee Details > General tab, an Employee must be in START mode in order to update the Timesheet Input Group Merge Code field, as shown in Figure 1: Merge Code for Employee in START Mode. Otherwise, the Timesheet Input Group Merge Code field cannot be updated, as shown in Figure 2: Merge Code for Employee Not in START Mode (such as CALCED) with the example of an Employee in CALCED mode.

NOTE:
The Timesheet Input Group Merge Code field can be blank on this screen.
