PR Reports and Processes

From/To/Matches Selection Criteria

From/To/Matches selection criteria options have been added to are available on the Timesheet Input Group Merge Code field on the Main tab of the following PR/SE processes (a sample field is shown in Figure 1: From/To/Matches TS Input Group Merge Code):

  • PR Convert Future Timesheets
  • PR Convert from Recurring Timesheets
  • PR Transfer Employee Self Service TS
  • PR Transfer FM Worksheets
  • PR Transfer SE Details
  • SE PR Transfer
  • PR Transfer Daily Reports
  • PR Move Employee Timesheets
  • PR Convert to Manual
From/To/Matches TS Input Group Merge Code

Convert to Manual

Convert to Group and Group Description fields have been added to are available on the Convert to Manual screen in Payroll > [Frequency] > Processing, as shown in Figure 2: Convert to Manual [%=Variables.Cheques caps%]. This process retains /timecards/overrides under the same PR Period, but a blank Convert to Group is still possible, in which case the /timecards/overrides will remain under the same groups.
