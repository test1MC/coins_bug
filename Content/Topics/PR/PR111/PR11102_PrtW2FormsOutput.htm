Print W-2 Forms Output

Depending on whether you or un the Print a Single Locality W-2 per State box on the Print W-2 Forms selection screen, the following processing occurs if an employee works in more than one Locality:

  • When the Print a Single Locality W-2 per State box is not ed, two W-2 Forms are printed, as shown in Figure 2: W-2 Form Output (when box is un[%=Variables.tick%]ed).
W-2 Form Output (when box is uned)

  • When the Print a Single Locality W-2 per State box is ed, all of the information is combined into a single W-2 Form, as shown in Figure 3: W-2 Form Output (when box is [%=Variables.tick%]ed).
    • MULTIPLE is printed for the Locality Name.
    • The Box 18 and 19 amounts are combined.
