Print (MCC) (US Only)

The Print process has been updated to use uses the actual Next Number at the moment of printing rather than the Next Number from the Print selection criteria screen. The Next Number field has therefore been removed was removed for Manual Calculator (MCC) from the following areas:

  • The Print selection screen in Payroll > Frequency > Input > Manual Calculator > Print button ( MCC Print Single [%=Variables.Cheques caps%] Button), as shown in MCC Print [%=Variables.Cheques caps%] Screen.
  • The Print selection screen in Payroll > Frequency > Input > Manual Calculator > select Print Single ( MCC Print Single [%=Variables.Cheque caps%] Option), as shown in MCC Print Single [%=Variables.Cheques caps%] Screen.
