Invoice workbench standard process

Use Invoice Workbench instead of Timesheet Analysis or Timesheet Cost Report.

Always use the Global Billing Items tab, not the SHO tab. There will be further updates on the SHO Tab in later versions of this topic.

Phase 1 Manual billing

Phase 1 is about using PB instead of Timesheet extracts to create your manual bill. See 1.2 regarding the PB process where you create the invoice from PB yourself.

  1. Filter to Project Manager SHO, and check that all SHO’s you bill are displayed. If any are missing (or any are not yours) get Bruce or Julie to amend.
  2. Use Column set selection to display different field options. This is flexible, so we/you can easily add fields into the view you prefer.
  3. Use the Report Button (better than the excel Button) if you need to get data extracted – by running the report button you can add more fields into the extract than are displayed on the screen. Over time, when you are familiar with on screen processing, the need to extract data to Excel should diminish, but whilst you are learning the system, it is a useful cross check to the timesheet data you used when doing your billing.
  4. If required, use the two sort options on the display: Contract/SHO/Date or Contract/SHO/Employee
  5. Review and resolve any records with status “No SHO Lines” (see section 2 for more explanation).
  6. Review and resolve any records which have been miscoded, using “Cost Re-Allocation” (see section 2 for more explanation).
  7. Special rules apply for “non-standard” SHOs (see section 2 for more explanation):
  1. SHO Line set up for “SHOFOC” – zero billing rate
  2. SHO Line set up for “CONNST” – non-standard rate
  1. Update the Bill Hours on the PB records for billable items.
  1. Don’t create manual summary lines in PB. If you have many lines that need to be consolidated, put all the Bill Hours into one of the lines, and make the other lines have zero Bill Hours. All lines (including the zero Bill Hours lines) should flagged as “To Manually Bill”.
  1. Manual lines can be created for expenses (see section 2 for more explanation).
  2. Flag selected PB records as either
  1. “To Manually Bill” (invoice prep), or
  2. Manually Billed (already invoiced, need to enter invoice reference), or
  3. Never Bill (will not be invoiced), or
  4. Deferred (need to confirm billing with client), or
  5. Leave as blank (action next month).
  1. Once you have done all of the status changes for all of your billing, you need to Apply your Changes (in Options, top right). 
  2. If required, extract your data into Excel to check your billing. In Phase 2 this will be done using Print Proforma Invoice
  3. Julie will create the Manual Invoice, and apply back to your records that have “To Manually Bill” status Invoice by Invoice (SHO by SHO). This will change the status to Manually Billed and the PB records will change from “Billing Outstanding” to “Billed”
