Options Menu

The Options Drop-Down provides the ability to copy the previous weeks timesheet, to specify what gets copied, and whether to display non-working days.

From the My Contract Timesheets screen, click to create a new timesheet and specify the relevant week ending date and click

In the blank Timesheet click Options

 

Copy Previous Timesheets

Select this option to copy the previous week timesheet into the current week.

Before using this option, to change the information that gets copied select Copy Preferences.

Copy Preferences

Selecting this option will present the option to Copy hours, descriptions, neither or both.

Display Preferences

This option will allow you to turn the display of weekends on or off (and enter time for weekend working).

Hide non-working days - on

Hide non-working days - off
