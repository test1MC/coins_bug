Setting up a new department and allocating employees

The following steps need to be followed:

  • Identify / create GL Element 1 for the department.
  • Create HR / Payroll department and JC Timesheet employee department.
  • Create T* contract for the new department.
  • Create X* contract for the new department (to allow cross charging into the new department). May need to create I* contracts also.
  • Create any new (if required). Create any new Professional Billing Products (if required) to match the new Cost Heads.
  • Always create Contract Specific Cost Code(s) for the new contracts.
  • Create any new Timesheet Activities specifically required for that department. This will likely include a single CHGx* activity for cross department charging. Map the timesheet activities to the appropriate debit and credit Analysis Codes (i.e. ).
  • Create Grade for new department, and allocate Timesheet Activities & cost rates.
  • Create / allocate employees into new department, ensuring that appropriate Grade and Analysis are added in JC Timesheet Employee maintenance, and the corresponding Payroll set up is done (so that Payroll is costed to the correct GL Element 1 department).

Employee role changes

Use the following as a checklist:

  • Is the employee changing department? If yes, then:
  • Amend payroll costing
  • Amend JC Timesheet Employee record – department, grade and analysis (T contract)
  • Contract security – amend code to new T contract, or just enter T if no longer entering timesheets
  • Contract Manager – if applicable, check that person is set up and user id is linked
  • User maintenance – amend user groups if required
  • Is the employee a leaver? If yes:
  • Contract security – blank out entries to remove access and remove of current employee list
  • User security – remove access
  • JC Timesheet employee maintenance – mark as a leaver and enter leaving date.
