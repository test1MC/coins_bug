Creating and posting Timesheet Batches

The first step (after any required timesheet approvals have been completed) is to create the timesheet batch:

If the reference is left blank, the system will generate the reference. A manually entered reference can be used. The Description defaults to blank – typically an entry such as the one above is made.

The “To” date needs to be entered. Care is required on the date entered and the timing around week end dates and month end dates. There should be a separate guidance note on this, but the assumption is that it would normally be the most recent week ending date (as used during Contract Timesheet Entry). This is Sunday in the UK and Australia, Friday in the US, Saturday in the Middle East.

The next step is to Post the timesheet batch (select JC > Timesheets > Timesheet Batches, then go to the last batch).


Select the batch and run the Post Action at the bottom.
