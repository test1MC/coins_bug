Overview

This guide deals with processes that occur after timesheets have been entered by employees (timesheet entry documented in a separate guide), and covers the following:

  1. An overview (this section)
  2. Approving timesheets within the same territory / kco  (US only)
  3. Approving cross territory / kco time (Global)
  4. Creating and posting timesheet batches

Components

The following are key elements of how the system works (menu options are under Contract/Job Status – Timesheets unless otherwise specified):

US same territory approvals.

JC Parameter settings shown below.

Menu option

Purpose

Approval > Employee Department Approval (Weekly)

Allow (US) department managers to Approve Timesheets

Approval > Company Department Approval (Weekly)

Allow (US) Accounts Administrator to Approve Timesheets

 

Cross territory (kco) approvals - Global.

Menu option

Purpose

Approval > Project Manager Workbench

Optional approval by Project Managers. Non US will not use.

Administration > Admin Workbench

Approval by Administrator, the required option for non US territories.

 

Creating and Posting Timesheet Batches
