SHO Header Information

Sales Handover

The first section (Sales Handover) contains a number of items which, although they are not mandatory fields, need to be populated in each instance.

Field Description Notes
SHO Number Allocated SHO number This is not entered by the Sales Person, this will be added later by Julie
Territory Deal Territory UK, Middle East etc.
ALF% ALF Percentage For the UK, the only valid percentage is 22%, selecting other Deal Territories will make different percentages available as appropriate.
Margin   Margin is calculated from the Financial Summary
Deal Type The type of deal being entered into.

A drop down list is available.
No. of Years Deal Term 1 to 5 years. This field is important as it controls how many years the SHO will calculate payments for. For example, if this is set to 1, only the first year payments will be calculated.

It is essential that these fields are completed ,otherwise a lot of the other calculations on the SHO will not work.

 

 

 

Customer/Contact Details

For new customers, all boxes must be completed. For exisiting customers the customers name will suffice.

In all cases, the following must be completed:

  • Company Name
  • Contact
  • Date
  • Sales Person
  • Financed (if applicable)
  • Sector
  • Order Number

The SHO will not be approved if all required information is not completed.
