In the following explanation we will primarily be covering the Surveyor workbench as the fundamentals are exactly the same as Buyers.

To Start, the Matrix Views replicate what is available in the development house type workbench, it allows you to apply changes to the matrix items and add variants to plots where applicable by the Variant Matrix View.

Matrix Views

The Matrix Views Tab allows you to see the resources in numerous ways, it also allows you to interact with the resources on different levels, this can be done by payment stage, variants matrix, Estimating Sections and resources.

on the Payment Stage Matrix, you have the ability to apply actions to rows or elements, this allows you to update individual items or a whole line per payment stage.

This is done with the following actions, clicking on the arrow actions the row or element action.

Row;Element;

Apply Price – applies any new price to a row or element where applicable

Print – gives you the ability to print the

Update VAP Budget – Updates the VAP budget based on the entries on this screen

Set as Ready to Use – sets the item as ready to procure

Add - adds the cost head to the house type when it is set to N/A

Set Stage as Required/Set Stage as not required – Makes item unavailable to use

Material, Subcontract and Plant Tabs

The Material, Subcontract and Plant tabs are exactly the same, they handle ordering but for each category, i.e materials and subcontracts. It is worth noting that the plant tab is not generally used, and that plant ordering is usually directed through the standard procurement process.

As you can see below, the material and subcontract menus exist in the exact same structure, therefore this how to guide will only cover Materials.

there are numerous ways in which you are able to order through both workbenches, you can order through each package and variation matrix, through the items directly or through the package browse, all require nominating a supplier and applying a price, ensuring that the item is set to ready to use.

The Package Matrix view and the Variation Matrix view look exactly the same, you are able to order items via package, or by variation (where applicable)

The element and row both contain the same value actions,

Seq Action, Nominate Supplier, Apply Prices, Set as ‘Ready to use’, Add to Item Basket and finally add to item basket (Bulk)

Seq Action allows you to specify more than one action, in general terms you are able to sequence a number of actions to run sequentially.

Nominating a supplier is essential to ensuring you are able to order a resource, and must be selected when trying to order.

Applying Prices allows you to apply any price changes that may have been made at standard house type, development house type or price workbench, this gives you full control over which items require the price increase and which are applicable.

Set as Ready to Use allows you to order an item, if this is not set then the item cannot be ordered, the system will alert you in several ways if you are not able to order the item.

Add to Item Basket, add to Item Basket (Bulk) allows you to add numerous items to the basket, ‘add to item basket’ allows you to add singular items to the basket, ‘add to item basket bulk’ allows you to order bulk items where specified (on the resource record) – NOTE: this also affects how it displays on the order form, Bulk items tend to group together as one order line.

There are two main ways of viewing the matrix, by value and by status, this is controlled by the filter located at the bottom of the page.

Item Basket & Ordering

The Item basket can be navigated to by going to ‘My Item Basket’ in the drop down under each of the tabs (Material or Subcontract)

Once in ‘My Item Basket’ you are able to select the specific items that you are wanting to order, these are displayed as ‘new orders’ under the reason column.

 

if you select the items that you are wanting and ensure that the actions bar has ‘Create Orders from Selected Items’ you can then click the arrow to proceed and create the order.

 

On clicking the arrow, you are presented with a standard procurement screen where you can select the order type, date, description and a number of subsequent fields that are specific to the order;

 

by clicking next, the system will create the order and subsequently remove the items from your item basket as seen below;

once removed from the item basket you can view and commit the order from the ‘Orders Raised’ option under the menu structure of the category that you are working with.

 

from here you are then able to select the order, commit, print, draft print, create cancellation and more using the action bar towards the bottom of the page. It’s worth noting that if you are confident the order is correct and you are wanting to commit, printing the order would auto commit it too.

Ordering from Package Browse

You are able to also order from the package browse screen; this is handy when the package contains all of the resources within the package, by navigating to the package browse screen you have the ability to select a number of alternative ways to add items.

 

It is worth noting that the package browse also shows you the status of ordering and if items are priced, the value and number of items associated to the package.

 

The actions available on the menu are listed below; these allow you to order items in a number of ways including variations, reorder, appending items, setting items as ready to use or not ready. All items work in the exact same way as the standard ordering method

 

Items

Items exist in the same menu, this item allows you to order directly from the list of items on the development and on the plot. You are able to add last-minute changes to a plot in this menu using the add type selector set to ‘resource line’. The selectors for adding to basket exist on the line also.

NOTE: you are not able to order items if a supplier and price has not been defined, the system will error advising you to nominate a supplier

Supplementary Items

Within the same menu drop down in each of the categories you have a number of other items that may be useful, it includes the following;

Orders requiring Variation shows you any orders that may have had quantity changes or price changes that need a variation raised.

Create Orders from Item basket allows you to create orders from all items in your order basket, this means you do not need to select individual lines. The system will automatically raise orders based on the supplier and order details (bulk vs singular)

My New Orders show you orders that have recently been raised and not committed, this allows you to easily navigate and commit the orders that need to go to the supplier.

Clear My Orders allow you to clear any orders that exist in the ‘my new orders’ screen, allowing you to easily maintain the list.

Development Items

Just like the House type items workbenches, the development items within the surveyors’ workbench allow you to add, delete and modify resources, it automatically adds in the information to the payment stages tab and where applicable the intelligent resources.

To Add a resource, ensure that the ‘add type’ on the workbench is set to ‘Resource Line’

 

if you now click on ADD, you’ll be taken to a screen where you’ll be able to add a new resource.

There are 4 main parts to the adding of a resource to the house type item tab. These are ‘Item Selection’, ‘Pricing Details’, ‘Other Details’ and ‘Ordering Details’, an example of what could be input is below;

Item Selection

 

it is worth noting that within the Pricing Details you MUST ensure that the ‘status’ is set to ‘Not Applicable’ against the categories that aren’t being used and also ensure that the ‘item ready to use’ box is not ticked.

 

Other Details is more memo data rather than specifically affecting anything to do with the resource and useage of it.

 

It is important to ensure that the ordering details are filled out and that a resource is associated to either a material or subcontract package.

 

once you are happy with your resource you can click ‘SAVE’, once it is saved, the screen will blank out all of the details you have just entered, this is so that you can add a new resource without navigating away. More Tab

Under the More tab you have numerous options, some of which we have already explored,

Apply House Type Changes to Plots – applies any changes that exist at house type level to the individual plots, in essence this copies each of the resources for each plot.

Plot Rebuild from House Types – this allows you to rebuild the from a house type if there are many changes that have happened at house type level that need a complete rebuild

Apply Prices to Plot – Applying prices to plots exists for the sole reason of prices that may have been changed on the prices workbench.

Update VAP Budget – updates the VAP budget based on items within the , this is useful for updating site start and current forecast.

Resource Substitution – allows you to substitute resources that have been labelled as substitute, this allows you to easily swap out bricks that may be of different or variance of some sort.

Package Setup Browse – allows you to see packages that have been set up, though this is set up in procurement you are able to see this against the also.

Stage Summary Report – a report to show you the stage of the in summary format, this report can be used as part of the pack if this level of detail is needed.

Plot Print Report – an easy way to print out the entire based on plot.

Package Status Report – this report shows you the packages that are currently available against the development and provides details of the status in terms of what has / needs to be ordered.

Procurement Plan – procurement plans can be set up to allow you to order, other than bulk, you are able to then schedule the ordering for when it is needed for each phase of the development.
