House Type Items Tab

Although you are able to add Individual Payment Stages, Estimating Sections and Packages to the Standard House type Workbench, the best way to be able to handle the house type items is through the house type item tab.

Adding Resources

To Add a resource, ensure that the ‘add type’ on the workbench is set to ‘Resource Line’

Click , you’ll be taken to a screen where you’ll be able to add a new resource.

There are 4 main parts to the adding of a resource to the house type item tab. These are ‘Item Selection’, ‘Pricing Details’, ‘Other Details’ and ‘Ordering Details’.

Item Selection

Field Description
Resource Code The unique code which identifies this resource item.
Variant The house type variant this resource is used on, if relevant.
Description The description of the resource.
House Type Dependent Whether this resource is house-type dependent.
Substitute Whether this resource should be substituted before raising an order.
Estimating Section The estimating section to which this resource belongs.
Enter the code or number.
Payment Stage The sequence number of the payment stage.
Quantity The quantity of this resource.
Unlabelled field The unit of measure for the quantity.
DIM Sheet Whether to use a dimension sheet to build up the quantity for this item.
Size The size that the resource comes in.
Unlabelled field The unit of measure for the size.
Wastage The wastage percentage for materials.
Unlabelled field The gross quantity (that is, the net quantity with the materials wastage percentage applied to it)

Pricing Details

It is worth noting that within the Pricing Details you MUST ensure that the ‘status’ is set to ‘Not Applicable’ against the categories that aren’t being used and also ensure that the ‘item ready to use’ box is not ticked.

Other Details

Other Details is more memo data rather than specifically affecting anything to do with the resource and usage of it.

Ordering Details

It is important to ensure that the ordering details is filled out and that a resource is associated to either a material or subcontract package, this allows you to view the resources against the individual plots within the surveyor workbenches, this must be associated

Once you are happy with your resource you can click ‘SAVE’, once it is saved, the screen will blank out all of the details you have just entered, this is so that you can add a new resource without navigating away.
