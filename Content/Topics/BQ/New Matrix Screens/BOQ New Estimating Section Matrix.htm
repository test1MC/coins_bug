New Estimating Section Matrix

The Estimating Section Matrix on the Development House Type  Workbench and the Surveyors Plot Workbench now uses the new matrix.

Actions

The following actions are now available on the right-click element and column menu:

  • Nominate Supplier
  • Apply Prices
  • Set as “Ready to Use”
  • Set as “Not Ready to Use”
  • Set Section as ‘Required’
  • Set Section as ‘Not Required’
