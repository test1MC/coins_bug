New Variants Matrix

The Variants Matrix on the Development House Type  Workbench and the Surveyors Plot Workbench now uses the new matrix.

Actions

The following actions are now available on the right-click element and column menus:

  • Add Variant to House Type
  • Delete Variant from House Type

Filters

You can filter by:

  • Variant
  • Description
  • House Type

Sort

Columns can be sorted by House Type (A-Z or Z-A).

Views

A selector allows you to view by:

  • Variant Status
  • Variant Value
