Navigation

The screens scroll left and right, and up and down. You can scroll using the scrollbars or the cursor keys on your keyboard, and scroll up and down using the mouse wheel.

Frozen columns to the left of the screen identify the rows (for example, the payment stage, variant, package or resource). You can resize the frozen columns to give more space for the remaining columns.

The current row in the frozen columns section, and the current cell, are both highlighted.

Highlighted Row and Cell

Two Search fields at the top of the screen (one for the rows, and one for the columns) allow you to jump directly to any cell in the matrix. Select the row and/or the column you want to jump to, and click the Apply button .

Search Fields

If you make more space available (for example, by resizing the window or changing the zoom level), additional rows and columns are displayed.
