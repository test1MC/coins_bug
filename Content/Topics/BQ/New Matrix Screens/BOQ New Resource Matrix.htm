New Resource Matrix

The Resource Matrix on the Development House Type  Workbench and the Surveyors Plot Workbench now uses the new matrix.

Actions

The following actions are now available on the right-click element and column menu:

  • Nominate Supplier
  • Apply Prices
  • Set as “Ready to Use”
  • Set as “Not Ready to Use”

Filters

You can filter by:

  • Resource
  • Description
  • Unit of measure
  • Estimating Section
  • Payment Stage
  • Materials Package
  • Subcontract Package
  • Plant Package
  • Plot Code

And choose whether to include legally completed plots.

Sort

Columns can be sorted by:

  • Plot (A-Z o Z-A)
  • Build Sequence (A-Z o Z-A)
  • House Type (A-Z o Z-A)

Views

A selector allows you to view by:

  • Resource Value/Items
  • Resource Quantity
  • Resource Status
