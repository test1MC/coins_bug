Standard Variant Setup

Standard variants cater for potential variations to the standard specification for a standard or house type. Examples include: a change of elevational treatment from all brick to half rendered; a change of roof from gable to hipped; a change of internal specification from open market to social housing, a change of all ironmongery to luxury quality, etc.

Variants allow you to maintain a single for a house type containing the possible variations, rather than have to setup multiple house types.

You can specify which variants should not be used together on the same plot.  For example, a plot might have standard or deluxe sanitary ware, but not both.

You specify which variants are relevant to each house type, then for each variant, you build up a list of items that are added to or omitted from the house type if that variant is used.

To set up variants:

  1. Navigate to > Setup > Maintenance > BQ Standard Varients

  1. Click

  1. Set up the code and description to identify the variant.
  2. Select any other variants which conflict with the variant you are setting up; that is, variants which should not be included on the same plot.  For example, all brick treatment would conflict with half-rendered treatment.

  3. Click

    NOTE

    If you add a conflicting variant y to variant x, variant x will also be added as a conflict on variant y; you do not need to add it manually.
