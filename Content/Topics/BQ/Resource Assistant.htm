Resource Assistant

When adding a new resource in , various other records must exist first:

  • Payment Stage
  • Package
  • Commodity
  • Estimating Section

Resource Assistant is a new function that allows you to add a new resource, and add the necessary records if they do not exist, all in one place.

(If you know that all the necessary records already exist, you can still add a new resource using the button on the Resources workbench.)

Screen and Processing Changes

A new function, Resource Assistant, has been added to the menus.

To add a new resource

  1. Go to > Setup > Maintenance > Resource Assistant.
  2. Select the Resource Type to add (material or subcontract).
  3. Enter or select the , Payment Stage (if used), Estimating Section (if used), Commodity or Trade, and Package. Resource Assistant Screen

    If any of these records do not exist:

    1. Click ADD next to the field.

      The relevant Add screen opens in a frame.

    2. Add the details for the new record and click .

      The frame closes and the field on Resource Assistant is populated with the new record.

  4. When you have filled in all the details, click .

    The standard Add Resource page is displayed, with the values you entered on the Resource Assistant screen populated.

    Resource Assistant – Add Resource

  5. Complete any additional fields as required and click .

Configuration

The following new BQ parameters need to be populated:

Parameter

Description

DEFMCAT

Default cost category for material resources added using Resource Assistant.

DEFSCAT

Default cost category for subcontract resources added using Resource Assistant.
