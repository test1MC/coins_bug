Packages

Material and Subcontract packages are currently set up within Procurement.

Within Procurement > Setup > Technical Information System > Packages, you are able to set up package items for subcontract and materials.

Packages do not tend to be extremely detailed but it allows you to group resources in and then effectively order the resources by package.

like most workbenches, packages contain ADD, Copy and deletion, packages can be added on the fly but please ensure that when adding packages, they are effectively labelled so that users are able to easily determine which package a resource needs to be created against.

When you open a package you are able to detail more in-depth information, if you require a default trade code you can input this on the main screen;

you are able to enter information pertaining to the construction stages, which allows you to specify when an item would potentially be needed in the construction phase;
