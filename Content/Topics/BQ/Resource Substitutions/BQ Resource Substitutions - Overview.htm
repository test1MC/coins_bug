BQ Resource Substitutions - Overview

Two new matrix screens are available in the House Type Workbench and the Surveyors Plot Workbench:

  • A Resource Substitution Matrix allows you to update resource substitutions for plots. It shows which resources require substitution, allows you to update the substitutions for multiple plots, and allows you to review the completeness and accuracy of the substitutions made.
  • A Resource Price view of the Resource Matrix provides an efficient way to update subcontract prices (which are specific for each house type).
