BQ Resource Matrix - Resource Price View

A new Resource Price view of the Resource Matrix in the Development House Type Workbench and the Surveyors Plot Workbench gives an efficient way to update prices for resources across all house types (or plots). This is particularly appropriate for subcontract prices which are specific to each house type.

Each cell displays one of the following:

  • “No Items” if the resource does not occur in the development house type (or the plot ).
  • The price if the resource does occur in the development house type (or the plot ). You can update these cells with a price. A tooltip shows the price units, quantity, unit, and value.

Actions

The following actions are available on the right-click cell and row menus.

  • Nominate Supplier
  • Apply Prices
  • Set as "Ready to Use"
  • Set as "Not Ready to Use"

These are the same actions as on the other views of the Resource matrix. The standard Copy and Paste actions are also available on the cell menu.
