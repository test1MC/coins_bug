BQ Resource Substitutions Matrix

The Resource Substitution Matrix shows resources requiring substitution as rows and plots as columns, with the substituted resource in each cell. The Resource Substitution Matrix is available in the Development House Type Workbench and the Surveyors Plot Workbench.

Each cell displays one of the following:

  • “No Items” if the resource requiring substitution does not occur in the development house type for the house type of the plot.
  • Blank with a red background if the resource requiring substitution does occur in the development house type for the house type of the plot and no resource code has been entered. These cells are updateable within the matrix: you can enter a valid resource code (which must be one not requiring substitution). A tooltip shows “Please enter the resource code”.
  • The resource code with a green background if a resource code has been entered. These cells are updateable within the matrix: you can update with a valid resource code. A tooltip shows the description of the resource code.

Actions

The following action is available on the right-click cell menu.

  • Lookup Resource Code. This provides a lookup of resources (only those not requiring substitution) and returns the selected resource to the cell(s).

In addition, the following standard actions are available:

  • Copy
  • Paste
  • Undo Changes

The following action is available on the right-click row menu.

  • Lookup Resource Code. This provides a lookup of resources (only those not requiring substitution) and returns the selected resource to all cells in the row.

In addition, the following standard action is available:

  • Undo Changes

Updating the resource substitution matrix also updates the existing resource substitution browse.
