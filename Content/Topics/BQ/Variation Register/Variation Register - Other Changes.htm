Variation Approval Workflow

An OA workflow can be launched to approve a variation. Specify the workflow code on the status code setup (see 1 , Variation Status Codes). When the variation moves to a status linked to a workflow, the OA workflow will be launched.

Web Service

A web service is available to create entries in the variation register and to amend entries. This allows variations to be created by third-party or bespoke sales extras software.

Item Basket and Order Raising

Any item that is currently part of a variation that is in progress will not be able to added to the item basket or have an order raised until the variation is made live.

Apply House Type Changes to Plots

Any house type item that is part of a variation that is in progress will not be copied down to the associated plots. Only once the variation is set to live will be the changes be applied to the associated plots.
