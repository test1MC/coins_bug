Posting the Payment Batch

Once the cheque has been printed, there is one further step to complete – posting the transaction entry batch:

Navigate to the Input menu in the module and enter the “Transaction Entry” screen

Navigate to > Cash Transactions > Cash Transaction Entry

Select the batch to be posted using the box to the left of the batch

Select the option “Post Batches with Report” from the dropdown menu and select the Apply Action button.  A report will be generated on the Report Status Workbench and the Batch status will update from Un-posted to Posted.
