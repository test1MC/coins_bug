Transaction Date

The date for the transactions in the batch. uses the transaction date you enter in the batch header as the default transaction date for all the transactions, but you can choose a different transaction date for individual transactions.

Effective Date

The cost-effective date of the batch. This is he date within a contract cost period to which a transaction belongs. Cost periods can be different from financial periods.
