Posting the Reconciliation Batch

Once you have entered all missing items for all the selected statement items, you can post the reconciliation batch using Transactions Entry.

The batch header control totals show the value of the statement items and the missing items entered at that point. If the value of the missing items does not equate to the value of the statement items you will be prevented from posting the batch.
