Positive Pay Parameters

The following parameters affect the new Positive Pay functionality.

Parameter

Description

CB/USEPOSPY

Will positive pay enhancements be used? If this is not set to Yes, the old-style Positive Pay Report will be used.

CB/POSEMPRP

If both the CB parameters USEPOSPY and POSEMPRP are set to Yes, show the payroll Employee name in the Positive Pay extract file (and on the Positive Pay Purge Report and Positive Pay Reset Utility report) otherwise just show the word EMPLOYEE.
