Positive Pay Document

The Positive Pay configurable document specifies the format to use for the positive pay extract file. The format to use for each bank is set up using the Format field on the bank account.

The form has been enhanced. If the CB parameter USEPOSPY = Y, a new Extras tab allows you to specify:

Field

Description

Delimited?

Whether the fields in the output file are delimited or fixed-length.

Delimiter

The delimiter to use. You can select one of comma, space, or tab.
