Import Scanned Images

New fields have been added to the Import Scanned Images screen. These fields only appear if the OCR engine is installed.

Fields

Field
Description

Index Direct to Company

If ed, this runs a report which imports the images as single-page invoices and runs OCR against the images. A invoice batch is created and any information that can be read from the images is used to populate the invoice records.

P/L Invoice Document Group

If indexing direct to the company, this is the document group that the scanned invoices will be assigned to.

The report informs you whether the images have been imported successfully.

The invoices are then available in a new invoice batch in Enter Invoices, with the images attached to the invoice records.
