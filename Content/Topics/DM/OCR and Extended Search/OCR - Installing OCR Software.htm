Document Management OCR - Installing OCR Software

Installing Tesseract

The Tesseract OCR software needs to be installed. As the installation of Tesseract is widely documented online, details are not covered in this document. Installation instructions for various platforms can be found at:

https://github.com/tesseract-ocr/tesseract/wiki

Installation instructions are provided in the wiki, plus the option to compile Tesseract if the UNIX/Linux distribution does not support one of the described installation methods.

Compile instructions can be found at https://github.com/tesseract-ocr/tesseract/wiki/Compiling.

A number of gui add-ons are provided for testing and training purposes. https://github.com/tesseract-ocr/tesseract/wiki/AddOns

Additional language packs are also available; see https://github.com/tesseract-ocr/tessdata. These are often referred to as training data for Tesseract.

Installing Other Components

You also need to install the following:

  • Leptonica image-processing software: http://www.leptonica.com/download.html
  • libtiff: http://download.osgeo.org/libtiff/tiff-4.0.6.tar.gz
  • libjpeg: http://www.ijg.org/files/jpegsrc.v9b.tar.gz
  • zlib : http://zlib.net/zlib-1.2.8.tar.gz
  • PDF Toolkit: https://www.pdflabs.com/tools/pdftk-the-pdf-toolkit/
