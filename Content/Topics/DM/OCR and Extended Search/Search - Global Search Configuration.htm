Document Management Extended Search - Global Search Configuration

There is a new entry in Global Search Configuration for the dm_document table : %DM_DOCUMENT. This needs to have the Use box ed.

If you need to make other changes to the configuration (for example, to address security - see 1, Document Management Extended Search - Security), copy the %DM_DOCUMENT record.

There is a new standard function and page %WSYDDOC to show the search results.
