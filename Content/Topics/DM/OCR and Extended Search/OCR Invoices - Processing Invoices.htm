Document Management OCR Invoices - Processing Invoices

Scan or import the invoices using the standard functionality.

NOTE

Currently the OCR will only process images scanned as TIFF files.

Run Index from Batch to process the invoices.

A new field has been added to the Index from Batch screen.

Field
Description

OCR Layout

The OCR layout to use to index these invoices. The layout identifies the areas of the screen that will be processed using OCR, and the keywords that the processed text will be associated with.

Select the layout you want to use when indexing the scanned images.

To confirm that you have chosen the correct layout, all configured keywords will be highlighted onscreen.
