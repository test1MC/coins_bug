Document Management Extended Search - Process OCR Queue

Process OCR Queue is a new function on the Document Management- Administration menu. It enables the OCR process to be run, either on demand or scheduled using the standard report scheduler. No input is required on this function.

Process OCR Queue processes any images (that are specified in OCRCAT) scanned since the last time OCR was run. It will only process those files added after the functionality has been installed and configured.

In addition, if the DM parameter GENSRCPDF is set to Y (and Tesseract version 3.03 or later is installed), the process will create a searchable PDF file for images specified in OCRCAT. These files are stored as revisions of the original scanned images in the database under the Document Management module. All subsequent image retrieval will return the searchable PDF file.

If the DM parameter OCROSOLR is set to Y, the extracted text will be passed to the Solr database.
