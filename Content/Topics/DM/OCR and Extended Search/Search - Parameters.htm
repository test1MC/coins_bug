Document Management Extended Search - Parameters

The following new parameters control the extended search functionality.

System parameters:

Parameter

Description

SRCHDOCS

A list of document extensions for documents to which full-text extraction and indexing is applied; for example: xls*, doc*, pdf, ppt*. This allows the extended search capability to find text-based attachments (such as Word, Excel and PDF files). If this is blank, no extraction of text will be done.
