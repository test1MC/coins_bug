Document Management Extended Search - Security

Existing and security is applied automatically to attached documents. When a user tries to open a document, either from Global Search or from any document browse in , the document is withheld if the user is not authored to see either the company or the (if the document belongs to a ).

To prevent the new search functionality allowing users to view documents you do not want them to see, you may need to consider additional security especially if HR or Payroll are in use. For example:

  • Restrict which documents are processed for extraction and indexing. This is managed by the %DM_DOCUMENT record in Global Search Configuration – the Condition Value field holds a “can-do” list of document groups; only documents matching this list will be indexed. The default value is “*” (for all). This is standard data so it will be reloaded during upgrades; if you want to change this you should copy %DM_DOCUMENT to a non-% record.
  • Restrict which search results the user can see. This uses Document Group security in DM only – it does not apply or GL security.
