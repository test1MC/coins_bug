Document Management OCR - Installing OCR Software

The Tesseract OCR software needs to be installed. This also requires Leptonica.

If ‘epel-release’ Is Available

If the ‘epel-release’ repository is available as an installation source (as it is with most modern instances of CentOS and Red Hat) you can use ‘yum’ to install the required libraries:

sudo yum install leptonica tesseract

This installs both Tesseract and Leptonica, and automatically adds any required libraries.

If ‘epel-release’ Is Not Available

To install the ‘ epel-release’ repository for RHEL7/CentOS7 if it is not available, use the following commands (these may need to be run as sudo):

wget http://dl.fedoraproject.org/pub/epel/7/x86_64/e/epel-release-7-10.noarch.rpm

rpm -ivh epel-release-7-10.noarch.rpm

NOTE

The version of ‘epel-release’ will change over time, so the link may need updating; for example, epel-release-7-10.noarch.rpm will become epel-release-7-11.noarch.rpm.

To process TIFF images, it is recommended to have the libgcj library available. This can be added from the pdftk library (this can be obtained from ):

sudo rpm -ivh pdftk-2.02-1.el7.x86_64.rpm

Then install the image libraries

sudo yum install libgcj libtiff zlib
