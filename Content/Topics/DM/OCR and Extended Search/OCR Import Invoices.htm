OCR – Processing Scanned Invoices

can process scanned invoices to automatically match the invoice to the and create invoice record with some of the data fields populated by information from the scanned image.

To process a scanned invoice

  1. Prepare the files to be imported by placing them into a network scanner location.
  2. Go to Import Scanned Images.Import Scanned Images Screen

  3. the OCR to P/L Invoice box.
  4. Select the Document Group to store the invoice against.
  6. Click Next.

Import Scanned Images runs as a report process. The report tells you which record or records it has created. The OCR process creates an invoice batch with the description “OCR DM Batch”.

  1. Go to Enter Invoices.
  2. Open the OCR DM Batch.
  3. On the Invoices tab, the invoices show as ‘Not Validated’ with a red background if they have not been updated. Invoice Batch – Invoices Tab

    The supplier reference, supplier account, invoice date and gross amount are shown if the OCR process has been able to determine them. Any values that break validation are shown with a red background.

  4. Follow the Internal Reference link for an invoice.
  5. The invoice record opens and displays the scanned invoice. Any fields that the OCR process has been able to identify are populated and validated, so any errors are highlighted.
  6. Fill in any remaining data and enter the invoice as normal.
