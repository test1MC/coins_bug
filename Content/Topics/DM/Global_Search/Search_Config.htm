Configuring DM Global Search Initial Configuration

Global Search Configuration

To use the Document Management Global Search, the search data for %DM_DOCUMENT must be generated using Global Search Configuration, and the Solr broker must be running.

Global Search Configuration

For more details about Global Search Configuration, see Global Search Configuration in the online documentation.

Parameters

The following new DM parameters control the display of the search results in Card View.

Parameter

Description
CVIMGHW The height and width of the image to display in DM Global Search results - Card View.
CVNUMCOL The number of columns to display in DM Global Search results - Card View
