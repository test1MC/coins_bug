Screen and Processing Changes

Document Browse

When viewing the list of documents for a record, the list now shows related documents. For example, if one of the related keys is "order number", when looking at a purchase ledger invoice record, if a document exists, selecting the paperclip button will display a list of not only the Purchase Ledger documents, but also all documents in the same company with the same order number, or any documents that are not company-specific with the same order number.

Multiple Invoices with the Same PO Number

Order Documents Including Related Invoices

The display of the paperclip button remains is dependent upon there being a document for the current record being viewed. If there is no invoice document when looking at a Purchase Ledger invoice record, but a purchase order document is available, the paperclip button will remain greyed out.

Category Description

To assist in identifying which image to view, the document category description is now included in the document browse.

Associated Documents Menu Option

When viewing a document, an ‘Associated Documents’ menu option lists all related documents.
