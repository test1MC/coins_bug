Viewing Related Documents - Initial Setup

Parameters

The following new parameters have been added to Document Management.

The following Document Management parameters relate to viewing related documents.

Parameter

Value(s)

USERELATEDKEYS

Set this to Yes to turn on the functionality to display related documents.

RELATEDKEYS

A comma-separated list of the long keyword descriptions (dkt_desc) included in lookups to find related documents.

Different keyword types can have the same long description. For example, poh_ordno on the Purchase order and ohd_num on the Invoice can both have the same long description of ‘Order number’.
