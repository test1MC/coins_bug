Viewing Related Documents Overview

When viewing a document in Document Manager you can now navigate to and view other documents with a relationship to the current document.  These documents may be in the same or different categories.  For example, when viewing a Invoice in an approval workflow, it is useful to also see the purchase order and/or GRN Packing Lists.   Another example is looking at Procurement Requisition Images when viewing a Purchase Order Image.

Note in the first example the field name(s) that relate the tables are different (poh_ordno  on the PO and ohd_num on the Invoice) so searching a single field would not return both documents.
