Document Management Separator Pages - Import Scanned Images

Several new fields have been added to the Import Scanned Images screen.

Fields

Field
Description

Blank Page Action

The action to be carried out if a blank page is detected.

  • No Action - Blank pages are imported as they are.
  • Delete - Any images that are identified as blank pages will be deleted and not imported into the batch.
  • New Document - The next image (after an image identified as a blank page) becomes the first page of a new document.

Max Dark %

This field is displayed if the Blank Page Action is Delete or New Document.

It allows you to specify the percentage of dark that indicates that the image is treated as a blank page.

The detection of a blank page requires there to be a threshold at which an image can be regarded as blank. Scanning a totally blank page may result in some data in the image; it is also possible that a blank page may have footer text when using stationary. For this reason, a minimum level of non-white noise must be specified using the Max Dark percentage setting.

NOTE

This process will not work if the network scanner automatically deletes blank pages.

Duplex

This is only displayed if the Blank Page Action is New Document. When scanning in duplex, the back of some scanned documents may be blank, therefore if Duplex is ed, it will require two consecutive blank images to activate the creation of a new document.

NOTE

Some scanners have the option of creating a mulit-page image file when scanning to duplex. Using a scanner with that feature enabled is not compatible with the option to create a new document.

Split PDF's to single pages

This is only available if DM/SPLITPDF is set to Y.

If this is ed, any PDF documents in the import will be split into individual PDF documents, and then imported as normal.

After the import, the report will show which PDF files have been identified as multi-page, and the pages generated from them.
