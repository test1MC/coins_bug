PDF Toolkit

For the function to separate PDF documents into individual pages, the PDF toolkit library needs to be installed on the server. This is a free to use software, and needs to be installed before the function to work.

For instructions on how to install the package, see:

https://www.pdflabs.com/docs/install-pdftk-on-redhat-or-centos/

If PDFtk is not installed and Split PDF is set, the process will be aborted and an error shown in the report with the above link.
