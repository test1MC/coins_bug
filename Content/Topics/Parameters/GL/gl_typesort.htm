GL/TYPESORT - Sorting of account types

This parameter is a list of comma separated values. The order of the values in the list is the order in which the GL Account Type will be sorted in reports (when the sort by Account Type is selected in reports). If this parameter is blank then the default sort order will be A,C,L,E,I (denoting that that the sort order will be in the order of Asset, Capital, Liability, Expense and then Income types)
