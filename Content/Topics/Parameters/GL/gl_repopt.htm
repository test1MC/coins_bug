GL/REPOPT - Post Printer Form Font Copies

For batches that are posted to General Ledger automatically, COINS generates a General Ledger report following the source ledger report. To override the settings for the source ledger report (for example, to print the report on a different printer), enter the settings (printer, form, font, copies) here. If you do not want a General Ledger report to be printed, enter 0 for the number of copies.
