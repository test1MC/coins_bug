GL/CPJRNLS - Cash Profile Transaction Types

A list of the transaction types that produce a cash movement in the Cash Profiles module.

To include certain General Ledger transaction types only: list the transaction types separated by commas.

To include/exclude transaction types from GL and other modules, list the transaction types prefixed with the module code and "/". In this case wild cards are allowed.

For example:

!GL/INT,GL/*,ST/I*

This would include all General Ledger transaction types except INT and would also include all transaction types beginning with I.
