GL/DEFANAL - Default analysis for txns

() Whether COINS copies the analysis from the last line in the browse when adding a new line in a GL journal batch (Y or N).

Y = The analysis will be filled in with the analysis used on the last line in the batch.

NOTE

If you insert a line, the analysis will default from the last line in the batch, not the line above the one you are adding.

N = The analysis field on new lines will be blank.
