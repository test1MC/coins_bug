GL/COBDM - Batch realtime datamarts

The details to use for creating datamarts when batches are posted.

This is a comma-separated list with three values:

kco,dataset,datamart

kco - The target for the GL/JC datamart. If cross- this will be the same in each .

dataset - The dataset to be built and written to the datamart.

datamart - The datamart to be written to.

This requires additional configuration; please consult COINS.
