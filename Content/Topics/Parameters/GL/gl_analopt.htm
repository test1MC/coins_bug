GL/ANALOPT - Analysis Options (C/A/G/V/W/X/P/L)

Codes for the analysis types, separated by commas. These are the codes users must select to change from one analysis to another when entering transactions.

The codes must be in the order:

, asset, General Ledger, , workshop, part-exchange, account, lettings recharge.

Only include codes for the analysis options you want users to use.

For example, if you are not using workshop analysis, and want to use the code N (for Nominal Ledger) for General Ledger, enter

C,A,N,V,,X
