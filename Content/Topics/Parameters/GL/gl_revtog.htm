GL/REVTOG - Toggle Revenue/Cost in GL Journal

In , if you want the analysis when entering a GL journal to default to Revenue, enter R, otherwise the analysis will default to Cost.

In , if you want to use a character other than R as the code that switches between cost and revenue input in General Ledger Batch Entry, enter it here. Do not use a character which is already used, such as C,A,G,V,W,X (, Asset, GL Account, , Workshop,HS Part Exchange).
