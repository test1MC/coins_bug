GL/GLWFJNL - Workflow Template to use for posting GL Journals

The workflow template for posting GL Journals

If GLWFJNL is non-blank, the GL Journal Entry and Sector Balancing journal in General Ledger and Status contains two tabs instead of one:

  • GL Source Batches shows all batches that have been sourced in the GL. The Post action is replaced with a Release to Workflow action, which will trigger the workflow based on the template identified by GLWFJNL. The workflow will need to be written so that the posting of a batch is approved, and the final stage of the workflow if approved will be to post the batch.
  • External Batches shows all batches that have been sourced from another module.
