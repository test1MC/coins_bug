GL/AUTOGEN - Auto Generate G/L Accounts

To have COINS generate accounts automatically if transactions try to post to them, set AUTOGEN to Y. COINS will generate the account provided the elements of the account are valid. If AUTOGEN = N, you must set up each account manually before transactions can post to it.

If AUTOGEN = Y, the lookup on account fields is by element rather than full account code. See General Ledger Account Lookup.
