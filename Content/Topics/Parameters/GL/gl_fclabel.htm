GL/FCLABEL - Column Labels for Yearly forecast browse in GL Forecasting

This allows you to choose format for column labels for forecast amount and actual amount in the Yearly Forecast browse and Rolling Forecast browse.

The following options can be selected:

  • Month and Year
  • Period Number
  • Period End Date

If this parameter is blank then "Month and Year" format will be applied (e.g. "Jan 18").
