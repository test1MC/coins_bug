GL/MASKTRUN - Truncate masked account with no sub level

This determines what happens when COINS masks two accounts, where the resulting account has sub levels that are not required as specified by the GL element setup. If this is set to Y, these elements will be removed from the resulting account. For example, if account _ _._ _ _ _.XYZ is masked with account 00.1234._ _ _ , and the element 1234 does not require a sub element, and MASKTRUN=Y, the resulting account code 00.1234.XYZ would be truncated to 00.1234
