GL/FCACCRGL - GL Clearing Account for Accrual Posting

This Parameter should hold a valid GL Account. This account will be used as a clearing account for accrual posting.

If GL/FCACCR is set and GL/FCACCRGL and GL/ACCRTYPE are non-blank then the following actions are accessible in GL Forecast Workbench browse.

Post Accruals - Post Forecast Accrual Amount to the GL Forecast Account. The negative Accrual amount will be posted to GL Account specified by GL/FCACCRGL.

Un-post Accruals - Cancel the Accrual Batch or create a reversal batch.

Copy Forecast to Budget - Copy Forecast values to GL Budget.
