GL/ACCTSEC - Account Security Field

Which account, element or analysis set is used to control user security.

Enter one of the following:

gla_acct for account code
gla_el1 for element 1
gla_el2 for element 2
gla_el3 for element 3
gla_el4 for element 4
gla_anal[1] for analysis set 1
gla_anal[2] for analysis set 2
gla_anal[3] for analysis set 3
gla_anal[4] for analysis set 4

or leave blank for no account security.

Use GL Security Maintenance to control which users are allowed to access which accounts.
