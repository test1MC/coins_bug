SW/RESCHDEL - Reschedule delay for Get Next Schedule Report (minutes)

The number of minutes after which Opti-Time Get Next Booking will automatically be re-run.

When using FM Integration with an external dynamic scheduling tool (such as DRS), .FM gets the currently-scheduled tasks from DRS via the Get Next Booking Report. If there are no tasks when the report runs, then if this parameter is set the report will automatically be re-run in the number of minutes set in the parameter.
