SW/STOPCANC - Cancel all tasks if stop job

Whether to cancel outstanding tasks when the Stop stage is run (Y or N).

Y = If there are outstanding tasks on a when the Stop stage is run, COINS cancels the tasks automatically. You would use this setting if you use the Stop stage to mean that there is no more work on the .

N = COINS does not cancel outstanding tasks when Stop is run. You might use this setting if a can be restarted after Stop has been run.
