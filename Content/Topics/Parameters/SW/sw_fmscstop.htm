SW/FMSCSTOP - If cost/FM tasks amounts not same then give Error or Warning

This parameter determines whether the user is allowed to post the SC certificate if its not been fully allocated to an FM task. The options are E-Error or W-Warning or leave blank to not validate.

If set to W then the user is prompted as to whether to continue or to go back and fix the certificate. If set to E then the SC certificate can't be posted unless the certificate amount is fully allocated.
