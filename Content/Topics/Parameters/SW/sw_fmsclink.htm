SW/FMSCLINK - Pay for Tasks using SC certificates

Whether FM tasks assigned to subcontractors can be linked to subcontract certificates (Y or N).

There must a valid order for the /subcontractor, and when entering a certificate for this order a screen displays all the FM tasks linked to it and shows the 'maximum value' associated with each of them.

The user selects which FM tasks the certificate pays for, and cost distribution lines are added to the certificate which reflect the 'maximum value' of the tasks selected, that is, [CONTRACT].[SECTION][ACTIVITY][].[CATEGORY]. The comes from FMSCJSC, the activity from FMSCACT, and the category from FMSCCAT.
