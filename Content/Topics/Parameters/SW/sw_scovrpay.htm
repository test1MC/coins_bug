SW/SCOVRPAY - Give Error or Warning if Certificate Value exceeds FM Task Maximum value

It is possible to link a SC certificate with a FM task. The FM task has a maximum value associated with it. This parameter determines whether you can assign more than the maximum value to the task.

The options to set this parameter are:

  • If set to blank or E, you are prevented from assigning a different value.
  • If set to W, you may set a different value and a warning will be given.
