SW/WINSIMS - Use WinSims system

Whether to interface with the WinSIMS system (Y or N).

If this is set to Y, a box on the indicates whether the for the need to be integrated with WinSIMS.

The following interfaces are available: Interface 1 - Pass all new FM ; Interface 2 - get file with FM tasks, update tasks and then send file with response; Interface 3 - send new FM Invoices which relate to WinSIMS Jobs.
