SW/USEPLBUD - Use Planned Budgets

Whether planned budget screens appear on the Workbench (Y or N).

Planned Budgets are entered at the Location/Major Category level. Planned Budgets require that an overhead section(s) be created for the . This can be done using a section of 0000 or 9999, for example.

The budget process utiles the Memo Data functionality in Status. The Status parameters BUDINIT, BUDVARI and BUDCURR must be set up. In addition, the must reference a Date, along with Period (entered in weeks).
