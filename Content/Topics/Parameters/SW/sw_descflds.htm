SW/DESCFLDS - Text and fields to add to end of Job description (and Section and WBS)

Text and fields that are added to the end of the description, to help ensure that the correct has been selected.

Fields entered here will automatically be displayed as a suffix to the brief description of the wherever it is used on screens or reports. For example, adding to DESCFLDS will show address line 1 at the end of the description. It is also possible to add text.

The JC section description and WBS description will also have the same suffix.
