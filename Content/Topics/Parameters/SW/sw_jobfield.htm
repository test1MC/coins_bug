SW/JOBFIELD - Additional fields in Job Load

Additional details fields to load from a spreadsheet.

You can load FM details from a spreadsheet using Load FM . Certain fields are required, but you can load additional fields if they are available.

The additional fields must appear as extra columns in the spreadsheet, after the required fields.

Enter a comma-separated list of the fields from sw_job into which you want to load the data from the additional columns. List the fields in the same order as the additional columns.

For example:

swj_taken_date, swj_taken_hh, swj_taken_mm
