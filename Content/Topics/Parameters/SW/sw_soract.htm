SW/SORACT - Enable a JC Activity field against an SOR

Whether to include an Activity field when adding an SOR to the system (Y or N).

If SORACT is set to Y and the is set to use activities, a new Activity field is available for input when adding a SOR item. This field allows the user to select from the standard activity list.

If the Activity field is populated with a budget code, this allows you to report on the time spent per budget code.

When the record is saved, this creates a WBS code for the FM number and the activity on the SOR.
