SW/LINKJOBS - Allow linked jobs

Whether to enable the feature that allows you to create linked "child ".

This allows you to specify a child on a minor category. When this minor category is chosen when adding a , then a duplicate will be created on this child . When the 'Confirm All tasks complete' stage is run on the child it will also be run on the originating parent and will complete any tasks existing on the parent . When the invoices on the child are posted to the as revenue, they will also be posted to the parent as costs. The 'operationally complete' stage will also be run on the parent .
