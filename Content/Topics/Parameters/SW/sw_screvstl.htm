SW/SCREVSTL - Default Review Style for SC Orders

The default Review Style for subcontract orders. When not blank, this parameter is used to set the Review Style when creating a subcontract order from an FM task.

Depending on settings in Procurement, this information may be required when raising subcontract orders. The value should be one of the existing Performance Review Styles applied to subcontracts, as set up using Performance Review Styles.
