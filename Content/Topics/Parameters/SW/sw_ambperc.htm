SW/AMBPERC - %age busy for Scheduling screen amber

The percentage at which the Single Day Scheduling screen shows time-slots as busy.

When the amount of resource scheduled in a given time-slot reaches this percentage of the resource available for that time-slot, COINS marks the time-slot as busy. Users will still be able to book appointments in the time-slot, but this is an indication that they should book appointments in other time-slots if possible.
