SW/DRSOIST - DRS Order is Coins FM Task

When using an external dynamic scheduling tool (such as DRS) this parameter determines whether s and tasks added in COINS FM should be automatically forwarded to the scheduling tool, or whether there should only be a manual action to do this.

If DRSOIST is set to Y then there is a one to one relationship between an FM task and a DRS order, and sub-tasks can be identified as follow-on jobs
