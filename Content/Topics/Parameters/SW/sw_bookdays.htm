SW/BOOKDAYS - Enable Day Rate Booking Functionality

Setting this parameter to Y enables the 'Book Days' functionality. This will create an extra tab on the Workbench to manage daily bookings. There will also be a new 'WIP ' function.

Daily Bookings allow the user a single workbench to manage a set of tasks that last over a number of days, and the subsequent invoicing of the work done (via an associated SOR Item)

The Workbench also tracks the progress of the booking.
