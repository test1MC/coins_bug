SW/SCAGTYPE - Default Subcontract Agreement Type

The default Subcontract Agreement Type for subcontract orders. When not blank, this parameter is used to set the Agreement Type when creating a subcontract order from an FM task.

Depending on settings in Procurement, this information may be required when raising subcontract orders. The value should be one of the existing Subcontract Agreement Types, as set up using Subcontract Agreement Types
