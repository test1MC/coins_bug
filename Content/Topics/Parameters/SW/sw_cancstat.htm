SW/CANCSTAT - Status if cancel invoice

The FM status to use after an invoice has been cancelled. If this is blank, the status is set to blank.

The cancel option allows you to cancel the invoice if it hasn't been printed. Depending on the cost type of the , it resets the to the state it was before the invoice was created.

  • For Transaction, complex SOR, SOR, Cost+ cost types: SOR items and/or cost transactions are reset so they are no longer marked as invoiced.
  • For 'simple measured' cost types: valuations are reset so they can be billed again.
