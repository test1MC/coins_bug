PL/BACSOVER - Override BACS Media on Supplier Account

Determines whether you can override the media on individual accounts.

Y = You can override the default format specified by BACSMED. In Details, press Return on the Payment Method field, then enter the media type in the BACS/EFT Media field. When overridden, the resultant output file has the media type APPENDED to the filename.

N = The BACS/EFT Media field does not appear.

This is to allow for more than one electronic clearing system, as in Australia.
