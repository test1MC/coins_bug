PL/ADMUSERS - List of Administrator Users on Invoice W/B

A list of administration users for the eCommerce Invoices.

Administration users are allowed to load and reject messages, and use sample messages. Non-administration users do not have access to the Load and Reject actions, or to the All Messages/ Live Messages/ Sample Messages filter.

The list of administration users can be specified in the following ways:

  • blank, * or !* (all users are administration users).
  • A comma-separated list of user IDs.
  • A list of non-administration users (user IDs prefixed with !), followed by *; for example: !johsmi, !tombro, *.

    NOTE

    Other features of standard can-do lists are not supported.
