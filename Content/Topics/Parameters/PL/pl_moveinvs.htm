PL/MOVEINVS - Move Unposted Invoices to Another Batch

Select Y to turn on the Move PL Invoices to Another Batch function, which allows invoices in an unposted PL invoice entry batch to be moved to another unposted invoice entry batch.

The purpose of this is to allow invoices that are not yet ready to be posted (because they need reviewing or re-negotiating with ) to be moved to another batch without holding up the posting of the remaining invoices in the batch.
