PL/POREQREG - OA only - PO Required on Registered Inv

( only) Whether an order is required when registering invoices (Y or N).

Y = You must enter an order number (unless the invoice type is included in the list in the PO parameter NONPOTTS).

N = You can enter registered invoices without entering an order number. When costing these invoices, COINS presents you with the PO Number field. If you enter a PO number, you can match the invoice to the order. If you leave it blank, you can cost the invoice directly (without matching to an order).
