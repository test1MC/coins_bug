PL/CIMINVDL - Allow Deletion of CIM-Generated Invoices

(COINSplus only) Whether you can use the F10 key in Invoice Entry to delete transactions generated by CIM.

Pressing F10 will delete the invoice - which is useful for documents that were scanned/indexed in error. The invoice reference number is retained (as part of the batch record) and is shown as a deleted internal reference number on the invoice posting report.
