PL/REFBASE - Int Ref Base (PPYYMMYYYY)

The format of the optional base part of internal reference numbers.

The base is a combination of digits taken from the current General Ledger period and financial date. REFBASE is a list of the positions of the required digits from the following sequence PPYYMMYYYY (where PP = ; YY = period year; MM = month; YYYY = year).

For example, a REFBASE of 9,10,5,6 and a REFFORM of 9999 produce an invoice reference number made up of the last two digits in the year, the two digits of the month, and four digits, such as 95050001.

See also REFFORM.
