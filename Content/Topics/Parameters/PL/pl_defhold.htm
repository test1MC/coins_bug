PL/DEFHOLD - Default Invoice Hold

Whether to place costed invoices on commercial hold by default (also see APPRCOST and APPRENT), that is, for COINS to automatically enter a 'Y' in the Invoice on Hold field:

Y = Invoices are put on hold automatically using the default hold code specified in DEFHCODE.

N = Invoices are not put on hold automatically.

NOTE

This parameter refers to the default Commercial Hold fields, which will always appear (with or without the Financial Hold fields).
