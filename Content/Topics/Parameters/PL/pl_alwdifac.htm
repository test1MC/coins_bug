PL/ALWDIFAC - Allow Different Supplier on Purch. Order

Allows you to specify whether a invoice can be matched/posted against a purchase order that has a different account number attached to it.

Y = When an order is added on a PL invoice, the from the order is automatically populated; this can be amended to another if appropriate.

N = The cannot be changed from the one set on the order.

You can override this for specific users by setting up appropriate permissions for the OA security function %WPL101WBPOHS1. Users who have root access are always allowed to change the account number on an invoice.
