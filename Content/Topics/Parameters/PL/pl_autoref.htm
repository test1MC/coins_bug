PL/AUTOREF - Auto Generate Reference

Whether to generate automatic internal reference numbers.

Y = COINS generates internal reference numbers for transactions automatically when the user presses Return.

N = The user must enter a new internal reference number for each transaction entered.

NOTE

This is different from the System parameter AUTOREF, which governs batch reference numbers.

You can use a different setting when loading invoices using plinv.p; add ,AUTOREF=N or ,AUTOREF=Y to the function parameter.
