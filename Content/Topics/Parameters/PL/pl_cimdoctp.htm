PL/CIMDOCTP - CIM Document Type Names (2 Entries)

The CIM document types for invoices and credit notes.

This is a comma-separated list of 2 entries. This allows you to specify the CIM document type names for invoices and credit notes separately.

The first entry is your description of a invoice (for example: PL Invoice). The second entry should contain your description of a credit note (for example: PL Credit Note). If this parameter is left blank (or either of the two entries is blank) then the default of XML PL Invoice will be used.
