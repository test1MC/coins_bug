PL/EHLDPINV - Allow PINV eInvoices to be Held

Whether to allow PINV eInvoices (transactions which require a purchase order) to be put on hold (Y or N).

If EHLDPINV is Y, hold processing is modified as follows:

  • If HLDREGIN is false and the distribution type is not "R" the user may change the transaction's hold status.
  • If HLDREGIN is true and the distribution type is "R" the transaction is put on hold. For a transaction on hold the user may then change the hold code.
