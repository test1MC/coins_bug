PL/AUTSUPAL - Automatically create Supplier aliases for PCards

Whether to automatically create aliases for PCards when user enters Account for a new merchant on a PCard transaction (see PCard Aliases).

2 = Two aliases (general and -specific) will be created automatically

1 = One -specific alias will be created automatically

0 = No aliases will be created automatically

If this parameter is not set, then the default value of '2' used.
