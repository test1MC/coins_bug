PL/SECTLED - Sector Based Ledger

Whether you are running on a sector basis.

To run on a sector basis, enter Y. Otherwise, enter N.

Running a sector-based ledger means you will be selecting invoices by sector. So in (PL), you will create one batch per sector and in (SL), you will create one receipt batch per sector. Payments/Receipts are not mixed across sectors in the same batch.

In order to use this parameter, the GL parameter SECTOR must be set to Y.
