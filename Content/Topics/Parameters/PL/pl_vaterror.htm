PL/VATERROR - Treat Diff VAT Code on Supplier as Error

Controls what COINS does if a user enters a code on a transaction that is different from the Code field in Details.

Y = COINS issues an error message and does not allow the user to enter the code on the transaction.

N = COINS issues a warning message; the user can enter the code on the transaction.

If the Code field on the is blank, COINS does not issue a message.
