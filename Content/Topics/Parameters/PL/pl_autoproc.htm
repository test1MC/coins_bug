PL/AUTOPROC - Process valid eInvoices automatically

Whether to allow valid eInvoices to be processed automatically, skipping the Electronic Invoice Workbench.

PL Batch would be created automatically only for auto-routed single-transaction PLINV eBatches from COINShub.

If eInvoice transaction have any validation errors, it will be loaded into Electronic Invoice Workbench automatically, but PL Batch creation will not be started.

Otherwise, PL Batch creation process will be started automatically in background (the same as usual OA PL batch creation). Default Batch Description and current will be used.
