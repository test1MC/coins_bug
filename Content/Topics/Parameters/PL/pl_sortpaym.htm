PL/SORTPAYM - Sort Order for Payments

Determines the sort order in which a / remittance print run, a payment list, or a payment posting report prints out.

The output can be sorted by selecting one of the following values:

ACCNAME = name.

ACCNUM = Account number.

CHQAMT = amount (ascending order).

CHQAMTD = amount (descending order).

PAYREF = Payment reference.

DEFAULT = Select this value if you do not want to influence the sort order of the print.
