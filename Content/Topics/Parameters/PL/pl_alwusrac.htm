PL/ALWUSRAC - Allow Supplier Account to be entered on the PCard WB - Cardholder Mode

Whether to allow card holders to enter the account number In the PCards Workbench - Card Holder Mode.

Y = In PCards Workbench - Card Holder Mode, users can enter a account code.

N = In PCards Workbench - Card Holder Mode, users cannot enter a account code; the account can only be specified by users running PCards Workbench - Administrator Mode.
