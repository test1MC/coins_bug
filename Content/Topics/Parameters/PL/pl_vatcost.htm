PL/VATCOST - Only Post VAT When the Invoice is Costed

Whether to post when the invoice is costed (Y) or registered (N).

Y = Only post when the invoice is costed. If some is non-recoverable, and can only be identified when the invoice is costed, set VATCOST to Y so that the analysis of the invoice can be edited as part of the costing.

N = Post when the invoice is first posted onto the ledger. The analysis of the invoice cannot be edited after the registration batch has been posted. If you are using invoice registration for some or all invoices, and you do not normally have non-recoverable , set VATCOST to N so that can be recovered on all invoices recorded on the ledger, based on the tax point date.

The normal setting in the UK is Y for house builders; N for other clients.
