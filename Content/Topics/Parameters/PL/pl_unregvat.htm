PL/UNREGVAT - Unregistered VAT Code

The code for who are not registered for .

UNREGVAT is either blank, or contains a zero-rated 'Unregistered' code.

If UNREGVAT is set to non-blank, then the following checks are enabled when entering transactions:

  • If the is not registered then the code will be set to the Unregistered code.
  • If the is registered then the code used must not be the Unregistered code.
