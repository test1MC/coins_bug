SY/LDAPCERT - LDAP SSL Certificate

The full path of the certificate repository (when implementing LDAP via Secure Sockets Layer (LDAPS).

When implementing LDAP via Secure Sockets Layer (LDAPS), this parameter must contain the full path of the Certificate Authorities (CA) certificate repository.

If this parameter is set to a valid directory on your COINS server, all LDAP communication with your LDAP Directory servers will be performed using a secured LDAPS connection. As such all LDAP Directory servers published on your corporate network and used by COINS must implement trusted, secured connections. COINS will demand a trusted secure sockets layer certificate from the LDAP Directory server.
