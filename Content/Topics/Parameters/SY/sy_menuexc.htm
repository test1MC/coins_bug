SY/MENUEXC - Main Menu Security Exclude

( only) A comma-separated list of letters that represent browse sub-functions (such as Bulk Update or the Delete button) that excluded when using menu item security.

The sub-functions represented by this list will not automatically be available even if the user is given Update access to a function. You can override this for individual users by explicitly granting access to such functions.

The letters are:

B = Bulk Add/Update box.

C = Copy button.

D = Delete button.

U = Update button.

X = Export button.

If you want to allow certain groups or individuals access to the excluded sub-functions, use Group Function Access or User Function Access. Filtering on %W*D would show the delete sub-functions, for example (though be careful as it would also include other functions that end in D).
