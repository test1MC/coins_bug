SY/WEBSTYLE - Web style base directory

The location of the directory that holds cascading style sheet files. This allows you to use different style sheets for different .

Normally, this should be set to $WEBBASE, to use files in the /style/ directory in as specified by the SY/WEBBASE parameter.

However, if you want to use different style sheets for different , set this as a company-specific parameter to the name of the directory that holds the /style/ directory. For example, if you have a directory structure:

coins
live
liveco1
images
style
liveco2
images
style

you would set this to liveco1 in company 1 and liveco2 in company 2.
