SY/LOCKTERM - Allow users to lock terminal

(COINSplus only) If you are using operating system logins (specified in Menu System Configuration) without allowing users to re-login (RELOGIN = N), how do you want COINS to behave if the user presses L?

Y = Pressing L locks the COINS session; the user must enter their COINS password (or press F4 to log out).

N = Pressing L logs the user out - they return to the operating system.
