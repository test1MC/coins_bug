SY/ISSFUNC - Issue Report Function

The function used for the input form that opens when the user selects the issue report option. If this is blank, the default function is %WSY2580FXXX

The title or tooltip of the button is taken from this function. If you need to change the default title "Report Issue", create a + function and change the context description. Alternatively create a custom input function/form and set the desired description, and enter it here, and will be used instead of the standard.
