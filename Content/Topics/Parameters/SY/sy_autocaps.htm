SY/AUTOCAPS - Automatically capitalise codes

This parameter controls whether COINS automatically capitales data that is entered in fields. It works differently in and .

If blank, no key fields are changed.

: If non-blank, key fields for main tables (for example: , codes) are capitaled.

: Set this to a list of tables that should be capitaled. For example, jc_job, ap_vendor, ar_cust, sc_mast would capitale key fields on , , Customer, and Subcontractor records.

NOTE

If you need more control over which fields are capitaled, you can maintain the capitalation of individual fields on the Fields tab of Table Documentation.
