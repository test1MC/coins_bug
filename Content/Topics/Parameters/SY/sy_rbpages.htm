SY/RBPAGES - Report builder page layouts

A comma-separated list of the report page layouts that are available on a report builder report. For example: %A4RLAND, %A4RPORT, %A3RLAND, %A3RPORT.

This allows just a few select layouts to be shown rather than all that are defined for other reasons in COINS. Blank will show all page layouts.

Page layouts for reports are set up using Report Page Layout.
