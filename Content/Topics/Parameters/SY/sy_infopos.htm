SY/INFOPOS - On-screen position of notification messages

On-screen position of COINS pop-up notification messages.

If this is blank, then value of 'bottom,5' is used.

Parameter format: '<position>,<offset>', where <position> could be 'top' and 'bottom', <offset> is a number of pixels from top/bottom edge of the screen.

Valid parameter values: 'top,50', 'bottom,30', 'top'.

A full page refresh (Ctrl + F5) could be needed before changes will take effect.
