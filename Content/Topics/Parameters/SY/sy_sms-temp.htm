SY/SMS-TEMP - SMS Template; use {phone} {msg} tokens

Template of message to SMS provider.

Simple text messages can be sent via an SMS provider by sending to the provider detail of the texts you want sent, in a format they recogne. Each different provider has their own format and this parameter allows you to specify the template of the message you send to them. Tokens ({phone} and {msg}) should be included in the template and the appropriate substitutions will be made when sending text message details to the provider.
