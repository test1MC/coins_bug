SY/LOGMSGS - List of message numbers to log

A list of COINS messages to include in the log file.

This is a can-do list of COINS messages (in the format xx999; for example SY617, SY618, SY619). If a user receives a message on screen or in a report that is in this list, that message will be written to the log file (WebSpeed log for pages/screens, report log for reports) with a tag alongside it of COINSErrorMessage and showing the program stack. This can be used to monitor for certain messages being raised.

There is a similar check that checks for System Error messages (which are specifically unexpected issues that the programmer has written a throwSystemError() message for). These are shown in the log file (regardless of this parameter) with COINSSystemError as a tag.

Both these tags can be searched for in the log file (using OE Management or other tools) to alert the system administrator to issues.
