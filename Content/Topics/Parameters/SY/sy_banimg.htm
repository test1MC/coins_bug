SY/BANIMG - Banner Image

( - up to v11.04 only) The URL of an image (such as a logo) to display on the title banner of each COINS screen.

This could be an absolute URL (for example: http:// 123.45.67.89/ resources/logo.gif) or a relative one (for example: /coins /live /images/ logo.gif).

You can set BANIMG as a -specific parameter, to display different images for different .

Since this image will be referenced by lots of screens, if the image location takes a long time to resolve, the performance of the software can be affected. It is best for the image to be local and not, for example, on a website.

If this is blank, the COINS logo (COINS_banner_logo.png) will be used.
