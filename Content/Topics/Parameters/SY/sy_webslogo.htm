SY/WEBSLOGO - Web small size logo URL

(This parameter is no longer used.) The URL of the small logo displayed next to the selector in the menu treeview.

This can be either an absolute URL (for example: http:// www.coins-global.com/ images/ logo.gif), or relative to the COINS installation (for example: /images/ logo.gif).

Since this image will be referenced by lots of screens, if the image location takes a long time to resolve, the performance of the software can be affected. It is best for the image to be local and not, for example, on a website. The image is displayed if SIDELOGO is not set to Y.

If WEBSLOGO is blank, the image identified by WEBLOGO will be used instead (but note that it will be resized to fit the space). If both WEBSLOGO and WEBLOGO are blank, the image coinslogo.png will be used. If you want to show an image other than the COINS logo next to the selector, we recommend you identify it using WEBSLOGO rather than using an image identified by WEBLOGO.
