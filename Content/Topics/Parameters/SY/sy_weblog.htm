SY/WEBLOG - Full path to Webspeed/AppServer log file

The full paths and file-names of the log files you want to view using Log File Viewer.

This is a list of name/value pairs. For example:

AppServer= /home/ dev/ as/ dev/ WebDevRTB.server.log, WebSpeed= /home/ dev/ ws/ dev/ WSDev.server.log, Apache= /usr/ local/ apache/ logs/ access_log

There are two special names - WebSpeed and AppServer, if these are used then the log files will be parsed to show agent, date and time. Any other names will just show lines of text so it can be used to view any text or log file.
