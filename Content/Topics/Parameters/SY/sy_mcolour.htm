SY/MCOLOUR - Message Colours (0 to 7)

(COINSplus only) The to use for different types of messages.

This is a list of pairs for message levels, separated by commas. There are seven message levels as follows: 1 = failure, 2 = error, 3 = processing, 4 = warning, 5 = information, 6 = hint, 7 = question. The list contains eight pairs, the first being the default to use for messages that do not have a level. The pairs are defined in the protermcap file.

For example, if M is defined as follows:

NORMAL, W/R, W/R, MESSAGES, MESSAGES, R/B, R/B

then failure and warning messages will be shown as white on red, processing and warning messages will be shown using the defined as MESSAGES, information and hint messages will be shown as red on blue, and all other messages will be shown using the defined as NORMAL.

If a is not specified, then NORMAL is used.
