SY/ECHONOLN - No New Line option for ECHO

The option for "no new line" in the UNIX echo command.

c = Use ECHO text \c

n = Use ECHO ? text

This is used in faxing; some UNIX shells (for example ksh on RedHat) do not recogne \c in the echo command for no new line. ECHONOLN set to n means use the -n option of echo instead of \c.
