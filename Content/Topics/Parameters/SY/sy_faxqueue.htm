SY/FAXQUEUE - Report queue to service Fax requests

The report queue to which reports or documents that are to be faxed should be sent. This allows the fax software to be installed on one server where these queues are serviced.

If a queue is specified here, when a user s the Fax option on the output selection tab, the report queue combo is hidden and the value of this parameter is used instead.

Report queues must first be configured using Report Queues.

If the lists in the DIRQUEUE and FAXQUEUE parameters are mutually exclusive then the user will not be allowed to print and fax at the same time.
