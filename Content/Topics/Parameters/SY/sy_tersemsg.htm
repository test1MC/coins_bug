SY/TERSEMSG - Terse messaging in webspeed log

Whether to set the program info/debug messages in the WebSpeed log file to terse (Y or N).

Many of the messages included in the WebSpeed log are for information only and are not normally required, though they are useful when investigating problems. This parameter allows you to exclude the information messages (significantly reducing the size of the WebSpeed log file) and only to go back to full logging when required.
