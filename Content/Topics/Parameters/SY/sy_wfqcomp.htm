SY/WFQCOMP - Workflow Quick Complete

Whether to show a quick complete option at the head of a workflow page in addition to the existing link. This will allow the user to quickly complete a task and return to the activity workbench (Y or N)?

If you follow the link from the Activity Workbench for a workflow task, you will see a button link to update the action at the head of the page. It is possible to select the link to show the task details and update and compete the task.

By setting this parameter to "Y", the completion of simple workflow stages (not requiring notes) can be achieved with a minimum number of mouse clicks.
