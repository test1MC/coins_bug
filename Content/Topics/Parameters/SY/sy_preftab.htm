SY/PREFTAB - Recall default tab

The default for whether, when a user displays a screen they have visited before, COINS displays the tab that the user visited previously, or the first tab on the screen.

The options are:

Full - Always shows the most recent tab that the user visited.

Day - Shows the most recent tab that the user visited today (otherwise shows the first tab).

Session - Shows the most recent tab that this user visited during the current session (otherwise shows the first tab).

None - Always shows the first tab on the screen.

You can override this setting for an individual user by setting the Recall Default Tab field on their user record.
