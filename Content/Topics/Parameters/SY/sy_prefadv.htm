SY/PREFADV - Recall advanced filters

The default for how COINS remembers the advanced filter settings on browse screens.

When a user opens a browse screen that they have used previously, COINS can pre-fill the advanced filter with the last settings they entered. This controls how long COINS remembers the settings.

Full = Always remember filter settings.

Day = Remember filter settings from earlier today (first time on each page would show no filters).

Session = Remember filter settings from the current session only (first time on each page since login would show no filters).

None = Never remember filter settings (advanced filters always show blank first time on a page).

You can change the setting for an individual user in Users.
