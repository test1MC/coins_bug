SY/WEBFOPAS - AppserverConnect string for remote FOP

The AppServer connection details that will allow FOP to be run remotely.

If this is blank, FOP is run locally, otherwise COINS will attempt to connect to the AppServer and run FOP remotely.

Common connection parameters:
-AppService application-service
-H [ host_name | IP-address ]
-S [ service-name | port-number ]
-DirectConnect

For example, connecting to an AppServer named "fopServer" on host douglas, set this parameter to:

-H douglas -AppService fopServer
