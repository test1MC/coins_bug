SY/WEBBASE - Web Base Directory

The location of an alternative base directory for the static pages and files on the web server. Normally you would leave this blank.

If this is blank, COINS will look for the static files (such as images, scripts and style sheets) in directories under the current COINS environment. However if you have the static files in under a different directory, you can specify the directory here.

For example, with WEBROOT set to http://195.40.14.50 and WEBBASE set to static, the file add.gif would be found at http://195.40.14.50/ coins/ static/ images/add.gif
