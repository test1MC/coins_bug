SY/IMAGEDDE - DDE Macro used for image retrieval

The image software or DDE macro for image retrieval (if you are using image retrieval software).

BATCHSCAN = Use BatchScan for displaying document images.

CIM = Use CIM for displaying document images.

IMACS = Use IMACS for displaying document images.

or set this to one of the standard DDE macros, for example: !IMSCAN. DDE macros are set up using Maintain DDE Macros.
