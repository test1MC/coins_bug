SY/EXTUSER - User for Extranet security

The user record that defines additional function security for Extranet users.

An Extranet user is one who has the Extranet User field on their user record ed.

By default, no functions are available; by adding function access for this user, functions are then available to Extranet users. The access permissions for this user ID are the superset of all functions that you will allow external users to run. Normal user function security is applied on top of this. If this parameter is left blank, no additional function security is applied.
