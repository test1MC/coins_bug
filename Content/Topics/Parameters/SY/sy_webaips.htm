SY/WEBAIPS - Auto Login IP CAN-DO list

A can-do list of IP addresses from which users can log in automatically (that is, without having to enter a password).

If a user attempts to log in from an IP address that is not in this list, COINS will not attempt to log them in automatically. For example, you can use this to allow internal users to log in automatically but force someone logging in from an extranet to enter a password.
