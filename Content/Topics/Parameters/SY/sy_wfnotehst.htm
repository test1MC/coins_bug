SY/WFNOTEHST - Display Workflow Note History

Whether to show all progress notes that have been entered for a particular workflow on the workflow navigation (Y or N).

Whenever a note is entered in the Progress field on the Workflow Navigation screen and the stage is completed, COINS records the text and the current date, time and user ID of the person completing the task.

If WFNOTEHST = Y, COINS displays these details in the Note History field.

If WFNOTEHST = N, the Note History field is not shown (but the details are still recorded).
