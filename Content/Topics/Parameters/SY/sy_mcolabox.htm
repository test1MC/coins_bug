SY/MCOLABOX - Alert Boxes Colour instead of MESSAGES

(COINSplus only) This controls the for alert boxes (if you are not using Window alert boxes - see SOFTABOX).

By default, the of an alert box depends on the severity of the message and the setting of M. However, if the alert box is displayed using the MESSAGES (or certain other combinations), the highlight on the buttons is confusing.

MCOLABOX allows you to override these for alert boxes. It is a list of pairs, separated by commas. The first pair defines the to use for alert boxes. The remaining pairs are those combinations which you do not want to use for alert boxes.

For example, if MCOLABOX is defined as:

Y/K,R/B

(and M is defined as in the example below), then COINS will display processing, warning, information and hint alert boxes as yellow on black (overriding the MESSAGES and R/B settings in M).
