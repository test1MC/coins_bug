SY/FTPLOCAL - Generalised FTP local directory

( only) The default directory on user's local computers when importing files using FTP. Do not include a final \.

Currently this is only used when importing orders.

  • If the user selects the Local PC radio button, the field defaults to the values of the parameters FTPLOCAL plus FTPFILE.
  • If the user selects the Host radio button, the field will be set to the values of the parameters FTPDIR plus FTPFILE.

In both cases the user can overwrite the field.
