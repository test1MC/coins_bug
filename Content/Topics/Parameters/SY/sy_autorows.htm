SY/AUTOROWS - Auto refresh if rows adjustment

Whether to automatically refresh browse pages to adjust the number of rows (Y or N).

When you load a page, COINS works out how much spare space (or not) there is, and changes the number of rows in the browse the next time the page is refreshed. If AUTOROWS is set to Y, that refresh is applied automatically if the number of rows calculated was different to what had just been built.
