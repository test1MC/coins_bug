SY/MENUSEC - Use Main Menu security

( only) Whether to use Menu Item Security (Y or N).

Menu Item Security allows you to control function access security at the level of items on menus, and to specify update or read-only access. See Menu Item Security.

If MENUSEC = Y:

  • If you grant access to a menu item, users have access to all its children, by default.
  • You can specify read-only or update access to functions. The extra option (Yes-Update) is available in Main Menu Security and elsewhere.
  • Users automatically have access to all functions with access type SYS, in all modules.
  • Users automatically have access to menus if they have access to a function on the menu (this also applies to tabs and container functions).
