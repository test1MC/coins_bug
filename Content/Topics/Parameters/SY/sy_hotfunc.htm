SY/HOTFUNC - Hot Functions From New Menu

(COINSplus only) A list of entries to set up hot keys. The entries are of the form key,function.

For example, D,%Gl510,J,%Gl101-A sets up the hot keys D to run General Ledger Dig Deeper, and J to run General Ledger Journal Entry.

Remember that if you set up hot keys that are already set up as menu keys, those menu keys will not work.
