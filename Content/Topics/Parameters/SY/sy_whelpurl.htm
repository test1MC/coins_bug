SY/WHELPURL - Location of OA Online Documentation

( only) The location of the (old-style) COINS Online Documentation (based on ).

By default (that is, if this is blank), COINS will display the Online Documentation from the COINS website. If you wish to host the documentation elsewhere (for example, on a local server), you need to enter the path of the top-level documentation folder (the \\coins\\ folder).
