SY/USRSTYLE - User Style sheet URL

( - up to v11.04 only) The path and file name of a cascading style sheet file (for example, user.css) to use to override the existing display styles.

You can create a user.css file instead of, or in addition to, changing the SY parameters in the Style group.

If this is blank, COINS uses the standard style sheet coinsstyle.css in the directory specified by WEBSTYLE.
