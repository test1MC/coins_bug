SY/IRMARKSH - IRMark script to run (if standard one overridden)

The location of the override version of the irmark shell script, if required.

If needed for both and this can be set as a comma-separated list of 2 shell script locations, the first for and the second for (on the report server).

Java 1.3 versions of CIS submissions have an override version of the irmark shell script.

This is also used when report servers are used; the value will depend on the configuration.
