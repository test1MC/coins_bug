SY/FTPFILE - Generalised FTP import file

The default filename to use when importing files into COINS using FTP.

Currently this is only used when importing orders.

In , this is used as the default filename when importing from either the UNIX host or the local PC. The default directory is specified in SY/FTPDIR or SY/FTPLOCAL.

In , this is used when importing from the UNIX host. When importing from the local PC, a Browse button is available.
