SY/WEBIMGS - Web images base directory

The location of the directory that holds graphic files (for buttons and other images). This allows you to copy and modify standard images, or use different images for different .

Set this to $WEBBASE to use files in the /images/ directory as specified by the SY/WEBBASE parameter.

If you want to use a different images directory (for example, if you want SY/WEBBASE to refer to the standard scripts and styles, but want to modify the images), set SY/WEBIMGS to the parent directory of the alternative images directory.

To use different images for different , set this as a -specific parameter to the name of the directory that holds the images directory. For example, if you have a directory structure:

coins
live
liveco1
images
style
liveco2
images
style

you would set this to liveco1 in 1 and liveco2 in 2.
