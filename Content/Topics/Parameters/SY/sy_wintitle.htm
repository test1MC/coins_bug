SY/WINTITLE - Window Title 1=Co 2=Prog 3=User 4=Branch

The format of the title in the Windows title bar. You can configure the Windows title bar to display your own text, and any of the following: COINS name, user name, procedure name, branch name.

Enter your own text, and/or the digits 1 to 4, separated by commas, to display the following:
1 = name
2 = Procedure Name
3 = User Name
4 = Branch (OASIS users only)

NOTE

In , COINS only takes the text at the beginning from the parameter. The other information is always shown, regardless of the setting of WINTITLE.

For example, if you set WINTITLE to COINS v11.04,1,3,2, COINSplus will display a title bar such as this:

COINS v11.04 - COINS Construction - DONBAR - Invoice entry
