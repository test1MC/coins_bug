SY/ESBSEND - ESB send message script

The script to be run when sending ESB messages out of COINS.

This script will be run when the message is ready to be sent. The following environment variables will be set:

BASE - The base directory of the environment.

FILE - The name of the file containing the message to be sent.

REF - The internal reference number of the message.

QUEUE - The endpoint/queue name of the message.

This allows the script to, for example, copy the file to a directory on disk, to execute an FTP transfer, or to send using a httpPostGet.sh script.
