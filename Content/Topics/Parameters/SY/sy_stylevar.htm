SY/STYLEVAR - Style Variant

Which COINS style sheet to use.

blue or blank = Use the blue style sheet that was available before 2012. This is the default.

or v1026b = Use the style sheet (first introduced with release v10.26 of COINS).

NOTE

To use the style sheet you must also have file caching enabled. A document explaining how to configure file caching is available on the Client Area of the COINS website.
