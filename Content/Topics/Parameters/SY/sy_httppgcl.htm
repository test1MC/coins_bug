SY/HTTPPGCL - httpPostGet class file to run (not inc .class)

The classname (without .class on the end) of the non-standard class to override httpPostGet.class (used by the UK EGov Submission XML), if required.

Some AIX versions of UNIX require a non-standard version of httpPostGet.class.

This is also used when report servers are used; the value will depend on the configuration.
