SY/LOGHISTR - Days of SYSAdmin log history to retain

The number of days that log history for the SYSAdmin user will be kept.

It will default to 365 days but can be overridden to any required value.

The SY/LOGHIST parameter is used for the number of days of log history that will be retained for all other logins. This enables, for example, the log history for users to be set to day 30 days and the log history for SYSAdmin to be 365 days.
