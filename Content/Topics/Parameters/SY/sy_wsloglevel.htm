SY/WSLOGLEVEL - COINS OA Webspeed Logging Level

4 Comma Separated Parameters:
  • OA Webspeed Logging Level: (0 - 3)
  • Perform Grouping: (Y or N)
  • Show Stack: (Y or N)
  • Stack Depth: (1 - 4)

Examples: 3,Y,Y,2 or 2,Y,N or or 3,Y or 1

OA Webspeed Logging Levels:
This is used to separate different web log messages into the following categories.
0 - None
1 - Basic/Terse
2 - Verbose (default when SY Param not set)
3 - Extended

Grouping:
Adds a id to each log line unique to that web request. Thus grouping a selection of web log lines into one request in log analysis.

Show Stack:
Usually used for debugging or developing by someone who is learning the code structures.

Stack Depth:
Will display a limited depth of 4 layers of the program stack.
