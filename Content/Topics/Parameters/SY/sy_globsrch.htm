SY/GLOBSRCH - Global search

Whether the Global Search is in use (Y or N).

If this is set to Y, when records in tables that are configured for searching are changed, the search index is updated.

If this is set to N, the search results only include COINS functions.

You can override this setting for individual users on the Preferences tab of Users.

NOTES

There is a menu-based function, Global Search, which is available regardless of the setting of SY/GLOBSRCH.

The search index process does not check the parameter so it is still possible to manually index a table using the browse actions on Global Search Configuration, but no subsequent changes will be indexed.
