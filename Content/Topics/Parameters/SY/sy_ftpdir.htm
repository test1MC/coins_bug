SY/FTPDIR - Generalised FTP import directory

(COINSplus only) The directory into which files are uploaded from the server. This must be a directory that all relevant users have access to.

This is used by data import programs that upload files using FTP:

  • Load Data Stores (jc906.p)
  • Load data into timesheet activities (jc910.p)
  • Import Performance Review data (pi902.p)
  • Import purchase order lines from CSV file (po101im.p)
  • Bulk Loading of SOR records (sw202.p)
  • Test Library Import (rp225.p)
  • COINS Plus FTP (syftp.i)
  • Import Site House Type Schedule (ti700im.p)
  • Plot Budget Maintenance (vp218.p)
  • Import House Type Budgets from Excel (vp705.p)
  • Import Expected Build Completion Dates (vp706.p)
  • Import House Type Budgets from C21. (vp707.p)
  • Import Report Stores (vp712.p)
  • SY Parameters (syusyp.i)
  • ASTA Export (hs714.p )
