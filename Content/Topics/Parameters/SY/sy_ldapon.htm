SY/LDAPON - Use LDAP Authentication

Whether to use LDAP authentication (Y or N).

Y = LDAP based authentication of user access to COINS will be enabled for all COINS user accounts that have the LDAP User@Domain field populated with their corporate network user identifier. This field is displayed and managed on the Details tab of Users.

N = All LDAP based authentication is disabled and the current COINS account password will be used for user authentication. The LDAP User@Domain field will not be available on the Details tab of Users.
