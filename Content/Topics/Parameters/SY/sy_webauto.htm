SY/WEBAUTO - Web Auto Login Request Page

The complete URL to the page which will perform the auto login request.

Automatic login allows users to log in to COINS without entering a password, by using the Windows network login information. If WEBAUTO is blank then auto login will not be used and the users will need to login manually to COINS using the COINS user ID and password.

The user's domain and user name must also be set correctly in the Domain\\User field on the user record.

NOTE

Automatic login only works when using Internet Explorer.
