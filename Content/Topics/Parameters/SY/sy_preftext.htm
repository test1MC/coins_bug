SY/PREFTEXT - Text on buttons

Whether to show (selected) buttons with text labels instead of, or as well as, images.

Select Icon, Text, or Text and Icon to show buttons with images only, text only or both text and images.

You can override this default setting for individual users with Users or User Preferences.

NOTE

Not all buttons have the option to show text. Those that don't will show as an image, whichever setting you choose.
