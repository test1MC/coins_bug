SY/WUNLOAD - Warn user if leaving page with unsaved data

Whether to show a confirmation prompt if the user navigates away from a page that has unsaved data (Y or N).

Y = Users will receive a warning if they try to navigate away from a page that contains active Undo and Save buttons. "Navigate away" includes following a link that would replace the page, refreshing the page or closing a tab or a window.

N = COINS allows the user to leave a page that has unsaved information on it, without warning.

You can override this setting for individual users: set the Unload Page Warning field (on the Preferences tab in Users).
