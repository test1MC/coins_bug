SY/PREFLKUP - Recall Filters for lookups

The default for how COINS remembers the filter settings on lookups.

When a user opens a lookup that they have used previously, COINS can pre-fill the lookup with the last settings they entered. This controls how long COINS remembers the settings.

Full = Always remember filter settings.

Day = Remember filter settings from earlier today (first time on each page would show no filters).

Session = Remember filter settings from the current session only (first time on each page since login would show no filters).

None = Never remember filter settings (filters always show blank first time on a page).

You can change the setting for an individual user in Users.
