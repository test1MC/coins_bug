SY/SIDEFOOT - Side frame footer logo

An optional logo to display at the bottom of the menu frame.

To display:

  • No logo, leave this blank.
  • The standard COINS logo, enter !.
  • A different logo, enter the URL of the image file. This can be either an absolute URL (for example, http:/ /123.45.67.89 / resources/ images/ logo1.gif) or relative to the COINS installation (for example: /coins/ oalive/ images/ logo1.gif).

    • Since this image will be referenced by lots of screens, if the image location takes a long time to resolve, the performance of the software can be affected. It is best for the image to be local and not, for example, on a website.

    NOTE

    This allows for a logo 50px high; if you want to use a logo with a different height, you will need to use a style sheet to modify the height of DIV.logoarea.
