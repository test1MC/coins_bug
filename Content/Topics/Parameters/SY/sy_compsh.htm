SY/COMPSH - Compile Script

The script that is used to start a COINS compile session.

A COINS compile session allows users to compile generate programs created from dataset definitions. These can also be hand-written and can include a FOP production section that speeds up the FOP file production over the configured reports. This requires an additional PROGRESS ; some aspects require PROGRESS 4GL skills and OA technical skills.

The output of the compile is captured and shown to the user, so output redirection should not be included.

For example: $BASE/bin/coinsbg.sh -b -p syucom.p

If COMPSH is populated and NCOM is set in the COINS , the Compiler option will be available on the Users screen.
