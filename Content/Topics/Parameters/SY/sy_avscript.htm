SY/AVSCRIPT - Antivirus command script

The script to run an anti-malware command.

If this is blank, whenever a document is uploaded to COINS the default anti-malware protection (ClamAV - www.clamav.net) is run. If ClamAV is not installed, users will be prevented from uploading documents.

To suppress the anti-malware protection, enter ! (exclamation mark) in this parameter. You might do this if you are confident that all endpoints (for example, PCs that access COINS) are secure.

To use an alternative anti-malware protection, enter the appropriate script to check a file.
