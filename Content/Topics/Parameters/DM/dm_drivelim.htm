DM/DRIVELIM - Maximum file size can transfer in one COINS Drive WS Call

Sets the File Size limit (in MB) that can be retrieved from COINS DM via the COINS Drive web-service (DMESB006) in one call.

COINS DM can be accessed from external systems via a web-service (DMESB006). The web-service can return file content in the form of base 64 encoding so care has to be taken that processing does not take up too much capacity.

This parameter restricts the amount of processing that the web-service will have to do in one call. If not set it defaults to 20MB.
