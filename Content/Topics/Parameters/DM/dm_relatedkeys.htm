DM/RELATEDKEYS - List of keywords descriptions that link documents

A comma-separated list of the long keyword descriptions that will be included in lookups to find related documents.

For example, poh_ordno on the purchase order and ohd_num on the invoice can have the same long description of 'Order number'. In this case, the paperclip button would show a list of all documents in the same with the same order number, or any documents that are not -specific with the same order number.
