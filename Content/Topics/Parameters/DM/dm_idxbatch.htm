DM/IDXBATCH - Document tables that can index from a batch in the paperclip function

A comma-separated list of the tables that can be indexed from a pool or batch when the paperclip function is selected on the table itself.

When you click the paperclip button to attach a document, if the table is listed in IDXBATCH, an Index from Batch tab is available. This allows you to filter on a document pool or batch.
