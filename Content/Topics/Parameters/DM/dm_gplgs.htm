DM/GPLGS - Use GPL Ghostscript to merge PDF documents

Set this parameter to Y if you wish to use GPL Ghostscript to merge PDF files.

The COINS standard merge scripts do not support the merging of PDF files created from Microsoft Office.

The latest versions of GPL Ghostscript do allow this.

If this parameter is supported; the software will assume that GPL Ghostscript is in the unix PATH of all report agents.
