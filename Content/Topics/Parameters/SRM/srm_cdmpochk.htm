SRM/CDMPOCHK - CDM Check - Subcontract Order Entry

When a user adds a subcontractor to a purchase order, what do you want to happen if the subcontractor does not have a current CDM approval or the approval will soon expire?

- None = Allow the user to add the subcontractor.

2-LEVEL = If the approval is due to expire within the run up period specified by SRM/CDMPOLD, issue a warning; if there is no current approval, do not allow the user to add the subcontractor.

ERROR = Do not allow the user to add the subcontractor.

WARNING = Issue a warning, but allow the user to add the subcontractor.
