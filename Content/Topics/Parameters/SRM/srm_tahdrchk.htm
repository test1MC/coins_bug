SRM/TAHDRCHK - Header code length to check Trading Agreements

Whether to allow for variable length Header Codes when checking Commodity/Trade Trading Agreements.

FIXED = (or blank) COINS checks trading agreements against Header code N characters long (N is set in PO/POHDRLEN for Commodities, PO/SCHDRLEN for Trades and PO/PLHDRLEN for ).

VARIABLE = COINS tries to find a Header code N characters long; if not found, then N-1 characters long, and so on; as soon as a Header code is found, it checks Trading Agreements for it.
