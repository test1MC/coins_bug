SRM/CDMPOLD - CDM Days to Expiry - Subcontract Order Entry

The run up period (in days) for purchase orders.

If you are using two-level checking for purchase orders (CMDPOCHK is set to 2-LEVEL), COINS will issue a warning if a user tries to add a subcontractor to an order and your CDM approval for that subcontractor is due to expire within this number of days.
