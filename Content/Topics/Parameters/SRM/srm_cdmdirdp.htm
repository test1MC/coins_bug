SRM/CDMDIRDP - Direct CDM (SSIP) Update - CAN-DO List of Data Providers

A can-do list of data providers from which CDM approval and updates for a vendor are processed directly; updates from any other source will appear in the Amended Companies tab of the Company Information Workbench.

The data providers are identified by numbers, as follows:

1 = iPortal.

2 = Constructionline.

3 = Builder's Profile (basic).

4 = Builder's Profile (premium).
