SRM/IPRTHOST - iPortal Host (for ESB Publish)

The iPortal host. This is needed so that the ESB Broker knows which system to send data to.

Typically, this will be something like "uat.coinsiportal.com" during testing, or "www.coinsiportal.com" for the live iPortal. This parameter is consulted by the ESB Database Publish and ESB Process Publish routines.
