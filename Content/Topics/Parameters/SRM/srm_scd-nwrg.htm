SRM/SCD-NWRG - Status Code for New Registrations from iPortal

The status code for new registrations received from the iPortal.

NOTE

The code used here should NOT be set up in the Group Status table. It is really an internal device that allows COINS to store and show new company information on the New Companies tab within the iPortal Monitor before the company is properly brought into the Company Information Workbench.
