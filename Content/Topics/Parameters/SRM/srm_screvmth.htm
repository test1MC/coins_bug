SRM/SCREVMTH - SC PO Performance Reviews - Set 'Rev Required'

The method for determining whether performance reviews are required for a subcontract order.

Manual = You specify on each order whether a review is required. The default value can be set using PO/SCREV.

Auto = Whether a review is required depends on the order value or the subcontractor's status.

In this case, a review is required if the order value is over the limit set by SCREVLIM or if the subcontractor status is one of those listed in REVSTAT. See also SCREVMIN.

Override = The Review Required field will be updateable (as if the parameter was set to 'MANUAL'), but will be automatically assigned, by looking at the Group Status Code of the company (see REVSTAT).

Weighted = This provides the option to hold weightings against each criterion. Set up review styles (using Performance Review Styles) to hold the weightings.

Weighted-OV = This provides the option to hold weightings against each criterion, as above. The Review Required field on the order is ed by default, but can be changed.
