SRM/POREVMTH - PO Performance Reviews - Set 'Rev Required'

The method for determining whether performance reviews are required for a material order.

Manual = You specify on each order whether a review is required. The default value can be set using PO/POREV.

Auto = Whether a review is required depends on the order value or the 's status.

In this case, a review is required if the order value is over the limit set by POREVLIM or if the status is one of those listed inREVSTAT. See also POREVMIN.

Override = The Review Required field will be updateable (as if the parameter was set to 'MANUAL'), but will be automatically assigned, by looking at the Group Status Code of the company.

Weighted = This provides the option to hold weightings against each criterion. Set up review styles (using Performance Review Styles) to hold the weightings.

Weighted-OV = This provides the option to hold weightings against each criterion, with the ability to override whether a performance review is to be carried out on the order in question.
