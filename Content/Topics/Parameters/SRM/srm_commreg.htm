SRM/COMMREG - Commodity Code for Registered Invoices

The dummy commodity code under which registered invoices are d in Spend (Commodity).

The commodity code must be set up as a real code but should not be used to identify real commodities.

Spend (Commodity) is by year, and treats registered invoices according to the period in which they were registered. Therefore, if an invoice was registered in the final period of one year and costed in the first period of the next, the invoice value will appear in the earlier year.
