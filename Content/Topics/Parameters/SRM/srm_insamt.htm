SRM/INSAMT - The Insurance Amount to import from the iPortal

Which insurance amount supplied by the iPortal (Total Cover or Single Claim) should be stored in COINS?

SINGLECLAIM = The value stored in COINS is extracted from the <cxc_single_claim> element in the inbound message sent from the iPortal. This is the default if INSAMT is blank.

TOTALCOVER = The value stored is extracted from the <cxs_amount> element in the inbound message.
