SRM/INSDIRDP - Direct Insurance Update - CAN-DO List of Data Providers

A can-do list of Data Providers (1,2,3,4) from which updates to vendors' insurance details are updated automatically in COINS without having to be accepted via the COINS Monitor "Amended Companies" tab.

If the inbound insurance is specified in the INSDIRTP parameter and the data provider of the company's information is specified in the INSDIRDP parameter then the company's insurance records will be processed directly. The company will only show on the Amended Companies tab if other changes are detected.

The data providers are represented by numbers as follows:

1 = iPortal

2 = Constructionline

3 = Builder's Profile (basic)

4 = Builder's Profile (premium)
