SRM/SCD-APP - Status Code for Approved Companies (OA)

A comma-separated list of Group Status Codes to use to identify "approved" companies.

Where more than one entry exists, the following rules apply:

  • The tab labels in the SRM workbench will be taken from the first entry.
  • When cascading-up the status of a supplier/subcontractor, the first entry will be used.
  • The first value will be used when running actions such as changing a company's status.

Set up the possible values for status types using Status Codes (Group Wide).

There is a hierarchy of status codes: the code in SCD-PART is the highest, followed by SCD-PREF, then SCD-APP, then SCD-REG, then any others.

COINS displays the description of this status code in the list of group status codes in the search tab, and the tab label for 'approved' companies. If SCD-APP is blank, the tab disappears.
