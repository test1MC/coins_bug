SRM/IPRTID - iPortal Services User Id

The services user ID to be used when COINS communicates with the Self Service Portal (iPortal).

Refer to COINS. This value is used to authenticate the web service communication between the local COINS system and the iPortal. There must be a user record with a user_type of "webservices" in the iPortal database that has the user_email field set to this value. Typically this will be in the format "coins#contractor.co.uk".
