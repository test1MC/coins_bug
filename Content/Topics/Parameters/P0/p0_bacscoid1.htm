p0/BACSCOID1 - BACS Record Type 1 Company Id

Data that should appear in Positions 14-23 of Record Type 1 of the file. (The Record Type value appears in Position 1 of the record.)

Unless requested differently by the bank, the first position of the field (Position 14) should be blank or 'b'. You cannot enter leading spaces for this parameter. If the length of the parameter you entered is less than 10 characters, the file generate process will add the leading spaces automatically. If the length of the parameter you entered is more than 10 characters, the file generate process will take into account only the first 10 characters.
