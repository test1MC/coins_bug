P0/ERROR-08 - Employee [val] has status [x] - cannot do load. [PR12]

The way the Timesheet Load should behave if it encounters this error.

Possible options are:

RESET = Resets this employee and then loads the line (does not apply to Post Calculation Cost Load and the other frequency-specific versions).

RjctFile = Prevents loading of the entire file (default option).

RjctLine = Prevents loading of only the line with the error.
