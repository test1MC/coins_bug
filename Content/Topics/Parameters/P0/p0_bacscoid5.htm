p0/BACSCOID5 - BACS Record Type 5 Company Id

Data that should appear in Positions 41-50 of Record Type 5 of the file. (The Record Type value appears in Position 1 of the record.)

  • This parameter does not require a leading blank.
  • Record Type 5 is a Header type record. The data from this parameter is also used in Positions 45-54 of Record Type 8, which is a trailing or footer record denoting the end of a group of transactions begun with a Type 5 record.
