P0/TSL-CNFG - Timesheet load file configuration

A Configuration parameter that specifies the data layout in the Timesheet Load File as a comma separated list of fields.

A simple load has one item value per line, for example:

EMPLOYEE,ITEM,INPUT,JOBNUM

You can load multiple values per line by specifying a column for each:

EMPLOYEE,ITEM_I11,ITEM_12,ITEM_I13,JOBNUM

The full list of possible entries is:

EMPLOYEE - Employee ID (clock or NI number)

ITEM - Payitem number

INPUT - Item input value

INPUTHHMM - Item input value (hours.minutes)

RATE - Item rate value

ITEM_I??? - The input value for item number ???

ITEM_H??? - The input value (hh.mm) for item number ???

ITEM_R??? - The rate value for item number ???

METHOD - Costing method

COINSCO - COINS company number

JOBNUM - Contract Number

PHASE - Contract phase

SECTION - Cost Code Section

ACTIVITY - Cost Code Activity

SECT-ACT - Combined Section and Activity

COSTHEAD - Contract cost head (or cost code)

CATEGORY - Contract cost category

PLTNUM - Plant asset code

PCH - Plant cost head

GLACCT - General ledger account code

WORKSHOP - Workshop reference

WCH - Workshop cost head

COMMENT - Comment text

HOLIDAY-PD - Employee holiday periods

LEAVE-DATE - Employee

DAYSSTRING- a hyphen separated list of days of the month and the timesheet code for that day. The pay item for the timesheet code will be updated with the day count.
