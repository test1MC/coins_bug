CS/REFBASE - Int Ref Base (PPYYMMYYYY)

The format of transaction reference numbers. Reference numbers consist of an optional base plus a format.

The base is a combination of digits taken from the batch General Ledger period and financial date. REFBASE is a list of the positions of the required digits from the following sequence: PPYYMMYYYY (where PP = ; YY = period year; MM = month; YYYY = year).

The format is a PROGRESS display format, such as 9999.

For example, a REFBASE of 9,10,5,6 and a REFFORM of 9999 produce a transaction reference number made up of the last two digits in the year, the two digits of the month, and four digits, such as 95050001.
