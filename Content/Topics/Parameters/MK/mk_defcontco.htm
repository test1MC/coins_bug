MK/DEFCONTCO - Comma separated Company and Office internal reference number automatically assigned when new Contact is created

This parameter allows you to assign a default and Office to a new Contact created in the New Contact function when the record does not have any or offices associated with it.

The main purpose of this functionality is to ensure that all contacts have at least one associated /Office (even if it is "None") so that such contacts can be displayed on the enhanced Contact lookup.
