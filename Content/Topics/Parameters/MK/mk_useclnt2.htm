MK/USECLNT2 - Use second Client

Whether to allow you to record a second client against a (Y or N).

This allows you to define a 'Contractual' Client and an 'Ultimate Client' for a .

When first setting this parameter, you need to update your current data. To do this, run the mkuppr01.p utility from the dot prompt within , then input the Internal Reference of the Client Company, which should be first Client Type for any currently having more than one client.
