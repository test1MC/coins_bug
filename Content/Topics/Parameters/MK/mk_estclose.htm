MK/ESTCLOSE - Estimated Close Date

Determines which date field to use when calculating totals of in x# of months, including current month. Valid values are pi_project.pij_close or pi_project.pij_estclose.

The value for this parameter should be a pipe-separated list of two items: the date field name from the pi_project table, and the corresponding description for the column in the report. For example: pij_close| Close Date.
