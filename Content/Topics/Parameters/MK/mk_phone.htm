MK/PHONE - The Phone number type related to the Phone field in Contact details

The phone number type that corresponds to the Phone field in Contact Details and on the Telephones tab.

When a user adds or updates a number in the Phone field, COINS adds or updates a record with this phone type on the Telephones tab. Similarly, changes to a number of this type on the Telephones tab are reflected in the Phone field in Contact Details.

Telephone number codes are set up using Lookup Codes.
