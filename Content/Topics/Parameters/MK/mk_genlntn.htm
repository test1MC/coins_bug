MK/GENLNTN - Generate the same Lead and Tender Numbers

Whether you wish to generate the same Lead and Proposal numbers. This is blank by default.

If this parameter is:

Blank - the system uses number generators based on the settings of other parameters, such as MK/GENLN, MK/GENTN, etc.

Set to Y - the MK/GENLN and MK/GENTN parameters also need to be set to Y in order for the GENLNTN functionality to work properly.
