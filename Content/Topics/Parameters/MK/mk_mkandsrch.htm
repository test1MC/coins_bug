MK/MKANDSRCH - Use alternative 'AND' company search algorithm

Whether uses an existing company search capability or a new search algorithm.

The options are:

Y = Company duplicate search in uses a new search algorithm based on a combination of company name, office name, and address values. This algorithm is used when adding a new company, performing a search for a company, and importing a company.

N = uses an existing company search capability when adding a new company, performing a search for a company, and importing a company.

Blank = Same as N.
