MK/PISECTYPE - Security Type

The type of security to be used.

The options are:

  • If MK/PISECTYPE is set to Full Record Security (RS), the Webservices do not display any records that the user is not allowed to access.
  • If MK/PISECTYPE is set to Field Security (FS), and field security should be applied to some records, this new field should be set to True.
