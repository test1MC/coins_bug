HS/CANRESIN - Allow cancellation of incomplete reservations

Whether reservations can be cancelled and deleted before the Reservation Entry Complete (RESCOMP) event has been run on the Sales Workbench. This allows a reservation to be cancelled in COINS more efficiently and prevents authoration workflows from being triggered on a reservation that will not be progressed.

If this is set to Y, the Cancel Sale event is available on the Sales Workbench options menu, and the reservation event can be deleted from the History tab without getting an error.
