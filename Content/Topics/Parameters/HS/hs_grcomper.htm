HS/GRCOMPER - Include plots completed within G/R period

Whether to include in the ground rent period plots that have been completed in the middle of the period.

Y = Plots that have been legally completed during the ground rent period you are billing are included. The charge is pro-rated. Charges that have already been billed will not be included.

N = Plots that have been completed during the period you are billing are not included.

If you are billing in arrears, this parameter should be set to Y.

If billing in advance, it depends on how the completion statement is configured:

  • If the completion statement is configured to include the rent, set this to N.
  • If the rent is not included on the configuration statement, set this to Y.
