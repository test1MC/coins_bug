HS/PWQAFLDS - Prospect quick add fields to show (OA)

Select the list of fields to show in Quick Add.

You can select to the show the following:

  • Source of - If you select not to show this then you should set up a default field value for the field hvi_init_htr_sourceEnq.
  • Reason for Move - If you select not to show this then you should set up a default field value for the field hvi_ReasonMove.
  • Status - If you select not to show this then you should set up a default field value for the field hvi_status.
  • Level of Interest - If you select not to show this then you should set up a default field value for the field hvi_level.
