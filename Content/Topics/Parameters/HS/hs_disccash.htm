HS/DISCCASH - Discount includes cash only

Whether the discount amount for a plot includes only cash incentives.

This affects the 'Discount/Allowances' amount displayed on the summary tab for the plot in the sales workbench and any report or using field vwb_discount (including the Financial Summary tab in the sales workbench). It does not affect the calculation of net prices or journals. If this parameter is changed it will only affect plots where the discount amount is updated in future - existing discount amounts will not be recalculated.

Y = The discount amount only includes incentives where the incentive code has a type of 'Cash'.

N = The discount amount includes all incentives.

NOTE

This is only relevant if you are using a breakdown of discounts and incentives - see DISCBRKD.
