HS/SCCOMPER - Include plots completed within S/C period

Whether to include in the service charge period plots that have been completed in the middle of the period.

Y = Plots that have been legally completed during the service charge period you are billing are included. The charge is pro-rated. Charges that have already been billed will not be included.

N = Plots that have been completed during the period you are billing are not included.

If you are billing in arrears, this parameter should be set to Y.

If billing in advance, it depends on how the completion statement is configured:

  • If the completion statement is configured to include the charges, set this to N.
  • If the charges are not included on the configuration statement, set this to Y.
