HS/SCHEXROL - Roles to be excluded from unscheduled tasks

The roles to be excluded from unscheduled tasks.

This allows one or more of the following sales staff roles:

  • Plot Sales
  • Part Exchange Sales
  • Sales Manager
  • Sales Director
  • Call/Issue Manager
  • Inspector
  • Site Agent
  • Lettings Advisor
  • Tasks

If this is changed then Regenerate Unscheduled Tasks should be run.

After the ‘Regenerate Unscheduled Tasks’ process has been run and completed,Initialise Summaries is need to be run.
