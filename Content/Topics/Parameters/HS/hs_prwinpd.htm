HS/PRWINPD - Default input date for Tick Sheet and Whiteboard (OA)

The default input date for the Construction Workbench progress tick sheet and the Construction Workbench whiteboard.

Options:

<blank> - Previous week end date (default)

BLANK - The date will be left blank

TODAY - Today's date

FRIPREV - Previous Friday

FRICUR - Current Friday

YESTERDAY - Yesterday's date
