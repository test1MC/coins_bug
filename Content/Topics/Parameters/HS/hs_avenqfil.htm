HS/AVENQFIL - Availability enquiry filter defaults

( only) Which fields on the availability filter to populate with data from the 's 'required' details.

The availability (within the summary) allows you to find plots that match the 's requirements. This parameter allows you to choose which of the requirements to use to filter which plots are included. This determines the default values for the advanced filter, but users can change the filter details to search on different criteria.
