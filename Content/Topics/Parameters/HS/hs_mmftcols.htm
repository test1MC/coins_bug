HS/MMFTCOLS - Market Mix Sq. Ft. Report Columns

The ranges of square footages that determine the columns in the Market Mix by Square Feet Report.

For example, setting this parameter as follows:

0-1000,1001-1500,1501

Would display 3 columns. The first column would include plots with a between 0 and 1000 square feet, the second column would include plots between 1001 and 1500 square feet. The final column would contain plots with a above 1500 square feet.
