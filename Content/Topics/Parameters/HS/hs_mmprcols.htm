HS/MMPRCOLS - Market Mix Product Report Columns

The ranges of market mix codes that determine the columns in the Market Mix by Product Type Report.

For example, setting this parameter as follows:

0-1Z,2-2Z,3-Z

Would display 3 columns. The first column would include plots with a market mix code that lies in the range 0 and 1Z. The second column would include plots with a market mix code in the range 2 and 2Z. The final column would contain plots with a market mix code between 3 and Z.
