HS/MMPRHEAD - Market Mix Product Report Column Headings

The column labels for the ranges of market mix codes that are specified in the HS parameter MMPRCOLS.

For example, setting this parameter as follows:

1 Bed,2 Beds,3 Beds

Would display the column labels as '1 Bed' for the first column, '2 Beds' for the second column and '3 Bed' for the last column.
