HS/SLAUSE - Use SLA dates?

Determines whether you want COINS to calculate an SLA date for issues.

On activating this the issue SLA date will be displayed on the various issue browses in the workbench.

The SLA days can be specified using Issue Priority Codes and Call Types. When you first enable this feature, or if you amend the SLA days later, you will need to regenerate the SLAs using Regenerate SLAs.
