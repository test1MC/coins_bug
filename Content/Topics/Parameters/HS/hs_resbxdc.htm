HS/RESBXDC - Expected build completion calculation contingency days (OA)

The number of contingency days to use when recalculating the expected build completion date at the point of reservation.

It consists of a a list of stages and the number of contingency days. For example: 50|5,80|7,99|14.

In the above example, if the current stage of the plot is less than or equal to 50 then an additional 5 days will be added as contingency. If the current stage of the plot is less than or equal to 80 and greater than 50 then an additional 7 days will be added as contingency. If the current stage of the plot is less than or equal to 99 and greater than 80 then an additional 14 days will be added as contingency.
