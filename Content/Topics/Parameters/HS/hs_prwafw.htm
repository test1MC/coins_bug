HS/PRWAFW - Actual/forecast warning days (OA)

The number of warning days used when comparing the actual construction stage date with the forecast date or the forecast date with today's date.

If this is set to a positive value, then the warning is based on how far behind the selected actual construction stage date is compared with the forecast date.

If this is set to a negative value, then the amber warning is based on how far ahead the actual construction stage date is compared with the forecast date.
