HS/CSFILL - Construction Methods for which earlier stages are auto-completed

A can-do list of construction methods for which earlier stages are auto-completed.

  • For plots with a construction method included in this parameter, a construction stage event will be automatically created for each incomplete earlier construction stage when a later construction stage is recorded as complete (so that there are no gaps in the recorded stages (within the list which is applicable to the build for the plot)). All fields (other than the construction stage) on these events will be set the same as for the construction stage event actually being recorded.
  • If this is left blank or for plots with a construction method not included in this parameter then COINS will work dependent on the settings of HS parameters CSDUP and CSSEQ.

NOTE

For plots with a construction method included in parameter CSFILL, this will effectively override the setting of parameter CSSEQ as any gaps will be automatically filled. However, parameter CSDUP works in conjunction with CSFILL ? controlling whether a user can input the same stage as currently recorded for the plot.
