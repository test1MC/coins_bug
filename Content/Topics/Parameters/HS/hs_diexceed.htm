HS/DIEXCEED - Action if Disc/Incentive level exceeded

Whether users can exceed their authored Discounts and Incentives level.

ERROR = COINS prevents users from entering Discounts and Incentives on a plot that exceed their D&I level.

WARN = Users will get a warning message but will be allowed to proceed.

For this functionality to work, users need to have a DI level set up (in Sales Staff), and the HS parameter PLOTLEV needs to be set.
