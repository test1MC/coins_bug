HS/LEVEXT - Level to define available plot extras

The level at which the list of available plot is defined.

Select one of the following:

COMPANY = A standard list for the whole (default). Define the list using Standard .

STANDHT = Each standard house type has its own list.

DEV = Each has its own list.

DEVHT = Each -specific house type has its own list.

PLOT = Each individual plot has its own list.

You define at one level and copy from that list to a lower level. For example, you could have a list, which you copy to each and modify, or a list for each standard house type which you copy to each plot and modify.
