HS/TPEMSAVE - Save Task Print into Document Management after email (OA)

Whether to store the document against the call after emailing the task print.

If set to Y the document will be stored against the call. An event will also be added to the call with the contents of the email and a link to the document. The task print document will also be stored to any tables configured against the configurable form.

This relies on Document Management being set up and a Document Management Link existing for the table hs_complaint.
