HS/CSLSTGV - Last construction stage additional validation (OA)

Determines the additional validation required when recording the last construction stage for a plot.

  • ERROR-ALL - If the actual date of any of the preceding stages have not been entered then an error will be given to the user.
  • ERROR-CUR - If the actual date of any of the preceding stages that have a current forecast date have not been entered then an error will be given to the user. Note the HS parameter FCASTCUR determines the current forecast.
