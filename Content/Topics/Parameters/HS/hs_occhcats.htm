HS/OCCHCATS - List of cost head/cat for manual charges

A list of and categories for those transaction categories against which you want to post manual charges as costs (rather than posting them to the HS transaction category).

Enter a comma-separated list of and categories; for example: M1002.MA, M1003.MA. would cost postings to the first transaction category listed in HS/OCCATS to M1002.MA, and cost postings to the second category to M1003.MA. Leave an item in the list blank to post to the transaction category; for example: ,M1003.MA.
