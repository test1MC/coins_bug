HS/EWDIUSED - Error/Warn if PX DI used > Plot DI

Determines whether an error or warning message is displayed if the part exchange discount/incentive used is greater than the discount/incentive allowed for the plot. (Only relevant if DISCBRKD = Y).

Select one of the following options:

<blank> = No message is displayed (default).

ERROR = Error message will be displayed.

WARN = Warning message will be displayed.
