HS/RETINST - Use return to enter issue instructions

(COINSplus only) Whether the Return "key" in task allocation can be used to select an issue or to enter issue-specific instructions.

Y = When assigning issues to a task, Return allows you to enter issue-specific task instructions; F7 allows you to assign the issue to the task.

N = When assigning issues to a task, F7 allows you to enter issue-specific task instructions; Return allows you to assign the issue to the task.
