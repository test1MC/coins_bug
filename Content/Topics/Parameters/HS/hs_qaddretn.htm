HS/QADDRETN - Allow return in quick add editor fields

(COINSplus only) In the Quick Add screen, whether you want the Return "key" to move the cursor to the next line (Y or N).

In the Quick Add screen, there are two editor fields, Issue Description and Task Instructions. QADDRETN controls how the Return key behaves.

Y = Pressing Return moves the cursor to the next line of the input field.

N = Pressing Return moves the cursor to the next field.
