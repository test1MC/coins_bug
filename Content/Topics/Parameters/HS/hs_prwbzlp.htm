HS/PRWBZLP - Stages to fix at zero duration if zero duration in construction program (OA)

A list of construction stages where if there is a zero duration in the construction then the stage duration should always be kept at zero.

This is useful for stages that are not on the critical path (e.g. brickwork stages in timber frame construction) so they have zero duration in the so as not to effect the build complete date. This will ensure these stages do not fill out any gaps in the Gantt chart report.
