HS/RESTACTU - Restrict update of actions from w/benches

Whether to to apply activity workbench security to the actions lists within House Sales workbenches.

Activity Workbench Access Maintenance allows you to control which users can view or update which other users' workbench activities This parameter allows you to apply this security to updating actions in House Sales workbenches.

Y = Users can only update actions if they have permission.

N = Users can update all actions.
