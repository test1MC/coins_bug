HS/TPEMSUB - Task Print email subject line configuration (OA)

The subject line to be sent on task print emails.

All fields on the task, complaint, plot, and records can be used in this description. The name of the field should be encapsulated in curly brackets. In addition you can add a pipe to specify the number of characters to show e.g.

{job_num} {vsi_name|15}

Would show all of the number (job_num) and the first 15 characters of the name (vsi_name).
