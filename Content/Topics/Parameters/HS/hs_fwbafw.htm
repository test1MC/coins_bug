HS/FWBAFW - Forecast workbench actual/forecast warning days (OA)

The number of warning days used when comparing the forecast date with today's date. If the date is within the number of days then it will be coloured amber.

If this is set to a positive value, then the warning is based on how far behind the selected date is compared with today's date.

If this is set to a negative value, then the amber warning is based on how far ahead the date is compared with today's date.
