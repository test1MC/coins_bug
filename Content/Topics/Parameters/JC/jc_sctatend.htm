JC/SCTATEND - Section placed at end of Cost Code

Whether to allow to place the section at the end of the cost code. This is intended for use by housebuilders, to enable plot costing.

Y = When setting up a new , you have to option to specify that the section is at the end of the cost code.

N = No can have the section at the end of the cost code.

If SCTATEND is Y, DEFCHLEN specifies the fixed length (not just the default length). In addition, the Cost to section field in Standard Maintenance is enabled.

On that have the section at the end of the cost code, you cannot use activities, so you would normally set the parameter DEFACT to blank.
