JC/CDRUPDPT - Allow update pending transactions on Submitted Daily reports

Allow updates on pending transactions in Submitted .

Comma separated list of values:

  • ALL - for all tabs in Daily Report
  • LABOR - For CDR Detail
  • EQ - For CDR Detail
  • MISCPR - For CDR Payroll
  • MISCEQ - For CDR

Samples:

  • ALL - Updates are allowed on all tabs
  • LABOR,MISCPR - Updates are allowed on Detail and Payroll tabs only.
