JC/MOVESTOR - Monthly movement datastores with types

A list of data stores that COINS uses to determine whether to generate a for a .

For example:

If MOVESTOR = storea|td, then for every Generate will evaluate the TD value of StoreA for the Open OA period. If the value is 0, no will be generated for that .

If MOVESTOR = storea,storeb|tp,ty then Generate will evaluate the TP and TY values of StoreA and StoreB (with respect to the OA period), and if all four values are 0, no will be generated.
