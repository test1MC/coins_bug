JC/COSECDEF - Default contract security when blank to NONE

Whether a missing record in in Security means 'None' or 'All'.

Y = If there is no record for the user in Security, the user has access to no . The exception is that if the security field (specified by JOBSEC) on a is also blank, the user will have access to that .

N = If there is no record for the user in Security, the user has access to all .
