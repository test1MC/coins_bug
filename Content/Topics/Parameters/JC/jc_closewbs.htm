JC/CLOSEWBS - Allow closure of WBS Codes

Whether COINS checks if WBS codes and cost codes are closed.

This allows WBS codes to be closed off once the work is complete, to prevent costs being allocated to them.

Y = COINS does not allow users to enter a WBS code in an application area (for example, requisition entry or purchase order entry) if it is closed for that application area.

N = COINS does not check whether WBS codes are closed.

Use Open/Close WBS Codes to close WBS codes.
