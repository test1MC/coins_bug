JC/QFORCH2 - Quick Forecast Cost Heads - Sales/GP

A comma-separated list of "dummy" to be used when entering the forecast to complete Sales (revenue) figures on the Monthly Review screen. For example: ZEAA,ZEAB,ZEAC where ZEAA = Net Order Value, ZEAB = Agreed Variations, ZEAC = Unagreed Variations.

The must be valid standard , but you must prevent revenue being posted to them (for example, by leaving the G/L codes blank or giving them a class that is not allowed on ). The first code in the list MUST represent the Original Order Value.

On the Monthly Review screen, actual revenue comes from the cost codes that are linked to these "dummy" codes; see FORQCON.
