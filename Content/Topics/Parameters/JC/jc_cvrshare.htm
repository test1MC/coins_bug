JC/CVRSHARE - Copy CVRs to shared directory

Whether to copy newly generated to a shared folder (Y or N).

Y = COINS copies newly generated into a folder of the same name as the COINS username of the user under folder $BASE/ var/ spreadsheets/ cvrshare. COINS automatically creates the folder cvrshare and all its subfolders. The user is expected to have Samba access to their respective subfolder to access their .

The user needs to click the COINS icon in CVRShare.XLS which initiates a macro to submit all their to COINS for processing.
