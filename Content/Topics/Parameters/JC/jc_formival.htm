JC/FORMIVAL - Forecasting - Manual Input of Sales values

A comma-separated list of the values of the field specified by FORMIFLD that identify the as one on which the Sales Value on the forecast is adjusted manually. For example, if of Type M1,M2 and M3 should allow the Sales Value to be entered (the values of the Type field on the are user-definable), FORMIFLD would be set to 2 (Type) and FORMIVAL would be the list M1,M2,M3.
