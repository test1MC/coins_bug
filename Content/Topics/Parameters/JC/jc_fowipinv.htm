JC/FOWIPINV - Memo Data: Invoiceable tasks not invoiced

The memo data type used to store the total value of invoiceable tasks that have not yet been invoiced. This is used on the Forecasting Workbench, for adjusting the sales to date forecast on those (typically maintenance ) identified by the FORMIFLD parameter.

If you leave this blank, the adjustment screen will not prompt for the invoiceable tasks value.
