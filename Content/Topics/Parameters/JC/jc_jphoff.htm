JC/JPHOFF - Switch off Contract Phases

Controls whether COINS allows you to set up phases as a way of dividing large into smaller units.

Y = Prevents the use of phases. Fields that relate to phases will not be displayed.

N = Allows the use of phases.

Contact COINS for clarification before changing this parameter. Before setting this parameter to Y, you should check that no phases already exist. You can use Phases in Report to do this.
