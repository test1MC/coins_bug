JC/ONGEN - On generate CVR notification

Whether the is sent an email notification when his/her are generated by COINS, and what type of email is sent.

blank = No email notification sent.

N = Email notification with link to Workbench but no attachment.

X = Email notification with link to Workbench and all of the user attached.

Z = Email notification with link to Workbench and all of the user attached in compressed form.

This parameter can be overridden at a personal level via OA -> Setup -> Notification.
