JC/FORQCON - Consolidation field for Quick Method

Actual costs and revenue displayed on the Commercial Summary screen are taken from cost codes that are linked to the "dummy" forecasting cost codes. This parameter specifies which analysis set is used to link codes.

Select one of the Analysis Sets.

For example if analysis set 4 is used to link codes, each "dummy" forecasting code will have a different value for that analysis set. If the first forecasting code has a value "x" for analysis set 4, the costs figure shown next to that forecasting code is the total of all cost codes that have "x" as the value for analysis set 4.

(The parameters QFORCH1 and QFORCH2 specify the "dummy" forecasting codes.)
