JC/USEWBSCH - Users who can change contract 'Use WBS'

A can-do list of users who are allowed to override the Use WBS Codes field on a .

You would use this if you have a standard way of setting up (either using, or not using, WBS codes), but want to allow some to be set up differently.

If this parameter is blank, all users can update the Use WBS Codes field.
