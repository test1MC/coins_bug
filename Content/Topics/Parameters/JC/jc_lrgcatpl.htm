JC/LRGCATPL - Large copy area templates

A list of templates which have more than 10000 cells in one worksheet marked as copy areas (for example: cvrnorth.xml,cvr22.xml).

This is provided to solve a problem with the Generate process failing on very large templates which had more than 10000 cells in one worksheet marked as copy areas.

Please note, the use of very large templates is not recommended by COINS. This parameter should only be used as a last resort. If a template has been in use without having been set in the parameter, and then the parameter value is changed to include the template, OR if the parameter value includes the template and the template has been in use, and then the template is removed from the parameter value, all the produced from the template in the next generation cycle will have empty copy areas (that is, their copy areas will not be retained from the last time they were uploaded).
