JC/RMPROCVR - Remove Processed CVR

Whether to delete the processed file after a has been uploaded (Y or N).

If this parameter is set to Y, and a is uploaded by the and is processed successfully, COINS deletes the processed file which is stored in the "processed" directory.

Y = Delete from processed directory.

N = Do not delete from processed directory.

Note: The processed file will only be deleted if the system has successfully compressed the file in the processed directory.
