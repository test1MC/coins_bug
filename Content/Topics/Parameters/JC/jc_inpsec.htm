JC/INPSEC - Create Sections during Cost Code Input

Whether to automatically create the required section when creating a new cost code for a that uses WBS (Y or N).

Y = In Cost Codes, you can enter or select an existing section, or enter a new section (and details such as the description and the classes that are allowed). COINS will automatically create the new section as long as all the details have been completed.

N = In Cost Codes, you can only enter or select an existing section.

The parameter also defines the default value for the related box on the Import Budget screens. When the parameter is set to Y, the Create Sections option will be ed by default.
