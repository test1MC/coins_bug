JC/FORSTVAL - Forecasting - (In)active Jobs values

A comma-separated list of two values of the field specified by FORSTFLD that represent inactive and active .

For example, if you use the Status field to represent the forecast status, the value of FORSTVAL would be I,A; if one of the analysis sets is used, the value of FORSTVAL would be two values set up on the analysis set.
