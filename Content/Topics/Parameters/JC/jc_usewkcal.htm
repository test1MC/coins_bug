JC/USEWKCAL - Use Weekly Contract Calendars

Controls whether COINS allows you to set up a calendar or use weekly costs.

Y = When setting up a new , COINS asks whether you want to set up a calendar.

N = COINS does not ask whether you want to set up a calendar (however, you can still set up calendars afterwards using Maintenance).

Set USEWKCAL to N if you do not intend to use weekly costs on (and do not intend to use the allocation routines available in Procurement).
