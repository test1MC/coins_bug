JC/ACCMETHP - Plant Accrual Method: (N G O or combn.)

The default setting for the accrual method held on the record. Enter one of the following values to determine the default for the Accrual Method () field in Maintenance:

G = Accruals are calculated from GRNs on all .

O = Accruals are calculated from Purchase Orders on all .

N = No allocation.

Blank = The user selects the accrual method (no default).

G,O,N = The user selects the accrual method, and the default is G (the default will be the first letter that appears in the comma-separated list).
