JC/EFFTDATE - Transaction Date in Eff/Date

Whether to use the same Date field on Status transactions as the transaction date.

Y = The Date field in transaction entry is treated as the transaction date, and the cost-effective date is the same as the end date. In these circumstances ensure JC/FDATELOC = N.

N = The Date field is the cost-effective date, and the transaction date is the same as the batch date.
