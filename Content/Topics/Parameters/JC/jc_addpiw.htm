JC/ADDPIW - Add project if not in PIW

When a new is created in Status, whether to automatically create a corresponding project in the Project Information System if no such project exists (Y or N).

Y = When a new is added in Status, a project is automatically created in the Project Information System in Central Repository.

N = Projects must be created in the Project Information System before they can be added to a .

The parameter JC/USEPIW must also be set to Y to enable this feature.
