JC/INPWBS - Create WBS during Cost Code Input

Whether to automatically create the required WBS when creating a new cost code for a that uses WBS (Y or N).

Y = In Cost Codes, you can enter or select an existing WBS code, or enter a new code and description. COINS will automatically create the new WBS code as long as a valid Section/Activity exists based on the WBS configuration.

N = In Cost Codes, you can only enter or select an existing WBS code.
