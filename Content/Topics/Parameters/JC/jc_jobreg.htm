JC/JOBREG - Contract Field denoting Buying Region

The field on the details record that represents the buying region (for rebate and price agreements).

Select one of the following options:

blank = Not used

jcl_loc = Location

jgr_group = Group

job_anal[1] = Analysis Set 1

job_anal[2] = Analysis Set 2

job_anal[3] = Analysis Set 3

job_anal[4] = Analysis Set 4

job_fore = Manager

jty_type = Type
