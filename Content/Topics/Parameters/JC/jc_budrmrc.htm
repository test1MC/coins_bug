JC/BUDRMRC - Approved Re-measure & Rate Change budget type

The code for the memo data (budget) type that holds approved re-measures and rate changes.

This includes the value of any approved which are a re-measure or rate change type.

If JC/BUDRMRC is not blank, the Budgets browsers in Status show a column that allows a drill down to show all approved which are a re-measure or rate change type and have a resource with the selected cost code/WBS code/section/activity.
