JC/JOBANAL - Contract Specific Analysis Set List

A list of cost code analysis set numbers that should be set up specifically for each (as opposed to using the default analysis sets which apply globally across all in the company). This allows you to the costs on different in different ways.

To enable this, create the valid analysis set codes in the WBS table with the Level field set to the analysis set number, then set JOBANAL to a list of the analysis set numbers that are to be -specific.

NOTE

This feature can only be used on that use sections.
