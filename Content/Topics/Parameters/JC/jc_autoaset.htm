JC/AUTOASET - Auto Create Cost Code Analysis Sets

Whether COINS should populate the cost code analysis set table when new cost codes are created (Y or N).

This is intended for housebuilders who have configured COINS to put the plot number at the end of the cost code. One of the cost code analysis sets is linked to the plot number; setting this parameter to Y populates the analysis set table with a list of used plot numbers. This facilitates Dig Deeper by plot.
