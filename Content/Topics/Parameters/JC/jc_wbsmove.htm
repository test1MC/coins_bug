JC/WBSMOVE - Allow WBS transfer

Whether to allow cost and transactions to be transferred between section/activity on a after posting (Y or N).

If this is set to Y, a Allow Cost/ Transfer field in allows you to enable this feature for individual .

On the cost and transaction and Single there is then an option to transfer the transaction to another section/activity. You can select multiple transactions and move them to a different section/activity. The new section/activity must be valid for the and all normal posting rules to allow the transfer apply. Transactions that have been transferred will show an audit button on the to show the movement made.
