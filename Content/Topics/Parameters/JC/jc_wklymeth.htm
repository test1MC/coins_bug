JC/WKLYMETH - Weekly Costs Mode

Which mode the Generate Weekly Costs Snapshot is run in.

Blank = None.

Online = Automatically update summary records during posting.

Simple = All costs against the are accumulated (regardless of date). The weekly cost movement is calculated simply by subtracting the previous cumulative figures from the newly calculated figure.

NOTE

The other option, STANDARD, is no longer supported. If you are using weekly costing, set WKLYMETH to Simple.
