JC/JOBSEC - Contract Security Field

Which field COINS uses to control security.

When determining whether to allow a user access to a , COINS checks the value of the field specified here against the settings for that user in Security Maintenance.

Enter one of the following:

job_num for code

jgr_group for group code

jty_type for type code

jcl_loc for location code

job_fore for manager

rcm_num for

job_anal[1] for analysis set 1

job_anal[2] for analysis set 2 - 4.

job_anal2[1] for analysis set 5, etc.

or leave blank for no security.

NOTE

If you change this parameter, to enable security or change which field governs security after the system has been set up, you need to run Security.
