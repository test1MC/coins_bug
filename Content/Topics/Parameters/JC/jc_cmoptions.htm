JC/CMOPTIONS - List of the Commercial Options which will be available in Contract Maintenance

A comma-separated list of the Commercial Options which will be available in . Blank means that all options will be available.

CMOPTIONS restricts the user to choose only from those Commercial Options which are specified in the parameter value.

Enter a comma-separated list using one or more of these codes:

LITE = Detailed .

DETVO = Detailed and SC .

LITEFULL = - and SC .

VOVAR = - and .

BQVOVAR = - and .

CMFULL = Full functionality.

NOTE

The "Not Used" option with blank code will always be available.

If the COINS module is not licensed, the CMOPTIONS parameter has no effect on Maintenance.
