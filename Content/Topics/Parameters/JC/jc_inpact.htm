JC/INPACT - Create Activity during Cost Code Input

Whether to automatically create the required activity when creating a new cost code for a that uses WBS (Y or N).

Y = In Cost Codes, you can enter or select an existing activity, or enter a new activity (and details such as the description and the classes that are allowed). COINS will automatically create the new activity as long as all the details have been completed.

N = In Cost Codes, you can only enter or select an existing activity.

The parameter also defines the default value for the related box on the Import Budget screens. When the parameter is set to Y, the Create -Specific Activities option will be ed by default.
