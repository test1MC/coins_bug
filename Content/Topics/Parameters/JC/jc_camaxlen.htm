JC/CAMAXLEN - Contract Costing Analysis maximum length (defaults to 19)

( only) This allows you to set the maximum length of the Costing Analysis. This can be between 19 and 30 characters. The default is 19.

This will change the size of the analysis fields on screens and reports, so you should only set it to the maximum length you need.

WARNING

You cannot use this functionality if you are still using . will still impose the 19 character limit.
