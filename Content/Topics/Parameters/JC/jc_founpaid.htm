JC/FOUNPAID - Memo Data: Unpaid invoices over 3 months old

The memo data type used to store the total value of unpaid invoices over three months old. This is used on the Forecasting Workbench, for adjusting the sales to date forecast on those (typically maintenance ) identified by the FORMIFLD parameter.

If you leave this blank, the adjustment screen will not prompt for the unpaid invoice value.
