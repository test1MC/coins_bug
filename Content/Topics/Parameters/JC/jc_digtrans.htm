JC/DIGTRANS - Max Txns on Dig Deeper (250)

A limit on the number of transactions COINS selects in Dig Deeper to improve performance (for example, 250 or 500).

If a Dig Deeper selects more than this number of transactions, COINS issues a warning message and asks the user to confirm that they want to continue with the (for example, only showing the first 250 transactions).
