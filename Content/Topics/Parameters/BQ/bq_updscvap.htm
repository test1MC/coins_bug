BQ/UPDSCVAP - Update VAP budget for the only category via Update VAP budget

Whether all cost categories in BQ should update the subcontract category in the VAP Budget (Y or N).

If set to Y, the materials (and other) values in Bills of Quantity will be added to the subcontract values (for each and payment stage) when updating the VAP budgets from bills of quantity (using the update VAP budget option from within the workbenches). Thus the VAP budget will all be held against the subcontract cost category. This can facilitate -sheet valuation at payment stage level since VAP Budgets only use payment stages for the subcontract cost category.

If set to N or blank, the VAP budget will be updated using the same cost category as in BQ. Thus the VAP budget will be held against separate cost categories for materials and subcontract.
