BQ/SCSTDESC - Split Subcontract Stages by Description

Whether VAP subcontract matrix order stages generated from should be split based on the item description.

Set to N (Default) to combine any items for the same and payment stage into a single matrix row. The description for the standard payment stage will always be used in the subcontract matrix.

Set to Y to create separate rows in the matrix for each description used in the even if the and payment stage are the same. The description from the item will always be used in the subcontract matrix.
