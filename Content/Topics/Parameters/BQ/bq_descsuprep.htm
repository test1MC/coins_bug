BQ/DESCSUPREP - Replace Bill Item Description with the Supplier code and Description

The code and description to use when any of the Apply Prices options are run or the resource is used for a Procurement order.

CODEDESCREP = Replace with 's code and description (if one has been set up on price file entry).

DESCREP = Replace with description only (if one has been set up on price file entry).

NOTREP = Don't replace (the standard resource description is used).

If you leave this parameter blank, the standard resource description is used.
