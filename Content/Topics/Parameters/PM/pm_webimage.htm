PM/WEBIMAGE - Image to use in reports

The name of the image file (for example, a logo) to display on HTML-formatted reports. The image should be one that will display well at 62x58 pixels.

If WEBIMAGE is blank, COINS uses a file called coinslogo.jpg. If WEBIMAGE does not refer to a full address including a server, COINS appends the value of the menu system parameter WEBSERVR to the beginning of the image name.
