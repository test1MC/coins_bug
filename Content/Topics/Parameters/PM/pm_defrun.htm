PM/DEFRUN - Default Run Type

( only) A list of the print methods available in , separated by commas. The first letter in the list specifies the default run type.

A = PDF output

B = Background

D = Delayed

G = GUI Preview

F = Foreground

P = Preview

S = Save

U = Urgent

W = Preview in web browser

If you do not want users to use some of the run types, or if they are not available on your computer or network, omit them from the list.
