PM/AUTODEL - Auto Delete Reports on Login

Do you want COINS to delete users' temporary files automatically?

Y = COINS automatically deletes unwanted temporary files.
(In , when a user logs in, COINS deletes that user's temporary files, subject to the settings of the associated parameters. Print files are only deleted if they have already been printed.)

N = COINS does not automatically delete print files; you should use Purge Print Requests () or the delete option in Report Status () periodically to remove unwanted files from the disk.
