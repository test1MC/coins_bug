PM/PRINTCP - Default Printer Code Page

The default code page used for enhancing COINSplus report output on printers, if there is no code page specified on the printer definition. (You can set different code pages for individual printers using Printer Maintenance.) See also SCREENCP and TERMCP.

Code pages determine which characters are used to display or print individual character codes. For example, COINS provides a code page called iso8859-1-enhanced which prints reports using graphics line-drawing characters, and one called no-graphics which omits line-drawing characters (for printers or terminals which do not support those graphics characters).

The code pages must be defined in the codepage file. See your Progress documentation for more details.
