PM/CPPDFOA - View COINSplus reports from OA as PDF?

If set to Y then reports will be automatically converted to PDF when viewed from the report status in . This replaces the ActiveX Print Preview control.

To enable this functionality FOP needs to be configured and the fop.sh script needs to be in the PATH for the Webspeed/AppServer agent. To do this, add a line like the following to the [Environment] section in ubroker.properties:

PATH=${PATH}:${BASE}/bin
