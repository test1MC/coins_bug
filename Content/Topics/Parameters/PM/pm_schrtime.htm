PM/SCHRTIME - Scheduled reports time restriction

A time restriction for the scheduled reports.

This parameter allows you to stop most users from being able to schedule reports during the specified time period (working hours, for example).

Parameter value should be a valid 24h time interval in the format of HH:MM-HH:MM.

For example:

08:00-17:00

20:00-23:00

00:01-03:15
