PM/GENBACK - Background Generator Command

The command that your operating system uses to start the background report generator.

For example:

$BASE/bin/session_%report.sh -p pm99b -b >$BASE/var/diag/genback.log

If you are using VSI-fax, you also need to include VSI-Fax in the parameter, before the command that calls pm99b:

export VSIFAX=/opt/vsifax/; $BASE/bin/session_%report.sh -p pm99b -b >$BASE/var/diag/genback.log

NOTE

If you are using report servers, GENBACK should be blank; the necessary configuration should be done on the report server.
