HR/HRJTDEF - Fields which default to Personnel from Job Title

A list of fields whose values default to Personnel based on the employee's job title. Choose the fields valid for your region based on the regions listed within the description of each field.

Examples of fields you can select in this parameter are: Employee Trade [ALL], Employee Class [ALL], Work Classification [US, CA, AUS] , Employment Utilation Classification [US, CA, AUS], Employment Utilation Trade [US, CA, AUS], EE0-1 Code [US, AUS], and FLSA Status [US].
