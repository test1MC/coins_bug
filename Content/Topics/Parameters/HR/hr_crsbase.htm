HR/CRSBASE - Course Reference prefix list (MMYYCCCCCCCC)

If you wish to automatically assign a reference to your training courses, this parameter can be used to build a prefix from the date and course code characters, as well as literal characters. This reference is created on saving the record, if the field is blank.

The format of the parameter is a comma separate list. If the entry is not a number it is taken as literal text. A number denotes a character from the string of the month and year of the course and the course code.

Example: If the course HSWAH3 was held in July of 2012, then we are selecting from the string 0712HSWAH3 and the setting 5,6,3,4 would give prefix HS12.

Note if you use the year or month, the course date may change after creation of the course, but this will not be reflected in the reference.
