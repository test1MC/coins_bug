HR/UDLEAVEDATE - Fields on user defined events to auto-populate with personnel leaving dates

A list of user defined events types, along with the field for each type, which should be auto-populated with the personnel leave date when this changes.

Only the latest events record for a person is updated and only if the date was the same as the personnel prior to this latter date being updated. The format is:

user defined type 1:date field 1,user defined type 2:date field 2,...
