HR/SSPREAS - Absence reason code to use for SSP

The absence reason code to use for SSP. This can be a single value for both paid and unpaid absence, or a pair of comma-separated values where the first is for paid absence and the second for unpaid absence.

When entering or updating a period of SSP in Payroll, if absences in HR and PR are linked COINS creates or updates a corresponding absence record in Human Resources, and sets the reason for absence based on this parameter.
