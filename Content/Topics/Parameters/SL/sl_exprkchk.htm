SL/EXPRKCHK - Exposure Risk Check Enabled

Whether to allow the live calculation of customers' credit risk exposure.

This is designed to stop users entering any type of transaction that will cause the customer's credit limit to be exceeded. A live calculation of a customer's credit risk exposure takes place, adding up all outstanding ledger applications and certificates, invoices and -specific data. This will be done for all .

If the value of the outstanding amount exceeds the customer's credit limit (for the current ), an error message will be generated, and the system will not allow the user to continue entering the transaction.
