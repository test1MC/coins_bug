SL/DEBTPCNT - Period Debt As % of Tot on Aged Debt Rpt

(COINSplus only) Whether to show as a percentage of total balance on Aged Debtors.

Y = Aged Debtors includes a row that shows each column as a percentage of the total balance for all customers. This row is immediately below the run totals, and is only included on the main version of the report (that is detail mode for the current period, with Break by GL Level not selected.

N = The percentage row is not shown on the report.
