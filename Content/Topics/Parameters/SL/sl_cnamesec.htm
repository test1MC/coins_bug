SL/CNAMESEC - Customer Name Security

Whether you are using customer name security. This controls which users can change a customer's name in Customer Details Maintenance.

Y = You can run Customer Name Security to set which users have permission to update customer names. You can then set security permissions for a user by entering an asterisk against their name; this allows the user to update customer names. If the security permissions field is blank or anything other than an asterisk, that user will not have security permissions to update any customer names.

N = You cannot run Customer Name Security.
