PT/SPKMAILF - 'From' email address when emailing Submittal Package

This parameter is used to determine the ?From? email address when emailing Package documents.

Options are as follows:

  • From ? use the email address from the ?From? person on the Package.
  • User ? use the email address from the current COINS user.

The ?From? person on the Package can only be used under these conditions:

  • It is linked to a Personnel record, not an external contact
  • Personnel record has an email address
