PT/SPKMAILT - 'To' email address when emailing Submittal Package

This parameter is used to default the "To" email recipients when emailing Package documents.

Options are as follows:

  • From – use email address from the "From" person on the Package.
  • To – use email address from the "To" person on the Package.
  • User – use the email address from the current COINS user.

Multiple options can be selected.
