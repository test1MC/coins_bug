SCIRE/IERCTSSL - RCT SSL Parameters

The value to be used during connection to ROS. Use the entry -sslprotocols TLSv1 -sslciphers AES128-SHA for connecting to the Live environment. For those clients who set up data to utile the ROS test environments the entry should be -sslprotocols TLSv1 -sslciphers DHE-RSA-AES256-SHA .
