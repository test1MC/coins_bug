SCIRE/LINKTAX - Link Companies to calculate tax

A comma-separated list of which are registered as a single Principal with Revenue for RCT purposes.

Enter a list of linked numbers, and separate each group of with a pipe character.

For example:
1,2,3|5,6|7
links 1,2 and 3 together, and 5 and 6 together. 7 is not linked. If a is not listed, it will not be linked. The main in each case is the first in the comma-separated list.

This allows you to link for reporting and submission separately from the CO/LINKCOMP list of linked , which are linked for maintenance. If LINKTAX is blank, COINS uses the list from LINKCOMP.
