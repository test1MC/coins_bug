ST/DIFFACTN - Auto update cost (+YNC/-YNC)

How do you want COINS to update the unit cost on records?

Y = Always update record.

N = Never update record.

C = Update record if the change in price is within the variance specified by the parameter DIFFMAX.

Specify the actions for increases and for decreases, separated by a comma. For example, C,Y means only update the cost if the increase is within the variance limits, and always update the cost if it decreases.
