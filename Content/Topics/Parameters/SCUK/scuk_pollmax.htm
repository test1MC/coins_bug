SCUK/POLLMAX - CIS 2007 Maximum number of polls

The maximum number of times that COINS will re-check for a response via the Government Gateway before timing out.

If COINS has not received a response after this number of attempts, the status of the submission is set to "Timeout" and COINS creates a new delayed request (see POLLNEXT).

To comply with HMRC guidelines on polling, we recommend you set this to 10.
