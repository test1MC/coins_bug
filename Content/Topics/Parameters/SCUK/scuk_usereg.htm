SCUK/USEREG - CIS 2007 Use Scheme Register

Whether to enforce the use of the CIS Scheme Register.

Y = You must use CIS Scheme Register to register the schemes you are going to use and which you are going to use to do the submission for each scheme. This is a stronger validation in that it prevents the same scheme from being submitted more than once. The CIS Scheme Configuration report will validate each and sector and show whether the configuration is "OK" or "Invalid", and the Month End Return process will only produce returns for registered schemes.

N = You do not need to register CIS 2007 schemes. The CIS Scheme Configuration report will show the current configuration.
