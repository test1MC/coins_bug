SCUK/DEFHMRCN - CIS 2007 Default HMRC Reg'd Name From

Where COINS takes the default HMRC Trading Name from when creating a new verification record.

N = Subcontractor Name. The HMRC Trading Name defaults from the Subcontractor Name field.

P = Payee Name. The HMRC Trading Name defaults from the Payee Name field on the subcontractor record.

If there is an existing CIS2007 verification record, COINS takes the name from that.

This does NOT affect Process Transitional Records, which will create the HMRC Trading Name from the name on the old CIS certificate record.
