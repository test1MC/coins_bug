EX/EXEMPSEC - Employee Name Security

( only) Whether COINS applies security to the Employee Name and Initials fields in Expenses Employee Maintenance.

User permissions are maintained via a program called exempsec.p. This needs a function parameter of EXEMPSEC. The program can only be run if the EX/EXEMPSEC parameter is set to 'Y'. When run, a browser shows all users, and user security permissions for the name fields can be set for update permissions by entering * in the last column, or left blank to give non-update permissions.

When Expenses Employee Maintenance is run, the program checks to see if the user has permissions to update the name fields; if the user does not have permissions and tries to amend the name fields then an error message will be displayed.
