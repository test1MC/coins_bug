CI/CENT-ADD - Use Centralised Accounts Addresses?

Do you want to handle the update of accounts addresses (PL, SC and SL) from within the Company Workbench (Y or N)?

Y = You can only update accounts addresses from the Company Information Workbench.

N = You can only update accounts addresses from the account maintenance functions in the ledgers.

See Centraled Addressing.

Also, to use the same address for all an office's accounts, set the CI parameters ADDPLAC, ADDSCAC, and ADDSLIAC to the same address type, for example, ACCOUNTS.
