CI/COUNTRY - Use Country Table (OA only)

Whether to use a drop-down list of countries for the Country field in the Company Information System (Y or N).

To turn this feature on, set COUNTRY to Y. Also, since data held in the Company Information System is not specific to a COINS , you must declare the list of valid countries using Countries in 0. Typically this will be a copy of all possible countries in use for all the COINS .

In this way, when an account is set up in the personal ledgers (PL, SC and SL), the country code needed on the account will be assigned as a default from the address record held in the Company Information System.
