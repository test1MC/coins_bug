CI/AUTOBASE - Account Number Base

The format for the base part of auto-generated account numbers.

Short = The first n characters of the account short name (where n is the value given in AUTOLEN) are used as the base.

Single = No base prefix is used; all account numbers are a single numeric sequence.

The parameters AUTOBASE, AUTOFORM, AUTOLEN, and AUTONUM control the way COINS generates account numbers for , customers, and subcontractors.
