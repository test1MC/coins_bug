CI/SCD-REG - Status Code when Registering

A comma-separated list of Group Status Codes to use when registering company details.

Where more than one entry exists, the following rules apply:

  • The tab labels in the SRM workbench will be taken from the first entry.
  • When cascading-up the status of a supplier/subcontractor, the first entry will be used.
  • The first value will be used when running actions such as changing a company's status.

Set up the possible values for status types using Status Codes (Group Wide).
