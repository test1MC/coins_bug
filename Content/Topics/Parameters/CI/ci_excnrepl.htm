CI/EXCNREPL - Exchange Server contact replication enabled?

Whether the Microsoft Exchange server contact replication feature is enabled.

This feature allows for one-way replication of contacts, in which contacts added or maintained in COINS are copied to a public folder on a Microsoft Exchange server. They can then be used in Outlook.

Part of the information on a contact is the company name and office address. If a company record changes, all contacts for the company will be replicated.

See How to Configure Exchange Contact Replication for more information.
