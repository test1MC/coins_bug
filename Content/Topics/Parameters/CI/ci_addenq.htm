CI/ADDENQ - Enquiry Addr Type (Excpts)

Address Type. If you specify an address type here, COINS uses this as the address type for documents.

Parameters beginning ADD identify the names you have given to different types of address (using Address Type Maintenance). Because you can choose whichever names you want for address types, you must use these parameters to inform COINS which names you have used. The possible values for company types can be found in the Procurement menu, Construction and Civils, Setup, Company Information Systems, Company Types. Company types can be set up in Company Type Maintenance.
