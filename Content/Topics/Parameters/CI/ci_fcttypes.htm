CI/FCTTYPES - OA only - Factor Types

The company type for factors.

When creating a PL/SC account, the list of companies will be those of type SUPTYPES/SUBTYPES and FCTTYPES. When a or subcontractor account is created for a company of type FCTTYPES, the Factor flag is set to Y. (Only the Sysadmin user can update this flag.)

If FCTTYPES is not blank, transactions (invoices and certificates) against factor accounts are prevented.
