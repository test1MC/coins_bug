CI/CITDIRHO - Upload/Download directory: Host

When downloading data to CIT C21, where on the host system do you want COINS to create the download files? This is the default for the "upload/download directory" prompt.

Parameters beginning with CIT control the of the Download COINS to CIT program (ci930). This program is used to export data that relates to companies and offices in the Company Information Workbench, for the purpose of loading into the CIT C21 third-party estimating product database. Download COINS to CIT can be run interactively (in which case the CIT parameters supply default values), or as a background task.
