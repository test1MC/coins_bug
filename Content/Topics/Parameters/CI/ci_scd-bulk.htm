CI/SCD-BULK - Status codes available for the Bulk Status Update

A can-do list (typically a comma-separated list) of Group Status Codes to be consulted by the bulk status update feature.

The bulk status update allows you to update the status of a company in the Company Information Workbench across multiple business units (views).

If the SRM parameter STATVIEW is not blank, the All Views screen within the Company Summary shows a Choose Action menu below the browse. This contains a list of the group status codes listed in SCD-BULK. So, for example, if SCD-BULK was set to SCP, PRF, APP, IL it would contain:
o Set status to: SCP-Supply Chain Partner
o Set status to: PRF-Preferred
o Set status to : APP-Approved
o Set status to: IL-In Liquidation
