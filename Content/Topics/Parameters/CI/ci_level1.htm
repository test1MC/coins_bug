CI/LEVEL1 - 1st level (eg Entire Database)

The description(s) for the 1st level view.

For each of the CI/LEVELn parameters, you can enter a long and short name separated by a comma (for example, Entire Database,Database). COINS displays the long name at the top of the Company Workbench screen, and the short name in the user view pop-up menu.

You can have an optional 3rd entry, which is an abbreviation used on column headings (for example, in SRM Workbench). If the 3rd entry is blank, COINS uses the second entry.
