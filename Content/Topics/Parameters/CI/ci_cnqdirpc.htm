CI/CNQDIRPC - Upload/Download directory: PC

The location on the PC where you want COINS to copy the files when downloading them to ConQuest.

COINS first creates a file in the host directory (specified by CNQDIRHO), then FTP the file to the location on the PC specified by CNQDIRPC. If this is blank, COINS does not copy the file.

Parameters beginning with CNQ control the of the Download COINS to ConQuest program (ci931). This program is used to export data that relates to companies and offices in the Company Information Workbench, for the purpose of loading into the ConQuest third-party estimating product database.

Download COINS to ConQuest can be run interactively (in which case the CNQ... parameters supply default values), or as a background task.
