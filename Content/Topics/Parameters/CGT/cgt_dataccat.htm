CGT/DATACCAT - Data consent category

The data consent category to use to determine if a person has opted in/out of a Communigator campaign. This will be used when creating a campaign in Communigator and when a person unsubscribes in Communigator.

Note this is only applicable when using the REST web service interface.

If this is left blank then the consent category flagged as 'Do Not Email' will be used.
