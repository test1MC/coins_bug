SE/SDHRWDTH - S&D Daily Board Hour Cell Width (pixels)

Enter the pixel width value to be used by the Hour cell on the S&D Daily Scheduling Board.

By default, the S&D Daily Board uses 1350 pixels. To control the width of each hour slot, it is recommended that you divide 1350 by the number of hours you would like to see on the Board simultaneously. The smaller the number, the more hours will be shown. The larger the number, the less hours will be shown and require more horizontal scroll.

  • When this parameter is not set, the default value for the Daily Board column width is 200.
  • There are no restrictions on the minimum/maximum value for this parameter.
