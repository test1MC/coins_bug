SE/SD2LNWKF - S&D Weekly Board Second Line Field

If parameter SE/SD2LNWKB is set to Y (two lines will be used on the S&D Weekly Board), enter the field to display on that second line.

This field should be entered by a COINS consultant during implementation and should be shown in the tooltip or on a screen to ensure the element of data is available in S&D. Database can also be used to provide a field name.
