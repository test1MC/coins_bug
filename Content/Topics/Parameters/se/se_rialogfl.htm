SE/RIALOGFL - Ria log file

The name of the RIA log file.

This can either be a specific file name (including the full path) or a directory.

  • If it is a directory, the logs are saved in the format of 'directory' + rialog.username.YYYYMMDD.log file in that directory.
  • If the parameter is blank, the system uses PM/SPOOLDIR as the 'directory' + rialog.username.YYYMMDD.log as the file name.
