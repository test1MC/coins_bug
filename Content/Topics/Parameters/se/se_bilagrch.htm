SE/BILAGRCH - Calculate Billable Value for Agreement Covered Cost Heads

Enter a comma-separated list of Covered that are to be switched to a billable when generating a billing line. The and Group will be changed. The system will fully discount the line after generating the billable value.

The comma-separated list will use the format of Source |Substitute for each set in the list. The Substitute must be listed below the as billable.

  • Example: ACR,AEX|APM,AEX
  • The value to the left of the pipe (|) should be the actual .
  • The value to the right of the pipe (|) should be the substitute .
