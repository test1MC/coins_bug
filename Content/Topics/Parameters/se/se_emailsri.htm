SE/EMAILSRI - Email Individual Visit Service Report Subject Text

Enter the subject text to be assigned to the system generated individual service report emails.

The subject has substitution capability to include the following values. You must Include the value as shown.

  • <ServiceOrder>
  • <VisitScheduleDate>
  • <VisitDispatchStartDate>
  • <Location>
  • <LocationAddress1>
  • <PO>
