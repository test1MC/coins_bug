PR/WPAYDAY - Weekly pay day

The day of the week that weekly-paid employees are paid.

The parameters WLTDAYS and WPAYDAY determine the default dates for payroll periods.

For example, if period 43 in tax year 1997 has tax period end date 31/1/98 (a Friday), and if employees are paid on a Thursday, the paid date will be 30/1/98. If there is one week lying time, then the week end date () when the work was carried out will be 24/1/98.

NOTE

If you are using two-weekly or four-weekly payrolls, set these up using Pay Frequency Parameters.
