PR/PR-ACCRUAL-LEAVETYPES - List of leave types for which leave is being accrued [AUS]

(Australian payroll only) A list of leave types which are accrued in the system.

This is used by the Leave Liability Reports which show the leave liability of employees accrued till a particular date.

Typical values would normally be HOL,SIC,LSL,RDO (denoting the following types of that are accrued for employees):
HOL - Annual Leave
SIC ? Sick Leave
LSL ? Long Service Leave
RDO ? Roster day off

Or for example if you have more than one leave type that denotes annual leave, you could have values like: HO36, HO38, SI36, SI38, LS36, LS38, RD36, RD38, PUB to denote different annual leaves accrued due to work done on 36 hour weekly contract, 38 hour weekly contract, and so on.
