PR/PP-COST - Pre-payroll costing

This is a list of four values, separated by commas, which determine whether the costing analysis can be entered during Timesheet Entry for each of the four pay frequencies.

= If the employee and pay item are costed, Payroll requires the user to fill in the Analysis field.

= Payroll System prevents users from filling in the Analysis field before payroll calculation has been run.

For example, Y,N,, means that weekly paid employees require costing during Timesheet Entry, and monthly paid employees require costing after Pay Calculations.
