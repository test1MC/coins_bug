PR/PR-ADJUSTTO-EMPBASE - Adjust Hours based on Employee Base Hours (y/n)

Used by the Auto Adjust Timecards function. This parameter determines if the total of hours in a timesheet is adjusted so that it is equal to the employee's base hours.

When set to "yes", "y", or "true", this parameter determines this process is going to occur when the Auto Adjust Timecards function is run.
