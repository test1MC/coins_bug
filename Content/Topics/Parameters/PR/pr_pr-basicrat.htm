PR/PR-BASICRAT - Item Holding Basic Hourly Pay Earn Types

A comma-separated list of items holding basic hourly pay earning types.

Nominate a list of Earnings Categories that represent the ordinary Hourly earnings of an employee. Timecard rates are checked for these earnings categories, and if the rate varies from the employee's base-rate + merit, the employee base rate is updated so that it is always the maximum rate. This is also used in the Overtime Costs Report to flag the timecards which have overtime or double time on them.
