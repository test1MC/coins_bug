PR/STPPROCESS - Single Touch Payroll pending transactions to be warned, errored or auto processed (Auto/Warning/Error)

This parameter will determine whether to error or warn if the single touch payroll has not been sent successfully to ATO before the payroll batch posting

Values can be Warning Error or blank. Blank would default to Warning.

Note: If set to Error, it will prevent the payroll batch posting from going ahead until such time that the payevent has been successfully processed at ATO and acknowledgement of the same has been received by Coins
