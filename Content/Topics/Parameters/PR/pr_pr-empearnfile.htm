PR/PR-EMPEARNFILE - Employee Earnings Export Filename [AUS]

(Australian Payroll only) The filename that the "Export Earnings Categories" will be exported (output) to.

If this parameter is blank, the default filename is EMPEARN.csv.

Export Employee Earnings outputs all the "earning categories" (like ordinary hours worked, shift hours worked etc). This can then be used to change employee earnings with a future date. For example: say employee E1 earnings from next month is to be 10 percent more than current rate. You would use Export Employee Earnings to export all employee earnings, change the rate on them and then use Load Employee Earnings Category Data to load them with the "future ".
