PR/DEFSPLIT - Default Cost Split Method

The default split method for post-payroll costing.

A = Split by amount.

I = Split by input.

P = Split by percentage.

U = Allow the input value to be updated before splitting the cost by input. This allows salaried people to have their cost split by the hours they worked, after they have been paid (for example, if you do not know the hours worked when payment is made).
