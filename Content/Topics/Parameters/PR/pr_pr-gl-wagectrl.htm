PR/PR-GL-WAGECTRL - Labour wage control for payroll

Stores the Payroll Control account. When payroll disbursements are associated with a , this GL account goes on the invoice.

This transfers the liability from the payroll control account to the creditor's control account (for disbursements such as SuperFunds, Taxes, Child Support, etc.).

For US only: For more information on this parameter and auto invoicing, see the PDF How To Guide entitled Use Auto Invoicing in OA Payroll (US Only) found in Payroll How To Guides.
