PR/SLIPCOLS - Payslip (non-std) layout

This parameter allows you to override the layout of the standard (original) payslip. This is a comma-separated list with 1, 2, or 22 entries.

1st entry: the number of columns to shift to the right (normally zero)

2nd entry: the number of rows in the columns section of the payslip.

3rd to 22nd entries: column positions and formats. You only need these if you want to override the default settings.
