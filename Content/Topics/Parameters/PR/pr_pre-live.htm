PR/PRE-LIVE - Is the payroll pre live (ie take on) [ALL]

Set this to Y before going live (for example, during data take-on or testing). Some fields will be enabled to allow data take-on during the pre-live stage when the flag is set to N.

When you are running Payroll live, set this to N.

If this is set to N:

  • When you are setting up employees before going live, COINS prompts you to enter details of previous NI contributions.
  • It will be possible to mark RTI returns directly as Submitted.
