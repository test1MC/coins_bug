PR/PENSBASE - Variable pension bases

This can be used to set up variable pension bases.

If you need to use different values of pensionable pay for different pay schemes, store the pensionable pay in separate pay items.

Enter a pipe-separated list of schemes and items. For example, 3=112|4=114 means for pension scheme 3 use the value of item 122 as the pensionable pay, and for scheme 4 use item 114.

This overrides the pensionable pay item specified in the configuration.
