PR/PR-ADJUSTHOURS-TYPE - Adjust Hours procedure to use this type to calculate total hours worked

Used in Auto Adjust Timecards. This parameter takes one of the following values: Factored-Hours, Factored-Hrs, Regular-Hours, Regular-Hrs, Sum-of-Hours, Sum-of-hrs.

Based on this, the "hours" of timecards in the current period are calculated and compared to the employee base hours, and the difference is adjusted on the timecard.
