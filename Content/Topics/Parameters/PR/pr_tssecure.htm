PR/TSSECURE - Use Timesheet Security Types and Timecard Rate Security Types

Indicate whether Timesheet Entry functions will use the standard Employee Security or the enhanced Timesheet and Timecard Rate security.

  • If this parameter is set to Yes, the security list provided for Timesheet Security and Timecard Rate Security is used in all Timesheet Entry functions and replaces the Employee Security List.
  • If this parameter is set to No, standard Employee Security applies and you only have access to employees listed in the Employee Security List and Code during Timesheet Entry.
