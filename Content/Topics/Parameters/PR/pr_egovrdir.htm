PR/EGOVRDIR - eGovernment Directory

The working directory for XML returns to HMRC.

The files that COINS creates during a submission attempt will be held in a subdirectory of this root directory. COINS creates the subdirectory automatically with the same name as the 'message class', which is HMRC-PAYE-RTI. COINS also copies the submission file to Document Management.

All users who will send the Internet submissions will require permissions to read and write from this directory.

If you are using both and to make Payroll returns this can be set as a comma-separated list of two directories, the first for and the second for (on the report server).
