PR/TIMESHT0 - Auto-update in time sheet entry (Pd. type 0)

The method of update in Timesheet Entry for weekly (Pd. type 0) timesheets. Enter a comma-separated list with Y for each option you want to allow.

a = Auto-update mode. The user steps through the items rather than having to select an item to update. This setting only applies to Coinsplus. In OA this auto-opening of the next record always occurs.

b = The user can update the Input field.

c = The user can update the Rate field.

For example, Y,N,Y enables auto-update mode and allows users to update the Rate field, but not the Input field.
