PR/UPDTSMPA - Update T/S when maintaining assignments?

Whether to update employee timesheets when maintaining pay agreements.

The parameter may be set to Yes, No, or Ask - prompt user when pay agreements are updated. COINS updates timesheets after maintaining the following types of data:

  • Rates of pay
  • Employee assignments
  • Employee trade details (specifically trade and/or grade)
  • Employee item standard values

An employee will be excluded from this update if they are not in an uncalculated state.

Please note that the existing parameter STDUPDTS is used in place of UPDTSMPA when maintaining standard values.
