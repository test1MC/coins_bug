PR/SPLITCOM - Input comments when splitting tsht line

You can add comments to the timesheet costs as you split them in post calculation costing. To switch this on, set SPLITCOM to Y.

Note that the composite field is now available as an option in the composite descriptions, and the parameter COSTDESC allows you to set certain and workshop to display the composite description. So you can comment on payroll costs and have this comment displayed in both and workshop cost .
