PR/P45LTOP - Laser P45 Top Shift (Printer=n)

This allows you to adjust the top margin for specific printers when printing P45s.

Enter a default value followed by a pipe-separated attribute list of printer specific settings. For example: 1|BigLaser=2| Accounts=3.

These give the number of characters to shift the printing on to pre-printed stationery and allows you to adjust to the margins on your printers without having to know about the printer hexadecimal escape sequences.

Positive shift will move the printing down the page. Negative values can be entered, but you are limited to subtracting from the default ? for example if COINS expects the first printing to be on line 5 of a form you can only remove 4.
