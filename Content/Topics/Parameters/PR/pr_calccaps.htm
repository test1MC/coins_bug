PR/CALCCAPS - Captions available in pay item calculations

If you have set up captions for pay item calculation, enter a comma-separated list of captions that can be used.

If a caption is used in this parameter, a pay item calculation variable will be created with the same name as the caption code, and it will be given the value set in the decimal value field in the allowable values list.

You can set up captions in Caption Maintenance using the F7 option. For example, you could set up a caption to hold what kind of healthcare benefit an employee has, then a list of allowable values for the caption and their decimal equivalents (for example, Family, Married, and Single with the values 536.00, 452.00, and 321.00 respectively).
