PR/SLIPTYPE - Payslip type

The format used for the payslip (A, B, C, D, or E). If you want to print the employee's default on the payslip, (types A or B), set SLIPTYPE to A- or B-.

If you are using the configurable payslip, set SLIPTYPE to Config so that the payslip can be viewed using the F7 View Payslip option. You also need to set the parameter CFGLFORM to the name of the form to use when viewing the payslip.
