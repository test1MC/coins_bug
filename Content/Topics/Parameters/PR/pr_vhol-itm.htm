PR/VHOL-ITM - Variable Holiday Item Number

The number of a pay item that will be used to store the factor to multiply fixed items by (if you need to vary the number of weeks over which to average ).

An example of this would be a sickness insurance scheme, where the amount payable is based on weeks worked, however during the number of weeks is based on the number of hours the employee has accrued rather than the physical time off. In this instance the number of periods can be calculated within the items, the result is assigned to the item specified by VHOL ITM, and this value plus one (to account for the normal week being paid) is then used as the multiplier.
