PO/PLALWSEQ - Allow 'Sequential' WBS allocation

( only) Whether to use sequential weekly allocation for that use WBS codes.

You need to specify which are to be run on this basis; the Configuration screen within Details prompts for a field labelled Seq?. During Allocation, the style of input will be varied depending upon the allocation method of the .

If the "Sequential" method of allocation is used, then all seven days of the week are displayed, and all the operator normally has to do is enter the WBS code for the days in question. Note also that subsequent days are set to the WBS code just entered.
