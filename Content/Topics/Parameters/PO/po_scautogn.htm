PO/SCAUTOGN - Auto-generate SO Nums ?

Whether COINS generates subcontract order numbers automatically.

Y = COINS automatically generates subcontract order numbers.

N = Users must enter subcontract order numbers manually.

If both SCALLWMN and SCAUTOGN are set to Y, users may enter their own purchase requisition numbers, but if they leave the field blank, COINS generates a number automatically. Do not set both parameters to N!
