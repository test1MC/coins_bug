PO/UNITWARN - (E)rror or (W)arning (Material)

How COINS responds if a user enters a unit for a material order that is invalid for a commodity (for example during GRN entry or order entry).

E = COINS displays an error message and prevents the user from proceeding until a valid unit is entered. Valid units are selected from a combo and are those set up against the commodity.

W = COINS displays a warning message, but allows the user to continue. You can select any unit from the units table via a lookup.
