PO/POHCODE - OA Only - Use Hold Codes on POs?

Whether to allow users to enter hold codes on material orders, which are then copied onto any invoices for the order.

If this is set to F, C or B, Financial Hold Code and Commercial Hold Code or both fields are available when entering the order. If a hold code is entered on the purchase order, then when an invoice is entered the code is copied to the invoice and the user is prevented from changing it. This would put all invoices on for this order on hold. Invoices can still be approved (if required) using standard approval functions.

If this is set to N, this feature is not available.
