PO/CHROOT - Length of Cost Head 'Root' [1-7]

The length of the initial 'root' part of the (which must remain unchanged).

This allows closer control over cost coding, where the buyer is able to determine the cost code.

The assumption when using this feature is that the standard held against the commodity code is a 'suspense' code (see CHSUS) that must always be overridden during order entry.
