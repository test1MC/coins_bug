PO/CONSTYPE - Consolidate House Type Lines

Do you want to consolidate house type lines on schedule purchase orders?

Y = When printing house builder's material orders built from the plot materials schedule, the PO is printed grouped together for the different house types and the details of the materials are then only printed once for each house type.

For example, if the same commodity is used at different locations, and this is set to Y, the order will show a single line for the total quantity. If this is set to N, there would be separate lines for each location (in which case you should configure the order form to display the location).
