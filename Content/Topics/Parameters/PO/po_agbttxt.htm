PO/AGBTTXT - Default Batch Headers for SC Agree (H/D)

Code to identify the default batch header text for batches of subcontract agreements. To specify different text for the two types of agreement ('under hand' and 'as a deed'), enter two codes separated by commas. For example: HAND, DEED. Text for the two types is set up as an Agreement Letter using Standard Text Maintenance in the Technical Information System.
