PO/POPLFRM - Plant Order Form

The name of the configurable form () or document () to be used for orders.

If you use purchase orders with different first and continuation pages, enter the two form names separated by a comma. The name for the second form must be the first name followed by "2"; for example, if your form name is PLO, the second form must be called PLO2.

If PO/FRMSELEC = "*" or "P", PO/POPLFRM is the default for the Print Form selector when printing an order.
