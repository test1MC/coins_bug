PO/SCVOIMAD - When importing SC VOs: Always create new lines

When importing a to an SOR order, whether to add the imported lines to the existing order.

Y = Imported lines are added to the bottom of the order.

N = The line values of lines on the original order are updated from the corresponding imported lines. This assumes that either:

    • The imported line is marked with a "pseudo-product code" (pol_artcode) in column I of the import file which uniquely matches pol_artcode on the original line. (For pol_artcode to be populated, the original order must have been imported or must come via eCommerce.)
    • The imported line contains the line number of the original line in column M of the import file.
