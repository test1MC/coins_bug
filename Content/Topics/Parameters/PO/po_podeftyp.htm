PO/PODEFTYP - PO Entry - default Order Type

The default order type for all material purchase orders (material, , overhead, etc.) except orders.

The order type you are using needs to be listed against BQ Parameter BQSCTYPE and PO parameter SCSORTYP. Also it must not be against the parameters PO Parameters HBSCORDT & SCNVPMTX and it cannot be any of N,T or R (These order types are normal, templates and reserved).
