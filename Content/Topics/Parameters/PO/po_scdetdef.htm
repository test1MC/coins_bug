PO/SCDETDEF - SC Default Setting for Subcontracts Processed with Detail Transactions

The default (Y/N) for the Detail field on new subcontracts if you are using detailed subcontract processing.

If you intend to do certification using , all subcontracts should be detailed, so we recommend setting SCDETDEF to Y and SCDETOVR to N. Creating a subcontract as 'detailed' does not mean you must restructure it, only that you can do so.
