PO/NTFY-INT - Email Template for PO Notifications to Internal Users

The email template for notifications to internal users in the Send PO Notifications routine.

If this parameter is blank or set to an invalid email template, no internal notifications will be sent.

See $O$Topics/PO/Notifications/Notifications.htm$O$.

Email templates are set up using Email Templates, type PONOTIFY-INT.

PO Internal Notification Rules can be configured in PO Internal Notification Rules.
