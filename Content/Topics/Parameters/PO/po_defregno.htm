PO/DEFREGNO - Default registration for road-going plant

The default registration number used when entering a GRN for a road-going item.

In order to report on exchanges of vehicles for the Department of Transport, you need to record the MID status of commodities and assets. When you raise an order or a GRN for a road-going vehicle, COINS prompts you for the vehicle's registration number. To avoid blank entries, if you do not enter a number on the GRN, COINS asks if you want to use the default registration number entered here.
