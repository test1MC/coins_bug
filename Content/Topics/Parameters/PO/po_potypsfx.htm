PO/POTYPSFX - Suffix PO No with PO Type (list of types)

This allows you to add the order type to order numbers that are created automatically.

* = Add the order type to all order numbers.
For example the normal order: M-1234/0001 (Material Order - Number/Order Number) Would become: M-1234/0001N (Material Order - Number/Order Number+Order Type).

blank = no order numbers will be suffixed by the order type.

comma-separated list of order types = Orders of those types will be suffixed. For example if POTYPSFX = C,B,N only call-off orders, bulk orders and normal orders would be suffixed by their order-type.
If, in the comma-separated list FORCE= is included, it is possible to set a letter to append to the end of orders raised by the Force Allocation method. For example if POTYPSFX = *,FORCE=F then all orders will be suffixed with their order-type, except forced-allocation orders which will be suffixed by "F".
