PO/_XACCTNO - Allow PO and GRN entry by PL account

Whether to allow entry of POs and GRNs by account number.

Entry of orders or GRNs by account number is disabled, as this may cause problems in future versions of COINS. You can enable entry by account number by setting this parameter to Y.

This parameter is password-protected; you will need to run xgivepwd.p to get the daily password for the PO module to update it.
