PO/IMSTYLE - Style of the Invoice Matching screen

The style of the invoice matching screen.

  • Brief = Users can switch between the 'Brief' matching screen and the 'Detailed' one.
  • Full = Invoice Summary shows only the detailed Matching tab.

Brief mode shows only item lines, and credit lines entered against item lines (that is, it does not show additional charges). A limited set of fields are available on the browse; concurrent update allows you to update several lines together.

Other details can be viewed using the button. The button only allows you to add simple credit lines. Other fields, which in Detail mode are available when updating a record or viewing details, are available on column sets in Brief mode.
