PO/SCCMMETH - Commit PO Method (P or M) (Subcontract)

How COINS commits subcontract orders.

M = Committing and printing subcontract orders are two separate processes.
(You can allow specific users to commit orders when they are printed by giving them access to the function %WPO1990FPOHS2.)

P = COINS commits orders automatically when they are printed.

Note that if this parameter is left blank, the existing PO parameter COMMETH will be consulted (even on subcontract orders).
