PO/ALWCHOV7 - Allow override of Cost Head during Asset PO Entry

Do you want to allow users to override the on asset orders (Y or N)?

Where order lines are entered on a separate tab, the derived from the commodity code is shown on order lines, and if ALWCHOV7 is set to Yes, the can be updated. On single form order entry, the field is only shown (and is updateable) if ALWCHOV7 is set.
