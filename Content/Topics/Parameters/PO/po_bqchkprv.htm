PO/BQCHKPRV - OA Only - Check whether resource ordered previously for plot

( only) Whether to allow users to raise material order lines with BQ resources if the resource has already been ordered.

WARNING = COINS will give a warning if an order line is entered for a plot where the resource has already been ordered.

ERROR = COINS will give an error if an order line is entered for a plot where the resource has already been ordered.

OFF = COINS does not check whether the resource has already been ordered.
