PO/TRUSTBNK - Bank Code to be used for LC Trust Receipts

The Bank Code to be used when LC Acceptances marked as Trust Receipts are being posted.

The Bank is used when a CB batch is created to pay linked invoices or certificates. Once this is done related LC acceptances are moved to TR status. When TR Maturity Date of the acceptance is reached then a corresponding inter-bank transaction will be produced from the Bank indicated in the parameter to the Letter of Credit Bank account.
