PO/PADPOFMT - OA Only - Order number format on order pads

The format of order numbers used on order pads.

This must be a digit format. For example, a format of 99999 would allow five-digit numbers from 00001 to 99999.

Order pad numbers are maintained using Order Pad Maintenance. Once pads are in use it would not be advisable to change PADPOFMT (although it is possible to retire pads and to delete pads for which no orders exist).
