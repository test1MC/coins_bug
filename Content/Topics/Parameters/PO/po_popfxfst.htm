PO/POPFXFST - P/O - Place Order Prefix first

Where COINS places the prefix in the order number.

Y = The order prefix is placed before the full order numbered part. For example: HM-1000/0245 . HM- indicates a head office material order. This order number has been generated for 1000.

N = The order prefix is placed after the Base of the order number. For example, 1000/HM-0245.
