PO/BUYERVAL - Buyer code validation

Whether COINS prevents retired buyers from raising or committing orders.

Error = It will not be possible to raise or commit a Purchase Order for a retired buyer; retired buyers are not displayed in the Buyer's drop-down list in PO Entry.

Warning = A warning message that the buyer is retired is shown but it will still be possible to raise and commit orders.

No validation = COINS does not check whether a buyer is retired.
