PO/POSPRICE - Use Supplier Prices on Purchase Orders

Do you want to use prices on material and purchase orders?

N = COINS does not default prices onto order lines.

Y = When the user completes the order header, if prices have been set up for the , COINS prompts Use Prices? If the user selects Yes, COINS defaults the price onto the order lines.

You can update the prices once they are on the order.
