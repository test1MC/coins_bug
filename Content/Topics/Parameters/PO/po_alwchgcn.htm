PO/ALWCHGCN - Allow change of Contract Number on Uncommitted Original PO

Whether to allow the number on an uncommitted order to be changed (Y or N).

Y = The Number field on uncommitted material or orders is enabled for update. This is only allowed if the order number uses a -based sequence (that is, if PO/PONUM1 = "comp").

N = The Number field on uncommitted material or orders is disabled.

Note that you cannot change the number if:

  • The order is a .
  • The order has been raised from a requisition, a project package or from the project resources screen.
  • The order has been raised using the House Builders module.
  • The order is a service-related order.

If the number on the order is changed, all order lines with the original number are changed to use the new .
