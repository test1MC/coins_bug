PO/DLYUNITS - List of weekly rated units

A list of the unit codes that represent daily and weekly rate units. (Units are defined using Units.) For example: DAY,5DY,WK.

COINS recognes four types of rate: hourly, daily, monthly and "cost per ". The parameters DLYUNITS, HRSUNITS, and MLYUNITS specify which codes represent hourly, daily and monthly rates. Any other rate is assumed to be "cost per ".

The valid rates for individual items are specified on the commodity code.

To enable daily (which treats the daily rate as the primary rate and allows you to apply a discount), enter the unit code for days as the first code in the DAYUNITS parameter, and as the second code in the DLYUNITS parameter.
