PO/SCSORCER - Order Types which should use unit price certificate entry

A comma-separated list of the Order Types which should use Unit Price Certificate entry.

Unit price certificate entry allows the data recorded on the subcontract order to be used during subcontract certificate entry.

This parameter defaults to blank.

Any order types listed in this parameter should also be listed in the SCSORTYP parameter; an error message will be displayed if this not the case.
