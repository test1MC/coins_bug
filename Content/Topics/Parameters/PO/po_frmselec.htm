PO/FRMSELEC - A list of Major Order Types that allow Print Form selection

A comma-separated list of Order Types where the Print Form selector is available.

Note: Where this function is enabled for, say, Material orders, the setting of POFRM becomes the default in the selector.

Order Types (asterisk "*" can be used to allow this functionality for any of the Order Types):

M – Material Order

P – Order

W – Workshop Order

A – Order

I – Order

O – Overhead Order

R – Orders

S – Subcontract Orders.
