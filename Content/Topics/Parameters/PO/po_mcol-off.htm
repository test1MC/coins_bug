PO/MCOL-OFF - Multi-Colour POs - Colour Off

For printing multi-part orders, enter the code for the to be used when you want to revert back to normal printing.

This will probably be the default of the printer, that is, black. This must be one of the control sequences set up on the printer definition for the printer that is used to print the Multi-Part Orders.

The codes for the used on multi-part orders are set up using Multi-Part PO Maintenance.
