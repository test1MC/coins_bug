PO/EVLSTPO - List of SC Agr Events for printing on PO (Max 5)

( only) Allows you to print subcontract agreement event information in the header or footer sections of the order form.

Enter a comma-separated list of up to five Agreement Event codes.

On the form in question, you can then select the following fields:

  • l-agevnotes[.] = Agreement Event Notes [1-5]
  • l-agevby[.] = Agreement Event Actioned BY [1-5]
  • l-agevdate[.] = Agreement Event Date [1-5]

Note that the [1-5] value specifies which event from EVLSTPO is to be printed. If the same event exists more than once against a particular subcontract agreement then the latest one in the list will be used.
