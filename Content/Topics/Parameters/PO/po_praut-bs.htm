PO/PRAUT-BS - Auto Base Prefix for Purchase Reqs

Do you want to use the semi-automatic numbering for Purchase Requisitions? If this is set to Y, a requisition number can be manually allocated an integer number, and COINS will automatically add the base prefix (using the PO "parameter"s PRNUM1, PRNUM2 etc.). The integer will be given a format using the parameter PRF-SAUT.

PRAUT-BS and PRAUTOGN must not both be set to Y.
