PO/SUSWBS - Suspense WBSs (if JC COST-WBS not in use default = 1st)

A comma-separated list of breakdown structure suspense codes. If JC/COST-WBS = N COINS uses the first code in the list as a default. If JC/COST-WBS = Y COINS uses the first code where the length of the code is equal to that of the length of WBS codes for the being used. The codes must already exist, and you must set up the same suspense codes on all .
