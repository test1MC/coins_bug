PO/POBUYRST - PO Entry: Restrict Buyer to own code

Whether to prevent buyers from using other buyer codes during Purchase Order Entry and Entry.

Y = A buyer will only be able to use his or her own Buyer Code. This can be overridden for an individual buyer by:

    • (in ) setting the user procedure parameter CANUSEANYBUYER against program po101.
    • (in ) giving a user access to the function %WPO101ZBPOH.

N = Buyers can use any buyer code.
