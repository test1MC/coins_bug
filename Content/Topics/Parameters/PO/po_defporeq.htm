PO/DEFPOREQ - Default PO Required for supplier

When entering a new record, do you normally want to want the to be set up to require Purchase Orders? DEFPOREQ is the default value for the P/O Required field of Maintenance in . You can override this value for individual .

The default for the P/O Required field in the Company Information Workbench is governed by the CI parameter DEFPORCI.

If PL/USECIS is set to Y, DEFPOREQ is ignored and the PO required field defaults in from the respective office record in Central Repository.
