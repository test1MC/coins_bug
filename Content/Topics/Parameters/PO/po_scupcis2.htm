PO/SCUPCIS2 - SC POs update CI Trade lists in detail ?

Should subcontract orders update Central Repository trade lists in detail (Y or N)? This is only relevant if SCUPCIS1 is set to Y.

Y = Trade codes entered on a subcontract order will automatically update the respective trade list as detail trade codes against the subcontractor in the Central Repository. If the relevant header code does not exist, it is also created.

N = Only the header codes, and not the detail codes, will be created.
