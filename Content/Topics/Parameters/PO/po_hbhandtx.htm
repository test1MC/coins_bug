PO/HBHANDTX - House Builders - Handing Text (As Is/Opp)

The text for the handing of plots, entered as a comma-separated list, for example, As,Opposite. This text is added to the order line description.

If HBHANDTX has a value against it, consolidation takes into account the handing: \'as\' and \'opposite\' plots for the same house type are grouped separately.

This only applies if you are consolidating schedule order lines; see CONSLINE.
