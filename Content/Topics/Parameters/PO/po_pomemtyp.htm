PO/POMEMTYP - Memo Plant Order Type

A comma-separated list of order types that represent zero-value memo orders.

These orders serve as a simple "authoration to proceed". They carry no value and so cannot produce a committed cost nor any form of accrued cost. Line details may be entered if required (for example to provide an indication what type of is to be and to provide some default cost coding for the 's invoice) but no quantities, prices, dates etc will be entered.
