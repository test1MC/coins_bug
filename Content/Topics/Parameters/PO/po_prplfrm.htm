PO/PRPLFRM - Plant Requisition Form

(COINSplus only) The name of the form used for printing requisitions.

If you use requisitions with different first and continuation pages, enter the two form names, separated by a comma. The name for the second form must be the first name followed by "2"; for example, if your form name is POA, the second form must be called POA2.

Forms are set up using Form Maintenance in the Technical Information System.
