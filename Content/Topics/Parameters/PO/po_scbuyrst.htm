PO/SCBUYRST - SC PO Entry: Restrict Buyer to own code

Whether to prevent buyers from using other buyer codes during Subcontract Order Entry and Subcontract Entry.

Y = A buyer will only be able to use his or her own Buyer Code. This can be overridden for an individual buyer by setting the user procedure parameter CANUSEANYBUYER against program po101.

N = Buyers can use any buyer code.
