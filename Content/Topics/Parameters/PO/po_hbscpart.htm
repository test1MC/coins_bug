PO/HBSCPART - HB Subcon POs - Part Order Schedules ?

Do you want to allow subcontract schedule orders to be part-ordered?

N = Only the whole schedule can be ordered.

Y = Users can select individual items from the schedule to order. They can still order the whole schedule by selecting all items.

We recommend you set this to Y (even if you intend not to part-order schedules) to allow for exceptions. See also HBPARTPO.
