PO/BQBUYERS - List of BQ users who can order any resource from any supplier

A list of users who can order any resource from any for BQ price orders.

If BQBUYERS is empty then the resource lookup shows all resources for the all users. If this parameter is not empty and the user is not in the list, then the resource lookup only shows resources which have an agreed price and valid date for the selected . If the user is in the BQBUYERS list, then the resource lookup shows all resources.
