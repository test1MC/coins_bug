PO/SCORDTEM - Raise SC POs from a Template Only

Whether to use SC template orders functionality.

If SC template orders functionality is being used:

  • The raising of against orders that have been raised via a template is prevented.
  • It is not possible to copy a Template PO if the Template Order has updateable clauses.
  • It is not possible to select Automatic clauses that are updateable.
  • SC PO Entry in will be prevented. (This is because the SC template orders functionality is based around entering long and complex Template PO's in - these templates and the functionality will not be available in ).
