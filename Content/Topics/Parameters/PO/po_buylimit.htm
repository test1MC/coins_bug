PO/BUYLIMIT - Buyer value limits when raising POs

SIMPLE = Only one maximum order value is available for each order type.

DETAILED = Three different maximum order values (Original Order Limit, Variation Order Limit and Total Variation Limit) are available for each order type.

When using detailed buyer limits, the buyer will only be able to commit an order if the original order value is within their limit; they will only be able to commit a VO if it is both within their limit and also if the total of all previous VOs has not been exceeded.

Buyer limits for each buyer are set using Buyers.
