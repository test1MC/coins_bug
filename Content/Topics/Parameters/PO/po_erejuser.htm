PO/EREJUSER - eOrder Conf <REJECT> button restricted by

Which users have access to the <REJECT> button on the Order Confirmation Workbench.

ADMIN = Restrict the use of the button to Workbench Administrators (PO/ADMUSERS).

SUPER = Restrict to Super-buyers (PO/SUPUSERS).

ADMIN,SUPER = Both (that is, restricted by ADMIN and SUPER).

* or blank = Available to all users.
