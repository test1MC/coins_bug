PO/SENDCLSE - Send open orders clauses electronically

This allows you to control whether COINS sends Open Orders clauses to the if require XML send. Parameter format is value|mode. The parameter value can be re-written by PORD settings <SendCLSE> with the same format as in the parameter.

Value can be set to the following values:

  • A - send all clauses
  • N - no clauses will be sent
  • "Number" - only top "number" of order clauses will be sent

Mode can be set to the following values:

  • 1 - send clauses once – it means that clauses will be send once only when the first XML for the order created
  • * - send clauses each time when order XML is created for the order

E.g. A|1, N|*, 3|*
