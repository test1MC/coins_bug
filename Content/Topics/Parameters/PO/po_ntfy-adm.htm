PO/NTFY-ADM - Email Template for PO Notifications to Purchasing Administrators

The email template for notifications to Purchasing Administrators in Send PO Notifications.

If this parameter is blank or set to an invalid email template, no admin notifications will be sent.

See $O$Topics/PO/Notifications/Notifications.htm$O$.

The email template is set up using Email Templates, type PONOTIFY-ADM.
