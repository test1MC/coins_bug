PO/MTCLRRST - Prevent Clearing a Missing Ticket if there are no documents attached

( only) Whether to prevent missing tickets from being cleared if there are no documents attached (Y or N).

If MTCLRRST is set to Y, the Clear Missing Ticket function is removed from the Missing Ticket Workbench browse screen (to prevent bulk clearing of missing tickets), and is only available as an option on the ticket summary screen. Clear Missing Ticket does not allow the user to proceed if no documents have been attached to the ticket.
