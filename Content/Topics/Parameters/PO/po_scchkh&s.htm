PO/SCCHKH&S Prevent releasing/issuing of subcontract instruction if H&S information box not ticked

Whether to check if the person issuing a Subcontract Instruction has confirmed that Health and Safety information has been received and reviewed.

Y = Error message (you won't be able to issue a SC instruction).

N = No errors or warnings.

The same check also happens when you release a SC instruction into Workflow.
