PO/AUTOCLPL - Close Plant POs automatically

Whether to complete order items automatically when posting a fully-matched invoice (Y or N).

This allows the purchase order to be marked as complete.

The iitem will be set as 'Complete' in the following situations:

  1. The Accrual Type is 'Due Date on Order / Maintenance' and the appropriate Maintenance records are all set as Complete;
  2. The Accrual Type is 'GRN / Maintenance', the item is fully delivered, and the appropriate Maintenance records are all set as Complete;
  3. The Accrual Type is 'GRN / Allocation', the item is fully delivered, the item is no longer and all related sheets are fully invoiced.
