PO/USEMTWB - Use the Missing Ticket Workbench?

Whether you are using Missing Ticket Workbench (Y or N).

Y = When a user enters a missing ticket during invoice matching, COINS creates a record in Missing Ticket Workbench.

Missing Ticket Workbench allows site users to "close" (approve) the missing tickets. This also takes the associated invoice off hold if there are no other problems with the invoice.

N = Invoices that relate to missing tickets will need to be approved using other means (for example, using the Approve Invoice screens) .

See Clearing Missing Tickets.
