PO/GRNLIMIT - GRN Limit for Unordered Goods (per GRN)

The limit on the value of unordered goods that can be entered on a GRN.

You can override the limit for individual users. In , use the parameter GRNLIMX. In COINSplus, set up a User/Procedure parameter against program po103n, for example: "GRNLIMIT=9,999".

You can control which users are allowed to enter unordered goods, by granting access to function %WPO1030BPGLA.
