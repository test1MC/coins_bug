PO/PONOTERD - Check if Red Notes exist in PO Entry

How COINS responds if a user enters a purchase order for a company that has red notes against it.

E = Error. The user cannot enter the purchase order if the company has red notes against it.

N = No Error. COINS does not check for red notes.

W = Warning. COINS displays a warning message if the company has red notes against it, but allows the user to enter the order.
