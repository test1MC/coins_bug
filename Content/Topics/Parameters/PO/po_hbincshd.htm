PO/HBINCSHD - Include schedules in HB purchase order print

Whether to print (or fax) the schedule with the order.

Y = The schedule the order was raised from is printed along with the Order.

N = The order is printed as normal.

NOTE

When faxing, pay attention to the overlay.dat file in the overlays directory (this directory is indicated in the PM parameter TESTDIR). The overlay.dat file determines whether or not an overlay is printed on the schedule part of the fax (which may be undesirable).
