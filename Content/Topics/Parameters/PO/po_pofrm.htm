PO/POFRM - Purchase Order Form

The name of the configurable form () or document () used for printing material purchase orders. This is also the default that is used for other types of order (except and subcontract) if the corresponding parameter is blank.

In both COINSplus and , you must set up a configurable document (configurable form) with a code that matches this value. Use Purchase Order - Forms (COINSplus) or Document Designer ().

If you use purchase orders with different first and continuation pages, enter the two form names, separated by a comma. The name for the second form must be the first name followed by "2"; for example, if your form name is POA, the second form must be called POA2.

If PO/FRMSELEC = "*" or "M", PO/POFRM is the default for the Print Form selector when printing an order.
