PO/USEPAD - OA Only - Order Pad validation of manually entered order numbers

Whether to verify order numbers against order pads. (Material Order Only)

To use this functionality you must allow the manual entry of order numbers (PO parameter POALLWMN = Y), and set USEPAD to either WARNING or ERROR.

Users will then be given either an error or a warning if they use an invalid order number. If they enter a valid order number, its pad and the owner of the pad will be displayed.

You must also set up order pads in Order Pad Maintenance.
