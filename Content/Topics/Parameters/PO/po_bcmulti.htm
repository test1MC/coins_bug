PO/BCMULTI - Allow multiple contracts filter in By Contracts browsers

Controls the of the number filter on the "By " browsers for Material, and Subcontract orders.

If Y then additional named filters are available:

  • All - number filter is ignored and orders for all are displayed.
  • Multiple - number filter can be a list and can contain wildcards.

The Single named filter is always available. If this is selected then the number filter must be a valid number and only orders for this will be displayed.

Note that security will still apply unless the user has access to function code %WPO101WBPOHS1 (Permit PO's across all ). If user has access to this special function then he/she will be able to see orders for any/all .
