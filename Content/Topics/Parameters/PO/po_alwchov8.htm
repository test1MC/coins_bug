PO/ALWCHOV8 - Allow override of Cost Head during Workshop PO Entry

Do you want to allow users to override the on workshop orders (Y or N)?

Where order lines are entered on a separate tab, the derived from the commodity code is shown on order lines, and if ALWCHOV8 is set to Yes, the can be updated. On single form order entry, the field is only shown (and is updateable) if ALWCHOV8 is set.
