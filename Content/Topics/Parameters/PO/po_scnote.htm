PO/SCNOTE - Check if Notes exist in SC PO Entry

How COINS responds if a user enters a subcontract order for a company that has notes against it.

E = Error. The user cannot enter the order if the company has notes against it.

N = No Error. COINS does not check for notes.

W = Warning. COINS displays a warning message if the company has notes against it, but allows the user to enter the order.
