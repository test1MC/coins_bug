PO/ALWMSTK - Allow Missing Tickets in Invoice Matching

(COINSplus only) Are users allowed to enter missing tickets during invoice matching?

N = Users cannot enter missing tickets.

Y = Users can use F9 to enter missing tickets.

You can vary this rule on a user-by-user basis by setting up a User Procedure Parameter for the individual user (or user group). The procedure in question is po104s.p. The parameter(s) column should be set either to ALWMSTK = Y or ALWMSTK = N.
