PO/HBPRLEN - List UOM Price for HB Order Length

The unit or units of measure for commodities that are specified by length.

House-builders material schedules and orders can use commodities (such as timber) that are sold as units of differing lengths. This means you only need to set up one commodity code for each different cross-section of timber; on the schedule, you specify the number of items, and an additional field allows you to enter the length of the item (for example: twelve 4.8m lengths of 50x25 timber @ 135 per 100m).

To do this, set up against HBSULEN the unit of measure that indicates that the commodity is specified by length. Against HBPRLEN, enter the unit of measure used to specify the length. For example, LNG against HBSULEN would indicate that the commodity is specified by length; and M against HBPRLEN would indicate that the commodity is measured in s. If you want to set up more than one such unit (say LNF and FT for commodities measured in feet), enter comma-separated lists in both parameters; the entries must be matching pairs, and both lists must have the same number of entries.

Since timber is normally priced in 100 lengths, you will also need to set up price conversion factors (using Unit Conversion Factors) from s to 100 s.
