PO/STFRM - Stock Order Form

The name of the configurable form () or document () used to print orders.

Forms are set up using Form Maintenance in the Technical Information System.

If this is blank, COINS uses the form specified by the parameter POFRM.

If PO/FRMSELEC = "*" or "I", PO/STFRM is the default for the Print Form selector when printing an order.
