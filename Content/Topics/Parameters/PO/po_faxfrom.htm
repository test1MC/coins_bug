PO/FAXFROM - The fax 'from' number for order fax cover

Which "fax from" number is displayed on the fax header when faxing Purchase Orders.

BUYER = The "fax from" number is taken from the buyer record (that is, the person who raised the purchase order). (It's possible to set up three fax numbers for a buyer; only the first one is used.)

COINS (the default) = The "fax from" field displays the COINS 's fax number (as set up in configuration).
