PO/USECOS - Allow Call Off Orders

Do you want to enter call offs against orders?

Y = Order entry displays a field that allows the order to be set as requiring Call Offs. The default setting for this is dictated by the purchase order types.

N = You cannot use call offs. Order entry does not display the Call Off field.

NOTE

This parameter does not affect "call off orders with no quantity", which are set up using a separate order type.
