PO/SCFRM - S/C Order Form

The name of the form (in ) or configurable document (in used for printing subcontract orders.

If you use subcontract orders with different first and continuation pages, enter the two form names, separated by a comma. The name for the second form must be the first name followed by "2"; for example, if your form name is SCO, the second form must be called SCO2.

If PO/FRMSELEC = "*" or "S", PO/SCFRM is the default for the Print Form selector when printing an order.
