PO/DEFPROFF - Switch off default email in Purchase Order Print

Whether to switch off default email in Purchase Order Print when default Printing Preferences are used for printing orders.

If DEFPRN is used and DEFPROFF is set to Y, the email will only be sent if the Default Print Method is set to EMAIL.

If DEFPRN is used and DEFPROFF is set to N, the email will be generated and sent if the office email is not blank and the Default Print Method is set to blank. Otherwise FAX will be used.
