PO/ALWINVEX - OA Only - Allow add of Extras in Invoice Matching

( only) Whether to allow commodity extras to be entered in invoice matching (Y or N).

Commodity Extras can be maintained using Commodity Extras. A maximum added-value per commodity extra per invoice can be set via this maintenance function.

If the maximum value is breached for a commodity extra added to an invoice then outside-tolerance rules apply. If an extra is added to an invoice for less than the maximum, then within-tolerance rules apply (a greater than expected amount has been invoiced, but the difference is within tolerance).
