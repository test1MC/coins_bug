PO/POLKPLVL - Commodity Code Analysis Set Dig Deeper level

The analysis level used in the commodity code lookup, if using analysis sets to look up commodity codes.

In order to look up commodities using analysis codes, the relevant codes need to be a hierarchy, so that codes at a lower level begin with codes from the next level up. For example, codes at level 1: AA, AB; codes at level 2: AA111, AA112..., AB111, AB112...

The lookup can use all four analysis set levels, or a subset of them; for example, levels 1 and 2, or levels 3 and 4.

blank = The lookup initially displays the descriptions from the highest available commodity analysis set level. The user selects one of these, and the lookup displays the descriptions for all codes at the next level that begin with the selected code. The user then navigates down through analysis set levels until the bottom is reached. After the bottom level is reached, the lookup displays the commodity codes which match the selected analysis set.

S1, S2, S3 or S4 = This specifies a start point. The lookup will initially display analysis descriptions from the specified level. The user then navigates down through analysis set levels until the bottom is reached. After the bottom level is reached, the lookup displays commodity codes which match the selected analysis set.

E1, E2, E3 or E4 = This specifies an end point. The lookup will initially display analysis descriptions from the highest level. The user then navigates through analysis set levels until the specified level is reached. After this specified level is reached, the lookup displays commodity codes which match the selected analysis set.
