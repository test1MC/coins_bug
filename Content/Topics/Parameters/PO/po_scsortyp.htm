PO/SCSORTYP - List of Schedule of Rates Order Types

A comma-separated list of the Subcontract Order Types that are for Schedule of Rates type orders. For example, a code for 'normal' SOR orders and a code for housebuilders subcontract schedule orders. See also HBSCORDT.

If you want to create SOR orders from template orders, include T in this list; you can then enter template orders as SOR-type orders.
