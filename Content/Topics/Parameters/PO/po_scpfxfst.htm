PO/SCPFXFST - S/C - Place Order Prefix first

Where COINS places the prefix in a subcontract order number.

Y = The order prefix is placed before the full order numbered part. For example: S-1000/0245 . This order number has been generated for 1000.

N = The order prefix is placed after the Base of the order number. For example, 1000/S-0245.
