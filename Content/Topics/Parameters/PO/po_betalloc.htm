PO/BETALLOC - Mat Betterment Restrict Budget Allocation

When users are entering purchase orders using Material Betterment, do you want to prevent them from allocating more than the available budget to the order?

E = COINS prevents the user from allocating more than the available budget.

N = COINS allows the user to allocate more than the available budget.

W = COINS issues a warning if the user allocates more than the available budget, but allows the allocation.
