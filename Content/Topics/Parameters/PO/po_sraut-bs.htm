PO/SRAUT-BS - Auto Base Prefix for S/C Requisitions

Do you want to use semi-automatic numbering for Subcontract Requisitions?

If this is set to Y, the user manually enters an integer for the requisition number, and COINS will automatically add the base prefix (using the PO parameter SRNUM1). The integer will be given a format using the parameter SRF-SAUT. (COINS issues a warning if the integer entered is more than 10 above or below the previous number.)

To use semi-automatic numbering, SRALLWMN must be set to Y and SRAUTOGN must be set to N.
