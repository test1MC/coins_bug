PO/STOCKDSC - Stock Item description in Procurement

Whether COINS uses the long or short descriptions in Procurement.

LONG = COINS uses the long description on Items in Procurement (for example, when raising Purchase orders, Requisitions or adding missing tickets in GRN entry).

SHORT = COINS uses the short description.

NOTE

Any commas in the long description will be ommitted when the description is copied over to the PO.
