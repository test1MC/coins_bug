PO/SCTRMDET - Special Payment Terms in detail on SC PO

Whether COINS prompts for payment terms during Subcontract Orders (for cases where a subcontractor working on several different projects has different payment terms on each project).

Due Date = During Subcontract Orders, if the Special Terms field in is set to Y, COINS displays the Terms frame. In this frame it is possible to set Due Date Type and Due Days. These values default to those set in the subcontractor record.

N = Overriding the due date type and days is not permitted.
