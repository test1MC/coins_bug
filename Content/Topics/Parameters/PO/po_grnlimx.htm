PO/GRNLIMX - Extra GRN Limit - Limit By Users (OA only)

( only) The limits (for specific users) on the value of unordered goods that can be entered on a GRN.

Specify the value, and the user IDs of the users to whom that limit applies. For example:
LIMIT=400:jsmith,pjones|LIMIT=300:kbrown

For these users, this overrides the general limit set in GRNLIMIT.
