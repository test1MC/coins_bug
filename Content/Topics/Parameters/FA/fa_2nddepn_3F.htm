FA/2ndDEPN? - Use second depreciation

Are you using two depreciation calculations (Y or N)?

COINS allows you to calculate depreciation in two different ways for each asset. This allows you to calculate the value to record in your company?s books and the value for tax purposes (for example, for capital asset calculations).

The parameter DEF2DATE determines the start date for the second calculation.

The parameter 2ndPOST controls whether the second calculation is posted to the General Ledger.
