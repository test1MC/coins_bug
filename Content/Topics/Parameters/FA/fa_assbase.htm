FA/ASSBASE - Asset Code Base (PPYYMMYYYYGGGGGGGG)

How COINS creates the base part of an asset code. You make up the base by combining digits from the , month or year when the asset is entered, and/or characters from the code for the asset group it belongs to.

The value of this parameter is a list of the positions of the required digits from the following sequence: PPYYMMYYYYGGGGGGGG (where PP = ; YY = period year; MM = month; YYYY = year; G = asset group code).

For example, to produce a base made up of the current year and the first two characters of the group code, ASSBASE would be: 9,10,11,12.
