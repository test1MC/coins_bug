FA/DPNTDOLD - Method To Calculate Depreciation To Date

How asset register depreciations to date are calculated.

This works in conjunction with the FA parameter INCTAKON.

Asset register depreciations to date can be calculated in four ways:

  1. DPNTDOLD = OLD and INCTAKON = Y: initial wdv - current wdv (or if disposed, the last before disposal) + take on depreciation.
  2. DPNTDOLD = OLD and INCTAKON = N: initial wdv - current wdv (or if disposed, the last before disposal).
  3. DPNTDOLD <> OLD and INCTAKON = Y: sum of posted depreciations + take on depreciation.
  4. DPNTDOLD <> OLD and INCTAKON = N: sum of posted depreciations.
