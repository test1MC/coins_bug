FA/ACHGPCHS - Cost Load Plant Cost Heads

This allows you to associate with the different file types when using Fixed Asset Costs Load.

The format for the parameter is a pipe separated attribute list, with the file extension as the attribute and the as the value; the value before the first pipe is the default list. If more than one is supplied they must be in the order internal cost, external cost, internal revenue, external revenue.

For example: PA,PB,PC,PD| mob=P1,P2,P3,P4 would mean that P1-P4 would be used for files with mob extension and PA-PD would be used for anything else.
