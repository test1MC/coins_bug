CB/VATIDCAN - Users able to cancel a VAT Return

A can-do list of users who have permissions to cancel a return.

You can also specify users who are not permitted to cancel a return by prefixing their user ID with an exclamation mark (!).

To globally allow any user who created a return the ability to cancel it, enter 'INPUTUSER' on the list.

If the parameter is blank then only SYSadmin users and Root users have permission to cancel returns.

For example:

  1. JOEBLOGGS,IANMIL - Only these two users can cancel returns.
  2. JOEBLOGGS,IANMIL,SYSADMIN - Only these two users plus SYSADMIN can cancel returns.
  3. INPUTUSER - Any user can cancel a return that they themselves have created but cannot cancel another user's return.
  4. INPUTUSER,!SYSADMIN,!IANMIL - Same as (3) except that SYSADMIN and IANMIL are prevented from cancelling their own returns.
