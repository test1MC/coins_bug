CB/MRKVALEW - Marked Items Value not Equal to Statement Item Value (E-Err/W-Warn/N-None)

Whether to display an error/warning message or no message when selecting a bank statement item for reconciliation that has marked Coins items and the value of the marked items does not equal the value of the statement detail item.

E = Error Message

W = Warning Message

N or Blank = No error or warning message
