CB/STTYDESC - Assign CB/GL transaction descriptions from statement line description when auto-reconciling bank statement type transactions

Whether to use the description from the statement line when auto-reconciling bank statement type transactions (Y or N).

Y = When running the Auto Bank Reconciliation process, the reconciliation transaction description for bank statement type transactions is assigned from the statement line description.

N = When running the Auto Bank Reconciliation process, the reconciliation transaction description for bank statement type transactions is assigned from the default bank statement type description.
