CB/BNKSECSC - Apply Bank Sector Security

Whether to apply GL security permissions to restrict users to banks within their own sectors when entering payments and receipts (Y or N).

Y = GL security permissions are applied to the bank's GL account. If the user's security permission does not include the bank's sector or the bank's GL account, then the user is prevented from using the bank and an error message is displayed.

The GL parameter SECTOR must also be set to Y.
