CB/AUTOCHQR - Auto Generate Cheque References

When entering a receipt or payment in a unique internal reference is generated for each analysis line, if the ledger selected is either PL,SL,SC,CS,EX,HS or LM. I.e. Those ledgers where a record gets created in the source ledger. This functionality can be extended to all ledgers in by switching on parameter AUTOCHQR which means that every analysis transaction will have a unique internal reference.
