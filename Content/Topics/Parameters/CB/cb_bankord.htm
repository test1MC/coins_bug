CB/BANKORD - Order of fields for BACS type 9 output

The format for {eft} type 9 configurable output.

This is a comma-separated list of output fields, denoted by the following:

c = code (for example, code or employee number)

n = name (for example, or employee name)

p = payee name

s =

a = account number

r = building society account number

bs = bank

ba = bank account number

v = amount

d = company name

For example, c,n,p,s,a,r,bs,ba,v,d or c,n,p,s.

NOTE

A format of s,a,n,d,v is compatible with LloydsLink import format "PC Pay CSV Format".
