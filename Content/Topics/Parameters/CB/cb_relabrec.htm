CB/RELABREC - Relaxed Auto Bank Reconciliation

Whether to use "relaxed" bank auto-reconciliation for receipts (Y or N).

N = Only standard auto-matching ( number and value must match) applies.

Y = If a unique match on number and value is not found, auto-matching is also allowed if a unique match is found for date and value. If multiple records are found with the same value on the same day, auto-matching is not performed. This might be used for example to reconcile payments.
