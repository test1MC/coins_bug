CB/MULTDET - Allow Cheque Analysis (multiple lines)

Whether to allow multiple cash analysis lines in Transaction Entry (Y or N).

Y = Allow multiple analysis lines (for example, for petty cash payments to more than one expense account).

N = Do not allow a multi-line breakdown. This means you will only ever enter transactions to a single analysis; you cannot update analysis lines on the Analysis Lines tab. It is not recommended to set this value for the US region, please make sure TAXTYPE parameter is not equal to SALES.
