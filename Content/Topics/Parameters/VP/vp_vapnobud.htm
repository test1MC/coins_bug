VP/VAPNOBUD - Create order element if no budget

When creating a subcontract order, do you want COINS to create order elements for cells that are blank in the budget?

Y = COINS automatically enters a zero value on the order if the corresponding cell in the budget is blank.

N = Blank cells in the budget produce blank cells on the order; users must press F9 to enter a value.

If you are not using budgets, set this to Y.
