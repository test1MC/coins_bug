VP/VAPCAT - Budget Category for VAP

A can-do list of cost categories for VAP (that is, categories against which you want to make VAP payments). This would normally be a subcontract cost category, and may include separate cost categories for dayworks and , for example.

COINS also uses the first cost category in the list for linking subcontract schedules to budgets.

If the VP/PSALLCAT parameter is set to N then only one of the categories included here should also be listed in the VP/BUDCAT parameter. If more than one category is included in both lists, VAP Budget Input will not work correctly unless the VP/PSALLCAT parameter is set to Y..
