VP/MATDESC - Matrix Description fields/format

If no payment stage description is available, what do you want to show as the labels for lines on the subcontract order and subcontract payment matrices?

Y = COINS displays the full description of the .

N= COINS displays the code and payment method (note that if you are using multiple payment stages on a , different lines will have the same label).

Alternatively, you can set VP/MATDESC to a comma-separated list of field names and lengths, for example: jsc_desc,30 or jsc_cc,8,jsc_shdesc,12.

The label may be truncated, depending on the settings of VP/MATCOL and VP/MATFORM.
