VP/USEPOCH - Use PO cost heads for new matrix order

When creating a new matrix order, do you want to automatically use the attached to the related purchase order (Y or N)?

If set to Y, a matrix line will be automatically created for each standard payment stage for each from the purchase order lines.

If set to N, a matrix line will be automatically created for each standard payment stage for the trade on the subcontract header.

In both cases, the values in the matrix will be populated from the VAP Budget.

Note that with either setting, no new lines are added to the matrix automatically when a subcontract order is committed.
