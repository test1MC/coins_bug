VP/VALTOVAP - Default Valuation to VAP Payment

Do you want COINS to copy order values automatically onto cells on the payment certificate based on the valuation (Y or N)?

Y = For cells that have been valued, COINS copies the order value to the equivalent cell on the payment certificate if the payment method on the cell is L, P or C.

N = Users must press Return on a cell to copy the value from the order (this is the default setting).
