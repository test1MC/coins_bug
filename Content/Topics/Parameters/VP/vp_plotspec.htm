VP/PLOTSPEC - {VAP} specifications for sellable plots

Determines the specifications that are only applicable to sellable plots. Only these specifications will be copied to sellable plots.

By setting this parameter only the house type budget values for payments stages flagged as Internal or Both will be copied to the plot budget for sellable plots.

Conversely, only values for payments stages flagged as External or Both will be copied to the plot budget for non-sellable plots.
