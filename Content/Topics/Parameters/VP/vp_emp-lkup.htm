VP/EMP-LKUP - Default VP/PR employee lookup

( only) The lookup level to be used when entering VAP/Payroll employees, if there are no employees listed on the subcontract.

A - All Employees = Make all employees available.

M - Monthly Employees = Only include monthly employees.

S - Select Type = Select the employees to be displayed. This is the default.

W - Weekly Employees = Only include weekly employees.
