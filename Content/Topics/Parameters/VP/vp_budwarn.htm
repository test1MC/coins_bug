VP/BUDWARN - Warn/Error if budget exceeded on order

Should COINS issue a warning if the order amount of the payment stage on a plot is more than the budget? For example, if you are not using budgets, you can disable the warning.

Y = Issue a warning if the payment stage on the plot is more than the budget.

E = Issue an error if the payment stage on the plot is more than the budget.

blank = Don't issue a message.
