VP/METHPAY - Permitted methods of payment for SC orders

A comma-separated list of the methods of payment permitted to be used for payments stages in a subcontract matrix order.

The payment types available are:

1 = Plot analysis set 1 green card.

2 = Plot analysis set 2 green card.

3 = Plot analysis set 3 green card.

4 = Plot analysis set 4 green card.

A = All plots green card.

C = Capped.

D = Detail/Dayworks.

G = Plot group green card.

L = Lump sum.

M = Multiple rate.

N = Deduction (similar to detail/dayworks except only negative values are allowed)

P = Provisional.

S = Single rate.
