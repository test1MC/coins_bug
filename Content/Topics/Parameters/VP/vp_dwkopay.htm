VP/DWKOPAY - Allow overpayment on dayworks stages? (OA)

Whether to cap payments on Payment Method 'D'. This allows you to ensure that variation payments for a plot do not exceed the order value for variations on the plot.

ERROR = If the total cumulative value of variation payments on a plot exceeds the variation order value for the plot, COINS prevents payment.

WARNING = If the total cumulative value of variation payments on a plot exceeds the variation order value for the plot, COINS issues a warning.

See Handling of Variations in VAP Subcontract Orders and Payments.
