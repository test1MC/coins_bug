VP/CASCSTAT - Sales status excluded from budget cascade

When using the 'Cascade' option in plot maintenance, you can exclude selected plots based on the system sales status.

To do this, enter a comma-separated list of statuses which you wish to exclude.

Valid plot system sales statuses are:

  • NEW - Unreleased plots
  • REL - Released plots
  • RES - Reserved plots
  • EXCH - Exchanged plots
  • COMP - Legally completed plots

By default, this is set to COMP to exclude legally completed plots.

This does not affect the cascade option in VAP budget maintenance.
