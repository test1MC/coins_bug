VP/MATPERID - Material element ID for payment card entry

When entering VAP matrix payments via the Create Certificates from Green Cards option, this determines the certificate line which is populated with materials element.

You should enter the itemid used in your SC certificate definitions for the materials line.

On setting this parameter you will also then be prompted to enter the default material percentage on the SC order payment stage details.

Normally only one of the VAP parameters LABPERID and MATPERID should be set.
