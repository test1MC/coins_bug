VP/MATOPTS - Matrix screen options

A comma-separated list of the update options to be made available to users in the subcontract matrix. When this parameter is left blank then by default all options will be available.

The available options are:

R = Enter subcontract rates.

C = Cascade values to a selected house type.

P = Apply a percentage increase to all uncertified cells.

S = Update payments stages.

D = Enter distribution amounts.

M = Revert all matrix values to their certified amount.
