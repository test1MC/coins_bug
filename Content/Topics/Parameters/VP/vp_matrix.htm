VP/MATRIX - Confirm Matrix on Subcontract Addition

When creating a new subcontract, does COINS ask whether it is a matrix subcontract?

Y = COINS asks whether it is a matrix subcontract. Use this setting if you will also be setting up non-matrix subcontracts for the same certificate types.

N = COINS automatically creates a matrix subcontract if the certificate type used is listed in VAL-CERT and the is set up as a . This is the recommended setting.

It is recommended that separate certificate types are used for matrix and non-matrix certificates. This removes the need for users to complete a "Matrix" box on every matrix certificate.
