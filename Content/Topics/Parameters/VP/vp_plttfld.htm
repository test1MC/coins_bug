VP/PLTTFLD - Matrix plot label tooltip fields (OA)

Determines the text to show in the tooltip for the plot column label in matrix style screens. If you leave this blank no tooltip will be shown.

You can use fields from the main plot table. You will need to enclose the field names in {} brackets.

For example:

Address: {vp_wbsdef.RO_vwb_fulladd} Caption: {vp_wbsdef.RW_vwb_caption__2^1}

Would show the plot address and the value from the second caption in caption group 1. Note you must prefix the fields with vp_wbsdef.
