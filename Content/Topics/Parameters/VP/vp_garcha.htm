VP/GARCHA - Character in the plot number that represent a garage plot (OA)

This determines the character in the plot number that represents a garage plot. On setting this parameter the costing for a garage plot on a matrix certificate will then be against the main sellable plot.

Note you must also set the VAP parameter GARPOS if you set this.

The main sellable plot will be determined by striping out the garage character from the plot number for the garage plot.
