VP/PMDEFDIS - Payment methods to default discount down

When creating a matrix order for payment methods specified in this parameter the discount on the order payment stages will default from discount entered onto the order header.

This parameter should be set to a can-do list of matrix payment methods. For payment methods not set in this parameter if the order stage is linked at payment stage level, the discount will default to the payment stage default discount. For order stages linked at level, the order stage discount will be set to zero.
