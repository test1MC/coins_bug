VP/VATFIRST - Create VAT Distribution first time only

Set to N unless you are using mixed certificates (that is, if some costed lines do not come from VAP). If you set both this and VP/VATUPD to Y, you must manually override on all certificates.

This is one approach if you treat retentions as VATable, rather than taking on the amount before retention. For an alternative approach, see VP/VATDIST.
