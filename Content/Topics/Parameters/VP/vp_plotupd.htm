VP/PLOTUPD - Auto Update to Number of Plots Field

In Maintenance, do you want COINS to calculate the number of plots automatically (Y or N)?

Y = COINS displays the number of plots you have set up (excluding non-sellable plots) in the number of plots field; you cannot update this field.

N = You must enter the number of plots on the manually. For example, you might want to indicate the total number of plots planned for the , but not set them all up at first.
