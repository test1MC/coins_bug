SC/CONCPAY - Whether concurrent payment batches should be available

Whether to allow more than one Subcontract Ledger payment batch to be open at the same time in the same .

For example, a standard monthly payment batch can be built up, but weekly payments can be made to some subcontractors without having to wait until month end.

If this is set to Y, when selecting invoices for payment automatically (using Select/Deselect Certificates (Batch)) or entering payment details for individual invoices (using Select/Deselect Certificates (Manual)), you can choose whether to use the default payment batch, use another existing batch, or create a new batch.
