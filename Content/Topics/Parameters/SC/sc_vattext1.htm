SC/VATTEXT1 - VAT text 1

(For Irish Reverse Charge ) The text to print on authenticated receipt subcontract certificates which are fully paid, and on invoice type subcontract certificates. For example: This is not a invoice.

In , use the field RO_sbp_vattext to print the appropriate text on the configurable document.

  • If the certificate has been set to Invoice (I), this will display the text from VATTEXT1.
  • If the certificate has been set to Self Billing (S):
    • If the balance on the certificate is not zero and the CO parameter CURRTAXP has "SC" in the list, it will display the text from VATTEXT2
    • Otherwise it will display the text from VATTEXT3.
  • If the certificate has been set to Authenticated Receipts (A):
    • If the balance on the certificate is not zero and the CO parameter CURRTAXP has "SC" in the list, this will display the text from VATTEXT2.
    • Otherwise it will display the text from VATTEXT1.
