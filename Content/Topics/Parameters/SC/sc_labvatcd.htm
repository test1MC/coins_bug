SC/LABVATCD - VAT code for PL to SC Labour

Whether to calculate on the element of invoices when linked to Subcontract Ledger.

This holds the code that should be applied to the value. This is required for the correct calculation of for the Irish Revenue.

COINS takes into account the Reverse Charge status of the transaction being paid, and does not add to the Net if the transaction type is in the Reverse Charge parameter list RCPLTYPE.
