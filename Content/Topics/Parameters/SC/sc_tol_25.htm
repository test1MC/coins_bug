SC/TOL% - Gross exceeds order tolerance %

The percentage tolerance for the cumulative gross value of certificates.

If the gross cumulative value of a certificate exceeds the order value by less than this percentage, the certificate can still be posted.

Both TOLAMT and TOL% must be set for the feature to work. A certificate will pass Cumulative Gross against Order Value validation if the Gross does not exceed either the order value plus the TOLAMT or the order value plus the TOL%.
