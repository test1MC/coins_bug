SC/BACSOVER - Override BACS media on Subcontractor A/c

Whether you want to be able to use different media formats for individual subcontractors (Y or N).

Y = You can override the default format specified by BACSMED. In Subcontractor Details, press Return on the Payment Method field, then enter the media type in the Media field. When overridden, the resultant output file has the media type APPENDED to the filename.

N = The Media field does not appear.
