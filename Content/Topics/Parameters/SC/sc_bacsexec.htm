SC/BACSEXEC - BACS O/S executable command

An optional operating-system command that is run after COINS creates a file. For example, a command to copy the file to a tape or send it to an output device, such as the port to which a box device is attached as a pseudo-printer.

The following Unix environment variables are available for use in the scripts:

  • $COMPANY - the number of the generating COINS .
  • $BACSFILE - the file name of the output file (as defined by the parameter BACSFILE) or in the case of an overridden media type, the full path name with appended media type.
