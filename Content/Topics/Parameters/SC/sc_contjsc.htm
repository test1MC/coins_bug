SC/CONTJSC - Cost Head Mask to identify Cost Transactions as awaiting Contra

Mask that defines whether to create corresponding Unallocated Cost Transaction automatically which allows you to raise related Contra using this transaction as a source

When any Cost Transaction is being posted (e.g. Invoice) and its analysis matches this parameter then Unallocated Cost Transaction is created automatically.

The parameter is taken into account only when Simple Contra functionality mode is enabled.
