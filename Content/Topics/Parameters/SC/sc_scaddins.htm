SC/SCADDINS - Additional Insurance Details

If you are using the old insurance method (see EXTINS), this controls whether COINS prompts for Contractors All Risk or Professional Indemnity insurance details when you enter Subcontractor Insurance.

To prompt for either or both of these additional insurance details, select the SCADDINS parameter, then select SCIPINDEM and/or SCICRISK from the list. The order you select them in determines the order in which COINS prompts for them.

To remove one of these values from the parameter, highlight the line, press Return, then replace the sequence number (1 or 2) with 0.
