SC/DISTTYPE - Valid Distribution Types

The list of distribution types that can be used for transactions on the subcontract invoice register.

Select one or more of the following options:

C-Costed - Allows transactions to be registered and fully costed. This requires use with a certificate definition which is configured not to post the costs.

R-Register Costs - Allows transactions to be registered with no costing.

If only one option is selected, users do not get a choice; if both are selected, users can choose.
