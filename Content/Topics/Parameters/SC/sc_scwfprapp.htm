SC/SCWFPRAPP - Workflow Template - BACS Result File Approval

The name of the workflow for the approval of payment result files.

Payment Results can automatically payments that are flagged as failures. SCWFPRAPP allows you to specify a workflow that is run to approve the auto- functionality and possibly auto-post the batch.

If this is populated, the Rejected Payments action on Payment Results is replaced by the Release for Approval action, which initiates the workflow.
