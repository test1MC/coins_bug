SC/ISLEOMAN - Isle of Man Jurisdiction

Whether you are operating under the Isle of Man jurisdiction (Y or N).

If set to Y then the Tax and tab in Subcontractor will allow you to enter the subcontractor's date of birth and first name. Also, an IoM submenu then appears on the SC Administration Period End menu. (COINS OA only).

If set to Y then the following parameters should also be set as follows to ensure correct operation of Isle of Man subcontract tax rules:

Set the SCUK parameter CIS2DATE to blank.

Set the SCUK parameter CISDATE to 31/12/2099.

Set the SC parameter STOP714 to N.

Set the SC parameter >EXP714 to Y.

Set the SC Parameter ACCPT714 to Y.

Set the SC Parameter RUN714 to Y.
