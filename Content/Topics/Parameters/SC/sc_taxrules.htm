SC/TAXRULES - Country Tax Rules

Which tax rules are you using? Depending on the setting of this parameter, COINS will apply different regional tax rules parameters.

blank = "old" UK tax rules (before the start of the Construction Industry Scheme).

UK = UK tax rules (current UK tax rules under the Construction Industry Scheme). Set the regional parameters using UK Tax Parameters. The parameter CISDATE on the UK Tax Parameters screen determines whether COINS uses the CIS rules in calculations.

AUS = Australian tax rules. Set the regional parameters using Aus Tax Parameters.

IRE = Irish tax rules. Set the regional parameters using Irish Tax Parameters.

IRE2 = Irish tax rules (after 1-Jan-2012). Set the regional parameters using Irish Tax Parameters.

WHT = Generic withholding tax rules are applied.
