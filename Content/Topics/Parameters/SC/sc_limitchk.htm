SC/LIMITCHK - User limit calculation (item id list)

A list of SC certificate definition items that are used to calculate the value to check against a user's authoration limit when they authore a draft certificate.

This is a list of item IDs, with prefix + or -; when summed together these provide the figure that is checked against for the authoration limit check for users in draft certification. If the limit is exceeded, the user is not able to authore the draft certificate.

If LIMITCHK is blank, the total payment plus PAYE this period values are used as a default.
