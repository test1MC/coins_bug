SC/STOPATR - Stop Payment if ATRs Outstanding

Whether to stop payment if the number or age of outstanding Authenticated Receipts exceeds a set limit (Y or N).

Y = If a subcontractor's outstanding Authenticated Receipts exceed the limits then certificates cannot be selected for payment.

N = If a subcontractor's outstanding Authenticated Receipts exceed the limits then certificates will not be selected using Select/Deselect Certificates (Batch), but can be selected using Select/Deselect Certificates (Manual) or paid using Manual Payment Entry. COINS produces a warning.

In both cases COINS refers to the settings of the SC parameters AGEAUTH and NOAUTH.
