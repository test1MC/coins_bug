SC/CASPAY - Override Payments linked to COINS Construction Cloud

This parameter is only applicable for subcontracts linked to the COINS Construction Cloud. If this parameter is set then users will be able to override payment entry in OA (but must also have set the Day Password).

This is a parameter to allow COINS Construction Cloud users to enter subcontract certificates in OA which have been on boarded. This enables the situation when the COINS Construction Cloud is down.
