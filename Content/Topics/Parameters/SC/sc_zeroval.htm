SC/ZEROVAL - Warn if cert exceeds zero order value

Whether COINS displays a warning if a certificate is entered for a zero-value order.

Y = COINS issues a warning message (or an error) if a certificate is input against a zero value order.

N = No warning message is displayed.

If the Gross line on the certificate definition is set to W (warning) or blank, users get warning messages when certificates are entered, effectively as reminders that the order value has not been entered. This allows you to leave subcontract order values as zero until the order has been confirmed, and still allow entry against those orders.

If the Gross line on the definition is set to E (error), COINS does not allow users to enter certificates for the order.
