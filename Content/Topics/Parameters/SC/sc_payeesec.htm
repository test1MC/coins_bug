SC/PAYEESEC - Payee Name Security

Whether COINS applies security to the Payee Name and Subcontractor Name fields in SC Subcontractor Maintenance.

Y = User permissions are maintained using Payee Name Security. This can only be run if the SC/PAYEESEC parameter is set to Y. When run, a browser shows all users, and user security permissions for the name fields can be set for update permissions by entering * in the last column, or left blank to give non-update permissions.

N = Any user can update the name fields.

When Subcontractor Maintenance is run, the program checks to see if the user has permissions to update the name fields; if the user does not have permissions and tries to amend the name fields, an error message is displayed.

In , as an alternative, you can use Field Access Maintenance; see Access to Fields.
