SC/USECONTR - Use Contra on Certificate Entry

This controls Certificate Entry and the option to pick up contra SC .

Y = Certificate Entry will pick up contra VOs. The Use Contras box will be ticked but not updateable.

N = Certificate Entry will not look at contra VOs and will be used just as a register. The Use Contras box will not be shown.

OVERRIDE = The user has the option to pick up contras on specific certificates. The Use Contras box will be ticked by default but users can update it.

The Use Contras box will only show if the line number on the subcontract certificate definition associated with the order entered is non zero.
