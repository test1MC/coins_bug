SC/WHENPAID - Use Pay When Paid Processing

Whether Pay When Paid functionality is available in this (Y or N).

The parameter has the following values:

Yes = Pay When Paid functionality is enabled for this .

No = Pay When Paid functionality is not enabled for this and is therefore not visible to users.

NOTE

Because paying subcontractors in this way is illegal in some countries (such as the UK), COINS will ask you to confirm before setting this parameter to Y.
