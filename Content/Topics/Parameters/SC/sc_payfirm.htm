SC/PAYFIRM - Allow pay if not firm contract(E/W)

Controls whether users can select certificates for payment if the is not a firm (as recorded in Subcontract Details):

E = COINS displays an error message; users cannot select the certificate.

W = COINS displays a warning message; users can still select the certificate.

blank COINS does not display a message.

This parameter is also checked in the creation of draft certificates. If set, and the setting is E or W, it warns if the subcontract is not firm.
