SC/PAYH+S - Allow pay if no H+S Certificate(E/W)

Controls whether users can select certificates for payment if the subcontractor does not have a Health and Safety Certificate (as recorded on the Unions, CITB, H&S etc screen in Subcontractor Details):

E = COINS displays an error message; users cannot select the certificate.

W = COINS displays a warning message; users can still select the certificate.

blank COINS does not display a message.
