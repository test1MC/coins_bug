SC/INVTXNS - Allowed Document Types

The types of document which can be registered.

Select one or both of the following options:

A-Applications - Allows applications to be registered.

I-Invoices - Allows invoices to be registered.

If only one option is selected, users do not get a choice; if both options are selected, users can use either choice. Note that the handling for applications is different from invoices.

The document type on each transaction is validated against the billing type of the subcontract. Invoices can only be used with billing type I. Applications can only be used with billing types A and S. Subcontract invoice register transactions cannot be processed aganst subcontracts with a blank billing type.
