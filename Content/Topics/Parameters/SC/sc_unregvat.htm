SC/UNREGVAT - Unregistered VAT code

The code for subcontractors who are not registered for .

UNREGVAT is either blank, or contains a zero-rated 'Unregistered' code.

If UNREGVAT is set to non-blank, the following checks are enabled when entering transactions:

  • If the subcontractor is not registered, the code is set to the Unregistered code.
  • If the subcontractor is registered, the code used must not be the Unregistered code.
