SC/VALBTYPE - OA Only - Validate Billing Type against VAT Registration

(COINS OA only) Whether COINS validates the Billing Type for subcontractors that are not registered (Y or N).

Y = The subcontractor's billing type cannot be set to A (Authenticated Receipt) or S (Self Billing) if there is no registration number recorded for them.

N = COINS does not restrict the billing type for non--registered subcontractors.

We consider that it is good practice to have this set to Y, since subcontractors who submit authenticated receipts or with self-billing agreements must be registered for ; however, clients with existing data that is not set up this way may need to set this to N.
