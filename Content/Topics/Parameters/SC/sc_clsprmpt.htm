SC/CLSPRMPT - Prompt to close PO if Cert value > PO value

The default setting for the Close Order when Certificate exceeds Order value box on subcontract certificates.

Y = The box is ed. Users will be prompted to close the order if the gross valuation on a subcontract certificate exceeds the order value.

N = The box is not ed. There will be no prompt (and the order will not be closed). This allows you to not put a value on subcontracts.
