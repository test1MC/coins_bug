SC/SBTOINV - Self Billing Certificates switch to invoice billing type

For subcontracts that are set up as self-billing, whether this only applies to certificates generated from VAP green cards (Y or N). If so, certificates entered using standard certificate entry are treated as invoices.

Y = Self-billing certificates generated from VAP green cards are treated as self-billing; certificates entered in any other way (for example through the Subcontract Ledger) are classified as Invoice.

Set this to Y if you have a mixture of billing types on the same subcontract. This ensures that the correct tax point is applied to the transaction, allows recoverable to be posted correctly, and means the appropriate wording is included on the certificate print.
