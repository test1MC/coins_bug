SC/NOAUTH - Authenticated Receipt Limit

The number of Authenticated Receipts that each subcontractor can have outstanding.

If the number of outstanding receipts exceeds this number, then depending on the setting of the SC parameter STOPATR, COINS either:

  • prevents payments to the subcontractor, or
  • prevents automatic payment selection using Select/Deselect Certificates (Batch), and issues a warning when using Select/Deselect Certificates (Manual) or Manual Payment Entry.

To produce a report on these and other conditions for subcontractors, use Defaulters Report.
