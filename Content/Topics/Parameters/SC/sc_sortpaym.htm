SC/SORTPAYM - Sort order for payments

The sort order for the / remittance print, the payment list, and the payment posting report.

Select the order to use:

    ACCNAME = By subcontractor name.

    ACCNUM = By subcontractor account number.

    CHQAMT = By amount (ascending order).

    CHQAMTD = By amount (descending order).

    PAYREF = By payment reference.

    DEFAULT = By payment reference.
