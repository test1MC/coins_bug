SC/DEFPRN - Use Printing Preferences in Cert/Remit Print

Whether to use the printing preference specified on the subcontractor record when "printing" SC certificates, remittances, and statements of deduction (Y or N).

If DEFPRN is not set to Y, the separate email, print and fax menu options, and the Print Options for remittances, are not available.

The printing preference (EMAIL, FAX or PRINT) is specified using the Default Print Method field in Subcontractor Maintenance.
