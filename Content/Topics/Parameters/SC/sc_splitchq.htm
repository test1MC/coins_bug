SC/SPLITCHQ - Split Cheques into contracts

Whether to split into (Y or N).

Y = Prints separate for every . This is a -wide parameter, which affects all payments to subcontractors.

N = Prints a single for all for the same subcontractor.

SPLITCHQ only applies if the SC parameter PAYTYPE = N; if PAYTYPE = Y, the Payment Type field on the certificate determines whether payments are split.
