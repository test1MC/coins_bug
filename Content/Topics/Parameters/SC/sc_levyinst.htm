SC/LEVYINST - Insurance types that trigger levy

The insurance types that COINS checks to determine whether to apply an insurance levy.

In Certificate Entry, an insurance levy can be calculated when a required insurance does not exist or has expired for the subcontractor.

If you are using processing (parameter EXTINS = Y), the Add Levy? field must be ed against the required types in .

If standard insurance processing is used ( only; parameter EXTINS = N), LEVYINST is a comma-separated list of codes that represent the insurance types to check:

  • SCEMPLIA - Employer's Liability
  • SCICRISK - Contractors All Risk
  • SCIPINDEM - Professional Indemnity
  • SCPUBLIA - Public Liability

If any of the selected insurance types in LEVYINST do not have a policy number set, or have an date set which is prior to the certificate date, then the levy will be calculated. If the date is not set then the processing assumes that the insurance does not expire.
