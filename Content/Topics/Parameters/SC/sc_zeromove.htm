SC/ZEROMOVE - Use ZERO MOVEMENT question for standard Certificate/Draft Certificates

This controls the of the Zero Movement box in Certificate Entry when entering standard certificates. For matrix certificates, the parameter ZEROMOVM is used instead for this purpose.

Enter W if you wish a warning to appear if a Gross value has been entered and the Zero Movement box is ed, or if no value has been entered and the Zero Movement box is not ed.

Enter E if you wish an error to appear instead.

Enter N if you do not wish to see the Zero Movement box or any errors/warnings connected to it.
