SC/EXTINS - Extended Insurance

Whether to run configurable subcontractor validation.

Y = COINS runs the configurable validation, which can be configured to validate more than the standard four insurance types. This is required in .

N = COINS only validates the four standard insurance types (Public Liability, Employer Liability, Professional Indemnity, Contractors All Risk). This is only available in .

NOTE

The configurable validation is not restricted to insurance policies; it can be used to check for renewals or missing details on any number of similar schemes (for example: insurance, pension, superannuation).
