SC/IDVAR - Certificate item ID for variation order lines

The item ID in the certificate definition for variation order lines entered in . This can be the same as SC/IDORIG.

  • If an item ID is entered, breakdown items are automatically created for this certificate line for each item on variation orders.

  • To split the variations over different certificate lines, use the format:

    <ITEMID>| <Sub-ITEMID>,<SC VO Type Mask>,<SC VO Status Mask>| ....repeat of other sub VO data.

    Where <SC VO Type Mask> is a single variation type code, or * for all, and <SC VO Status Mask> is a can-do list of variation status codes.

    An example format is:

    IDVAR=VOITEM| VOSUN,C,SUN| VOUNAPP,*,!APP,*

    This means the item VOITEM if it exists will be populated with the subcontract value for all Approved . The item VOSUN will be populated with all subcontract with a type of C and Status of SUN, and the item VOUNAPP will be populated with all subcontract types with all statuses except APP.

  • If blank, the breakdown items are not automatically generated.
