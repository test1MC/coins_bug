SC/PAYINSPL - Override PAYINS = Warning for PL

Allows you to control whether the PL Automatic Selection routine selects certificates for payment if any of the have expired.

The SC Automatic Selection routine will not select certificates for payment if any of the have expired and the parameter PAYINS is set to Warning. The PAYINSPL parameter allows you to control whether the equivalent routine in also rejects such certificates.

N = The PL Automatic Selection routine will not select certificates for payment if the have expired and the parameter PAYINS is set to Warning.

Y = The PL Automatic Selection routine will select certificates for payment under these conditions.

The manual selection routines in both PL and SC allow you to select certificates when PAYINS is set to Warning.
