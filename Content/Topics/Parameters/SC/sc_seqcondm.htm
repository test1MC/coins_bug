SC/SEQCONDM - Use seq for conditional messages in header

( only) Whether to use conditional message fields as 'normal' fields in the header and footer.

Y = You can specify particular messages in the header or footer using condMessage[n], where n is the sequence number of the message. The order the conditional messages appear in does not depend on the sequence in the message maintenance function.

N = The condMessage[n] format only allocates the lines for the conditional messages; use this if you already have forms configured this way.

See How To Configure and Use Conditional Messages.
