SC/MAXTIERS - Draft Cert - Max number of Levels per user

The maximum number (1,2,3) of draft certificate authoration levels a user is allowed. (Only valid if TIERAUTH = Y).

MAXTIERS is usually set to 1 less than NOTIERS to prevent the same person from being able to control all stages of a draft certificate. A user cannot have level 1 and level 3 security set.
