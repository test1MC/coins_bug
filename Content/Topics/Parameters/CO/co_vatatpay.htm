CO/VATATPAY - VAT Liability at payment by VAT code

Whether to allow users to specify codes for which COINS recalculates liability when the is raised.

If this is set to Y, a At Pay field is enabled on codes, which allows you to mark a code so that COINS recalculates and posts the liability when you raise the rather than using any previously entered value.

If this is set to N, the At Pay field is not shown.
