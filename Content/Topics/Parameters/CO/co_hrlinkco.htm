CO/HRLINKCO - List of Companies to link PR & HR (CAN-DO)

A can-do list of in which the HR/PR link is enabled. For example: !15,* for all except 15. If this is blank, the existing payroll configuration HR Link setting - which is global - will be used.

Leading zeros will be stripped out: "00*" would save as "*", so if you want 1 to 9 either enter longhand or use ".".
