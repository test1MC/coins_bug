CO/FAXTAGS - Enter list of Fax tags

A list of fax tags that COINS can replace with data.

COINS expands each tag with the appropriate value from COINS data and other parameters. This is then appended to the faxing software command. You can use the following tags, but only if using Vsifax Gold (NOTE: enter the tags in the order shown):

  1. tfn - to fax number
  2. fll - file
  3. fnm - from name
  4. fco - from company
  5. ffn - from fax number
  6. tnm - to name
  7. tco - to company
  8. ntf - note file
  9. sub - subject
  10. ovl - overlay
  11. cvr - coverpage name
  12. fls - (trailing) attachment

NOTE

The tags ovl, cvr and fls are essential if using overlays, cover sheets and attachments in VSI-Fax.
