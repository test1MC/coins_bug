CO/DAILYSUM - Type of Daily Summaries to store

Controls if you use Daily Summaries, which are stored values for Cost/ Revenue/ Accrual/ Commitment.

TOTAL = Daily summaries are used.

DETAIL = Daily summaries are used, and records detailing the build up are also created.

The Daily Summaries are generated by scheduled report. The stored value will be used unless it needs to be re-evaluated due to activity since the last update.
