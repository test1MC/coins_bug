CO/VATABLAB - VATable Label

The label for the amount liable to , to print on a configurable document (for example: "able Amount"). This is for use with Irish Reverse Charge .

Where a line from the certificate is specified for printing (in the list of Item IDs) this will be suppressed if the certificate is Reverse Charge. If you use the field vatablab (in ), RO_vatablab (Subcontract Ledger in ) or RO_cst_vatablab ( Ledger in ) for the description of the VATable amount elsewhere in the configurable document, COINS will not print it for a reverse charge certificate. VATLAB does the same but for the label used for the amount.
