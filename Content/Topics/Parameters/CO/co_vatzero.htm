CO/VATZERO - Apply strict VAT rate/amount rules

Whether to apply strict rate/amount rules when entering (Y or N).

If this is set to Y, anywhere where a code is entered with a amount, the amount must be zero if the rate is zero. Conversely if a code has a non-zero rate then the amount cannot be zero. If a normal user breaks these rules then they receive an error (if logged in as SYSAdmin then just a warning will be shown).

These checks are applied to the following transactions: SL Invoice, PL Invoice, SC Certificate, CS Certificate, CB Transaction, GL Journal, SC Invoice Register.
