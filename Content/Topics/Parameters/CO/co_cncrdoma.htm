CO/CNCRDOMA - Concur Domain for user names

The Concur login ID is globally unique and styled like an email address. This parameter specifies the domain part of the ID that will be used as a suffix to the COINS user ID.

If the value of this parameter is set as @myCo.com then the user JOHSMI will get a Concur login of JOHSMI@myCo.com. This login needs to not clash with any other Concur user globally, so the domain part needs to be unique to your company.
