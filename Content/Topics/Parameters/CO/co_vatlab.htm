CO/VATLAB - VAT Label

The wording for amount, to print on a configurable document (for example: " Amount"). This is for use with Irish Reverse Charge .

Where a line from the certificate is specified for printing (in the list of Item IDs) this will be suppressed if the certificate is Reverse Charge. If you use the field vatlab (in ), RO_vatlab (Subcontract Ledger in ) or RO_vatlab ( Ledger in ) for the description of the elsewhere in the configurable document, COINS will not print it for a reverse charge certificate. VATABLAB does the same but for the wording used for the able amount.
