CO/UNESTVAT - Treat Non-Established As VAT Reg

Whether to allow the option to mark a subcontractor as "Not Established" for Irish purposes (Y or N).

If this is set to Y, an Established field will appear on Subcontractor Maintenance.

Under the Irish Reverse Charge rules, certificates from subcontractors not established in the state will still be liable to Reverse Charge . Setting the Established field to No will allow the user to set " Registered" to Yes without having to enter a registration number. This will allow the use of non-zero rate codes.
