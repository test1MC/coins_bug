CO/BATPRMOD - List of modules to process with my open batches

A comma-separated list of modules enabling the user to obtain a list of unposted batches when adding a new batch.

For example, if this parameter contains module PL, and the parameter BATPREXC does not contain batch type PLINV, if a user adds a PL invoice batch and they already have one or more unposted invoice batches, a combo list labelled Open Batches allows them to select a batch to which they can add transactions.
