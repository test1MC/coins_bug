CO/RCPHRASE - Reverse Charge Statement

The phrase required on subcontract invoices and certificate documents which fall under the Reverse Charge rules.

Configure the appropriate documents to use the following field:

  • In : rcphrase
  • In : Ro_Rcphrase (or use the generic field reference method coc_sysparam^co|rcphrase)

This field will not display the text unless the code on the certificate is Reverse Charge.
