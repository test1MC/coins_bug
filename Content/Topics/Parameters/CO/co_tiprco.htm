CO/TIPRCO - COINS Ti PR Company number

When this field is filled in with a company number, it indicates that the user is using COINS Ti Payroll.

If this field is filled in, Maximum Annual Contributions do not apply to the OA HR user using Ti PR. As a result, the OA HR Plan Workbench > Plan Rates > Pay Info Calc screen displays N/A for the Maximum Annual Contribution fields.
