CO/BACSEOL - BACS End Of Line (ASCII List)

A comma-separated list of the ASCII values required at the end of each file line. This only affects media types 4,6,7,8,11,12,14.

For a Carriage-Return (ASCII 13) and Line-Feed (ASCII 10) combination this would be "13,10"; for a standard line feed (the default if nothing is put in this field) it would be "10".
