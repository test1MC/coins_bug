CO/RCTEXT2 - Reverse Charge Text2

Optional text that can be printed on a reverse charge subcontract certificate.

Configure the appropriate documents to use the following field:

  • In : rctext2
  • In : RO_Rctext2 for Subcontract Ledger; RO_cst_rctext2 for Ledger; (or use the generic field reference method coc_sysparam^co|rctext2)

This field will not display the text unless the code on the certificate is Reverse Charge.
