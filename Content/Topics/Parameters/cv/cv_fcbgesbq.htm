CV/FCBGESBQ - ESB Endpoint for background recalculation of forecast items

ESB Endpoint. If not blank then an ESB process queue can be configured to process the dependency recalculations as they occur.

May contain curly bracket expressions which will be replaced using jc_job fields, e.g. jc_job.jgr_group - this allows for processing to be spread across multiple ESB endpoints and brokers

ESB Process Queues can be added to process the messages for the end points. The process queue should run program cvuesb001.processMessage

The ESB Endpoint will only be considered if CV/FCBGCALC set to Y.
