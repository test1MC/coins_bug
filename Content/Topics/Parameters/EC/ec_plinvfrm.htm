EC/PLINVFRM - Document code for imported invoices

The document definition code that should be used to create an image of invoices imported via eCommerce.

When this parameter is populated the import of an invoice will spawn a print job, and the document should be set to store an image against the invoice in Document Manager.

The first entry of the parameter specifies a default form, and you can append alternatives based on the batch transaction type. For Example:

PLINVmat|HII=PLINVhire|HIC=PLINVcredit
