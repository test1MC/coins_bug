EC/ARCHCMD - Archive command

The Unix command used by Archive Messages to archive eCommerce messages.

Archive Messages uses this command to copy the data in the source directories to the archive directory specified in EC/ARCHDIR.

This can include the token $ARCHDIR, which will be expanded to the value of the EC/ARCHDIR parameter.

Use a variation of the UNIX cpio command compatible with your Operating System. This works well on many versions of UNIX: cpio -pumd $ARCHDIR

See How To Archive eCommerce Data.
