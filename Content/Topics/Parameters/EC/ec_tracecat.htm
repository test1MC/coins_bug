EC/TRACECAT - ECatalogue Message Trace File

The file to log conversations with the e, for diagnostic purposes.

If you are experiencing connection problems with the e, specify a log file to record the HTTP conversation between COINS and the e. Specify the log file using the format TERSE or VERBOSE followed by a pipe character followed by the name for the log file; for example: VERBOSE|/coins/var/ECDir/ecatlog.log
