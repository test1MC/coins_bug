EC/SWWBMEMO - Small works Workbench memo

Whether to look for small works order numbers (ending with a WBS code) on electronic invoices.

Y = Electronic Invoice Workbench first breaks down the order number found in the XML <BuyersOrderNumber> tag in the according to the order number format and prefix set in the PO/PONUM1, PO/POORDPFX and PO/POPFXFST parameters.

Having decided what the order number should look like the workbench then looks for another / character. If one is found then the workbench assumes that any characters after this are a small works reference or WBS code. See Order Numbers on the eInvoice Workbench.

N = Electronic Invoice Workbench does not look for WBS codes in order numbers.
