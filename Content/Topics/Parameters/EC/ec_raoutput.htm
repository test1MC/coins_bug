EC/RAOUTPUT - Remittance Advice output defaults

Where to put XML remittance advices.

This is a comma-separated list with four entries, as follows:

  • Entry 1 - Should always be folder
  • Entry 2 - The name of a directory on the database server
  • Entry 3 - The maximum package size. Remittance advices are bundled together to be sent to Hub. This is the maximum number in any one package. 100 is good choice.
  • Entry 4 - How frequently (seconds) the sender background process (ecu002) looks to see if there are new remittance advices to be sent. 3 is a good choice.
