EC/EVISIBLE - Workbench visible Order types

A list of order types you want to be visible on the eCommerce Invoices. Blank means all are visible.

Normally we recommend you leave this blank. If you need to filter the messages (for example if you are running two parallel procurement systems), enter a comma-separated list of pipe-separated pairs where the first part of the pair is either MATERIAL or TRADE, and the second part is an order type. For example: MATERIAL|O,MATERIAL|P.

WARNING

Messages that are filtered out don't go anywhere else.
