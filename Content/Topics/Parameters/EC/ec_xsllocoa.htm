EC/XSLLOCOA - Stylesheet location in OA

The location of XSLT stylesheets to enable XML documents to be viewed from the workbenches. Should be set to a relative (/coins/xsl) or absolute (http://coinsserver/coins/xsl) web path.

If this parameter is empty, then the value of /coins/$WEBBASE/xsl is used, where $WEBBASE is the value of the SY/WEBBASE parameter.
