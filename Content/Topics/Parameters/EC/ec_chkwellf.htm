EC/CHKWELLF LOAD checks for well-formed XML

Whether checks that incoming messages use well-formed XML (Y or N).

Y = If a document fails the check for well-formed XML, moves it to the $ECDIR/REJECT directory. The System Administrator should monitor this directory.

N = The check is bypassed, which may save a few milliseconds per transaction. Only set this to N if you are confident that you will only receive well-formed XML messages.
