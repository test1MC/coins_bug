PI/PLNWFMA - Workflow Template for Plan Dates Approval - Materials

The workflow template for approving material plan dates.

This is only relevant if the PI parameter USEPLNMA is set to Y.

If PLNWFMA is populated with a valid workflow template, the Send Package Dates for Approval action is available on the Key Dates browse.

If PLNWFMA is blank, the Approve and Freeze Plan Dates action is available to freeze material package dates manually.
