PI/PLNWFPL - Workflow Template for Plan Dates Approval - Plant

The workflow template for approving plan dates.

This is only relevant if the PI parameter USEPLNPL is set to Y.

If PLNWFPL is populated with a valid workflow template, the Send Package Dates for Approval action is available on the Key Dates browse.

If PLNWFPL is blank, the Approve and Freeze Plan Dates action is available to freeze package dates manually.
