PI/TLAWFSC - Workflow Template for Tender List Approval - Subcontract

The workflow template for approving subcontract lists.

This is only relevant if the PI parameter USETLASC is set to Y.

If TLAWFSC is populated with a valid workflow template, the Send List for Approval action is available on the List browse.

If TLAWFSC is blank, the Approve and Freeze List action is available to freeze the subcontract list manually.
