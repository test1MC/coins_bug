PI/TLAWFPL - Workflow Template for Tender List Approval - Plant

The workflow template for approving lists.

This is only relevant if the PI parameter USETLAPL is set to Y.

If TLAWFPL is populated with a valid workflow template, the Send List for Approval action is available on the List browse.

If TLAWFPL is blank, the Approve and Freeze List action is available to freeze the list manually.
