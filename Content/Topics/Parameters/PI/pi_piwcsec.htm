PI/PIWCSEC - Apply Contract Security settings to PIW?

Whether security settings are reflected in the Project Information Workbench (Y or N).

This means that if a user doesn't have access to a particular contract, he or she won't have access to the corresponding project in the PIW either. Note that the contract security settings will only influence those projects with a corresponding contract. Other projects (for example and Leads) will be unaffected.
