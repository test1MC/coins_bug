PI/MULT-ENQ - Using Multiple Enquiry Documents?

This allows you to have more than one document against a package. (Each document will be sourced and priced separately).

Within the document entry program, you have an option to create additional . If you have this parameter set to Y and have more than one per package, you are given an option to change your Working document from within the package's event list (press F6 to change your Working document).
