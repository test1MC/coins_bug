PI/TLAWFMA - Workflow Template for Tender List Approval - Materials

The workflow template for approving material lists.

This is only relevant if the PI parameter USETLAMA is set to Y.

If TLAWFMA is populated with a valid workflow template, the Send List for Approval action is available on the List browse.

If TLAWFMA is blank, the Approve and Freeze List action is available to freeze the material list manually.
