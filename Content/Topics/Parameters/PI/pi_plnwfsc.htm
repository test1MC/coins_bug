PI/PLNWFSC - Workflow Template for Plan Dates Approval - Subcontract

The workflow template for approving subcontract plan dates.

This is only relevant if the PI parameter USEPLNSC is set to Y.

If PLNWFSC is populated with a valid workflow template, the Send Package Dates for Approval action is available on the Key Dates browse.

If PLNWFSC is blank, the Approve and Freeze Plan Dates action is available to freeze subcontract package dates manually.
