PI/POPCKSC - Package Code to be created by Subcontract Orders

The subcontract package code to use when raising subcontract orders from the main menu (rather than from within a package).

If you to intend to allow subcontract orders to be raised from the main menu (rather than from within a package) and you would like a package to be automatically created, enter a subcontract package code against this parameter. You would typically do this for small value orders where your workflow will carry out the relevant validation.
