VTUK/EGOVRDIR - VAT Return XML root directory

The base directory for XML submissions to the Government Gateway.

For example, $BASE/ var/ egovxml.

The ASCII text files that COINS creates during a submission attempt will be held in a subdirectory of this root directory. The subdirectory name is the same as the 'message class', and is created automatically. These are:

  • HMRC-VAT-DEC - Live Submissions
  • HMRC-VAT-DEC-TIL - Test in Live Submissions

All users who will send the Internet submissions will require permissions to read and write from this directory.

If you have more than one submission , you may want to create separate subdirectories for each submission , to make it easier to locate the XML for each . For example: BASE/ var/ egovxml/ company16

NOTE

There is an equivalent parameter in Payroll and Subcontract Ledger. You may want to configure all these parameters to point to the same root directory.
