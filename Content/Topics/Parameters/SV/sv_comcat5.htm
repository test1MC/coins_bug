SV/COMCAT5 - Commodity Code for 5th cost category

Commodity Code to be used for the fifth cost category.

The format for this should be the tip type (for example, MATERIAL, TRADE, , or ) followed by a comma and then the detail commodity code (for example, ,1902)

This is used when not using resource details on a type. A simple resource breakdown is generated using the six parameters of CMCAT[1-6] depending on how many cost categories are being used. This is required so that budgets can be generated and items can be valued without having to worry about the resource details.
