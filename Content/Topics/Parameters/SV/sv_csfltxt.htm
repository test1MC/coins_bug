SV/CSFLTXT - Final Certificate Text

The title that can be used on a CS certificate OA form when the status of the certificate is submitted within .

This is controlled via the RO field that needs to be included on the OA form. This field is called Ro_cert_text^CERT. This uses the text from this parameter if the certificate is submitted; otherwise, it uses the parameter CSDRTXT that is text for a draft certificate, which is a certificate that has not been submitted yet.
