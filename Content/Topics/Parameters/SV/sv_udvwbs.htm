SV/UDVWBS - Totals - Pricing Method variation WBS codes

A comma-separated list of six WBS codes to use when the pricing method is Totals , which can be the same WBS code or six different WBS codes.

Details in the WBS code format for cost category 1 to 6, separated by a comma, such as A1P,A1P,A1P,A1P,A1P,A1P.
