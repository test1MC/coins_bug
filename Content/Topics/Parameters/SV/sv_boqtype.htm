SV/BOQTYPE - Global BOQ Group for auto-creating Contract BOQ Types

Enter the Global Group for auto-creating Types if not copied from a template .

When a new is created, types are created automatically using the following precedence:

  • Template if specified on the
  • Global group if specified on the
  • Global group as specified in this parameter

If a is selected as the current working and no records have yet been created, then types are copied from the group specified in this parameter.
