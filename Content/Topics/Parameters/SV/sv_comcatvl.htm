SV/COMCATVL - Allow same commodity types and codes

Whether to allow a simplified set up using the same commodity types and codes.

Y = The parameters for COMCAT[1-6] can have the same type and codes to simplify the setup, if you are not concerned about the cost category breakdown.

N = Each of the six parameters (COMCAT[1-6]) must be unique, that is, they must relate to the cost category types defined in Status.
