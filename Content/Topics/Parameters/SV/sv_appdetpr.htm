SV/APPDETPR - Detailed Application Report function

The report function to run to produce the application detail.

If SV/APPDETPR is populated, the Choose Action menu on the Applications screen includes an Application Report option which runs a report showing the application detail. There is a standard COINS report, %WSV3030RSVD, but you can create a different report and enter its report code as the value of SV/APPDETPR.

If SV/APPDETPR is blank, the Application Report option is not available.
