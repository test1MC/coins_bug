SV/TUPWVAL - Shows what value are we valuing against

The value being valued against. This determines the value shown in the browse and what we should apply percentages against.

The options for this parameter are:

  • TUP -
  • ITEM - Full value

For example, if a item is for $2000 and has a of $1000 (50%), $1000 would show in the browse if TUP is selected. Valuing this item at 75% would be a value of $750. $2000 would show in the browse if ITEM is selected. Valuing this item at 75% would be a value of $1500.
