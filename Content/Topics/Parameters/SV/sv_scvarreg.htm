SV/SCVARREG - Label for Dynamic Folder in Subcontract Variations Maintenance

A list of titles for up to five user-defined tabs that COINS displays on the Subcontract screen.

The user-defined tabs allow you to record and report on additional information when entering subcontract .

Use Subcontract to set up the additional fields you want to display, and which of these tabs you want each field to appear on.

NOTE

COINS only displays a tab if you have set up at least one field on it.
