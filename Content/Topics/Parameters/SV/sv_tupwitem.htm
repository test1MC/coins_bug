SV/TUPWITEM - Shows what Items can be valued against

The items that can be valued against. This always includes the original .

NOTE

If any setting other than APPR is used, and once values have been entered into a Client Application against a , that should be prevented from being deleted.

The options for this parameter are:

  • APPR - Approved only.
  • UAPPR - Approved and unapproved (excluding rejected ); this is the default.
  • ALL - All items (including rejected). This would not be required often - probably only to fix a where the user has valued an item and the has since been rejected.
