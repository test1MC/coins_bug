SV/CRTDETPR - Detailed Certificate Report function

The report function to run to produce the certificate detail.

If SV/CRTDETPR is populated, the Choose Action menu on the Certifications screen includes a Certification Report option which runs a report showing the certificate detail. There is a standard COINS report, %WSV3020RSVD, but you can create a different report and enter its report code as the value of SV/CRTDETPR.

If SV/CRTDETPR is blank, the Certification Report option is not available.
