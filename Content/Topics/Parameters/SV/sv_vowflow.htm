SV/VOWFLOW - Use workflow for Variation Approval/Rejections

Whether to use workflows for approval and rejection (Y or N).

If set to Y it will not be possible to manually change the status of to approved or rejected. The following actions will appear on the workbench: "Release for Approval" and "Release for Rejection". These will use the workflow template from the VOWFBDTF, VOWFCLI, VOWFINT, VOWFRTC, VOWFREM and VOWFUSR parameters with a function parameter of "APPROVE" or "REJECT"
