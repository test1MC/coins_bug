SV/SCVWFLOW - Use workflow for Subcontract Variations

Whether approving or rejecting subcontract can be controlled by workflow (Y or N).

If set to Y then it is not possible to manually change the status of to approved or rejected and the following actions will appear on SC workbenches: "Release SC for Approval" and "Release SC for Rejection". These will use the workflow template from the SCVWFCON, SCVWFRTC, SCVWFREM or SCVWFSTD parameters with a function parameter of "APPROVE" or "REJECT".
