SV/VALROUND - The rounding factor to use when entering valuations in the browser

The rounding factor to use on .

For example:

  • If blank, no rounding is done.
  • If you enter 1, the value rounds to the nearest pound or dollar.
  • If you enter 10, the value rounds to the nearest 10 pounds or dollars.
  • If you enter 0.01, the value rounds to 2 decimal places; if you enter 0.1, the value rounds to 1 decimal place, etc.
