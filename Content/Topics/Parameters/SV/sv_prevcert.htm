SV/PREVCERT - Certificate Posting method when using Applications/Certificates

Whether to use combined certificate posting (Y or N).

If PREVCERT is set to Y, as part of the certificate posting routine each line of the certificate will be checked against the application that is linked to the certificate. This will include the original , lines and any material on or off site entries.

If any of the lines of the certificate do not match the linked application, the certificate will be converted to a combined application / certificate then application element of the certificate will be updated to match the values on the certificate prior to the certificate being posted.
