SV/VOTREE - Display the BOQ Treeview when entering a Variation

Whether to display the tree when entering a (Y or N). The default is YES.

Y = The tree is displayed. This allows you to place at different points within the structure.

N = The tree is not displayed. The tree may be unnecessary if you are putting all in a single section.

SV Parameters VOQUICK and VOTREE don't work together. If VOTREE is turned on then is still shown even if VOQUICK is turned on
