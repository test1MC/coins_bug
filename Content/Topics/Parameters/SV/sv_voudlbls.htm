SV/VOUDLBLS - Labels for layout and the dynamic tabs which display user defined fields on Variations Register

Parameter VOUDLBLS can have maximum of 6 comma separated entries. First entry is the label for the layout which groups all the user defined fields which have the Tab as “Main Details”. This layout is displayed on the Details screen. The next five entries are the labels for dynamic tabs which are displayed on .

Use to setup user defined fields for . User has the ability to configure user defined fields on 5 different Tabs. Those tabs are configured through the parameter VOUDLBLS.

Note that the dynamic tab is only displayed on the if at least one user defined field has been setup with that tab.
