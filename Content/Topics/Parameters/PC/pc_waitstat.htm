PC/WAITSTAT - Status code of assets waining for decision

The status code of assets waiting for decision.

This status is used when the person doing first line assessment is unsure and refers the decision (whether to make the asset available or create a ) to someone else. If this is populated, a third action – Awaiting Decision – becomes available on the Maintenance Desk - Assets tab, to allow you to set the status of assets that are to be inspected after they are returned.
