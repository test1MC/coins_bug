PC/CHGBREAK - Charge Break (YN@YN@YN)

This determines the default invoice breaks; in other words, whether charges for different charge , site addresses, or dispatches appear on the same or different invoices.

This is a list of three Y/N choices corresponding to the three sorting criteria specified in CHGSORT. These must be Ys first, then Ns, or all Ns, or all Ys.

For example, if CHGSORT is C,L,D and CHGBREAK is Y,Y,N, the charging run will generate a separate invoice for each charge and site address, but not for each dispatch.

You can define different settings for individual customers or using Invoice Breaks.
