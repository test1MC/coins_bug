PC/UNCRTCHG - Include Unconfirmed Returns In Charge Run

Whether to include unconfirmed returns in the charge run (Y or N).

If this is set to Y, unconfirmed returns are included in the charge run. The items are still seen as until the return has been confirmed, but the charges will not generate while the return stays unconfirmed. This applies only for items charged based on the days , weekly or daily.

This can reduce the number of credit notes due to charging beyond the unconfirmed return date of an asset, item or , and also means that the (s)/ customer(s) will have up to date, correct costs.
