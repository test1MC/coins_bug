PC/EXPREFIX - Exchange Ref Prefix (DDDDDDDD)

The prefix for exchange return reference numbers. This allows you to differentiate between normal returns and exchange returns.

The prefix can be made up of characters from the depot ( location) code, using numbers to refer to the positions in the code. For example, 1,2,/ will give a prefix made up of the first two characters of the depot code, followed by a slash.
