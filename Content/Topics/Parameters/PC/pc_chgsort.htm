PC/CHGSORT - Charge Sort (DCLTSHO@DCLTSHO@DCLTSHO)

When processing a charging run, how do you want to sort the dispatches?

This is a list of three letters to indicate which order the dispatches are sorted in:

C = by charge

D = by dispatch number

L = by location (address code)

T = by dispatch type

S = by Day Hire Ref

H = by Hire Contract number

O = by Customer Order number

So for example, C,L,D will sort dispatches first by charge , then by address code, then by number.
