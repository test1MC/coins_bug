PC/HDFIELDS - Choose which fields to show on the Hire Desk

Controls which fields are displayed on the tabs.

The following options are available.

ALL = Show Customer, Charge and /Phase fields;

ALLDESC = Show ALL fields plus the description field;

ALLTOOLTIP = Show ALL field plus the description as a tooltip;

CODE = Show the client type (/Customer) and code fields;

CODEDESC = Show the client type, code and description fields;

CODETOOLTIP = This setting will show the code field and the tooltip will include the client type and description. The type field will only be displayed when adding and/or updating.

DESC = Shows only the client description field;
