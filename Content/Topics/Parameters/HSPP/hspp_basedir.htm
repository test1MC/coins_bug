HSPP/BASEDIR - Base directory for files

The base directory on the COINS server that the brokers will use to store new messages that are downloaded from a property portal. By default they will use $BASE/var.

  • Rightmove files are stored in a subdirectory Rightmove.
  • Zoopla files are stored in a subdirectory Zoopla.
  • Smart New Homes files are stored in a subdirectory SmartNewHomes.

Each portal directory (for example $BASE/var/Rightmove) also has an ftp and old directory. The ftp directory temporarily stores the files downloaded from the portal's FTP site. The old directory is an archive for files processed into COINS; once processed they are removed from the ftp directory and moved to the old directory. The portal directory also contains a log file (for example Rightmove.log) and a lock file (for example Rightmove.lk). Removing the lock file will allow the COINS portal process to be terminated, for example to allow for overnight backup.
