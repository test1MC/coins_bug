Defining the Content of the Document

Manual Definition

The information that is displayed on the document can be either the value of a database field or fixed text.

In the Fields tab, add a separate field record for each database field and each bit of text on the document.

Anything that is to be printed on the same line must have the same Row number. The Columns indicate the order of items in the row. Different rows can have a different number of columns, and the columns can be of different widths.

The Text field prints text on the form. Database fields can be embedded within the Text box by adding { } either side of the field.

The Field field allows you to reference fields in the database or add the path of an image, see the Images of Configurable Documents section of this document.

The Text and Field boxes must be used exclusively

Use the Include Form field to reference other forms in the document, see the Nesting Forms section of this document.

The Width field determines the width of the field. The minimum Height of a row is determined by the height of the first item on the row (other heights are ignored).
