Installing OpenSSL on the server

OpenSSL is an open source package that provides a set of utilities for administering certificate files. In this instance OpenSSL is used to:

  1. Convert certificates to a format usable by ; and

  2. Identify certificate files via their unique hash key.

Perform the following steps

  1. Log into the server using the root account login details.

  2. Determine whether or not the OpenSSL package is already installed on the server:

    Using RHEL

    If OpenSSL is not installed then proceed to step 3. If it is already installed you can proceed to Installing Certificates for COINS

  3. Install the OpenSSL libraries.
    From the command prompt issue the command: yum install openssl

    The following will be displayed:

    Choose y and press enter to continue.

    If no errors are encountered you will see the line: Complete!
