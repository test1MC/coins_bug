Configure LDAP-Task Summary

The tasks required to enable LDAP Directory authentication from are as follows:

  1. Confirming the Active Directory server resolution

  2. Preparing the server

  3. Configuring user accounts

  4. Implementing Secure Sockets Layer - optional

The tasks must be performed in the order outlined above.

Each task is broken down into a series of steps that will need to be completed. The final task is optional – this will depend on your specific installation.

 

Please contact support if you require any clarification of tasks contained within this How To guide
