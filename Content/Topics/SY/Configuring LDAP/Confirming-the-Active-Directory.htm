Confirming the Active Directory Server Resolution

The LDAP Directory user authentication implemented in , uses your network’s domain name services (DNS) lookup to identify all available directory servers that are available for user authentication.

The server must be able to resolve directory servers using: _ldap._tcp.<domain> where <domain> is the domain name of your network. The following task shows the steps involved in testing the ability of your server to identify available LDAP directory servers.

Testing the directory server lookup

No LDAP directory servers

If your server is unable to identify any LDAP directory servers on your network, talk to your network administrator to determine whether or not your server can be configured to access a domain name server that is able to identify LDAP directory servers on your network.

The above option is outside of the scope of this How To guide.
Please refer this to your server and/or network administrator for resolution.
