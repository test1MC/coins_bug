Prerequisites

This “How-To” guide assumes that the configuration of the environment for ODBC access has been completed by a COINS System Administrator. End users will not have the required access or capabilities to perform the required configuration.

The user performing the tasks within this “How-To” must have:

  • A Microsoft Windows based computer
  • Administrative rights to the computer on which the SQL Client software is being installed
  • The Progress OpenEdge SQL Client Access installation package for your operating system.

This software can be provided free of charge for all licensed customers. Please contact COINS to arrange for software and licence codes.

The settings required to make the ODBC connection to the environment, these include:

  • The server’s fully qualified domain name (FQDN)
  • The environment’s database SQL port number
  • Their SQL user name
  • Their SQL password
