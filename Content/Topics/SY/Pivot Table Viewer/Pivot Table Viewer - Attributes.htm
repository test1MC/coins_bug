Pivot Table Viewer - Attributes

Attributes are shown on the left – these can be dragged as columns or rows and can also be used for filtering to show a subset of the data.

Pivot Table Viewer Attributes

The viewer shows only the ed items. You can use the Filter results field to limit which items are shown in the attribute selector, to make it easier to find them in a large list, but it is only by ing or uning items that you include or exclude them from the pivot table.

Once filtered. the field name will appear in italics.
