Pivot Table Viewer - Overview

A new Pivot Table viewer is available in to assist in analysis of data. It can be added to Data Mart extracts.

It is also available on all browse reports, and can be added to reports created using Report Designer. For more information see the documentation for PDR 24636 (OA-CE-SY020 Pivot Table Viewer for Reports). For more information see 1, Pivot Tables in Reports - Overview.
