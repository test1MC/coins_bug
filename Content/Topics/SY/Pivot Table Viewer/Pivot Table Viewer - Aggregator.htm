Pivot Table Viewer - Aggregator

The Aggregator determines how values are grouped together from the underlying data.

Pivot Table Viewer Aggregator Selector

The following options are available:

Field
Description
Count

Count of records for each cell.

Count Unique Values

Count of records with unique values – prompts for field.

List Unique Values

List of unique values for each cell – prompts for field.

Sum

Totals a specified numeric measure.

Integer Sum

Totals the rounded values for a specified numeric measure

Average

Averages the values from a specified numeric measure.

Minimum

Minimum value from a specified numeric measure.

Maximum

Maximum value from a specified numeric measure.

Sum over Sum

Divides the sum of a specified measure by the sum of another specified measure to give a ratio.

Sum as Fraction of Total

Each cell as a percentage of the grand total for a specified numeric measure.

Sum as Fraction of Rows

Each cell as a percentage of the row total for a specified numeric measure.

Sum as Fraction of Columns

Each cell as a percentage of the column total for a specified numeric measure.

Count as Fraction of Total

Each cell as a percentage of the total number of records.

Count as Fraction of Row

Each cell as a percentage of the number of records in the row.

Count as Fraction of Columns

Each cell as a percentage of the number of records in the column.
