Pivot Table Viewer - Saving and Loading Views

You can save the settings for a view, so that when you run the same report or data mart again and use the Pivot Table Viewer, the same view is used.

At the bottom of the viewer is a save and load toolbar:

Pivot Table Viewer Save and Load Toolbar

To save a view – enter a name into the text box (bottom left) and click the Save button ().

To load a previously saved view – select it from the drop-down list (<Load Existing>) and click the Load button ().

To delete a previously saved view – select it from the drop-down list (<Load Existing>) and click the Delete button ().
