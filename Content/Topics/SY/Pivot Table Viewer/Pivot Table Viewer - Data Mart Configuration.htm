Pivot Table Viewer - Data Mart Maintenance

Data Mart Tab

There is a new flag on the Data Mart tab in Data Mart Maintenance.

Data Mart Maintenance Data Mart Tab

Field
Description

Pivot Table

This enables the pivot table functionality for the data mart.

Columns Tab

If the Pivot Table flag on the Data Mart tab is ed then another field will appear on the body update of the columns browse to allow the user to select whether the column is available for filtering and viewing in the pivot table. The default is ed.

Data Mart Maintenance Columns Tab

Extracts Tab

If the Pivot Table flag on the Data Mart tab is ed, two new columns show on the Extracts browse:

Data Mart Maintenance Extracts Tab

Field
Description
Pivot Extract

This indicates that a Pivot Table data file has been created for the extract.

The pivot data file is saved in $BASE/var/datamarts and is named as follows: dw_{kco}_{dwt_type}_{dwe_seq}.json.

Pivot Viewer

If a Pivot Extract exists then a button is available to allow the user to launch the Pivot Table viewer for that extract.

A new Generate Pivot Table Data action on the Extracts browse can be used to recreate the pivot table data file for the extract – this needs to be done if the data mart configuration has changed.

The Delete Extract action now deletes the pivot table data file associated with the extract if it exists.
