Report Designer - Pivot Tables

There is a new option in Report Designer that allows you to export pivot data when the report is run.

Field
Description

Pivot Data

If this is ed, the body query is used to export all the fields from all the tables in the query to a pivot data file. This data file is stored alongside the other report output files in the spool directory. When a report with this option selected is run, the Pivot Table Viewer button () appears on the Report Status workbench and allows the user to view the data in the Pivot Table Viewer.
