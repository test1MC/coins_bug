Pivot Tables in Reports - Overview

There is a new capability in Report Designer to allow the designer to specify what data is dumped to the Excel file produced.

A new Pivot Table viewer is available to assist in the analysis of data. It is available on all browse reports, and can be added to reports created using Report Designer. It can also be made available on Data Mart extracts; for more information see the documentation for PDR 24582 (OA-CE-SY019 Pivot Table Viewer for Data Marts). for more information see 1, Pivot Table Viewer - Overview.
