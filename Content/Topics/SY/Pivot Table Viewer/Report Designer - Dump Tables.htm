Report Designer - Dump Tables

It is now possible to specify on the report configuration what data should be exported to the Excel file when running the report.

A field has been added to the Report Designer Query tab.

Field
Description

Dump Tables

A comma-separated list of temp tables or datasets to export to the Excel file. The list of temp tables can either be prefixed or not with “tt”. The tables listed will be dumped in the Excel output.

If you leave this blank, the report sort fields and body form fields are used and these are built in to the export. This is the default.

NOTE

You can also specify what data should be exported to the Excel file when running a report by specifying DUMPTABLES= in the function parameters. This will override the new feature above.
