Browse Reports - Pivot Tables

The generic browse report has been enhanced so that the pivot table viewer is available for any browse report. Any data that is produced from a browse report option is also then available for pivot.

On the browse, filter to the records required and click the Print button ().

Select the required fields as usual and run the report.

The Pivot Table Viewer button () is available on the Report Status workbench:

Report Status Workbench with Pivot Table Viewer Button

And also on the popup menu:

Unread Print Requests Popup Menu with Pivot Table Viewer Button

Click the button to display the data in the Pivot Table Viewer.
