Using the Pivot Table Viewer - Renderer

The Renderer determines how the data is shown.

Pivot Table Viewer Renderer Selector

The following options are available:

Field
Description
Table

Simple table with rows and columns.

Table Barchart

Each cell in the table shows both a value and a bar to indicate relative magnitude.

Heatmap

Cells are coded – dark red indicates largest number, white indicates smallest number.

Row Heatmap

Each row has its own heatmap range (largest number in each row is dark red).

Col Heatmap

Each column has its own heatmap range (largest number in each column is dark red).

Table with Subtotal

Table with subtotals for nested row attributes. Also allows rows to be closed or opened to show detail.

Table with Subtotal Bar Chart

Combines features of Table with Subtotal and Table Barchart.

Table with Subtotal Heatmap

Combines features of Table with Subtotal and Heatmap.

Table with Subtotal Row Heatmap

Combines features of Table with Subtotal and Row Heatmap.

Table with Subtotal Col Heatmap

Combines features of Table with Subtotal and Col Heatmap.

Line Chart

Columns are X-axis. Each combination of row attributes is shown as a separate line. Good for showing time series data.

Bar Chart

Columns are X-axis. Each combination of row attributes is shown as a separate bar.

Stacked Bar Chart

Columns are X-axis. One stacked bar per column showing combination of each row attribute.

Area Chart

Similar to Line Chart but fills in the area below the line.

Scatter Chart

Shows the relationship between two measures (for example, Quantity and Profit). Both rows and columns should be numeric values.

TSV Export

Tab Separated Values. You can copy the data and paste it into Excel.
