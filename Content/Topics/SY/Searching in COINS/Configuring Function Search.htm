Configuring Function Search

Menu Search (see Menu Search) and the function search within Global Search (see Global Search) allow users to find and run  functions. Both searches rely on the same data, which needs to be generated beforehand. Run System > System Setup > Build Function Search Data to generate the initial data, and run it again whenever changes have been made to functions or menus.

WARNING

The menu search shows functions that the user has access to. If your security model relies on allowing users access to everything then denying access to specific menus, this could mean that users will have access to functions you do not want them to. We do not recommend this method of controlling access to functions.
