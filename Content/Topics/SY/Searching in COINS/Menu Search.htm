Menu Search

Menu Search allows you to quickly find and run any function you have access to.

To search for a function

  1. Open the main menu panel. To do this, either:
    • Press Ctrl+Shift+>.
    • Click the "open menu" button in the page header.
    • Click the "open menu" handle in the middle of the left-hand border of the screen.

      • The Menu Search field is at the top of the main menu panel.

  2. In the Menu Search field, type part of the function name.

    NOTE

    To position the cursor in the Menu Search field, you can press Ctrl+Shift+> even if the menu panel is already open.

  3. displays functions that match what you type.

    If you move the mouse pointer over a result row, a tooltip shows the module that the function belongs to.

    This can be useful if the results include similarly-named functions from different modules.

  4. To run a function from the search results, either:
    • Press the Down Arrow key until the function is highlighted, then press Return.
    • Click the function.

NOTE

You can also search for functions using the Global Search.
