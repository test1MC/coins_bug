Searching in - Overview

The Search button in allows you to look for different types of information.

  • You can find and run functions. NOTE: You can also do this using the menu search.
  • You can find records in the database, such as customers, , or people. A panel within the search frame shows you more details, and you can open the record directly from the search frame. For performance reasons, this is typically restricted to 'static' data, which does not change frequently.
  • If you have an external search engine configured, you can use the extended search capability which allows you to search large volumes of data such as invoices or purchase orders, to find specific transactions, as well as 'static' data. This allows you to navigate directly to records, more quickly than using specific . The extended search capability uses the Global Search frame.
