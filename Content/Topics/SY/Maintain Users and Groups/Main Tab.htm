Main Tab

 

Field

Description

Buyers Code:

The User ID of the buyer.

Buyers Name:

The name of the buyer.

Buying Department:

The buying department.

Represents Company:

The company that the buyer represents. This is memorandum text only and can be printed on the purchase order

Represents Region:

The region that the buyer represents. . This is memorandum text only and can be printed on the purchase order

Retired:

Whether this buyer is retired.

You can prevent retired buyers from raising or committing orders.

 

Associated Parameters

 

PO/BUYERVAL

Materials:

The maximum value that the buyer is allowed to order on material orders.

Plant:

The maximum value that the buyer is allowed to order on plant orders.

Subcontract:

The maximum value that the buyer is allowed to order on subcontract orders.

Can F9 in the eOrder Workbench:

Select this if the buyer is allowed use the F9 option in coinsplus in the eOrder Workbench.
