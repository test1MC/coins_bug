Identifying State as part of Company Set Up

In order to communicate with Third Parties, such as ATO, State is often a field that is required as part of the validation routine.

Please ensure that the following configuration is applied to

Step 1 - Update Terminology

Navigate to System > System Setup > Help Tools > Terminology

Locate and open the term global address4

Within the Display Text field, enter "State"

Click to save the line.

Step 2 - Review/Update Company Set Up

Navigate to System > Company Setup > Configuration > CO Configuration

Address Line 4 has now been updated with the field name "State".

 

Open the record and add the relevant state. Save the record.
