Keyboard Shortcuts Configuration Overview

There are some pre-defined keyboard shortcuts for the most common actions in OA, like Ctrl+Shift+S for saving the record. The problem is that some browsers disallow intercepting some shortcuts – for example, the standard Ctrl+Shift+N shortcut (Add/New) does not work in current versions of Google Chrome. Keyboard Shortcuts Configuration allows you to configure the list of OA shortcuts to avoid browser-specific problems, and to define alternative shortcuts.

Initial Configuration

No additional configuration is required before using these features.

Screen and Processing Changes

Keyboard Shortcuts Configuration

Keyboard Shortcuts Configuration is a new function that allows you to configure the list of OA shortcuts, and to define alternative shortcuts.

To register new shortcuts:

  1. Click Open.
  2. Click either the Shortcut or the Alternative Shortcut field.
  3. Press the desired key combination.
  4. Click Save.

NOTE

Only shortcuts with Ctrl, Alt and Shift modifiers are allowed.

You may need to clear your browser cache, or fully reload the page (Ctrl+F5 in IE; Ctrl+Shift+R in Chrome) to get the changes to take effect.

If you try to enter a shortcut that has already been used, displays an error message and highlights the duplicate shortcut.

To delete a shortcut, click the Shortcut or Alternative Shortcut field (as appropriate) and press Delete on the keyboard.

To reset all the shortcuts to the pre-defined defaults, click Restore Defaults.

Tooltips

For shortcuts that are linked to buttons, (like Add, Open, Save, and Next), the shortcut is displayed in the button tooltip:
