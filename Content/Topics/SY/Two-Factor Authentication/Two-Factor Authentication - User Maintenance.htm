Two-Factor Authentication - User Maintenance

A new field has been added to User Maintenance.

Field

Field
Description

The user’s number (for SMS messages).

If using two-factor authentication, one or both of the Email or fields must be completed for the user to be able to access the system with a normal interactive login.

If the AUTH2SMS parameter is set to Y, SMS is used as a preference, and if the field for a user is not completed then the email method will be used.
