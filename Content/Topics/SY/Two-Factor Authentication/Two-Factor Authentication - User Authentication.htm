Two-Factor Authentication - User Authentication

When you attempt to log in to OA if two-factor authentication is enabled, you must fill in the existing login screen as usual.

When the user ID and password have been submitted, an additional form is presented that prompts you for a onetime use passcode. At the same time an email or SMS (depending on the configuration) is sent to you, to provide you with the passcode.

Enter the code in the passcode field and click log in.

Provided the passcode is correct and was entered within the allowed time limit, you will be taken to your normal login page, menu and desktop.

If you enter the passcode incorrectly or after it expires, an error is shown.

You can click Cancel to be returned to the Log In page to try again.

If someone attempts to access using your user ID and password, you will be notified with the passcode message (via email or SMS) which would alert you to this breach.
