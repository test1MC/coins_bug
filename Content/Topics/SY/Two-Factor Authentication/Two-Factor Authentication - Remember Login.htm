Two-Factor Authentication - Remember Login

If not using the Remember Login feature, the user will be required to enter a valid passcode after the existing Log In screen every time they enter . This is not recommended. The recommendation is to switch on Remember Login (SY/WEBRIPS) so that once they are authenticated, users are able to access the system for a set number of days (specified in SY/WEBRDAYS or overridden on the user record).
