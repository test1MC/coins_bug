Inline Widget

Inline uses a "widget" to launch the inline side panel. Standard user controls allow the placement of the widget at any point on the edge of the screen. However, as COINS uses the full screen space, the widget obscures at least one or more COINS screen items.

COINS code changed to display an icon in the Header (controlled by the parameter SY/MANTYPE) that launches the Inline side panel. Inline administrators will need to disable the Panel widget when they create their sites. Instructions added to documentation.
