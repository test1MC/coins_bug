Inline Manual Setup

It is now possible to embed documentation produced using the third-party product "Inline Manual" in COINS.

Two new parameters enable this functionality:

MANKEY - The access API key issued by the manual provider.

MANTYPE - The manual provider.

 

MANKEY for Inline Manual is a 32 character code. This code will be generated when you create your Inline site for use in COINS. Within the Site settings in Inline, navigate to the Player Installation Menu option. The 32 character key can be found within the Embed Player code.

MANTYPE for Inline Manual should be set to Inline to embed Inline Manual. This will provide a new icon in the COINS Header which will make the Inline Manual side panel available.

If MANTYPE is blank then no manual will be embedded and the Icon will not be displayed.

 

Hiding the Inline Manual Widget

COINS makes maximum use of the available screen area. This makes the default use of the Inline Manual widget impossible as COINS cannot control which areas of the screen may be obscured by the default widget. For this reason, COINS have provided the new Header Icon to activate the Inline Panel and the Default Widget must be switched off on your Inline site.

To hide the widget:

Login to the Inline Manual portal with your e-mail and password.

Go to Sites and choose the desired site.

Go to the Settings tab.

Go to Player settings.

Scroll down to the Panel trigger section and mark Hide trigger. You can also hide the widget for mobile devices (all mobile devices that are smaller than a tablet) by checking Hide trigger only for mobile devices.

Confirm with Save button.
