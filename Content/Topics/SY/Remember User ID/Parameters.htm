Remember User ID – Parameters

The following parameters need to be set:

Parameter

Description
WEBRIPS A list of IP addresses that if identified from the browser will be allowed to remember the user ID and allow pass through access. If this is blank (default) then this feature is switched off. This is intended to allow for internal IP addresses to be identified and only allow this pass through access if the user is on the corporate network.
WEBRDAYS The number of days that a user ID will be remembered. The default is for the remainder of the current day (based on the server date), but this can be set to any integer. This will be the number of days that the user will be allowed access to without further authentication provided they are on the network defined in WEBRIPS
WEBRPWD An additional password, which is encrypted and checked on login to allow access. This is intended to prevent a user setting up a similar system, creating a login token, transferring to the live system and gaining access. Without this password they would not be able to set up the correct token. You can re-set this if you want to force all users to log in.
