Remember User ID – Logout

When a user whose IP address is defined in the WEBRIPS list logs out, a box appears on the logout confirmation :

Field

Forget User

If this is ed and the user clicks OK, the next time the user accesses they will see the login screen and will have to log in manually. This means that if a user logs in to using someone else's computer, they can ensure that no-one else can access using their credentials. If the user logs out without ing this box, the next time they access on the same computer they will not need to log in.

When a user whose IP address is defined in the WEBRIPS list logs out, they are taken to the page specified by the SY parameter LOGOUT (or to a default page if LOGOUT is blank).
