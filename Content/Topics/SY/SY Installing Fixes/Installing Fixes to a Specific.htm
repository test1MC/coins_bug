2.5 Installing Fixes to a Specific Environment

To install a fix or fixes into an environment is all done from the COINSadmin menu. In this section we will be going through the steps for installing Fixes to a specific environment. This should be done during a quiet time, i.e. during a lunchtime or in the evenings.

  1. Log in to the COINSadmin area and locate the Administration Main Menu screen

  2. Select the Environments option from the Administration Main Menu screen, using the Arrow keys. Press the Return key.

    The Environments Menu screen will be displayed.

  3. As we are deploying the Fix to a specific environment we need to inform the system. Select the Administer an Environment option from the Environments menu using the Arrow keys. Press the Return key.

  4. A list of the current environments that you will be displayed. Select the specific environment you require. For our example we are selecting the Learn environment using the Arrow keys. Press the Return key.
  5. The Environment Menu screen is displayed, showing the environment you have selected in brackets, i.e. (LEARN).

    The environment will be displayed alongside the title of each screen going forward. If this is not the environment that you required, press the F4 key to return to the previous screen.

  6. Select the Install option on the menu using the Arrow keys. Press the Return key.

    The Deployment Menu screen is displayed for you to select the option required.

  7. Select the Deploy Fix option on the menu using the Arrow keys. Press the Return key.

    The Select Fix screen is displayed listing the fixes that are available, with additional details in a table format. 

    The fields available are

    FIX

    The unique number of the fix

    Date

    The date the fix was released for deployment

    FIX Type

    There are various types of fix available. It will also advise you if this is a monthly service pack.

    User

    The developer/user that raised the fix

    Reason

    Short description of the fix.

  8. Select the fix that you require using the Arrow keys and the details panel will change giving you full details of the fix.

    If you want to view the files that will be affected by the Fix, press the F7 key to view the files window. (Press the F4 key to return to the list).

  9. Using the Arrow keys select the file you require to install. Press the Return key to confirm the Fix you require. This will mark the Fix to show it has been selected with an * or Asterisk adjacent to the number.

    Continue using the Arrow keys and Return key, until all the required files have the * or Asterisk adjacent to the number.

  10. Press the F1 key. This will start the process of deploying the fix. The Deploy Fix screen will be displayed showing details of the “Install Fixes” you have selected. Enter details into the relevant fields.

    User Name

    Your Name. The Fix Status enquiry uses this data to show who installed fixes

    Multi User Compile

    The Default is Y = Yes the database brokers are running. Change to N = No if they are not running.

    Delayed

    Do you want to delay the installation of the fix? The system will ask you to enter the date and time required

  11. Press the F1 key to continue to install the fix. The Fix Compile screen will be displayed and show the progress of the Fix File or Service Pack being installed.

    COINSadmin copies the files to the correct directory and recompiles any programs affected by the fix.

    The Fix Installation Message screen will be displayed informing you that the Fix has been successfully installed.

  12. Press the F4 key to return to the Deployment Menu screen.
  13. To show details of the fixes you have installed select the Fix Status Enquiry option using the Arrow keys. Press the Return key.

    The Fix Enquiry screen is displayed listing details of the fixes.

    Sequence

    The sequence the fixes were deployed

    User Name

    Who installed the fixes

    Date / Time

    The date and time the fixes were installed. Note - this could be different as the installation could have been delayed.

    Vrs

    The Version of the environment, i.e. V1022

    Pch

    The Patch number

    Reason

    A brief reason of what the fix relates to

  14. To return to the COINSadmin menu press the F4 key. You may have to press the F4 key more than once to return to the Administration Main Menu screen.

You will now be required to trim the webspeed agents. See Trim the Webspeed Agents.
