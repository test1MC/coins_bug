Installing Multiple Fixes to Multiple Environments

 

The COINSadmin Tool has the facility to apply fixes to multiple environments during one session. This is beneficial when applying the same fixes to multiple environments, however all the environments must be on the same server.

 

It is strongly recommended that you deploy and test all fixes in a non-live environment BEFORE you deploy them in a live environment.

Log in to the COINSadmin area and locate the Administration Main Menu screen.

Select the Configure COINSadmin option from the menu using the Arrow. Select the Return key.

The Configuration Menu screen will be displayed. Select the Multiple Environment Fix Install option from the menu using the Arrow keys. Select the Return key.

The Select Environments screen will be displayed. Select the environment that you require to apply using the Arrow keys. Select the Return key. This will mark the selected environment with an Asterisk.

When you have selected all the environments select the F1 key. Note - The environments must be on the same release and service pack level.

The Select Fix screen is displayed the list of fixes available. Select the Fixes that you require to apply using the Arrow keys. Select the Return key. This will mark the selected fixes with an Asterisk.

When you have selected all the fixes you require, select the F1 key.

The Install Fixes screen is displayed. This will start the process of deploying the fixes. The Deploy Fix screen will be displayed showing details of the “Install Fixes” you have selected. Enter details into the relevant fields.

User Name

Your Name. The Fix Status enquiry uses this data to show who installed fixes

Multi User Compile

The Default is Y = Yes the database brokers are running. Change to N = No if they are not running.

Delayed

Do you want to delay the installation of the fix? The system will ask you to enter the date and time required

Select the F1 key to continue to install the Fixes or Service Packs. The Fix Compile screen will be displayed and show the progress of the Fix File or Service Packs being installed.

COINSadmin copies the files to the correct directory and recompiles any programs affected by the Fix or Service Pack.

The Fix Installation Message screen will be displayed informing you the Fixes or Service Packs have been successfully installed.

Select the F4 key to return to the Configuration Menu screen.

To show details of the fixes you have installed select the Fix Status Enquiry option using the Arrow keys. Select the Return key.

The Fix Enquiry screen is displayed listing details of the fixes.

 

Sequence

The sequence the fixes were deployed

User Name

Who installed the fixes

Date / Time

The date and time the fixes were installed. Note - this could be different as the installation could have been delayed.

Vrs

The Version of the environment, i.e. V1022

Pch

The Patch number

Reason

A brief reason of what the fix relates to

To return to the COINSadmin menu select the F4 key. You may have to select the F4 key more than once to return to the Administration Main Menu screen.

You will now be required to trim the webspeed agents.
