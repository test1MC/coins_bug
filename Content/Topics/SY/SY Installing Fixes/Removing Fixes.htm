Removing Fixes

 

In some instances you may need to remove a fix. There is a set process to follow when removing the fix. It is recommended that all users are logged out of the environment before you remove the fix, especially in a Live Environment. 

Some fixes may take some time to collate the files and be removed. It compiles the previously installed fixes to ensure compatibility. Do not interrupt this process as it could take up to an hour to complete.

Log in to the COINSadmin area and locate the Administration Main Menu screen

Select the Environments option from the menu using the Arrow keys. Select the Return key.

The Environments Menu screen is displayed. Select the Administrator an Environment option from the menu using the Arrow keys.

The Environment Menu screen will be displayed. Select the Install option using the Arrow keys. Select the Return key.

The Deployment Menu screen will be displayed. Select the Fix Status option using the Arrow keys. Select the Return key.

The Fix Enquiry screen will be displayed. A list of fixes will be detailed in the enquiry screen. All fixes that have been installed and deployed will have an Asterisk next to the fix number.

Select the fix required to be removed using the Arrow keys, for example, Fix Number 063368. Select the F10 key. The Fix will be removed from the list.

Select the F9 key to recompile the remaining fixes.

The Install Option screen is displayed. Enter the details in the relevant fields. This will start the process of deploying the remaining fixes.

User Name

Your Name. The Fix Status enquiry uses this data to show who installed fixes

Multi User Compile

The Default is Y = Yes the database brokers are running. Change to N = No if they are not running.

Delayed

Do you want to delay the installation of the fix? The system will ask you to enter the date and time required

The Deploy Fix screen will be displayed showing details of the “Install Fixes” you have selected. Enter details into the relevant fields.

Select the F1 key to continue to install the fixes. The Fix Compile screen will be displayed and show the progress of the Fix File or Service Packs being installed.

COINSadmin copies the files to the correct directory and recompiles any programs affected by the fix.

The Fix Installation Message screen will be displayed informing you the Fixes have been successfully installed.

Select the F4 key to return to the Configuration Menu screen.

You will now be required to trim the webspeed agents.
