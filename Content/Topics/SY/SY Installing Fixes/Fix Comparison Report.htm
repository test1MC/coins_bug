Fix Comparison Report

 

The Fix Comparison Report enables you to compare the fixes installed on two environments on the same Server. (This is similar to the Compare Documents on two word documents to see if there are any differences, i.e. a fix that has not been installed etc. This would go into the trainer’s notes and not in here)

The best example to use would be when you are applying the fixes to the Live Environment. It will ensure that the Live Environment has all the same fixes installed and deployed as the environment you have tested the fixes in.

Log in to the COINSadmin area and locate the Administration Main Menu screen.

Select the Configure COINSadmin option from the Administration Main Menu screen using the Arrow keys. Select the Return key. The Configuration Menu screen will be displayed.

Select the Fix Comparison Report option using the Arrow keys. Select the Return key.

The Admin screen will be displayed. Enter the two environments you are comparing. In this example we are using learn and cplive. Select the Return key.

The Compare Environment Fixes Report is displayed. You can see that two fixes have been installed and deployed on cplive, whilst only one has been installed on the learn environment.

You will now be required to trim the webspeed agents.
