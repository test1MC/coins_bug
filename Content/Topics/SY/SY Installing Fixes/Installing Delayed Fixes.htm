Installing Delayed Fixes

 

You can delay the Fix or the Service Packs to a time that is more suitable for you and your business, i.e. during the lunchtime, or out of office hours. Going through these stages will enable you to select a specific time that you wish to start installing to fix.

Log in to the COINSadmin area and locate the Administration Main Menu screen

Select the Environments option from the Administration Main Menu screen, using the Arrow keys. Select the Return key.

The Environments Menu screen will be displayed.

As we are deploying the Fix to a specific environment we need to inform the system. Select the Administer an Environment option from the Environments menu using the Arrow keys. Select the Return key.

A list of the current environments that you will be displayed. Select the specific environment you require. For our example we are selecting the Learn environment using the Arrow keys. Select the Return key.

The Environment Menu screen is displayed, showing the environment you have selected in brackets, i.e. (LEARN).

The environment will be displayed alongside the title of each screen going forward. Should this not be the environment that you required select the F4 key to return to the previous screen.

Select the Install option on the menu using the Arrow keys. Select the Return key.

The Deployment Menu screen is displayed for you to select the option required.

Select the Deploy Fix option on the menu using the Arrow keys. Select the Return key.

The Select Fix screen is displayed listing the fixes that are available, with additional details in a table format. 

The fields available are

FIX

The unique number of the fix

Date

The date the fix was released for deployment

FIX Type

There are various types of fix available. It will also advise you if this is a monthly service pack.

User

The developer/user that raised the fix

Reason

Short description of the fix.

Select the fix that you require using the Arrow keys and the Details panel will change giving you full details of the fix.

Should you wish to view the files that will be affected by the Fix, select the F7 key to view the files window. (Select the F4 key to return to the list).

Select the Return key on the fix that you require to be installed. This will display an “*” (asterisk) adjacent to the Fix Number to show which fixes or Service Packs you are installing. Note - you can select more than one fix or service pack.

Select the F1 key. This will start the process of deploying the fix. The Deploy Fix screen will be displayed showing details of the “Install Fixes” you have selected. Enter details into the relevant fields.

User Name

Your Name. The Fix Status enquiry uses this data to show who installed fixes

Multi User Compile

The Default is Y = Yes the database brokers are running. Change to N = No if they are not running.

Delayed

Do you want to delay the installation of the fix? The system will ask you to enter the date and time required

 

Enter Y = Yes into the Delayed field and select the F1 key. This will display the delayed fields details for you to enter in the relevant fields.

Enter the time you require the install to start in the Time field in the format HH:MM. Enter the valid email address in the Email field that you want to have the confirmation message sent. Select the Return key.

The message will be displayed detailing when the install is scheduled. Select the OK option by select the Return key.

You will return to the Deployment Menu screen. The fixes and service packs will be installed in the background.

When the install is complete an email is sent to confirm that all the selected fixes and service packs have been done.

You will now be required to trim the webspeed agents.
