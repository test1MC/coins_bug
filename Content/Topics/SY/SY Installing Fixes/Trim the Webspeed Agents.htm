Trim the Webspeed Agents

 

After a Fix File or Service Pack has been installed and deployed you must trim the Webspeed agents so that users will use the updated programs, with all the new fixes and files. This would be the same as removing all the history and cookies on the website.

You must use the same environment the fix was installed from, i.e. LEARN.

Log in to the COINSadmin area and locate the Administration Main Menu screen.

Select the Administer Servers and Brokers option from the menu using the Arrow keys. Select the Return key.

The Administer Servers and Brokers screen is displayed. Select the Webspeed Administration option using the Arrow keys. Select the Return key.

The Select Webspeed Broker screen will be displayed. Select the environment required, i.e. learn, using the Arrow keys. Select the Return key.

The Administrator Webspeed Broker Menu screen will be displayed. Select the Trim Agents option using the Arrow keys. Select the Return key.

The Trim Agents screen will be displayed. Enter the Number of Agents in the field.

We would recommend you enter 99 to trim, which is the maximum and limit available to trim. This will ensure that no agents will be omitted.

The message will be displayed showing the number of agents successfully trimmed. Select the Space Bar to continue.

The Administer Webspeed Broker Menu screen will be displayed

The fixes have been deployed (Or removed) and are available for all the users.
