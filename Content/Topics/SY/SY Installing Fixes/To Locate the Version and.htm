To Locate the Version and Environment

 

There are various ways to find the details of the version number of your current environment and you may feel comfortable using one of these methods.

However we are going to use the Configuration Report to find this information and to do this you are required to log into COINSadmin.

Select the Configure COINSadmin option from the Administration Main Menu screen, using the Arrow keys. Select the Return key.

This will display the Configuration Menu screen. Select the Configuration Report option using the Arrow keys. Select the Return key.

The Configuration Report is displayed detailing information regarding the environments and the versions.

Use the Page Up and Page Down keys to scroll through the report. The Repository will show information of the versions currently loaded.

By scrolling further down the report you will locate the environment that you require.

This will also confirm the version number of the environment that you wish to load fixes in. In our example you can see that the Learn Environment is on V10.22 (101102 – is the latest release).

Select the F4 key to return to the Configuration Menu screen.
