Integration Application - Command Line Arguments

There are two recogned command line arguments for the OA Outlook Integration Application (COINSExchangeReplication.exe):

/configFile

Name of a configuration file. If omitted then the standard configuration file name of _exchange.settings is used.

/logFile

Name of log file.  If omitted then the standard log file name of _exchange.txt is used.

Sample command line with command line arguments:

"C: \coins_exchange\ COINSExchangeReplication.exe"  /logfile=COINS_exchange.TEST.txt /configFile=COINS_exchange.TEST.settings
