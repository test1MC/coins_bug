Configuring Activity Workbench Replication

The steps for configuring Activity Workbench Replication are:

  1. Set up an AppServer on the Unix server (see 1, AppServer)
  2. Create the replication user account in Active Directory (see Active Directory Replication Account)
  3. Assign the Impersonation account permissions (see 1, Impersonation)
  4. Set the replication values on the user records (see 1, Activity Workbench Replication)
  5. Install replication software (see Install the Replication Software)
  6. Configure replication software (see 1, Settings and 1, Command Line Arguments)
  7. Schedule replication (see 1, Scheduling)
  8. Configure Outlook (see 1, Activity Workbench Replication)
