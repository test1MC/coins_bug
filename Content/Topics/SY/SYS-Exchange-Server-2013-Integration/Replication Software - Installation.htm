Install the Integration Application

The OA Outlook Integration Application needs to be installed on a Windows machine. It is not recommended to install it on the same server as is used for Exchange.

will provide the install files.

  1. Run the installer, which creates a directory to hold the replication install files and to provide a working directory for the execution of the replication process.  A sample directory structure may be:

    C:\_Exchange
    C:\_Exchange\Installation
    C:\_Exchange\wrk

    Note: The working directory needs to be writeable by the replication account.

  2. This creates the following files in the working directory (for example, C:\_Exchange\wrk):
    • _exchange.txt - this is required for logging.
    • _exchange.settings - this is the configuration file
  3. Update the configuration file as necessary; see 1, Replication Software Settings for details of what to put in the configuration file.
