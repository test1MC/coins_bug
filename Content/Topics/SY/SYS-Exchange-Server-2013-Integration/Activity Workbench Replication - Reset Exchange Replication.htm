Reset Exchange Replication

The first time replication is run for a user all the user’s appointments and tasks will be created in Exchange.  Once this is complete replication will switch into “changes only” mode, in which only appointments and tasks changed since the last replication run will be copied to Exchange.  If the user’s replication status is reset then the next replication run will again replicate all appointments and tasks to Exchange.
