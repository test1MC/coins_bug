Exchange Integration - Prerequisites

Exchange Integration requires the following:

  • Active directory account for replication.
  • Exchange 2013 or later, or Exchange Online.
  • A separate Windows server for replication (recommended)
  • Direct connection between the Windows server and the Exchange server.
  •  OA for the appropriate role and concurrent user count.
  • HTTPS to be enabled with a valid SSL certificate on the web server for the OA environment.
