Configuring Contacts Replication

The steps for configuring Contacts Replication are:

  1. Setup an AppServer on the Unix server (see 1, AppServer)
  2. Set parameters (see 1, Contacts Replication - Parameters)
  3. Create the replication user account in Active Directory (see Active Directory Replication Account)
  4. Create the contacts public folder (section 1, Contacts Replication)
  5. Install replication software (see 1, Installation)
  6. Configure replication software (see 1, Settings and 1, Command Line Arguments)
  7. Schedule replication (see 1, Scheduling)
  8. Configure Outlook (section 1, Contacts Replication)
