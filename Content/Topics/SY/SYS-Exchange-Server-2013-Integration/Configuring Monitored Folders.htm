Configuration of Monitored Folders

The steps for configuring import from Monitored Folders are:

  1. Set up an AppServer on the Unix server (see 1, AppServer)
  2. Create the replication user account in Active Directory (see Active Directory Replication Account)
  3. Create the public folders to be monitored (see , Monitored Folders)
  4. Associate the folders with an OA program (see 1, Monitored Folders)
  5. Install replication software (see 1, Installation)
  6. Configure replication software (see 1, Settings and 1, Command Line Arguments)
  7. Schedule replication (see 1, Scheduling)
