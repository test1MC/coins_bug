Reset Contacts Replication

The first time replication is run the process will copy all contacts to Exchange.  Once this is complete replication will switch into “changes only” mode, in which only contacts changed since the last replication run will be copied to Exchange. 

Reset Contacts Replication (%WCI7070FPCN) resets the contacts replication status so that the next replication run will again replicate all contacts to Exchange.

To reset contacts replication, the box and click the “next” button.
