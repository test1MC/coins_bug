Monitored Folders

Exchange Monitored Folders (%WSY2000BSTE) lists any Exchange Public Folders to be monitored. Once defined here, the Exchange Integration utility periodically checks the folder for new messages and runs the specified program, passing in the message sender, recipients, subject, text, and any attachments.

Field Definitions

Field
Description
Code

The code number automatically generated when you add a new monitored Exchange Public Folder.

Description

A description for the monitored Exchange Public Folder.

Folder Path

The full path to the monitored Exchange Public Folder.

Archive Path

The full path to the archive Exchange Public Folder.

If set, messages are moved here after processing.

Program

The OA program to be run for each message in the monitored folder.

This program must include syu136.i and have the following signature:

  • input pcSubject as char
  • input pcText as char
  • input pcFrom as char
  • input pcTo as char
  • input pcCC as char
  • input pcParam as char
  • input table ttAttachHdr
  • input table ttAttachDat
  • output pcMsg as char
  • output pbOk as log

Parameters
The parameters are passed to the specified OA program, which allows the same program to handle different folders in different ways.
