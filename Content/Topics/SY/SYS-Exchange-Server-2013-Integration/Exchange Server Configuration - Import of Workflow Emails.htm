Import of Workflow Emails

Import of workflow emails can be done using either a mailbox or a public folder.

Using a Mailbox

Using a mailbox to import workflow emails is the preferred method.

  1. Set up an email account for the inbound workflow emails.
  2. Set the SY/WFETO parameter to this email address.
  3. Set the WorkflowMailBox parameter in the replication software settings file to this email address.

The user account running the replication software needs access to the workflow mailbox if these accounts are different.

Using a Public Folder

A new public folder should be created for this purpose.  The folder needs to be given an email address and this address needs to be stored in the SY/WFETO parameter as described in section 1, Import of Workflow Emails.

The replication account should be given permission level of at least “Editor” on the workflow folder.

The Default permissions should be set to “Reviewer” or lower.
