Import of Workflow Emails

The following SY parameters relate to Exchange Replication (under System, System Setup, SY Parameters):

ID

Description

Configuration

WFETO

Workflow Response Email Address

Set this to the email address of the public folder for replies to workflow emails.

WFMON

Workflow monitor error email

An email address that will receive an email containing the error message if the workflow monitor stops with an error. The detail of the error will be shown in the body of the email. Details of what the monitor was processing will be contained in the workflow monitor log file.

The Workflow Monitor (%WWF1000MWFT) must be running to send the workflow emails to users.
