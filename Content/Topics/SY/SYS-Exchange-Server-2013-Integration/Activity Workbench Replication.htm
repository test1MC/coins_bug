Activity Workbench Replication

Set the replication values on the user records.

Two fields on User Setup relate to Exchange replication:

Field

Label

Configuration

sur_exchsvr

Replication Mail Server

A code to identify the replication process used for this user.  This needs to match an environment name in the replication software configuration file.

sur_mailbox

Replication Mailbox

Mailbox account name.  This is the user name of the windows account.

Make sure that these fields are set correctly for all users.
