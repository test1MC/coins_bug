Configuring Import of Workflow Emails

The steps for configuring import of Project Emails are:

  1. Set up an AppServer on the Unix server (see 1, AppServer)
  2. Set parameters (section 1, Import of Workflow Emails)
  3. Create the replication user account in Active Directory (see Active Directory Replication Account)
  4. Create the workflow mailbox or public folder (section 1, Import of Workflow Emails)
  5. Install replication software (see 1, Installation)
  6. Configure replication software (see 1, Settings and 1, Command Line Arguments)
  7. Schedule replication (see 1, Scheduling)
