Imported Exchange Emails

The Imported Exchange Emails browse (%WSY5220BSDO) displays any emails imported from monitored folders (as shown in Figure 1: Imported Exchange Emails Screen). This is not found on any OA menu; you can view it by specifying the following at the end of the URL at the top of the page:

&MainArea=%WSY5220BSDO.
