Contacts Replication - Parameters

The following CI parameters relate to Exchange Replication (under Procurement, Setup, Configuration, CI Parameters):

ID

Description

Configuration

EXCNMETH

Exchange Server contact replication API

Set to “EWS”.

EXCNREPL

Exchange Server contact replication enabled?

Set to “Y” to enable contacts replication.  Once this is done, changes to companies, offices and contacts will be stored for replication to Exchange.

EXCNAUTH

Authentication token for Exchange Server contacts replication

No longer required.

EXCNURL

URL for Exchange Server contracts replication

No longer required.
