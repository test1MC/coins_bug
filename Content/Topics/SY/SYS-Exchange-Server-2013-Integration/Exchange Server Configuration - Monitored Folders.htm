Monitored Folders

Monitored folders are public folders on the Exchange Server.  The replication account should be given permission level of at least “Editor” on each of the monitored folders.

The Default permissions should be set to “Reviewer” or lower.

Each monitored folder may also be given an email address (optional)

If an archive folder is specified in the Exchange Monitored Folder maintenance in OA (see 1, Monitored Folders) then a public folder must also be created with this name and the same permissions as the monitored folder.  Archive folders do not need an email address assigned.
