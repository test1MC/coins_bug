Named Users

Location: System > User Maintenance > Named Users

Named Users allows you to specify which users are currently licensed to use . It provides an alternative to maintaining the Users individually.

 

To maintain a user, click next to the User ID and tick/untick the appropriate columns. Click to save your changes.

Field

Description
User ID The ID of the user.
Click the column to sort
The user ID is the code which a person uses to identify themselves in the system; for example, when logging in.
Name The name of the user or group with this ID.
Click the column to sort
Named User Whether this user is a 'current' user and has access to .
The number of current users you can have is limited by your licence.
Designer

Whether the user is licensed to run OA Reporting & BI Designer functions.
The number of users allowed to run designer functions is limited by your licence.

OA Reporting & BI > Document Designer > Document Designer is available whether this is ticked or not.
Writer Whether the user is licensed to run Report Writer functions.
The number of users allowed to run Report Writer functions is limited by your licence.
ODBC Adapter Whether the user is licensed to access the ODBC adapter.
The number of users allowed to access the ODBC adapter is limited by your licence.
Webservices Whether the user is licensed to access Webservices.
These allow XML program requests from other systems. They are used for integration of 3rd party applications with .
The number of users allowed to access Webservices is limited by your licence.
Workflow Whether the user is licensed to run workflows.
The number of users allowed to run workflows is limited by your licence.
