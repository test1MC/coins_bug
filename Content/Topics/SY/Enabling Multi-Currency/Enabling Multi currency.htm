Enabling Multi-currency

  • Configure the company using System > Company Setup > Configuration > CO Configuration, but leave the Currency field blank initially.
  • Set up the country codes
  • Set up the General Ledger account codes for the exchange gains and losses account, unrealised variance account and rounding account.  See GL Accounts.
  • Set up the codes for the currencies you will use in that company.  See Currency Codes.
