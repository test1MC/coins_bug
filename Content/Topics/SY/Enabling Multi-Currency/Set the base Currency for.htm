Set the base Currency for the Company

To set the currency you MUST be logged in to as SYSAdmin rather than your normal user login.  This ensures that only the system administrator is able to set the base currency for a company.

  1. Ensure that no other users are accessing the company during this process.
  2. Change to the company you want to set up.
  3. Go to System > Company Setup > Configuration > Set Currency.
  4. Enter the code for the base currency of the company - this is the currency that the company will work in.

If you are converting an existing single-currency company, and there are already transactions on the system, sets all transactions and supplier and customer records to the base currency. This may take several minutes.  The process works through every account and every transaction present on the company and calculates the various currency fields required.

Before entering any transactions the Currency will need to be set on the company. Each company has a base currency and can also have an alternate base. You need to configure each company separately. You can have some companies using multi-currency and some using single currency operation (although if you do, you will not be able to do General Ledger cross-company consolidation).
