Set the Colour for displaying the country code

In enquiries and other programs, you can set up to highlight the currency code with a different if it is for a foreign currency. The base currency code will not be highlighted

  1. To set the for the currency code:
  2. 1. Go to Company Parameters.
  3. 2. As the value of the parameter CURCOL, enter a pair, for example: K/Y for blacK on Yellow.
