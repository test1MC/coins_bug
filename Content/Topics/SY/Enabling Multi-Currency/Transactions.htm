Procurement

An order can be input into the system using a different currency to the base providing all the set-up as above has been completed.

Order put into system in GPB

 

When the order is displayed in the Material Orders screen this will be displayed in the foreign currency and the Currency noted.

 

On the Contract this will be displayed in Company Base Currency 

 

Currency conversion rate will be the latest rate that is prior to the transaction date. If there is a change in the rate prior to invoicing this will be handled at the invoicing stage.
