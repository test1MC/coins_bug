Currency Exchange

Currency Exchange allows you to maintain:

  • The exchange rates between your base currency base currency (the currency that the current company works in. Different companies can have different base currencies) and other currencies.
  • The conversion rates used for inter-company consolidation and multi-base currency revaluation.

You need to set up the exchange rates separately in each company.

When you are setting the system up, you must ensure that there are exchange rates dated before the earliest transaction you will set up.
