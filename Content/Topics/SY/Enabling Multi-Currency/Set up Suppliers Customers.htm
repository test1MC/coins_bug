Set up Suppliers, Customers

 

By default, when you set up an account for a new supplier, customer or subcontractor, will assign the base currency to that account.

When you set up a new account from the Company Information Workbench and link the Office to an account already existing in the Purchase(Supplier)  Ledger , assigns the account currency according to the set up on the office in the PL of the company that you select.

To set up the office:

1. Go to Company Information Workbench.

2. Select the company.

3. Select the Offices TAB and add a New Office.

4. Leave the Company Information Workbench and go to Supplier Maintenance in the .

5. Set up a new or Sales Ledger account. The details should be the same as on the old account, except that the Currency code should be set to the new currency.

If you make a mistake with the currency code, you can go back and change it provided that you have not yet entered an invoice against the supplier. Once you have entered an invoice, you cannot change the currency code on the account.

7. In the Company Information Workbench the Office will now be stamped with the Account number and the  PL Currency Code, SL Currency Code and SC Currency Code fields will display the currency. If the company does not yet exist in the Sub-ledger the currency will not be displayed.

 

 

 

6. Then set up the account in the ledger in the usual way.

If the account already exists in the ledger of any company, the currency code you set up on the office must match the currency of the account – will not let you enter a different one. This also means that on accounts with the same number in different companies, the currency code must be the same.

 

Base currency

The currency that the (current) company works in. Different companies can have different base currencies.

Foreign currency

A currency other than the base currency of the company you are working in

Document currency

The currency in which the transaction is entered.
