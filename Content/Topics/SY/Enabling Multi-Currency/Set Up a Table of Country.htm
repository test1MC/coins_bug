Set Up a Table of Country Codes

Maintain Countries allows you to maintain country codes. County codes are used to identify the countries in which your company and your customers or suppliers operate. Even if you only deal with customers or suppliers in your own country, you need to set up a code for that country.

 

 

Field

Description

Country

A code for the country.

Note: You must use the reserved code GB for Great Britain. This allows to validate contract, supplier and subcontract VAT registration numbers according to the UK VAT registration number algorithm.

Name

The name of the country.

EC Member

Is the country a member of the European Community (Y or N)? This is used to determine whether Intrastat processing is required.
