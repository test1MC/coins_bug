Set up Contracts

The Currency on the Contract allows you to set up a Contract in a different Currency to the Base currency of the Company and report the contract in the Document Currency.

 

  1. In the set-up of the Contract define the Currency . It is advisable to keep the contract at one Currency and not have the ‘Allow Alternate Currencies’ flag on.

 

To Note:

  • You cannot change the currency on a Contract once any codes etc have been set up or any costs , commitments or accruals have been recorded on the contract . This also includes inputting Budgets.
  • If you have incorrectly set up a currency on a contract this can be undone providing NO costs have been incurred.
  • You can set up the Base currency on the contract then record orders and transactions in the foreign currency, the system will adjust the amount for reporting according to the set up of the currency exchange rates.
