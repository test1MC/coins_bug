Dynamic Queries - Functions

It is possible to create functions for a specific dynamic query that can be placed on menus or as tiles on the desktop.

Dynamic Query Function

Create a function with parameters as follows:

  • Set STN_CODE to %WSY1352FXXX (for the ), %WSY1353RXXX (for the report) or %WSY1354RXXX (for the extract).
  • Set DYNQ to the query that is required.
  • Set Sy_tableTLA to ste61 (which is the RSP for sy_table used for dynamic queries).

This function can then be placed on the menu and/or as a tile on the desktop.
