Mail Assistant - Overview

The Mail Assistant provides additional features when using Outlook:

  • Save incoming and outgoing emails and attachments to records (such as a purchaser or a ).
  • Search for information in directly from Outlook.
  • Use email templates to compose emails in Outlook.

Mail Assistant can be used with Outlook 2013 or later, and with Office 365.

NOTE

Mail Assistant only works if there is a network connection to the environment.
