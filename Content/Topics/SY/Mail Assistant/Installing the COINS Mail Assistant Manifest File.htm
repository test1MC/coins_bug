Installing the Mail Assistant Manifest File

The Mail Assistant manifest file tells the Outlook client where to find the Mail Assistant files. Each COINS environment needs a different version of the manifest file. Either COINS will provide the appropriate version(s) of this file based on information you supply, or you can generate the manifest file for your environment(s) (see Generating the Mail Assistant Manifest File).

You then need to install the manifest file and make it available to your users.

  1. Log in to on-premises ECP Exchange 2013/2016 or Exchange Online portal.

  2. Navigate to Organization > Apps.

  3. Select "Add from file".

  4. Browse for the manifest file .
  5. Edit the properties and choose how you want Mail Assistant to be made available to your users. (The screenshot below is for illustration only and is not a recommendation.)
