Modifying and Installing the Manifest File

The Mail Assistant manifest is an XML file that tells the Outlook client where to find the Mail Assistant and its associated files. To use Mail Assistant, each environment needs a different version of the manifest file.

You can generate the manifest file using OA (see Generating the Mail Assistant Manifest File). Then the manifest file needs to be installed on Exchange and made available to your users (see Installing the COINS Mail Assistant Manifest File).
