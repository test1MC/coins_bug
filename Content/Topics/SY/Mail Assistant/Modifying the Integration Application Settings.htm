Modifying the Integration Application Settings

If you need to change the settings of the OA Outlook Integration Application:

  1. Run setup.exe again.
  2. You will be prompted to stop the current activity:

  3. Click Stop all, or click the buttons to stop each activity individually.
  4. Choose what you want to update:

  5. You can then update the integration application settings (see , Integration Application Settings).
  6. Click the Install button to save your changes.
  7. You then need to restart the replication service: in Control Panel - Administrative Tools - Services, right-click the COINSExchangeReplication service, and choose Start.
