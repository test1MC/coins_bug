Modifying the Mail Assistant Manifest File

The manifest file is supplied by ; most of it does not need to be changed, but if has not provided you with a version that is configured for your environment, the URLs need to be changed to point to the correct locations.

The highlighted areas in the illustration below show some of the entries that need to be changed.

In the DefaultValue attribute of the following elements, change the text before "/scripts/lib..." to the base directory for (which is the value of the SY/WEBROOT parameter, plus "/coins/" plus the value of the SY/WEBBASE parameter):

  • <IconUrl>
  • <HighResolutionIconUrl>
  • <Resources><bt:Images> <bt:Image> (there are three elements like this)

In the DefaultValue attribute of the following elements, change the text before "/woumsai.p..." to the script directory for (which is the value of the SY/WEBROOT parameter, plus "/" plus the value of the SY/WEBSCRPT parameter):

  • <FormSettings><Form xsi:type="ItemRead"><DesktopSettings><SourceLocation>
  • <FormSettings><Form xsi:type="ItemEdit"><DesktopSettings><SourceLocation>
  • <Resources> <bt:Urls> <bt:Url> (there are three elements like this)

 

Note

The URLs need to use https:// (not http://) - in other words the webserver needs to have a valid security certificate.

For example, if the parameters are set as follows:

SY/WEBROOT = https://mycoinsenvironment.com

SY/WEBBASE = live

SY/WEBSCRIPT = cgi-bin/wslive

 

The elements within the <Resources> tag would look like this:

<Resources>

  <!-- add information on resources -->

  <bt:Images>

      <!-- Blue icon -->

      <bt:Image id="coins-icon-16" DefaultValue="https://mycoinsenvironment.com/coins/live/scripts/lib/msmailAddin/images/_Icon_16x16.png"/>

      <bt:Image id="coins-icon-32" DefaultValue="https://mycoinsenvironment.com/coins/live/scripts/lib/msmailAddin/images/_Icon_32x32.png"/>

      <bt:Image id="coins-icon-80" DefaultValue="https://mycoinsenvironment.com/coins/live/scripts/lib/msmailAddin/images/_Icon_80x80.png"/>

    </bt:Images>

    <bt:Urls>

      <bt:Url id="composeTaskPaneUrl" DefaultValue="https://mycoinsenvironment.com/cgi-bin/wslive/woumsai.p?mailtype=compose"/>

      <bt:Url id="readCustomPaneUrl" DefaultValue="https://mycoinsenvironment.com/cgi-bin/wslive/woumsai.p?mailtype=read"/>

      <bt:Url id="readTaskPaneUrl" DefaultValue="https://mycoinsenvironment.com/cgi-bin/wslive/woumsai.p?mailtype=read"/>

    </bt:Urls>

When you have modified the manifest file, you need to install it on Exchange and make it available to users; see 1, Installing the COINS Mail Assistant Manifest File.
