Generating the Mail Assistant Manifest File

The manifest file tells your Exchange server and Outlook clients where to find the Mail Assistant.

NOTES

You must generate the manifest file from within the relevant OA environment. Do not generate the manifest for example for your test environment from within your live environment.

The SY parameter WEBROOT should be set to a URL, not an IP address; the Web Root URL is taken from this parameter.

To generate the manifest file:

  1. Go to Generate Mail Assistant Manifest (on the System > System Setup menu).

  2. In the Mail Assistant Unique Identifier field, select the appropriate identifier for the current environment (for which you are generating the manifest file).
  3. In the Mail Assistant Environment Name field, enter the name you want to use in Outlook for this environment.

    The other fields are populated from parameter settings.

  4. Click Process to generate the file.

    The file is available on the My Files tab of Report Status.
