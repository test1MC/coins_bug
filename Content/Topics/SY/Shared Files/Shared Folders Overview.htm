Shared Folders - Overview

 

Shared folders is a new feature that allows several users to access one or more shared folders in addition to their own home directory.

This is intended to remove the need for SAMBA shares (which are increasingly a security concern) to allow users to share files within .  These folders might be used for example for files, by configuring a parameter to place output file in a shared directory so that a different user can download the file and process it with the bank.
