Shared Folders – My Files

If shared folders have been set up, an additional body selector or combo is shown on each of the “My Files” tabs.

My Files - My Files Tab

On the My Files tab the selector allows you to pick your own home directory or one of the shared directories to which you have READ access.

The Get File tab has a combo selector to allow you to get a file from a specific folder.  This is just another option to get a file and provides the same functionality that is available on the My Files tab by selecting the appropriate folder.

You can only GET files from directories to which you have been granted access to the READ function as defined on the shared folder configuration.

My Files - Get File Tab

The GET will default to your home directory.

Similarly the Put File tab allows you to select the target folder for the uploaded file.

My Files - Put File Tab

You can only PUT files to directories to which you have been granted access to the WRITE function as defined on the shared folder configuration.
