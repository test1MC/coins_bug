Shared Folder Setup

The shared folders required are configured by the system administrator using the Shared Folders function on the System Setup menu.

Shared Folders Setup Screen

Saving a record will create a directory, if it does not exist, in $BASE/var/home with the specified directory name (in lower case).  Do not use a user ID as this is where the user’s home directories are also stored.  If you delete a directory, it and its contents will be removed.

The Read Function and Write Function control which users have access to this directory. These functions must be set up in Function Maintenance. The functions can be called anything you want; you then need to set the user function access to these functions, to control access to the shared directory.

Function Maintenance – Access Functions

You can use the same function for both read and write permissions, and for different directories, if required.  If the user has access to the function, the corresponding read/write capability will be made available.  The function access is granted in the same way as would normally be done for functions on menus.
