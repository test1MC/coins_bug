Reporting Issues - Workflow

You can define a workflow that will be launched when an issue is raised and the ISSTYPE parameter is set to Workflow or Both. The workflow template to be launched is specified in ISSWF. The workflow template must exist in the company specified in ISSWFKCO.

Issue Report Log - Workflow Screen

The fields on the issue report page are available to be assigned in to workflow variables.

Please contact if you need further information or assistance with using Workflow.
