Reporting Issues - Email Template

A new function Report Issue Configuration (%WSY1320SMNP) allows you to configure the title and body of the email that will be sent.

Report Issue Configuration Screen

You can use {} to replace values from fields on the issue report page (either the standard or any customed version). Field names must match the fields on the page; for example : {RW_su$$userid} (using the standard $$ to replace the hyphen). Fields from the function from which the issue report is launched are also available; for example: {ms_fulldesc}.

If either of the fields is blank then a standard layout is used. The title consists of the function description and function code and the body is a simple table of the fields.
