Mobile Form Reporting

Sequence numbers are now shown for sections and fields. These values can be used with RO fields to access values of a completed mobile form.

New fields have been added to allow you to optionally store field values on the header of the form. This allows queries to be written to find forms with particular values. For example, if a safety audit form has fields for number on staff on site and construction phase, a query could find audits done during groundworks, where there were more than 50 people working on the site.

Mobile Form Printing - Simple and Complex

There are changes to the mobile forms print in Document Designer, to simplify and easily identify the fields used in the mobile form. The mobile form field ID (FID) can be used when designing the output print document, which makes mapping input easier. This is useful for simple forms that do not have repetitive sections (see 1.1, Simple).

Additionally there is complex way already available and detailed in this document (see 1.2, Complex), which is determined by use of Mobile Form Section element.

See also the Enhancement document OA-CE-SS002 Health and Safety Audits.
