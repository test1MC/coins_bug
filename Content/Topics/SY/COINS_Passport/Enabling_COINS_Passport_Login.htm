Enabling Passport Login

Two new SY parameters control the “log in via Passport” functionality.

Parameter

Description

SY/CASAUTH

The URL to the Construction Cloud Single Sign-On (SSO) server, provided by Construction Cloud for this OA Environment. Usually looks like https://cas-sso-example.com/authorize/[ext-system-id].

SY/CASAUSEC

A private key for JWT token verification, also provided by Construction Cloud.

If both parameters are non-blank, the “Log in via Passport button” is available on the OA login screen.

The OA Environment must be configured on Construction Cloud as an external system, available for CCC SSO authentication.

The UNIX ‘openssl’ command-line tool (https://www.openssl.org/) must be available for this functionality to work.
