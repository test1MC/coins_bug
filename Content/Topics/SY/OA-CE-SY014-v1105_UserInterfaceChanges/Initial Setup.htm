Initial Setup

The SY parameter STYLEVAR will be automatically set to v1105 as part of the upgrade process. The previous style settings are available in case you need to refer back to them temporarily, but are no longer supported.

You should remove existing custom style sheets (for example, user.css), as these may conflict with the new styles. If you want to make style changes that are not covered by the new palettes (for example, to change the background ), you can create a new user.css to override the standard settings in the new style sheet.

Note that the SY parameters in the “Style” group are no longer used and will be removed in a future release.
