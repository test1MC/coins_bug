Column Sets and Browse Menu

The column set selector has been moved to a browse menu, which is available by moving the mouse pointer over the menu icon in the top-left corner of a browse.

The menu shows the following (depending on your permissions

  If you have access to the Query Editor, the menu includes an option to open the query for the current page in the Query Editor, in a new tab.

and on whether column sets are enabled for the browse):

  • Show Selected/Show All (you can still click the icon to show the selected records, or Shift+Click to select all records).
  • Maxime – This appears if there is an inline browse on a tabbed page. It allows you to open the browse in a frame, so that you can see more records. When you close the frame, the inline browse is refreshed. See Maxim[%=Variables.is%]e Browse.
  • Column Sets – This allows you to maintain column sets.
  • The available column sets are listed below the Column Sets option, with the currently-selected column set highlighted at the top. If the column set has resizable columns, buttons to allow you to save or clear the column set widths are available.
