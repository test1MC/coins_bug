Advanced Filter

To make the advanced filter accessible while still showing as many rows as possible, the advanced filter is now shown in a popup frame.

The Advanced / Simple button toggles between the advanced and simple filter.

A Not Filtered / Filter Applied button on the button bar shows whether the advanced filter has been applied, and clicking the button will display the Advanced Filter.

If an advanced filter has been applied, you can quickly see which columns have been filtered: move your mouse pointer over the Filter Applied button - a tooltip shows the filtered columns.

On the Advanced Filter popup:

  • The Apply button applies the filter and hides the filter popup.
  • The Clear button removes the filter and hides the filter popup.
  • The Simple button switches to the simple filter.

You can use the following shortcut keys: Ctrl+Shift+F to display or hide the Advanced Filter; Ctrl+Shift+A to apply the Advanced Filter; Esc to hide the Advanced Filter popup.
