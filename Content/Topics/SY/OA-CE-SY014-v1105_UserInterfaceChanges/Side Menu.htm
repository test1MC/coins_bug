Side Menu

The default side menu now lists all modules to which you have access, rather than including a drop-down list of modules. The icons for function categories are now -coded; the s can be changed (see Function Category Maintenance).

The company selector has been moved from the previous side menu to the page header (see "Page Header" on page 1).

A menu search has been added – this allows you to search for any function on the menus (it is the same as the function part of the Global Search).

New shortcut keys Ctrl+Shift+> and Ctrl+Shirft+< open and close the menu panel.

Note: If you change the width of the menu panel, this is remembered when you next log in.
