Screen and Processing Changes

The v11.06 changes described in this section were made for the Workbench in the module.

Generate Forecast Spread Utility Selection Form

The Generate Forecast Spread Utility selection form was changed as follows:

  • Spread Method field: The Replace with S-Curve Distribution option was renamed as Replace with S-Curve (Normal Distribution).
  • A new Replace with S-Curve (Library) option was added (see Figure 1: Generate Forecast Spread Utility Selection Form). When selected, a new S-Curve Library field displays. This field cannot be blank, must be complete, must exist in S-Curve Library Maintenance, and cannot be marked as Dormant.
Generate Forecast Spread Utility Selection Form

Generate Forecast Spread Utility Processing

The Generate Forecast Spread Utility program was enhanced to use Spreading Utility Library Methods. If you select the Replace with S-Curve Distribution option, a spread is generated using a pattern from the S-Curve Library selected in the S-Curve Library field.
