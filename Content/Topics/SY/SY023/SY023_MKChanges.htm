MK Screen and Processing Changes

The v11.06 changes described in this section were made in the module.

Forecasting Options

Source field options were changed in Forecasting Options details found in > Setup > Setup > Forecasting Setup, as follows (see Figure 1: Source Option):

  • Option S-Curve was renamed as S-Curve (Normal Distribution).
  • A new Source option labeled S-Curve (Library) is now available in the Billings, Cost, Cash Inflow, and Cash Outflow layout boxes. When selected, a Code field appears that can be used to select the Code you wish to use for this Source option. An error occurs if this field is left blank or the Code is flagged as Dormant.

Forecast

The Forecast browse can be accessed in the Forecasting tab of the Workbench or by drilling down on the Description link in the Forecasting Workbench. Source field options were changed in these areas, as follows:

  • Option S-Curve was renamed as S-Curve (Normal Distribution).
  • A new Source option labeled S-Curve (Library) is now available in the Billings, Cost, Cash Inflow, and Cash Outflow layout boxes (see Figure 2: [%=Variables.Mk Project caps%] Forecast Browse). When selected, a Code field appears that can be used to select the Code you wish to use for this Source option. An error occurs if this field is left blank or the Code is flagged as Dormant.

The new Forecasting options are processed by the Generate Forecast Spread action program.
