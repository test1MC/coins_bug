SY Screen and Processing Changes

The v11.06 changes described in this section were made in the System module.

S-Curve Library Maintenance

A new S-Curve Library function was added to System > System Setup. A sample browse screen is shown in Figure 1: S-Curve Library Browse Screen.

S-Curve Library Browse Screen

Field Definitions

Code

The code for the S-Curve header. It links to the S-Curve Library Detail page (see Section 1.1.1 Detail Screen).

Description

This is a description of the S-Curve.

Notes

Notes regarding the S-Curve are entered here.

Complete

Indicates whether the S-Curve spread is complete (last period cumulative value = 100.0).

Last Revision

This is the date and time of the most recent change to the record, and the user ID of the user who changed it.

Dormant?

Whether the S-Curve is dormant. Dormant S-Curves are not shown in drop down lists.

Spread Tab

The S-Curve Spread tab shows an updateable browse of 100 periods (automatically created when you enter an S-Curve header), as shown in Figure 6: S-Curve Spread Tab.

S-Curve Spread Tab

Field Definitions

Period

The S-Curve Spread period number can be from 001 to 100.

Movement

This is the Movement amount for the period. An error occurs if the number is negative.

Cumulative

This is the Cumulative amount for the period. If this amount is changed, the remaining spread records are recalculated sequentially.

This field is disabled for Concurrent Update and Bulk.

Notes

Notes for the spread period are shown here.

S-Curve Spread Tab Actions

Two actions are available on the S-Curve Spread tab if the parent S-Curve is Incomplete: Spread Unallocated (Prorate) and Spread Unallocated (Evenly). Both of these actions run as foreground processes using a Spread Completion Utility program (see Section 1.1.2 S-Curve Spread Completion Utility), which distributes unallocated values on selected records evenly or by prorating. Only selected Spread records are changed.

S-Curve Spread Options Menu

This menu contains the following single option if the parent S-Curve is Incomplete: Complete Spread using Known Values. It is run as a foreground process using a Spread Completion Utility program (see Section 1.1.2 S-Curve Spread Completion Utility), which uses the curve fitting method to complete the spread.

Used in Tab

The S-Curve Used In tab (Figure 7: S-Curve Used In Tab) shows the usage of a current S-Curve Library record in the system. It finds the records to use by:

  • Searching through Forecasting Option Sets and collecting Forecasting Options that reference the current S-Curve Library record as a source of Billings, Cost, Cash Inflow, or Cash Outflow.
  • Searching through if the following conditions are true:
  • A An is active and no exists for it.
  • A An uses a Forecast Options Set that references an S-Curve Library record as the source of any S-Curve.
  • The first incomplete Forecast of the is found that references an S-Curve Library record as the source of any S-Curve.
S-Curve Used In Tab

S-Curve Spread Completion Utility

The S-Curve Library Spread Completion Utility uses Spreading Library methods to distribute unallocated amounts by using the following methods:

  • Curve Fit
  • Spread Unallocated (pro-rate)
  • Spread Unallocated (evenly)
  • Replace with Even Distribution
  • Replace with S-Curve (Normal Distribution)
