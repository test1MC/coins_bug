System Monitor

The system monitor shows all the processes in the top layout of the page.

System Monitor Screen

The dots show:

  • Green if the correct number of processes (as defined in the coins.cfg file) are running.  
  • if no processes are defined and none are running. 
  • Yellow if fewer processes are running than expected.
  • Red if a process is running and none are expected (this would only happen if someone had manually run a process outside this control).

The toolitip on the dot shows in brackets the number of processes found and the number expected.

These dots are based on the AppMon snapshot data which is taken at intervals of five minutes.  They are therefore slightly out of date and you may need to wait for up to five minutes for the status to shown to catch up with reality.
