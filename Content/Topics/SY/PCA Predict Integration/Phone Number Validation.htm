Phone Number Validation

If the PCA Predict phone number validation service is enabled, phone number fields in can be validated automatically.

If the phone number is valid, a message is shown to confirm the valid phone number, and details of the type of phone number (such as landline or mobile) and the network. If the phone number is not valid, a warning is shown.

In addition, the validation checks that a mobile phone number is being entered in the mobile phone number field and a mobile number is not being entered in the home phone number field. As the work phone number could be a mobile number this check is not done on this field.
