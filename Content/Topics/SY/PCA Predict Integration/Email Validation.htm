Email Validation

If the PCA Predict email validation service is enabled, email address fields in can be validated automatically.

If the email address is valid, a confirmation message is shown. If the email address is not valid, a warning is shown.

Depending on the setup of the email server to which the address belongs, it may not be possible to validate the user name, in which case only the domain name can be validated. The validation message informs you if this is the case.
