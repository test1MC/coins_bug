International Dialling Codes

International Dialling Codes Maintenance has been amended to allow allows the standard ISO2 country code to be stored. This is required when validating telephone numbers.

International dialling codes are provided as standard data so these will be populated automatically as part of the upgrade.

Maintain Countries has also been amended to allow also allows the ISO2 code to be stored. This is used when validating phone numbers that are not prefixed with an international dialling code, or where there is no country on the page.

If there is no country, looks at the country record from the company configuration record.

If there is no ISO2 code associated with this country, it consults the CO parameter DEFCNT to determine the country and uses the ISO2 code from the country specified in this parameter.
