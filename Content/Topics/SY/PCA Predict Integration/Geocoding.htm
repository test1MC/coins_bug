Geocoding

New fields have been introduced in House Sales to store the geo-coordinates for the home address on the People Workbench and the development address on the Development Workbench. This includes the easting, northing, longitude and latitude. This will allow future functionality such as heat maps to be developed. These new fields are placed after the address field.

The House Sales parameter PIWXFLD determines if these new fields are shown. The new System parameter PCASERV determines if the geocoding service is to be used.
