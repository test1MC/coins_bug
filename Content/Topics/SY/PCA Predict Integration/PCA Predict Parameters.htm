PCA Predict Parameters

The following new SY parameters affect the of integration with the PCA Predict web service which validates addresses, phone numbers, email addresses and bank account details.

Parameter Description
PCASERV

Which validation services you want to use. You can select one or more of the following:

  • Address Search
  • Phone Number Validation
  • Email Validation
  • Bank Account Validation
  • Geocoding
PCAKEY The key for the PCA Predict web service.
PCAADFLD

This allows you to override the default mapping and specify what each address line should be populated with. This is a list of PCA fields to use. It does not include the .

The following HS parameter now allows you to select additional fields to display if using the geocoding service.

Parameter Description
PIWXFLD

Additional fields that can be prompted for when entering or updating a person in House Sales, Customer Care or Lettings Management. The list now includes:

  • Easting/Northing
  • Latitude/Longitude
