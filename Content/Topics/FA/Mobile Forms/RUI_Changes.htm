Changes

Menu

A new menu entry Asset Forms has been added to the menu.

Menu

Asset Forms Home Page

This has a single section with tiles: Forms, Assets, Daily Check. The Forms tile shows the number of incomplete forms you have. Daily Check indicates the number of checks that you need to perform today.

Asset Forms Screen

Forms

The forms tile takes you to a browse of asset mobile forms. Tap a form to view the details, then tap to update it.

Asset Form

Assets

The Assets tile takes you to a browse of assets which can be filtered by item, group, depot and status. Type T and A assets are excluded and depot security is applied.

Assets Browse

This shows all assets allocated to you or for which you are responsible.

Tap an asset to take you to mobile forms for that asset.

Tap to add a new form; select the type of form to add, and tap .

Daily Check

The Daily Check option is designed to make it as simple as possible to enter a daily equipment check. The tile is only shown if you are allocated an asset of a group that specifies a daily check form, or are currently responsible for one. This function takes you to the asset form summary, to add or update a form specified by the asset’s group.
