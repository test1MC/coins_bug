OA Changes

Mobile Forms

A new option, Asset, has been added to the Type field. If Asset is selected, it must be the only type selected.

Mobile Forms Maintenance Screen

You can now only have form elements when the section type is Table.

Asset Groups

A new Mobile Form field has been added to allow you to specify a daily check form for the asset group. If a form is specified for the asset group, the user allocated to or responsible for an asset in the group will be prompted to complete a daily check using that form.

Asset Workbench

A new browse has been added to show mobile forms associated with the asset, similar to the Contract Mobile Form browse.

Asset Mobile Forms Workbench

Asset Mobile Forms Workbench is a new function that allows you to view completed mobile forms for assets.

The initial browse lists the available asset mobile forms.  A link on the Code field displays a browse of instances of that form.
