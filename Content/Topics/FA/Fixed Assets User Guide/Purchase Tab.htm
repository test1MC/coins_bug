Purchase Tab

You should leave this tab blank if you are going to be using the Asset/PL Reconciliation functionality within – this function pulls the invoice details onto the Asset automatically without manual entry. As the PL invoice can come in after the asset has been delivered, the Asset/PL Reconciliation can be carried out at a later date and the Asset entered onto the system at the delivery stage.

 

Field

Description
Supplier Select the Supplier from the Lookup
Reference The PL Invoice reference number
Purchase Date The Purchase Date
Price The Purchase Price
Minimum Book Value The value of the Asset after being fully depreciated – usually 0 or 1

 

Note: The Fixed Asset Purchase Batch is a system-generated batch, the date and period defaults to the date of entry of the first asset in the batch, and the current Financial period in Global Periods Maintenance. If there is no current Financial period identified the Asset Purchase will go into the earliest open period. Populating the purchase date and purchase price in this tab creates a batch in the Fixed Asset Purchase Batch which is later manually posted (after running the Purchase Posting Report).
