Asset Purchase Order

The initial asset purchase is done using an Overhead Order and a commodity code which is linked to the Asset Purchases control account.

From the Procurement module select Overhead Orders

Create the Overhead Order.

Enter the header details for the Overhead Order in the relevant fields. Click .

Field

Description
PO Number The order number will either be entered manually or system-generated, based on your Procurement parameters setup
Department The Balance Sheet department for your purchase
Use Order Form The form which will be used when printing the Order
Supplier Supplier Details
Order Date The order date used for this purchase
Due Date The due date for this purchase
Description The short description of your Order
Buyer The user entering the Order

An Order Number is allocated and the detail page of the Order is displayed.

Click and fill in the details of the order lines.

Code Select the Z001 – Capital Purchases Commodity Code

  • Click to store the line
  • Fill in the other order lines and when you have finished click
  • Commit the order
  • Enter the Invoice as per the invoice input functionality

The Order will put the commitment on the Fixed Asset Purchases Control account. E.G. 00.5000

The invoice will become available in the Fixed Asset module for reconciliation to the Asset purchase.
