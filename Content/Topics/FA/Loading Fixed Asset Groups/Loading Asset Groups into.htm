Loading Asset Groups into OA

 

Master Asset Load Template

Create the Load File

 

In the Master Asset Register Load file, within the tab titled “Groups Load Format”, copy row 18 (the red line)

Highlight multiple rows beneath it and “Insert Copied Lines” to allow you to enter multiple Asset Group records

Enter the Asset Group details into the Master Asset Load File Register

Copy the rows that have been completed

Paste them into a new Excel document as “Paste Special > Values and Number Formatting”

Save the file as a .csv

Close down that file and do not save again despite being prompted to

Fixed Asset Group Load

Navigate to Fixed Assets > Administration > Fixed Asset Group Load

Click on the “Choose File” button

Select the file just saved as previously shown and ensure “Upload Mode” is set to “Check”.

Then Click on the “Next” button

Check the Report Status queue for the report titled “Fixed Asset Group Load” to see if there are any issues with the load file. If there are issues the report will detail which lines are the offending lines. Those lines will need to be corrected and re-uploaded in “Check” mode until they have all been checked successfully.

An example of an issue such as missing data that is required:

Once any errors have been corrected and they have all been checked successfully re-upload in “Load” mode

Again, check the Report Status queue for the report titled “Fixed Asset Group Load” to see if there are any issues with the load file. If there are issues the report will detail which lines are the offending lines. Those lines will need to be corrected and re-uploaded in “Load” mode.

*NOTE*

If there are any issues with some of the lines, only those lines will need to be re-uploaded. Some Asset Groups will have been uploaded successfully so do not attempt to re-upload them again otherwise an error message will appear on the report stating the same.

Alternatively you can use the tickbox “Overwrite Existing Asset Group” and upload all lines (those that have been loaded and those that haven’t):

To see the Asset Groups that have been uploaded navigate to Fixed Assets > Setup > Maintenance > Asset Groups

Now that these Asset Groups have been loaded into , move them into the “Loaded” section of the Master Asset Register Load file
