Asset Captions

Captions are fields that allow you to record information about an asset that would not normally be held in the system as standard, and for which fields are not provided.

Captions on their own are for information only; you can enquire and report on them, but there is no processing associated with them. However, you can link asset captions to triggers, so that when you update a caption, updates the triggers linked to it. This means can issue warnings, or stop users selecting the asset, when certain expiry conditions have been met (for example, when a service or insurance renewal is due); see Linking Triggers to Captions.

To set up captions:

  1. Set up caption groups (using Fixed Assets > Setup > Maintenance > Maintain Caption Groups).

Caption groups can be configured as:

  • Update - you can link triggers to the captions in the group, and update the captions at various stages in the plant hire process.
  • Warn - the caption will be displayed at selected points in the plant hire process.
  • No Action - you can only update these captions on the asset itself.
  1. Use Fixed Assets > Setup > Maintenance > Caption Maintenance to set up Captions.
  • The Group and Type you select on the caption govern the additional tabs available after you click Save.
  • For character captions, you can update a list of allowable values. If you want to allow any value (in other words, to make the caption a free-text field) leave the list blank.
  • For captions that belong to groups configured as Update or Warn, specify which functions allow you to update the caption: Dispatch, Return, Workshop Job Card, Timesheet, Daily Timesheet, Picking List Workbench.

For captions that belong to groups configured as Update, add the triggers.

Workshop Job Card already allows update of an Odometer field. If you want to use the odometer reading to update a trigger, use a caption for odometer and ignore (or hide) the existing Odometer field.

  1. On Fixed Assets > Setup > Maintenance > Asset Groups (if SHOWCAPS = OnlyWithDef) select which captions in each caption group are appropriate on assets in the asset group. The group default record is required for the caption to be available on an asset. When you add a new asset, you update only the captions that are set up on the asset group. So on cherry pickers you could have captions for height, reach, safe working load; on compressors you could have maximum pressure, etc.

Caption Defaults

You can configure default values for captions (that is, the values that the captions have when you add a new asset).

  • You can set a default on the asset group; this takes precedence.
  • If there is no default on the asset group, uses the value set up in Fixed Assets > Setup > Maintenance > Asset Captions Defaults.
  • If both of these are blank, uses the default on the caption.
