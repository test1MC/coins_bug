Print a Subcontract Draft Certificate

If a hard copy of the payment certificate is required then you are able to print either a single certificate or multiple.

You are also able to print a certificate after it has been released

Select the box next to the Certificate(s)

From the Actions drop down list select Print Certificates , and Apply Action .  Acknowledge any warning messages and retrieve the print.

Printing multiple Certificates will produce one PDF and not individual PDFs
