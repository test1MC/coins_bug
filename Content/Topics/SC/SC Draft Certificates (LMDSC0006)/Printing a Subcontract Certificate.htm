Printing a Subcontract Certificate – After Posting

 

After a Certificate has been posted by the Finance Department, it will no longer be available to view in the Draft Certificate area of the Subcontract Ledger module.

This section will demonstrate how to print a Subcontract Draft Certificate after it has been posted

    Select Enquiries > Select Draft Certificates

 

‘%’  Note - The Certificate Enquiry workbench is accessible by all

‘%’  Note - From the Certificate Enquiry workbench the only action available is to print certificates

    Select the box next to the Certificate(s)

    From the Actions drop down list select Print Certificates , and Apply Action . 

‘%’  Remember -  that printing multiple Certificates will produce one PDF and not individual PDFs

‘%’  Remember - Certificates are created as a  PDF.  Therefore you may wish to store the Subcontractor application and also Payment Certificate along with the Subcontract Order documentation
