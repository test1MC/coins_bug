Release a Subcontract Draft Certificate for Approval

 

Locate the required certificate(s) on the Draft Certificate Entry screen

‘%’  Note - that just as in printing certificates you are able to select and release multiple certificates in one action

    Select the box next to the Certificate(s)

    From the Actions drop down list select and Apply Action .

    Acknowledge any warning messages.

‘%’  Note - that the Certificate has now been removed from the Draft Certificate Entry workbench
