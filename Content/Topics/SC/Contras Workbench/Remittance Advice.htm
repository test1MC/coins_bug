Remittance Advice

In Document Designer, new fields are available on the Extras tab of the SC Remittance Advice and the PL Remittance Advice documents if SC/CONTWB is set to SIMPLE.

Field
Description

Contra Header Form

This allows you to specify a form that is used to include header details for contra invoices below each certificate record.

Contra Detail Form
This allows you to specify a form that is used to include contra invoices that are offset against the certificate or payment. The value offset against the payment can be shown using the fields sc_pay.RO_spy_contra and ap_pay.RO_apy_cur_contra.
