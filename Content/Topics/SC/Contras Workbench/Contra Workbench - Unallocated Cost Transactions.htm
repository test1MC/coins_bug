Contra Workbench - Unallocated Cost Transactions

If the SC parameter CONTWB has been set to SIMPLE, any new cost transactions that match the criteria set by the following SC parameters will be created as unallocated costs:

CONTJSC A cost mask to identify cost transactions as awaiting contra.

CONTCAT A cost category mask to identify cost transactions as awaiting contra.

CONTSEC A section mask to identify cost transactions as awaiting contra

CONTACT An activity mask to identify cost transactions as awaiting contra

NOTE

The new contras functionality does not work with existing data.
