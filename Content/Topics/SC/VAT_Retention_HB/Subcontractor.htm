Subcontract and Subcontractor Maintenance

The following changes have been made to both Subcontract Maintenance and Subcontractor Maintenance.

The CITB Applicable field has been removed. This is now configurable on the certificate definition based on Net of Tax for CIS.

The CITB Deduct Applicable field has been re Contractual Training Contribution. This field is now always enabled. By default the field is not ed (unless a default value has been set using Field Default Maintenance).
