Releasing Draft Certificates

 

  1. Go to one of the draft certificate workbenches.
  2. the certificates you want to release and select Release Draft Certificates from the Choose Action menu.

    This releases the current selected record if the certificate is the earliest one for the order at the 'Attention Required' level; otherwise, gives an error.

    A comments box is displayed (30 characters). If you cancel from here (), then the releasing does not proceed.

    Releasing a draft certificate (level 0 to level 1):

    • Makes it available for authoration.
    • Only if it's the earliest level 0 draft certificate for the order.
    • Can add comments.
    • Release or commercial hold is not allowed if menu parameter BILLTYPE is set and the certificate has a different billing type.
