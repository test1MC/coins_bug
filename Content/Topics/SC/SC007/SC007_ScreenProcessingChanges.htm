Screen and Processing Changes

Reports Menu

The new Subcontract Status Report is found as a menu item on the Subcontract Ledger > Reports > Subcontract Reports menu, as shown in Figure 1.

Menu Item

Selection Screen

When you select Subcontract Status Report from the SC menu, the selection screen appears, as shown in Figure 2. This screen consists of four tabs (Main, Selection, Subcontractor Selection, and SC Order Selection), which are discussed separately below.

Main Tab

The Main tab is the default screen that displays when you select the Subcontract Status Report from the menu, as shown in Figure 2.

Main Selection Screen

Include Unposted

Whether you wish to include unposted certificates in the report.

Field Set

Select the field set you wish to use for the report.

Selection Tab

The Selection tab allows you to specify which to include on the report. It uses the standard  OA  selection screen, as shown in Figure 3.

Selection Tab

Subcontractor Selection Tab

The Subcontractor Selection tab allows you to select which subcontractors to include on the report. It uses the standard  OA selection screen, as shown in Figure 4.

Subcontractor Selection Tab

SC Order Selection Tab

The SC Order Selection tab allows you to specify which subcontract orders to include on the report. It uses the standard  OA selection screen, as shown in Figure 5.

SC Order Selection Tab

Sample Report

The Subcontract Status Report shows all the SC Certificates for the selected , subcontractors, and subcontracts. Data is sorted in the following order: , Subcontractor, Subcontract, Transaction Number of Certificate (Cert No). An example of the report is shown in Figure 6.
