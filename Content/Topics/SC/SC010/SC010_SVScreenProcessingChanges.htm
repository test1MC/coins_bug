SV Screen and Processing Changes

Changes have been made in the module to allow Contractors to know whether a Statutory Declaration has been received from a Subcontractor,

SV SC Certificates Entry

When parameter SC/TAXRULES is set to AUS, a new Statutory Declaration Received field is found on the SV SC Certificates Entry screen, as shown in Figure 1. This field defaults to uned, and any changes made to it also update the same field on the SC Certificates Entry screen in Subcontract Ledger.

SV SC Certificates Entry Screen

SV SC Certificates Browse

The following new fields are found as columns on all tabs in the SV SC Certificates Browse (the screen in Figure 2 shows one of the columns as an example):

  • Statutory Declaration Attached - This column is visible but not updatable under the following conditions:
  • Parameter SC/TAXRULES is set to AUS.
  • Document category sc_certificate has the Document Group value defined in parameter SC/SCSTATDEC.
  • Statutory Declaration Received - This column is visible when parameter SC/TAXRULES is set to AUS. It is updatable and defaults to uned.
