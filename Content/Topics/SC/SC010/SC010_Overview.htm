SC Subcontract Statutory Declaration – Overview

It is now possible for a Contractor to know whether a Statutory Declaration has been received from the Subcontractor, which indicates that the Subcontractor has paid all s, employees, etc. This functionality has been added to the Subcontract Ledger and modules.

This document describes the changes that have been made in Subcontract Ledger and to provide this functionality for the AUS Region only.

Configuration

A new parameter has been added to Subcontract Ledger to identify Document Groups to be configuted for Subcontract Statutory Declarations.

Parameter

Description

SCSTATDEC

Document Group used for Subcontract Statutory Declarations

A configured Document Group for Document Statutory Declaration is used when SC/TAXRULES is set to AUS.

Certificate Document Group

The Subcontract Ledger and modules now allow Contractors to know whether a Statutory Declaration has been received from a Subcontractor. This functionality can occur only when the Certificate Document category has a Document Group (see Figure 1: Certificate Document Group) defined in parameter SC/SCSTATDEC, as shown in Figure 2: SC SCSTATDEC Parameter Setting.
