SC Screen and Processing Changes

Changes have been made in the Subcontract Ledger module to allow Contractors to know whether a Statutory Declaration has been received from a Subcontractor,

SC Certificates Entry

When parameter SC/TAXRULES is set to AUS, a new Statutory Declaration Received field is found on the SC Certificates Entry screen, as shown in Figure 3. This field is updatable and defaults to uned.

SC Certificates Entry Screen

SC Certificates

The following new fields are found as columns in SC Certificates (the screen in Figure 2 shows one of the columns as an example):

  • Statutory Declaration Attached - This column is visible under the following conditions:
  • Parameter SC/TAXRULES is set to AUS.
  • Document category sc_certificate has the Document Group value defined in parameter SC/SCSTATDEC.
  • Statutory Declaration Received - This column is visible when parameter SC/TAXRULES is set to AUS. It is updatable and defaults to uned.
