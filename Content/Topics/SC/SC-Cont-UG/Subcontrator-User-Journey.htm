Subcontractor User Journeys

 

 

 

Subcontractor users can view the SC Orders shred with them and Create Application for Order:

SC Users can enter Application for Payment Values in the Add new Application Screen.

 

Total = Gross Application – Discount – Retention

 

Labour Liable for Tax is an input field that users need to enter how much of the total is Labour Liable for Tax.

 

Users must add an attachment of their backup to support their Application.

 

Submit will send Application to the Main Contractor.

 

 

The SC Orders screen gives users a view of Applications against SC Orders and Notices.

 

 

The Applications Screen shows the status of Application and corresponding Notice.
