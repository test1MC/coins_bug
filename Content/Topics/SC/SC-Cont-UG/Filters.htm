Finding Records

To find particular records, you can filter the list.

  1. Click the filter bar below the column heading.

  2. In the filter field below the column heading, enter the text or value you want to filter on.
    • You can enter text or values in more than one column.
    • For date columns or numeric columns, you can enter a range of values to filter on.

    • For text columns, type the first few characters of the text you want to filter on. If you want to filter on text that is not at the beginning of the field, put asterisks on either side of the text; for example: *Hill*.

    The filter is applied as you type. The filter bar below the column heading turns green to show that you are filtering on that column.

  3. By default, the filter applies “all rules”; that is, only records that match all the filters you have filled in are shown. If you want to show records that match any of the filters, click “any rule”.

  4. If you want to look for text or values in any column, click “all columns”.

  5. To clear the filter (and show all records), click the “Cancel all filters” button.

Browse screens may also have some pre-defined filters, which you can apply from the menu.

  1. Click the arrow to the right of the menu item.
  2. Choose the filter from the popup menu.
