Your Construction Cloud Desktop

The COINS Construction Cloud workspace consists of a Desktop that contains tiles from which you can run a selection of functions and/or see information. There are also one or more other tabs (for example, SC Controller), which have menus from which you can run all the functions that are assigned to you.

Set Up Your Desktop

When you first log in to COINS Construction Cloud, your Desktop shows in Edit Mode. Using Edit Mode, you can set up the tiles on your Desktop.

To go to Edit Mode after the first time you log in:

  1. Click the User logo.
  2. Then click the Custome Desktop button.

Add a New Section

Your Desktop is organed into sections, which contain groups of tiles. You must create at least one section before you can add tiles to the desktop.

  1. Click the Add Section button on the desktop:

  2. Enter a name for the new section.

Add Tiles to a Section

The Edit Mode panel shows the functions that are available to you; these depend on the role or roles that have been assigned to you. You can arrange these as tiles on the Desktop, to suit the way you work.

  1. Click a function from the Edit Mode panel on the right and drag it onto the section:

  2. Alternatively, you can double-click a function in the Edit Mode panel to add it to the top section.
  3. You can resize a tile by clicking the edge of the tile and dragging it to the size you want.

  4. You can drag tiles between sections.
  5. When you have finished adding tiles to your desktop, click the (Close) button in the Edit Mode panel to save your changes and exit Edit Mode.

Remove Tiles from the Desktop

You can remove a tile from the Desktop.

  1. Click the tile and drag it off the desktop.

    The Edit Mode panel opens automatically.

  2. Drag the tile onto the edit mode panel.

    The tile changes to show a red outline.

  3. Drop the tile onto the Edit Mode panel.
