Publish Order to SC Controller

 

This is achieved by committing SC Order for ‘onboarded’ Subcontractor. Once an Order is Published it is visible on the All SC Orders Screen:

 

User can right click on an SC Order line and the following options will appear:

Assign to Me – Assigns the Order to you so it appears in your ‘My SC Orders’ grid so it can be worked on.

Assign to allow users to assign to another SC controller user:

 

View allows users to view the SC Order Details held at this point:

 

The SC Order Statuses should be as follows:

• Unenhanced and Enhanced Orders – New

• Once Order has been ‘shared’ with Subcontractor then - Active

• Once Order Value has been fully applied against and or OA deems the Order complete status - Closed

• If Order is Cancelled in OA then this status is published from OA to SC Portal then status - Cancelled

Enhance – Ability to add details to the Order including Contract Type and Valuation Dates.
