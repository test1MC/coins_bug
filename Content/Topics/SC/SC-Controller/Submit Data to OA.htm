Submit Data to OA

OA user and password are required to call webservice in OA to submit data from Construction Cloud. Required webservices need to be licensed with a role so if the user is assigned a named role type user and the specific webservice role(s) ticked then that is all they will be able to run.

See user CAS as an example on DevMain and DevStage.

There were created two web services: Create Certificate Batch SCESB001 and Create Draft Certificates (SCESB006).

The web service definitions are:

  • https://ukazux01.coinscloud.com:4433/env/devmainws/wouesb.p?service=SCESB001
  • https://ukazux01.coinscloud.com:4433/env/devmainws/wouesb.p?service=SCESB006

The web service is called using JWS in CAS System:
https://cas-alpha-docs.azurewebsites.net/JsonWebService/
