Subcontractor Maintenance - Analysis Tab

The Analysis tab enables you to select a way of grouping records together as an alternative way of sorting and filtering for Subcontractors.

These can be set and defined by the company at the configuration stage. In our training environment we have the analysis sets to just looking at three fields.

  • Select the Dropdown button on the Type of Subcontractor field. Select the option required, i.e. labour only, supply and Fit.
  • Select the Dropdown button on the Trade of Subcontractor field. Select the main trade of the subcontractor from the list.
  • Select the Dropdown button on the Payment Frequency field. Select the option required from the list.

There could be other fields available to analysis. Enter all the information in the relevant fields.

When all the information has been entered select the Warnings tab.
