Subcontractor Maintenance - Main tab

The information in the Main tab will have all the data that you originally entered from the Company Information Workbench. However there are additional fields to enter.

The full details of the Subcontractor record will be displayed to enable you to add the details of the Subcontract Ledger account.

Enter the Subcontract Payment details in the relevant fields. The main fields are listed in the table below.

 

Fields

Description

Default Print Method:

Select the default method by which SC certificates and remittances for this subcontractor are "printed".

The valid options for the default printing method are blank, EMAIL, FAX, and PRINT.

Company Reference

The subcontractor’s company reference number

Remarks:

Any remarks that you wish to be added on the Subcontract record. These will be shown in enquiries and are free format. Max 200 Characters.

When you have entered all the information select the Payment Details tab.
