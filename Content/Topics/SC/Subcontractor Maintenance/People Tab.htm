People Tab

Subcontractor People allows you to record the employees that work for the subcontractor.

This links to Human Resources. The lookup shows employees with a payroll type of Subcontractor, and if you add a new subcontractor employee here, this will create an employee in Human Resources with a payroll type of Subcontractor.

Field

Description

Safety Card ID

An ID that allows you to identify people uniquely - like Safepass or CSCS card number - so you can search to see if there is an existing record for this person.

If nobody is found with the ID you can use the lookup to search based on name and other information, and if they aren't on the system yet, then add a new record.

People change employers; linking to an existing record will give you a continuous history of their training and other experience, so it can save money by reducing the amount of training and inductions that are repeated.
