Insurance Tab - Adding a New Scheme

To record details of an Insurance Policy or other scheme against the Subcontractor Account, click on the Insurance Tab.

The system will allow you to select the type of scheme to be recorded

Choose the appropriate scheme to add and click

Field

Description

Expiry Date

The expiry date of the scheme.

Policy Number

The scheme number.

Provider

The name of the insurance provider if applicable.

Name on Policy

The actual name to be found on the policy document (if different from the supplier / subcontractor).

Reference

The reference number

Amount

The policy limit (for example, the limit of liability). This is for your information only.

Start Date

The scheme start date.

Notes

Notes for this scheme.

Validate

Whether to check this policy for this subcontractor.

This also determines whether the record displays on the defaulters report.

Validation is set up in Subcontract Ledger > Configuration > Insurance / Schemes.

Order Entry

This describes the validation that will occur for the selected scheme when an order is entered.

The date used for validation is the order entry date.

Order Commit

This describes the validation that will occur for the selected scheme when an order is committed.

The date used for validation is the certificate commitment date.

Certificate Entry

This describes the validation that will occur for the selected scheme when a certificate is entered.

The date used for validation is the certificate entry date.

Payment Due

The validation that will occur for the selected scheme when a payment due date is entered.

Payment

The validation that will occur for the selected scheme when a payment run is performed.

The date used for validation is the payment run date.

Ignore If Labour Only

If checked, Do not validate this scheme if the subcontract is marked as 'Labour Only'.

If blank, Validate this scheme even for labour only subcontracts.

Scheme Status

The Status of the scheme.

Mandatory - The scheme applies to all subcontractors (and will be added automatically for new subcontractors) and cannot be removed.

If the SC parameter INSMAND is set to Y then expiry date, policy number, and provider must be entered.

Default - The scheme applies to all subcontractors (and will be added automatically for new subcontractors). The record can be removed for individual subcontractors.

Blank - The scheme is added manually to individual subcontractors.

Address

Postcode

The subcontractor's address.

Contact

The name of the person to contact for enquiries.

To store the details, click .

The details will now be shown on the Insurance Tab
