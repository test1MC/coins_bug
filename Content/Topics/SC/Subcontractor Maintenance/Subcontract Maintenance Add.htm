Subcontractor Maintenance - Warnings Tab

Subcontractor Warnings shows warnings based on the current status of the subcontractor.

 

Any warning messages will be displayed on the screen along with the number of times the warnings have been made.

This screen shows a warning if:

  • There is a problem with the subcontractor's tax status.
  • Outstanding authenticated receipts exist for the subcontractor.
  • The subcontractor is dormant.
  • Unallocated payments exist.
  • The subcontractor is on stop.
  • One or more configurable scheme (insurance) checks has failed.
  • There are subcontracts with currency different to the subcontractor's.

If there are any warnings present, a red notification is also shown in the context area at the top of the screen, you can see that there are warnings even if you are not looking at the Warning tab.

Click the Refresh the Regenerate button to update the list. You cannot add any messages to this list, as these are generated elsewhere within
