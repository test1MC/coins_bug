Subcontractor Maintenance - Contacts Tab

This tab allows you to enter details of contacts at the subcontractors company.

These fields are for information purposes and are not mandatory. However if you have the information it is always good practice to enter the details.

 

Fields

Description

Enquires Contact:

The main contact for enquires, references and quotes.

Directors:

The main company directors.

Senior Officers:

The name of the manager etc.

Administration:

The number of administration staff.

Technical Staff:

Technical staff relate to IT technical staff.

Site (Dir):

The number of staff employed on the site.

Site (Sub)

The number of site contract staff on the site.

Enter the details in the relevant fields and select the Unions, CITB, H&S tab.
