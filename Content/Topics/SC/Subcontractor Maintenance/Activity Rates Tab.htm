Activity Rates Tab

This tab allows you to set up the subcontractor's rates of pay for different activities.

The rate codes are set up using Subcontract Timecard Activities.

Field

Description

Contract

The contract (and phase, if appropriate) to which the rates of pay apply; blank means the rate applies to all contracts.

Activity

The code for the activity to which the rates of pay apply.

Unit

The unit this activity is measured in. This is a memorandum field and can be any text.

Analysis

The analysis code for this activity.

When you select the activity, fills in the default analysis code for this activity. If a different analysis is appropriate for this subcontractor, you can change it here.
