Import/Export Definition

Location: Subcontract Ledger > Configuration > Export Definition

The Import/ Export routines allow certificates to be moved between Companies or COINS environments.

Click the hyperlink of the certificate to be exported

displays the document definition data in the Document Data field.

Select all the text in the field (click in the field and press Ctrl+A).

Copy the selected text and paste it into a file or an email message.

You can then transfer this data to a different environment.

To import document definitions:

Location : Subcontract Ledger > Configuration > Import Definition

Paste the text in the Report Data field and click .

The imported document definitions will now be available in Standard Certificate Definition.
