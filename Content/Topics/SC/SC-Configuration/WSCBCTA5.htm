Column Descriptions

Location: Subcontract Ledger > Configuration > Column Descriptions

Column Descriptions allows you to set up column headings that are used in the Subcontract Summary Report, the Subcontract Summary Report (Short Version), and the Certificate Posting Report. The headings correspond to the Item IDs in the certificate definition. After you have defined the subcontractor certificate, set up the column descriptions.

Field Description
Code The item ID from the certificate.
Description A description to explain what the code means.
