Completing the Payment Workbook

There are some fundamental dos and don’ts to adhere to when completing the workbook:

  • Never add, delete or alter the sequence of the tabs in the workbook.
  • Never use formulas on any of the input cells (it may look like an excel format but it is an xml file. Any formulas will result in a failure in the upload process).
  • Ditto no comas, full stops / (use un-broken text).
  • Always insert numbers to a maximum of two decimal places.
  • Heed the warning & error messages
  • Insert data into the yellow highlighted cells
  • Ensure when adding variations that you always insert a cost code and that data to all cells in the rows are completed.
  • You can only pay up to a maximum of the order value against the CSA.
  • All variations and extras must be paid via the recoverable or non-recoverable tabs.
  • The workbook consists of a number of tabs:
    • Payment
    • Bill of Quants
    • Payment Notice
    • Materials on site
    • Contra charges
    • Add Recoverable
    • Recoverable history
    • Add non-recoverable
    • Non-recoverable history

When the payment workbook has been completed the workbook needs to be saved to the regions shared drive in the folder designated for S/C payments ready for upload. Remember -

 

DO NOT CHANGE THE FILE NAME & ENSURE THE FILE IS SAVED AS AN XML SPREADSHEET

Produce a hard copy payment certificate, back-up documentation and proceed with the authorisation procedure and sign-off.

Payments will not be uploaded until the authorisation sign-off has been achieved.
