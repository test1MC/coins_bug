Types Tab

The Company Types is available to enable you to search on types of companies. Select the company types that are associated with the company, as you can allocate more than one. For example the company could be a supplier of materials, a subcontractor or a company.

Select the box adjacent to all the required company types. A company must have at least one type (for example, supplier).

If you select the Save button without selecting a company type, a message is displayed stating “You must select a Company Type”.  Select the OK button. You will return to the Company Add screen. Select the Types Tabs to allocate the company types.
