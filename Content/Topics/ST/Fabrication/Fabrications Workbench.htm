Fabrications Workbench

The Fabrications Workbench is a new workbench, similar to the Workbench, but which shows only fabrications.

Fabrications Workbench

Several fields that are not required for fabrications have been removed, and new fields have been added.

Fabrication Entry

Fields

Field
Description

Item

The item that will be made using this fabrication, if you are making it for your own . The item must be set up beforehand.

Quantity

The quantity of the item that will be made.

Fixed Price

If the item is not being made for your own , this is the fixed price that you will charge for the item.

Completion

The fabrication can be completed either while updating the , or by using the Complete action on the Fabrication browse. Completing the creates a transaction batch which updates the balance for the designated item, by creating a negative issue. This posts automatically.
