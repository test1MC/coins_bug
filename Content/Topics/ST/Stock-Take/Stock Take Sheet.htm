Sheet

 

The Sheet is provided to check-off the items against.

The report is initially generated by the Batch routine. It lists quantity on hand at each location, and then shows adjustments made to the quantities through the > Input > Entry routine.

When creating the sheet in , options to format it are presented.

To create Sheet select it from the ‘Input’ menu folder.

 

 

Click the link of the batch number to access the Sheet Option

 

 

Include On-Hand ? allows the Yardsperson to view the balances within on the Sheet. Typically, this is uned for an accurate count with no indication given to the Yardsperson as to how much is in the yard.

In addition the Sort order can be decided. Bin Number (if used) can be selected or the code.

Click to generate the Sheet.

The Sheet acts like any other report in in that it is output to the Report Status screen - allowing a PDF or spreadsheet version to be obtained.

The Excel version is more versatile as you can manipulate the output (adding columns etc).
