To apply adjustments go to the Entry page in the ‘Input’ folder.

 

It is important to be aware that the batch is incomplete and unposted until adjustments are confirmed.

Please note - Unwanted batches such as those created in error should still be posted/completed (even if no adjustments are needed) as unposted transactions can cause problems at month-end.

Filters

A number of filters are available to enable various searches to be made on the Sheet, for example the Bin number (if used) or the item code or the item description.

Other filters can help whilst making the adjustments to categorise the results – for example there is a filter to show all items that have been changed to show more items than expected (highlighted above and below). To activate these filters select one and click on the ‘Apply Filler’ button.

 

 

Zero Count – This filter shows items with zero quantity

Not As Expected - This filter shows items that have been adjusted to be more or less.

Less Than Expected - This filter shows items that have been adjusted to be less than expected.

More Than Expected - This filter shows items that have been adjusted to be more than expected.
