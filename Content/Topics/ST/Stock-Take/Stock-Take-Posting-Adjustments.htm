Posting Adjustments

 

When the is completed, whether adjustments were needed or not, or whether amendments were done line by line or by bulk methods, the batch will need posting. The posting can be done from either the main screen or within the batch itself.

From the main screen select the batch ( the box) select Post from the action menu and click the Apply Action button

A Posting Report will be generated to confirm the Batch Posting, or to highlight reasons why the batch could not be posted.

A successful posting will change the Batch status to "Posted" and the relevant financial implications will be automatically updated.
