Assembly Parameter

The following new parameter specifies the transaction type to use for receipts of assembled (or dismantled) items. This can be used to adjust levels if you assemble items – for example, stop signs – from component parts and hold the assembled items in stock, or if you split a box of items to issue the items individually.

Parameter

Description
ASMBTTYP

The transaction type to use for receipt of assembled items.

When the transaction type is used you enter the codes for the assembled items to receipt, and posting will automatically create a negative receipt for the parts used to assemble. To split assembled items, you enter a negative quantity.
