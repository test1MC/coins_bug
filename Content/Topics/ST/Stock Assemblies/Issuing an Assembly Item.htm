Issuing an Assembly Item

In issue transaction, enter the assembly item code and quantity required.

When you save the transaction, the component parts required are all calculated and added automatically.

Assembly Item Transaction

ed indicators differentiate assembly items (green) and their component parts (red). The cost analysis for the parts is copied from the assembly item.

If there is a failure in the calculation or if there are insufficient parts an error is reported.

Insufficient Parts Error

You can delete individual parts, or update the quantity. If you delete the assembly item, this will delete all of the component parts, and if you change the quantity of the assembly item, this will re-calculate the required components.
