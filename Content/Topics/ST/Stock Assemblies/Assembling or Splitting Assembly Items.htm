Assembling or Splitting Assembly Items

If you assemble item from its component parts and hold it in stock, or if you split a box of items so that you can issue individual items separately, you can adjust the levels by entering a receipt transaction.

Enter receipt transaction using the transaction type specified by the ST parameter ASMBTTYP, and enter the code of the assembly item.

  • For assembled items, enter a positive quantity; the posting will automatically create a negative receipt for the parts used in the assembly.
  • For split items, enter a negative quantity for the assembly item (for example, the box); the posting will create a positive receipt for the separate items.
