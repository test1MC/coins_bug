Model Specific Items and Styles: Export/Import Setup

This section includes information about functions found on the >  Setup menu that were modified in v11.09 to allow the use of Model specific Items and Styles.

Export Setup

If parameter CV/ITEMMODL is set to Y, the Export Setup function in > Setup selects Model specific Items and Styles for export. Otherwise, it only works with Company specific Items and Styles in the Model and disregards any Model specific Items/Styles.

Import Setup

If parameter CV/ITEMMODL is set to Y, the Import Setup function in > Setup assigns a Model code to the Items and Styles.
