Manual Adjustments

When creating a new by copying a previous one, manually entered data can be copied to New or Reset if the Reset Item when Copying ? field is ed in Items properties. This was extended to allow a Copy/Reset conditionally using the value of another Item. This section describes what has been done in  OA to accommodate changes made for this.
