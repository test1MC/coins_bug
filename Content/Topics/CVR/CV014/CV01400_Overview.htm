Overview

This document covers a A number of changes that have been were made in the module for the v11.06 release. These changes include:

  1. Decimal Items were stored using 2 decimal digits precision. This precision has now been increased to 6 decimal digits. It is therefore possible to configure precision on a Item. See Section 1, Decimal [%=Variables.CVR%] Item Changes. (PDR24417)
  2. When creating a new by copying a previous one, manually entered data can be copied to New or Reset if the Reset Item when Copying ? field is ed in Item properties. This has now been extended to allow a Copy/Reset conditionally using the value of another Item. See Section 1, [%=Variables.CVR%] Manual Adjustment Changes. (PDR24418)
  3. Browse Views previously could only be filtered and sorted using source table fields (e.g., Cost Code Section, WBS Code, etc.). A new feature now allows filtering and sorting by Items. See Section 1, [%=Variables.CVR%] Filtering and Sorting Changes. (PDR24419)
