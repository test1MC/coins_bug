Subcontract Package Items

New Subcontract Package items are available in the Value field in > Setup>  Items, as shown in Figure 1. You can use these options to set up Subcontract Package information you would like to capture and display in the Workbench. These options are:

  • Subcontract Package Calculation
  • Subcontract Package Manual Input
  • Subcontract Package RSP Field

These options are also listed as named filter items on the Items Workbench, as shown in Figure 2.
