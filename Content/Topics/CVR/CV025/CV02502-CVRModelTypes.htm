Model Types

The following new view options are available on the Type field under the Views tab in > Setup > Models:

  • Subcontract Packages Browse - see Section 1.2.1 for details
  • Subcontract Packages Forecast - see Section 1.2.2 for details

Subcontract Packages Browse

When you select Subcontract Packages Browse from the Type field, an Add screen appears, as shown in Figure 3. You can use this screen to set up the browse.

Subcontract Packages Browse Add Screen

The Non-Zero Items lookup only shows Items of the Subcontract Package type.
