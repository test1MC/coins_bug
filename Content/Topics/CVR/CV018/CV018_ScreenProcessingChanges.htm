Cost Code Report

The new Cost Code Report in the module generates detailed Cost Analysis or Earned Value cost code views of the for a selected Period.

Menu Item

If parameter CV/USEBPFXT is set to Y for the current company, Cost Code Report now appears as a menu item on the > Reports menu, as shown in Figure 1.

Menu Item

Report Selection Screen

When you select Cost Code Report from the Reports Menu (see Figure 1), a report selection screen appears with the Main tab as the default screen, as shown in Figure 2. You can use the Main tab to select the Period and Field Set you wish to show on the report. The selection screen also contains tabs that allow you to filter and sort the and Cost Code information you want to show on the report.

Selection Screen

Field Definitions

Period

The period you wish to report on. This defaults in with the current Period, but can be changed.

Field Set

Select either Cost Analysis or Earned Value as the cost code view to be shown on the report for the selected Period.

Sample Reports

Based on the Field Set you choose on the report selection screen (Figure 2), a report is generated with either Cost Analysis or Earned Value as the cost code view for the selected Period.

Cost Analysis Report

When you select Cost Analysis as the Field Set on the report selection screen, the report is shown in Cost Analysis view, as shown in Figure 3.

Earned Value Report
