Configuration

A new CV parameter lets you decide whether or not to record the history audit of s.

Parameter

Description

CV/CVRAUDIT

Use History Audit

Whether to record history audit of s (Y or N).

Workbench

s Browse

If new parameter CV/CVRAUDIT is set to Y, a History button link appears on the s screen to the left of the Status column, as shown in Figure 1. If clicked, a new History browse screen opens, showing a detailed history of the selected (see Section 2.2 for details).

This button link also provides you with the ability to attach and review event specific PDF reports and log files.
