Items Types

The following new types of Items are available in the Items browse, as shown in Figure 1:

  • Item Calculation
  • Item Manual Input
  • Item RSP Field

A Items named filter is also available on this browse.

Item Types and Filter

When you select the Add button on the Items browse for any of these three Item Types, a properties screen appears (it is the same for all three types), as shown in Figure 2.

Item Properties Screen

Field Definitions (Selected Fields)

Summare By

Indicate whether you wish to summare Items by or by Series.

From/To/Matches

Indicate how you wish to filter Items by entering From/To data in the following fields:

  • Level Codes 1-5

Item Type

Select the type of item: Initial, , or Both (the default).

Row Type

Select the boxes you wish to use (all of them are ed by default).

  • Qty/Rate
  • Cost Based
  • Calculated
  • Subtotal
  • Heading

Post to Data Store?

This box is only available if the Summare by field has been selected. If ed, standard options appear in the Report Store and Value to Post fields.

Workbench

Items Browse

The following new page sections are available for the Items Browse View:

  • Item Browse
  • Enhanced Item Browse

Named filters on this browse are:

  • All, Non-Zero (as per any existing Browse page)
  • All, Initial Items, Items
  • All, Originally Entered, Amendment Items Only
  • All, Print Items

Items View

A new View is available for Items of type Item.

Named filters on this view are:

  • From, To Period
  • Item filtering criteria
  • Non-zero Amount / Amounts / Balance

Other Changes

Update Items

Update Items processing now includes type Items.

Item Snapshot

Items Snapshot processing now includes type Items.

Post Items

Post to Data Store processing now handles type items for summaration by . Also, the posting report now shows Items.
