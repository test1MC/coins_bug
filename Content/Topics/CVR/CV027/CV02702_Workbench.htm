A Cost Code / Category Forecast View is generated on a base of fields defined for forecast view and filtering criteria provided in Cost Code / Category Forecast View properties.

Cost Code / Category Forecast View Filters

The filter on the Cost Code / Category Forecast View now has additional filter criteria. In a forecast spread view, you can get to that criteria by clicking the Filter button in the upper left of the screen. The filter then pops up, as shown in Cost Code / Category Forecast View Filters.

Update Items

Update Items can now provide Forecast values for Cost Code / Category Items.

  • If a Cost Code / Category Forecast Item is selected for update, forecast values of that item are now recalculated.
  • Category Split Summary forecast values are recalculated and stored at the and Category levels.
  • The processing of Cost Code / Category Item forecast values affects the following:
  • Update Items, Recalculate Summary Values
  • Re-Spread forecast Values (as a part of Update)
  • Generate Forecast Spread
  • Roll Up Spread to Item
  • Process Snapshot data

Generate (New)/(Copy Previous)

When a new is generated (via the Copy previous adjustments option), previous adjustments are copied as part of the initial Update processing.

When an existing is copied using the Workbench, manually entered data are copied from the original to a new .

Print

Print now supports custom query filtering in Forecast Grid Views and the new Code / Category Forecast Grid.
