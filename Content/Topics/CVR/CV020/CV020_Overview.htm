Overview

Previously, the authoration process in the module could only be controlled by using a Release for Authoration browse action. To provide another option for this process, you can now use a new CV parameter (WFAUTRLS), which controls whether to automatically release a completed for Authoration workflow.

In addition, this process has been extended for the following actions: Update Items, Lock/Unlock , Generate Spreadsheet, Set As Complete/Incomplete, Post Budgets and Memo Data, and Create GL Batch.

This document discusses the changes that have been made to provide this alternative processing for authoration.
