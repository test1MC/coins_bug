Screen and Processing Changes

Project Status

The Project Status function has been removed from the Project Management > Global Setup tree menu. It has been reassigned as an MK Project function under the Setup > Project > Project Setup tree menu, as shown in Figure 1: Project Setup Tree Menu.

Companies Tab

A new Project Companies tab has been added to the Project Team function in Project Management > Selected Project > Project Setup, as shown in Figure 2: Project Companies Tab.

Project Companies Tab

This tab allows you to browse companies associated with the Project.

Contacts Tab

Although the Contacts tab in Project Team is a PT function, it contains hyperlinks to MK functions. For example, when you click the Contact Name link in this PT function (Figure 3: Contact Name in PT Function), you are taken to an MK function screen (see Figure 4: MK Function Screen).

People Setup

Changes that have been made in Project Management > Global Setup > People Setup are described in this section.

Job Group Function

A new Job Group function has been added to the People Setup menu, as shown in Figure 8: Job Group Menu Item. It is an MK function that allows you to set up the Job Groups and the Job Roles that belong to them.

Job Group Menu Item

People Setup Functions

The following functions on the People Setup menu have been changed from SY to MK functions:

  • Titles
  • Qualifications
  • Membership
  • Skills
  • Experience Categories

Personnel

The Personnel function in Project Management has been changed from an SY module to an MK module function.
