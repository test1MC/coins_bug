Screen and Processing Changes

Print [PT Type] Action

The Print [PT Type] action is available on the following PT browses: RFIs, Submittals, Transmittals, Meetings, Daily Reports, Drawing Sets, , SC , and Material Tracker. When selected, the screen shown in Figure 1: Print [PT Type] Screen is displayed, showing all the documents linked to the PT Type that are available to be emailed.

Print [PT Type] Screen

Previously, the Available list on the Print [PT Type] screen could only show documents attached to the PT record and documents attached to the current PT Item. This has been changed so that the Available list can show all of the following documents:

  • Documents attached to a Project Management record.
  • Documents attached to the current Project Management Item.
  • Documents attached to the record (when a Project is associated with a ).
  • Documents attached to the linked records of a Project Management Item.

The file names in the Available list are shown in the following format:

<PT Type> - <Reference> (<File Name> [File Size])

Example: SUBMITTAL-SUB0001-Submittal (Koala.jpg [.74MB])

The PT Type can be RFI, Submittal, Transmittal, Meeting, or Daily Report.

PT Items Linked to Submittal Packages and Drawing Sets

PT Items can be linked to Submittal Packages and Drawing Sets. The Available list that shows linked documents when printing a PT Item linked to a Submittal Package/Drawing Set can be any of the following:

  • Documents attached to a Project Management record.
  • Documents attached to the current Project Management Item.
  • Documents attached to the record (when a Project is associated with a ).
  • Documents attached to the linked records of a Project Management Item.
  • Documents attached to Submittal records (when a PT Item is linked to Submittal Packages).
  • Documents attached to Drawing records (when a PT Item is linked to Drawing Sets).

Email [PT Type] Documents

The Email [PT Type] Documents action now exists on the following browses: RFIs, RFI Revisions, Submittals, Submittal Packages, Transmittals, Meetings, Issues, Daily Reports, and Drawings. When selected, the screen shown in Figure 2: Email [PT Type] Documents Screen is displayed, showing all the documents linked to the PT Type that are available to be emailed.

Document Drop-Down

CONS OA did not allow users to view all the documents linked to Material Tracker, , and Subcontract . To resolve this, a Linked Documents tab is now available on the Material Tracker and Summary screens. A Documents drop-down has also been added to the Subcontract Summary screen. These changes are discussed in more detail below.

Material Tracker

The Documents drop-down on the Material Tracker Summary screen now includes a Linked Documents option, as shown in Figure 3: Material Tracker Summary Screen. This drop-down option can show any of the following documents:

  • Documents attached to the Project Management record.
  • Documents attached to the current Material Tracker record.
  • Documents attached to the record.
  • Documents attached to the linked records of the Material Tracker record.

Material Tracker Summary Screen

Subcontract

A Documents drop-down tab is now available on the Subcontract Summary screen, as shown in Figure 5: Subcontract [%=Variables.Sv Variations caps%] Summary Screen. This drop-down has the following three tabs:

  • Browse Documents – Displays all the documents attached to the SC .
  • Linked Documents – Displays all the documents attached to linked records, SC , , , and Project Management records.
  • Upload Documents - Allows you to upload documents for the SC .
