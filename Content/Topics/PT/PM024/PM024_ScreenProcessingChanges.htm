A new Item Types field has been added to the Transmittal Details layout in Project Management > [working ] > Communication Management > Transmittals tab, as shown in Figure 1: Item Types Field. This field allows you to any types of items you would like to transmit.

Transmittal Print

Transmittal Print uses the OA Document Designer in System > System Setup. This form previously contained a Comment Header, Comments, Item Header, and Items fields. The Item Header and Items fields have been replaced with Reasons Body and Reasons Header, as shown in Figure 4: Transmittal Print Screen.
