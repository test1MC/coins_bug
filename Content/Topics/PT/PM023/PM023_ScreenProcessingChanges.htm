Screen and Processing Changes

Daily Reports

A Ref column is now available on the PT > Communication Management > Daily Reports tab, as shown in Figure 1: Ref Column. This field, which represents the date and sequence of a report, is automatically generated, not updatable, and mandatory.

Ref Column

Multiple Child Reports

Since you can now have multiple Daily Reports for one day, you can also now have multiple sets of child reports on the same day for each Daily Report.

Daily Reports can have multiple child reports for the following categories:

  • Subcontract
  • Materials

Daily Report Details Context Area

The following two fields were added to the context area of the Daily Report Details screen, as shown in Figure 2: New Context Area Fields:

  • Reference
  • Reported By

Print Daily Report
