Production Tracking Functionality: Administration

Import WP / Tasks

The Import WP / Tasks function is found on the Administration menu in Project Management > [Working Project] > Production Tracking. It allows you to import Production Tracking Work Packages and Tasks into . A sample screen is shown in Figure 1: Import WP / Tasks Screen.

IMPORTANT: You cannot use any spreadsheets that have been exported via the Export Work Packages or Export Tasks functions, which are described in Section 0.1 Production Tracking Functionality: Work Packages/Tasks/Actions.

Import WP / Tasks Screen

Field Definitions

Import Type

Select Work Packages or Tasks as the type of import to be performed.

Spreadsheet File

Select the spreadsheet to be used for import.

Import Contains Headers?

If this option is selected, the system imports the headers (first text line) of the import file.

Validate Only?

Whether you wish to import the file for validation mode only.

Auto Create Work Packages for new Tasks

This field only appears if Tasks is selected above as the Import Type. It allows you determine whether you wish to automatically create Work Packages for new Tasks.

Error Handling

Select how you wish to handle errors during import: Load if no errors or Load all records without errors.
