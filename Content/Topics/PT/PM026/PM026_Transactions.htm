Production Tracking Functionality: Transactions

Production Tracking contains two separate Transaction functions in Project Management > [Working Project] > Production Tracking: Transactions and Cost Code Transactions. Both of these functions are discussed below.

Transactions

The Transactions view-only function in Project Management > [Working Project] > Production Tracking allows you to view the underlying incremental transactions created by Task and Action functions (Figure 1: Transactions Screen). Some of these transactions may update the Status quantities.

Transactions Screen

Cost Code Transactions

The Cost Code Transactions view-only function in Project Management > [Working Project] > Production Tracking allows you to view the underlying Cost Code transactions created by Task and Action functions. A sample screen is shown in Figure 2: Cost Code Transactions Screen.
