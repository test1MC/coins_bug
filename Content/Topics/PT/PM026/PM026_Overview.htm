Production Tracking Functionality: Overview

New Production Tracking functionality is found in the Project Management module. It allows contractors to track the progress of detailed sets of predefined tasks by recording installed quantities against the tasks. Tasks can be grouped together as Work Packages, and progress can be viewed at Work Package level. By assigning a Cost Code to each task, the Cost Code’s percentage complete can be calculated from the task quantities.

Production Tracking allows Project Managers to categore tasks differently than Accounting Cost Codes.

  • Tasks are the detail items that need to be performed/installed at the site (such as Pipe, Pipe Hanger, Valve, etc.). Each task has a defined quantity, hours per quantity, and total hours to install.
  • Tasks are grouped together in a Work Package. Work Packages can have a quantity but are only for reference.
  • Actions are the recording of the installed quantity in the field. These can also be scheduled install values, which can then convert into actual installed values.

This document explains the changes that have been made to provide Production Tracking functionality within  OA.
