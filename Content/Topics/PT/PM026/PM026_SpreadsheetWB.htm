Production Tracking Functionality: Spreadsheet Workbench

In order to make data entry easier for some projects, installed quantities can now be entered and loaded into  via a new Spreadsheet Workbench function in Project Management > [Working Project] > Production Tracking.

This Workbench is divided into three separate functions: Generate Spreadsheet, Download Spreadsheet, and Upload Spreadsheet. Below are the steps you would perform to load quantities into a spreadsheet by using these functions.

  1. Define a template Spreadsheet (.xml format) in Microsoft® Excel and load it into  OA.
  2. When you are ready to enter the installed quantities, use Generate Spreadsheet to generate a spreadsheet (using the appropriate template you created previously). See Section 0.1.1 Generate Spreadsheet for more details.
  3. Use Download Spreadsheet to download the spreadsheet with installed quantities to your computer. See Section 0.1.2 Download Spreadsheet for more details.
  4. When the spreadsheet is ready to be uploaded, use Upload Spreadsheet to load the data into OA. See Section 0.1.3 Upload Spreadsheet for more details.
  5. After the data is validated, it is loaded directly into the Project.

Generate Spreadsheet

The Generate Spreadsheet function (Figure 1: Generate Spreadsheet Screen) generates a spreadsheet with data based on the selection criteria you enter. The selection screen contains a Template field where you can select the spreadsheet template (created in Excel and uploaded to OA) to use for the generated spreadsheet. Once generated, the new spreadsheet is placed as an entry in the Download Spreadsheet function, where you can then access it.

Download Spreadsheet

The Download Spreadsheet function (Figure 2: Download Spreadsheet) allows you to easily find and download spreadsheets that have been generated. It lists these spreadsheets for the Project in newest order first.

  • To download the spreadsheet, click the Spreadsheet link.
  • If a spreadsheet has been downloaded and subsequently uploaded, you are warned that the spreadsheet was previously uploaded.
