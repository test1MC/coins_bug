Production Tracking Functionality: Setup Maintenance

Constraints

Constraints Screen

The Constraints function is found in Project Management > [Working Project] > Production Tracking > Setup Maintenance, as shown in Figure 1: Constraints Screen. It allows you to define Constraints specific to the Project.

Constraints Screen

Merge Into Action

The Merge into action is found at the bottom of the Constraints screen and can be used to merge together Constraints that have been used. When you click this action, a Merge into popup screen appears, as shown in Figure 2: Constraints Merge Into Screen.

As explained in the message at the top of this screen, the data merge is done in a background report. Also, the current Constraints function will close after you complete this screen, and you should wait until the background report function has finished before returning to the Constraints function.

Constraints Merge Into Screen

Responsible Party Screen

The Responsible Party function is found in Project Management > [Working Project] > Production Tracking > Setup Maintenance, as shown in Figure 3: Responsible Party Screen. It allows you to define a list of contacts linked to the Project.

Search Fields

The Search Fields function is found in Project Management > [Working Project] > Production Tracking > Setup Maintenance, as shown in Figure 5: Search Fields Screen. It allows you to define the Project specific Search Field labels to be associated with the Work Package and Task of a Project.

Search Fields Screen

Merge Into Action

The Merge into action is found at the bottom of the Search Fields screen and can be used to merge together Search Fields that have been used. When you click this action, a Merge into popup screen appears.

The data merge is done in a background report. Also, the current Search Fields function will close after you complete this screen, and you should wait until the background report function has finished before returning to the Search Fields function.

Upload Template

The Upload Template function is found in Project Management > [Working Project] > Production Tracking > Setup Maintenance, as shown in Figure 6: Upload Template. It allows you to upload a template spreadsheet to the correct location in (the $BASE/ var/ spreadsheets/ template directory).

Upload Template

XML Templates

The XML Templates function is found in Project Management > [Working Project] > Production Tracking > Setup Maintenance, as shown in Figure 7: Excel XML Templates Screen. It allows you to create and maintain Excel XML templates for Production Tracking.
