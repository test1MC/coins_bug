Production Tracking Functionality: Screen and Processing Changes

Production Tracking Menu

A Production Tracking menu is available under the Project Management (PT) menu, as shown in Figure 1: Production Tracking Menu. This menu appears once a Working Project is selected from the PT menu.

Production Tracking Menu

Select Working Project

Menu Item

The Select Working Project function on the Project Management menu lets you select a Working Project. The Production Tracking menu (see Figure 2: Select Working Project) is now found within any Working Project that you select from the Select Working Project screen.

Select Working Project

Screen

When you click Select Working Project from the menu, the screen shown in Figure 3: Select Working Project Screen appears.

The following changes were made to this function in order to accommodate Production Tracking:

  • You can add Projects to the My Projects list via the Add to My Projects action at the bottom of the screen.
  • The last selected Project is used as the default, thus allowing you to bypass the screen and not requiring you to select it each time you access Production Tracking.
  • Project security is in place and allows you to limit the Projects that users can see. This security allows group and subgroup values to be assigned to the Project and users. security is also applied if the Project has turned into a .

The linked ’s security access only comes into play when you attempt to assign a /Cost Code to a Task. If you do not have rights to a linked , you cannot assign it to a Task. In addition, if you attempt to enter an installed quantity for a Task that has a for which you do not have rights, the system does not allow the quantity to be saved.
