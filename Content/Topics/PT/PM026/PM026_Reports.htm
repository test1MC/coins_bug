Production Tracking Functionality: Reports

Report Overview

There are a number of Production Tracking Reports found in Project Management > [Working Project] > Production Tracking > Reports. A description of each of these reports is given below.

  • Production Progress Report - This report provides Cost Code level detail data by defined Groups.
  • Production Progress History Report - This report provides Task level detail data.
  • Production Worksheet Report - This is a report in spreadsheet format that provides Task level detail and is used to fill in installed quantities.
  • Task Hours and Quantity Report - This is a report in spreadsheet format that provides Task Level dates by defined Groups.
  • Commitment Statistics Report - This is a report in spreadsheet format that provides Date level data by defined Groups.
  • Commitment Report - This report provides commitment data by defined Groups.
  • Task Listing - This is a report in spreadsheet format that provides Action (scheduled) level data by defined Groups.

Report Selection Screen

In general, all of the Production Tracking reports listed above have the same basic selection screen, as shown in Figure 1: Main Report Selection Screen. Although most fields are shared by all reports, there are also some fields specific to only certain reports. All fields (basic and report-specific) are defined below.

Main Report Selection Screen

Standard Fields (Filters Area)

Filter From/To/Matches

The type of data you want to report on, which is selected from the Filter drop down menu (such as , Cost Code Type, etc.). You can report on the individual data or span of data.

Period From/To

The period or span of periods to report on.

Include Zero Lines

Whether to include lines that have zero values for the period.

Exclude Tasks with No Open Action

Whether to exclude tasks with no open action from the report.

Include Scheduled Actions Thru

If a date is entered here, the report shows only actions with a target date that is blank or less than/equal to this date.

Tasks Status

Select the tasks to be printed based on their status: All, Complete, or Incomplete.

Include Tasks

Select the tasks to be printed based on their actions: All, With Open Actions, or Without Open Actions.

Standard Fields (Report Design Area)

Report Group Selector (Available/Selected)

Select an available Report Group and place it in the Selected area to be included in the report.

Page Break on

Select which group(s) will be printed on a new page. Options appear in this field for each Group selected in the Report Group Selector.

Group Totals

Whether to print the Total lines for each Group on the report.

Report Totals

Whether to print the Total lines for the report.

Double Line Space

Whether to print the report with a double space between each line.

Column Selector (Available/Selected)

Select a Column from the Available area and place it in the Selected area for inclusion in the report.

Report Format

Select the Standard or Quantity field set to print on the report.

Include Actions List

Whether to include an Actions List under each Task on the report.

Include Tasks Notes

Whether to include Task Notes under the Task row on the report.

Exclude Hours

Whether to exclude Hours from the report.

Include Hours

Whether to include Hours on the report.

Plain Report

Whether to print the report in plain mode.

Banded Report

Whether to print the report in banded mode (an alternate background of white and gray rows).
