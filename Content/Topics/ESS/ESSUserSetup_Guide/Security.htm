Security

In addition to user security there is the ability to set which employees any given employee can have access to using Personnel security.


Navigate to Human Resources > Personnel Security.

A calculation is required to dictate which employees they can have access to. Examples below:

 

Access To

Code

Own record only

CAN-DO(pp_organisation.ppo_seq + ',' + pp_organisation.ppo_seq, '{sysuser.getLink^HRKEY}')

Work group ‘700’

pp_organisation.hrg_code = '700'

Work groups ‘700’ or ‘500’

pp_organisation.hrg_code = '700' OR pp_organisation.hrg_code = '500'

All apart from employee ‘123456’

pp_organisation.hrp_id <> '123456'
