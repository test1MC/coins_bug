HR Setup for Employee Self Service

This topic covers the setup and areas of a HR record that need completing to successfully utilise Employee Self Service.

User / HR link

The user record needs to be linked with the corresponding HR record for the user account to know which HR details to show. If not, the employee self service module will show blank details for that employee/user. In the HR record (Main tab) enter the user ID of the corresponding user account as shown below:


HR fields – Main tab

    System User ID

    Title

    First Name

    Surname

    National ID

    Date of Birth

 


HR fields – Address tab

    Address (line 1)

    Address (line 2)

    Address (line 3)

    Address (line 4)

    Postcode

    Country

HR fields – Organisation tab

    Director / Manager / Supervisor / Works To (At least one of these will need to be completed for use as holiday authoriser)

    Holiday Authorisers (Dictates which from the above will be sent holiday requests)

    Director / Manager / Supervisor (These tick boxes enable this employee to appear in the above hierarchical positions for other employees)
