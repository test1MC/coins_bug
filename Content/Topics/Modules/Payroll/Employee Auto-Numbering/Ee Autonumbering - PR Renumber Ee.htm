Employee Auto-Numbering - PR Renumber Employee

The Renumber Employee Utility uses the format defined in PR/EMPCODEF when generating the default new employee numbers. The user can override these if PR/EMPAUTON is not set to ALWAYS (except when reversing an earlier renumbering).

When auto-generating an employee number or checking its validity (whether auto-generated or manually entered), the utility will check for any clash with either another Payroll employee number or another HR personnel ID.

The user interface has been simplified by removing the options Add Prefix to Employee Numbers, Manually Enter New Employee Numbers and Prefix to New Employee Numbers.
