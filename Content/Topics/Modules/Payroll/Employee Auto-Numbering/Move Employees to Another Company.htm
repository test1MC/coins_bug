Move Employees to Another Company

Move Employees to Another PR Company is the US/Canadian payroll menu option for moving from one company to another. Previously this used the original employee number in the new company. It now considers the standard employee number validation rules for the new company. If the original employee number is acceptable, this will be used in the new company. If not a new number will be auto-generated.

  • In the phase I changes this auto-numbering only works when EMPAUTON is not blank (that is, when standard auto-numbering has been enabled).
  • In phase II the auto-numbering works regardless of the setting of EMPAUTON.

When auto-generating an employee number and when checking its validity, the function checks for any clash with either another Payroll employee number or another HR personnel ID.

The new employee number is shown in the Message column on the report.
