Employee Auto-Numbering - HR Parameters

The HR parameter AUTOID has been updated to accept new values, and two new parameters HRIDFRMT and HRIDUNIQUE have been added.

Parameter

Description
AUTOID

The type of auto-numbering which is enabled. The options are:

(i) <blank> – no auto-numbering (was N);

(ii) IFBLANK – will set the number if the user leaves it blank (was Y);

(iii) ALWAYS – will always set the number.

HRIDFRMT The template for the format. See 1, Employee Auto-Number Format Details.
HRIDUNIQUE A ‘Y/N’ flag field to make employee numbers unique across all companies.
