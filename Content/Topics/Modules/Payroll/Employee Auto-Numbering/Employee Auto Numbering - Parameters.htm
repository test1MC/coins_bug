Employee Auto Numbering - Parameters

The existing Payroll parameters that govern employee auto-numbering have changed.

Parameter Description
EMPAUTON

The options that specify which type of auto-numbering is enabled are now:

(i) <blank> – no auto-numbering;

(ii) IFBLANK – (was AE) will set the number if the user leaves it blank;

(iii) ALWAYS – (was AD) will always set the number, though this could be initially in HR so it will be possible to enter an existing HR personnel ID number for which the Payroll employee is still to be created.
EMPREFIX This parameter has been removed.
EMPCODEF

This is a template style field that specifies the format. See , Employee Number Format.

New employee numbers are validated against this format.
EMPUNIQUE No change. A ‘Y/N’ flag field to make employee numbers unique across all companies.
