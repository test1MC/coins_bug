Print Sample P60s – Alignment Test

Using this year’s P60 (Single Sheet) (201x-1x), print ten P60s to the printer that you intend to use for this year’s P60 print, to check alignment, page throws, and font size.

You will need to test the alignment with an actual P60 (single Sheet) (201x-1x) form and not a printed copy of the above .pdf form.

  1. Go to Payroll menu > Monthly Payroll > Maintenance > Print P60s.

  2. Select the relevant Form (and tax reference if you are using multiple tax references), and the Year you are submitting for (only the current tax year will be displayed this time around).
  3. On the Employee Selection tab, select a range of employees for test purposes, then click the button.

The report will be produced as a PDF file and will be available on the Report Status workbench.  You can then print the PDF file and confirm that it aligns with the pre-printed stationery.

  1. Go to Payroll > Period Payroll Administration > P60 Print.

  2. If you are using multiple tax references, select the relevant code, tax reference and year you are submitting for (only the current tax year will be displayed this time around).

  3. Press F1 to continue.
  4. On the Employee Selection screen, select a range of employees for test purposes, then press F1.

Print the report and confirm that it aligns withthe pre-printed stationery.
