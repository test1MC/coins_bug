Run the Tax Code Increases Procedure

NOTE

The basic personal allowance for - will be increased to £11, 850 and the tax code for emergency use will be 1185L W1 or 1185L M1 or 1185L X.

As a result there will be a general uplift of tax codes with suffix ‘L’ which will be increased by 35.

Run the Tax Code Increases procedure to increase the suffix by the appropriate amount.

The Tax Code Increases procedure is a company-wide procedure by pay frequency, so make sure all payrolls have run the end of year clearing procedure above before performing this step and selecting the required pay frequency to update.

Enter the tax increases as follows:

  1. Go to Payroll > Company Setup > Employee administration > Tax Code Increases.
  2. Go to Payroll > Period Payroll > Company Setup > Tax Code Increases.

  3. Enter the increases.

  4. When all the rates have been entered click Next to run.

  5. When all the rates have been entered, press F1 to accept and save the data.

    The report shows the old and new tax codes.
