Order New P60 Stationery

Order P60 stationery together with any other stationery that you require. Refer to HM Revenue & Customs forms Order line. This should be available as a contingency in case HMRC has any technical issues during submissions.

NOTE

HMRC have not yet released details of the new format for the P60 so please refer to their website for any updated details. We expect that the only change will be that the ASPP field will be replaced with the ShPP field. (16 Feb 2016)

The P60 format we support is the P60 (Single Sheet) (201x-1x). http://www.hmrc.gov.uk/ebu/p60snglsheet.pdf

This is a single sheet laser form and you will require a laser printer.
