P60 Setup

To set up the new P60 form for printing:

  1. Download the file p60-laser.txt from the website client area.
  2. Go to OA Reporting & BI > Document Designer > Import Document Definition.
  3. Copy and paste the content from the p60-laser.txt file, then save.

  4. Go to OA Reporting & BI Setups > Report Page Layout.
  5. Add a new page as below. Please make sure you have ed Form Page at the bottom of the page.

  6. Finally go to OA Reporting & BI > Document Designer > Document Designer.
  7. Set up the document as show below. You should only need to set Extra Formatting to font-family=Times Roman,padding=0mm, and the Page and Class fields.

You should now be able to print to the new P60 form.
