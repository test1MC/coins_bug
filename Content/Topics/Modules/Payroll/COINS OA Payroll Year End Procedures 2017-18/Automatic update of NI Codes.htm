All current employees will also have any contracted out number and flag cleared when starting 2017.

There is a pre-existing apprentice flag in Employee Maintenance which will be used in doing the above initialisation.

The exact definition of a qualifying apprentice for NI purposes is still being decided, which means this may not line up precisely with how the existing apprentice flag is being used. Hence this initialisation will have to be a best guess and users will be free to set an NI code which does not agree with the apprentice field value.

Running the OA Employee NI Detail report in ‘Normal’ mode using date of birth selection on the Employee Selection tab can be used to get a list of employees aged between 16 and 24 on the 6th April 2017 to check through.

An additional age based automatic NI code update will be added. The pay calculation will not attempt to check if an employee has become or ceased to be an apprentice.
