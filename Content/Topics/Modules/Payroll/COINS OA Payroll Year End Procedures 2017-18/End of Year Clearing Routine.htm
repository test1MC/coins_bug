End of Year Clearing Routine

In OA the Clear down routine is run when you start new Pay period for period one of the new Tax year.

Run this once only for each frequency as follows:

  1. Go to Payroll > Period Payroll > Period Administration > End of Year Clearing Routine.

    displays the message 'Are you sure you wish to clear all the date totals?'.

  2. Answer Yes.

The Year-end-clearing routine:

  • Clears the “to date” figures on items that have Year End Clear set to Y, for all employees within the pay frequency for this company.
  • Advances the Tax year in Payroll Configuration from Tax Start Year 2014 to 2015 and changes the Payroll Period (this means you do not need to run “Start Pay Process”).
  • Changes the leaver code from 2 (Left) to code 3 (Left in Prior Year).
  • Changes the employee last paid period appropriately
  • Changes any “week 1” or “month 1” tax codes to a cumulative basis.
  • Updates the weekly pay periods with reference to parameters WLTDAYS and WPAYDAY. Note: Monthly periods will have to be created manually.
