Set Week 53 Indicator to Yes

This is only applicable to Weekly Payroll (including two-weekly and four-weekly) and is the only variance between monthly and weekly.

If it is necessary to run a payroll for week 53 pay period /, you must ensure that the ‘Week 53 year’ box has been ticked. Do this before starting the pay process for week 52, or sooner if possible.

To set the week 53 indicator:

  1. Go to Payroll menu > Company Setup > Company Configuration.

  2. Go to Payroll > Weekly Payroll > Company Setup > Payroll Configuration.

  3. Tick the Week 53 box.
  4. Set the Week 53 field to Y and press F1 on each of the following screens to accept the details.
  5. Go to Payroll > Weekly Payroll > Setup > Periods Maintenance, and make sure that a valid week 53 period exists if required.

  6. Go to Payroll > Weekly Payroll > Weekly Setup > Periods Maintenance, and make sure that a valid week 53 period exists if required.
  7. If you need to add a period, click Addpress F9 to insert an additional line and populate it with dates and other values as appropriate. Please ensure that the Week 53 dates are different to those set up for Week 52. Once you have entered the dates, press F1.
