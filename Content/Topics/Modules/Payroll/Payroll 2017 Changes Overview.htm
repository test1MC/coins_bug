Payroll 2017 Changes Overview

This document covers the updates to the RTI returns for 2017 and a couple of pension reporting updates which already apply in 2016.

  • FPS Changes:
    • Start Date Reporting
    • Pay Frequency Reporting
    • Serious Ill Health Lump Sum Payment Reporting
  • EPS Changes - Apprenticeship Levy

Reporting of payrolled company cars via RTI is optional for the 2017 tax year and only becomes mandatory from the 2018 tax year. The software changes will not be available until mid-2017 and are therefore not covered in this document.
