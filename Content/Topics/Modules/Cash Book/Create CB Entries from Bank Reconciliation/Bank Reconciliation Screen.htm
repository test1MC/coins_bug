Bank Reconciliation Screen

Several changes have been made to the Bank Reconciliation - Statement Items screen.

New Action

A new action is available, named Enter Missing Items. You can select a single unreconciled statement item and apply this action. This takes you to a new Missing Items Entry Screen (see 1, Missing Items Entry Screen) in which you can enter details of missing COINS items for the unreconciled statement item.

The missing COINS items for all unreconciled statement items are entered in a single special open batch, with a description of “Missing Items Reconciliation”. If there is no missing items reconciliation batch currently open, the Enter Missing Items action creates one.

New Column

A new column has been added to the Bank Reconciliation browse.

Field
Description

Missing Items

This shows the balance of all missing COINS items that have been created for each unreconciled statement item.

New Filter

A new filter called ‘Missing Items’ has been added. When selected, the filter shows only those unreconciled statement items that have had missing items entered against them.

New Statuses

Two new values are possible in the Status column:

  • ‘Missing Items Incomplete’ means that missing items exist against the unreconciled statement item but their balance does not match the statement item value.
  • ‘Missing Items Complete’ means that the balance of the missing items matches the statement item value.

Statement Item Date Link

If missing items have been entered against a bank statement item then clicking on the statement date link takes you to the Missing Items screen where you can maintain the missing items. If there are no missing items against the statement item then clicking the statement item date field takes you to the COINS items screen as usual.

Other Actions

The actions Un-reconcile Items, Reconcile Unmatched Item and Enter and Reconcile Unmatched Item cannot be used if any missing items have been entered against the statement item.
