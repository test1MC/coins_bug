Statement Items Browse (%WCB1060BCBL)

Two new columns have been added to the browse.

Fields

Field
Description

Marked Items

Each row shows the value of all items that have been ‘marked’ against the statement item.

Status

This shows the status for statement items that have marked items.

  • Ready to Reconcile - The value of all marked items equals the statement item value.
  • Items Marked - The value of all marked items is not equal to the statement item value.

Marked Items Page Filter

A new page filter with a label of ‘Marked Items’ shows only those open statement details records with marked items.

Statement Items Actions

Two new statement item actions have been added.

Reconcile Marked Items

This action allows a user to reconcile a single statement item to marked items.

Only one statement item at a time can be selected.

If the user tags an item with no marked items then an error message will be displayed.

If the total value of the marked items equals the value of the statement item, then the action will reconcile the items to the statement item.

If the total value of the marked items does not equal the value of the statement item, a warning or error message may be displayed depending on the new CB parameter MRKVALEW (see 1, Configuration). The action will reconcile the items to the statement item and also display a variance screen in which the user will be able to write the difference off to a variance account.

Deselect Marked Items

This action will deselect all items that are marked to the statement item.

Existing Statement Actions

If a statement item is tagged that has marked items, it will not be possible to use the existing statement actions namely; ‘Unreconcile Item’, ‘Reconcile Unmatched Item’ and ‘Enter Missing Items’. If the user tries to use these actions then an error message will be displayed.
