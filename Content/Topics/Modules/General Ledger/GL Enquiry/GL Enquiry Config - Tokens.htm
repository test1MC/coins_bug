GL Configuration - Tokens

A new tab allows you to define tokens. Tokens allow you to include run time selection options in the which the user can specify. For example, you could produce a single that can be run for individual departments or sectors within the business (choosing which department or sector when you run the ).

GL  Configuration - Tokens Tab

Fields

Field
Description

Token

The token that is used in the definition. Tokens are a $ sign followed by one or more characters.

Description

The description of the token, to be used as a prompt when the is run.

List

Whether the token is used as a list of items.

Blank

Whether a blank value is allowed.

Type

The type of value that should be assigned to the token at run time. This will depend on where the token has been used in the definition.

Generate Tokens Option

A new option allows you to generate the tokens. First, add the tokens to the row definitions you want them to apply to.

GL Configuration - Tokens on Row Definition

Then run Generate Tokens. This adds and new tokens to the list on the Tokens tab, where you can set the labels and other properties of the tokens.
