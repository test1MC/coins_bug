GL Configuration - Rows

Configuration of rows has been enhanced to make the selection of accounts more like the GL Report Configuration.

GL  Configuration - Rows

Field
Description

Account

The previous From Account, To Account and Account Mask fields are now presented in the standard From, To and Matches format.

Account Type

From, To and Matches fields allow you to specify which account types to include.

Elements

For each element of the General Ledger account, From, To and Matches fields allow you to specify which accounts to include.

Analysis Sets

For each account analysis set, From, To and Matches fields allow you to specify which accounts to include.

Element Analysis Sets

For each element analysis set, From, To and Matches fields allow you to specify which accounts to include.

Percent Line

The number of an row that you want to show the amount on this row as a percentage of.

Row Divide
The number of an row that you want to divide the value of this row by.
Save Into
Renamed from Accumulator. The name of the accumulator into which you wish to save the values of this row. The Operand field determines whether the values are added to or subtracted from the accumulator.
Operand
Renamed from Sign. Whether the values for this row are added or subtracted from the accumulator value.
