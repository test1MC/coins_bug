Customer Care Scheduler - Parameters

The following parameters control the of the Customer Care scheduler.

Parameter

New or Existing

Description
SCH2STIM

New

 The default start time for the scheduler Day view. The Day view opens with this time-slot aligned at the left of the view, though the view can be scrolled horizontally to show earlier times.
SCH2USE

New

 Whether to use the new scheduler.

SCHDEFHR

New

The default duration to use on the scheduler for tasks that have no duration specified.

SCHTFLD

Existing

The text shown on the first row of a task on the scheduler.

SCHTFLD2

New

The text shown on the second row of a task on the scheduler.
