Customer Care Scheduler - Scheduling a Task

To schedule an unscheduled task, drag it from the To Schedule list and drop it onto the calendar for a specific and time.

The duration of the task is shown at the bottom right of the scheduled task. If an unscheduled task has duration specified, when the task is dropped onto the calendar a default duration will be assumed. This is specified in the new HS parameter SCHDEFHR.

Tasks are ed based on the SLA dates of the associated issues: tasks that breach the SLA are ed red; tasks within the target SLA are green; tasks on the target SLA date are orange.

Rescheduling a Task

You can reschedule a scheduled task by:

  • Dragging it to a different and/or time on the calendar.
  • Double-clicking the task (or using the context menu; see 1, Customer Care Scheduler - Task Context Menu) to bring up the Update Task box and updating the date, time and/or duration.
  • Making a task the first task of the day. Right-click the task and choose Set As First Task from the context menu. This moves the selected task to the start of the ‘s working day, then reschedules the other tasks to follow it.

In Single Day view, you can change a task’s duration by dragging the left or right side of the task.

Unscheduling a Task
