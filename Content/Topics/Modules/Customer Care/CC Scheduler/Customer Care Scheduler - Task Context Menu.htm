Customer Care Scheduler - Task Context Menu

Right-clicking on a task in the calendar brings up a menu that allows you to do various things with the task.

  • Update Date/Time (see 1.1, Rescheduling a Task)
  • Unschedule (see 1.2, Unscheduling a Task)
  • Joint Task (see 1, Customer Care Scheduler - Creating a Joint Task)
  • Add Schedule Note (see 1, Customer Care Scheduler - Adding a Schedule Note)
  • Schedule Additional Task (see 1, Customer Care Scheduler - Creating a New Task)
  • Set as First Task (see 1.1, Rescheduling a Task)
