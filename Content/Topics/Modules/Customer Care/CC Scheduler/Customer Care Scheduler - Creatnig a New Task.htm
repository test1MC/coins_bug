Customer Care Scheduler - Creating a New Task

Tasks should only span one day. If a task will run over more than one day, schedule a new task based on the existing scheduled task.

Right-click the task and choose Schedule Additional Task. This provides the following sub-menu options:

  • Today
  • Next Day
  • Next 7 Days - This has further sub-options that list the next seven days.
  • Select - This allows you to choose a specific date using a date picker.

    • Schedule Additional Task Menu

You will be prompted to confirm that you want to create a task on the date you specify.

This creates a new task with the same duration, and at the same time of day.
