Customer Care Scheduler Screen

The Scheduler allows you to schedule Customer Care tasks by dragging and dropping them onto a calendar-style view of and time slots.

Scheduler Day View

To Schedule

The To Schedule panel on the left lists the unscheduled tasks. These are the same records as shown on the ‘Unscheduled’ tasks tab in the Customer Care workbench. You can filter the tasks, for example to show only tasks for selected developments; see 1, Customer Care Scheduler - Filtering.

For each unscheduled task, the following details of the task are shown:

  • Task Reference
  • Task Reason
  • Development Number
  • Development Name
  • Plot
  • First part of instructions
  • Duration
  • SLA Date

The SLA Date shown is the earliest SLA date based on the issues assigned to the task. This is ed red if the date is before today, amber if today and green if in future.

Scheduled Tasks

The main part of the schedule screen shows the tasks that have already been scheduled, as a calendar view. The calendar also shows other Activity Workbench entries, such as holidays. Entries not related to Customer Care are not updateable.

You can change the timescale for the tasks view by clicking the buttons at the top of the calendar:

  • Single day view shows details of the tasks for a specific day for each . This is broken down into one-hour time slots, running from midnight to midnight. By default the leftmost time slot is the time specified in the new HS parameter SCHSTIME.
  • Multi-day views (five day view, seven day view or 14 day view) show details of the tasks for multiple days for each .

The Work Week button allows you to exclude weekends from the view.

Day View Buttons

Navigation buttons allow you to move backwards or forwards through the days or weeks, and the date picker button allows you to choose a specific date to show.

Date Navigation Buttons

The Scheduler remembers the view you select; next time you run the scheduler it will show the same view.

Refresh

The refresh button reloads the list of unscheduled tasks, and the current information shown in the schedule window.
