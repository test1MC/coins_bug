Set Home Page

When logging into , you may select a specific home page that you will be returned to automatically.  This may be useful if you always navigate to a particular function after login.

To set your preferred home page, navigate to the screen you wish to set as your default and select the Set Home Page button on the Global Shortcuts menu.

The following message will be displayed:

To set the home page, click OK.  To cancel the operation and leave your homepage on the current setting, select Cancel.

To reset the Home page, either set a new homepage using the steps above, or select the User Preferences button on the Global Shortcuts menu and blank out the Home Page setting on the User Preferences Tab.
