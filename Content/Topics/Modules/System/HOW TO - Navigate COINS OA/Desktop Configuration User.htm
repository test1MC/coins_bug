Desktop Configuration - User

Desktop Setup Wizard

If a blank Desktop has been set up for you, there is a link to the Desktop Setup Wizard at the top of the screen.

Link to Desktop Setup Wizard

NOTE

You can also run the Desktop Setup Wizard from a link on the User Preferences screen.

The Desktop Setup Wizard allows you to set up the layout of your Desktop (which tabs and sections it contains) and which functions to add to it.

The wizard consists of a sequence of screens:

  • The first screen allows you to choose an existing Desktop to copy. This is used as the 'starting point' for your Desktop.

Desktop Wizard – Desktop Selection

  • If you select an existing Desktop, creates a copy of that Desktop, which you can then modify (without affecting the original).
  • If you select the Desktop that has your own user ID, deletes that Desktop, and creates a new, empty desktop for you.
  • If you leave this blank, creates an empty desktop for you.
  • The Sections screen allows you to name the tabs you want on the desktop, and the sections you want on each tab.

Desktop Wizard - Sections

  • The History screen allows you to select the functions to add to the Desktop. This shows a list of functions.

Desktop Wizard - History

At the top of the list are the functions that are already on your Desktop (if any). Below that are other functions you have run, arranged according to how often you have run them. You can add any of these to your Desktop, and choose which section to add them to.

creates tiles for the functions you choose, on the sections you specify. You can then modify the appearance of the tiles, and how the functions open, using the Desktop tab in User Preferences.

When you click the Next button on the History screen, refreshes the Desktop and applies the changes.
