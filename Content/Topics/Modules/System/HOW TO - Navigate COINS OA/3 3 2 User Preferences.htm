3.3.2 User Preferences

User Preferences now opens in a frame

If a desktop is configured for you (that is, there is a desktop menu in the Desktop field in User Maintenance), User Preferences has a new field.

User Preferences

Desktop Image

Allows you to specify the background image for your desktop. This is a drop-down list of the available image files (see 3.2.3, Images).

The background image also shows on the log in screen.

A new tab on User Preferences allows you to configure your desktop (if you have permission).

User Preferences – Desktop Tab

A link to the Desktop Setup Wizard allows you to set up your Desktop (see 3.3.1, Desktop Setup Wizard).
