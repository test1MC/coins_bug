Expanding the Menu

The Menu Text box will enable you to view the Menu Option description even if the menu itself truncates the description in the menu panel.

You can extend the Menu panel to the width you require.

  1. Move the mouse pointer over the panel line dividing the menu and the screen. The cursor will change to reflect a line with double arrows.

     

  2. Hold the left mouse click down. This picks up the line.
  3. Drag the cursor across to where you wish the menu panel to be and release the hold. The panel will move with you as you drag the cursor.  remembers the width of the menu panel, so next time you open the menu, it will open to the width you selected.
