Using the Back/Forward Buttons

You may wish to go back the previous screen when using , especially when you have added filters to a browse screen or workbench.

If you select the option again from the menu, you will lose all the filters you have input.

Using the back and forward buttons is the best way to keep your filters. This is available as each page is separate and therefore can be recalled.

  1. Select the Back button to go the previous screen. This will go back the previous screen with all the filters and search criteria. 

    2. An alternative option to go back to the previous screen would be to select and hold the ALT key and select the Left Arrow key.

    As you go back through the previous pages, you may wish to return or go forward and have all the filters originally selected saved and in place.

  2. Select the Forward button to go to the next page. You must have seen this page to be able to move forward to it.

    3. An alternative option to go back to the previous screen would be to select and hold the ALT key and select the Right Arrow key.
