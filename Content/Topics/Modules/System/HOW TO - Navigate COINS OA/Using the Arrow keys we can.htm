Using the Arrow keys we can start to compile our print. Select the Name from the Available field and select the Add button. This will move the selected item from the Available field to the selected field.

To select more than one line, hold the Crtl or Shift key and select either a block, or specific lines. Select the Add button (not the Add All button) and this will add all your selected items.

    Using the different Arrow buttons will enable you to add or delete one or more lines of data.

    When you have added all the required items to the list, you may wish to change the sort order of the list.


Locate the item you wish to move up and highlight it. Select the Move Up or Move Down button and the item will move one space up/down the list. You may need to do this a few times to be in the right position.

    The Output Options tab is available for you to select how you wish the print to be produced. Go to the Report Output Option section in this document for more information.

    Select the Forward button. A message will be displayed stating “Task Has been submitted successfully”

    Select the OK button. The Print will be available in the Report Status workbench. Go to the Report Status section in this document for more information.
