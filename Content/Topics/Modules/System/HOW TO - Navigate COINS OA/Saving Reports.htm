Saving Reports

There is a box on the Saved Reports browse.

Field Definitions

Pack

If selected this allows this saved report to be included in a report pack.

Once a report has been included in a pack (see 0.0.1, Report Pack Reports) then the pack selection cannot be deselected nor the saved report deleted until it is removed from all report packs that use it.

There is similar functionality on scheduled reports to indicate that the report can be included in a report pack.

The existing process for saving a report has not changed:

  1. Set the selection criteria for the report.

  2. On the Output Options tab of the report, select Save.

  3. Run the report.

    4. The report is added to the Saved Reports tab in the Report Status Workbench.

  4. Open the report record to select the date offsets.
