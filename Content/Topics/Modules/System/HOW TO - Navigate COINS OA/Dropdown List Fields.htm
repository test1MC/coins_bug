Dropdown List Fields

You must use the dropdown lists for selecting the right data to be entered into the field. You are unable to input the data manually into the field. It has to be populated from the dropdown list.

  1. Select the Dropdown Arrow on the right of the field.
  2. This will display a list of valid data to select to populate the field. Move the cursor over the line required and select. This will close the list and automatically populate the field with the selected data.

    Only one selection can be made. You cannot type into the field therefore the data you require must be on the list.  However if the information you require is not shown on the list, contact your system administrator.

  3. If the field shows <Select> as input this Dropdown field is mandatory and you must select from the list. You will not be able to save the record if you have not made a selection.
