Search within the Browse Screen

The Search Field with the Browse screen relates to the Headings of the columns and will enable you to search the data based on the column headings.

This is the recommended way of searching for specific records. The list will be reduced to display only those records matching the search criteria.

If you are looking for specific information on the screen, for example, a User in the User Browse screen, you can hold the Ctrl key and press the F key, then type the word or words your are searching for.

Every match will be highlighted with the Browse screen list.
