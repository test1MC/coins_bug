Select the Dropdown button on the right of the field. This will display a list of option for you to select.

Recall Filters Field

The Recall Filters field relates to any filter fields on any Browse screens. These are usually available for you to search within. A typical example would be to set the filter to a specific type of order and then search within by Order number.

Recall Filters in Lookups

The Recall Filters in Lookups field relates to any filters in lookups screens, these are additional screens which are available using the Lookup button at the end of certain fields.

 

Filter Options

Description

Default

Use the company setting for the filters.

Full

The filters you set are remembered constantly. You will have to physically change them yourself if you wish to filter a list on a different option

Day

The filters set will only be remembered until the end of the current day

Session

The filters set will only be remembered until you are logged out

None

Do not remember any filters

    Select the option required and the field input will change to the option selected.

    Select the Save button. It is not necessary to log out and back in for the selection to take effect and can be used. It is effective immediately after being saved.
