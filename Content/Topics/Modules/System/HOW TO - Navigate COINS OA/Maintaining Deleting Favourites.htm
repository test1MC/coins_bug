Maintaining/Deleting Favourites

When a favourite page is no longer required we would recommend you delete it from your list.

You can use the User Preferences screen and select the Favourites tab and follow the steps below to delete the record.

  1. Locate the Favourite panel on the Home Page.
  2. Select the Edit button. This will enable you to maintain your favourites list.

  3. The Favourites screen will be displayed. You will notice there are no tabs available on this screen.

  4. Locate the favourite you want to delete. Select the Tick Box adjacent to the record. This will highlight the record required.
  5. Select the Delete button.  The system will display the warning message

  6. Select the OK button. The screen will automatically refresh and the record will be removed.
  7. Refresh the Home Page to remove the record in the Favourites panel. Right Mouse Click on the Home Page to display the Menu. Select the Refresh option.
  8. The screen will refresh and the record will be removed.
