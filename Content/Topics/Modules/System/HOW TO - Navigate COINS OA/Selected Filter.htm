Selected Filter

  1. Move the mouse pointer over the Selected Filter button to see how many records you have selected. The number will be displayed at the end of the Text Box tooltip in brackets, i.e. (3). This is very useful if you are only selecting specific records.

  2. To see how many records you have selected, move the mouse pointer over the browse menu button. The menu shows the number in brackets.

  3. Select the Selected Filter button Show All option and only the selected records will be displayed.
  4. To show all records, click the Selected Filter button againselect the Show All option.
