Selecting Companies

Some organisations may have more than one trading entity (or Company) for which there are separate ledgers.  If you have access to these separate companies, the method to switch between them is as follows:

  1. Select the Dropdown button on the right of the Company field. This is the first field at the top of the menu
  2. Select the Dropdown list in the header bar.

  3. Select the Company required from the list. The list will only show those companies to which you have been granted access.

  4. The view on your current page will automatically refresh with the data from the company selected.
