Browse Screens

has many Browse screens which enable you to view a set of records matching predefined search criteria. Tabs are available on Browse screens that are used to filter the same data in different ways or the sort order, i.e. All Material Orders and By .

They enable you to page up and down, move to the first record, or the last record, select records specific records or a block of records.

You can change the sort order, drill down on certain information using hyperlinks, export data into Excel, and print certain information to suit your specific requirements.
