Using an * Asterisk to filter

Using an asterisk when filtering on a search will enable you to search on words, codes and values. Depending on where you place the Asterisk will return different results. This represents any string of characters.

  1. Enter West* in the Filter field. 
    2. The Filter West*

  2. The filter is being used with the Search field on the Name column to return any records starting with West and have any other characters afterwards.
    3. The Results of the Filter West*
