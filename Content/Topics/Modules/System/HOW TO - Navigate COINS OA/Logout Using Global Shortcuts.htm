Logout Using the Logout Icon

Using the Logout icon for logging off is the recommended way of exiting . This Logout icon is found on your user menu:

 

at the bottom right of the main screen.

The Logout icon enables you to exit the system correctly. It is recommended you close all records, enquiries and reports before you exit the system.

  1. Select the Logout icon. A message will be displayed asking “Logout?”
  3. Select the OK button and you will return to the Login screen.
  4. Alternatively select the Cancel button to stay logged in. You will stay on your current screen.
