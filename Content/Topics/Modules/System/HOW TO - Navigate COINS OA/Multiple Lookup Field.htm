Multiple Lookup Field

When using Lookup fields you may have a requirement to select more than one line of data. For example, you may want to search for for a number of locations.

  1. Select the Lookup button on the field. This will display the table of values and data.

  2. Select the multiple records you require using the Tick Box adjacent to each record, using the block selection or specific selection.

  3. Select the Return Selected button at the bottom of the table. This will return all the records ticked.

    You cannot use the Select button adjacent to the line, as this will only select the one line.

  4. The table will close and all the selected information will be populated in the field, separated by commas.
