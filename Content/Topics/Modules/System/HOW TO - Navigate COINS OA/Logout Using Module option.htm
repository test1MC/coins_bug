Logout Using Module option

The Module option can be found under the Modules list on the left of the screen, just above the menu.

It is recommended you close all records, enquiries and reports before you exit the system.

  1. Select the Dropdown Arrow at the end of the Module field to display the list of modules you have available.
  2. Use the scroll bar to scroll to the bottom of the list. Select the Log Out option.
  3. A message will be displayed asking “Logout?”

  4. Select the OK button and you will return to the Login screen.
  5. Alternatively select the Cancel button to stay logged in. You will stay on your current screen.
