Frame Menu

If you minime a frame, it moves to the "Frame Menu"in the page header. When frames are minimed, an indicator on the Desktop button shows how many there are.

The Frame Menu hides automatically, but you can move the mouse pointer over the Desktop button in the page header to show the Frame Menu with the minimed functions on.

Click the restore or maxime button on a frame to expand it again.

 

Frame Menu

Although the only way to run functions in frames is from a Desktop tile, if tiles have been minimed, the Frame Menu is available from the page header of standard screens, and functions in frames can be restored on top of standard screens.
