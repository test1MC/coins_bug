Email

To send the report by email once it has completed, the email field.  Additional selection fields will be displayed.

If your user preference are set to the email default, these fields will displayed automatically – in which case, deselecting the field will hide the options.

  1. Selecting the Email box will enable you to select the email type as well, i.e. HTML.
  2. The recipient does not have to be a user or have access to the specific data, as this will be sent in a PDF format to them.
  3. will automatically create attachments with a standard naming format consisting of the Module and a sequential number (for example SC000360.pdf).  If this is acceptable you need take no further action.  If however, you need to specify a file name (for situations where an external system expects an import file with a specific name for example), the Attachments Name field allows you to specify a name for the PDF and Excel attachments in the email. 
  4. Add additional comments or message if you wish.
  5. The report will run in the background and when complete will send the email and also place the report on the Report Status.
