Selecting Modules

Users will have access to the records, ledgers and reporting for different modules. To do this they need to be able to change the module access during the working day.

  1. Select the Dropdown button on the right of the Module field. This is the second field at the top of the menu
  2. Select the Module required from the list. The view on your current page will automatically refresh with the Home Page and a new menu from the module selected.


Be aware that only those modules to which you have been granted access will appear on the drop-down.
