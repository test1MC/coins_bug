Change Filter Settings

Filters are available to be selected when using Browse screens, Reports and Enquiries. There are also different views of specific screens relating to these filter options.

The filter settings are available to be changed. The company will have already preselected the standard options, however if the field is available you can override the option. 

You can also set the reports to be sent to you via email as a default. This is only from the reports and not when the Print button is used.
