Add to Favourites

This will add the specific screen you are currently viewing to your favourites list. You must be viewing the screen to add this as a favourite. This will also include any filters that you have used if you are in the browse screen.

  1. Locate the screen you wish to add as a favourite.
  2. Select the Add to Favourites icon from the Global Shortcuts icons.

  3. The Add to Favourites screen will be displayed showing the URL link in the Favourite Description field.

    4. You can amend the description of the link to be meaningful to you. This will make it easier to identify when maintaining your list.

    Ticking the Save Page Options field will store the settings of any filters etc. that you have used.

  4. Enter the description in the Favourite Description field and select the Forward button.
  5. The link will be added to your favourites list in your Home Page.

  6. Select the hyperlink to go the Favourite page. This will be opened in a new window.
  7. The Favourites Menu is also displayed when you move the mouse pointer over the Favourites button. It shows the functions you have set up as your favourites. You can click any of the links to go straight to that function.
