Display / Hide the Menu

When you are looking at certain browse screens or , you may not be able to view all the columns on the screen. You will need to scroll across the bottom.

If you wish to see the entire screen, you can hide the menu to enable the screen to stretch out.

  1. Select the Trianglearrow icon to the left of the screen title to hide the menu. The trianglearrow will point in the direction it will move, for example, push the menu closed.

  2. The menu will have closed and the screen can now been seen in full.

  3. To open the menu and return to the standard screen layout select the White Trianglearrow. The trianglearrow will point in the direction it will move.
  4. The menu will be visible.
