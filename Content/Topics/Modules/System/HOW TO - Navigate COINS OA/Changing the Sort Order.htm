Changing the Sort Order

The information is listed in columns of data and therefore has a predefined sort order allocated when the screen is initially opened.

Each column has a triangle next to the name where a sort is available. You can only have one sort on each page.

A closer look will show one triangle is a single arrow pointing up or down and the rest are double arrows. This shows the single triangle is currently the main sort and its direction indicates ascending or descending order.

The heading of the column that is currently sorted is highlighted in a different , and the arrow shows whether the records are sorted in ascending or descending order. Other columns that can be sorted have a double arrow, and are highlighted when you move the mouse pointer over them.

  1. To check the sort order, move the mouse pointer over the column heading to check the sort order.

  2. To change the current sort from ascending to descending, click the column heading. In our example click the PO Number heading hyperlink.

  3. In our example screen, to change the sort from the PO Number column to the Supplier column, to select the Supplier heading hyperlink.

  4. When you move the mouse pointer over the Supplier column heading hyperlink, there is no text box tooltips as there is no sort on this column. Select the Supplier column heading hyperlink.

  5. The sort order has changed and the Supplier is now the first main sort. The text box tooltips are availableso the records are sorted by supplier name, and the single arrow indicates the sort order.
