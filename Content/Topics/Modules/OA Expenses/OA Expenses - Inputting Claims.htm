OA Expenses - Inputting Claims

When you add a claim batch record on the Claims Workbench and follow the link on the claim number field, displays a browse that allows you to add claims.

Expense Claim Records

Fields

Employee

The employee number of the claimant. A lookup of Expense employees is available.

Name

Name of expense claimant.

Claim

The claim number (generated automatically by ).

Claim Date

The date on which the expense claim was filed.

Expense

The date on which the expenses were incurred.

Gross

The total amount of the expense claim. If the EX parameter ERRCTRL is set to N, calculates this total as claims are entered.

Net

The claim total minus . If the EX parameter ERRCTRL is set to N, calculates this when the claim is marked as complete.

The total in the claim. If the EX parameter ERRCTRL is set to N, calculates this total as claims are entered.

Route

How the expense claim will be paid. This can be through Payroll (only if the employee is linked to a valid Payroll employee record) or through Expenses.

External Ref.

An external reference for this claim.

Authored By

The user who authored the expense claim. This field is for information only.

Complete

Whether the claim input is complete. This must be ed before the claim can be to posted.

Posted

An indicator to show whether the claim has been posted.
