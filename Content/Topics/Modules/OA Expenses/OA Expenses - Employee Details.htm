OA Expenses - Employee Details

Expenses Employees allows you to create Expenses employee records.

Expenses Employee Maintenance

If employees will enter expense claims using CAS Expenses, they need to be set up with person records in HR, and employee records created in CAS Expenses to link to the person records (see the CAS Expenses documentation).

If expense claims are not coming from CAS Expenses, the employee records can be stand-alone.
