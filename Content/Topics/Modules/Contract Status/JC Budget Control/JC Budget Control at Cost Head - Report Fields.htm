JC Budget Control at - Report Fields

Two new fields are available for use in workflows when comparing order values to budgets at level:

  • RO_CostHeadsOverBudget returns a list of whose budget would be exceeded if the order in workflow were to be committed.
  • RO_iCostHeadOverBudget returns the value 1 or 0. The value 1 is returned if there are any whose budget would be exceeded if the order in workflow were to be committed.
