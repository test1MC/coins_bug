Open

Open Date

The date on which the became open. This field is only available if Open is ed.

Location Name

The site location. This field is only available if the JC parameter SITELOC is set to Y.

It allows you to enter or select a location from the list of site locations (see 1, Site Locations – Locations). If the location is populated, the address fields on the default to the address set up on the site location record.

If the CO parameter COMPLOC is set to US, PR information on the is also populated with information from the site location record.

On the Maintenance Configuration tab, you can select the values of any additional analysis sets that have been configured (see 1, Contract Fields - Analysis Sets).
