Fields - Additional Analysis Sets

An additional six analysis sets can be defined against a . Once populated these fields allow standard configuration of the analysis sets and are available against all , and all reports and where analysis can be filtered.

NOTE

The additional analysis sets are only available in OA. COINSplus screens will show only the original four sets.

Analysis sets must be configured in sequence and no blank analysis sets are allowed out of sequence (for instance, you cannot configure analysis set 3 and not configure 1 or 2).
