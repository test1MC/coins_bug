Site Locations – Locations

Locations allows you to set up the location records.

Locations Maintenance Screen

  • The Customers tab. Several fields on the Main tab are suppressed.
  • The Contacts tab.
  • A new tab. This is a read-only browse that shows the that reference the location (based on the current ). There is also a new field (RO_slc_NumOfJob) that can be included on column sets; this shows the number of in the current that are linked to the location, and a link on the value takes the user to the new tab.
