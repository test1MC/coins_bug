Configurable Fields

Configurable fields allow you to capture and report on additional information that is not covered by standard fields. You can configure up to 20 fields, of different types (for example, text, date, lookup) for each supported record type. You can configure fields globally (across all ) or for a single .

Currently, configurable fields are only supported for records.

Setting Up Configurable Fields

  1. Go to Configurable Fields Group.

    This shows the available fields groups. These are hard-coded, according to which records support configurable fields.

  2. Select the group you want to update, and from the action list choose either:
    • Change Global to set up fields for use in all .
    • Change -Specific to set up fields for use in the current only.

    Configurable Fields Maintenance

  3. For each field you want to set up, complete the following:

    Field
    Description

    Label

    The label to display against this field.

    Type

    The type of field to display. The types available are:

    None

    		Text fields, including date and number fields. You can specify a format, such as a date or number format.

    Editor

    		Text editor field. Allows the user to enter large amounts of text.

    Lookup

    		The field is populated using the specified lookup.

    Lookup Company

    		The field is populated using a lookup on companies in the Central Repository.

    Lookup Contact

    		The field is populated using a lookup on contacts in the Central Repository.

    Combo

    		The field is populated from a drop-down selection list (specified in the Populate/Key Field field).

    Selection List

    		The field is populated from multiple selection list (specified in the Populate/Key Field field).
    boxThe field is displayed as a box.

    Format

    A standard Progress display format to apply to the field; for example, a date or number format.

    Populate/Key Field
    The RSP and key field used to populate specific lookups, combos and selection list, as advised by .
    Function
    The lookup function to use (for lookup-type fields).

  4. the In Use box and click (Next) to save your changes.

The configurable fields are available in a new layout box when updating the record.
