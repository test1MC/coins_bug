Managing Subcontractor Tenant Users

To begin managing users, open the Administration app, then under User Administration, click Users. This shows a list of all the users that have been created.

Modifying a User

Right-click the user you wish to modify, or left-click the user and click the menu icon , then click Modify. This opens the Manage User window.

Edit the details as necessary. Note that some details are locked and cannot be changed.

Click to save your changes.

Managing User Groups

The User Groups assigned to a user determine which sections of which apps that user has access to within COINS Construction Cloud. To begin managing User Groups for a user, right-click the user, or left-click the user and click the menu icon , then click Modify User Groups. This opens the Modify User Groups window.

To add a User Group:

Select the User Group from the drop-down menu at the top of the screen.

Click to add the user to that group.

 

To remove a User Group:

Right-click the User Group, or left-click the group and click the menu icon , then click Remove From Group. You will receive a warning message asking if you are sure.

Click to confirm the removal.

De-activating and Re-activating a User

A user can be de-activated and re-activated by an administrator at any point, even before they have activated their own account via email.

To de-activate a user:

Right-click the user you wish to de-activate, or left-click the user and click the menu icon , then click Deactivate. You will receive a warning message asking if you are sure.

Click to confirm the de-activation.

 

To re-activate a user:

Right-click the user you wish to re-activate, or left-click the user and click the menu icon , then click Activate. You will receive a warning message asking if you are sure.

Click to confirm the re-activation.

Re-sending the Activation Email

The activation email that is sent out automatically when a new user is created can be re-sent. This may be necessary if the user cannot find the email, or if more than twenty-four hours have passed since the email was sent, in which case the activation link will have expired. Users who have not yet activated their accounts via email can be identified by the email icon under the Active column.

Right-click the user to whom you wish to re-send the activation email, or left-click the user and click the menu icon , then click Re-send Activation Email. You will receive a warning message asking if you are sure.

Click to re-send the email.

Resetting a User’s Password

Users can reset their own passwords via email, or they may request an administrator to reset their password for them.

To reset a user’s password:

Right-click the user for whom you wish to reset their password, or left-click the user and click the menu icon , then click Reset Password. This opens the Override Password screen.

Enter the new password for the user. It must satisfy all four of the listed criteria.

Click to save the password.
