Client Access to Portal

One feature available from OA v11.08 onwards, is it is possible to open the directly from .

This replaces the OA documentation, so the Documentation item on the user menu will open the help on the website without the user having to log in to the client area.

Links from the side-frame help will also open the appropriate topic in the portal. This has not been announced directly to the clients as the feature has not been exhaustively tested, and we wanted to get feedback on the portal generally before encouraging clients to switch.

However, if individual clients want to enable this on 11.08, all they need to do is to set the SY parameter DOCBASE as follows:

 

https://www.coins-global.com/resources/documentation/uk/Default.htm

 

Individual users may also need to allow popups from this URL.

 

If DOCBASE is blank, will continue to use the existing online documentation.

 

If any clients on 11.08 do choose to implement this, please let Don Barwick and Nigel Longley know.

 

Please also let us know if you have ANY feedback from clients on the new portal.
