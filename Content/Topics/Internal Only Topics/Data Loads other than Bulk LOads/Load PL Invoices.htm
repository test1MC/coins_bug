Import Invoices

(Example file RG IAD Data Load PL Invs)

 

Go to Purchase Ledger>Invoices>Import Invoices

Enter Location & Name of file to be imported.

Typically, this would be in a CSV format so box to say that it includes a Header Line & choose Comma for the Delimiter.

Add a Description.

There can be up to five files loaded at once.

 

Check the report for any errors.

 

Import Invoices Workbench

Go to Purchase Ledger>Invoices>Import Invoices Workspace.

 

The filter which should be set to Imported.

 

Highlight & Select the files to process, then Apply Action - Validate Workspace.

If the Validation is successful the files will clear from the screen so you need to change filter to Validated.

 

If however one of more the files remain that is because of validation errors, to investigate & resolve these open the file by clicking on the hyperlink, then open each invoice that has validation notes against it.

Once all the invoices have blank Validation Notes click Next to continue then Validate again, or choose Action - Validate from within the open file.

Change filter to Validated & then Highlight & Select files to be posted, then Apply Action - Post.

Check posted report carefully for any errors, then check PL & GL / JC posted reports.

At any stage before posting you can choose to delete file by using Clear Workspace Action.
