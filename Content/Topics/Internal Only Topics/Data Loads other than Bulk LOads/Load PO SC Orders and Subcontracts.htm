Import Subcontract Orders

(Example file RG IAD Data Load SC Orders)

 

This has to be csv format.

Go to Procurement>Administration>Import Subcontract Orders

Enter Location & Name of file to be imported.

Use Upload Mode - Check to validate file. Fix any errors.

Then use Upload Mode – Load to import file.

 

Go to Procurement> Subcontract Orders

Select the Orders to be Created and Apply Action – Commit Orders.

 

When complete reverse parameter settings.
