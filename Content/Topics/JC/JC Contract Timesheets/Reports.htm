Reports

 

Summary Report - produces a summary list of contract timesheet data

Detail Report - allows you to produce a report that shows, for each employee or a single employee, the activities they worked on each day, the number of hours, and the analysis.

Posting Report - allows you to produce a report showing the details of each timesheet batch.

Monthly Timesheet Report – As per detail report by period in summary or detail

Timesheet Cost Report - produces a report of costs generated from contract timesheets.

Monthly Missing Timesheet Report - allows you to produce a report that shows, for each employee, the days on which timesheets are incomplete or have not been entered
