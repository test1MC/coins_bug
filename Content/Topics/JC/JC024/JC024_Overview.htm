Overview

The Daily Reports function has a Deliveries tab that allows COINS OA users to see all the information about deliveries for a . However, since many supervisors review Daily Reports in the Review and Approval function, changes have been made in that function in Status to allow users to also view the information about Deliveries for a . Changes were also made in Project Management to allow users to view Deliveries information for a in the Daily Reports function.

This document describes the changes that were made in both modules to provide users with this functionality.
