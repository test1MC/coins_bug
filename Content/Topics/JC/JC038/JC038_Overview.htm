Quick Budget Entry: Overview

Enhancements have been made to the Budget Entry Workbench in COINS OA  Status to allow The Budget Entry Workbench in COINS OA  Status allows for the input of At Complete and To Complete values. This functionality also lets you set the Budget Input Method to Qty (Hours) on more than one Category when the Cost Code/Category version of the Workbench is used.

In addition, the Billing Item field on the and Cost Code maintenance screens has been relabeled as Item ID.

This document describes the changes that were made in v11.09 to accommodate the new Budget Entry Workbench functionality.
