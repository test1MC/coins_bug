Quick Budget Entry: Standard

The Standard function is found in Status > Setup > Setup Maintenance. It allows you to configure the Budget Input Method and Rate Source at either the level (JC/USECCCAT set to Not Used) or the Category level (JC/USECCCAT set to Used or All).

Standard Browse

The Standard browse screen now includes MULTI (multiple update) and BULK (bulk update) buttons at the bottom of the screen (Figure 1).

Field Definitions (Selected Fields)

Budget Input Method

If JC/USECCCAT is set to Not Used, the Budget Input Method field is now shown on the properties screen (Figure 3). This field has four options:

  • None
  • Value
  • Hours Input (labeled as Hours in previous versions)
  • Quantity Input (labeled as Quantity in previous versions)

If a specific Category has been set up, the following two new options are also shown on this field:

  • Hours Based
  • Quantity Based

Note: Both of these input types can be placed on numerous categories.

[Enter by]

This new unlabeled field is found to the right of the Budget Input Method field. It has the following options, as shown in Figure 4:

  • At Completion (the default)
  • To Complete

This field is disabled when Budget Input Method is set to None.

Forecasting Tab

Since the screen layout of the Forecasting tab was too wide, the following changes were made (as shown in Figure 6):

  • The Calculated Cost At Completion fields are now combined with Estimated Cost At Completion into a single Cost At Completion layout.
  • The Budget Input layout has been moved down and spanned to full page width.
  • New Enter By fields are now available on the Budget Input layout.
