Quick Budget Entry: Cost Code Maintenance

The Cost Codes function is found in Status > > Cost Code Maintenance with WBS tab. It allows you to configure the Budget Input Method and Rate Source overriding the settings. This can be done at either the Cost Code level (JC/USECCCAT set to Not Used) or the Cost Code - Category level (JC/USECCCAT set to Used or All).

Browse Screen

The Cost Code Maintenance browse screen now includes MULTI (multiple update) and BULK (bulk update) buttons at the bottom of the screen (Figure 1).

Field Definitions (Selected Fields)

Budget Input Method

If JC/USECCCAT is set to Not Used, the Budget Input Method field is now shown on the Main tab of the Cost Code Maintenance screen (Figure 3). This field has four options:

  • None
  • Value
  • Hours Input (labeled as Hours in previous versions)
  • Quantity Input (labeled as Quantity in previous versions)

If a specific Category has been set up, the following two new options are also shown:

  • Hours Based
  • Quantity Based

    Create Cost Codes Processing

    Auto Generate Cost Codes

    When a new Cost Code is created by the Create Cost Codes process, any Enter By field on the new Cost Code is now assigned from the Enter By field on the used for creating the Cost Code.

    Automatic Cost Code Creation

    A Cost Code can now be created automatically during either of the processes listed below.

    1. During user input from various areas, such as: Resource input when parameter SV/VOAUTOCH is set to Y.
    1. During XML Spreadsheet processing (load Items / Resources). Processing will be modified consistently.

    Regenerate Cost Code Data

    The Main tab of the Cost Code Data utility in Status > Administration > Rengerate Data now has a Budget Enter By box, as shown in Figure 6. If this field is ed, selected Cost Codes are updated from the .

    Generate Cost Code - Category Data

    Budget Input Enter By options are now copied to Cost Code – Category records created by the Generate Cost Code – Category Data report utility. Any further updates in the Cost Code Enter By fields are automatically reflected in the Cost Code – Category records.

    Cost Code Billing Item Label

    The label for the Cost Code Billing Item field (jcc_svitemid)[1..6] is now labeled Item ID in the areas discussed below. This field is used to link any Items to the Cost Code for Cost Based Billing purposes.

      • If parameter JC/CATCCBOQ is set to N, a Billing Item field labeled Item ID field now appears on the Main tab of the Cost Code properties. The label of this field is supported by a Lookup that uses a standard Item ID label.
    • If parameter JC/CATCCBOQ is set to Y, the Item ID field is suppressed on the Main tab of the Cost Code properties. Instead, the Billing Items section that has been relabeled as Item IDs is now used as the tab option.
