JC Parameters

The JC parameters listed below were added to Status to allow Owner Budget processing.

Parameter

Description

JC USEOWNBD

Use Owner Budgets (Y/N)

This enables the facility to record owner budgets and allow a Rate Type to be associated with a .

JC OWNRTYPE

Owner Rate Default Type

If set and valid and USEOWNBD=Y, then new default to use this Rate Type.

The Owner Rate Type on a is used if populated on the Owner Budget Status Report; otherwise, the standard Rate Type is used.

JC OWNPLNBD

Owner Plan Budget Type

JC OWNFORBD

Owner Forecast Budget Type

Changes

When JC/USEOWNBD = Y, ana new Owner Rate Type field is available on the Financial section of the Advanced Settings tab in JC , as shown in Figure 3. The value in this field depends on which of the following actions you perform:

  • Select a Rate Type from the field lookup.
  • Enter a value in JC parameter OWNRTYPE, which defaults into the field.
  • Leave the field blank so that the standard Rate Type applies.

Owner Changes

Owner Plan

When a value is set in parameter JC/OWNPLNBD, an a new Owner Plan screen is available in Status > Input > Memos,Budgets,Allowances > Input Budgets by Period, as shown in Figure 5. This is based on the existing Budget Input; the Budget Type value defaults in from parameter JC/OWNPLNBD and cannot be changed.

  • The Owner Plan Budget value defaults as blank if nothing has been entered in the cell. This allows for a zero value to be entered on some cells. The values show cumulative values to date, and zeroes are shown as blanks. You can override these values.
  • The Owner Plan Budget populates the Owner Forecast Budget Type when parameter JC/OWNFORBD is set. The Copy Budget function can be used to set the Forecast Budget to be the same as the Owner Plan.
NOTE:
Since the Standard Budget Input now excludes Budget Types related to parameters JC/OWNPLNBD and JC/OWNFORBD, they can only be maintained via this input procedure.

Owner Plan Screen

Owner Forecasts

When a value is set in parameter JC/OWNFORBD, an a new Owner Forecasts screen is available in Status > Input > Memos,Budgets,Allowances > Input Budgets by Period, as shown in Figure 6. You can use this screen to enter override values as cumulative (similar to Owner Budgets).

  • Each Cost Category field is blank unless you override it with a value or zero, which stores a period movement when the record is saved. Data are stored by Period and can be reported on, but the screen always shows as cumulative. A cumulative value can be entered for a Period (such as 0117 for Jan 2017) and then left as blank in another Period (such as 0217 for Feb 2017). Reporting still reports back the first value entered (such as in January).
  • If no override is entered for the month, the Forecasted value is calculated at the time the report is run.
  • The Owner Plan Budget populates the Owner Forecast Budget Type when parameter JC/OWNFORBD is set. The Copy Budget function can be used to set the Forecast Budget to be the same as the Owner Plan.
NOTE:
Since the Standard Budget Input now excludes Budget Types related to parameters JC/OWNPLNBD and JC/OWNFORBD, they can only be maintained via this input procedure.
