Standard Listing Report

The standard Listing Report lists all Subsidiary and their values for each selected.

Menu Option

The standard Listing report is found as a menu option in Status > Reports > Operational Reports > Standard Reports, as shown in Figure 1.

Menu Option

Selection Screen

Main Tab

When you select the Listing report option from the menu above, the Main tab report selection screen appears, as shown in Figure 2. The fields on this screen are described below.

Selection Screen

Standard fields found on this selection screen are: , Group, Type, Location, Manager, Customer, Active, Complete, and Supervisor.

Field Definitions (Selected Fields)

The value of this field defaults to the current period.

Recogned Revenue Method

This option is only available if Recogned Revenue Methods are set up in the logged-in company. The lookup options for this field include all Recogned Revenue Methods that have been set up under Status > Setup > Setup Maintenance. Sample options are shown in Figure 3.

Sample Lookup Options

Field Set

Select the field set you wish to display on the report. (Field sets are set up in Main Web Menu > OA Reporting & BI > Field Sets > Report Field Sets.) The following field set options to select from are:

  • Cash Analysis
  • Cost and Revenue Summaries
  • Cost Budgets
  • Current Budget
  • Forecast at Completion
  • Initial Budget
  • Revenue Budgets
  • Standard WIP - this includes all the fields found in the Standard WIP Report (which only exists when CO Parameter COMPLOC is set to USA)

Sort Tab

The Sort tab can be selected from the Listing Report selection screen, as shown in Figure 4.

Sort Tab

The following new options may be included in the dropdown menu of the Sort By field, depending on parameter settings:

Site Name

This option is only available when JC Parameter SITELOC is set to Y.

Recogned Revenue Method

This option is only available when Recogned Revenue Methods are set up in the logged-in company.

Existing Phase

This option is only available when JC Parameter JPHOFF is not set to Y.
