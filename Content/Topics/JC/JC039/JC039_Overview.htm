Overview

The Summary available in Status for the US region only uses LIVE data, which caused performance issues when larger, more complicated column sets were used on the browse. A Populate Summary has therefore been added to a new Data Mart submenu in Status to resolve this issue. It uses stored data using data marts and dataset definitions. It is only available for the US region and if there are any populated data marts.

This document describes the changes that were made in v11.09 to accommodate the Populate Summary functionality.
