Overview

A number of changes have been made to the Status in COINS OA. Below is a summared list of those changes, which are explained in more detail in this document.

Changes to the Cost and Summaries section include:

  • A Categories tab was renamed to “” and extended to show more information.
  • A new Periods tab shows historical data on and Cost Code levels. If PO/COSTMOVE is enabled, this can also show historical commitment and accruals.
  • A new Weeks tab was added to the standard Summary .

Changes to the Budgets section include:

  • New and Analysis tabs are now available.
  • A new Periods tab shows historical budget data on the and Cost Code levels.
