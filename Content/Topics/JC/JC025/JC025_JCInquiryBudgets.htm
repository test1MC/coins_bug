JC Budgets Section Changes

Browse Elements

Columns

An Unapproved column (Figure 1) was added to all browses of the Budgets section. This column shows values for an unapproved budget type.

Unapproved Column

Margin and Margin % columns (Figure 2) were also added to all browses of the Budgets section where the Cost Category is shown, such as the and Period browses.

Margin Columns

Select Field Set for Cost and Revenue

A Select Field Set field has been added to the Section, Activity, , and Analysis Set browses of the Budgets section. The options in this field are Cost and Revenue, as shown in Figure 3.

Select Field Set for Cost and Revenue

Unapproved Field Set Option

An Unapproved option has been added to the Select Field Set field in the WBS and Cost Code browses, as shown in Figure 4. After selecting this option, the WBS and Cost Code browses display the category split values for the Unapproved budget type.

Unapproved Field Set Option

Budgets Tabs

Tab

The tab screen (Figure 6) in the Budgets section of Status displays all of the budget types shown in the Cost Code tab aggregated on level.

Tab Screen

  • Data are sorted by and Description.
  • The named Filter has the following options: All, Non-zero only.
  • All numeric values show the Totals.
  • The Select Field Set field at the bottom of the screen allows you to switch between Revenue and Cost views.

Analysis Set Tab

The Analysis Set tab screen (Figure 7) in the Budgets section of Status displays all of the budget types shown in the Cost Code tab aggregated on the Analysis Set level.

Analysis Set Tab Screen

  • Data are sorted by Code and Description.
  • The named Filter has the following options: All, Non-zero only.
  • All numeric values show the Totals.
  • The Select Field Set field at the bottom of the screen allows you to switch between Revenue and Cost views.

Period Tab

The Period tab screen (Figure 8) in the Budgets section of Status displays a level Budget history browse showing the category split values and values of budget records per period.

Period Tab Screen

Budget History by Cost Code Browse

If you drill down on the Date column in the Period browse (Figure 8 above), you are taken to a Level Budget History by Cost Code Browse screen (Figure 9), which is filtered by selected Budget Type and ordered by Period.

Period Budget History by Cost Code Browse

At the bottom of this screen is a Budget field, which allows you to select the Budget Type view you would like to display, as shown in Figure 10.

Budget Field Options
