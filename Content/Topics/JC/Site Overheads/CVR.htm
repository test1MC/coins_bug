Update

The Update option has a new box, Update Site Overheads. This allows you to update the site overheads values in the forecast. This option is only shown if the model contains at least one item linked to a site overhead and the is set up in the Site Overheads Workbench.

In addition, a separate Update Site Overheads option is available on the update Options menu. This allows you to update only the site overheads.

This is required as some home builders don’t allow users access to the update items option and the only time the can be updated is when it’s first added. As the site overheads will likely be updated later after the initial creation process a separate function is required to allow the site overheads and only the site overheads to be updated.

The update process then takes a snapshot of the Site Overheads Workbench and updates the item records and the associated summary records.
