Site Overheads Workbench - Options

The following options are available from the Options menu of the Site Overheads Workbench:

Recalculate Site Overheads

Recalculate Site Overheads allows you to update the site overheads with actual costs and recalculate the actual cost variance rows. It also allows the accrual-related rows to be recalculated. This runs as a background report process; the context area shows the status of the recalculation process.

Lock Site Overheads

Lock Site Overheads allows you to lock the site overheads figures up to a selected period. Once locked, users will not be able to change the overhead item values for a locked period.

Unlock Site Overheads

Unlock Site Overheads allows you to unlock the site overheads figures, if they are locked. This allows for the correction of errors and to make any adjustments.

It is expected that access to this function will be restricted by function access so only designated users can unlock a forecast.

Update VAP/BoQ

Update VAP/BoQ runs a report that can update a selected VAP Budget and the current items Estimated Final Cost figures calculated on the Site Overheads Workbench.

VAP Budget Snapshots

VAP Budget Snapshots allows you to view any of the existing VAP budget snapshots.

What Ifs

What Ifs allows you to generate, view, and amend What If scenarios.

If you create a new What If, it will be based on a copy of the existing site overheads workbench. As this could take some time to process due to the number of records, this runs as a background report process. When viewing a What If scenario, it is updateable in the same way as the normal site overheads.

Delete What If

Delete What If allows you to delete the currently viewed What If scenario. It is only available when viewing a What If scenario.

Show Current

Show Current allows you to switch to the current view of the site overheads workbench, if you are viewing a What If scenario or VAP budget snapshot.

Gantt Chart

Site Overheads Gantt Chart allows you to report on the site overheads items in a Gantt chart style. Tasks are shown for each item where an actual or forecast costs exists in the financial period. When viewing a What If scenario or VAP budget snapshot, this reports on details from the snapshot.

The report can also be run for more than one or , from the Status and VAP report menus.

Site Overheads Report

Site Overheads Report allows you to print the details shown on the Matrix tab of the Site Overheads Workbench (or the snapshot if you are viewing a What If scenario or VAP budget snapshot).
