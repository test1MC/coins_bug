Site Overheads Workbench

The new Site Overheads Workbench is a single place in which surveyors can see and manage information about their current site overheads. Two functions provide separate routes to the workbench:

  • Site Overheads () is a browse of for which a site overheads record has been added.
  • Site Overheads () is a browse of for which a site overheads record has been added.

A link on the or column takes you to the main site overheads workbench for the or . Column sets allow you to show the information you want.

Click to add a new site overheads record. This prompts for the or and the Template (for , only General templates are available; for , only Home Building templates are available).

Click to go to the main Add Site page.

Enter the initial dates for the site.

When you save this form, the site will be added to the workbench and the selected template will be copied to the or . The initial overheads and cost allocation will also be calculated. As this may involve updating large number of records, it runs as a background report process. The context area shows the status of the initial site creation process; when it is complete, the page refreshes. If the template configuration is changed, this will not affect the existing site overhead records.
