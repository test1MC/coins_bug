Site Overheads Workbench - Initial Configuration

Parameters

The following new JC parameters affect the Site Overheads functionality.

Parameter

Description
SOCHMASK A can-do list of that are within the scope of the Site Overheads Workbench.
SOOVRUSR A can-do list of users that are exempt from Site Overheads template item override restrictions.
SOVARSGN

Whether variances on the Site Overheads Workbench should be treated as positive or negative sums.

  • N = Cost savings are treated as negative values.
  • P = Cost savings are treated as positive values.
SOCFAST Which construction forecast should be used for the equivalent unit calculation.

Terminology

The term “EFC” (Estimated Final Cost) is customable, so wherever it is used you can replace it with an alternative such as “FFC” (Forecast Final Cost) or “FCAC” (Forecast Cost at Completion). Use Terminology to enter System Override text for the token jc efc.

To use the site overheads functionality, you need to set up future that cover at least the duration of any overhead forecasts.
