Site Overhead Templates

Site Overhead Templates is a new function that allows you to configure the templates for site overheads. The templates specify the dates, , standard overhead items, and additional fields that are shown on the Site Overheads Workbench.

There are two types of template: General and Home Building; Home Building templates have additional features that are not relevant for general contracting. The template allows you to maintain the dates and fields to be shown, the and their accrual method and the individual detailed overhead items.

Template Dates

The ‘Dates’ tab allows you to define the dates that are to be updated for a . These dates form the basis of the time frame over which weekly or -based overheads items will be calculated. You can set up dates to be either entered manually, or calculated based on a number of days, weeks or months before or after another date on the template.

Template Fields

The ‘Fields’ tab allows you to define the fields which will appear on the summary tab in the Site Overheads Workbench. Any field from the record (and for home building templates, from the record) can be used.

Template

The ‘’ tab allows you to define the that can be used in the template, and their accruals calculation method. Only specified in the new JC parameter SOCHMASK can be used.

To avoid the need to make manual adjustments where the difference between the actual cost and the budget/forecast cost is small, you can specify an Allowed Difference. If the accrual method is accrual, but the difference is within this percentage, an accrual will not be created.

A row action Update Site allows you to add and update their accrual method and allowed difference on live sites in the Site Overheads Workbench. The update runs as a background report.

Template Items

The ‘Items’ tab allows you to configure the items that make up the breakdown of the site overheads.

A row action ‘Update Site Items’ allows you to add the selected items to existing live sites that don’t have the item. The check if an item exists on a site is based on the and item description. The update runs as a background report.

Generate

The Generate function on the Options menu allows you to automatically generate the for a template based on the specified in the new JC parameter SOCHMASK. You can specify the accrual method and allowed difference, which apply to each new added to the template.

Template Export/Import

Template Export/Import is a new function that allows you to export a template and its dates, fields, cost heads and items, or to import an exported template.
