Site Overheads Workbench - Summary Page

The site overheads summary page allows you to maintain the dates, items and actual/forecast costs.

Summary Tab

The ‘Summary’ tab shows the fields as configured using the template fields option, and the template dates. You can update manual dates, or the Override box to update calculated dates. If you change a date, any items with dates that are linked to it are updated, and the EFC (estimated final cost) total is recalculated.

Items Tab

The ‘Items’ tab displays a browse of the site overheads items. From here you can add bespoke items which are specific to the site; a * in the final column indicates a bespoke item. Column sets are available.

Restrictions on Updating Items

All users can add, update or delete bespoke items that are specific to the site.

Items linked to a standard template item cannot be deleted by any user. Only users listed in the new JC parameter SOOVRUSR can update all fields for all standard items. Other users can only update all fields if the Allow Override option is set on the standard item, otherwise they can only update the quantity- and price-related fields from here.

Matrix Tab

The ‘Matrix’ tab shows the detail for each item and allows you to update the amounts and forecast allocation of costs. The matrix also shows the actual cost variance, accrual and accrual reversal.

If you change a cell value for a cost head that is set up to accrue, a message in the context area informs you that the accruals need to be recalculated.

A row action allows you to allocate costs to a forecast period.

Gantt Chart

A link in the context area of the Site Overheads Workbench allows you to view a non-updateable ‘Gantt Chart’ view of the site overheads forecast. (A separate option allows you to print out a Gantt chart. See 1.9 - Options.)
