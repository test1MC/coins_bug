Quick Budget Entry - Standard

Changes have been made to Standard to allow you to specify the budget input method and rate. If using cost code/category data, these are entered at /category level, otherwise they are entered at level.

Details

If the JC parameter USECCCAT is not set to All or Used, new fields are available on the Details tab.

Fields

Field
Description

Budget Input Method

This is a drop-down selector with the following options.

  • Blank - budget costs cannot be entered against the corresponding cost code.
  • Value - the default when the default category is not .
  • Hour - the default when the default category is .
  • Quantity.

Rate

If the Budget Input Method is Hour or Quantity, this is available; it is a drop-down selector with the following options:

  • Actual.
  • Initial Budget.
  • Approved Budget.
  • Current Budget.
  • Forecast at Completion.
  • Each of the calculation programs specified in SY/BUDCALC.

Forecasting Tab

If the JC parameter USECCCAT is set to All or Used, a new section Budget Input Method is displayed.

This shows a drop-down selector for each cost category defined in JC Configuration. These have the following options.

  • Blank - budget costs cannot be entered against the corresponding cost category.
  • Value.
  • Hour.
  • Quantity.

If the Budget Input Method is Hour or Quantity, the Rate field is available for the corresponding category; it is a drop-down selector with the following options:

  • Actual.
  • Initial Budget.
  • Approved Budget.
  • Current Budget.
  • Forecast at Completion.
  • Each of the calculation programs specified in SY/BUDCALC.
