Quick Budget Entry - Parameters

The following new JC parameters are relevant to the Input Budget Workbench.

Parameter

Description
BUDCALC An optional comma-separated list of additional calculation methods (defined in System > Miscellaneous Maintenance > Calculation Programs) which are used for calculating the Budget Input Rate , which in turn is used for calculating budget cost.
BUDQTY The options for forecasting by quantity.

BUDMANUPD

A comma-separated list of budget types used for manual update; any entry in BUDMANUPD that exists in Memo Data Types and has type “U-Unlocked” is available.

If BUDMANUPD is blank, only the Forecast At Completion budget specified by FORECURR is available.

BUDVALID

Whether to display a Warning, an Error or no message, if the Budget Input Method for a category is blank and a GL Account exists for that category.
