Quick Budget Entry - Cost Code Maintenance

Changes have been made to Cost Code Maintenance. to allow you to override the budget input method and rate for the cost code. If using cost code/category data, these are entered at cost code/category level, otherwise they are entered at cost code level.

Main Tab

If USECCCAT is not set to All or Used, new fields are available on the Main tab: Budget Input Method and Rate. The default values for these fields are taken from the values on the corresponding .

Forecasting Tab

If USECCCAT is set to All or Used, a new section Budget Input Method is displayed on the Forecasting tab. This shows a drop-down selector for each category defined in JC Configuration. The default values for these fields are taken from the values on the corresponding .

Cost Codes Browse

The following columns have been added to the Cost Codes browse. If the JC parameter USECCCAT is set to All or Used, the columns show fields for each category; otherwise only one field is shown.
