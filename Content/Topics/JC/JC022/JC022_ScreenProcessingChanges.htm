Configuration

The three JC parameters listed below were added to accommodate Accrual Reporting in Status.

Parameter

Description

JC/LABACCR

Accrual Reporting

JC/INJCTACCR

Include JC Timesheets in Accruals

JC/CDRSTAT

List of CDR Status codes to be included in Accruals

Screen and Processing Changes

Accrual Rate Types

A Accrual Rate Types function is now available on the Status menu at Setup > Setup Maintenance. When selected, it displays the browse screen shown in Figure 1. You can use this browse screen to set up and maintain Accrual Rate Types.

Accrual Rate Types

Maintenance

A new Accrual Rate Type field is found in the Configuration tab of Maintenance, as shown in Figure 2. This field is only visible and enabled when the JC/LABACCR parameter is set. It lists all non-dormant Accrual Rate Types, which are set up in the Accrual Rate Types function (see Section 2.1).

Configuration Tab

Accrual Amount Calculation

The Accrual Amount is calculated from the following records:

  • Unposted PR Timecards.
  • Unposted JC Timesheets when the JC/INJCTACCR parameter is set.
  • The Daily Reports with statuses, as listed in the JC/CDRSTAT parameter, which have not been transferred to Payroll.

Accrual Amount

The Accrual Amount now appears on the following :

  • Costs by
  • /Phase Costs by Activity
  • /Phase Costs by Section
  • /Phase Costs by WBS
  • Costs by Cost Code
  • /Phase Costs by Analysis set 1
  • /Phase Costs by Cost Code
  • /Phase Costs by Cost Code Analysis set 1

When you drill down on the Accrual Amount field, you will see the details of the records that have been used for calculating that amount.
