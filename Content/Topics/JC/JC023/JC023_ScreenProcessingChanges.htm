Details Screen Changes

The Details screen was previously a tab-based screen with a group of options. The UI Details screen instead has horizontal sections and tabs for related information, as shown in Figure 1. The sections are built according to configuration; for example, information is not shown if it is not configured on the .

The Options menu on this screen only includes actions against a or Utilities.

UI  Details Screen

Horizontal Sections

Main Section

The Main section in Details contains four layouts, as shown in Figure 2: Summary, Location Information, Information, and Information.

Notable information about this screen includes:

  • The following fields were moved from the section and added to the Information layout:  Proposal Number, Architect, and Architects Project Number.
  • The Pay When Paid and Allocation Code fields have been moved from the Advanced Settings section and added to the Information layout.

Configuration Section

The Configuration section has been relabeled as Advanced Settings and moved with its contents toward the bottom of the Details screen (see Section 1.1.3). The contents of the section remaining, which is labeled as Configuration, is divided into three layouts: Description, WBS Configuration (moved from the Main section), and Analysis, as shown in Figure 3.
