PT Communication Management Changes

Daily Reports Tab

Daily Reports (Figure 1) can be accessed in Project Management > [selected Working Project] > Communication Management.

Both the Daily Reports and Review and Approval browses previously used the Number on the CDR Header to show CDRs entered for the Project for one only. The CDR now appears on each Project referenced on either the CDR Header or in any details.

  • The Review and Approval browse now uses a linking table instead of selecting CDRs by Number on the CDR Header.

PT Daily Reports Tab

Project Daily Report Summary

Daily Reports are displayed as part of Project Management > [selected Working Project] > Communication Management > Daily Reports tab > [click Date to drill down to Project Daily Report details] (see Figure 2). You can then choose Daily Reports to access Daily Reports (see Figure 3).

All CDRs linked to the current Project appear in the browse. When drilled down to the CDR Summary, detail lines display that are filtered by the current Project. You can clear the filter to see all CDR details.
