Status Changes

JC Daily Reports

If there is at least one Pay Shift record set up in Pay Shift Codes in Payroll > Global Setup > US Specifics, a new Pay Shift column is used on the JC Daily Reports screen, as shown in Figure 1.

Daily Reports

JC Quick Add

If there is a Pay Shift record in PR Pay Shift File Maintenance, Pay Shift is displayed at the top of the Quick Add tab screen in Status > Daily Reports, as shown in Figure 2.

Quick Add Tab

JC Payroll Tab

If there is a Pay Shift record in PR Pay Shift File Maintenance, Pay Shift is displayed at the top of the Payroll tab screen in Status > Daily Reports, as shown in Figure 3.

Payroll Tab

JC Detail

Pay Shift is displayed at the top of the  Detail tab screen in Status > Daily Reports, as shown in Figure 4.

Detail Tab

JC Daily Report Transaction Report

If there is at least one Pay Shift record in PR Pay Shift File Maintenance, a Pay Shift field is displayed in the Daily Report Transaction Report in Status > Daily Reports.

JC Review and Approval Vertical

If there is at least one Pay Shift record in PR Pay Shift File Maintenance, a Pay Shift column is displayed in the Review and Approval Vertical browse in Status > Daily Reports, as shown in Figure 5.

Review and Approval Vertical Pay Shift

JC Accrual Rate Types

If there is at least one Pay Shift record in PR Pay Shift File Maintenance, a Pay Shift column is now displayed in the Accrual Rate Types browse in Status > Daily Reports.

Accrual Amount Calculation

Amount recalculation was changed to now consider Pay Shift Rates entered in Accrual Rates Maintenance.
