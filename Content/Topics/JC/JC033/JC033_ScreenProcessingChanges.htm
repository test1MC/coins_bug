The description of the JC Parameter CDRLTFLD - Crew Daily Report Labour Tab Extra Fields now includes two new options listed below (also see Figure 1). They are available for the PayCompany with Tax/NI Type 6 and 11 (USA and Canada).

• PWTRADE – Prevailing Wage Trade

• PWSKILL – Prevailing Wage Skill Level

3.1 Daily Reports

New fields PW Trade and PW Skill Level are found on the Labour Quick Add screen and Payroll browse of Contract Status > Daily reports > Daily Reports.

NOTE:

Prevailing Wage Trade and Prevailing Wage Skill fields are only available on Crew Daily Reports when the Tax/Ni Type of PayCompany is 6 and 11 (USA and Canada).

The following new fields are found in Daily Reports Detail of the Additional Columns on Labour Tab section (Figure 3):

5 Other Changes

5.1 COINS JSON Services Changes

The following changes have been made to JSON Services in order to accommodate PW Wage and PW Skill functionality.

• Publish PWTRADE and PWSKILL – PWSKILL and PWTRADE records were added to the CDR mobile app. This provides the user with a list of all prevailing wage skills and trades to select from when adding a new employee on the report.

• Change Contract Labour - In addition to Labor Skill and Trade, Prevailing Wage Skill and Trade have been added to the CDR Mobile app.

• Change Contract Job - The CDR Mobile app has been changed so that it can identify if the job or any of the job section has the Prevailing Wage Schedule set.

• Change Submit Daily Report – Since the CDR mobile app returns the Prevailing Wage Skill and Trade, OA needed to be able to handle them.

5.2 COINS Mobile Changes

5.2.1 Display Prevailing Wage

The app displays the PW fields only if Prevailing Wage is set for the job.

Figure 16: (JC033_CrewMemberReviewSign.png)Crew Member Review and Sign
