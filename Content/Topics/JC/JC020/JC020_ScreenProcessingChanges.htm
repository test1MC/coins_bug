Listing Report

Menu Item

The new Listing Report can be found in Status > Reports > Master , as shown in Figure 3. This report has been developed for Master . It lists all Subsidiary and their values for each Master selected.

Listing Menu Item

Selection Screen

Once you select Listing from the Reports > Master menu, you are presented with the selection screen shown in Figure 4. There are two tabs on this screen: Main and Sort, which are explained below.

Listing Selection Screen

Main Tab

The Main tab for the Listing Report is shown in Figure 4 above. Along with the standard selection screen fields, it also includes the following fields:

The value in this field defaults from the current period date.

Currency

This drop down field defaults to the Company's default Currency Code. Only the visible company is set to multi-currency.

Field Set

This field allows you to select the field set (a grouping of related fields) to display on the report.

A number of pre-defined field sets exist, but others can be created as required; please contact COINS if you require additional field sets.

Sort Tab

The Sort tab for the Listing Report is shown in Figure 5. It contains the following fields:

Sort By

Includes drop down options for sorting by Group, Type, Location, and Analysis fields.

Totals

When ed, the report shows the totals for the sort options.

Page Level

After selecting how you wish to sort the report, you can also select the number of the sort level that controls page breaks. The report will start a new page each time the value of this sort field changes.

Cost Code Detail Report

Menu Item

The Cost Code Detail Report can be found in Status > Reports > Master , as shown in Figure 5. This report has been developed for Master . It shows the details of a Master and the corresponding Subsidiary at the Cost Code level. Subtotals are shown per Subsidiary , and the Grand Total is shown per Master .

Cost Code Detail Menu Item

Selection Screen

Once you select Cost Code Detail from the Reports > Master menu, you are presented with the selection screen shown in Figure 8. There are three tabs on this screen: Main, Cost Code, and Sort. The Main and Sort tabs are discussed above in Section 1.2.2, Selection Screen. The Cost Code tab uses the standard selection screen fields.
