Parameters

This section describes the parameters that are related to Leave of Absence (LOA) processing.

HR System Parameters

Two HR System Parameters (Figure 1: HR System Parameters) found in  OA in System > System Setup > Global Parameters (under HR Module) are now used for Leave of Absence (LOA) processing. These parameters are as follows:

  • STATLOA - a new parameter used in HR Status to indicate a Leave of Absence.
  • STATACT - used in HR Status to represent an Active status (such as ACT).

Company Parameters

Two Company parameters, TIPRCO and TIPRLOA, are found in System > Company Setup > Configuration > Company Parameters, as shown in Figure 2: Company Parameters. They are used in the OA HR/Ti PR Leave of Absence process, as explained below.

Company Parameters

TIPRCO

When filled in with a company number, this field indicates that the user is using Ti Payroll.

Note: Benefit Enrollments are not end-dated when an HR LOA record is created in OA HR Employment History.

TIPRLOA

Whether the Layoff Date can be used to indicate an employee's Leave of Absence (L) or Termination (T). If set for Termination (T), OA treats the Layoff Date from Ti PR as a Termination Date.
