HR Employees in Multiple Courses: Training/Assessments

This section describes how HR Training/Assessments can be used to allow multiple course for employees.

Main Browse

The Training/Assessments browse screen now contains two tabs (as shown in Figure 1: Main Training/Assessments Browse): Training/Assessments and Group .

  • The Training/Assessments tab was previously shown as a browse with no tabs. Even though it now shares the screen with another tab, its browse remains the same as before.
  • The new Group tab is discussed below in Section 1.2.2.
Main Training/Assessments Browse

Group Tab

A new Group tab has been added to the Human Resources > Training/Assessments function, as shown in Figure 2: Group Enrollments Tab. This is where you can add and maintain information about Course Group .

Group Enrollments Tab

Field Definitions

Group Code

The code for the course group, which links to a Courses Group Assignment browse (see Section 1.2.3).

Date From

The start date of the course being offered.

Ideal # Attendees

The ideal number of Attendees on the Course Master record. The value in this field shows the minimum value against all included in the group courses.

# Attendees

The number of enrolled attendees.

# Courses

The number of enrolled courses.

Archived

Whether the course is archived.

Courses Group Assignment

When you drill on the Group Code field in the Group tab, you are taken to a Courses Group Assignment browse, as shown in Figure 3: Courses Group Assignment Browse. This browse contains the following tabs:

  • Details (see Section 1.2.2.2)
  • Add Personnel (see Section 1.2.2.3)
  • Group (see Section 1.2.2.4)
  • Enrolled Personnel (see Section 1.2.2.5)
Courses Group Assignment Browse

Details Tab

The Details tab on the Courses Group Assignment browse is shown in Figure 4: Details Tab. This screen gives you more information about the courses group.

Details Tab

Type

The type of course, such as Training or Assessment.

Course

The code used for the training course/assessment.

Name

The name of the training course/assessment.

Venue

The venue where the training course/assessment if being held.

Places

The number of places for the training course/assessment.

Start Time

The start time for the training course/assessment.

Add Personnel Tab

The Add Personnel tab on the Courses Group Assignment browse is shown in Figure 5: Add Personnel Tab. This screen allows you to add one or more employees to a training course/assessment. You can select employees and then use the Selected action (Figure 6: Enroll Selected Action) to them.

Enrolled Personnel Tab

The Enrolled Personnel tab on the Courses Group Assignment browse is shown in Figure 8: Enrolled Personnel Tab. This screen includes the status of each employee attending the training course/assessment. Even if an employee may not necessarily be enrolled, you can still select Confirmed, Attended, Successful, etc.
