HR Lookup Codes Maintenance

To maintain HR Lookup Codes, Navigate to the HR Module and open the HR Maintenance sub-menu. Select the Lookup Codes option.

 

 

Filtering the relevant Lookup Codes

Before being able to add the lookup codes it is required to filter to the relevant data set. For Example: if you wish to add the Types of s.

Go the bottom of the work bench under the “Selectors Section” and Select the “Lookup Type“ you wish to use from the drop-down menu ( in this example: “ HR type ”)

 

· Click Apply Filter.

You must click apply filter otherwise the screen will not be set to the correct code when you start to add/amend.

Amending a Lookup Code

 

To amend a lookup code, ensure that you set the filter at the bottom of the screen to the correct Lookup Code Type and Apply the filter.

Select the code to be amended and click against the code.

You can amend the description, but you cannot amend the code. To prevent un-used codes from appearing in the drop-down lists within the Personnel Workbench, click the dormant box. To re-instate dormant codes, simply un- the dormant box.

Click to save your changes.
