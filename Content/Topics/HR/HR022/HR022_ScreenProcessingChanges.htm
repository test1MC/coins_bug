Screen and Processing Changes

HR Disciplinaries

Additional fields are now available on the HR Disciplinaries screen in Human Resources > Personnel Workbench > [Personnel link] > Events tab menu, as shown in Figure: HR Disciplinaries Fields. These fields allow for more options when recording a disciplinary action.

HR Disciplinaries Fields

Field Definitions (Selected Fields)

Superintendent

The Superintendent of this disciplinary action. This is in addition to the existing Manager field.

General Foreman

The General Foreman of this disciplinary action. This is in addition to the existing Manager field.

Foreman

The Foreman of this disciplinary action. This is in addition to the existing Manager field.

Level

The Level assigned to this disciplinary action. The HR Disciplinary Levels available in this field are set up in HR Maintenance > Lookup Codes.

Site

The name of the Site where this disciplinary action occurred.

Location

The location of the Site where this disciplinary action occurred,

Unsafe Work Act

A description of the unsafe work act involved in this disciplinary action.

Witness 1

Witness 2

The names of up to two witnesses for the disciplinary action.

Steward

The Steward of this disciplinary action.
