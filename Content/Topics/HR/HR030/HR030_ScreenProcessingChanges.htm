Selection Screen

The Incidents/Accidents screen in Health & Safety > can contain Analysis Set selection criteria (Figure: Analysis Sets Selection Criteria Screen). This screen also now contains an Type field (Figure: [%=Variables.Enquiry caps%] Type Field), which allows you to select which type of you would like to generate (Incidents Count by , Incidents Count by Year, or Accidents Count by Company).
