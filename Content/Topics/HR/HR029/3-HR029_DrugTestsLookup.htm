Function Specific Lookup Codes: Drug Tests Lookup Codes

The Drug Tests Lookup Codes function allows you to set up and maintain Drug Test related Lookup Codes.

Menu Option

A new Drug Tests Lookup Codes option is found on the HR Maintenance menu, as found in Figure: Lookup Codes.

Lookup Codes

Screens

When you select Drug Tests Lookup Codes from the menu shown above, a screen with numerous tabs appears, as shown in Figure: Lookup Codes Screen. You can use this screen to set up and maintain drug related Lookup Codes. Each code is a tab within the function.

Code, Description, and Dormant fields are found on all of these tabs.
