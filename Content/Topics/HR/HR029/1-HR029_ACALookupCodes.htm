Function Specific Lookup Codes: ACA Lookup Codes

The ACA Lookup Codes function allows you to set up and maintain ACA (Affordable Care Act) related Lookup Codes for the US market only.

Menu Option

The new ACA Lookup Codes option is found on the Human Resources > Administration > ACA Processing menu (as shown in Figure: Menu Option) and the Payroll > Administration > ACA Processing menu.

Menu Option

Screens

When you select ACA Lookup Codes from the menu shown above, a screen with numerous tabs appears, as shown in Figure: ACA Lookup Codes Screen. You can use this screen to set up and maintain ACA related Lookup Codes for the US market only. Each code is a tab within the function.

ACA Lookup Codes Screen

Code, Description, and Dormant fields are found on all of these tabs. The only exception to this is the ACA Coverage is Affordable field, which is found only on the ACA 4980H Safe Harbor Codes tab.
