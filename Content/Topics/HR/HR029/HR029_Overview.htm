Overview

Lookup Codes are simple file maintenance functions embedded in a single function for the Human Resources module. In order to provide better security for Lookup Codes, the following new functions have been created in HR: ACA Lookup Codes, Benefit & Accrual Plans Lookup Codes, and Drug Tests Lookup Codes. These functions contain process specific Lookup Codes that are shown as tabs.

The Lookup Codes set up in these Lookup Code functions are available in the HR Personnel Workbench.

This document provides more detail on these new Lookup Code functions in HR.
