Function Specific Lookup Codes: Benefit & Accrual Plans Lookup Codes

The Benefit & Accrual Plans Lookup Codes function allows you to set up and maintain Benefit related Lookup Codes for the US/CA/AUS/ME markets only (ME companies also include Plan Workbench and Benefit Groups items).

Menu Option

A new Benefit & Accrual Plans Lookup Codes option is found on the HR Maintenance > Benefits menu, as found in Figure: Menu Option.

Menu Option

Screens

When you select Benefit & Accrual Plans Lookup Codes from the menu shown above, a screen with numerous tabs appears, as shown in Figure: Lookup Codes Screen. Each code is a tab within the function.

Note: The Standard Uniform Premium Type Tab is for the US/CA regions only.

Lookup Codes Screen

Field Definitions (Selected Fields)

Code, Description, and Dormant fields are found on all of these tabs. Exceptions to this are:

Group (*Accrual Plan Type tab only*)

The specific Group for the code.

Accrual Anniversary Date (*Accrual Plan Type tab only*)

The specific type of accrual anniversary date for this code. The options are:

  • A-Computed Adjusted Service Date
  • C-Custom Adjusted Service Date
  • H-Hire Date
  • M-Manual Adjusted Service Date
  • R-Rehire Date

ESS Displayable (*Plan Type tab only*)

This field determines the actions in ESS for the code, as follows:

  • When ed, the code is displayed in ESS.
  • When not ed, the code is not displayed in ESS.

ESS Tile Type Group (*Plan Type tab only*)

The name of the ESS Tile Type Group for the code. Examples of options are: 401k, Dental Plans, Medical, Flexible Spending, LTD Plans, STD Plans, and Vision.

Monthly Max on Pay Calculation (*Plan Type tab only*)

The specific type of monthly maximum on the Pay Calculation for this code, such as:

  • A-Pass Monthly Maximum for ALL calculations
  • B-Pass Monthly Maximum for Both Employee & Employer Calculations
  • E-Pass Monthly Maximum for Employee Calculation only
  • I-Pass Monthly Maximum for Imputed Income Calculations only
  • N-Do not pass Monthly Maximums
  • R-Pass Monthly Maximum for Employer Calculation only
