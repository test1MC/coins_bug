New Level of ESS Approvers: ESS Approver Workbench

The ESS Approver Workbench includes functionality that lets you see the second level of Approvers set up on the HR Personnel Workbench.

Approvers

The Approvers field (see Figure: Approver Details) on the Timesheet Approval tab in Employee Self Service > ESS Portal > ESS Approver Workbench can now display a comma-separated list of Approvers. It also now includes the User Ids of Approvers set up in the HR Personnel Workbench. Previously, it only contained the current Approver set on the Timesheet Group in ESS. When drilling into the Timesheet Group on the ESS Approver Workbench, it is now possible to view the second level of Approvers who have been set up on the Human Resources Personnel Workbench.

Submit Current Report Line Action

The Submit Current Report Line action is found on the Timesheets by Date screen in Employee Self Service > ESS Portal > Approver Workbench > Awaiting Submit tab > Employee ID link > Timesheets, as shown in Figure: Submit Current Report Line Action. This action allows an Approver to submit time for the Employee he/she is responsible for; for example, when the Employee is sick or on . This now includes a second level of Approvers defined in the HR Personnel Workbench.

Timesheet Managers Workbench

A Timesheet Managers Workbench appears on the Timesheet Approval tab in Employee Self Service > ESS Portal > Approver Workbench (see Figure: Timesheet Managers Workbench) when a Manager is added to the ESS TS Manager field in the HR Personnel Workbench. All Managers from these records are displayed with WE Date and Period Desc as columns and TS Period as a body filter. This HR Approver has the same functional access as the ESS Timesheet Group Approver, thus providing the additional ability to be set on an individual employee, not solely on the ESS TS Group.
