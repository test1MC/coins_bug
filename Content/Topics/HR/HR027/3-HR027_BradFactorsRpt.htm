Bradford Factor Calculation: Bradford Factors Report

The Bradford Factors Report allows you to report on personnel with Bradford Factor values in a selected range.

Menu Option

The new Bradford Factors report (Figure 1: Report Menu Option) only appears on the Human Resources > Reports > Personnel Statistics menu if some Bradford Factors have already been set up.

Report Menu Option

Report Selection Screen

When you select Bradford Factors from the menu (see Figure 1: Report Menu Option), a selection screen appears, as shown in Figure 2: Report Selection Screen. This screen contains From/To fields for each of the configured Bradford Factors.

If both the From and To values for a given Bradford Factor equal zero (0), that zero value is not included on the report.

Report Selection Screen

Field Definitions

Sort By

Select whether to sort the report By Bradford Factors (see Figure 3: By Bradford Factors Sample Report for sample report) or By Hr Group/Subgroup (see Figure 4: By HR Groups Sample Report for sample report). Data is sorted by Personnel , then First Name.
