Overview

A statistic called the Bradford Factor assesses personnel absence patterns. The Bradford Factor is calculated as follows:

B = S² x D

where:

  • B is the Bradford Factor score
  • S is the total number of spells (instances) of absence of an individual over a set period
  • D is the total number of days of absence of that individual over the same set period

Improvements have been made in the Human Resources module so that it is now easier to set up, view, and report on Bradford Factors for HR personnel. These include a new Bradford Factor Configuration page for setup, and a new Bradford Factor Report that reports on these statistics.

This document describes all of these changes in more detail.
