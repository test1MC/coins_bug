Bradford Factor Calculation: Personnel Workbench

This section describes how the HR Personnel Workbench is set up to use Bradford Factor calculations.

Bradford Factors Layout

A new Bradford Factors layout is found on the Other tab of the Personnel Workbench, as shown in Figure 1: Bradford Factors Layout. It displays only if Bradford Factor values have been set up in Bradford Factor Configuration.

Bradford Factors Layout

Column Sets

The configured Bradford Factors also appear on the list of available Column Set fields, as shown in Figure 2: Column Set with Bradford Factors. This ensures that they can be included on an initial browse screen view, if desired.
