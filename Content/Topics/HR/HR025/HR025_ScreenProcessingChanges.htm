Screen and Processing Changes

The information below describes how OA users in the Australia region can now use the Human Resources module to track the Visa information of employees.

HR Visa Information Function

A new Visa Information drop down option is found on the Australia menu in Human Resources > Personnel Workbench > Events tab, as shown in Figure 1: Visa Information Option. You can use this option to add and maintain Visa information for an employee.

Visa Information Option

Main Screen

After you select Visa Information from the drop down menu, the Main screen appears, as shown in Figure 2: Visa Information Main Screen. You can add or update Visa Information records on this screen.

Visa Information Main Screen

Add/Update Screen

When you add or update Visa Information records from the Main screen (Figure 2: Visa Information Main Screen above), a selection screen appears, as shown in Figure 3: Visa Information Add/Update Screen. Use this screen to enter or update information for these records.

Visa Information Add/Update Screen

Field Definitions (Selected Fields)

Visa Type

The type of Visa, which may include lookup options such as 457 Visa, Working Holiday Maker, Bridging Visa, etc. Lookup codes for this field are set up under the VISATYPE lookup type.

Residency Status

The type of Residency status, which may include lookup options such as Temporary, Permanent, etc. Lookup codes for this field are set up under the RESISTATUS lookup type.

Sponsor Relationship

The type of Sponsor Relationship, which may include lookup options such as Company Sponsor, Individual, Father, Mother, etc. Lookup codes for this field are set up under the SPONREL lookup type.

PR Missing Timecard Report

The Missing Timecard Report in Payroll > [frequency] Payroll > Reports determines if an employee has a corresponding HR personnel record. If so, it can now check if there is an active Visa (the Visa Date is greater than the current time card period Effective Date).

A message appears if the time card hours do not meet the minimum and maximum hours of the Visa criteria of an employee. (The Earnings Categories configured against the PR/PR-BASICRAT parameter are used to check the total hours worked for the employee.)

HR Visa Information Report

A new Visa Information Report function is found on the Australia menu in Human Resources > Reports > Personnel. It allows you to show the Visa details of employees. The Main tab of the report selection screen is shown in Figure 4: Report Selection Screen (Main tab). (The Personnel tab is standard and does not need to be shown here.)

Report Selection Screen (Main tab)

Field Definitions

From/To Issue Date

The starting/ending issue dates of the Visas to show on the report.

From/To Date

The starting/ending dates of the Visas to show on the report.

Visa Information Details

Select the field set you wish to display on the report. Field sets are set up in OA Reporting & BI > Field Sets > Report Field Sets.
