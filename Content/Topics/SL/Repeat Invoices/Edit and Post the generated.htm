Edit and Post the generated Repeat Invoices

To review and edit the generated repeat invoice(s) go to Sales Ledger Input menu folder and select Invoice & Credit Note Entry. There you can see a list of all sales ledger batches, where invoices were created by using the Invoice & Credit Note Entry menu. Use the navigation arrows to go to the last entered records.

Click on the underlined Batch Number to see the invoices in the batch. Again click on the underlined invoice Internal Reference number, to access the invoice details.

Note:  You can only edit the batches, where the batch Status is ‘Unposted’.

Use the Open button to open the invoice details and edit the fields that need changing.

To apply the changes and to save the invoice, select the Next button at the bottom left corner of the page. And click the Next button AGAIN, to exit and save the batch.

To Post the batch, select the batch to be posted and Set the Action Dropdown at the bottom of the screen to Post, then click the Apply button.
