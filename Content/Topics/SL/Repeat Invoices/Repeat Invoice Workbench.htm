Repeat Invoice Workbench

In the Repeat Invoice Workbench you can review and edit EXISTING repeat invoices, including the invoiced amount, but cannot add new ones. You access it by selecting the Sales Ledger module, the Input folder menu and the Repeat Invoice Workbench

 

Make sure the Filter is set to All Invoices to ensure you can see all of the already setup repeat invoices templates. Use the Search option to find the one you are looking for. When you select a new filtering or search option, click the Apply Filter button to apply the changes to the screen.

By clicking on the underlined repeat invoice number (RINVxxxxxx) in the Invoice column you can view the individual repeat invoice template, edit it (the Invoice tab) and review the invoicing to date (the Actual Invoices tab with links to the Invoice Details information).

 

You can only edit certain information under the Invoice tab. Use the Open button () to open and change information. Go to the Lines tab to change the Net Amount and/or the invoice text and to the Revenue Distribution tab to edit the distribution.

Note any changes you make Repeat Invoice Workbench are changes you make to the Repeat Invoice template and will affect all the invoices generated from this template from that moment on.
