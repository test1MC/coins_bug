Repeat Invoice Template

When you have saved the initial setup details, the next stage of the Repeat Invoices is displayed.

Select the Invoice Tab.

Click to setup up the template details.

Since you are setting up a repeat invoice template, remember to enter more general descriptions in the Customer Reference and Description field of the Invoice tab.

Please note to leave the checkbox next to the multiple lines ticked and NOT to enter the Net Amount under the Invoice tab. You will enter the invoice amount under the Lines tab that will appear after you select the save button on this screen.

For more detailed instructions on entering invoices see the guide HOW TO…Enter Sales Ledger Invoices and Credit Notes.

Click to store the template header details.  Additional Tabs will then be made available.

Select the lines Tab.

 

Click to begin adding lines to the invoice. Again remember, whatever you enter will appear on all the generated repeat invoices created from this template.

Note the line that opens in the repeat invoice template is somewhat different from the one in standard Invoice entry. Here you need to enter the Analysis and the VAT Code (the required fields) for each entered line, even if the entered value is 0

Enter the Description, the Quantity, and the Price.  You may select the Unit and the VAT Code from the dropdown menus.

Enter the Analysis. If you have entered the contract number under the main details in the Invoice tab, the contract part of the analysis will be automatically populated. Contract number needs to be followed by a full stop ‘.’ and then the cost code.

For example: 10000.B1234

To save the entries, click located at the beginning of the line and continue to the next line, always entering the VAT Code and the Analysis.

 

The Revenue Distribution Tab

 

When you have entered all the lines, select the Revenue Distribution tab. If you have entered lines with zero values, you should have at least two entries : one where the Amount is 0, and one or more with real values in the Amount column.

To prevent the 0 appearing on various reports, it is important you delete that line from the repeat invoice template.  You do that by ticking the box () at the beginning of the line with the 0.00 amount and selecting the Delete button ().

The basic setup of the recurring invoice template is now complete, so you select the Next button at the bottom of the page to save the invoice template.

Review the invoice template and when you are confident all the details are entered correctly, return to the Main tab of the Repeat Invoice template. Tick the Active box, to activate the generation of the repeat invoices and select the Save button

You can un- and deactivate the template at any time to stop the generation of the repeat invoices. Just select the Open button in front of the appropriate repeat invoice template on the Repeat Invoice Workbench screen.
