Report Scheduler

The Repeat Invoice function makes use of the Report Scheduler in OA to control the frequency of invoicing.  In this example, we will set up a new monthly schedule called MTEST.

Navigate to System > Report Setup > Standard Schedules

Set the type filter to the correct type of schedule that you wish to create.  Here we have set it to Monthly.  Click ADD.

Set up the schedule as required.

In our example below, the schedule will run on the 26th day of each month.
