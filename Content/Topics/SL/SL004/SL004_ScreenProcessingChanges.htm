Screen and Processing Changes

SL Print Statements

You can use the new Division option in the Statement By field of the Main tab of SL Payments > Print Statements to sort data by Division (Figure 1: Division Option). This option also lets you use the GL sector address (not the company address) in the header of the SL Print Statements Form.

Division Option

Service Order Selection

If Service Management is configured, the Service Order Selection tab is displayed on the SL Payments > Print Statements screen, as shown in Figure 2: Service Order Selection Tab.

Service Order Selection Tab

Include Zero Balance Invoices Option

The new option Include Zero Balance Invoices (Figure 3: Include Zero Balance Invoices) on the Main screen allows you to print invoices with zero balances. This option is not ed by default.

The value of this same option on the Extras tab of the SL Statements Print forms is d; if ed, invoices with zero balances are printed.

Include Zero Balance Invoices

Include Related Invoices Option

The new option Include Related Invoices (Figure 4: Include [%=Variables.Jc Contract caps%] Related Invoices) allows you to print related information. This option is ed by default.

If the option is ed and a CSCERT form is defined on an SL Print Statements form, related data are printed.

Include Related Invoices

Print CB Unallocated Cash

The Print CB Unallocated Cash utility lets you print CB unallocated cash created through the Quick Cash Receipt Entry and Quick Cash Payment Entry utilities, as well as CB unallocated cash linked to CS and SL ledgers and created through the Cash Transaction Entry utility.
