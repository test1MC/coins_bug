SL parameter SECSTMT was added to the module.

Parameter

Description

SL/SECSTMT

Allow to print SL statements by sectors (Y/N)

This SECSTMT parameter lets you print SL statements by sector without switching on the SL parameter SECTLED. Different combinations of these two parameters result in the following:

(GL) SECTOR=Y, (SL) SECTLED=Y, (SL) SECSTMT = N - Division is on the Statement Screen

(GL) SECTOR=Y, (SL) SECTLED=Y (SL) SECSTMT = Y - Division is on the Statement Screen

(GL) SECTOR=Y, (SL) SECTLED=N (SL) SECSTMT = N - Division is not on the Statement Screen

(GL) SECTOR=Y, (SL) SECTLED=N (SL) SECSTMT = Y - Division is on the Statement Screen

 

NOTE:

If SL parameter MULSECST = N, Select Division in > Payments > Print Statements appears as a combo field (Figure 1: Division Combo Field); if SL parameter MULSECST = Y, Division appears as a Division Selection section (Figure 2: Division Selection).
