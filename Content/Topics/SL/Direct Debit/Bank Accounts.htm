Bank Accounts

New fields have been added to Fields in Bank Accounts to allow you to override the default identifiers used in Australian ABA files.

Fields

Field
Description

User Preferred Specification

A text string to identify the user, often their name. This is required for Australian ABA files and is usually dictated by the financial institution. If blank, the value from the SL/BACSUSER parameter is used.

User Identification Number

A number to identify the financial institution. This is required for Australian ABA files. If blank, the value from the SL/BACSUSER parameter is used.
