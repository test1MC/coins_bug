Direct Debit Receipts - Statement Print

If the new SL parameter USEDD is set to Y, new additional fields are available on the Statement Print selection screen.

Field

Field
Description

Create Direct Debit Records

If uned, no Direct Debit records are created. This means you can run the process multiple times, to see what is produced on the statement print before finaling. The email and FAX options (if configured) are also disabled to prevent customers getting multiple emails or faxes.

If this is ed, running the statement print will generate direct debit records for those customers configured as Direct Debit.

Bank Code

The bank code that the cash receipt will be linked to when the receipt batch is created from the new Approval Workbench. This is only enabled if Create Direct Debit Records is ed.

These fields are disabled if the Direct Debit approval process is running.
