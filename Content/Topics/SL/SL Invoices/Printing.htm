Printing

Print Invoices


From the Sales Ledger module select the Input submenu and select Print Invoices.

 

Select the Form from the dropdown list and enter the criteria for the invoices that are to be printed in the relevant fields.

From the Main tab you can select the invoices using the following criteria.

Fields

Description
Reference A range of reference number using lookups in the From and To fields
Date A Date range using the From and To fields
Batch Invoices from a particular Batch

From the Customer Selection tab you can select the invoices using a variety of criteria.

Use the lookups in the From and To fields to select a range of invoices by Customer, Status Code, Currency or Customer Name.

To print invoices for specific Customers, Status Codes, Currency or Customer Name use the lookup in the Matches field.
