Multiple Lines Credit Note

The multiple lines credit note enables you to enter and cost each of the credit note lines individually. This may be used if there is more than one item purchased and/or the revenue from each item will be distributed to different contracts or ledgers.

Note - You can add the multiple lines invoice to the same batch as a consolidated invoice.

On the Invoices Tab ensure that Multiple Lines is ticked

Click

Select the Lines tab to enable you to enter and cost individual lines for the credit note

Click

Enter details of the lines in the relevant fields.

Field

Description
Line Defaults to the next line number in sequence
Product Use the lookup to select the code for the item being sold
Description Enter free text to describe item being sold for example Engineering Bricks
Quantity The amount of items being sold
Price Enter the price of the item relating to the unit
Unit How the item is being sold for example EA = Each, UN = Unit
VAT Code The VAT Code by using the dropdown list
Analysis Revenue analysis code for this invoice line. This will be made up of the Contract Number and a for example Contract 1000 and Other Sales Revenue 9100

Click to save the line. A new line will be given to enter further distribution.

When all entries are complete click to remove the additional line.

Select the Revenue Distribution tab.

The revenue will have been distributed automatically by consolidating all lines for any one contract. This can be changed by selecting the Open button and making necessary amendments.

Select the Credit Control Notes tab if any notes are required for Credit Control.

Click to enter the details.

Click to save the line.

Click and a list of invoices/credit notes in this batch will be displayed.

Click to return to the batch listing
