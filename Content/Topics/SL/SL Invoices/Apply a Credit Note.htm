Apply a Credit Note

This menu is used for internal adjustments. It enables you to allocate credit notes to an invoice, for example, an invoice has been entered twice.


From the Sales Ledger module select the Input submenu and select Apply Unallocated Cash

    Click and create a batch.

Click

Select the Transactions tab and Click to enable entry of the customer details.

Enter the Customer Account required for the allocation and select the Save button

Select the To Allocate tab. A list of transactions on this account will be displayed

Locate the invoice and credit note to be balanced and select the Tick Box adjacent to the details. Select Pay Invoices from the Choose Action dropdown list and select the Apply Action button.

 

The Amount Paid column will now show the values. Select the Forward button to return to the Transactions tab.

 

 

Click to return to the batch listing.

You can now Post the batch.
