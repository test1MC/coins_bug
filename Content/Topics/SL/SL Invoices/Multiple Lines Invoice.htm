Multiple Lines Invoice

 

The multiple lines invoice enables you to enter and cost each of the invoice lines individually. This may be used if there is more than one item purchased and/or the revenue from each item will be distributed to different contracts or ledgers.

 

You can add the multiple lines invoice to the same batch as a consolidated invoice.

From the Sales Ledger module, create an invoice batch

Select SINV – Sales Invoice from the Transaction Type dropdown list.

Click

Select the Invoices tab and Click

The Invoice tab will be displayed enabling you to enter the details for the invoice. Enter the details of the invoice in the relevant fields ensuring the Multiple lines Tick Box is selected.

Field

Description
Customer Account The customer account number that this invoice is to be debited to
Name This is auto populated from Customer Maintenance record
Address This is auto populated from Customer Maintenance record
Transaction Type Defaults from the batch header but can be changed using the dropdown list
Analysis Contract the invoices will be costed to
Customer Ref Customer Order number where available
Description A description that explains what the invoice is for. This will appear on the customers’ paperwork
Invoice Date Defaults to the transaction date entered in the batch header. This can be changed
Due Date Automatically calculates using the Customer Maintenance record, for example the Customer record shows that the invoice will be due 30 days from invoice date
Discount Date Automatically calculates using the Customer Maintenance record, for example the Customer record shows that to be eligible for discount the invoice must be paid within 15 days from invoice date
Sales Analysis Allow categorisation of invoices for reporting purposes, for example Salesman initials
Multiple Lines To enter the sales invoice line by line. When selected the Net Amount Discount, GST and Gross Amount fields will grey out
Net Amount Net Amount of the invoice
Discount % Any agreed discount percentage to be applied to the invoice
VAT Select the VAT code from the dropdown list

Click

Select the Lines tab to enable you to enter and cost individual lines for the invoice.

Click

Enter details of the lines in the relevant fields.

Field

Description
Line Defaults to the next line number in sequence
Product Use the lookup to select the code for the item being sold
Description Enter free text to describe item being sold for example Engineering Bricks
Quantity The amount of items being sold
Price Enter the price of the item relating to the unit
Unit How the item is being sold for example EA = Each, UN = Unit
VAT Code The VAT Code by using the dropdown list

Analysis

Revenue analysis code for this invoice line. This will be made up of the Contract Number and a for example Contract 1000 and Other Sales Revenue 9100

Click to save the line. A new line will be given to enter further distribution.

When all entries are complete click to remove the additional line.

Select the Revenue Distribution tab.

The revenue will have been distributed automatically by consolidating all lines for any one contract. This can be changed by selecting the Open button and making necessary amendments.

Select the Credit Control Notes tab if any notes are required for Credit Control.
