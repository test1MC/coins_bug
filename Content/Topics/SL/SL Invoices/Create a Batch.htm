Create a Sales Ledger Batch

 

Before entering any transactions in the Sales Ledger a batch will need to be created. Each batch has an automatic unique batch number and an internal reference number (automatically allocated or defined). The process to create a batch header is generic to all ledgers. The exception within the Sales Ledger is when you are entering Invoices and Credit Notes.

When creating a batch to input invoices and credit notes you will need to select a Transaction Type in the batch header.

From the Sales Ledger module, select Invoices > Invoices and Credit Note Entry.

Click

Enter the batch header details in the relevant fields.

 

Field

Description

The period end date that all the transactions in this batch will be posted to, for example 210 – 28-02/10. 2 = the month, 10 = the year, 28/02/10 = the month ending

Transaction Date

The transaction date used for all transactions with this batch

Transaction Type

Select the transaction type of the transactions included in the batch. Select alternative types for individual transactions

Description

A unique description to identify the batch for example xxxxxx

 

Click

The Invoice Entry Batch Screen will be displayed.

Two tabs are available, Batch and Invoices.

The Batch Tab shows the batch header information. The batch number has been allocated, along with the reference number. The description is also displayed to enable you to identify the correct batch.

Select the Invoices tab to enter the details for the transactions to be included in the batch.
