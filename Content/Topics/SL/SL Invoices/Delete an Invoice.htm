Delete an Invoice

If an invoice has not yet been posted, it can be deleted from the batch, e.g. if the invoice has already been added to a batch, however the customer cancels their order and therefore the invoice does not need to be posted.

From the Sales Ledger module select the Input submenu and select Invoice and Credit Note Entry.

Locate the batch where the invoice is listed. Use the hyperlink in the Batch No. column to enter the batch invoices.

 

Select the Invoices tab. Locate the invoice to be deleted and select the Tick Box adjacent to the invoice details. Click .

The system will display a message asking ‘Delete this record? Deletion is permanent and cannot be undone’. Select the OK button.

 

The Invoices tab will be displayed. Click to return to the batch listing.

If an invoice has been posted you cannot delete it. A credit note should be raised for the full amount of the original invoice.

You can now post the batch.
