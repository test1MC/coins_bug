Amend a Posted Invoice

If any amendments need to be made to the invoice, for example the Customer Reference was incorrect and needs to be amended, this has to be done in the Change Posted Invoice menu.


Note – Invoice Date and Transaction values cannot be amended on a posted invoice.

From the Sales Ledger module select Change Posted Invoice.

Locate and select the invoice to be amended.  Click .

The Invoice tab will be displayed so that the amendments can be made.

 

When complete, click

The system will return to the SL Change Posted Invoice browse list.
