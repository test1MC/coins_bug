Enter a Credit Note

From the Sales Ledger module select the Input submenu. Select Invoice and Credit Note Entry option.

Select the Add button and create a batch. Select SCRN – SL Credit Note from the Transaction Type dropdown list.

Credit notes can be entered as an SCRN as described in this section, or alternatively you can use still use SINV (as an Invoice) in which case you must enter negative amounts.

Click

Select the Invoices tab and Click

The Invoice tab will be displayed enabling you to enter the details for the credit note. Enter the details of the invoice in the relevant fields.

 

 

Field

Description

Customer Account

The customer account number that this credit is to be placed against

Name

This is auto populated from Customer Maintenance record

Address

This is auto populated from Customer Maintenance record

Transaction Type

Defaults from the batch header but can be changed using the dropdown list

Analysis

Contract the credits will be placed against

Customer Ref

Customer Order number where available

Description

A description that explains what the credit is for. This will appear on the customers’ paperwork

Invoice Date

Defaults to the transaction date entered in the batch header. This can be changed

Due Date

This will default to be the same as the Invoice Date field

Discount Date

This will default to be the same as the Invoice Date field

Sales Analysis

Allow categorisation of invoices for reporting purposes, for example Salesman initials

Multiple Lines

To enter the credit line by line. When selected the Net Amount, Discount, GST and Gross Amount fields will grey out

Net Amount

Net Amount of the credit. This will be entered as a positive figure. The SCRN transaction type indicates to the system that it is a credit

Discount %

Any agreed discount percentage to be applied to the credit. This will need to be included if the original invoice had a discount applied

VAT

Select the VAT code from the dropdown list

Click

Consolidated Credit Note

A consolidated Credit Note will contain a net total only. The lines are not broken down individually.

Due to UK VAT rules, UK clients are advised to use Multiple Line Credit notes NOT consolidated.

If multiple lines is not ticked (on the screen above), when you click save a Revenue Distribution tab will be displayed.

Click

Enter the details of the revenue distribution in the relevant fields.
