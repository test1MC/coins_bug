Posting Batches

A batch needs to be posted to the ledger to enable the invoices and credit notes to be allocated to the relevant contracts and customer accounts. Batches can be posted individually or collectively.

The batch can remain in an Unposted Status until all the relevant transactions have been entered.

From the Sales Ledger module select the Input submenu and select Invoice and Credit Note Entry.

Locate the batch to be posted from the batch listing and select the Tick Box adjacent to the batch details. Select Post Batches from the Choose Action dropdown list. Select the Apply Action button.

 

Note - If a report is required for this batch posting select Post Batches with Report.

You return to the Invoice and Credit Note Entry Batch screen and the status will be updated to Posting, while it is being processed. Once complete the status will change to Posted.
