Input of Amended Details

When you view the details of a return with a status of “AMENDING” (but not with any other status), you can amend the details for individual subcontractors.

  1. To update subcontractor details from the CIS Monthly Return Submissions browse, click the link on the AO Reference / UTR / Period column for the return you want to amend.
  2. On the Subcontractors tab, open the subcontractor record.

    CIS Monthly Return - Subcontractor Details

  3. Amend the values in the Total Payments, Cost of Materials or Total Deducted fields as necessary.
  4. Enter notes to explain the reason for the amendment (this is for internal use only and will NOT be submitted to HMRC);
  5. Save the amended subcontractor record.

When you save the record for a subcontractor, the summary figures for the whole return will be updated to reflect the changes.

You can also add a new subcontractor record.
