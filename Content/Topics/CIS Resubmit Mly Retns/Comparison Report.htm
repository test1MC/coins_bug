Comparison Report

The Comparison Report compares an amended CIS return to the previous version of the return for the same period or to the original return for that period. This enables you to check the changes made and help ensure that they have been made correctly prior to submission (or following submission).

To run the Comparison Report, on the CIS Monthly Return Submissions workbench select a return and choose the Comparison Report action.

Comparison Report Selection Screen

Fields

Field
Description

Selected Return

The amended CIS return that you have selected.

Compare against Return

The CIS return to compare against the selected return.

Show Added / Changed Entries Only

Un this box to see all entries on the selected return even if they are identical to the entry on the return you are comparing it with.
