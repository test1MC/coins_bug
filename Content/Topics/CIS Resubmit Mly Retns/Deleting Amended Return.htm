Deleting Amended Return

If you realise that you have started to amend the wrong return or have made incorrect amendments to the return, and have not yet submitted it, you can delete the amended return using the Delete Amended Return action on the CIS Monthly Return Submissions workbench.

If the amendment has a status of “COMMITTED” (that is, it is ready for submitting), you need to un the Information Correct box before you can delete the amendment.
