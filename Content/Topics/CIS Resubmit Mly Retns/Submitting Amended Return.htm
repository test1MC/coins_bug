Submitting an Amended Return

Once the necessary amendments to the detail records have been made, open the header to complete the Information Correct, Employment Status and Subcontractor Verification boxes and update the password if necessary.

Amended CIS Header Details

When you save the record, its status will be changed from “AMENDING” to “COMMITTED”. At the same time, the status of the original return will be changed from “COMPLETE” to “SUPERSEDED”.

CIS Monthly Return Submissions - Committed Status

You can then submit the amended return in the same way as an original return, by selecting the Submit action for the return in the CIS Monthly Return Submissions workbench.
