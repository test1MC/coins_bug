CIS Monthly Return Submissions

A new action on the CIS Monthly Return Submissions workbench allows you to update a copy of a completed return.

Running the Amend Return action creates a copy of the selected return, with the following details:

  • A new submission ID.
  • A status of “AMENDING” (this is a new status).
  • Last Action equal to the date and time the copy of the return was made.
  • By is set to the user who ran the Amend Return action.
  • The Information Correct, Employment Status and Subcontractor Verification boxes are cleared. The user must re-complete these before submitting the amended return.
  • All other header fields are copied from the original return.
  • All the subcontractor details (including negative returns) are copied from the original return.
  • The attempts, errors and IRMark are not copied.

CIS Monthly Return Submissions - Amending

Only one return with a status of “AMENDING” is allowed at any one time for each combination of AO Reference, UTR and Period. Further amended returns can only be created if the current return is submitted before the next one is created.
