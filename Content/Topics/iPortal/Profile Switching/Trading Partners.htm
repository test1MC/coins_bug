Trading Partners

 

The Action menu on each applicable subcontractor record contains an option to Switch Profile.

 

 

Selecting this option invokes a confirmation dialog.

 

 

The dialog indicates which Profiles are known in the iPortal database; which is the current Profile linked to OA; and which Profiles are available to be linked.

Selecting the Confirm Change option will:

  1. Send a message to the OA Company Information Workbench to use the newly selected profile.
  2. Associate with the new profile all questionnaires assigned and completed on behalf of previous profile. Any incomplete questionnaires will be transferred to the newly selected profile and the subcontractor user can still complete them as normal but will need to be representing the correct profile at the time of completion.
  3. Associate all completed and future assessments with the newly selected profile.
  4. The previous profile will be put in the contractor’s iPortal Vendor Pool.
  5. If the newly selected vendor is awaiting secondary approval, the profile switch will allow the secondary approver to approve this profile.

No notifications will be sent to the vendor telling them about this change.

Selecting the Close option will close the dialog and no switch will be made.

In the example shown above, the vendor, 1st Line Defence Ltd, has profiles linked with data-providers Builders Profile and ConstructionLine; the ConstructionLine profile is currently associated with the vendor’s record in the linked CIW; the Builders Profile record is targeted to be newly linked to CIW. Pressing the Confirm Change button will remove the CIW link to the ConstructionLine record and substitute a link to the Buildres Profile record.
