Screen and Processing Changes

Details Tab

The sections in the Details tab on the Workbench (see Figure 1) are now positioned one below another in the order listed below. Please note that the Captions section is only visible if the MK/USECAPTIONS parameter value is set to Y.

  • Main – See Section 2.1.1 for details
  • Custom – See Section 2.1.2 for details
  • Location / Dates / Forecast – See Section 2.1.3 for details
  • Captions – See Section 2.1.4 for details
  • Activities – No changes made
  • Notes – No changes made

Details Tab

Main Section

The Main section is divided into four separate columns, as shown in Figure 2.

  • A Work Types field in the third column appears under the Main Work Type field if parameters MK/MAINWT and MK/WORKTYPES are set to Y. You can select from a number of different options that appear for various work types (such as Sheet Metal or HVAC).
  • Outcome fields are only visible on this screen when the Stage field equals Closed-Lost, Close-Won, or Inactive. Outcome fields (as shown in Figure 3) include: Outcome Reason, Outcome Comment, Our Outcome Category, Contract Awarded To, and Additional Outcome Comments.

Main Section

Outcome Fields

Custom Section

The Custom section (Figure 3) contains fields that are found in Setup > Setup > Custom Fields maintenance.

Custom Section

Location / Dates / Forecast Section

The Location / Dates / Forecast section contains the three columns listed below, as shown in Figure 4:

  • Location – This column always appears in this section.
  • Date – This column always appears in this section.
  • Forecast – This column only appears if the MK/USEBPFXT parameter is set to Y.

Location / Dates / Forecast Section

Captions Section

The Captions section (see Figure 5) contains Caption fields from the Update screen in > Setup > Information System > Caption Group Definition. This section is only visible if the MK/USECAPTIONS parameter value is set to Y.

Captions Section
