Screen and Processing Changes

Email Templates

Main Screen

A new Triggers option is available in the Select Template Type field in > Setup > General Setup > Triggers & Templates > Email Templates, as shown in Figure 1. Use this option to create new email templates for Triggers.

Email Templates Main Screen

Add Screen

When you select the Triggers option on the Email Templates screen (Figure 1 above) and then click the Add button, you are taken to an Add screen, as shown in Figure 2. You can use this screen to create a new email template for Triggers.

Email Templates Add Screen

Field Definitions

Template

Enter the name you want to use for the Trigger template.

Description

Enter a brief description of the Trigger template.

Comments

If needed, add any comments that relate to the Trigger template.

Subject

Enter the subject to be used for this Trigger template. You can enter merge fields in this field by surrounding the field with brackets {}, as demonstrated in the sample below.

Message

Enter the email message for this Trigger template. You can enter merge fields in this field by surrounding the field with brackets {}, as demonstrated in the sample below.

Triggers

Menu Option

A Triggers function is now available as a menu option in > Setup > General Setup > Triggers & Templates, as shown in Figure 2. This function allows you to add, update, or delete triggers.

Triggers Menu Option

Main Screen

When you select the Triggers menu option, a main screen appears, as shown in Figure 4. This screen lets you add, update, or delete triggers, as needed.

Triggers Main Screen

Column Definitions

Name

The name of the trigger.

Table

The table where the trigger is stored.

Email Template

The name of the email template being used for the trigger.

Detail Screen

When you click the Add button on the Main Trigger screen (Figure 4 above), a Detail screen appears, as shown in Figure 5. You can fill in or update the trigger's information on this screen.

Triggers Detail Screen

Field Definitions

Trigger Name

The name of the trigger.

Select Email Template

This lookup field allows you to select an email template previously added in Email Templates (see Section 1.1).

Table

Select a table available on the list, from which the trigger can be started. Examples of tables you can choose from are , Contacts, Events, Journals, etc.

New/Deleted/Update Record

Select one of the following boxes:

  • New Record - creates a new record
  • Deleted Record - deletes an entire record
  • Update Record - updates an existing record

Field Triggers/Calc/Value

Select the field triggers, which are a set of fields (listed below) that determine the criteria to use for getting the email notification.

  • Select Field - select the field to be used as a trigger
  • Calc - select the calculation to be used, such as Equal To
  • Value - select the field value

For example, in Figure 6 below, the email notification will be sent only if the kco company (kco) is Equal To the Value of 80, and the Revenue Reserve % (mkm_reserveperc) is Less Than the Value of 50.

Field Trigger Example

After you fill in the Calc/Value fields on a row and press Enter, a new row is automatically added for the next selected field. The Maximum amount of allowed field numbers is 20.

Send To

The fields in this area (shown below) allow you to select from a list of options to determine which role types and personnel should be notified.

  • Roles - Select the types of roles to which the email should be sent. This field only appears when is selected in the Table field.
  • Personnel - Select from a list of personnel to whom the email should be sent.

An example of how this area appears is shown below.

If = Then

Select the conditional fields, which are a set of fields (listed below) used to determine the criteria that must be met in order to receive the email notification.

  • If - select the field to be used as the first condition
  • = - enter the second condition, which determines what the first condition (If) should be equal to
  • Then - select the name of the user to be used if both conditions above are met

For example, in Figure 7 below, if the employer (pij_employer) is equal to (=) to Another User, Then the Employer with user ID katrey (katrey) receives an email notification. Also, if the Journal text (mkj_text) is equal to (=) to This is a test only, Then user sabstr also receives an email notification.

If - Then Fields Example

NOTE:

After you fill in the If=Then fields on a row and press Enter, a new row is automatically added for the next selected field. The Maximum amount of allowed field numbers is 20.

Additional Emails

This is a list of email addresses that will be included in the email notification. It must contain a semicolon between each email address.

CC

The fields in this area (shown below) allow you to select from a list of options to determine which role types and personnel should be sent a copy (CC) of the email.

  • Roles - Select the types of roles to which a copy of the email should be sent. This field only appears when is selected in the Table field.
  • Personnel - Select from a list of personnel to whom a copy of the email should be sent.
