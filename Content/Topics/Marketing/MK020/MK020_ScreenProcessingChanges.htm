Link/Copy

You can select one of the following options for mapping the field:

  • Y = Copy and Link
  • N = Copy Only

If parameter MK/JCLINK is set to N, this field automatically defaults to Copy Only and cannot be changed.

Dormant

If ed, this field will not be mapped until the Dormant field is uned.

Note: In Update mode, this is the only field that is enabled and can be changed.

Action Items

The following Action items are found at the bottom of the Mapping File Maintenance browse: Repopulate data and Delete data.

Repopulate data

After records have been added to the screen, you can select one or more of those records and run the Repopulate data action. The process reads through all current records, finds the related , and updates the selected fields with the associated data from the field.

Delete data

Use this action to delete selected field(s) from the Mapping File Maintenance screen. When you first run this action, the following message appears: "Do you want to clear out all side data related to this record?" If Yes, the process reads through all records, finds the related , blanks out the noted field, and deletes the record.

Creation

When a new is created from the , and parameter MK/JCLINK is set to Yes, data are now copied from the fields to the fields of the current company.
