System and User Requirements

In order to use COINS MS Outlook integration functionality, there are certain system and user requirements that must be met. This section discusses these requirements.

Add-on Installation Requirements

You must install a preconfigured COINS MS Outlook Add-On onto a machine with MS Outlook. While installing, you should only be prompted for an installation path; all other configuration and COINS connection settings are predefined.

Please consult your COINS representative for more information on installing this Add-On.

3.4.6 Email Integration

When link an email to COINS there should be a possibility to associate additional contacts or personnel but only to One Project or One event

3.4.6.1 New Emails

 

• COINS Icon within email. User clicks on icon to get a popup box.

o Company field: Lookup table, linked to [ci_company] table in COINS, include columns for Company Name, Office Name, Address and Contact. Within lookup, user can filter and search by Company Name, City, State and Country, Work Group and Sub Groups, Sales Rep and My Companies.

o Contact field: Lookup table includes columns for Last Name, First Name, Middle Name. If Company is selected, only Contacts of that Company are available. If no Company is selected, all Contacts are available. When a Contact is selected, the Company of that Contact will autofill the Company field.

o Project field: Lookup table, linked to [pi_project] table in COINS, include columns for Project Name, Client Name, Description, Project Number, Status, Sub Status. Filter/search by Project Name, Address (city/state/country), Project Number, Sales Rep, Work Group, Sub Groups and My Projects.

o Personnel: Lookup table linked to pp_organisation table in COINS.

o Event: Lookup linked to event table in COINS.

o Mailing: Lookup linked to mailing table in COINS.

o Attachment: Email attachments will load as document attachments to the email integrated in COINS. The user can uncheck the checkbox if the attachments are not needed in COINS.

o Autofill: User can use lookup or begin typing in each field for an auto-fill dropdown. Dropdown results by any part of the record field name. No wildcards needed for search.

o Smart Autofill: Based on the email address in the “To” field of the email, the Email Integration will Autofill with the corresponding Company, Contact and/or Personnel that matches that email address with COINS.

 If more than one email address is listed, the popup will search and display the first email listed. When the user selects “Add” within the popup, the fields will Autofill with the next matching email address to COINS.

3.4.6.2 Existing Emails in Outlook

 

• Select email or multiple emails in Outlook from any Outlook folder and synchronize them to selected Companies, Contacts, Project, Event, Mailing and Personnel in COINS.

• Ability to integrate with additional Companies, Contacts, Projects and Personnel with the Add button.

• Ability to save the attachments to the Email Activity in COINS (see checkbox in above example). Also have the ability to not save attachments to COINS by unchecking the Attachments checkbox.

• Ability to update the COINS CRM Events records.

o COINS CRM has an Events workbench. An Event contains lists of contacts. Currently COINS has the capability of sending email invitations to these event lists. With Outlook Integration, need the ability for return emails from the Invitation Lists to automatically update the Accepted or Declined fields in the COINS Event (will also possible have a “Tentative” field – to be determined). These emails should also automatically link to the event, the contact and the contact’s company.

 

3.4.7 Definition of Offline

The Offline for MS Outlook Add-on should be any state when COINS server is not available, and Add-on to synchronization transaction cannot be executed, from this point all changes to COINS integrated entities in Outlook should be stored locally and will be synchronized by time stamp in any possible occasion to synch with COINS Server (such connection availability will be checked by scheduled synch operation or on demand by pressing the Synch Now button).
