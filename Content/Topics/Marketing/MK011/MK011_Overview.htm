Overview

The COINS system needed an integration solution with Microsoft© Outlook (MS) as an activities tracker and emailing client. New Add-On based functionality is now available to add users with the ability to conveniently interact with data in MS Outlook and the COINS system (see Section 1.1 below). This functionality allows COINS OA and Project Management users to focus on their day to day work more efficiently via broader access and a more convenient way to access information in both COINS and MS Outlook.

Add-On Description

The new COINS MS Outlook Add-On provides integrated functional capabilities of the MS Outlook client and some COINS modules, especially and Project Management. This Add-On is responsible for managing the synchronization process (in both online and offline modes), as well as sending and receiving the data of two systems. It must be installed on a user’s workstation where an MS Outlook client is installed. Once installed, it adds new commands and menus to the MS Outlook user interface.

When integrated, you can store references to e-mail messages sent and received from MS Outlook, as well as synchrone the details of contact persons and activities between the two systems.
