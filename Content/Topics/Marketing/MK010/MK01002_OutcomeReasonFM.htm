Outcome Reason Field Maintenance

Outcome Reason Field maintenance is found in > Setup > Setup, as shown in Outcome Reason Field Maintenance.

Outcome Reason Field Maintenance

Column Definitions

Stage

The Stage to assign an Outcome Reason. Only Stages that have Closed-Lost, Inactive, or Closed-Won qualifiers are available.

Code

The code for the Outcome Reason.

Description

A description of the Outcome Reason.

Dormant

Whether the reason is dormant.
