Outcome Report

The new Outcome Report menu option has been added to can be found in > Reports. It is used to report on Outcome Reasons for Closed-Won, Closed-Lost, and Inactive Stages.

Report Selection Screen

When you select Outcome Report from the menu, you will see a report selection screen, as shown in Report Selection Screen.

Report Selection Screen

Sample Reports

You can generate the Outcome Report with or without Competitors:

  • A sample Outcome Report with Competitors is shown in Report Sample (With Competitors).
  • A sample Outcome Report without Competitors is shown in Report Sample (Without Competitors).
