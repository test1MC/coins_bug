Browse Screen

After you use the selection screen to filter information you wish to display and then click Next, you are taken to the main Summary browse screen, as shown in Figure 4. This browse screen is divided into five different tabs:

  • Status Conversion by Month Tab (see Section 1.3.1)
  • Stage Conversion by Month Tab (see Section 1.3.2)
  • Stage Qualifiers by Month Tab (see Section 1.3.3)
  • Status Conversion by Quarter Tab (see Section 1.3.4)
  • Summary Filter Tab (see Section 1.3.5)

For each tab, the header fields remain the same. Also, the # links in the columns of each tab allow you to drill down to the screen to see more detailed information (see Section 1.5 for more information about the screen).
