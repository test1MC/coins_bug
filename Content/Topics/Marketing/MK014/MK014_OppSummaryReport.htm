MK  Summary Report

The new Summary Report function has been added to the module to allow for data mining and reporting.

Menu Option

A new Summary Report menu option has been added to the menu under the Reports submenu, as shown in Figure 1.

Selection Screens

When you select the Summary Report option from the Reports menu shown above, you are taken to a selection screen that allows you to filter on what you want the Report to display. As with the Summary selection screen, there are two slightly different versions of this screen, depending on whether you choose or Date Range at the top:

  • (see Figure 2) - Shows as the first field. The resulting information on the Report is therefore shown by .
  • Date Range (see Figure 3) - is replaced by the fields Date and Show where Date is in the Range. The resulting information on the Report is therefore shown according to the dates entered here.

Report Sample

After you use the selection screen to filter information you wish to report on and then click Next, the Summary Report is generated, as shown in Figure 4. This report is divided into five different areas that mirror the Summary screen, except that only the report may include the Conversions Tile graph:

  • Status Conversion by Month
  • Stage Conversion by Month
  • Stage Qualifiers by Month
  • Status Conversion by Quarter
  • Conversions Tile graph (if you ed Include Graphic)
