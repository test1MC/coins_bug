Screen and Processing Changes

Activity Types Maintenance

A new column named Default Action Type is found on the Activity Types screen in MarketingCRM > Setup > General Setup > Activity Types, as shown in Figure 1. It allows an Action Type to be assigned as the default for each Activity. This is a mandatory field.

Activity Types Screen

Activities Workbench

Completed Column

In MarketingCRM > Activities, the Done column has been renamed as Completed, as shown in Figure 2.

Activities Workbench Column

Default Action Type Value

When an Activity Type is added at the top of the Activity data entry screen, a value defaults into the Type field farther down the screen, as shown in Figure 3. The default value in this field can be changed if desired. Note: Some Activity Types do not display the Action Type.

Activities Workbench Default Value

Named Filters

The Named Filters at the bottom of the Activities Workbench screen have been changed as are described below.

  • Named Filter 1 options are now:
  • My Activities
  • All Activities
  • New Named Filter 2 options are:
  • All Categories
  • Appointments
  • Tasks
  • Other (this is used for blank categories)
  • Named Filter 3 options for Due Dates are:
  • All
  • Open
  • Due Today
  • Due in Next 7 Days
  • Due in Next 14 Days
  • Due in Next 30 Days
  • Overdue
  • Completed
