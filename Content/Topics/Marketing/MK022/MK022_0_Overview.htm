GDPR in : Overview

New General Data Protection Regulations (GDPR) came into effect in May 2018, for which companies involved in any direct marketing must be able to provide evidence that individuals have positively opted in to receive such marketing. The existing data protection fields in COINS can now track such evidence in the Contact records, together with a source of consent, comments, and the ability to attach documents for each.

The functionality now available in COINS OA for GDPR allows you to:

  • Record consent and how it can be used
  • Record how consent was captured and by whom
  • Record if consent is withdrawn and when
  • Store and use personal data only when it is lawfully required
  • Delete/remove personal data where necessary
  • Prevent the override of data protection rules when running mailshots, etc.
  • Extract details

This document describes any changes that were made to the module to accommodate GDPR functionality.
