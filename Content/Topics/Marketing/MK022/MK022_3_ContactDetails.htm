GDPR in : Contact Details

The Contact Details tab in MK Contacts shows new sections depending on how the MK parameter CONSUSE is set. These sections include: Main, Social/Personal, Consent, Further Details, Notes, and Activities. Each section is explained in better detail in the paragraphs that follow.

  • If CONSUSE is set to N, a Further Details tab appears, and the Consent tab does not appear, as shown in Figure 1.

CONSUSE = N

  • If CONSUSE is set to Y, the Consent tab appears, and the Further Details tab does not appear, as shown in Figure 2.

CONSUSE = Y

Main Section

The Main section no longer contains Notes and Activities, which are now found as sections. A sample screen is shown in Figure 2.

Main Section

The new Upload Contact Photo link shown below is found on this screen. It allows you to upload a Contact photo onto the Main Details section.

Social/Personal Section

The Social/Personal section appears as shown in Figure 3. It allows you to maintain a Contact's social information (such as Twitter, YouTube, etc.) and personal information (Date of Birth, Personal Assistant, etc.).

The bottom of this section also includes a Relationships browse where you can record friend/family relationships.

Social/Personal Section

Consent Section

The Consent section (Figure 5) collects and displays the data from Data Consent Categories. You can enter and update this information as needed. NOTE: This section only appears if parameter MK/CONSUSE is set to Y.

Consent Section

Field Definitions (Selected Fields)

Data Consent Notes

This new field lets you enter information about the consent data, as needed.

NOTE: Parameter MK/CONSNOTEW determines whether you will receive an error or a warning when trying to save the record if you have changed one or more consent values but have not changed this field.

Data Consent Document

When you click the Choose File button in this new field, it opens a window where you can choose a document to be uploaded that is associated with this Contact (it can support proof of the consent given). Once uploaded, the file name is shown next to the button. Note: Only one document can be uploaded per update.

After the record is saved, the uploaded document is stored against the Contact's record in Document Manager.

Further Details Section

The Further Details section (Figure 6) lets you set up the Mailing Preferences information as needed. NOTE: This section only appears if parameter MK/CONSUSE is set to N.

Further Details Section

Notes Section

The Notes section allows you to add, update, and review notes for a Contact.

Activities Section

The Activities section allows you to add, update, and review activities for a Contact.
