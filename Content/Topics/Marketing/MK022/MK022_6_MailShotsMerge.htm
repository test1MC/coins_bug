GDPR in : Mail Shots and Mail Merge

When a Contact’s consent category is No or Unknown, that Contact is excluded from any Mailing Lists, Mail Shots, and Mail Merges.

Email Triggers

In > Setup > General Setup > Triggers and Templates > Triggers, you can send emails to users based on the activity of an a . In order to comply with GDPR rules, the Consent flag for an email is checked before adding the email address to the outgoing message.

Mailing Lists

In > Mailing Lists, you can define the list of Contacts to whom an email or text can be sent.

First, select the Contact(s) you wish to send the email/text to, then select the Create Mail Shot action, as shown in Figure 3.

Create Mail Shot Action

When you click Next, you can use the screen to select the Select Destination Media (as set up in parameter HS/MSMEDIA). You can also select the Letter/Template to be used for that media (Figure 4 shows the Email screen and Figure 5 shows the Text Message screen).

Destination Media - Email

Destination Media - Text Message

After you fill in the screen, you can select Next to begin the process of checking the Consent flags of each person for the type of message being sent (text, email, etc.) before adding them to the final file.
