GDPR in : Contacts Report

A new Contacts Report is available as a menu option in > Reports. It is used to display data about Contacts, including Consent Data. When selected, it displays a selection screen with a number of tabs, all of which are discussed below.

Main Tab

The Main tab of the Contacts Report selection screen is shown in Figure 1. This tab allows you to select which Available fields you would like to move to the Selected area in order to include them on the report.

Main Tab

Fields (Selected Fields)

Other Companies

If selected, an inline report displays using Company/Office, Current, and Main columns.

Mailing Lists

If selected, the Mailing Lists to which the Contact belongs are displayed as a comma-separated list.

Company Tab

The Company tab of the Contacts Report selection screen is shown in Figure 2. This tab allows you to select any information about the company you wish to include on the report, such as the Company Offices, Company Types, etc.

Company Tab

Contact Tab

The Contact tab of the Contacts Report selection screen is shown in Figure 3. This tab allows you to select any Contact information you wish to include on the report.

Contact Tab

Field Definitions

Contact

Select one or more Contacts to report on.

Contact

Select one or more to report on.

Contact Origin

Select one or more options to report on regarding where the Contact's information came from.

Grade

This field appears if MK/CONTGRAD is set to Y. Select one or more Grade types of a Contact to report on.

Job Group

Select the Group to report on.

Job Role

Select the Role to report on.

Create Date

Enter the start and end dates to report on.

Show Consent Categories

When you select this field, the Consent to Receive categories are displayed for selection, as shown in Figure 3 above. For each category, select one of the following options to indicate what you wish to report on:

All - All consent types for the category are included on the report.

Yes - Only "yes" consent types for the category are included on the report.

No - Only "no" consent types for the category are included on the report.

Unknown - Only "unknown" consent types for the category are included on the report.

Exclude - All consent types for the category are excluded on the report.
