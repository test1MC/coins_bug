Two-Factor Authentication (24849)

There is a new optional feature to require users to authenticate with a time-limited passcode (which is sent to the user via SMS or email), in addition to the normal user ID and password. This new feature greatly enhances the authentication of users in to and is particularly relevant if the system is available on the public Internet. The password hashing algorithms have also been strengthened. See OA-CE-SY025 for details.
