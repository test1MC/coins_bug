Expense Types

Expense types are records that define how different types of expense claim item are treated: for example, how they are costed and what information is required.

To add a new expense type

  1. Open Expense Types.
  2. Right-click the list and click Add New.
  3. On the Details tab, fill in the information as follows:
  4. Company The  company that this expense type is used in (expense types are specific to a single company).
    Apply Validation To Type

    This determines the fields that are required when entering the amount of the claim and how they are validated.

    • Limit. Users enter an amount, which is validated against a limit, such as a maximum value.
    • Rate. Users enter a number of units; the rate per unit is set on the policy. This is used for expenses such as mileage which is paid at a fixed rate.
    • Per Unit. Users enter a number of units and a rate per unit; the policy has a limit per unit. This is used for expenses such as hotel accommodation (where the units are the number of nights and the rate is the cost per night).
    Parent Type

    You can group related expense types together as children of a parent expense type. For example, a "public transport" parent expense type might have children "train", "bus" and "plane". When the user selects the expense type, children of the same parent are shown together, below the parent.

    To make the current expense type a parent, leave this field blank.

    To make the current expense type a child, enter the name of the parent type.
    Type Name The name of the expense type.
    Label Used for Amount The label that is displayed for the amount field.
    Label Used for Rate/Unit If the expense type is a Rate or Per Unit type, this is the label that is displayed for the rate per unit field.
    Show Odometer controls and map link If the expense type is a Rate type, you can allow users to calculate the mileage by entering the start and end odometer readings, or by plotting the distance on a map. To make these features available, this box.
    General Ledger Code The General Ledger code that this expense will be costed to (when it is costed to the General Ledger).
    Contract The contract cost head that this expense will be costed to (when it is costed to a contract).

    Tax Code

    		 The tax code that applies to this expense type.
  5. Click Save to save your changes.
