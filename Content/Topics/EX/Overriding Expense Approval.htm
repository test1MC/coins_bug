Overriding Expense Approval

The Expenses Administrator can override the approval of an expense report if it has not yet been paid. For example:

  • If no-one has reviewed the expense report (perhaps because the approving manager is unavailable and has not set up delegation), the administrator can approve it.
  • If a user has submitted an expense report in error, the administrator can cancel it.

To override an expense report:

    Open Approval Override.

    This shows a list of all expense reports that have been submitted but not yet paid.

    You can view the details of a report before overriding it:

    Right-click the report line and click Approval Override.

      Right-click the report line and click View Expense Report.

      The Report Details screen opens.

      Click Close when you have finished viewing the report.

    Type an explanation of why you are approving or cancelling the report.

    Click Approve Report or Cancel Report.
