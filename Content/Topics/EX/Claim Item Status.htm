Claim Item Status

 

The status of a claim item changes as it is processed. You can see the status on the Claim Items List.

Possible statuses are:

Draft

You are still working on this claim item. You can add it to an expense report.

Awaiting Prior Approval

You have requested prior approval of this claim item (see Requesting Prior Approval, ). You cannot modify the claim or add it to an expense report.

Prior Approval Rejected

You cannot add this to an expense report (or request prior approval again) unless you make a change to the claim item (which sets the status back to Draft).

Prior Approval Given

You can add this claim item to an expense report. Note that if you modify the claim item, its status will change to Draft.
