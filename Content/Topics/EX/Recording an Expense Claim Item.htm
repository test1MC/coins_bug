Recording an Expense Claim Item

    Open Create New Expense Claim Item.

    You can save the Expense Claim Item at any point. For example, you could simply create and save an item without entering any details, to remind yourself that you have incurred an expense.

    Enter the details of the expense. Only claim items that include at least the minimum information: date, expense type and amount can be added to an Expense Report.

Field Explanation

Transaction Date

The date you incurred the expense. You can use the calendar lookup, or use standard abbreviations, such as $-1w for “one week before today’s date”.

Your company’s policies may restrict how far in the past you can enter claims.

Expense Type

The type of expense. These are set up according to your company’s policies, and determine things like what information you need to provide, and how much you can claim.

Cost to Contract

 

(This is only available if it has been enabled for you.)

Tick this if the expense is to be costed to a contract. This makes the Apply Costing Code field available.

Merchant

The merchant with whom the expense was incurred (for example, the shop where you bought an item, or the hotel you stayed at).

Description

A description of the expense.

Apply Costing Code

If you Cost to Contract, enter the code for the contract to which the expense is to be costed.

Apply

This may change to “Cost Details”.

Cost Details

This section allows you to enter details of the amount you are claiming. The information required depends on the Expense Type.

Does this claim contain any personal expense?

In some cases, for example if using a fuel card, you may be required to account for all expenses, even if you are not claiming for them. If you are recording an expense but not claiming, the “Personal Expense” box, then enter the amount of the personal expense part of the claim.

Billable to Client

If the expense is to be charged to a client, the “Billable to Client” box, and select the contract.

    Depending on your company’s policies and the Expense Type, you may be required to provide additional information. If the “Further Required Information” tab is shown, click it to enter the additional information.

    To save changes to the claim item, click Save & Close (or Save & Repeat if you want to enter more claim items). You can make further changes before adding the claim item to an expense report; see Modifying an Existing Claim Item.
