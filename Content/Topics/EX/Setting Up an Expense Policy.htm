Setting Up an Expense Policy

To set up an expense policy:

  1. Open Expense Policies.
  2. Right-click a line and click Add New.
  3. Fill in the fields as follows:
  4. Company The company to which this policy applies.
    Name The name of the policy.
    Applies To

    Which users the policy applies to.

    Global - The policy applies to all users in the company.

     

    Locations / Departments / Grades - The policy applies only to users who belong to specified locations, departments or grades. If you select this option, additional tabs allow you to select which locations, departments and/or grades the policy applies to.

    Specific Users - The policy applies only to specified users. If you select this option, an additional tab allows you to select which users the policy applies to.
