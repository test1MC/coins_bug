OA Expenses - Inputting Claim Items

When you enter the claim record and save it, this takes you to a screen where you can add the claim items.

Claim Item Screen

Fields

Expense

The date on which the expense was incurred.

Item

The code for the type of expense.

Comment

A description of the expense.

Gross

The gross value of the claim.

The code for this expense. The default code is specified on the expense item type, but this can be overridden; for example, if no receipt is provided, the code would need to be changed to a non-recoverable code.

Amount

The value, which calculates using the percentage on the code.

Recoverable

The amount of that is recoverable, which calculates based on the settings on the code.

Cost Code

The cost code for the expense (if the expense is to be costed to a ).

Resulting Analysis

The initial value to create the distribution, which can be amended if required.

If a has been entered, this will be the , plus the from the item defaults. If no has been entered, this will be the GL account derived from the account mask on the item and the defaults on the employee record.

Cost

The amount to be paid for the claim item.

If each claim item has a single costing and analysis, you can enter all the details for the claim item on the Items tab. The and Cost Distribution tabs allow you to split the across more than one code, or to split the cost across more than one analysis.

A "paperclip" button is available to allow you to attach documents (for example, images of expense receipts).
