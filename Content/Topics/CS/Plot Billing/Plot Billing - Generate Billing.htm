Plot Billing - Generate Plot Billing

Generate Plot Billing is a new background report process that generates the Applications or Certificates for a based on the group valuation. It can be run from the menu, for multiple , or as an Option from Valuation Matrix (Tick Group) for a single .

This processes each open series that is set up as a ‘Plot Billing Series’ and flagged as a ‘Tick Group Valuation Generation’ series. For each billing stage and plot where there is an order amount and the billing stage is linked to a group, the valuation percentage will be calculated for the tick groups and plot. If the billing stage and plot are fully valued, then an application or certificate will be raised for the order amount.

The application or certificate date is set to today’s date, and the transaction type is determined by the VP parameters BILLCSTA and BILLCSTC. If the VP parameter BILLPOST is set to Y, the batch is posted automatically.
