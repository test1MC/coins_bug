Applications Using Plot Billing

If you enter an application (or combined application and certificate) for a series that is marked as a ‘plot billing series’, additional tabs are available to allow you to enter the matrix and variation amounts for the application.

Application Matrix

The Application Matrix tab shows a matrix of plots and billing stages. If the billing stage input method is ‘Amount’, you can enter the amount applied for from the client (either the cumulative amount or the amount for this application).

When you update a matrix amount, the application line associated with the billing stage is updated, and any dependent lines in the application definition are recalculated.

The cell selector in the header row allows you to display different values:

  • Cumulative (updateable) - the cumulative amounts applied for.
  • This Application (updateable) - the amounts applied for on this application.
  • Order value (read-only) - the order value agreed with the client.
  • Unapplied (read-only) - the remaining balance that is to be applied for.

A tooltip on each cell shows:

  • Order amount
  • Previously Applied
  • Previously Certified
  • Balance to Apply For
  • Balance to be Certified
  • The disputed amount (if there is a disputed amount).

Cell Actions

The following cell actions are available:

  • Detail Enquiry. This shows details of the order amount, application amount, certified amount, balance to apply, balance to certify, applied not certified amounts and disputed amounts, and a browse of the applications and certificates relating to the cell.
  • Update Amount. This prompts for an application amount and updates the selected cells with that amount. This is only available when the selected cell value is ‘Cumulative’ or ‘This Payment’.
  • Update with Order Amount. This sets the application amount to the order amount. This is only available when the selected cell value is ‘Cumulative’ or ‘This Payment’.
  • Disputed Amount. This allows you to record any details of disputed amounts, and notes relating to the disputed amount.
  • Undo Changes. This undoes any changes made in the current update.
  • Delete This Application. This allows you to delete the amount for this application. This is only available when the selected cell value is ‘Cumulative’ or ‘This Payment’.

Column Actions

The following column actions are available:

  • Cascade from Plot. This allows order values to be copied from one plot to another. This prompts for House Types and Plot Start/End/List. Cells where an application or certificate has been entered will not be updated. This action is only available when the selected cell value is ‘Order Value’.
  • Undo Changes. This undoes any changes made in the current update session.

Matrix Actions

The following matrix action will be provided:

  • Update with Order Amount. This sets the application amount to the order amounts for all stages/plots. This is only available when the selected cell value is ‘Cumulative’ or ‘This Payment’.
  • Undo Changes. This undoes any changes made in the current update.
  • Delete This Application. This allow you to delete the amounts for this application.

Variations

The Variations tab allows you to enter the variation application amount and/or variation certified amount. This is a browse of the variations set up for the series. When you update a variation, the corresponding matrix cell and application/certificate line are updated with the revised amounts.

A row action allows you to delete the ‘This Application/Certificate’ amounts if you make a mistake.
