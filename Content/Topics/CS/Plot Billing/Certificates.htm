Certificates Using Plot Billing

If you enter a certificate (or combined application and certificate) for a series that is marked as a ‘plot billing series’, additional tabs are available to allow you to enter the certified matrix and variation amounts.

Certificate Matrix

The Certificate Matrix tab shows a matrix of plots and billing stages. If the billing stage input method is ‘Amount’, you can enter the amount paid by the client (either the cumulative amount or the amount for this certificate). When you update a matrix amount, the certificate line associated with the billing stage is updated, and any dependent lines in the certificate definition are recalculated.

The cell selector allows you to display different values:

  • Cumulative (updateable) - the cumulative amounts certified.
  • This Certificate (updateable) - the amounts certified on this certificate.
  • Order value (read-only) - the order value agreed with the client.
  • Uncertified (read-only) - the remaining balance that is to be certified.

A tooltip on each cell shows:

  • Order amount
  • Previously Applied
  • Previously Certified
  • Balance to Apply For
  • Balance to be Certified
  • The disputed amount (if there is a disputed amount).

Cell Actions

The following cell actions are available:

  • Detail Enquiry. This shows details of the order amount, application amount, certified amount, balance to apply, balance to certify, applied not certified amounts and disputed amounts, and a browse of the applications and certificates relating to the cell.
  • Update Amount. This prompts for a certificate amount and updates the selected cells with that amount. This is only available when the selected cell value is ‘Cumulative’ or ‘This Payment’.
  • Update with Order Amount. This sets the certificate amount to the order amount. This is only available when the selected cell value is ‘Cumulative’ or ‘This Payment’.
  • Disputed Amount. This allows you to record any details of disputed amounts, and notes relating to the disputed amount.
  • Undo Changes. This undoes any changes made in the current update.
  • Delete This Application. This allows you to delete the amount for this certificate. This is only available when the selected cell value is ‘Cumulative’ or ‘This Payment’.

Column Actions

The following column action is available:

  • Undo Changes. This undoes any changes made in the current update session.

Matrix Actions

The following matrix action will be provided:

  • Update with Order Amount. This sets the certificate amount to the order amounts for all stages/plots. This is only available when the selected cell value is ‘Cumulative’ or ‘This Payment’.
  • Update with Applied Amount. This updates the certified amounts with the latest applied-for amounts.
  • Copy Specific Application.This prompts for an application, and copies the applied-for amounts from that application to this certificate.
  • Undo Changes. This undoes any changes made in the current update.
  • Delete This Application. This allow you to delete the amounts for this application.

Variations

The ‘Variations’ tab allows you to enter the variation application amount and/or variation certified amount. This is a browse of the variations set up for the series. When you update a variation, the corresponding matrix cell and application or certificate line are updated with the revised amounts.

A row action allow you to delete the ‘This Application/Certificate’ amounts if you make a mistake.
