VAP Billing - Overview

Plot-based billing allows you to build up sales applications or certificates by plot and stage, based on the sheet valuation in VAP. This allows  users who are subcontractors to home builders to bill the home builder, based on a valuation or measure of the work done to date. This process is also required when doing work for a housing association.

series can now be set up to use a matrix for billing, and there is new functionality to generate applications or certificates based on the sheet valuation in VAP.
