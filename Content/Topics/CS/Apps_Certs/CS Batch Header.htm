Applications/Certificates

CS Batch Header

The system will allow the entry of Applications and certificates against contracts.

Select > Applications/Certificates > Applications/Certificates

Batch Header

As with other ledgers, transactions are entered via a batch, Click to add a new batch.

The batch number is allocated by the system

Enter any reference to identify the batch

The financial period will default to the “Current” period, but can be changed if required

The transaction date will default to today's date.

Select he transaction type to use for transactions in the batch. If there is more than one type available, uses this as the default transaction type for each new transaction, but you can choose a different transaction type for individual transactions. For example, if you were entering both certificates and applications in the same batch, you could choose 'certificate' for the transaction type in the batch header, and change it to 'application' for the application transactions only.

Enter any description to identify the batch

Click at the bottom of the page.
