Posting the Batch

When all Application/Certificate Transactions have been entered into the batch, from the Transactions Tab select a Posting option from the Action Drop-down and click apply

Alternatively, you can leave the batch unposted and post it at a later date. In this case, the batch can be selected from the Batch Listing Screen and the Posting option selected from the Action Dropdown at the bottom of that screen.

In either case, if a report option is selected, this will be generated and will appear on the Report Status Workbench.
