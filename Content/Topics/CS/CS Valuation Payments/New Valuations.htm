New Valuations

To view the VPS

Navigate: /Input/Valuation Payment Stages

Navigate to > > Valuation Payment Stages

 

  • You will need to select which contract you wish to see the valuation payment stages for.
  • To undo the entries, click the undo icon.
  • To save the entries and continue, click the next icon.

 

  • The valuations screen will appear.

  • Click to amend the initial due date and add notesto save your changes
  • To view the details, click the detail icon.  
  • Click on the underlined number of the valuation you wish to update.
  • Under the main tab, you will see the latest figures to date.

The stages tab will show you the values entered at each stage so far.

For each valuation you will need to enter the applied and expected values.  To add a new VPS stage click .

On the Main Tab enter the ‘actual’ valuation date, will default the values into all other fields but they can be amended if required.

Once you have made all your entries move to the Details Tab.

Coins will default in the Cumulative figures from the applied stage. At this point you enter the Gross Expected value.  will update the Discounts & Retentions.

If you wish to add any comments relating to this valuation use the Comments tab and use the save icon   to save your entries. Once the values have all been saved, click the next arrow to continue.

We can now see the added applied stage.
