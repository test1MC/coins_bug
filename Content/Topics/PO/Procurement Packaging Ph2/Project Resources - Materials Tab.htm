Links

Links on both the Quantity Ordered field and the new “UO” field open a new desktop frame containing the list of orders shown in the corresponding tooltip. This list contains the same fields as shown on the standard Material Orders workbench. The Context area at the top of the frame shows details of the resource line selected. Just as with the standard Material Orders browse, it is possible to drill down into the order and carry out the normal actions available from the order’s summary screen.

Similar links on the Quantity Requisition field and the new “UR” fields open a frame containing the list of requisitions shown in the corresponding tooltip.

Raise Requisition Action

A new Raise Requisition action is available on the list of resources:

Raise Requistion Action

Once one or more resources have been selected the user is taken to the requisition header screen. The Number is pre-populated and is now display-only.

On saving the requisition header, the user is taken to the Requisition Lines screen where the selected resources and quantities are pre-populated. All the standard requisition entry parameters are consulted, so, for example, if requisitions are allowed to be priced, the Price and Per columns are visible and available for input.

Issue and Recall Requisition

If a requisition is issued, or if an issued requisition is recalled, the Quantity Requisitioned column is updated to reflect the total quantity of the resource that has been requisitioned (and issued).

Merging Lines into Budget Line

If requisitions and orders are raised outside of the Resources > Materials tab within the Project Information Workbench, attempts to associate the requisition or order line with the corresponding budget line (where the Via column is “IMP”) on the resources list. It does this by looking for a line one which the (commodity) code, the first 30 characters in the description and the unit of quantity are the same. If it cannot locate this record, a new line is created with the Via column set to “ORD”.

A new action on the Materials tab allows you to associate requisitioned and/or ordered lines with the correct budget line. You may want to do this when has been unable to locate the corresponding budget line itself (typically because the first 30 characters of the description are not identical to the description on the budget line). Select one (and only one budget line) and select one of more requisitioned or ordered lines. The (commodity) code and unit of quantity of all selected lines must be identical.
