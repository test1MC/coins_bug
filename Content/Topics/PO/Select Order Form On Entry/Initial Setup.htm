Selecting an Order Form During Order Entry - Initial Setup

A new The following PO parameter governs whether the print form selector is available when entering an order.

Parameter

Description

FRMSELEC

The parameter is a list of order types (e.g. M – Material, P – , O – Overhead and etc) for which this selector should be available. If an order type is not in the list or if this parameter is not set the selection of the order print form will work the same way as it worked before. An asterisk (*) can be used for allowing this functionality for any Major Order Type):


M – Material Orders

P – Orders

W – Workshop Orders

A – Asset Orders

I – Orders

O – Overhead Orders

R – Orders

S – Subcontract Orders
