The field defaults to the value of the appropriate PO Print Form parameter. It is possible to amend the Order Form if the original order is not committed.

The field is only available for update when entering new orders, not for .

This functionality is available on the purchase order entry pages (from Orders Live / Inc. Archived) as well as when raising an order from requisitions / subcontract instructions and from packages:

Raising an Order from a Requisition Line

Head/Site Order

Depending on whether the order is a head office order or a site order, the value of the appropriate PO parameter will be used: PO/POFRM for material head office and PO/POFRM‑S for material site order.

Service Order Related Orders

The default values of the order forms for material/subcontract orders are taken from the appropriate PO parameters – POSEFRM or SCSEFRM.

Templates

The select form functionality is not available for order templates but the order form is available for amendment when copying a template to a regular order.

Copying Orders

When an order is copied, the value of Use Order Form is not copied. The default form for a copied order is the same as when an order is manually created, but can be amended.
