Selecting the Order Form During Order Entry - Overview

New functionality is available that allows you to You can set a specific print form against each new order when you enter it. Order forms for printing different types of purchase orders are created using Document Designer. The form’s name is set in the appropriate PO parameter for the order type, for example: for material orders it is set using the parameter POFRM; for subcontract orders using SCFRM; for overhead orders using OVFRM and so on. All new orders use the specified order form by default but you can now choose another form during order entry.
