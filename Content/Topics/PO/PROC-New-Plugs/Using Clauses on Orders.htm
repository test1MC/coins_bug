Using Clauses on Orders

  1. When you are entering the lines on an order, choose Clause from the Add Type drop-down at the bottom of the browse.

    displays a drop-down that shows the clause codes.

  1. Select the clause you want to add.
  2. Click  to add the clause.
  3. If the clause uses plugs, these are presented as input boxes within the Description field; enter the required text, date or number.

  1. Click to save the clause.

NOTE

You will not be allowed to commit an order if any mandatory plugs have not been filled in.

Clauses on Template Orders

Updateable clauses can be updated only in normal orders. In template orders all clauses are treated as non-updateable.  When copying a Template PO, plugs with default and auto-fill values are re-calculated with the new PO values for all clauses (even for updateable ones – which will only really be updateable on a normal order).
