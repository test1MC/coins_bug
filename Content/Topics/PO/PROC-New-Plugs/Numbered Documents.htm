2.4 Numbered Documents

Numbered document plugs allow you to produce structured subcontract orders with placeholders for standard numbered documents such as standard contract, health and safety documentation, and schedules.

Set up the numbered document sets using Numbered Documents (%WTI2181BTND).

 

To insert a plug for a numbered document set:

  1. While you are maintaining the clause, click .
  2. In the lookup, select Numbered Document Set.
  3. inserts the plug token $$NDS^CODE=##.
  4. After the = sign, type the code for the document set you want to include.  For example:

Documents:

$$NDS^CODE=STANDARD##
