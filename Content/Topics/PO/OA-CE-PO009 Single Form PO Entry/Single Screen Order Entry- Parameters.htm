Single Screen Order Entry- Parameters

The following parameters allow you to specify the number of order lines available when entering different types of order. In each case the maximum number of lines is 50, and the default (if the parameter is not set) is 5.

Parameter

Description
MAXLINMA The number of lines available on single form material order entry screens.
MAXLINOV The number of lines available on single form overhead order entry screens.

MAXLINST

 The number of lines available on single form order entry screens.
MAXLINAS The number of lines available on single form asset order entry screens.
MAXLINWS The number of lines available on single form workshop order entry screens.
