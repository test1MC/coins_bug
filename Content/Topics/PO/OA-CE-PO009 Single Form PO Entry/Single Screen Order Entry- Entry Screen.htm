Single Form Order Entry - Entry Screen

For each type of order, the single form entry is available as a tab on the order browse.

Single Form Order Entry Screen

The screen consists of the order header fields, arranged in layout frames, and tabs that allow you to enter the different types of order line.

  • Simple PO Entry – Materials
    • plus and Overhead Tabs (if the PO parameter ALWMULMA is set to Y)
  • Simple PO Entry – Workshop
    • plus Asset and tabs (if the PO parameter ALWMULMA is set to Y)
  • Simple PO Entry –
    • plus Overhead tab (if the PO parameter ALWMULMA is set to Y)

Each screen also includes an Other Details tab.
