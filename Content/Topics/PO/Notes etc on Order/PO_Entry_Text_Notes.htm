Comments, Inclusions and Exclusions on Orders

Order Types

The A new Show Comments Tab on PO Entry box field on the Order Types screens allows you to specify on each order type whether the text editor fields are available.

Order Entry - Comments Tab

If the Show Comments Tab on PO Entry box on the order type is ed, a Comments tab is available on the order header. This contains three fields.

Fields

Field
Description

Comments

These notes are primarily for printing on the purchase order form.

Inclusions

This field allows you to clarify what is included in the overall scope of the order.

Exclusions

This field allows you to clarify what is excluded from the overall scope of the order.

NOTE

When copying a template order to a new order, or copying any other type of order to a new order, the contents of these fields on the template/original order are copied onto the new order.

Printing the Comment Fields

The text can be printed on order forms using the following fields:

  • poh_notes
  • poh_inclusions
  • poh_exclusions
