Trade Codes

This topic demonstrates how to set up Trade Codes.

Trade Codes are purchasing codes that will be used on the Subcontract order.

  1. Navigate to Procurement > Commodity, Trade, Equipment Workbench > Trades.

  1. Select .

  1. Enter desired Trade Code and Description. Click Header if you wish to break up the code list. You can group the detailed codes (individual commodities, trades, etc.) under Headings (which can be used when sourcing Trades).

    Click Retired if the code is retired.

  2. Click the drop down lists Units of Quantity and Units of Price to choose a relevant measurement. These are mandatory fields. If desired, enter an Extended Description in the field.

 

It is recommend to choose "EA-Each" for both quantity and price. This has been the standard used for the codes that already exist in the system.
