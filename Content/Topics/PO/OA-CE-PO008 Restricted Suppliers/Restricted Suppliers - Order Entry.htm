Restricted and Subcontractors - Order Entry

If restricted supplier or subcontractor lists have been set up for a , will only allow you to enter purchase orders for valid companies on those lists.

Validation applies to all versions of material, and subcontract order entry. So, for example, it also applies if you “Raise PO” from a list of packages or resources on a project or "Raise PO" from requisitions.

Material and Orders

When you enter the number when adding a material or order, if a restricted list of suppliers is present displays a message below the field.

You will only be able to add suppliers from the 's restricted supplier list, and you will not be able to select a supplier:

  • If either the Valid From or Valid To dates have been populated for that supplier, and the order date is not within the valid range.
  • If the field is ed for that supplier.

Subcontract Orders

When you enter the number when adding a subcontract order, if a restricted list of subcontractors is present displays a message below the field.

You will only be able to add subcontractors from the 's restricted subcontractor list, and you will not be able to select a subcontractor:

  • If either the Valid From or Valid To dates have been populated for that subcontractor, and the order date is not within the valid range.
  • If the field is ed for that subcontractor.

Multi- Orders

If the order in question is a multi- order, then once you have selected the on the line, validates the supplier entered on the order header with reference to the new ’s list of Restricted Suppliers (if present).
