Requisitions - Parameters

The following new PO parameters affect requisitions.

Parameter

Description
GENCREQ Whether requisitions are in use (Y or N).
ALWRQNCD Whether a blank code is allowed on requisition lines (Y or N). This only applies if GENCREQ is set to Y.
NCDDEFU The default units (Uoq, Per) for requisition lines with a blank code. This is expected to be a comma-separated list of two units. The first entry is the default Unit of Quantity and the second entry is the default Per. For example: NO,EA.
ALWTYPCH

Whether the line type on requisition lines can be changed (Y or N). This only applies if GENCREQ is set to Y.
