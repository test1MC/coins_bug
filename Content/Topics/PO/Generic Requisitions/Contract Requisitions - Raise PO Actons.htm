Raise PO

On requisitions the Raise PO option is replaced by two new options:

  • Raise Material PO
  • Raise PO

If the requisition contains only lines, the Raise Material PO action is not shown (because material orders cannot contain lines).

On the Requisitions browse, the Raise PO action is replaced by two new choices:

  • Raise Material PO
  • Raise PO

If the selected requisition contains only lines, the Raise Material PO action cannot be used, and an error message is displayed.
