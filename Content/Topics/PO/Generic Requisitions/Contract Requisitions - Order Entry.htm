Requisitions - Tag/Add Requisition Lines

It is possible to Tag/Add Requisition Lines from the Options menu on the Order Summary screen. If the new PO parameter GRNCREQ is set to Y then this option takes the user to the new Requisitions list for the in question.

When the user takes the Add to PO (or the Add Requisition Items) action on the subsequent screen, if the purchase order being created is a material one then no line can be selected and an error is shown.

Requisitions - Multi Update

The Multi Update feature on the Order Lines screen now includes the following additional fields:

  • Commodity Code
  • Unit of Quantity
  • Per
  • (only on material and orders)
  • Cost Category (only on material and orders)

Note that if the PO parameter ALWCHOV1 is set to N, both and Cost Category become disabled.
