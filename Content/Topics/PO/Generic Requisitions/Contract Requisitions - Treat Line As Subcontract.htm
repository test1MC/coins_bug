Requisitions - Treat Line As Subcontract

If an individual line cannot be ordered or fulfilled from (and only if no orders have already been raised against the line), a new action - Treat as Subcontract - allows the line to be marked accordingly. Essentially, the line is being marked as closed so both the Quantity and Quantity Outstanding are set to zero.

A new unlabelled column on the Requisition Lines screen shows the text “Treated as Subcontract”.

If this action is taken in error, an additional action is available to Undo Treat as Subcontract (including the Quantities that were set to zero). This action is only allowed on requisition lines that are currently marked as “Treated as Subcontract”.

These two options are only available of the requisition has been issued.

These options are available for Requisitions and also for all major types of requisition except subcontract (in other words, for material, , overhead, workshop, asset and ).
