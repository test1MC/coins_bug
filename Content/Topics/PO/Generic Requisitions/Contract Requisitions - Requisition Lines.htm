Requisition Lines

If a Requisition contains at least one item, an unlabelled column is shown as highlighted below. This column contains the word “” if the line has been raised against code. Also, in this instance, an additional Named Filter called “ lines only” becomes available.

Requisition lines on Requisitions can be updated after the requisition has been issued. This means that the settings of the PO parameters ALWUPRQL, ALWUPDUN, ALWUPDSC, ALWADRQL, and ALWDLRQL are ignored.

Add Type Selector

If the requisition being entered is a requisition, the “Material” option in the Add Type selector is changed to “Material/”.

If you choose this option, the Code can be either a commodity code or code. The lookup button shows a list containing both commodity codes and codes, with appropriate filters.

You can also add a Text line to record your requirements. The buyer will be able to convert this to the appropriate material/ or line later.
