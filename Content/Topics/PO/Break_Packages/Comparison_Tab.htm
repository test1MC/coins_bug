Price – Comparing Quotes

When you have received quotations from suppliers, you can compare the prices. If necessary you can ‘plug’ any missing prices to allow you to make a realistic comparison (see Price - Plug Rates). If you decide it is better to order some items from one supplier and some from another, you can move items to a new package and raise a separate order from that new package (see Placing Orders for a Package with Different Suppliers).

To compare quotes

  2. Go to the Comparison tab. This shows a matrix with the items from the schedule as rows; the columns show the suppliers who replied, and the prices they have quoted.
