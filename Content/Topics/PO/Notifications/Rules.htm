Notification Rules

PO Internal Notification Rules is a new function that allows you to specify which users are notified about new orders, and under what circumstances. The screen only shows rules that pertain to records in the current .

PO Internal Notification Rules Screen

For each user who is to receive notifications, enter their user ID and select the major type of order for which they will be notified. You can narrow down the orders for which they will be notified by entering a wildcard list for the Entity (that is, , assets, overhead departments, locations or workshop job references, depending on the setting of the Major Order Type) or the Order Type. If the major order type is , you can also narrow down the orders based on various other fields such as Contract Group, Contract Manager, or the analysis sets.

PO Internal Notification Rules Screen - Setting the Major Order Type

You can have more than one rule for a user.
