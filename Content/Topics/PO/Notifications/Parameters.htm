PO Notifications - Parameters

The following new PO parameters affect the PO notification feature.

Parameter

Description

NTFY‑INT

The email template for PO Notifications to internal users. If this is blank, no emails will be sent to internal users.

NTFY‑EXT

The email template for PO Notifications to and subcontractors. If this is blank, no emails will be sent to or subcontractors.

NTFY‑ADM

The email template for PO Notifications to purchasing administrators. If this is blank, no emails will be sent to purchasing administrators.

DEFNTFEX

Whether to send PO Notifications to and subcontractors by default. If this is not set, the N value is assumed.

The settings in these parameters can be overridden for individual or subcontractors using the PONOTIFY setting in EC Organation Setup (see EC Organation Setup).
