Single Form Order Entry - Parameters

The following parameters allow you to specify the number of order lines available when entering different types of order. In each case the maximum number of lines is 50, and the default (if the parameter is not set) is 5.

Parameter

Description
MAXLINMA The number of material lines available on single form entry screens.
MAXLINOV The number of overhead lines available on single form entry screens.
MAXLINST The number of lines available on single form entry screens.
MAXLINAS The number of asset lines available on single form entry screens.
MAXLINWS The number of workshop lines available on single form entry screens.
