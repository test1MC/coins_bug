Overview

Changes have been made to improve the order entry process. These changes include:

  • New functions that allow you to enter order header information and order lines on a single page (referred to as "single form PO entry").
  • The ability to include multiple analysis types on the same order.

Single form PO entry is currently available for the following major types of orders:

  • Material orders
  • Overhead orders
  • orders
  • Asset orders
  • Workshop orders

, , subcontract and service order related orders are excluded.
