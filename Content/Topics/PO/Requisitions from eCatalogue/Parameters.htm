Requisitions from e - Parameters

The following new PO parameters affect the raising of requisitions from the e:

Parameter

Description

RQECTTPS

A list of major types of requisition which can be raised from the e. This can be set to any combination of the following:

  • (M)aterial
  • (O)verhead
  • (W)orkshop
  • (A)sset

RQECTHST

The e host.

RQECTURL

The URL of the e cross- screen.

RQECTUSR

The e user ID.

RQECTPWD

The e password.
