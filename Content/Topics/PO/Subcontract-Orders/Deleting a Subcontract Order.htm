Deleting a Subcontract Order

 

If an Order has NOT been sent to the Subcontractor, it can be deleted.  Deleting the order will reduce the liability on that Order to 0.

From the Procurement Module select Subcontract Orders.

Select the tick box next to the order.

‘%’  Note - that a Subcontract Order can only be deleted from the Subcontract Orders Workbench

Select the Delete Button.  Select OK on the warning message to permanently delete the order.

‘%’  Note - that Change No DEL appears and that the status changes to CANCELLED
