Subcontract Amendment Variation Order

If an Order changes there is no requirement to cancel it and start again.  Users have the option to create a Subcontract Amendment Variation Order. 

From the Procurement Module select Subcontract Orders.

Select the tick box next to the order.

 

An Amendment Variation Order can be created from the Subcontract Orders Workbench OR from within the Order using the Options drop down list

From the Actions drop down menu select Create amendment Variation Order and Select the Apply Action Button.  You are taken into the Subcontract Orders screen.

‘%’  Note - that a Change number appears under the Order Number and that certain fields are greyed out to prevent changes

On the Main Tab enter a description that explains the reason for the variation.

All other fields in the Main tab are left as they are
