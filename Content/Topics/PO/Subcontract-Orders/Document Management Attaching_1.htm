3.7 Document Management – Attaching the Order PDF to the Order

Once the Signed Order has been received from the Subcontractor, it is to be scanned and stored electronically on the system against the order.

    From the Procurement Module select Subcontract Orders.

    Select the tick box next to the order.

    Select the Add Order Header Documents button.

    Select the Upload Document tab.

    Select the Browse button, locate the PDF and Select on the save button.

‘%’  Remember - Select ‘OK’ on the message stating that the image has been uploaded

    Select the Subcontract Orders workbench and confirm that the document has been uploaded.

‘%’  Note - that the Add Order Header Documents Button will now have changed colour to indicate that documents exist

‘%’  Remember - that any Subcontrat Order documents can be scanned and uploaded / stored within .  This includes any offline order documentation that is created (Word, Excel etc)
