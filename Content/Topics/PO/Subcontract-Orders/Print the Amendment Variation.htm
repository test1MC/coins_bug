3.4 Print the Amendment Variation Order

Once the Amendment Variation Order has been committed it must be printed and sent to the Subcontractor.

    Select on Option drop down list .

    Select Print Order .

    Select the Forward Button to create the PDF of the Order.

‘%’  Note - that printing the order will create the order as a PDF to the Reports Status Workbench and not direct to a printer

‘%’  Remember - Select ‘OK’ on the message stating that the document has been selected to print

‘%’ Remember - Select ‘OK’ on the message stating that the task has been submitted succesfully

    Select the Report Status Workbench and confirm that the Order PDF has been created. 

‘%’  Note - You are NOT required to print a ‘hard copy’ of the order during this Training Course.

    From the Procurement Module select Subcontract Orders and review the order.
