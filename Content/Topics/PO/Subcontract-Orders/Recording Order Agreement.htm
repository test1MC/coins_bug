2.6 Recording Order Agreement Received Date

Once the Signed Order has been received from the Subcontractor, the date agreement received is to be recorded.

    From the Procurement Module select Subcontract Orders:

    Select the Hyperlink into the Order.

    Select the Order Tab and then select the Agreement Sub Tab.

    Select the Open Button.

‘%’  Remember - Select ‘OK’ on the message stating that the Order has been Committed – you will only be able to update certain fields

    Enter the Date Received.

    Select the Save Button.

‘%’  Note - No payment can be raised against the Order until the Agreement Received date has been recorded
