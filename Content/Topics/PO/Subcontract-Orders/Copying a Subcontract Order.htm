Copying a Subcontract Order

An Order can be copied for many reasons, allowing the use of previously completed detail such as Subcontractor – Line Detail, the user is only required to change detail such as Contract No - Value.  Or you may have set up Template Orders.

This section will demonstrate how to copy a Subcontract Order.

From the Procurement Module select Subcontract Orders.

Select the tick box next to the order.

Select the Copy button

Complete the remainder of the Order as required

 

‘The PO Number is blank until you save the Order Header.
Remember to review / change details such as Contract Number, Subcontractor etc
