Cancelling a Subcontract Order

These are for cancelling an order. They print as orders showing the original lines, but with the text NIL instead of the quantity.

To create a cancellation order:

  1. Go to Subcontract Orders.
  2. the original Purchase Order.
  3. Select Create Cancellation Order from the Choose Action list.

    4. adds the below the original order, and automatically assigns it a change number. Cancellation orders show with a D after the change number.

  4. You still need to print and commit it in the usual way. Once you do so, you will not be able to uncommit it.

 

A Cancellation Order can be created from the Subcontract Orders Workbench OR from within the Order using the Options drop down menu
