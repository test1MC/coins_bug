Print the Order

Once the Subcontract Order has been committed it must be printed and sent to the Subcontractor.

‘%’  Note - will produce the covering order to accompany all offline order documents that are produced to support the main order

On the Subcontractor Orders screen, select the order(s) to be printed.

Select Print Subcontract Orders from the actions menu and click apply.

 

Alternatively, within the order you can used the options menu to access the print order option.

 

Select the Output Options tab. Enter a name in the Title field to easily locate it. Select the output type required.

 

Click to generate the PDF of the Order which can then be sent to the subcontractor.
