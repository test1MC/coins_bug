4.2 Print the Cancelation Order

Once the Amendment Variation Order has been committed it must be printed and sent to the Subcontractor.

This section will demonstrate how to print a Subcontract Cancellation Order.

    From the Procurement Module select Subcontract Orders.

    Select the tick box next to the order.

‘%’  Remember - that a Cancelation Order can be created from the Subcontract Orders Workbench OR from within the Order using the Options drop down menu

    From the Actions drop down list select Print Subcontract Order and Select the Apply Action Button. 

‘%’  Remember - Select ‘OK’ on the message stating that the document has been selected to print

‘%’  Remember - Select ‘OK’ on the message stating that the task has been submitted succesfully

    Select the Report Status Workbench and confirm that the Order PDF has been created. 

‘%’  Remember - You are NOT required to print the order during this Training Course.

    From the Procurement Module select Subcontract Orders.
