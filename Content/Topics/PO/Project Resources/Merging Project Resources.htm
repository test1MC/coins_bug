Merging Project Resources

Project resources can be created when requisitions or orders are raised directly from the requisitions or orders workbenches.

If the item that was requisitioned or ordered has the same commodity code, rate, unit of quantity and first 30 characters of description as an existing resource line, that resource line is updated accordingly. If the item is different, it shows as a resource on the appropriate project resources screen with the text ORD in the Via column. This means that the resource has been created as part of the ordering process.

Resources added as part of the ordering process may be the same as existing budget resources, but because they have been given a different description when they were requisitioned or ordered they appear as separate resource lines. In this case you can merge them with the existing budget resource.

  1. On the project resource browse, select one or more resource lines with ORD in the Via column,
  2. Select one line with BUD in the Via column. This is the line that the other lines will be merged into. All lines must have the same commodity code and unit of quantity.
  3. From the actions menu, choose Merge line(s) into budget lines.
