Project Resources

Project resources can be used as a budget and buying list: requisitions and orders for resources can be raised directly from the Resources screens.

You can create the list of project resources by:

  • Extracting resources from .
  • Importing resources using a CSV file (for example, from an estimating package.
  • Adding and updating resources manually.

When resources are created by extracting from or importing from a CSV file, any source lines that have the same type, code, and unit of quantity, and on which the first 30 characters of the description are the same, are combined as a single resource. Similarly, you will be unable to add a resource manually that has the same type, code, unit of quantity, and first 30 characters of the description.
