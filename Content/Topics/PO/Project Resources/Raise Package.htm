Raise Package

You can raise packages directly from the resources screen.

  1. Select the resources you want to add to the package, and use the Raise Package action.

  2. On the Raise Package from Resource List screen, enter the package code and fill in any additional details.

  3. Click (Next).

    This takes you to the package summary screen, from which you can process the package as normal.

On the resources screens, any resources that have been added to a package are marked with a P, and a tooltip shows which package or packages they have been added to.
