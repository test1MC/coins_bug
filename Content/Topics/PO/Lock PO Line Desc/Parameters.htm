Locking the Description of PO Lines - Parameters

The following new parameters are used to lock the description of order lines.

Parameter

Description

POLKDSCA

Purchase Orders – Lock Description (Assets)

POLKDSCI

Purchase Orders – Lock Description ()

POLKDSCM

Purchase Orders – Lock Description (Material)

POLKDSCO

Purchase Orders – Lock Description (Overheads)

POLKDSCP

Purchase Orders – Lock Description ()

POLKDSCW

Purchase Orders – Lock Description (Workshop)

The following existing parameters are used to lock the description of requisition lines.

Parameter

Description

PRLKDSCA

Requisitions – Lock Description (Assets)

PRLKDSCI

Requisitions – Lock Description ()

PRLKDSCM

Requisitions – Lock Description (Material)

PRLKDSCO

Requisitions – Lock Description (Overheads)

PRLKDSCP

Requisitions – Lock Description ()

PRLKDSCW

Requisitions – Lock Description (Workshop)

NOTE

If the existing PO parameter ALWLUKOV is set to N, the order line description for a Luckins item cannot be changed, even if the parameters above are not set to Y.
