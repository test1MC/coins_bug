Locking the Description on PO Lines - Items

If the existing Allow additional description box is ed, and if any of the parameters used to lock order line descriptions are set to Y, the following new field is available in Item Maintenance:

Field
Description

Coaching label for additional description

Text explaining to the user that they should enter additional information specific to the selected code. So, for example, on code that represented some sort of conduit you might set this text to “Please specify size (e.g. 10m) and finish (e.g. Steel)”.
