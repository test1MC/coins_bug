Locking the Description of PO Lines - Overview

You can now lock the description of commodities entered on purchase orders to prevent users changing them. This may be necessary if you are using a detailed set of commodity codes (or are using the file either as a true entity or as a “product” file for use on material orders). This allows a meaningful analysis of purchases based on the standard description, without the concern that the person raising the order may have completely changed the standard description and ordered something unrelated to the commodity code used.

You can allow users to enter an additional description (for example, to specify or size).
