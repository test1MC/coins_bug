Locking Requisition Line Descriptions

If the "lock requisition line descriptions" parameter that corresponds to the type of requisition is set to Y, the “main” description and the “Extended Description” held against the selected commodity code are displayed but cannot be overridden. A new An “additional description” text box is displayed, with the “coaching label” immediately above it.

Requisition line update has also been changed to display as an inline update frame, rather than as a separate update frame.
