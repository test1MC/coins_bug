Terms and Conditions Sets - Order Entry

When you enter an order, determines whether a terms and conditions set is relevant based on the order type or the project. If so, a new the Terms & Conditions tab is available.

Terms & Conditions Tab

When you first enter the order, the lines from the specified terms and conditions set are displayed on the Terms & Conditions tab. Depending on how the individual lines are configured, you can do various things:

  • If the Title or Additional Text allow it, you can update them.
  • If the line is not mandatory, you can delete it using the button.
  • You can add lines using the button in the header (or by pressing Tab on the last field of the last existing line).
  • You can change the order of lines by dragging the line number up or down.
  • The Reload button allows you to reload the terms and conditions lines; this will replace any changes you have made with the original lines. You can also use this if you are updating an order that was not entered using standard Order Entry (but was loaded using Import Purchase Orders, for example). In this case the terms and conditions set will have been assigned to the order but the lines will not have been created.

If you choose a different project or order type, or the Terms & Conditions tab is shown or hidden during order entry, a warning message appears.
