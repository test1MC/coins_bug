Terms and Conditions Sets - Project Information

You can specify terms and conditions sets for material, subcontract and orders on a project.

The Order Forms tab has been renamed Order Forms / T&Cs, and the Head Office and Site columns have been renamed Head Office Form and Site Form.

New fields have been added:

Fields

Field
Description

Material

The terms and conditions set to use on material orders on this project. This is a drop-down list of terms and conditions sets that have been configured.

This is optional, and overrides the set that is specified on the order type.

Similar to the Material field, this is the terms and conditions set to use on orders on this project.

Subcontract

Similar to the Material field, this is the terms and conditions set to use on subcontract orders on this project.
