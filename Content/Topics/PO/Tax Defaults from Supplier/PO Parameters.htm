PO Parameters

The following new PO parameters allow you to specify how US sales tax codes are to be used on PO Entry:

Parameter

Description

DFTAXHDR

Where the sales tax code on the order header defaults from:

J – (the default)

S – Supplier / Subcontractor
DFTAXLIN

Where the sales tax code on order lines defaults from:

H – Order Header

A – Order Line Analysis (the default)
