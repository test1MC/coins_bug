Purchase Order Header

Material, Subcontract and Equipment Orders

On the order header, once a valid number has been entered, the value defaulted into the Sales Tax (code) field initially comes from the .

If the new PO parameter DFTAXHDR is set to “S” then once the supplier and office have been entered, and if the office relates to an existing account on which the Default Sales Tax Code is not blank, the sales tax code is defaulted from this account (even though the may have previously defaulted in a different sales tax code).

Overhead, Workshop and Asset Orders

The Overhead Department, Workshop Reference and Asset Code (which are the equivalents of the Number) currently do not carry a default sales tax code. In this instance, the sales tax code on the order header always defaults from the account when the PO parameter DFTAXHDR is set to "S".
