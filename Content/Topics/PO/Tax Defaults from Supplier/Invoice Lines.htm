Invoice Costing Lines

When an invoice costing line is created the sales tax code on the line is set to the sales tax code on the invoice header.

If the new PL parameter DFTAXLIN is set to “H”, the sales tax code from the invoice header does not change, regardless of the costing analysis entered on the line. The user can manually change the sales tax code on a line by line basis.

If the PL parameter DFTAXLIN is set to “A”, the values entered in the costing analysis (for example, , commodity code, WBS code, , or cost category) can result in the analysis line being populated with a different sales tax code.
