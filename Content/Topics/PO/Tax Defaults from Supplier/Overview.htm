Overview

For US companies (in other words where the CO parameter TAXTYPE is set to SALES), it is now possible for the default sales tax code on purchase orders to be governed by the or subcontract record instead of the , and to default the sales tax code from the order header instead of regenerating the code on each order line based on the , section, and cost category.

The same functionality is also available on invoices that are not related to a purchase order.
