Order Lines

When an order line is created, the sales tax code on the line is set to the Sales Tax (Code) on the order header.

If the new PO parameter DFTAXLIN is set to “H”, the sales tax code from the order header does not change, regardless of the analysis entered on the line. The user can manually change the sales tax code on a line by line basis.

If the PO parameter DFTAXLIN is set to “A”, the values entered in the analysis fields (for example, , commodity code, WBS code, , or cost category) can result in the order line being populated with a different sales tax code.
