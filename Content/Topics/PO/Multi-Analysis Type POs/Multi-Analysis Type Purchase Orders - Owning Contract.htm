Multi-Analysis Type Purchase Orders - Owning

The number entered on the order header screen is deemed to be the “owning” ; only the owning is shown on the orders browse. This means that if an order relates to more than one , you can only filter or search for it using the number entered on the order header.

A single purchase order can only have one accrual method. This accrual method is typically dictated by the , although it can be derived from the supplier or the order type. Where it is the number that dictates the accrual method, it is the owning number that determines the accrual method, rather than any other , overhead department, location etc entered on order lines. On the Order Summary screen this field is referred to as Accrual Type.

For purchase orders to operate correctly, GRNs must be entered. For this reason, if the accrual type stamped on the order header implies that GRNs are not required, “ Location” is excluded from the Analysis Type selection drop down box during order line entry.
