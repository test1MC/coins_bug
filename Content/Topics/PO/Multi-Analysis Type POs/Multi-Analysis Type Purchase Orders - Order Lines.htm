Multi-Analysis Type Purchase Orders - Order Lines

New columns have been added to the order lines browses.

Order Line Fields

Columns

Field
Description

Analysis Type

This is available if multi-analysis is allowed on the major order type (see 1, Multiple Analysis Types - Parameters). The possible values in the Analysis Type list depend on the major type of order being entered.

Material Orders:

  • Asset
  • Location
  • Dept

Workshop Orders:

  • Workshop Ref
  • Asset
  • Location

Asset Orders:

  • Asset
  • Workshop Ref
  • Location

Orders:

  • Location
  • Dept

If the purchase order’s Accrual Type means that GRNs are not required, the “ Location” option is omitted from the list.

Analysis

The analysis code for the order line. The code, field lookup and validation are determined by the analysis type selected.

This column is labelled Analysis if multi-analysis is allowed on the major order type (see 1, Multiple Analysis Types - Parameters). Otherwise, the field label relates to the major order type (for example, , Workshop Ref).

NOTE

If you use a item on an order line with the analysis type of Location, it is for . If you use a item on a line with a different analysis type, it is treated as a product code.

Item Summary

If multi-analysis is allowed on the major order type (see 1, Multiple Analysis Types - Parameters), and only if the order contains more than one analysis type, the Analysis Type and Analysis columns (as on the order lines browse) are shown.
