Multi-Analysis Type Purchase Orders - Order Browses

The “M” column, previously shown on Material Orders browses, has been updated, and has been added on the Orders, Overhead Orders, Asset Orders, Workshop Orders and Orders workbenches.

Column

Field
Description

M

This now shows:

  • “MA” if the purchase order relates to more than one analysis type.

  • “M” if the order relates to more than one but does not relate to more than one analysis type.

On material order browses, this only appears if either of the PO parameters ALWMULMA or ALWMLAMA is set to Y.
