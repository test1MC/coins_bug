Pick Up Orders Overview

It is common practice, particularly for service technicians, to physically go to a supplier to purchase and pick up required materials when the need is identified. Recommended best practice is to request a purchase order at time of purchase and provide the required details to the back office. As the goods are being immediately picked up it is known at the time of entry that the goods are "received".

A purchase order can now be marked as “Pick Up” at the time of order entry. When a Pick Up order is committed, a GRN for all items on the order is created.

The following major types of order cater for “Pick Ups”:

  • Material
  • Overhead
  • Workshop
  • Asset

Note that whilst orders are included in the list, additional fields such as Serial Number that are normally entered as part of the manual GRN Entry routine are set to blank.

Initial Configuration

Pick Up Orders - Parameters

The following new PO parameter allows you to enable the pick up order functionality.

Parameter

Description
ALWPUORD Whether to allow pick up orders.

Screen and Processing Changes

Pick Up Orders - Order Entry

If the new PO parameter ALWPUORD is set to Y, a new box, Pick Up Order?, allows you to mark the order as a pick up order.

If the accrual type of the order (which depends on the type of the order, the analysis (, department), and the supplier office) means that GRNs are NOT to be entered, or if the order being created is a template order, this box is automatically uned and disabled.

If the order being created is order and the PO parameter AUTOSTK is set to Y, the option to “Commit/Print/GRN/Batch” does not appear in the Options menu on the Order Summary screen. This is because the pick up order functionality effectively replaces that option (although the order will not be automatically printed).

PO Commit

When a pick up order is committed, creates a GRN for all items on the order.

If the order being committed is a variation order, only additional quantities are auto-GRN’d. Note that you cannot reduce the quantity on an existing line because if a GRN line is present, the revised order quantity cannot be lower than the delivered quantity.

Order Browse Screens

If the PO parameter ALWPUORD is set to Y, an additional filter called “Pick Up Orders Only” is available on all browses for the affected major order types, and the Pick Up Order? field is available in the column sets.
