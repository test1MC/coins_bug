National Purchasing Agreements - Parameters

The following new CI parameters need to be set to enable National Purchasing Agreements:

Parameter

Description
USENPA Whether National Purchasing Agreements are in use (Y or N).
NPAGROUP The Document Group for National Purchasing Agreements. If a document in this group is attached to a company record, when an order is raised for that company, a link on the order header allows the user to view the document.

 

NOTE

Both of the above parameters should always be set in “global” mode. In other words, any company-specific version should be ignored. This is because specifying a supplier as having a National Purchasing Agreement is done in the CIW which is not company-specific.

In addition, we recommend that you set the PO parameter COSTDET to Y.
