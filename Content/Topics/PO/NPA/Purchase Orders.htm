National Purchasing Agreements - Purchase Orders

Order Header

If you place an order against a company that has the National Purchasing Agreement box ed, the text “NPA” is shown alongside the Supplier name in the order header. If at least one agreement document has been uploaded against the company, a hyperlink on the “NPA” text opens a new tab/window which displays all documents that have been uploaded to the company record using the Document Group specified in the new CI parameter NPAGROUP.

Order Header NPA Link

Note: If the CI parameter NPAGROUP is blank then no “NPA” text will be shown.

Line Entry

If the CI parameter USENPA is set to Y, two additional fields appear when a purchase order line is entered or amended.

Order Line with NPA Fields

Field
Description

Eligible NPA Purchase with Supplier

Whether this line is covered by a National Purchasing Agreement.

This is ed, by default, if the supplier entered on the order header is marked as having a National Purchasing Agreement.

Eligible Manufacturer

This field is optional. If there is a National Purchasing Agreement in place with the manufacturer of the goods (as opposed to the supplier of the goods), choose the manufacturer. This denotes that this line should be marked as being eligible.

The lookup displays a list of manufacturers from the Central Repository with whom you have a National Purchasing Agreement. A manufacturer, in Central Repository terms, is company whose lists of Company Types includes the value set against the CI parameter COT-MANU. Typically, this parameter is set to “MANUF” or something similar. Only manufacturers that also have the National Purchasing Agreement field ed appear in the lookup list .

Both of these new fields are updateable even after the purchase order has been committed.

NOTE

If the agreement is only with the manufacturer of the goods, not the supplier, leave the Eligible NPA Purchase with Supplier box unticked, but choose a manufacturer.
