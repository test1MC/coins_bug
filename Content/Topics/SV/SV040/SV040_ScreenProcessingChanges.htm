Status Maintenance

A new Proposal Details column is available as a box on the Status Maintenance screen in > Global Config, as shown in Figure 1: [%=Variables.Sv Variation caps%] Status Maintenance. It only appears when SV Parameter PROPDET is not blank. You can select this field when you wish to set up a Status that will show proposal details (Proposal Name and Proposed Number).

Type Maintenance

A new Proposal Details column is available as a box on the Type Maintenance screen in > Global Config, as shown in Figure 2: [%=Variables.Sv Variation caps%] Type Maintenance. It only appears when SV Parameter PROPDET is not blank. You can select this field when you wish to set up a Type that will show proposal details (Proposal Name and Proposed Number).

Details

The new Proposal Number and Proposed Date fields appear in the Last Change Date layout box on the Details screen in > [working ] > , as shown in Figure 3: [%=Variables.Sv Variation caps%] Details. They only appear when:

  • The SV Parameter PROPDET is not blank.
  • The Proposal Number is zero.
  • The Proposal Details box on the corresponding Type or Status is ed.

Details

Proposal Number

A sequential number is generated either by the Series or by the , depending on SV parameter PROPDET. It cannot be changed.

Proposed Date

This defaults to today’s date, but it can be changed.

Browse Screen

Update Status Action

When a Status is updated by using the Update Status action on the browse screen (see Figure 5: Update [%=Variables.Sv Variation caps%] Status Action), the Proposal Number and Proposed Date are assigned to that under the following conditions:

  • The SV Parameter PROPDET is not blank.
  • The Proposal Number is zero.
  • The Proposal Details box on the corresponding Type or Status is ed.

Update Status Action

Workflow-Controlled Status Update

When the Status is updated via workflow, the Proposal Number and Proposed Date are saved on the under the following conditions:

  • The SV Parameter PROPDET is not blank.
  • The Proposal Number is zero.
  • The Proposal Details box on the corresponding Type or Status is ed.

History

The new Proposal Number and Proposed Date fields on a are copied to the History record.
