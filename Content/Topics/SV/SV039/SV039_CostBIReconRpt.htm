SV Cost/Billing Reconciliation Report: Cost/Billing Reconciliation Report

Menu Item

The Cost/Billing Reconciliation Report is now available on the > [working ] > Reports > menu, as shown in Figure 1: Menu Item. This report shows data from transactions and source transactions.

Menu Item

Selection Screen

When you select Cost/Billing Reconciliation Report from the menu, the selection screen appears, as shown in Figure 2: Report Selection Screen.

Report Selection Screen

Field Definitions

From/To

Enter the period or range of periods to report on.

From/To Transaction Date

Enter the date or range of dates to report on.

Include Transactions

Select the transactions you want to report on: All, Billed, or Unbilled.

The following records can be found on the report when All is selected:

  • Source Transactions records for the selected .
  • All the transactions belonging to Certificates that have a non-blank source (if the Sales Series uses Applications and Certificates or Certificates only).
  • All the transactions belonging to Applications that have a non-blank source and were not copied to Certificates.
  • All the transactions that have been populated by source transaction records but not associated with APP/CERT (unbilled).
