SV Cost/Billing Reconciliation Report: Screen and Processing Changes

Populate Transactions

transactions can be populated from the following areas: , Applications, and Certifications. These areas have been modified to provide the Cost/Billing Reconciliation Report with relevant transaction data.

Applications and Certifications

The following areas of Applications and Certifications have been modified to accommodate the Cost/Billing Reconciliation Report:

  • Copy Linked Application Measures (Certificates)
  • Populate Measures Actual Costs (Certificates)
  • Empty Applications/Certificates
  • Manual Delete of Cost Transaction records
