SV Parameter

The new SV parameter below is found in OA to define the Currency Conversion Type for use with multi-currency functionality.

Parameter

Description

CURCTYPE

Currency Conversion Type used for multi-currency functionality for SC in

How CURCTYPE Is Used

Since SV parameter CURCTYPE defines the Exchange Rate Types, this parameter should use one of the Conversion Rate Codes listed in System > Company Setup > Setup > Conversion Rate Codes, as shown in Figure 1: Conversion Rate Codes. If it is blank, the Currency Exchange Rates defined for the transactional type are used. Otherwise, Currency Exchange Rates are used as defined for the type by parameter CURCTYPE.

Conversion Rate Codes

Exchange Rate Types can be defined for different currencies. Exchange Rates for each Exchange Rate Type and for each currency can be defined for various effective dates in the Currency Exchange function in System > Company Setup > Setup, as shown in Figure 2: Currency Exchange.
