SV SC Cert Measure

Copy Measures

SV Measures for an SC Certificate can be copied from the following:

  • Measures
  • Application Measures
  • Certificate Measures

All of the above measures are in Currency; however, the SC Certificate is in Order Currency. When copying these measures to SC Certificate Measures, all of the values listed below are converted from Currency to the Order Currency against which the SC Certificate is raised. The effective date used to find the exchange rate is the SC Certificate date.

  • Value To Date
  • Value This Measure
  • Value This Period

Copy Materials

You can copy Application and Certificate materials to an SC Certificate. However, since Application and Certificate materials are in Currency and SC Certificates are in Order Currency, Application and Certificate materials must be converted from Currency to Order Currency during the Copy process. The effective date used to find the exchange rate is the SC Certificate date.
