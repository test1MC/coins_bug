SV Subcontract Reports

Subcontract Reports in can now show amounts in multi-currency format, as described in this section.

Subcontract Report

A new Show Amounts In field is found on the selection screen of the Subcontract Report.

Subcontract Report

  • When Subcontractor Currency or Order Currency are not selected, the title of the Report shows the selected Report Currency with the label Currency.
  • The Order header shows the Report Currency with the label Currency. Original, Committed, and Estimated values of the Order are shown in the report.
  • The Body also displays the currency with the label Currency. The total for each is suppressed when either Subcontractor Currency or Order Currency is selected.

Subcontract Items Report

A new Show Amounts In field is found on the selection screen of the Subcontract Items Report (see Figure 1: Subcontract [%=Variables.Sv Variations caps%] Report for an example of where this would be located on the screen).

  • When Subcontractor Currency or Order Currency are not selected, the title of the Report shows the selected Report Currency with the label Currency.
