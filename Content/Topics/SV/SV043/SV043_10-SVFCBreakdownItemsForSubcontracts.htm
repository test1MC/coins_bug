SV Breakdown Items for Subcontracts

Breakdown Items can be created for by selecting or values, as shown in Figure 1: [%=Variables.CVR caps%] Breakdown Items.

Breakdown Items

Breakdown Items are created for Client and Subcontract . Breakdown Items are in Currency. While creating Breakdown Items from an SC , the and values of the SC must be converted from Order Currency to Currency to get a new value in Currency. The old Breakdown Item value remains in the source currency.

  • If creating a Breakdown Item from an SC , the old value is in Order Currency.
  • If creating a Breakdown Item from a Client , the old value is in Currency. The date used for finding an exchange rate is the current Period.
