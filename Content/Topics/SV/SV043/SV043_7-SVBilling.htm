You can copy SC Certificate Measures and SC Certificate Materials to Applications, Certifications, and Measures through certain actions on the Application/Certification/ Workbench, as discussed in this section. The effective date used to find the exchange rate is the Application/Certificate date.

Applications/Certifications/

The Application/Certification/ Workbench has the following two actions (see Figure 1: Copy SC Actions) that pertain to copying:

  • Copy SC Certificate Measures – see Section 1.2 Copy SC Certificate Measures for more information
  • Copy SC Materials – see Section 1.3 Copy SC Materials for more information

Copy SC Actions

Copy SC Certificate Measures

You can copy SC Certificate Measures to Applications, Certificates, and . Since SC Certificate Measures are in Order Currency, and Application/Certificate/ are in Currency, To Date Value, This Measure Value, and This Total Value amounts must be converted from Order Currency to the exchange rate code defined by SV parameter CURCTYPE (if not blank) or the transactional rate code (if SV parameter CURCTYPE is blank) for Currency. The effective date used to find the exchange rate is the Application/Certificate date.

Copy SC Materials

You can copy SC Certificate Materials to Applications, Certificates, and . Since SC Certificate Materails are in Order Currency, and Application/Certificate/ are in Currency, the Value and Rate must be converted from Order Currency to the exchange rate code defined by SV parameter CURCTYPE (if not blank) or the transactional rate code (if SV parameter CURCTYPE is blank) for Currency. The effective date used to find the exchange rate is the Application/Certificate date.
