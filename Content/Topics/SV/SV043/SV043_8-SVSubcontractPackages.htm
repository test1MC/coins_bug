SV Subcontract Packages

The Subcontract Packages menu item (see Figure 1: Subcontract Packages) is only visible when SV parameter USEPACK equals Y. You can use this function to allocate Items and Resources to an appropriate package by using the Allocation (XL) option.

Using the Populate PO action on the Subcontract Packages browse, PO Lines can be populated from Items and Resources that have been assigned a package.

Subcontract Packages

A Subcontract Order can have a currency different from Currency. Values on Items and Resources are in Currency. When populating PO Lines from Items and Resources, currency conversion must be done from Currency to Order Currency. The effective date used for finding the exchange rate is the Order date.
