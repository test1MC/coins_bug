Change Rate Type Utility

Menu Option

A new Change Rate Type Utility menu option is available in > Selected > Admin, as shown in Figure 1: Change Rate Type Utility Menu Option. This utility lets you replace an existing Rate Type with another Rate Type.

Change Rate Type Utility Menu Option

Selection Screen

When you select Change Rate Type Utility from the menu (Figure 1: Change Rate Type Utility Menu Option above), a selection screen appears, as shown in Figure 2: Change Rate Type Utility Screen.

Change Rate Type Utility Screen

The following rules apply when attempting to change a Rate Type using this utility:

    1. If no transactions exist, you can make any changes to the new Rate Type.
    2. If no Payroll transactions exist, you can create any new Rate Type with the same source types.
    3. If Payroll transactions exist, but are not linked to an application or certificate, you can make changes according to Rule #2 above, and all transactions will be recalculated for the new Extra Payroll Groups data.

Before Running Utility

Before running the utility, you should first void all posted certificates if any of the following conditions exist:

  • The new Rate Type has different source types and existing transactions,
  • The new Rate Type has the same source types but different Extra Payroll Groups, and billed Payroll transactions exist.
NOTE:
An error message displays for Transactions billed to posted certificates, which states that you need to void all of them first.

For Transactions billed to voided certificates, the utility does the following:

  • Purges all transactions linked to voided certificates.
  • Resets all source transactions back to unbilled.

The new Rate Type is set for the level; it is set to blank for all series.
