Named Filter

The bottom of the browse has a named Filter that can be linked to Risks/Opportunities using one of the following options:

  • All - displays all linked and unlinked Risks and Opportunities.
  • Linked - displays all Risks and Opportunities linked to the current .
  • Unlinked - displays only Risks and Opportunities that can be linked to the current . This filter option now takes into account the new Full Link Type; Risks and Opportunities linked in Full mode do not appear in the Unlinked view.

Link Risks and Opportunities Browse Action

The Link Risks and Opportunities action at the bottom of the browse screen (see Figure 3: [%=Variables.Sv Variation caps%] - Risks and Opportunities Browse) now takes into account the new Full link type between the and the Risks/Opportunities record.

  • If the current is already linked to a Risks/Opportunity record with a Full link type, the action is suppressed.
  • When a Risk/Opportunity record is linked to the by a Full link type, both and values are updated.
