Standard Cost Based Report Changes - Naming Conventions

When documents attached to Transactions are exported using the Standard Cost Based Report, the naming conventions are as shown below, where:

  • Sequence = Sequence of Transaction (bic_seq). This is a new field that is shown on the Standard Cost Based Report and can be shown on the Cost Transaction browse.
  • Number = Serial Number with format 999.

Documents

The naming convention for exported documents related to transactions whose source is CTPL is:

Sequence + “_” + Supplier Reference of Invoice (ain_supref) + “_” + Number + Document Name.

The Supplier Reference of an invoice can be blank if the cost transaction did not originate from . In that case the naming convention of the exported document is:

Sequence + “_” + Number + Document Name

Subcontract Ledger Documents

The naming convention for exported documents related to transactions whose source is CTSC is:

Sequence + “_” + Subcontractor’s Reference of SC Certificate (sbp_subref) + “_” + Number + Document Name.

The Subcontractor’s Reference of a certificate can be blank if the cost transaction did not originate from the Subcontract Ledger. In that case the naming convention of the exported document is:

Sequence + “_” + Number + Document Name

JC Timesheet Documents

The naming convention for exported documents related to transactions whose source is CTJC/JCTS is:

Sequence + “_” + Employee (jem_code) + “_” + Number + Document Name.

If the employee code is blank, the naming convention of the exported document is:

Sequence + “_” + Number + Document Name.

Charges Documents

The naming convention for exported documents related to transactions whose source is CTPC/PCINV is:

Sequence + “_” + Invoice Number (bic_iref) + “_” + Charge Line(bic_xref) + “_” + Number + Document Name.

If the invoice number is blank, the naming convention of the exported document is:

Sequence + “_” + Number + Document Name.

Other Transaction Documents

The naming convention for exported documents related to transactions whose source is CTOT is:

Sequence + “_” + Number + Document Name.
