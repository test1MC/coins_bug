Standard Cost Based Report - Extra Payroll Groups

Extra payroll groups can be set for Rate Types used for a . By default, the Standard Cost Based Report displays these extra payroll groups. If you do not want clients to see the values entered for extra payroll groups, display of these values on the report can be suppressed.

A new column has been added to the Extra Payroll Groups browse in Rate Maintenance.

 

Column

Field
Description

Show on Report

If this is ed (the default), the Extra Payroll Group is displayed on the Standard Cost Based Report.
