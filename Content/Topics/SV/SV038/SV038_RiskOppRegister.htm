Field Descriptions

Reference

This field can be automatically generated or manually entered depending on the setting of SV parameter ROPMANNO. A Reference code is generated automatically when ROPMANNO is set to N.

Category

Select the category for this risk/opportunity.

Subject

The subject of this risk/opportunity.

Status

Select Open or Closed from the drop-down list as the status for the risk/opportunity.

Closed Date

The date that the risk/opportunity was closed.

Cost Code

The cost code for the risk/opportunity. If this is not blank, validation occurs.

Description

A description of the cost code.

Link to

Indicate whether there is a link to for this risk/opportunity.

This field is only used when parameter LNKROPVAR is set.

Type

This field is enabled when Link to is ed. When it is enabled, you may select either or .

This field is only used when parameter LNKROPVAR is set.
