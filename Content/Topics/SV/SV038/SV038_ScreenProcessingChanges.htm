SV Risks and Opportunities : Screen and Processing Changes

Configuration

A number of SV parameters have been added to the module so that the new R&O  can be used.

Parameter

Description
LNKROPVAR Link Risks and Opportunities to ?
ROPCODE Characters to precede Risk and Opportunities Reference code

ROPDEFCAT

The default category for Risks and Opportunities

ROPDEFMTH

Default Method for Risks and Opportunities

ROPDEFST

Default Status for Risks and Opportunities

ROPFORM

Risks and Opportunities Reference code format

ROPMANNO

Allow manual entry of Risks and Opportunities Reference code

ROPUDFRM

Page section for Risks and Opportunities user defined fields

Global Config Menu

The following menu items are available on the > Global Config menu, as shown in Figure 1: Global Config Menu:

  • R & O Categories (Section 2.2)
  • R & O Mitigation Methods (Section 2.3)
  • R & O Status (Section 2.4)

Each of these menu items is discussed in more detail later in this document.

Global Config Menu

R & O Categories

The R & O Categories browse allows you to create and update a list of R & O Categories, as shown in Figure 2: R & O Categories Screen. All of the fields are updatable, and the Code field is mandatory.

R & O Mitigation Methods

The R & O Mitigation Methods browse allows you to create and update a list of R & O mitigation methods, as shown in Figure 3: R & O Mitigation Methods Screen. All of the fields are updatable, and the Code field is mandatory.

R & O Mitigation Methods Screen

R & O Status

The R & O Status browse allows you to create and update a list of R & O statuses, as shown in Figure 4: R & O Status Screen. All of the fields are updatable, and the Code field is mandatory.

R & O Status Screen

An R & O Item can be linked to only when SV Parameter LNKROPVAR is set to Y and the Linked to flag on an R & O record is set.

Expected and Expected fields are available on the . Their values are used for calculating the Expected values of an R & O Item.

Details

There is now a Risk & Opportunity Margin Impact layout on the Details browse screen, as shown in Figure 5: Risk and Opportunity Layout Screen. You can update Expected and values here when a is not approved.

The Pessimistic and Optimistic values are RO fields that are calculated and display-only.

The Expected and Expected values are used for calculating the Expected values of an R & O Item.
