Workspace

The Workspace browse has column set capability so that you can view the elements of data you have imported. Here you can edit, validate, and post the imported data. A sample browse is shown in Figure 4: Workspace Browse.

The Status column is found in the Workspace browse. Also, a record is flagged in an Alert (red flag) column if it does not pass validation to qualify as a valid Planned Inspection.

Workspace Browse

Options Menu

The Options menu at the top of the Workspace browse contains the following options: Validate All Records, Post All Records, and Clear Workspace, as shown in Figure 5: Workspace Options Menu. Each option is explained below.

Workspace Options Menu

Validate All Records

Use this option to perform validation of the Workspace records. A record is flagged if it does not pass validation to qualify as a valid Planned Inspection.

The Status of a validated record is shown as a green circle with the tooltip appearing as "OK", as shown in Figure 6: Green Circle Status.

Green Circle Status

Post All Records

All records in the Workspace must be valid in order for the Post to process. If at least one record is not valid, an error message is shown (see Figure 7: Validation Error Message) and posting is aborted.

The Post process removes all Workspace records upon successful completion.

Validation Error Message

Clear Workspace

This function lets you clear the Workspace of all data.
