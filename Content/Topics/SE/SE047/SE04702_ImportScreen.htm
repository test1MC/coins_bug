Import Screen

The Import screen (Figure 2: Import Screen) allows you to import data for Planned Inspections from a CSV file. This data can then be imported into the Workspace, where it can be edited, validated, and posted.

Also found on this screen is a link to an Example CSV file. For more information on this CSV spreadsheet, see below.

Import Screen

Example CSV Spreadsheet

The columns in the Example CSV file are configurable, i.e., you can rearrange or remove them as needed. In order to create new Planned Inspection records, the CSV spreadsheet must reference the mandatory fields required to create an Planned Inspection.

The spreadsheet (and Workspace) only reference the Planned Inspection header information (see Figure 3: CSV Header Information), including , Location, Service Department, and Renewal. It does not reference monthly columns since the monthly columns reflect each ’s Renewal Date information and are therefore different between . When posting the imported data, a monthly schedule is generated and based on the Standard Schedule referenced.
