RO Fields

Since some Planned Inspections may not be set up to generate the Covered Inspection data, you must individually manage these . Nine new RO fields (Figure 1: RO Fields) have therefore been added to the function and the to help you do this. They can help identify in which the Planned Inspection values and Covered Asset values are different.

All of these RO fields are found in a column set at the bottom of the screen. They contain Planned, Tasked, and Remaining totals in columns for # of Pieces, Hours, and Amount($).

RO Fields

  • The RO fields are the sum for all Locations, all Service Departments, and the current Renewal (based on today).
  • The RO fields support parameters for Location, Service Department, and Renewal.
