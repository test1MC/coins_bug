Import Data Menu

A new An Import Planned Inspections submenu is available under the Import Data menu in Service Management > Administration, as shown in Figure 1: Import [%=Variables.Se Agreement caps%] Planned Inspections Submenu. This submenu allows you to create Planned Inspections. It includes two functions:

  • Import - This provides you with an Example CSV spreadsheet file. The columns in the Example CSV file are configurable, i.e., you can rearrange or remove them as needed. See 3 - Import Screen for more information.
  • Workspace - The Workspace browse has column set capability, so that you can view the elements of data you have imported. See 1 - Workspace for more information.

Import Planned Inspections Submenu

This import is designed as a multi-step function:

  1. Import the data into a temporary Workspace.
  2. Use the Workspace browse to make corrections to the data. You can also use the Delete button to remove individual rows of data.
  3. Select options from the Options menu to validate the data (Validate All Records), post the data (Post All Records), and clear out all data (Clear Workspace).
