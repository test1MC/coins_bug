RO Fields

New RO fields have been added at the level to provide access to consolidated comment records. Consolidated Service Reports generated in Service Management > Utilities > Print Service Reports - Consolidated can use these consolidated RO fields.

These RO fields include:

  • For Problem Codes: RO_sso_ProblemConsolidated with Comment Type parameter, which allows access to a specific type of consolidated comment. The “ALL” value can allow you to access consolidated comments of any Comment Type.
  • For Resolutions: RO_sso_ResolutionConsolidated with Comment Type parameter, which allows access to a specific type of consolidated comment. The “ALL” value can allow you to access consolidated comments of any Comment Type.
