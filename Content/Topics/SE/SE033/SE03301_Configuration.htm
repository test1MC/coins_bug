Two new parameters have been added to COINS OA Service Management to provide Comment Types for Consolidated Problem Codes and Resolutions.

Parameter

Description

CMPROBCN

Comment Type for Problem Code Consolidated Log Entry

CMRESCN

Comment Type for Resolution Consolidated Log Entry

The Comment Type referenced in these parameters is used for Consolidated comments and is considered to be an Invoice Comment (set up in the Comment Types function, as shown in Figure 1: Comment Types Setup). During the invoice generation process, Invoice Comments are copied to the Invoices function and can be further edited before printing the invoice.

Comment Types Setup

NOTE:
SE Parameters CMPROBCD and CMRESOL are not affected by this functionality. The Comment Type referenced in these parameters will continue to be used only for Source comments and is not considered to be an Invoice Comment.
