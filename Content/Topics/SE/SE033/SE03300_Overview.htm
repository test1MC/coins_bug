Overview

COINS OA users now have the ability to consolidate invoice comments into a single text area from the function. The consolidated comments can then be edited before being seen on an invoice.

  • Invoice comments are printed once per invoice, after the invoice totals. Comments that appear on the invoice are set up in the Comment Types function in Service Management > Setup > Configurable Setup.
  • COINS OA consolidates similar comments, so that all Problem Code descriptions are consolidated into a single comment, and all Resolution comments are consolidated into a single comment. Service Reports and Invoices then use the new consolidated comment, allowing applicable data to print at designated locations within the document.

This document discusses the enhancements that have been made in Service Management to allow the consolidation of comments, which then become an invoice’s data. It does not address the consolidation of Additional Work comments.
