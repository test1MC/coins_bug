Summary Screen

COINS OA users had to run reports or in Service Management to obtain information showing costs, billings to date, commitments, and estimated gross margin prior to invoicing. This was not an efficient way to view the information. As a result, the function now provides users with a Summary screen option on the Finance tab.

This document provides you with more details about the Summary screen in Service Management.

Summary Option

A new Summary option has been added to is found on the pull down menu of the Finance tab in , as shown in Figure 1: Summary Option. It allows you to view Summary information for costs, billings to date, commitments, and estimated gross margin prior to invoicing.

Summary Screen

When you select Summary from the Finance pulldown menu, a Summary screen displays, as shown in Figure 2: Summary Screen.

Summary Screen

The following details apply to this Summary screen:

  • Cash Receipt values are not displayed.
  • Invoices can reference multiple .
  • Partial payments are not applied toward specific invoice distribution lines.
  • The link on the Last Invoice Date can be used to display the Invoices browse.
  • The Invoices browse contains Amount Paid and Balance columns (see Figure 3: Invoices Browse), as well as referenced by the Invoice.
