The following new SE parameters have been added to Service Management to allow for text notifications to customers.

Parameter

Description

NOTIFEM1

Email Template for Email Notification when Entered

NOTIFEM2

Email Template for Email Notification when Scheduled

NOTIFEM3

Email Template for Email Notification when Rescheduled

NOTIFEM4

Email Template for Email Notification when Travel Started

NOTIFEM5

Email Template for Email Notification when Started

NOTIFEM6

Email Template for Email Notification when Completed

NOTIFTX1

Message Content for Text Notification when Entered

NOTIFTX2

Message Content for Text Notification when Scheduled

NOTIFTX3

Message Content for Text Notification when Rescheduled

NOTIFTX4

Message Content for Text Notification when Travel Started

NOTIFTX5

Message Content for Text Notification when Started

NOTIFTX6

Message Content for Text Notification when Completed
