Notification Templates

Email notification subject and body is based on Email Templates of SENOTIFY type. To get correct template the system needs to pick template name from corresponding (based on event) NOTIFEM parameter value.

Text notification is stored directly in corresponding NOTIFTX parameter.

If parameter value is empty or the value does not correspond to Email Template, then use default template from the table below.
