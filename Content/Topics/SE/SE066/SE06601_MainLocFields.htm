Main Location Fields

Functionality on the Location Details and Location Selection screens in Service Management can now be company (kco) specific for main Location fields, as described below.

However, the main Locations browse (Figure 1: Locations Browse) does not contain Company Specific data, even though you can use an In Company View filter.

Location Details

Main Tab

The following fields on the Main tab in Service Management > Locations > Location Details are now considered to be global values and therefore were changed to include "All Companies" in their labels, as shown in Figure 2: All Companies Fields.

  • Service Email - All Companies
  • Coverage Zone - All Companies
  • - All Companies
  • Location Comments - All Companies

All Companies Fields

Company Specific Tab

A new Company Specific tab is now available in Service Management > Locations > Location Details, as shown in Figure 3: Company Specific Tab. This tab contains all fields whose values are considered to be company specific.

Wherever a is created in COINS OA, any non-blank values in the Service Email, Coverage Zone, and fields on this tab are used. For any of these fields that are blank, the values in the corresponding All Companies fields on the Main tab are used.

Location Selection Screen

The common fields on the Location Selection screen were previously only based on global values. The following fields on this screen are now considered to be global values and therefore were changed to include "All Companies" in their labels, as shown in Figure 4: Location Selection Screen.

  • Coverage Zone – All Companies
