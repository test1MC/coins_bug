Configuration

A new parameter in Service Management allows for the calculation of the Billable value for covered .

Parameter

Description

BILAGRCH

Calculate Billable Value for Covered

This parameter supports a list of paired values, comma-separated. The value to the left of the pipe(|) is the actual referenced. The value to the right is the billable to substitute. For example: ACR|AEX,APM|AEX (though APM likely will not be listed here).
