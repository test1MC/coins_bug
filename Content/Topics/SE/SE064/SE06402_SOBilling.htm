Billing

In order for a billable value to be calculated for a nonbillable , the must reference the substitute within a billable Group (for example, AEX). The billable Group defines the Billing Method to use for the nonbillable activity. To calculate a billable value, the Bill-To field in the must be set to Customer via Allowable Work.

When Generating Detail Lines from Costs, Detail, and PO Lines, the Apply Billing Rates process is run if a line's Billing Method is NOT nonbillable. Processing a nonbillable line now occurs when the parameter SE/BILAGRCH contains a matching . Processing is as follows:

  • The Line’s is substituted before Apply Billing Rates is run.
  • The substitute , Group, and Billing Method are assigned to the billing line.
  • The Generate From Costs process sets the Billing Method based on the billable values returned. Generate From Detail and other processes must set the following fields on the Detail Line from the substitute Group:
  • Billing Method
  • Cost Multiplier
  • The calculated Subtotal Amount for the line is copied to the Discount Amount and multiplied by -1, resulting in a Total Amount of $0.00 for the line, as shown in Figure 1: Subtotal and Discount Amounts.
