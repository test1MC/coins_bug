Renewal Renumber Toolbox

Menu Option

A new An Renewal Renumber Toolbox function is now found as an option on the System > Toolbox > Service Management tree menu, as shown in Figure 1: Menu Option. This toolbox is a renumber routine that updates existing records, and any user who accesses the function will see pending changes for all .

Menu Option

Browse Screen

When Renewal Renumber is selected from the Toolbox menu, a browse listing is displayed with pending changes for all , as shown in Figure 2: Toolbox Browse Screen. The new New First Renewal column is included on the browse.

Toolbox Browse Screen

Browse Features

  • Process All - updates all tables with a reference to a Renewal value. The offset value is the difference between the New First Renewal and the actual First Renewal.
  • The button bar for the browse contains the following: Open, Concurrent, Multi, and Bulk. You can use any of these buttons to enter the First Renewal.
  • A Process New First Renewal action at the bottom of the screen allows you to process selected records on the browse.
  • An Options menu is found in the top-right corner of the browse. It contains the following two options:
    • Clear All - removes all New First Renewal values.
    • Process All - updates all tables with a reference to a Renewal value. The offset value is the difference between the New First Renewal and the actual First Renewal.
  • Whether processing individual via either the Process New First Renewal action or Process All option, the process runs on the server in the Report Log, and an applicable Toolbox warning may appear on the selection screen.
