Overview

The Scheduled Hours value was automatically set to 2 hours when a did not have Estimated Hours (Estimated Hours were automatically set to 8 hours). The Service Management module needed to provide a default value for the hours in a Group, and this functionality has now been implemented. In other words, the system now defaults the Scheduled Hours value instead of automatically using 2 hours.

This document describes how the Scheduled Hours defaulting works in Service Management.
