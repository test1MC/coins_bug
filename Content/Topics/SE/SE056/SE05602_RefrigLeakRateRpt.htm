Refrigerant Leak Rate Report

The Service Management module now has a Refrigerant Leak Rate Report. If an requires Refrigerant, this report allows you to see any Leak Rate for that Refrigerant that exceeds the EPA threshold (excessive leakage must be reported).

Menu Item

A new Refrigerant Leak Rate Report option has been added to is found on the Service Management > Reports menu, as shown in Figure 1: Report Menu.

Report Menu

Selection Screen

When you select Refrigerant Leak Rate Report from the SE Reports menu, a selection screen appears, as shown in Figure 2: Report Selection Screen. Changes that were made to this screen for v11.08 are:

  • Sort By is no longer available at the top of the screen; as a result, the report now sorts by Location only.
  • New Page Per /Customer is now labelled New Page Per Location.
  • The Selection tab name is now labelled Location Selection.
  • The Refrigerant (All/Added/Recovered) action is no longer available. Instead, the system now determines the appropriate actions when calculating the Leak Rate.
  • Added Refrigerant is included, with the exception of any with a Sub Action of RRTU – Recovered Same Unit.
  • Recovered Refrigerant within the last 365 days is included as additional information.

Report Selection Screen

Field Definitions (Selected Fields)

Leak Rate Threshold %

This new field allows you to print only when the Annualed Leak Rate exceeds the value entered.

From/To Service Date

This date is mandatory. The To Service Date is used to look back 365 days. If there are multiple transaction records within the From/To Service Dates for an /Circuit, only the last transaction displays. The values from all transactions for the last 365 days (based on the calculation To Service Date less 365 days) for the /Circuit are included.

Line Source

Since the All option is no longer available, you now choose Choose whether Refrigerant Action data should come from either Detail (the default) or (Billing Detail Lines).

Report Sample

After you enter information on the report selection screen, the Refrigerant Leak Rate Report can be generated. A sample report is shown in Figure 3: Report Sample.

  • The report does not contain subtotals or report totals. It simply lists each with a reportable Leak Rate. are sorted by Location. The Location Name, Address, and Primary Customer (Number and Name) are part of the sort heading.
  • The report lists the /Circuit with a Material record containing Refrigerant that has been serviced within the From/To Service Dates.
  • The column values are calculated for the /Circuit. The /Circuit appears on the report if either the Annualed Leak Rate or Rolling Leak Rate exceeds the Leak Rate Threshold %.

Report Sample

Column Definitions (Selected Columns)

Pounds Last Added

The value taken from the last transaction in the From/To Service Date for the /Circuit.

Date of Prior Addition

The date of the previous transaction for the /Circuit where the Refrigerant Action is A-Added and the Sub Action is not RRTU – Recovered Same Unit. If this is more than 365 days, the date is not printed.

# Days Between Additions

This value is calculated using Date Last Added – Date of Prior Addition. However, if the Date of Prior Addition is blank, a 0 value is shown.

Pounds Added (Not Recovered From Unit) Last 365 Days

The number of pounds added but not recovered in all prior transactions for the /Circuit within the last 365 days where the Refrigerant Action is A-Added and the Sub Action is not RRTU – Recovered Same Unit.

Pounds Recovered Not Returned to Unit Last 365 Days

The number of pounds recovered but not returned in all prior transactions for the /Circuit within the last 365 days where the Refrigerant Action is R-Recovered and the Sub Action is not RTU – Return To Unit.

Note:

This note is not involved with the Leak Rate calculations. If removed Refrigerant was not returned to the Unit, it may actually affect the amount of added Refrigerant. If virgin/reclaimed Refrigerant was added to replace the recovered Refrigerant, the Leak Rate may be different. You must review the Refrigerant transaction in detail to determine the true Leak Rate for the /Circuit.

Annualed Leak Rate

The rate for the Annualed Leak Rate is calculated as follows (see Section 1, EPA Leak Rate Calculation Requirements for more information about Leak Rate calculations):

(Pounds Last Added / Capacity) * (365 / # of Days Between Additions) * 100

Note: If # Days Between Additions equals 0, 365 is used as the denominator.

Rolling Leak Rate

The rate for the Rolling Leak Rate is calculated as follows (see Section 1, EPA Leak Rate Calculation Requirements for more information about Leak Rate calculations):

(Pounds Added Last 365 Days / # of Days Between Additions) * 100
