Revenue Recognition Methods File

A new Revenue Recognition Methods function is available on the Service Management menu in Setup > Company Setup. When you select this option from the menu, the screen shown in Figure 1: Revenue Recognition Methods Screen appears. It allows you to maintain a company-specific master list of the Revenue Recognition Methods that apply to only.

This function is also where you can select the new C-Cost Multiplier Calculation to use for Revenue while the order is unposted.

Revenue Recognition Methods Screen

Field Definitions

Revenue Recognition Method

The code used to represent the Revenue Recognition Method.

Description

A description of the Revenue Recognition Method.

Calculation

The calculations available for use are: B-Billings, C-Cost Multiplier, D-Detail Lines, P-Percentage of Completion.

[Cost Category Multipliers] , Materials, , Subcontract, Other

When you select the C-Cost Multiplier option, you are allowed to enter information into any of these cost categories.

Dormant

Whether the Revenue Recognition Method is dormant. When dormant, inserting a new record that references the RRM Code is not allowed. Any codes listed in the SE/DORMRRMS parameter are marked as dormant.
