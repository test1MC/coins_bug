Configuration

The SE parameters listed below were added to allow for Service Report Email Subject functionality, as described in this document.

Parameter

Description

EMAILSRI

 

Email Individual Visit Service Report Subject Text - Will accept the following substitution values:

<ServiceOrder>
<VisitScheduleDate>
<VisitDispatchStartDate>
<Location>
<LocationAddress1>
<PO>

EMAILSRC

Email Consolidated Service Report Subject Text - Will accept the following substitution values:

<ServiceOrder>
<Location>
<LocationAddress1>
<PO>
