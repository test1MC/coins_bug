Add Material PO

When you select a from the screen and then select the Add Material PO action (see Figure 1: Add Material Order Action), an Add Material Order for Service - Single Form Entry screen from Procurement is opened, as shown in Figure 4: Add Material Order for Service Screen.

The following information is true for this screen:

  • The field value defaults in.
  • The Allowable Work, Number, and Tax Code fields default in, based on the field value.
  • The Number, Supplier, Description, and Due Date fields are mandatory.
  • The field lookup in the Material table shows specific for the Allowable Work, rather than all available Standard .
