Overview

The Type Renumber Toolbox is now available on the System > Toolbox > Service Management menu. It is a renumber process that can be used by operating companies to update existing records to use a standard list of values. Users who access this function can see pending changes for all Types.

Changes made to accommodate this Toolbox are discussed in this document.

NOTE:
It is recommended that you test the Toolbox changes in a non-Live environment first, as this Toolbox will permanently change the Types.
