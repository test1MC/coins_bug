Type Renumber Toolbox

The Type Renumber Toolbox is a renumber process that can be used to update existing records to use a standard list of values.

Menu Option

A new An Type Renumber Toolbox function is now found as an option on the System > Toolbox > Service Management tree menu, as shown in Figure 1: Menu Option.

Menu Option

Browse Screen

When Type Renumber is selected from the Toolbox menu (Figure 1: Menu Option), the function opens as a browse listing of all Types, as shown in Figure 2: Browse Screen. A New Type column is included on the browse when added from the function’s standard column set.

Browse Screen

Browse Features

  • The button bar for the browse contains the following: Open, Concurrent, Multi, and Bulk. You can use any of these buttons to enter the New  Type.
  • A Process New Type action at the bottom of the screen allows you to process selected records on the browse.
  • An Options menu is found in the top-right corner of the browse. It contains the following two options:

Clear All

Removes all New Type values.

Process All

Updates all tables in all companies with a reference to an Type value.

  • Whether processing individual Types via either the Process New Type action or Process All option, the process runs on the server in the Report Log, and an applicable Toolbox warning may appear on the selection screen.
