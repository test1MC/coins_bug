A new Scheduled Billing Taxable % field is available under Scheduled Billing in > Locations tab > Allowable Work Details > Billing & Sales Tax tab, as shown in Figure 1: Scheduled Billing Taxable % Field. It is only visible and enabled when Scheduled Billing is selected as Revenue Taxable and parameter SE/ABTAXPCT is set to Y.

The value in this field allows for two decimal positions; it enforces the value to be positive and to not exceed 100. (In COINS Ti, this value is a whole number that cannot exceed 100.) The value of 0 in this field is recogned as 100 percent.
