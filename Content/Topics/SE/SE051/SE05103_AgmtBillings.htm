Billings

COINS Ti supports separate Taxable Amount values on a transaction through the use of a separate Taxable Amount field. However, in COINS OA, every transaction is taxable, and the value of the tax is based on the Tax Code referenced. The Generate Scheduled Billings and Generate Time of Service functions now allow you to create two separate Scheduled Billing Detail Lines (see Figure 1: Revenue Tax Defaults): (1) one line holds the taxable portion; and (2) the other line holds the non-taxable portion.

  • The taxable portion references the ’s Revenue Taxable Tax Code and is the first line created.
  • The non-taxable portion references the ’s Revenue Exempt Tax Code. Any rounding result is applied to the Taxable Detail Line. The sum of the two lines must equal the value of the Scheduled Amount.
