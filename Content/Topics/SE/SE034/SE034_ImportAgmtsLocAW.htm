Import Including Locations & Allowable Work

A new An import called Import Including Locations & Allowable Work is available in Service Management to import with Locations, Allowable Work, Renewals, and . This function also supports a Workspace area.

Import Data Menu

A new An Import Including Locations & Allowable Work submenu is available under the Import Data menu in Service Management > Administration, as shown in Figure 1: Import [%=Variables.Se Agreements caps%] Including Locations & Allowable Work Menu. This new submenu allows you to import with its Location, Allowable Work, Renewal, and . It includes two functions:

  • Import - This provides you with an Example CSV file. The columns in the Example CSV file are configurable, so that you can rearrange or remove them as needed. See Section 1.2 Import for more information.
  • Workspace - The Workspace browse has column set capability, so that you can view the elements of data you have imported. See Section 1.3 Workspace for more information.

Import Including Locations & Allowable Work Menu

This import is designed as a multi-step function:

  1. Import the data into a temporary Workspace.
  2. Use the Workspace browse to make corrections to the data. You can also use the Delete button to remove individual rows of data.
  3. Select options from the Options menu to validate the data (Validate All Records), post the data (Post All Records), and clear out all data (Clear Workspace).

Import

The Import screen (Figure 2: Import Screen) allows you to import data for , Locations, Allowable Works, and from a CSV file. This data can then be imported into the Workspace, where it can be edited, validated, and posted.

Also found on this screen is a link to an Example CSV file. The columns in the Example CSV file are configurable, so that you can rearrange or remove them as needed.

Workspace

The Workspace browse has column set capability, so that you can view the elements of data you have imported. Here you can edit, validate, and post the imported data. A sample browse is shown in Figure 3: Workspace Browse.

The Status column is found as a column in the Workspace browse. If a record does not pass validation to qualify as a valid / Location/Allowable Work/Renewal/, the record shows a red circle, as shown in Figure 4: Red Circle Status.

Options Menu

The Options menu at the top of the Workspace browse contains the following options: Validate All Records, Post All Records, and Clear Workspace, as shown in Figure 5: Validate All Records Option. Each option is explained below.

Validate All Records Option

Validate All Records

Use this option to perform validation of the Workspace records.

The Status of a validated record is shown as a green circle with the tooltip appearing as "OK", as shown in Figure 6: Green Circle Status.

Green Circle Status

Post All Records

All records in the Workspace must be valid in order for the Post to process. If at least one record is not valid, an error message is shown (see Figure 7: Validation Error Message) and posting is aborted.

The Post process removes all Workspace records upon successful completion.

Validation Error Message

Clear Workspace

This function lets you clear the Workspace of all data so a new, corrected file may be imported.
