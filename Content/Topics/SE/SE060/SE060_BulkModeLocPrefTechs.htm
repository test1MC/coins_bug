Bulk Mode Location Preferred

A new Bulk Mode Location Preferred function has been added to can be found on the Related Functions menu in Service Management > Locations. It allows you to set up and maintain Preferred information for multiple locations in one place. This function can also serve as a tool to mass add and delete records.

Menu Option

Bulk Mode Location Preferred is now available as an option in Service Management > Locations > Related Functions, as shown in Figure 1: Bulk Mode Location Preferred [%=Variables.Se Technicians caps%] Option. It does not support Bulk Open (update).

Bulk Mode Location Preferred Option

Browse Screen

The browse screen that appears when you select Bulk Mode Location Preferred from the menu is shown in Figure 2: Bulk Mode Location Preferred [%=Variables.Se Technicians caps%] Browse. It contains the standard column set, including Location Name, Location Dormant, and Location Closed columns.

Bulk Mode Location Preferred Browse

Add/Delete Buttons

You can use the Add and Delete buttons on the Bulk Mode Location Preferred browse (see Figure 3: Add/Delete Buttons) to add and delete Preferred .

Sequence Column

The Sequence column (see Figure 4: Sequence Column) on the Bulk Mode Location Preferred browse can be used to sequence Preferred .
