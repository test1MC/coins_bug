Overview

COINS OA users can now include zero-dollar billing lines on an Invoice to show the customer there is no charge for the item stated. Previously, Detail Line processing forced the Ready for Invoicing (RFI) flag to be set as uned (not to show) when a Detail Line had a $0 Subtotal Amount.

Changes that have been made to the Detail Line entry process to handle zero-dollar billing are discussed in detail in this document.

NOTE:
The updated process does not allow for a completely zero-dollar invoice. A valid invoice must contain a non-zero balance and therefore be worthy of posting to either or Status.
