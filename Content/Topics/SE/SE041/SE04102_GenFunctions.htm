Generate Functions

All Generate functions now automatically set the Ready for Invoicing flag to uned when the value of a Detail Line is $0. This section describes the various Generate functions to which this applies.

Detail Line Generation - Details

If you generate lines from Material and lines set up in Details, only the line with a non-zero Subtotal Amount is marked as Ready for Invoicing. Below is a sample setup in which Material line 2 and line 2 will be marked as ready for invoicing.

  • Material line 1 - includes FA-Flat Amount billing method and zero flat amount.
  • Material line 2 - includes FA-Flat Amount billing method and non-zero flat amount.
  • Material line 3 - set to NB-Nonbillable.
  • line 1 - includes non-zero Regular hours, non-zero Charged hours, and Billing Method = HR-Hourly, with no corresponding Billing Rates record (zero amount).
  • line 2 - includes non-zero Regular hours, non-zero Charged hours, and Billing Method = HR-Hourly, with a corresponding Billing Rates record (non-zero amount).

Detail Line Generation - Quantity Installed

If you generate a Detail Line in a Quantity Installed record of the > Finance tab menu > Quantity Installed menu option based on the setup below, the Quantity Installed has a zero Subtotal Amount and the Ready for Invoicing flag is not ed.

  • An I-Unit Price group is set up and contains costs used for the UP Billing Item setup.
  • Cost Codes have been created for the in Utilities > Create Cost Codes.
  • In Status > Input > Enter Quantity Installed, a matches the costing of the 's Allowable Work.

Detail Line Generation - Add-ons

If you generate an Add-on for lines based on the setup outlined below, the Flat Amount Add-on generates with a zero Subtotal Amount and the Ready for Invoicing flag is not ed.

  • Add-on is a Group with Billing Type = FA-Flat Amount and a zero Flat Amount value.
  • There is at least one other Detail Line marked as Ready For Invoicing but not invoiced yet.

Detail Line Generation - Costs

If you generate Detail Lines from Costs in a JNL-GL/JC Journal Batch containing a zero amount transaction and non-zero (positive/negative) hours, the lines generated have a zero Subtotal Amount and the Ready for Invoicing flag is not ed.

Detail Line Generation - Purchase Order

If you generate Detail Lines from a Purchase Order with a zero price/line total, the lines generated have a zero Subtotal Amount and the Ready for Invoicing flag is not ed.

Invoice

If you generate an Invoice from a with at least two Detail Lines (non-zero Subtotal Amount and zero Subtotal Amount), the Detail Line with a zero value is available in the Invoice.
