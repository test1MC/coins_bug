Void Action

When you select a from the SE screen and then select the Void action, a new dialog screen is displayed, as shown in Void [%=Variables.Se Service Order caps%] Screen.

Void Screen

Field Definitions

Void Reason

The void reason code must be entered when parameter VOIDREAS is set to Y. The description of the code is read only.

Comment

Use this field to add extra information manually, or select a standard phrase from the lookup.

RO Fields

The following RO fields are now available for the Browse and column set and export forms:

  • RO_sso_voidcode (Void Reason Code)
  • RO_sso_voidby (Voided By)
  • RO_sso_voiddttz (Voided Date/Time/Time Zone Offset)
  • RO_sso_voidcomment (Void Comment)
