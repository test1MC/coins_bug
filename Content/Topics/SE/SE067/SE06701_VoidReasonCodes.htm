Void Reason Codes

A new Void Reason Codes function is now found as a menu item in Service Management > Setup > Configurable Setup. It allows you to maintain a master list of codes identifying void reasons.

When selected, it displays the screen shown in Void Reason Codes Screen.

Void Reason Codes Screen

Field Definitions

Reason Code

The code used for the void reason.

Reason Description

A brief description of the void reason code.

Dormant

Whether the void reason code is dormant.

Lookup Table

A Void Reason Codes table is now found in Service Management > Company Configuration > Lookup Tables tab, as shown in Lookup Table. If set to Global, the void reason code data can be seen by all companies. If set to Company Specific, the data can only be seen by the company itself.
