Invoice Summary Report

The new Invoice Summary Report lists invoices posted in a range, with the option to print by Invoice Date range. Emphasis is on invoice value, with Gross Margin included for reference. All Cost and Gross Margin figures on the report are To Date values.

filtering is not available for the Invoice Summary Report since the report does not list only a portion of an invoice. If an invoice is selected, all for that invoice are shown.

Please note that each within the Invoice is listed individually on this report. Also, there are no individual Invoice totals.

Column Definitions

Inv Type

The value of the Invoice Type depends on the following:

  • If all invoice lines are Scheduled Billing, the type equals SB.
  • If no invoice lines are Scheduled Billing, the type equals SO.
  • If both apply, the type equals MIX.

Costs TD

Agree Costs TD

These fields are not shown in the sample report.

These fields show posted costs only for the . If the Inv Type = SB and there are no costs, the two rows do not display. Unless it is a Time-of-Service Billing (Invoice Type = MIX), Scheduled Billing do not have any costs.

Commitments

This field is not shown in the sample report.

Commitments are not shown if the total value of corresponding commitments is zero.

Commitments

This field is not shown in the sample report.

Commitments are not shown if Show Covered Costs was not ed on the selection screen. They are also not shown if the total value of corresponding commitments is zero.

Allocations

When there are Allocations, they are shown on the report as a sixth J/S Cost Category column; otherwise, the sixth category column is not shown.

The sample report above did not have Allocations and therefore does not show this column.

Other Billing Types

Quoted Price, Add-ons, etc. may appear in this column when applicable.
