Additional Work Mass Complete Toolbox

A new Additional Work Mass Complete toolbox gives COINS OA users the ability to mass complete Additional Work when they have fallen behind on managing the information submitted by Mobile . This toolbox mass completes the additional work stage at both the level and level.

SY Toolbox Menu

A new toolbox titled Additional Work Mass Complete is available in Service Management under the System > Toolbox menu, as shown in Figure: System Toolbox Menu.

System Toolbox Menu

SE Toolbox Selection Screen

When you select Additional Work Mass Complete from the SY Toolbox menu, a selection screen displays, as shown in Figure: [%=Variables.Se Visit caps%] Additional Work Mass Complete Selection Screen. Note that the selection header contains a warning in red text stating that you should have a current, verified data backup before executing the Toolbox utility.

Note: The parent does not need to be open when you run this Toolbox utility.

Additional Work Mass Complete Selection Screen

Field Definitions

Mode

Select whether the system should run in Validate or Complete mode. If run in Validate mode, no changes will be made.

Select the or range of to be included in the utility run.

Schedule Date

You must select a range of Schedule Dates, which helps prevent the Toolbox utility from running on all .

Status

Select one of the following statuses to be run in the utility: All, Scheduled, Dispatched, or Completed.
