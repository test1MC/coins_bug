PR Transfer SE Details

A new Distribute Daily Hours box is found in the Transfer SE Visit Details function in Payroll > [selected period] > Input, as shown in Figure 3: PR Transfer SE Visit Details Screen. It only appears on this screen when the Default Daily Hours field is not ed in PR Company Configuration (see Section 1.1).

PR Transfer SE Details Screen

When Distribute Daily Hours is ed, the Reg/OT/DT Day Hours of the created timecards are populated during the transfer process using the date of the SE  Report, regardless of the Default Daily Hours field of PR Company Configuration.
