Screen and Processing Changes

PR Company Configuration

If the Default Daily Hours box on the Timesheet Setup layout of the Other tab of Payroll > Company Setup > Company Configuration is uned (as shown in Figure 1), an additional Distribute Daily Hours field is present when running the following functions: PR Transfer Daily Reports, PR Transfer SE Details, and SE PR Transfer Check.

This field allows you to indicate that the Daily Hours values should be populated during the transfer process by using the date of either the or the Daily Report.

PR Company Configuration Screen

PR Transfer Daily Reports

A new Distribute Daily Hours box has been added to the Transfer Daily Reports function in Payroll > [selected period] > Input, as shown in Figure 1. It only appears on this screen when the Default Daily Hours field is not ed in PR Company Configuration (see Section 1.1).

PR Transfer Daily Reports Screen

When Distribute Daily Hours is ed, the Reg/OT/DT Day Hours of the created timecards are populated during the transfer process using the date of the Daily Report, regardless of the Default Daily Hours field of PR Company Configuration.
