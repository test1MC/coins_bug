PR Company Configuration

If the Default Daily Hours box on the Timesheet Setup layout of the Other tab of Payroll > Company Setup > Company Configuration is uned (as shown in Figure 1: PR Company Configuration Screen), an additional Distribute Daily Hours field is present when running the following functions: PR Transfer Daily Reports, PR Transfer SE Details, and SE PR Transfer Check.

This field allows you to indicate that the Daily Hours values should be populated during the transfer process by using the date of either the or the Daily Report.
