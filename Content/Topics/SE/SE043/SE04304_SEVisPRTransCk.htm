SE PR Transfer Check

A new Distribute Daily Hours box has been added to the PR Transfer Check function in Service Management > , as shown in Figure 4: SE Visit PR Transfer Check Screen. It only appears on this screen when the Default Daily Hours field is not ed in PR Company Configuration (see Section 1.1).

SE PR Transfer Check Screen

When Distribute Daily Hours is ed, the Reg/OT/DT Day Hours of the created timecards are populated during the transfer process using the date of the SE  Report, regardless of the Default Daily Hours field of PR Company Configuration.
