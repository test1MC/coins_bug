Select Multiple to Renew

To make the process of selecting multiple more efficient, a new Select Multiple to Renew function has been added to is found in Service Management > Renew .

Selection Screen

The Select Multiple to Renew function is a process/report that provides you with the ability to both select and deselect for validation and reporting purposes. When selected, the selection screen appears, as shown in Figure 1: Select Multiple [%=Variables.Se Agreements caps%] to Renew.

The From/To Renewal End Date fields are mandatory and determine the range of end dates for renewal.
