Amount Schedule

Menu Option

A new An Amount Schedule function is available in Service Management > > Operational Setup tab, as shown in Figure 1: Menu Option. It can be used to record the future renewal Amounts for with a Term that extends beyond the chosen default Renewal Duration (typically 12 months).

Menu Option

Browse Screen

When you select Amount Schedule from the menu, a browse screen appears, as shown in Figure 2: Browse Screen. This browse is only visible when the following conditions are true:

  • One or more Term records are on file.
  • The last Renewal’s End Date is before the last Term’s End Date, and users choose not to use a default Renewal Duration that is less than the Term Duration.
  • One or more records are on file.

Browse Screen

Field Definitions (Selected Fields)

Start Date

This can be a date within any of the Term records on file. When selecting an for renewal, a record is only used if the Amount Schedule Start Date matches the Start Date of the new renewal period.

Amount

The Schedule’s Amount is used when creating the Renewal’s Amount record.
