Select to Renew

This section discusses the Select to Renew function found in Service Management > Renew . By introducing the ability to concurrently update Agreements in the Select for Renewal browse, you have one place to quickly review and adjust the elements of data that are used in the renewal process.

Browse Screen

The Current Term Duration, Current Term Amount, Current Term Billing PO, and Current Term PO fields are available on the Select to Renew browse (Figure 1: Select [%=Variables.Se Agreements caps%] to Renew Fields) if they were added to the column set of the main browse (Figure 4: [%=Variables.Se Agreements caps%] Column Set). These fields and the other fields allow you to have access to data that can assist you during the renewal process.

NOTE:
The Number of Statuses field is now located between Type and Current Renewal End Date.

Amount - Pending Renewal Field

Many times, you may only need to update the Amount upon selecting an for Renewal. To streamline the renewal process, the new Amount – Pending Renewal value is enabled in the Select for Renewal browse when the following conditions are met:

  • The has been selected.
  • The references only one Location/Service Department.

If the Amount – Pending Renewal field is updated on the browse, the changes are added to one Amount record and the Booking Amount is automatically adjusted.

When new Amount records are generated, they are Location/Service Department specific.
