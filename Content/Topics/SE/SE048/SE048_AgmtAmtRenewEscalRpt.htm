Amount Renewal Escalation Report

A new An Amount Renewal Escalation Report is available in Service Management. It shows the increase/decrease in Amounts between renewals.

Menu Item

A new An Amount Renewal Escalation Report option is available in the Service Management > Reports menu, as shown in Figure 1: Reports Menu.

Reports Menu

Selection Screen

When you select Amount Renewal Escalation from the SE Reports menu, a selection screen appears. This screen contains three tabs and the standard Output Options tab, as shown in Figure 2: Report Selection Screen: Main, Selection, and Location Selection. Only the Main screen is shown here since the and Location selection tabs are standard.

Report Sample

After you enter the information in the selection screen, an Amount Renewal Escalation Report is generated. The report data is listed by /Location/Service Department. The report lists with a Renewal Start Date that falls between the From/To Renewal Date range. A sample of the generated report is shown in Figure 3: Report Sample.
