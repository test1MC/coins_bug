At the bottom of the Detail Lines - All browse, you can use the Billing Complete button to mark a and all of its as Billing Complete. This button should be used to update data when either an event is not triggered by Post Invoice or the data could not be set to DONE at the time of Post Invoice.

A new Ready for Invoice button displays to the left of the Billing Complete button (see Figure 1: Ready for Invoice Button) when a Ready for Invoice Stage is both on file for the and is not set to DONE. When clicked, the Ready for Invoice Stage is set to DONE.
