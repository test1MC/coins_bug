Discovery BI0002.6 - Review: Writing a Standard Page

 

In this discovery, we go over creating a basic Contract Browse screen (as we did in BI0002.1). Once created, Discoveries BI0002.7 onwards will introduced additional browse components.

A Contract Browse Screen

The first exercise is to create a simple contract enquiry screen. The screen will select contracts and return basic information from the jc_job record.

Add the Fields

On the Fields Tab go to the FORM selector at the bottom of the page, select Body and click Apply Filter.

You should always do this to ensure you are adding fields to the correct form – adding fields to an incorrect form will mean that they do not display.

Linking a Field to another function

A field on an enquiry can be linked this to another function – for example to dig deeper into further information. This can be an existing standard enquiry or another user defined enquiry. Use the Function option in Field Maintenance to do this.

To Link the job_num field to the contract record:

Edit the job_num field by clicking on the Open button On the Function field enter %WJC5014SJOB (this is a standard coins enquiry).

Adding a Sort to a Column

A column header can be used to sort the data. To add a sort enter the field name followed by either a + for ascending or – for descending.  Add a sort to the column for job_num. (kco+,job_num)

This will also provide a simple search filter to the Enquiry screen.
