Testing a Dataset

It is possible to use a dataset in the Query Editor to check the data being returned.

For our dataset created in Discovery BI0009, open up query editor and enter the following query:

FOR EACH ttcontract

 

Dataset tables are known as Temporary tables (as they are only exist at the time of the query).  For this reason, any Dataset table must be prefixed by tt when used in a query.

 

Enter the name of the data set in the Data Set box – for our example we enter xxM2DS1 (replace xx with your initials). 

Enter the name of the data set in the Data Set box – for our example we enter xxBI0009 (replace xx with your initials). 

Set the number of rows to 10 and click Run.
