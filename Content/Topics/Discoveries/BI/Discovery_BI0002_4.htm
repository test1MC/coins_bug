Discovery BI0002.4 -Adding Filters to the Contract Browse Screen

 

This exercise is to modify the contract browse screen to include advanced filters.

 

Adding a Page Browse Filter

Edit the page section by clicking on the Open button and save or follow the link on the Page Name.

Select the Browse Filters Tab.

Click to Add fields to the filter as follows:

Field Label Types
job_num Contract GE,LE,MA
jcl_loc Location GE,LE,MA
jgr_group Group GE,LE,MA
jty_type Type GE,LE,MA

 

Click after adding each field.
