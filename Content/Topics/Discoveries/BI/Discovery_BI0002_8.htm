Discovery BI0002.8 - Summary Pages

 

In this discovery we will be adding a summary screen to be used from xxM3Exercise2 and which will contain the details from the page it was called from.

 

 

Create the Function

Create a new function. (which will be the page linked to from Exercise 1 instead of the standard link).

 

Function Code Initials + Exercise No   Eg    NLM3EXercise3
Function Name Description identifier NL M3 Exercise3
Function Type Function
Function Module Contract Status
Function Category Browse
Function Context Leave as defaulted from the Function Description.
Function Program wou005

 

 

Create the Page

Create the page section (with the same name as the function).

Enter the information into the Page Summary as follows:

 

Header Table jc_job,jc_config,co_config
Body Table jc_job
Summary (On Header Tab)

This page will be used as a container of the costcode page. Note that the Header tab must contain any tables use in the calling page and the body table will be that of the summary. (In this example that would be 'Contract' table.

Click Save to complete the Page Summary.

 

Create a Detail Page Form

Select the Form tab and create a Detail form.

Click the Add Button and enter the following information to create the Form :

 

Form Detail
Field Type Detail

Click the Save Button to create the form.

Add the Fields

In the Fields Tab go to the FORM selector at the bottom of the page, select Detail and click Apply Filter.

NOTE: You should always do this to ensure you are adding fields to the correct form – adding fields to an incorrect form will mean that they do not display.

Add the following Fields :

Function View-As
xxM3Exercise2 Inline Frame

 

 

Please note that the Function Name used here will be the name of the function created in exercise 2.

Click Save   after entering each field.

Add the in-line frame to a tab.

Go to Function Maintenance and copy function used in exercise 2 and add a T to the end.  Change the context of this new function to read   -   COSTS

Now add the tab name to the end of your in-line frame on the page of your current exercise (Exercise 3).

 

Add new detail page to the Original Page.

Instead of calling the standard link on your original page, we can now call our bespoke page.  Go back to the page created in Discovery BI0002.6 (xxM3Exercise1) and change the function to be that of the function created in this exercise.
