Search Field and Button

Type a search query. Then click the button or press Enter to initiate a search.

Case-insensitive Matches Search is not case-sensitive. For example, a search for the word "run" will find matches for "Run" and "run."

Matches with Variant Endings For example, a search for the word "run" will also find matches for words such as, "runner," "running," and "runs." Since matches are not case-sensitive, the results will include topics containing matches such as, "Runner," "Running," and "Runs."

Phrase Search Search for phrases by enclosing the search terms in quotation marks. This is useful when you want to restrict a search to locate terms that appear in an exact order. For example, you might want to search for a phrase such as "Smart Tiles"

Asterisk (*) wildcard and boolean search capabilities are available.
Filters List

Provides the ability to narrow the scope of the search results to categories defined by a search filter set. For example, you can search for results in parameters only.

 
Search Results

Lists search results in ranked order.

Search Highlighting

Highlights matching keywords in topics.

Matches are not highlighted when users perform searches with the asterisk (*) wildcard character.
Remove Highlighting Button

Removes search highlighting from topics.
Boolean Searches

AND

Use to narrow the search and retrieve only the topics that contain all of the words it separates.

Type AND between two or more terms. This is not case sensitive.

When separating terms with a blank space, the AND is always inferred. The plus symbol + and the ampersand symbol & can be used instead of the word AND

OR

Use to search for two or more words. This broadens the search results by retrieving topics that contain any of the words it separates.

Type or between two or more terms. This operator is not case sensitive.

NOT (^)

Use to search only for matching topics that do not contain the term following the operator.

Type a term, followed by the carat symbol (^), and then type another term. For example :

Desktop ^Tiles will retrieve all desktop topics, but not the ones that refer to tiles.
