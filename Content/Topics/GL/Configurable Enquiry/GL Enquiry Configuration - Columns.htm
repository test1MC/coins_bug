GL Configuration - Columns

Fields for the configuration of columns have been added or updated.

Fields

Field
Description

Percentage

A new box to control whether the column displays a percentage value. If this is ed, the value in each line is displayed as a percentage of the column total; “%” is added to the column label. This allows you to show a currency amount in one column and a percentage in another.

Print Column

A new box to control whether the column is displayed on the generated (this is ed by default). If the column is not displayed, it can be used in calculations (for example a quarterly with three columns with offsets which are not shown, then a calculation column adding them together).

Budget

This has been changed from a drop-down list to a lookup field, and allows tokens to be entered.

Company Group

This is a new lookup field for the Opening Balance, Closing Balance, Movement, Actual, Budget and Variance column types. The totals the values from all companies in the group (which can be configured to link to multiple companies or a single company).

You can enter an token to allow the group to be chosen when the is run.

Calculation Add Type

A new option, Calculation, has been added to the Add Type selector for columns. A calculation column allows you to perform calculations based on other columns. Enter the calculations as the column numbers and the operations (+, -, *, /) to perform. For example: 1+2+3 to add three columns together or 1-2 to show the difference between two columns.
