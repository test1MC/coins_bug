GL Configuration - Rows

Fields have been added to the configuration of rows.

Fields

Field
Description

Row Desc Alignment

The alignment of the row description. This can be one of:

  • Default
  • Left
  • Right

Print Line

Whether the line is displayed on the generated (this is ed by default). If the line is not displayed, it can be used in percentage calculations (for example to add a line to represent total sales, and then to use the line as the denominator of percentage for other lines).

Variant

A drop-down selector with the options blank, Total or Subtotal.

  • If Subtotal is selected, the row is displayed on the as a subtotal line with a double underline.
  • If Total is selected, the row is displayed as a total line with a bold underline.
