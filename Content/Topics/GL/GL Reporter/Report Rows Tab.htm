Report Rows Tab

The Report Rows tab is available to add data and content to a report.

The Report Rows list all the data that you want to use in a report. This includes account numbers, unique codes and descriptions.

The report rows tab shows a list of information that will be used for the report. Each row will allow you to define where particular selections of accounts will be shown in the report and where heading, subtotal and total lines will show.

There are 7 different types of rows that can be used when adding data lines to the report.

Allows you to configure lines of text that you can use in different places on the report - for example, underlines to use as formatting.

Allows you to specify a paragraph of additional text to include in the report.

X

Page Break

Allows you to force a page break. For example, if the report is in two sections you might insert a page break to make sure that the second section starts on a new page.

 

Blank Line

Allows you to insert blank lines within the report.

B

Contract Line

Allows you to specify groups of contracts to report on. Selected information from the contract record is available.

J, H, F

Each row type has a defined role within the report and some will affect the data that is displayed and others will perform an action.

The "Show On" code is used to link rows and columns so that on the columns you can specify which row types the column is shown on. For example, to show the account balance on account rows and total rows, you would include ‘D.’ and ‘T.’ in the ‘Show On’ field for the account balance. H and F refer to the sort headers and footers for account and contract rows. The Show On field determines which line types will show in which defined columns.

The codes are:

Code ‘Show On’
B Blank Rows
Dn Account (Detail) Rows
Fn Sort footers
Hn Sort headers
Jn Contract (Job Costing) rows
Sn Standard text rows
X Additional text rows

Where n is a number that represents a level. To use the same row type for different things, for example, if you want one set of account lines to show closing balance and movement, and a second set to show budget and forecast, give the two row types different levels, and put the relevant level number after the "show on" code for each column
