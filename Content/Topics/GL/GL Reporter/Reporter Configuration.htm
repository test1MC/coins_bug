Reporter Configuration

Report Configuration will enable you to configure and create new General Ledger reports. Reports may also be reviewed, amended and updated, however it is recommended that you copy the standard or original report first before changing. Reports set up in one company will need to be exported and imported into other companies.

  1. From the side menu, select General Ledger > Setup Maintenance > Reports Setup Maintenance > Define GL Reports

    The Reporter Configuration browse screen will be displayed showing all the reports that have been configured.
