Formatting

Text styles

You can choose the text style (for example: typeface, , border and size) for the whole report using the Class field on the Configuration tab. You can also apply different styles to lines or columns on the report using the Class field on the row or column. Classes are set up using Report Class in the System module.

You can also use additional formatting on individual rows or columns.

Character formatting

You can apply standard Progress formatting to a column on the report to control, for example, the width of the column or how numbers are displayed. Enter the format in the Format field on the column.
