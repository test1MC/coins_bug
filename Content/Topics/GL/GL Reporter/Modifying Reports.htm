Modifying Reports

Change Report Rows

Existing report rows may be changed by amending, deleting or copying rows. 

Amend Report Rows

To amend a Report Row select the line to be amended by using the Open button. The line is then available for amendment.

Delete Report Rows

  1. To delete a Report Row or multiple rows, select the line(s) to be deleted.

  2. Click the Delete button at the bottom of the browse screen to delete the line. 

    A message will warn that deleting the line will be permanent.

    If multiple lines are selected then the number of rows to be deleted will be shown in the message.

  3. Click OK to continue and delete the line.

    The line(s) will be removed from the Report Rows.

Copy Report Rows

  1. Select the line to be copied by ticking the box of the required line.

  2. Click the Copy button at the bottom of the browse screen

    The line selected will be copied; you can then modify the new line.

Move Report Rows

  1. Select the line(s) to be moved by ing the box of the required line(s). It is possible to move a group of lines either by clicking individual boxes for lines or using ‘Shift+Click’ for a range of lines or ‘Ctrl+Click’ to add lines to already selected lines.

  2. Click the Move Up or Move Down buttons at the bottom of the browse screen.

    The selected lines will either be moved up or down one row at a time.
