Change Report Columns

Existing report columns may be changed by amending, deleting or copying columns.

Amend Report Columns

To amend a Report Column select the line to be amended by using the Open button. The line is then available for amendment.

Delete Report Columns

  1. To delete a Report Column select the line to be amended. 
  2. Click the Delete button at the bottom of the browse screen to delete the line. 

    A message will warn that deleting the line will be permanent.

  3. Click OK to continue and delete the line.

    The line will be removed the Report Columns.

Copy Report Columns

  1. Select the line to be copied.

  2. Click the Copy button at the bottom of the browse screen.

    The line selected will be copied and the line may be modified.
