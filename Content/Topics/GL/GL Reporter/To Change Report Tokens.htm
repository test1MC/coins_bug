To Change Report Tokens

Report Tokens may be changed by amending, deleting or copying columns.

Amend Report Tokens

To amend a Report Token click the Open button on the line to be amended. The line is then available for amendment.

Delete Report Tokens

  1. To delete a Report Token select the line to be amended. 

  2. Click the Delete button at the bottom of the browse screen to delete the line. 

    A message will warn that deleting the line will be permanent

  3. Click OK to continue and delete the line.

    The line will be removed from the Report Tokens.

Copy Report Tokens

  1. Select the line to be copied.

  2. Click the Copy button at the bottom of the browse screen

    The line selected will be copied and you can modify the new line.
