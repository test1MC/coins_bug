Underlining

To add underlines into a report to break up the printout when showing totals and subtotals a row or single ‘-‘ or double ‘=’ lines can be printed as a separate row on the report.

To achieve this effect, add the standard text lines into the report as a row, where you wish them to appear. Standard text lines allow you to insert underlining etc. into the report. In order to do this, on the report rows tab you can punctuate the report with Standard Text lines wherever you wish to underline.

When you then go to the report columns tab, you then input what you would like to be displayed on a standard text line. will then display these items at every occurrence of a Standard Text Line on the report rows.

Note that you can use differing effects with Standard Text Lines when you use different ‘Levels’ in the report.  i.e. in the example above, you could put single underlines on level 1 (and therefore ‘show on’ s1) and double underlines on level 2 (and therefore ‘show on’ s2).
