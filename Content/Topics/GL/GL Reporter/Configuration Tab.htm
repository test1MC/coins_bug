Configuration Tab

The Configuration tab creates the layout of the report and includes information relating to the page layout, font size, and page length.

  1. Enter the details of the page size and layout of the report and the fonts required in the relevant fields.

    The fields are defined in the table below.

    Field:

    Description:

    Divide by:

    Determines how the balances are displayed in the report. The General Ledger balances displayed will be divided by this number which is usually blank or zero to display the balances with no division or to use 1,000 or 1,000,000.

    Round:

    Determines how the rounding of balances will be displayed. Blank or zero values do not round.

    Page Layout:

    The page layout for the report e.g. A4 Portrait or Landscape

    Class:

    The font name and size for the report e.g. Arial, Bold, 10pt, 12pt

    Variant:

    A specific variant to use on the line e.g. Header or Footer

    Extra:

    The format and font for the variant

    Column Ratio:

    The column width for the font i.e. points.

    Page Length:

    The number of lines for the report page.

    You have entered all the details for the page layout and the fonts required. You can change this at any stage of the report, however by changing the page orientation or the fonts will impact the data displayed.

  2. Select the Save button.

    The report layout and title have been added to the Reporter Configuration browse screen.

    The Report Configuration screen is displayed with additional tabs for you to build the report. This report is now available in the Report Configuration Browse screen however there is no data on the report as yet.

    Additional tabs are now available to add data and define how to display the data on the page.
