Report Columns

The Report Columns tab will enable the data to be displayed in the report. This takes the data in the Report Rows tab and displays it using the Report Columns tab. Each report row can have its own set of columns, or share a common set of columns.

There are many different Column Type to choose, however these are group into 5 main groups.

Column Type:

Code

Description:

Character:

CH

Columns used for data that is Alpha, Alpha Numeric or Account Numbers. e.g. Account Full Description

Literal

LI

This will display exactly what has been entered into the Column Description Field.  e.g. Trading Account

Decimal

DE

Columns that are using figures, or values that are decimals. These can be used as calculations.

Integer

IN

The data is numerical and is not decimal, i.e. period numbers

Date

DA

Columns where the data displayed are dates.

You must select the right Column Type to ensure that the data and formatting is displayed correctly.

Column headings are entered by using a literal column type and positioning the header using a negative number for the row value. To make it easier to review the report these columns should be defined first.
