Adding a Character Type

There are two types of Character types to the Report. The first will enable you to select information from the General Ledger record. The second is to show the exact details you have entered on the Description field in the Report Rows tab.

Adding Account Details

This will be a Character Column type.

  1. Select the Account Number option from the dropdown list and click the Add button.

    The Line will be available for you to enter the details for the display.

  2. Enter the details into the relevant fields.

    Field:

    Description:

    Label:

    The label to identify the Data

    Format:

    The format you want the data to be displayed

    Show On:

    Relates to the Level of the Data Rows and also the type of Column. This can be a comma separated list if more than one value is required. The

    Row:

    The number of rows down on the page to start the data

    Column:

    The number of columns across the page to start the data. The starting position is (approximately) the number of characters across the page. Note that if the format of a column is wider than the difference between the two Col fields, the following columns will not be aligned.

  3. Enter the format that we are using for the data. The Account Number is X(9), which means it will display 9 digits.
  4. Enter the Show On, which in our example id D1.

    The Show On field determines which line types the column will be printed on. Use a comma-separated list of one or more from the following:

    B

    Blank Rows

    Dn

    Account Rows

    Fn

    Sort footers

    Hn

    Sort headers

    Jn

    Contract rows

    Sn

    Standard text rows

    X

    Additional text rows

    Where n is a number that represents a level. For account and contract rows, you can think of levels as variations of the same type of line on which some of the columns may be different. For example, if you want one set of account lines to show closing balance and movement, and a second set to show budget and forecast, give the two row types different levels, and put the relevant level number after the "show on" code for each column. For header and footer lines, the levels correspond to the sort levels as defined on the row. A dot instead of a number means "show on all levels for this type of line".

  5. Click the Save button.

    The report can be run and show the details on the report.
