GL Reporter – Adding a Line

A Line report row will display balances on the report, based on the selections entered in the line. This allows information to be displayed on a GL report.

  1. Select Line as the row type option from the type field drop down list and click .

    A new line is created for the Line in the report.

    The Detail and Print fields will automatically default in order to print the Line on the report.

  2. Enter the description for the Line e.g. Africa Contracts.

  3. Enter the Line detail in the relevant fields. The fields are defined in the table below.

    Field

    Description

    Level

    lines at the same level will be displayed together in a column on the report. 

    Contract, Group, Type, Location, Manager, Client, Status of Contract

    Use these fields to select which to include.

    Active

    Select whether to include only active , only inactive , or both.

    Complete

    Select whether to include only complete , only incomplete , or both.

    Include Future Contracts

    the box to include future .

    Company Group

    The Company Group to report on. 

    Currency Type

    The base currency for conversion if you are using more than one currency.

    Percent Line

    The line number to reflect as a percentage of another row; for example, Profit as a percentage of Total Sales. 

    Divide Row

    The line number to divide a row by another row. 

    Sort

    The order to sort the report.  Max 5 levels e.g. 1 = Account, 2= Analysis Code, 3 = Branch, 4 = Dept, 5 = Detail. 

    Header/Footer Level

    The level corresponds to a set of header and footer column fields configured in Report Columns.

    Print Line

    the box to print the line on the report or un to just process to accumulate report values.

    Print Zero:

    the box to print the line on the report even if it contains zero balances.

    Reverse Value

    the box to display negative values as positive and vice versa.
