GL Reporter – Adding a Blank Line

You can add blank lines within the report to enable the data to be easily viewed and read. You may also want to have a blank line to space out the data and information from the totals.

When adding Blank Lines you may wish to use , rather than . This will enable you to design the report first and add the blank lines where appropriate.

  1. Select Blank Line as the row type option from the type field drop down list. the row immediately below where you require the Blank Line if you wish to insert a new blank line.

  2. Click to add to the end of the report or click if you wish to insert above the currently selected line.

    A new line is created for the Blank Line in the report.

    Leave the description field blank for a Blank Line to be printed on the report.

    The Detail and Print fields will be selected automatically in order to print the blank line on the report.

  3. The report will show the additional Blank Line before the total.
