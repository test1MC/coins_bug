GL Journal Transactions

On the Transactions Tab of the Journal Batch, select the Add button. Enter the details of the transaction in the relevant fields.

Field

Description

Type

The type of account (contract, General Ledger, asset, workshop, VAT or part exchange account) to which you are entering the journal.

Unlabelled Field

If the transaction is to an account other than a General Ledger account, this shows:

Rev if the transaction is a revenue journal.

Input if the transaction is input VAT.

Output if the transaction is output VAT.

Cost if the transaction is a cost journal
Analysis

Financial analysis where you want to record the value of the transaction

GL Account Lookup

GSTable Amount excluding GST

VATable

The amount excluding VAT (for VAT journals).

The VATable amount is shown with the natural sign; VAT amounts (in the Value column) are displayed with the sign which depends on whether the entry is a debit or a credit.

For example, for a VATable input with a positive figure in the VATable column, the VAT amount is displayed as a positive figure in the Value column since this will be a debit entry in the journal.

For a VATable output with a positive figure in the VATable column, the VAT amount is displayed as a negative figure in the Value column since this will be a credit entry in the journal.

A negative VATable output figure is interpreted as reducing the VATable outputs which requires a debit entry in the journal, so this is displayed as positive in the Value column.
Value

Value of the transaction.

If the Rate field on the batch header is not zero, calculates the value of the transaction from the quantity, when you complete the line.

To enter alternative base currency values manually, leave the value in this field as 0.00.
Currency Currency code of the transaction
Description Description will default to either the batch header or the previous transaction or you can enter a specific description per transaction line.
Date Date of the transaction
Quantity The quantity for the transaction.
Units The unit of quantity.
