Entering values for the non-fixed journals

Click the hyper-link on the non-fixed journal to amend.

Click Open against the line to be amended.

The user can update this line to a different value from the configured standing journal value for this posting only. This will be reset to the original value after the next posting.

When you have entered the values, click to save the line. Changes to the configured standing journal are indicated by the values having a red background.

When you have finished amending values, click .

You will be returned to the Standing Journals Browse. The Ready to Post field will be Amber to indicate that changes have been made to the Journal for the next posting.
