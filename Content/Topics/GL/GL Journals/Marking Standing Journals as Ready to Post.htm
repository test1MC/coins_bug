Marking Standing Journals as Ready to Post

You will only be able to set the standing journal as ready to post if there are no errors associated with it.

If errors are present the field will have a red background.

You will be prevented from setting this flag if:

  • There are no detail lines
  • The batch isn't balanced
  • The next period date is outside the active period

If there is a warning associated with the standing journal then the field background will be amber.

The reason for this would be one or more of:

  • There are detail lines but they are all zero.
  • The standard journal batch has been amended with values and/or new lines for the current posting only.
  • The next period is not the current GL period.

The reason for the error or warning is shown on the body detail.

To mark selected journals as ready to post, select the line/s and Use the Choose Action selector.

The actions you can apply to these records include:

Ready to Post lets you indicate that a standing journal is ready to post.
Not Ready to Post lets you indicate that a standing journal is not ready to post.
