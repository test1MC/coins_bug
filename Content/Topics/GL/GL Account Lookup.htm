GL Account Lookup

Depending on how has been configured, when you use lookup on an account field, the lookup can be by element or by full account code. If the lookup is by element, you can use to build up an account code from the individual elements. Each time you click , the lookup shows the available codes for the next blank element. You can select one to use as your input to the field.

For example, if your account structure is Division.Account:

On the Account field click

displays a lookup of all the division codes.

Click against the code you want to select.

displays a lookup of all the account codes.

This is the last level, so click

inserts the completed account code into the field.
