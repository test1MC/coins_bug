Budget Patterns

Budget Pattern Maintenance allows you to set up budget patterns which act as templates defining how the total budget for an account is applied across the financial periods of a year. Once you assign a budget pattern to an account, you can change the value for any financial period.

Navigate to General Ledger > Budgets > Budgets Setup Maintenance > Budget Patterns

To add a new pattern Click .

Description
Code The code for the budget pattern.
Description The description for the budget pattern.
Period 1 For each of the financial periods, enter a value to represent what proportion of the budget is assigned to that period. calculates the individual budget figures by dividing the annual budget total by the total in the budget pattern, and multiplying the result by the value in this field.
Total The total of the budget proportions.
