GL Account Forecast Browse Actions

The following actions are available on the Rolling Forecast and future year Forecast browses.

  • Empty Forecast resets the forecast values to zero on each selected account.
  • Copy Previous Forecast copies the forecast (future period) values for each selected account from the any previous period forecast to the current forecast.
  • Copy Budget copies the values for each selected account from the budget defined in GL/FBUDTYPE a selected budget to the current forecast. The budget defined in GL/FBUDTYPE is the default.
  • Apply Multiplier prompts for a decimal multiplier, and for each selected account, multiplies the forecast amounts by that multiplier.
  • For a single account, Apply Value prompts for a new forecast value (or a new remaining value on the Rolling Forecast screen) and a spread method, and updates the forecast spread values accordingly.
