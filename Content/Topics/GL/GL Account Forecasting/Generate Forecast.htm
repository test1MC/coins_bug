Generate Forecast

Generate Forecast creates a new General Ledger forecast. Generate Forecast is available on the General Ledger Forecasts menu. It is also available on the Option menu of Forecast Workbench.

Fields

Field
Description

Period

The General Ledger period to create the forecast for. This defaults to the first period (from the current period onward) which has no existing forecast.

Sector

The sectors to create forecasts for. A separate forecast will be created for each sector specified.

These fields are only available if the GL parameter FCBYSECT = Y.

Description

The description of the forecast.

Prepared By

The user who prepared the forecast. This defaults to the ID of the current user.

Copy Previous Period Forecast

Whether to copy forecast data and notes from the previous forecast. This copies any forecast values that are in periods later than the current forecast’s .

Copy Previous Period Documents

Whether to copy any documents attached to the previous forecast. This is only available if Copy Previous Period Forecast is ed.

Notes

Notes that relate to the forecast.
