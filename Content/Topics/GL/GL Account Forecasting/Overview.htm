GL Account Forecasting Phase 2 - Overview

GL Account Forecasting was developed in version 11.08. New features have been added to extend the existing functionality.

  • Save Forecast Structure on forecast header
  • Improve performance for summary level GL forecasting
  • Browse actions for summary level
  • Bulk update functionality at GL account level forecasting
  • Configure column labels in Yearly Forecast browses and Rolling Forecast browse.
  • Ability to copy any GL budget.
  • Ability to copy any previous forecast.
  • Post/Un-post Accrual amount.
  • Links on Summary Level browse.
  • New GL Transactions browse.
  • GL Forecast columns in configurable GL and reports.
