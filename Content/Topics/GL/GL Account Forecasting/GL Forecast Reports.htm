GL Forecast Reports

A new GL Forecast Reports submenu in the General Ledger Reports menu includes the reports below.

GL Rolling Forecast Detail Report

The Rolling Forecast Detail Report provides the ability to review detailed values of actual balances, accruals, budgets, and forecast entered for selected GL accounts in any .

GL Yearly Analysis Report

The Yearly Analysis Report provides output consistent with the GL Forecast input screens, comparing Actual, Budget and Forecast summaries for last year, this year and next year values. It is possible to choose any level of Forecast Structure defined in Company and Account level as a more detailed version of the report output. Field sets allow you to choose between showing budget variance, accruals or forecast movement.

GL Rolling Forecast Report

The Rolling Forecast Report shows monthly forecast values.
