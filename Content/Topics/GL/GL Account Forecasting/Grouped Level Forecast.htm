GL Account Forecast Workbench - Grouped Level Forecast

The Grouped Level Forecast is shown when you follow a link from a forecast on the GL Forecast Workbench, or from a higher-level forecast on the Grouped Level Forecast screen.

Grouped Level Forecast displays the following tabs:

On each of these tabs, a link on the Code column takes you to the same tab on the Grouped Level Forecast for the next level (if there is another level in the forecast structure), filtered by the parent code.

The links on any of the Actual or Forecast columns take you to the same tab on the Account Forecast, filtered by the parent code (see - GL Account Forecast Workbench - Account Forecast).

Budget Variance

For each analysis row, the Budget Variance browse shows the actuals for the previous year; the budget, actual and variance (budget – actual) for the selected period and current year to date; and the budget, forecast and variance for the current year and as many future years as are defined by GL/FCNUMYRS.

Accrual Browse

The Accrual browse is shown if GL/FCACC = Y.

For each analysis row, the Accrual browse shows the actuals for the previous year; the actual, accrual and this period values for the selected period; the actual, accrual and year to date values for the current year; and the budget, forecast and variance for the current year and as many future years as are defined by GL/FCNUMYRS.

Rolling Forecast Browse

The Rolling Forecast browse shows summared actuals for the previous year, actuals or forecast (depending on the forecast period selected) for each period of the current year, and forecast, budget and movement for the whole year. Tooltips on all the period columns show the budget, forecast and variance.

Future Year Forecast Browses

The Future Year Forecast browses show forecasts for the periods of the selected year; the summared forecast for the previous year; and totals for the selected year.
