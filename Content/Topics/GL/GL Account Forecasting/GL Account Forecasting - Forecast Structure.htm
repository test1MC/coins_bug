GL Account Forecasting - Forecast Structure

If GL/USEBPFXT = Y, a new Forecast Structure function is available (on a Forecast Setup submenu in GL Setup Maintenance). Forecast Structure allows you to define up to three levels of GL Account grouping. If defined, levels allow grouped views of GL Accounts in the Forecast Workbench.

Each level is associated with one existing analysis set (account or element analysis), selected from a drop-down list of the account and elements analysis that have been defined.

You can use the From, To and Matches field to limit the accounts that are included in the forecast.
