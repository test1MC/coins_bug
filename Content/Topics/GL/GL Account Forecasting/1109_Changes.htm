Screen and Processing Changes

GL Forecast Workbench

The performance of the workbench has been improved by using generate programs.

If posting of accruals is enabled in the Forecast Workbench (using the GL parameters FCACCR, FCACCRC* and ACCRTYPE), new columns show the number and status of the accrual batch.

Forecast Workbench Actions

The following new actions are available on the Forecast Workbench:

Grouped Level Forecast

Browse Actions

The Empty Forecast, Copy Previous Forecast, and Copy Budget actions are now available at the summary level (the Grouped Level Forecast) as well as on the Account Forecast browse. If used at summary level, a warning is displayed that the forecast values of all General Ledger accounts which use the summary level code will be updated.

Copy Previous Forecast now allows you to copy from any previous forecast.

Copy Budget now allows you to copy from any budget; the budget defined in GL/FBUDTYPE is the default.

Budget Variance Browse

Accrual Browse

The future year forecast columns have been removed from the browse.

Rolling Forecast Browse

Future Year Forecast Browses

The link on the Forecast Value column for the previous year has been removed.

Account Forecast Workbench

Bulk update has been added to the Account Forecast Workbench.
