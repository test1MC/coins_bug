GL Forecast Workbench

If GL/USEBPFXT = Y and at least one level of forecast structure is defined in the company (see 1 - GL Account Forecasting - Forecast Structure), a new Forecast Workbench menu item is available.

Forecast Workbench allows the input of GL forecasts against (optionally using a sector-specific forecast). Each forecast holds period-specific forecasting data for GL accounts.

A link on the Date column takes you to the first level forecast (see - GL Account Forecast Workbench - Grouped Level Forecast).

An option on the Options menu allows you to generate a new forecast (see 1 - Generate Forecast).

Sector-Specific Forecasts

If sectors are in use in the current , and the new GL parameter FCBYSECT is set to Y, sector-specific forecasts are used. A Sector drop-down selector is available on the forecast screens.

If the GL Security field is set to the Sector element (in GL/ACCTSEC) , GL security applies to the Forecast Workbench browses, in which case:

  • Only forecasts which are in sectors the user has access to are displayed.
  • The Sector selector only shows sectors allowed for the user.
  • Users can only add, change or delete forecasts in sectors to which they have access.
