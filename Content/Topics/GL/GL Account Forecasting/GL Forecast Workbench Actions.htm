GL Forecast Workbench Actions

The following actions are available on the Forecast Workbench.

  • Regenerate Forecast rebuilds the selected forecasts (if they have not been marked as Complete) based on the current GL forecast structure, including any changes in account/element analysis and any accounts added after the forecast was generated.
  • Delete Forecast deletes header, detail and forecast spread data for the selected forecasts (if they have not been marked as Complete).
  • Set Forecast as Complete s the Complete box on the selected forecasts.

  • Set Forecast as Incomplete clears the Complete box on the selected forecasts.

  • Authore Forecast s the Authored box on the selected forecasts (if they are marked as Complete).
  • Unauthore Forecast clears the Authored box on the selected forecasts (if they are already marked as Authored).
