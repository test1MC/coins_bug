Change Asset Economic Life allows you to change the useful economic life (UEL) or a range of assets. You can run the process as report only (no update to the assets) so you can see what changes would be made before committing to making them.
More...

This operation is restricted to straight line deprecation methods; you cannot change from or to other depreciation methods.

You cannot run the process in update mode when there is a depreciation run in progress.

See Changing the Economic Life of Assets.
