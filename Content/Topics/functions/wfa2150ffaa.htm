Depreciation Type Conversion allows you to change from the non-cumulative method of depreciation to the cumulative method (see Depreciation Types).

Do not use this procedure without first consulting Construction Industry Solutions Ltd.

More...

Before the conversion, you can produce a report showing the effect conversion will have. You can then note assets whose value would change significantly (for example, if the purchase transaction was posted a long time after the actual purchase), and change the details for those assets after you have run the conversion.

The report works by calculating depreciation by the old method and by the cumulative method. If you run the report, you must do so before depreciation has been calculated, or after depreciation has been posted. You can use the Depreciation Type Exceptions report (see Depreciation Type Exceptions) to report on assets whose depreciation method has been changed, or for which manual depreciation has been entered (that is, the calculated amount has been overridden).
