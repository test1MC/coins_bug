Asset Groups allows you to maintain your asset groups. Asset groups save you time and effort when entering assets. An asset "inherits" the characteristics of the group it belongs to, so you can use asset groups to set up defaults.
More... You should give some thought to setting up asset groups. For example, it might be more sensible to set up several groups for different types of dump truck, each with different rates, than to have a single group for dump trucks and have to enter new rates for each asset.
