Workshop {} allows you to maintain a list of elements. You can build up workshop codes from a series of elements, allowing you to set up the according to the level of detail you require.
More... For example, a code might be made up of three elements, denoting "part" (brakes, ), "activity" (replace, clean) and "reason" (accident, wear and tear):
  • 1st element (part) BR = brakes
  • 2nd element (activity) CL = clean
  • 3rd element (reason) WE = wear and tear

So the for "cleaning brakes because of wear and tear" would be BRCLWE.
