Depreciation Type Exceptions allows you to produce a report that shows assets for which:
· the depreciation method has been changed,

· the calculated value of depreciation has been overwritten, or

· the depreciation method used DL (diminishing line) calculation

This report is intended to highlight assets that may not convert cleanly if the depreciation calculation type is changed. That is, the rate of depreciation on those assets may change.

Once you have run this report, you can check the assets it highlights by running Depreciation Type Conversion in report mode.
