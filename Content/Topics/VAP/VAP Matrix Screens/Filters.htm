Filters

You can filter the data in the matrix. The fields that are available for you to filter by depend on the specific screen.

  1. Click the Filter button to display the filter panel.

    2. Filter Panel

  2. Click Filter On.
  3. Type the values you want to filter by, or click to use the lookup. Standard wildcards are allowed (for example: ASHE* to filter on all codes that begin with ASHE).
  4. Click OK to apply the filter.

The matrix refreshes and shows only the data you have filtered by. The filter button turns yellow to show that a filter is applied.

If you are showing totals, the totals only include the filtered data.
