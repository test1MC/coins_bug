Standard House Type Budget Matrix

The Standard House Type Budget screen now uses the new style matrix. The new matrix budget input process presents a ‘flatter’ view of the data. Rather than having to drill down to update payment stage level budgets, all the budget amounts are available from a single screen. The new matrix screen also allows multiple house types to be viewed and updated at the same time.

Actions

The following element actions are now available on the right-click element menu:

  • Update Budget

Filters

You can filter by:

  • House Types
  • Stage Descriptions
  • Cost Categories
  • Trades

Sort

Columns can be sorted by House Type (A-Z or Z-A).

Options Menu

The Options menu gives access to the following functions:

  • Load House Type Budget - This is a new option to load the house type budget from a file, and works in a similar way to the development house type budget matrix action.
  • Copy House Type budget
  • Uplift House Type Budget
  • Switch Budget - This allows you to switch between different VAP budget types.
