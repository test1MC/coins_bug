Plot Budget Matrix

The Plot Budget screen now uses the new style matrix. The new matrix budget input process presents a ‘flatter’ view of the data. Rather than having to drill down to update payment stage level budgets, all the budget amounts are available from a single screen. The new matrix screen also allows access to all house types used on a development.
