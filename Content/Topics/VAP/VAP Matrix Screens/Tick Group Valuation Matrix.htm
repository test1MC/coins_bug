Tick Group Valuation Matrix

The Tick Group Valuation screen now uses the new style matrix.

Tick Group Valuation Matrix

Tooltips on the cells show the budget amount, costs and previous valuation.

Cell Tooltip

For groups that have more than one payment stage linked to them, a hyperlink on the cell allows you to drill down and value by payment stage.

Actions

The following element actions are now available on the right-click element menu:

  • Detail - this was previously available as a button in the cell.
  • Fully Value
  • Unvalue Element
  • Mark as Exempt
  • Mark as Non-exempt
  • Percentage Complete

The following column actions are now available on the right-click column menu:

  • Cascade - this was previously available as a matrix action.

Filters

You can filter by:

  • Plots
  • House Types
  • Tick Group Code
  • Group Description
  • Show/Hide Retired Plots
  • Build Status (All, Build Complete Only, Build Incomplete Only)

Show

A selector at the top of the matrix screen allows you to change between amount and percentage view.

Options Menu

The Options menu gives access to the following functions:

  • Valuation photos
  • Create Certificate from Valuation
  • Unvalue Development
  • Auto Value Build Complete Plots
