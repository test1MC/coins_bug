Development Budget Workbench

Development Budget Workbench is a new function that combines the development house type budget and plot budget screens and allows access to the key functions relating to maintaining a budget.

The initial screen shows a list of all developments and all budgets, with coding to show whether the budget is Empty, Unlocked or Locked.

Development Budget Workbench - Initial Screen

A button allows you to see the lock/unlock history for the development.

Column sets allow you to choose to see additional information for each budget:

  • Budget Comments
  • Budget Locked/Unlocked
  • Budget Locked/Unlocked By
  • Budget Updated
  • Budget Updated By

From this screen you can select a development and budget type. Clicking on the link, you are taken to the development house type and plot budgets.

Development House Type and Plots Budget

If the budget is locked, the header shows "Locked By" and the name of the user who locked it.

"Locked By" Indicator in Header

Options Menu

The Options menu gives access to the following functions:

  • Apply House Type Changes to Plots - Whenever a development house type budget amount is updated, it is flagged as ‘changed’. A message is displayed at the top of the screen:

    Run the Apply House Type Changes to Plots option to update the plot budgets for plots linked to the updated house types.

  • Load House Type Budget - This is a new option to load the house type budget from a file, and is equivalent to the previous development house type budget matrix action.
  • Load Plot Budget - This is the equivalent of the previous development budget load.
  • Copy House Type budget - This is the equivalent of the previous house type budget matrix copy options.
  • Uplift House Type Budget - This is the equivalent of the previous development house type budget matrix action:
  • Uplift Plot Budget - This is the equivalent of the previous development budget matrix action.
  • Lock/Unlock Budget - These are the equivalents of the previous lock and unlock budget functions.
  • Switch Budget - This allows you to switch between different VAP budget types.
