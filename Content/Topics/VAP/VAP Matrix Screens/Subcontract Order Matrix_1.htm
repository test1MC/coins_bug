Subcontract Order Matrix

The subcontract order matrix now uses the new style matrix.

Subcontract Order Matrix

Tooltips on the cells show the certified amounts.

Actions

The following element actions are now available on the right-click element menu:

  • Detail
  • Percentage Uplift
  • Set to Budget Amount(previously 'Update from Budget').
  • Set to Certified Amount (previously 'Zero Unpaid'; this also sets the order amounts to the certified amounts for paid/part-paid cells).
  • Delete Element

The following matrix actions are now available on the row right-click menu and the matrix menu:

  • Percentage Uplift
  • Set to Certified Amount (previously 'Zero Unpaid'; this also sets the order amounts to the certified amounts for paid/part-paid cells).
  • Set to Budget Amount(previously 'Update from Budget').

The following actions are now available on the column right-click menu:

  • Cascade from Plot (this is the equivalent of the previous matrix action 'Cascade Plot to Plot', but is based on selecting a single plot.)

Filters

You can filter by:

  • Plots
  • House Types
  • Stage Descriptions
  • Build Status (All, Build Complete Only, Build Incomplete Only)
  • Fully Certified Plots (Show, Hide)
  • Fully Certified Stages (Show, Hide)
  • Plots with No Order Values (Show, Hide)

Show

A selector at the top of the matrix screen allows you to change between order value and card number.

Options Menu

The Options menu gives access to the following functions:

  • Copy Matrix Details
  • Print Subcontract Matrix Report
