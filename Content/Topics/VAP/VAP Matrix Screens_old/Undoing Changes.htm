Undoing Changes

All the right-click menus, and the matrix menu, include an option to Undo Changes (these are equivalent to the Reset Element and Reset Matrix actions on the old style screens).

Changes you make in the matrix screens are saved automatically. but you can undo them as long as you have not navigated away from the screen.

  1.  Do one of the following:
    • To undo the changes to a cell, right-click the cell and choose Undo Changes from the menu.
    • To undo the changes to a row, right-click one of the frozen columns in the row and choose Undo Changes from the menu.
    • To undo the changes to a column, right-click the column header and choose Undo Changes from the menu.
    • To undo changes to the whole matrix, click the menu button and choose Undo Changes from the menu.
  2. Then answer Yes to the question Are You Sure?

The values in the cells whose changes you have undone revert to the values they had before you began the current update session, and are no longer highlighted.

Note
As the Undo Changes options are standard on all matrix screens, we have not listed them separately in the sections on changes to specific screens.
