VAP Certificate Matrix

The VAP certificate matrix now uses the new style matrix.

Certificate Matrix Screen

Tooltips on the cells show the order value and the previously certified amount.

Note
The Dayworks/Variations tab still exists, but is now only.

Actions

The following element actions are now available on the right-click element menu:

  • Detail - this was previously available as a button on the element
  • Update with Order Amounts (previously 'Update from Order ')
  • Mark as Complete
  • Mark as Incomplete
  • Update Notes
  • Zero Cumulative Amount
  • Delete this Payment - This is a new option which removes the payment from the certificate.

The element action ‘Previous Certified Amounts’ has been combined with the cell detail option available from the ‘Detail action.

The following matrix actions are now available on the matrix menu:

  • Update with Order Amounts (previously 'Update from Order')
  • Mark as Complete
  • Mark as Incomplete
  • Zero Cumulative Amount
  • Delete this Payment - This is a new option which removes the payment from the certificate.

Filters

You can filter by:

  • Plots
  • House Types
  • Stage Descriptions
  • Build Status (All, Build Complete Only, Build Incomplete Only)
  • Fully Certified Plots (Show, Hide)
  • Fully Certified Stages (Show, Hide)

Show

You can show:

  • This Payment
  • Cumulative
  • Order Value
  • Unpaid
  • Card Number
