Tick Group Valuation Matrix

The group valuation matrix now uses the new style matrix.

The following element actions are now available on the right-click element menu:

  • Detail
  • Fully Value
  • Unvalue Element
  • Mark as Exempt
  • Mark as Non-exempt
  • Percentage Complete
  • Reset Element (now ‘Undo Changes’).

The following column actions are now available on the right-click column menu:

  • Cascade
  • Reset Matrix (now 'Undo Changes').

The following matrix action is now available on the matrix menu:

  • Reset Matrix (now 'Undo Changes').
