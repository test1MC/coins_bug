Screen Flow

After login, the user will be presented with a list of Contracts. The contracts the users can view are defined by security in the Contract Status module.

Each contract will show the contract name & number. The name of the screen is highlighted by the white coloured large text.

After the user selects a contract from the list, they will be presented with a list of Purchase Orders for that Contract.

Each PO can render the following details

  • PO number
  • Short description
  • Supplier name
  • Contract name & number(When viewed)

Selecting a Purchase Order will present the View Screen. This will display the Orders made against the PO. 

It will also allow the user to create new GRN by tapping on the line item or by selecting the ‘Create GRN’ button.
