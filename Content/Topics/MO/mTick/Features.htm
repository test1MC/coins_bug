Features

On Plot selection, the user will get the Tick Groups for that Plot and selected Tick Group will be highlighted.

Number of groups will be displayed in the top right hand corner.

The Show Fully Valued button, allows the user to see Tick Groups that are fully valued.

Starring Plots will mark them as favourites. These plots will be cached onto the device and displayed to the user. The can be filtered by favourites.

Only previously viewed and starred plot will be cached.

Users can search from the developments/plot/materials screens.

Search feature works with any number/characters and searches all fields.

Tapping on the ‘Refresh’ button, the application will fetch data.

The user will be able to use the search and refresh button on all screens apart from photo screen.

SORT BY on development screen will allow the user to sort developments in order of their number or distance.

FILTER BY on the development screen allows the user to filter the developments by ‘Favourites’ or ‘None’.

Choosing ‘None’ will remove all filtering options that were previously selected.

When the user taps to sort by ‘Favourite’, the following option will be given. Here the user can select to view developments which have been starred or all of those except starred developments.

A total will always be displayed at the top right hand corner.

This will be shown for Developments, Plots, Materials and Tick Groups.

SORT BY on plots screen will allow the user to sort the plots by ‘Build Sequence’ and ‘Plot Number’.

To return to previous screen user has to click back button or the mTick home button in the top left corner.

The user can filter by any of the following options:

  • None
  • Group
  • House Type
  • Sellable
  • Favourites

All filter by options are persistent, meaning that when the user logs out and in again, the users filtering preferences will be saved.

If the user tries to use a filter that has not been set. i.e. ‘Groups’, then the following message will be shown.

For the materials the user the can ‘SORT BY’ the following:

  • Commodity
  • Price
  • Value

A ‘Total Value’ will be displayed at the bottom of the material screen. This will automatically get updated whenever the user makes amendments to the quantity.

All sort by options are persistent, meaning that when the user logs out and in again, these sorting preferences will be saved.

Tapping onto the calculator symbol will bring up the will bring up a pre-installed calculator on the device.
