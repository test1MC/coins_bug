Plots

Within a Development, the user will see the Plots on site.

The following plot information is available to the users to view and utilise features like sort and filter by. More information in section with features.

  • Type
  • Sequence
  • Group
  • Status

Tap on ‘Star’ to choose favourite plots. The application will cache data for that plot

Tap Show Completed to see the ‘legally completed’ plots.

The dark background on Plots suggests the plot has cached data as user has either previously accessed it or has starred it. This will allow user to continue working if there is a loss of internet connectivity on device.

To get to the Developments, the user must tap the back or mTick home button.
