Materials

Within a Development site, the user can also see the Materials on site.

Materials are shown in a list and their quantities are editable.

Materials will display the following details:

  • Commodity
  • Description
  • UoP
  • Price
  • Value
  • Quantity (current)

Selected material will be highlighted.

Use the calculator to work out quantity and tap on the ‘+’ button to accumulate the materials on site.

If the user taps to add a quantity, an input dialog will appear.

Once the input has been confirmed, the user can continue to enter more materials and view the ‘Total Value’ of materials on site.

After materials have been entered, they must be submitted by tapping ‘Submit’ button before leaving this screen.

If the user tries to exit this screen without submitting data then the following error message will be prompted.
