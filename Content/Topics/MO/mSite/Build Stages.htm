Build Stages

Users can open a Plot to view/update the build stage for this Plot.

To update the stage, users can the box which will then prompt the with a send button.

Users may or may not be able to skip any stages (subject to set up). The stages not previously ticked will be highlighted in red. It is possible to tick on all previous incomplete stages automatically – please refer to CW Parameter settings.

Tick boxes for the complete stages are editable, but the boxes ticked in previously cannot be edited unless FCWALLDEL is set to ‘Y’.

The users can cancel the submitting data by either tapping the back buttons or refreshing the screen by swiping down

The stages screen shows user additional information at the top of the screen depending on the set up. In this case we can see the Plot Address, Expected Build Completion and House Type.

Stage photos can also be captured.  Access to photos is via the image icon. Photos can be captured via the camera and attached to the plot from gallery. These are automatically linked to a build stage. Notes can also be appended.
