Log In/Log Out

The Login page will be displayed prompting the user to enter the Username & Password. After typing username and password, user should tap Sign In.

By tapping on the settings icon located top right corner.  It will take the user to the Settings screen where they can update or check Master Host, Domain and Application version.

The Master Host and Domain details are provided to device administrators by Coins Mobile. Also modifying the Master host will clear any cached data & settings on device.

An error alert will be prompted if there is no internet connectivity or the Master Host & Domain details are incorrect.

If the username and password are wrong, a message will pop out to inform the user

User would log in using the details set up in Coins OA and device administrators can control each user passwords and roles. If this is not set properly the user will not be able to log in and get authenticated.

Once authenticated, the user has to select a ‘KCO’ to proceed to the Sites, the next screen. If the user does not have access to multiple KCO’s then they will not be prompted with this.

To log out from the application users can tap on the menu icon located in the top left corner or the back button of the device. This will prompt the Log out prompt.
