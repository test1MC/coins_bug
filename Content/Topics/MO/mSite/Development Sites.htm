Development Sites

Each tile will entail Sites details that will include the following.

  • Site Name, Number and Address.
  • Distance to Site from current location.
  • Number of standard and Housing Association Plots.

To set some sites as Favourites, users can tap on the Star button which will allow users to filter by favourites and also store site information onto the device. Users will then have access to the site when there is no mobile coverage available

To filter by favourite sites or vice versa, tap on the options button at the top right. Then select Filter by>Favourites/All . Sites can be filtered by either Favourites/Non Favourites.

The search button at the top left lets you search all fields and values on the current screen.

When a site has address details, a navigation icon appears with the distance from the current location

To Refresh the sites screen, users can swipe down. An arrow icon will appear which retrieves information from

Users can Sort by Name, Number or Distance of Sites, either in Ascending/Descending orders.

Information can be viewed by tapping the menu icon located lop left.

The Date which appears on the top left within menu will show the current date. This can be changed by tapping onto the date.
