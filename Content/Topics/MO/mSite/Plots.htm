Plots

The Plots within a site will be displayed that can be searched and can be sorted by Plot or Sequence number either in Ascending/Descending order.

Each plot tile contains following details:

  • Sequence
  • Address (Optional)
  • Build Stage
  • Forecast Date

Access to photos is via the image icon (Figure 1). Photos can be captured via the camera and attached to the plot from gallery. These can also be optionally linked to a build stage and have notes appended.

Photos captured at Plot level will need to be sent via tapping the send button that appears at the bottom right.
