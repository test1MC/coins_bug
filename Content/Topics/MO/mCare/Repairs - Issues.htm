Repairs - Issues

When the user taps On site from the Visit Details screen then they will be taken to the "Actions" screen where the "ISSUES" tab will be displayed.

The "ISSUES" tab will contain the following information:

  • Issue number
  • description
  • Complete & Incomplete button
  • Number of photos
  • Add issues button (+)

 

Tapping on the green add issue button will load the Issue "Add Issue" screen. Similar to adding an issue from an inspection when failing an item, the will need to enter the following

details:

  • Description
  • Location
  • Category
  • Priority
  • Notes

The can also add photos against the issue, save or discard the issue.

Tapping on the back or the cancel ("X") button the will be shown the following message where they will need to confirm their action before proceeding.

If the issue is added it will appear on the "Actions" screen where the will have the option to delete it.

The user can continue to add additional issues.
