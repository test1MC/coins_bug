Repairs - History Tab

The "Plot History" tab will display a list of issues that have been assigned against the task.

 

Each historical repairs task will contain the following details if captured during the visit:

  • Call Reference number
  • Date of the visit
  • Issue reference number
  • Priority
  • Location
  • Sub-Location
  • Category
  • Sub-Caretgory
  • Description
  • Notes
  • Status
