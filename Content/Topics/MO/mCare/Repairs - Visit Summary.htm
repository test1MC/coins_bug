Repairs - Visit Summary

By tapping on Summary the user can view a list of issues that were reported during the visit.

This will display the following:

  • Issue ref
  • description

 

Issue Details

If the wishes to view more information about the task then they can tap onto the issue from the Visit Summary screen and will be taken to the Issue Details screen.

Here the can see the following information about the task:

  • Reference
  • Description
  • Location
  • Sub-location
  • Category
  • Sub-category
  • Priority
  • Notes

To proceed to the Job Completion screen, the user will need to tap on the Issues Details heading, then on Continue (where they will be taken to the Visit Signature screen).

Once they tap Continue from the Visit signature screen they will be taken to the Visit Details screen for that plot.

Here they can view the following:

  • On-site date and time
  • Completed Date and time

Tapping complete will close and remove this task from the ‘Visit List’

Once the user taps on Complete, the task details will be sent back to OA where the backend user can view all issues, notes, photos that have been submitted.

This can be viewed from the Customer Care module, in the Customer Care Workbench under Tasks.
