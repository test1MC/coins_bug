Repairs - Actions

From the "Actions" screen the will first see the Issues tab, they can also swipe across to the "NOTES", "SIGN" and "COMPLETE" screens.

The hamburger menu will always be displayed in the top right corner of the screen, this will give the user the ability to navigate their way to alternative screens.

 

From the above example there are 3 issues. issue with the reference number 120 is the issue that came from , the two additional issues were added from the device during the visit.

We can see the 3 different (blue, grey and green).

  • Blue - issues that have not been given a "Complete" or "Incomplete" status. The will not be able to complete the visit without a status being selected.
  • Grey - Incomplete
  • Green - Complete

Tapping on the issue that came from will display the issue details. This screen will contain the following information regarding the issue:

  • Issue reference number
  • Call reference number
  • Description
  • Location
  • Sub-Location
  • Category
  • Sub-Category
  • Priority
  • Notes
  • Photos captured from previous visit

The will also be able to cpature photo against the issue on this screen.
