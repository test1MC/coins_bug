Access Jobs - Signature

The user can obtain the customers signature to confirm the visit.

The dropdown option allows the user to select the signatory and the name can be inserted in the text box.

To obtain the signature, the user has to tap the ‘Sign’ button.

This will display the screen where user can sign and if need to change, they can tap on Sign Again button.

If the user selects OK then they will be prompted to answer a few questions.

Clicking OK will take the user to the Survey screen.
