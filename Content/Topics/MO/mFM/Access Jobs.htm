Access Jobs

Task List

This will display a brief snippet of the tasks assigned to the user for the current day. This will include details such as: job, task, contact, task title, summary (drop down arrow) and the time the task is allocated for.

Task Details

Contact Tab

Contact screen contains details for the customer contact name, address, contact number and notes. (Figure 2.1)

Task Tab

Task screen contains the title, notes and a list of SORs assigned for the task. (Figure 2.2)

History Tab

History screen will show the history of this job.

Actions Tab

Actions gives the user options to book onto the task by choosing On Route or Navigate.

When the user taps the On Route, it will prompt a confirmation alert message.

When the user taps Navigate , the application will also prompt a confirmation alert message.

Once user proceeds, a choice of Navigation applications will be available depending what is already pre-installed. The user can select the desired application and navigate to destination

When the user has arrived at the location for the task, they will need to notify arrival. They will be given two options.

  • On Site
  • No Access
