Access Jobs - Task Photo

The user can capture photos after tapping the Camera icon on the Task Details screen.

The user has to tap the Take Photo button.

The application will launch the camera screen where user can take photos.

Once the photo is taken, user can view it by tapping on the small view of the captured photo.

Notes can be appended in the text box and user can also Tag photo as a Pre or Post task.

Users can delete the photo by tapping the ‘X’ icon.
