Features

Camera

The user can utilise the camera icon from the main Menu and they will be prompted to take the photo.

Camera options and layouts may vary depending on device; however, the functionality of the application will remain the same.

Timesheets

The user can select the type of timesheets that they wish to complete: Activities and Task Activities.

When the user selects Activities they will be able to add Absence, Holiday and Training hours. (Figure 4.1)

When the user selects Task Activities they will be able to select the task which they wish to amend.

Once a task is selected the user can amend the time for On Route, On Site and Off Site

After the user has made amendments they will need to click on the icon at the top right corner of the screen to save the changes.

When the icon disappears this indicates that the changes have been saved.

Navigate

If the user taps the Navigate button, they will be prompted with a confirmation alert message.

Once user proceeds with the chosen Navigation application tool. The user can operate the application to navigate to any destination.
