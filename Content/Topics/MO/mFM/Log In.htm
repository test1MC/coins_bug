Log In

Locate and open the mFM application from the device application menu.

The Login page will be displayed prompting the user to enter their Username & Password.

If the user inserts incorrect details or if nothing is entered, the following error alert messages will be displayed: “Please insert username and password”

An alert error will be prompted if the incorrect credentials have been entered

An alert error will be prompted if there is no internet connectivity.

Once the internet connection is active, the user is then authenticated. This will result in successful log on and subsequently the ‘Menu’ screen is displayed.

From the settings cog there is an option to change the Menu display from Grid to Carousel
