No Access Jobs

To book on to the task, the user can tap the On Route or Navigate button.

Once the user arrives at the location for the task, they may not be able to access it. Therefore they will need to tap the button No Access.

After selecting No Access, the user has to record information for the visit. This includes the options to if carded, and select a Reason from the drop down options.

There is also an option to enter Notes for any observations.

Once the user has filled in as much information as possible and clicks continue, the user will be prompted to confirm the reason. They can select alternative reasons or confirm their previous selection by clicking on the drop down menu.

Once this has been confirmed and user selects “OK”, the application will close this task and return to the task list screen which will show remaining/incomplete tasks.
