Date/Time and GPS Photos Stamp

FMSTAMPPHOTO and FMSTAMPPHOTOGPS definitions are available under:

Mobile Configuration Settings (Company)

Mobile Configuration Settings (User)

Date/Time photo stamp

If FMSTAMPPHOTO setup definition is set to true, every photo that is taken will be stamped with date and time (at the bottom of the picture), if the setup definition is set to false none of the photos will be stamped.

GPS coordinates photo stamp

If FMSTAMPPHOTOGPS setup definition is also set to true,  every photo that is taken will be stamped with date, time and GPS coordinates, if GPS location is available and its accuracy is less or equal to 50 meters.

If coordinates are not available or the accuracy greater than 50 meters photos will be stamped with date, time and “Coordinates Not Available”.

If Location Services are turned off on the device the user will be prompted to turn it on. The user will have to click OK to continue.

This will display the Android Location Services setup screen. Once the user click the back button, the camera screen will be displayed.
