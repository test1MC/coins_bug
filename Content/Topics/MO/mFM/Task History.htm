Task History

FMTASKHISTORY definition is available under:

Mobile Configuration Settings (Company)

Task history tab

If FMTASKHISTORY setup definition is set to true a history tab will be added to the Task Details screen.

If task history is available, a list of the last completed jobs for this location will be displayed in the history screen.

The number of displayed jobs will be set by HISTJOBS parameter in OA.

Tapping on a task will display the task details.

Tapping on an SOR item in the Task Details screen will display a pop up with the SOR details.
