SOR Variation Limits

FMSORVARIATIONLIMIT definition is available under:

Mobile Configuration Settings (Company)

Mobile Configuration Settings (Contract)

Mobile Configuration Settings (User)

FMSORVARIATIONLIMITONOFF is available under:

Mobile Configuration Settings (Company)

Mobile Configuration Settings (Contract)

FMSORVARIATIONTYPE definition is available under:

Mobile Configuration Settings (Company)

Mobile Configuration Settings (Contract)

Values:

  • TOTALCOST
  • TOTALVALUE
  • VARIATIONCOST
  • VARIATIONVALUE

 

FMSORNEEDAPPROVAL is available under:

Mobile Configuration Settings (Company)

Task SOR variation limit

If FMSORVARIATIONLIMITONOFF setup definition is set to true and the varies task SOR where the variation value exceeds the limit set on the FMSORVARIATIONLIMIT setup definition, the user will be prompted with a warning.


The user will be able to continue but will not be able to that SOR as completed, until it has been approved.

If SOR needs an approval, user will not be able to select Task Status in the Visit screen; instead the Status will be set to the value of the setup definition FMSORNEEDAPPROVAL

Variation limit change

If a task is flagged as approval required and the application receives an update with new value for FMSORVARIATIONLIMIT, which allows the SOR to be completed, the user will be prompted to complete pending SORs. Clicking yes will display the SOR screen, where the user can the SOR as completed and select the proper status in the Visit screen before completing the task.
