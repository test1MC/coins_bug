Photo Capture Enhancements

FMPHOTOMAXNUMBERPERTASK, FMPHOTOMANDATORY and FMPHOTOMINNUMBERPERTASK definitions are available under:

Mobile Configuration Settings (Company)

Mobile Configuration Settings (Contract)

FMPHOTOMAXWIDTH and FMPHOTOMAXHEIGHT definitions are available under:

Mobile Configuration Settings (Company)

Mandatory Photo Capture

If the FMPHOTOMANDATORY setup configuration is set to true and number of photos captured is less than the value of   FMPHOTOMINNUMBERPERTASK setup configuration the will be prompted with warning letting him know he cannot continue without taking the minimum number of photos.

Maximum number of photos

The maximum number of photos that can be taken per task is limited by the value of the setup definition FMPHOTOMAXNUMBERPERTASK.
If the user tries to take more photos than the maximum value, he will be warned he cannot take more photos.

If FMPHOTOMAXWIDTH and FMPHOTOMAXHEIGHT setup definitions are set, the photos will be resized accordingly.
