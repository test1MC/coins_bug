Task list types

FMTASKLISTTYPE, FMTASKLISTNUMBEROFDAYS, FMTASKLISTUSECACHE and FMTASKLISTACTIONFUTUREWORK definitions are available under:

Mobile Configuration Settings (Company)

Mobile Configuration Settings (User)

FMTASKLISTTYPE values:

  • STANDARD (Current day view)
  • MULTIDAY (Multiday view)
  • SINGLETASK (One task at a time view)

Task list type is configurable on OA

Standard task list

  • Standard task list will display the tasks that are scheduled up to the current day.
    The tasks will be ordered according to the appointment time.

Multiday task list

  • Multiday task list will display the tasks that are scheduled up to the current day and tasks that are scheduled to the next days (number of future days is configurable on OA).

Single task list

Single task list will display only one task at a time. The displayed task will be the one with the earliest appointment.


Once the task is complete, the next task will be displayed.
