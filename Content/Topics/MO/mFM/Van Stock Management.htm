Van Stock Management

FMVANSTOCK, FMVANSTOCKREQUEST and FMVANSTOCKREQUSTDEPO definitions are available under:

Mobile Configuration Settings (Company)

Mobile Configuration Settings (User)

 

FMVANSTOPISS definition is available under:

Mobile Configuration Settings (Company)

Van stock in main menu

If FMVANSTOCK is set to true, the van stock button will be displayed in the main menu.

Parts screen is be added to the list of screens in task actions.

Van stock screen

The van stock screen will display a list of all items held in the van, including each item code, name and available quantity. Tapping on an item will display additional information.

The list can be searched by tapping the magnifying glass icon.
If FMVANSTOCKREQUEST is set to true, a plus icon will be displayed in the action bar.

Tapping on this icon will display the stock request screen.

Van stock request screen

Tapping on Add button in the van stock request screen will display the stock screen.
Search option can be used to filter the list.

Only one item can be selected at a time and quantity has to be selected before adding the item by tapping ok.

Added items will be displayed in the van stock request screen; each item will have an Edit and Delete buttons.

Tapping the Delete button on each item will prompt the user to confirm the action.

Tapping on the Edit button with display an item edit screen in which the user can update the required quantity and save any changes by clicking on OK.

Tapping Continue in the stock request screen will prompted the user to confirm the request.

If the user selects No then the request will be cancelled and the application will return to the Van Stock list screen.

If the user selects Yes then the request will be submitted and the application will return to the Van Stock list screen

If the user returns to the main menu without submitting the request, and the returns to the stock request screen the selected items will still be displayed.
Logging off without submitting the request will clear the stock request list.

Parts screen

If FMVANSTOCK is set to true, the Parts screen will be displayed after tapping continue in the SORs screen.

Tapping on the Add button will display the van stock screen, which will include the stock items.

If the user taps and holds on a part, further details such as: Location, Balance, Code, Group, Unit Cost and Description will be displayed in a popup box

The user can select the required item and the quantity.
After tapping OK the setup definition FMVANSTOPISS will be checked.
If FMVANSTOPISS is set to E the selected quantity will not be allowed to exceed available quantity.
If FMVANSTOPISS is set to W, the selected quantity will be allowed to exceed available quantity and the user will be prompted to confirm the action.
If FMVANSTOPISS is not set, the quantity will not be checked.
After the selection, The Parts screen will be displayed with the selected items. 

Each item will have edit (pencil icon) and delete (X) buttons.

Tapping the edit button will display item edit screen in which the user can update the required quantity.

Tapping the delete button will prompted the user to confirm the action.

The selected items will be saved in the parts screen even if the user goes back to the main menu and returns later to the active task.
If the user logs off the selected list will not be saved in the task, unless the task was put on hold before logging out.
