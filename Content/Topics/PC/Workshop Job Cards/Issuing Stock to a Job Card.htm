Issuing Stock to a Job Card

 

To issue stock to job cards, Navigate to Stock > Input > Input Stock Transactions.

Click to create a new batch.

Field

Description
Select the correct for this batch
Transaction Date Enter the transaction date for the transaction within this batch.
Transaction Type

Select Stock Issue from the drop-down list
Description Enter a description to help you identify the batch later.
Location Select the stores from which you are making the issue

Click . The Transactions Tab will be displayed.

On the Transactions Tab, Click

Field

Description
Stock Code Enter, or select from the lookup, the stock code for the item to be issued.
Description This will default from the Stock record, but may be amended if required.
Quantity Enter the quantity of the item(s) to be issued.
Analysis Select ‘Workshop’ from the drop-down list. The analysis cost head will be partly completed for you based upon the stock group record that the stock item belongs to. DO NOT overtype this. Use the Lookup, to locate the appropriate Job Card – which will complete the analysis with the job card number automatically, or enter the Job Card number as the prefix before the displayed cost head.
Transaction Date The date of the transaction – defaults from the batch header, but may be amended if required.
Our Reference Our reference of the transaction. (optional)
Responsible Person The person responsible for the item of equipment (optional)

 

Click to save the transaction.

Continue to add further Stock Issue transactions as required. When complete, Click to return to the Batches screen.

Posting the Batch

From the Batches screen, select the batch to be posted.

From the Actions drop-down, select Post and click Apply Action .

When the Batch has completed posting, the stock levels will be updated and the items costed to the Job Cards.
