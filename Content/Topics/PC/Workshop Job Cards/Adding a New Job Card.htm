Add Screen

The screen is split into three areas: Detail, Recharge and Completion. When setting up a new , only the Detail and Recharge sections should be completed.

DO NOT fill in the Completion section until the work is actually complete.

Detail Fields

Field

Description
Workshop Enter the location, depot, or workshop where the asset is kept.
Asset Enter a code for the asset (a lookup is available)
Event Select an Event Code (e.g. Repair, Service etc.)
Title The Title of this Workshop Job
Reg. No The registration Number (if relevant to this workshop job)
Meter Reading The meter reading (if relevant to this workshop job)
Date The date of the workshop job.
Order Enter the Customer Order Number, if relevant. May be left blank.
Received Enter the date received (optional)
Required Enter the required date (optional)
Authored Enter the person who authored the workshop (optional)
Estimate Enter the estimated cost (optional)
Reference Enter a reference for the estimate (optional)
Description Enter brief details of the work to be done

Recharge Fields

Field

Description
Recharge

Whether costs to this workshop get recharged. This field will initially be ed. You will not be able to un it until the Details section has been completed.

If ed, the following fields are available for completion
Charged Whether this workshop has been fully charged
Ready to Charge Whether the workshop is ready to charge
Recharge To The type of analysis (customer, job, or split). Selecting the analysis will allow the correct lookup in the adjacent field.
Charge The charge . A lookup is available.
Description The recharging description.
