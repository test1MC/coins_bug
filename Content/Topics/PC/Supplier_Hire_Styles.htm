Supplier-Specific Styles

suppliers can have varying styles depending on the type of equipment being . To make it easier to choose the correct style on order, styles can now be linked to a company record in the Company Information Workbench, so that when an order is raised, the styles dropdown is limited to only the styles provided by the .

There are two new subtabs on the company record 'Other' tab:

  • Styles allows you to create a list of the styles in use by the , and to identify a default style.
  • Code Styles allows you to specify the 's default styles for each type of item, by associating particular styles with specific codes.

When you enter code on an order line, the style is determined as follows:

  • If the code you entered exists in the Code Styles list held against the , that style is defaulted in, and the Style combo-box is populated with that style and any -specific styles from the Styles tab.
  • If the code does not exist in the Code Styles list, the Style combo-box is populated with all the -specific styles from the Styles tab. The default is taken from that list if a default has been specified; if no default has been specified then the style is taken from the code’s style as held on the Code Workbench. However, if that last style does not exist in the -specific Styles list then the first style in that list is taken to be the default.
  • If no -specific style has been specified then the default is taken from the code’s style on the Code Workbench and the combo-box is populated with all the standard styles.
