Loading Web Images

The below files are in the ".gif" file format.

 

 

 

FTP to the server

 

Populate the images into an easy to find and relative directory.

In the example below I have gone into /home/coins/ and created the images folder, so the directory is /home/coins/images/

FTP the files into the directory, make sure the owner and group are ‘’.

Change the permissions on the directory and the files to 666

 

 

Navigate to the web server path = /opt/hpws/apache/htdocs/coins

 

 

Then link the images directory that you have just created to this path... “the web server path = /opt/hpws/apache/htdocs/coins”

 

 

Check the PC parameter, IMGDIR, is set to the correct directory:- /coins/images/
