Plant Web Parameters

Below are some of the most relevant Web parameters that will need to be amended in order for the images and filepath to work correctly.

 

If you are a Hosted client, log a Support Request for the Web .gif images to be loaded in by our Technical Support Team. The filepath should be provided to you by them after which you'd need to reference it in the "IMGDIR" parameter.

 

"IMGDEF" is the default catalogue image if there is none specified in Plant Hierarchy or against the Asset Group in Assign Plant Hierarchy.

 

"IMGDIR" is the filepath for where the images are loaded in.

 

"IMGEXT" should be set to "gif" and the images should be in .gif file format.

 

 

In addition to this document, there is another titled "Loading Web Images" which must be followed if you are a non-Hosted client.
