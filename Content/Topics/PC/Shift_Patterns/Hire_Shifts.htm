Specifying Shifts for

When making requests using the web or – Requisitions, you can specify the shift pattern (for scheduled asset and codes).

Web – Add to Basket Screen

NOTE

If a shift pattern is available, you do not have to select one. You only need to use shift patterns if you are concerned about scheduling using hourly timings.

If the shift pattern uses groups, the Group field is populated. You can choose which groups apply by deleting the ones you don’t need.

Optionally, you can enter an Offset if the shift does not start at the beginning of the pattern. If the shift pattern starts on a specified day of the week, the offset identifies which shift of the start day. If the pattern has no specific start day, the offset specifies any shift of the pattern.
