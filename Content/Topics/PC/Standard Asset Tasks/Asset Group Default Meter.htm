Asset Group Default Meter

You will need to set the default meter for each Asset Group. Navigate to Fixed Assets > Setup > Maintenance > Asset Groups

 

Select your asset group, and on the Detail tab, Open the record, and select the relevant caption from the drop down in the “Caption Used for Service Meter” field.

This is the caption that holds the meter value you want base service cycles on for the group.

You only need to set this if overriding PC/SERVMETR. The caption needs to be a numeric data type and must be set up to retain history, as the history is used to work out historical rate of change used in calculating the expected next service
