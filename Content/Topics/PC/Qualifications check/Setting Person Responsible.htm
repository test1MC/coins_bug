Setting the Responsible Person

When setting a responsible person, if that person has a job which PC/OPERJOBS nominates as an operator, then their qualifications will be checked. An error is shown if the person is not qualified, or if renewal is needed. A warning is shown if the qualification is optional or the date required is beyond the qualification window.

This validation happens on Person Responsible browses, and on records that will lead to a Responsible Person record being created: Requisition, Pending , Fast Dispatch and Pre-confirmation Dispatch. Validation occurs when the record is saved because several fields (item, person, start and end dates) need to be set before validation is sensible.
