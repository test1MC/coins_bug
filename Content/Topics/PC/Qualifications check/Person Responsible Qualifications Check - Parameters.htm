Person Responsible Qualifications Check - Parameters

The following new parameters control this functionality.

Parameter Description
OPERJOBS A can-do list of HR job groups that identify Operators.
QUALWNDW

The window of time where lack of qualification will give an error rather than warning. This is the number of days in the future when a missing qualification changes from an error to a warning. If you schedule responsibilities well in advance this allows you to schedule ahead of qualifications being gained. There is a report which looks for qualifications missing or needing renewed; see 1 , Missing Qualification Report.

If you set the qualification window to 0, you are in effect setting all missing qualifications to warning only.
