Breakdown Processing - Maintenance Desk

A new version of the Maintenance Desk has been introduced. The existing version of the Maintenance Desk has been moved to the WIP & Deprecated menu.

The Maintenance Desk has three tabs:

  • The Assets tab lists P and V type assets. A body selector allows you to show All, For Inspection, Awaiting Decision and Breakdowns. Actions allow you to set the asset status to Available or to create for the asset, or (if PC/WAITSTAT is populated) to set the status to Awaiting Decision.

  • The Inspection/Workshop tab shows the Workbench.
  • The Breakdown tab is a new browse that shows that have the event specified by PC/BDWNEVNT. There is a filter for Open, All, Complete with Open being the default. The Exchange Asset action allows you to swap the asset. A field shows the number of free days in the period of the breakdown, with a link to the Free Days screen for the dispatch linked to the breakdown.
