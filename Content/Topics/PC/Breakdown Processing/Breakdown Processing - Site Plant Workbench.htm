Breakdown Processing - Recording the Breakdown

A new An action - Report Breakdown - on the Site Workbench allows you to record that an item has broken down. The same feature is available on the Desk.

You are prompted for the date of the breakdown.

This action creates for the asset, using the depot from the dispatch and the event from PC/BDWNEVNT.

If the PC/BDWNMAIL parameter is set, an email detailing the asset, the depot, dispatch, and user recording the breakdown, is sent to the users or roles listed. If a role is used, an email will be sent to the user or users who are set up with that role on the depot or .

If the PC/BDWNSTAT parameter is set, the asset status will be set accordingly.

A new column shows the reference of any created for breakdown of the asset, with a link to the summary.
