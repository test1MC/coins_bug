Posting a VAT Return

After the VAT Return has been submitted, it must be posted.

To post the Return:

Go to VAT > VAT Returns Workbench.

On the VAT Returns tab, select the Return.

Choose the Post VAT Return to GL action.

Posting a VAT Return will transfer the returned balances from the VAT control accounts to a separate VAT Debtor/Creditor Account. The balance on this account will then be cleared by posting of the cash payment/receipt to/from HMRC. This process avoids the need to allow manual postings to the VAT control accounts for this monthly/quarterly cash posting.

Only unposted returns can be posted. This will create and post a GL batch with one line per unique GL control account for VAT codes (excluding non-recoverable input VAT codes) included in the return with a value of the total VAT for VAT Codes linked to that GL Control Account (Positive output VAT creates a Debit entry and Positive input VAT creates a Credit Entry). The total of these lines is then reversed and posted to the Creditor/Debtor Account for the VAT Registration. The Posting Status for the Return is changed to "Posted".

A posted return can be unposted.

Go to VAT > VAT Returns Workbench.

On the VAT Returns tab, select the Return.

Choose the Unpost VAT Return to GL action.

This makes the reverse entries to those that were made when the VAT Return was posted.
