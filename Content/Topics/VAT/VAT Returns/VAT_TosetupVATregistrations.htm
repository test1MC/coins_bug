To set up VAT registrations:

If you have Multi Sector VAT Registrations remove VAT registration from Company Set up before setting up VAT registrations

Using VAT > VAT Registration, set up a record for each VAT registration (each entity that is registered with the tax authority and that will make VAT returns).

For companies or sectors that are not VAT registered, set up a separate "dummy" registration record with the Regd field unticked.

Using VAT > Global Company Configuration > to indicate which VAT registration each company or sector belongs to. 

Click on the Sector VAT reg

The browse screen shows all companies; each company must be linked to one of the VAT registration records (or if sectors are in use, each sector must be linked to a registration record). 

The Registration Number field is -coded to indicate any VAT registration numbers that need updating. 

To change the registration number for a company, update the record in VAT > Global Company Configuration >Global Company Configuration.

To change the registration number(s) for sectors within a company, go to General Ledger > Chart of Accounts > Sector Details Maintenance, and update the sector record.  You can leave the registration number on individual sectors blank if it is the same as the company that the sector belongs to.

Sector VAT Codes can only be entered if the parameter GL/SECTGLEL has been set.

GL/SECTGLEL

If you are using sectors, enter a number between 1 and 4 to identify which element in the General Ledger account code denotes the sector.
