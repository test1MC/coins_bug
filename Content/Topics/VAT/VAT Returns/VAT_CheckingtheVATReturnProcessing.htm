Checking the VAT Return Processing is Complete

The following tabs of VAT > VAT Returns Workbench all provide coded status columns to help you check that your VAT Return processing is complete:

VAT Returns – for all VAT Returns for VAT registrations submitted from this company;

Company Summary – for the latest VAT Return for each VAT registration submitted from this company;

Group Summary - for the latest VAT Return for each VAT registration (subject to security access to the submission company for each registration).

 

Posting Status

Unposted - The posting batch has not yet been created.

Posting - The General Ledger posting batch has been created but has not finished posting.  This should normally be a transitory status unless there is a problem with the batch posting.

Posted (with Dig Deeper to General Ledger batch) - The batch transferring the entries from the control accounts configured against each VAT code to the creditor/ debtor account configured against the VAT Registration has been posted in the General Ledger.

Manually Posted - The necessary General Ledger postings have been dealt with manually.
