Configuration - VAT Codes

VAT Codes are setup VAT>VAT Codes

Separate VAT codes are used for input and output VAT

To add new VAT codes Select “Add” from the menu

For each code enter the code required a description and a rate and the relvant boxes for recoverable and returnable, the inactive tick box is used to retire unused VAT codes

Select from the dropdown to define if the code being enterd is input or output VAT

If the code is to be used for EU returns select if it is for Goods or services

Enter the Input & Suspense GL codes to be used for this VAT Code

Save the entry.
