Reverse Charge VAT - VAT Codes

A new field has been added to the VAT Codes maintenance screen.

Field

Field
Description

Auto-generated?

If ed, this code is only used on VAT distribution lines that are auto-generated as a result of the settings in the VAT Code Substitution table.

Codes on which Auto-generated? is ed are excluded from normal VAT code lookups (but are visible in the lookup on the VAT Code Substitution screen itself).
