To mark a Return as Manually Submitted:

  1. Go to VAT Returns Workbench.
  1. On the VAT Returns tab, select the Return you want to submit.
  1. Choose the Mark VAT Return as submitted manually action.

This updates the submission status from "Unsent" to "Manually Submitted".

 

 

Currently only supports electronic submission of UK VAT Returns.  Submissions to other jurisdictions, including Ireland, should be marked as manually submitted. do not support electronic submissions for VAT Returns under the annual accounting scheme as this requires additional entries, and very few clients would qualify for this scheme
