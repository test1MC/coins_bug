1 Overview

This guide is intended for anyone with OA Designer skills to create and maintain Desktops. 

This guide is not intended for end-users for which Desktop Wizards are available to build desktops from pre-defined components.  End User documentation is available from the OA Navigation section of the Learning Resources on the Client Area:
(www.coins-global.com)

The guide will work through the creation of an example Desktop. It covers the creation of the Desktop structure and explorers the configuration of the various Tile types that are available.
