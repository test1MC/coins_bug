Task Summary

 

The tasks required to implement alternate line shading in reports and forms are as follows:

    Create report field styles

    Applying to forms (optional)

    Applying to reports (optional)

The tasks are to be performed in the order outlined above. Each task is broken down into a series of steps that will need to be completed.

Some tasks are identified as optional. Whether or not a task is optional will depend on your specific installation.

Please contact support if you require any clarification of tasks contained within this “How-To” guide.
