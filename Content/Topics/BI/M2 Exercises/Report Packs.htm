Report Packs

Exercise 3 – Creating a Report Pack

This exercise creates a simple report pack.

Create the Function

Function Code Initials + Exercise No.e.g. NLM2Exercise3
Function Name Description identifier.e.g. NL M2 EX 3
Function Module Contract Status
Function Catgory Report
Function Context Leave as defaulted from the Function Description.
Function Program wou005

For a report pack we need this function to specify a page that will contain all of the selection criteria needed by EACH of the reports in the pack.  For this example we will be using the input page of the first report as it has all the selections needed.  If however you had multiple reports all needing different selections, you may need to build a bespoke page to be used by the function.

The input page is specified by the parameterstn_code=  followed by the page to be used e.g.:

stn_code=NLM2Exercise1

This is then followed by a comma separated list of the reports that you wish to run prefixed with rtn_code =

e.g.:

rtn_code=report1,report2,report3

We are going to create a report pack for the vendor listing report we created in Day1/2 and the contract report we created earlier so our parameter will be as follows (replace xx with your initials):

Parameters: stn_code=xxM2Exercise1&rtn_code=xxM1Exercise6,xxM2Exercise1
