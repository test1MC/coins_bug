Totals

Press the totals button to automatically add totals to the footer forms.

Select the footer forms that should be created and press OK.  Any measures in the body will be automatically added to the footer with suitable formatting and padding to line up with the body fields.

The Preview button will show the result.

You can add, change or delete fields in the footer forms but if you run the automatic totals option again they will be removed and the standard footer replaced.
