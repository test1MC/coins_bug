Parameters

Parameters are single value variables (such as selection criteria for the report) that are required for the data source to produce data.  They can either be optional or mandatory.  Optional parameters may be left blank and will default to some sensible value.  Mandatory parameters must be completed or no data will be produced.  Mandatory parameters are initially highlighted in red.

Double-click the parameter to launch the parameter dialog.

Fixed

If you this, the parameter is fixed; that is, it cannot be changed at run time.

Preview

The value that is used when you press the preview button to preview the report.

Run Time

The value that is used as at run time. 

Hide

If you this, the run time will be set for the report and the user will not be able to change it.  If this is not ticked, the run time will be shown as a default on the report selection but will be changeable.

When you have updated the mandatory parameters, they will be shown with a green highlight instead of red indicating that they have been set.
