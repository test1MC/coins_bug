17 Method to force the closure of Workflow:

There is a method which can be used within the context of a report or within the Calculation editor if it is applicable for one record.  The syntax for the method is as follows:

method('wfuwfp01.completeWorkFlowForRowid','Rowid','WFName','FromStage','To Stage','Comment');

Example of use:

method('wfuwfp01.completeWorkFlowForRowid','0x0000000000443c85','EX_1','*','END','Forcing the Workflow to close via method calc');
