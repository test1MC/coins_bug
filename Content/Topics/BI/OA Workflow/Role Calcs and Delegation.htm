     However if you add in the following method to the calculation you can get around this.  

If you add the following Method to the calc   then if a delegation is assigned then it will kick in.  In the example the user is hard-coded to Kerbro but could be a variable such as $pob_code:

$thisuser='kerbro';

$UserID=method$('wfuwfp01.returnDelegateUser',thisuser,'',today);

Please note that when you use the generate My roles on the Workflow Delegation Tab  a Role will not be generated for this but it is covered by whatever is set in the ALL   statement.
