Workflow Groups

Workflow groups are simply a means to organise Workflows into business areas, for example Procurement, HR, etc. At a later stage you can specify which roles belong to a group; and if you then assign a Workflow to a group, any member of the group can action it (see Workflow Roles and Users).

Field Description
Group The workflow group code. Use this code to group roles and workflow templates.
Description The description of the workflow group.
