Licensing of Workflow

 

There are three elements to licensing Workflow in :-

WF - Workflow Module

This is the standard module code for workflow. This needs to be at least 1 to allow a user to build and test workflows in the system. This value should reflect all the users who will be using the workflow module. (This is not the number of users who will have responsibilities within a workflow, just those users who will be building, maintaining and monitoring workflows.)

NWF - Named Workflow Users

This is the number of individual users who will be assigned responsibilities within the configured number of workflows.

X-NWF - Number of Live Workflows

This is the number of Live workflows available within an environment. NOTE – a single workflow is company specific, so a workflow, available in more than one company will require a for each company it is to be used in.
