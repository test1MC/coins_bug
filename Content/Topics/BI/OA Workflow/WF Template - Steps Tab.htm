WF Template - Steps Tab

The stages tab shows the workflow in a step by step fashion (starting from the start of the workflow) showing the current stage and the before and after stages

 

The workflow can be navigated to preceding or following stages by following the links in the before and after areas of the screen.

The Stage frame shows the current stage; the label, description, type and role (for appropriate types).  Any of the enabled fields can be updated and the save button will save those changes. 

If you enter a new link label then “Not Found” will be shown.  You can follow the link to this new stage and it will be created when you save it.

If you enter a link to an existing stage then the link to go to that stage will only be shown once the link label is saved.

Blanking out a link (and saving it) will remove the link.

Once the basic outline of the workflow is established the “Stages” tab can be used to fill out the details of the stages e.g. calculations, processes, conditions, durations etc.
