ENTRY$

Purpose:

 

Returns a character string entry from a list based on an integer position.

Syntax:

 

ENTRY$(element,’list’,’delimiter’)

 

element:

An integer value that corresponds to the position of a character string in a list of values.  If the value of the element does not correspond to an entry in the list, an error condition will occur.

 

list:

A list of character strings.  Separate entries with a delimiter – usually a comma and surround the list with single quotes.

 

delimiter:

A delimiter you define for the list.  This allows ENTRY$ to operate on non-comma separated lists.  The delimiter must be a single character.  Surround the delimiter with single quotes.
