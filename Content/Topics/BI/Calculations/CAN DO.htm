CAN-DO

Purpose:

The CAN-DO function tests a value against a comma seperated list of criteria in a similar way to INLIST but will allow the use of wildcards and exclusions. It should be used in the format

Syntax:

CAN-DO(test,comparison)

 

where test is a single value, comma separated list, or wildcards such as ‘*’ for multi-character tests or ‘!’ for exclusions and

comparison is a string, string variable or character field against which the test will be compared

 

Calculation returns 1 if True, 0 if False

 

 

 

A string, string variable or character field against which the test will be compared
