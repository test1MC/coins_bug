IF

Purpose:   Evaluates one of two expressions depending upon the value of a specified condition.
Syntax:   IF(TestA,Operand,TestB,TrueValue,FalseValue)
Parameters: TestA The first numeric value to be compared
Operand: One of the following:
EQ Equal To
NE Not Equal To
LT Less Than
GT Greater Than
LE Less Than or Equal To
GE Greater Than or Equal To)
TestB The second numeric value to be compared
TrueValue If the test is true, the IF statement will return 1, you may specify a value here to be used instead of the default
FalseValue If the test is false, the IF statement will return 0, you may specify a value here to be used instead of the default
