Note : PlotSelectionGenerate

The Plot Selection generate should be used in future as the preference. The old vwbSelectionGenerate only gave you the Plot From/to/Matches. and did not take into account the Developments that you have chosen.

This means that the standard query for House Sales Reports should be

FOR EACH jc_job WHERE jc_job.kco = {kco} {jobSelect},

EACH vp_site OF jc_job WHERE TRUE {vsiSelect},

EACH vp_wbsdef OF vp_site WHERE vp_wbsdef.vwb_sale = TRUE {PlotSelect}

The new PlotSelectionGenerate will work - ONLY when you only select one development - But it then gives you many more options for Plots relevant to that selection.

e.g. If you leave the Development Selection with more than one development…. You will get the following error on the Plot Selection if you select one of the lookups.
