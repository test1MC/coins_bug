ExportType

ExportType=

  • X – XML Output
  • S – Spreadsheet
  • P – PDF

This parameter may be used to restrict the output types for reports. By default OA will generate all three report types. Note this does not change the output format of the file, only which of the output types are created.

 

Examples:

ExportType=S (only the Excel file is generated, no PDF or XML)
ExportType=P,S (Both PDF and Excel are generated, no XML)
