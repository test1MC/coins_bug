Multiple Tabs/Layouts

Currently our example has the costs information appearing on one tab.  It is possible to add multiple tabs to a screen and then to have different tabs grouped together.

Copy the Tab Function

Create multiple versions of the TAB function.   Change the Context of each copy.

xxM3Exercise2T2 - TAB 2
xxM3Exercise2T3 -TAB 3
xxM3Exercise2T4 - TAB 4
xxM3Exercise2T5 - Group 1
xxM3Exercise2T6 - Group 2
