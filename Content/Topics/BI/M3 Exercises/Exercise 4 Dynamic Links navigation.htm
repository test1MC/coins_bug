Dynamic Links/Navigation Buttons/Options Menus

Currently the in-line page has been hard-coded to always look at one contract.  We are now going to change this so that the drill-down will dynamically know from which contract it is being called.

Edit the Function of the Detail Page

We will now add the ROWID of the tables known to the calling page. As we need to tell the Detail Page (Exercise 2 & 3) where it has come from.   We add ROWID's to the parameter of a function.

Function Parameter jc_jobRowId={jc_jobRowId}

 

Update the Query to be Dynamic

 

In the Body Tab on Exercise 2 we need to update the Query to read.

Header Table

jc_job

Body Query

FOR EACH jc_costcode WHERE jc_costcode.kco = {kco}
AND jc_costcode.job_num = '{job_num}'

 

 

Click the Save Button to save the form.

4.3 Running the Function

Calling page now passes rowID of jc_job record to the detail pages.

However, in order to choose another contract, the user must return to the previous screen.

4.4 Add Navigation Buttons to Detail Screen

On the Header Tab of the Summary Page (Exercise 3) - you can enter the function name of the calling procedure (Exercise 1) into the Header Query Field. 

This will add the Navigation buttons to the context section of your page enabling the ability to Navigate to different Contracts without the need to go back to the previous screen. The navigation buttons which are available -  FIRST,NEXT,PREV,LAST.

Add Options Menu to Screen.

Go to Function of the Detail Screen (Exercise 3) - Copy this function as a Menu Function and Add an 'O' to the end of the function Name.

Function

xxM3Exercise3O       (This is an 'O' not a zero)

Name

Can be left or changed to identify as Options Menu

Type

Change to Menu

Module

Leave as Current Module

Category

Change Category to also be Menu.

It will not appear on the page until a program has been added to the menu.

Select the new function via the link and go to Menu Items

 

Add the following standard function to the menu.    %WJCRJOB – (Contract Summary Profit Report).

Now run the function again, and an Option menu should be available on the top right of the screen.
