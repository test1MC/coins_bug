Exercise 12 -In-line Reports

 

Goal : To create a report which will run and list the costs by costcode for one specific contract.

 

Create the Function.

Functionxx M3Exercise12 - (Share the page with the function created in exercise 10)
Parameter stn_code=xxM3Exercise10

 

 

12.2 Create the Report.

Header Tables jc_config,co_config
Body Tables jc_costcode
Query

FOR EACH jc_costcode WHERE jc_costcode.kco= {kco}

AND jc_costcode.job_num = '1000'

 

 

Sort By jcc_cc

12.3 Add the ROWID of header table.

Add the following parameter to the function of the in-line report.

jc_jobRowid={jc_jobRowid}

 

If this is not the first parameter then you must precede the parameter with an &.

 

Add the jc_job Table as a header table.

 

Header tables should include any table used in the calling procedure.  The header tables record for your in-line report should now read

Header Tables jc_config,co_config,jc_job
Query

FOR EACH jc_costcode WHERE jc_costcode.kco= {kco}

AND jc_costcode.job_num = '{job_num}'

 

Add the in-line report to outer-report.
