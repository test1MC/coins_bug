Existing Reports

If you are returning to an existing report to maintain fields, you can access each field individually or add a new field as required.

To Add fields to the Report simply click from the selection of icons on the bottom of the screen.

Choose the Button to choose from a list fields available to appear on the report.

Use the Button to select the appropriate field.

Variable If the field is required to be used in a calculation then a variable can be assigned
Label The Name which will appear on the top of each column on the report.
Width Report Writer will create default column widths, these can be overridden as required by entering a value (in mm). (Note that the fields should not exceed the number of mm assigned in the page layout i.e. Standard A4 Landscape Report Layout is 297mm).
Total

If you require the field to be totalled at the end of each column then the box. In conjunction with this field you can select from the Total Types as to which type of Total you want to see.

Available Options are the Maximum, Minimum, Average, Distinct Values or you can count the number of records by using the Count Option – By default all totals will Add.
Hidden You can now hide a field, so if you require the information from a field (e.g.:- to be used in a calculation) but you do not necessarily want this field to appear on the report then this box.
X & Y These two fields are the axes for the graph. 
Calculation

You can now put complex or simple calculations into fields. Example of a simple calculation would be after assigning two fields to variables a and b – a third field can be added displaying the sum of these two fields by entering a + b into the calculation field.

Format

The format of a field will take on the default format as seen in the Database Enquiry however if you use a calculation a format will have to be defined on the field if the information is required to be standardised on the report. 

Some example formats are:

->>>,>>>,>>9to display monetary values without pence

(>>>,>>>,>>9.99)to display negatives with parentheses

->>9.99%to display values as percentages

 

Always ensure your formats allow for negative values.
