User Defined WebServices

Where a Webservice does not exist that meets your requirements, User Defined Services, as their name implies, allow Web Services to be configured to access data to your specification.

User defined services come in several forms:

  • Data Sets – These allow you full control of the information being retrieved from
  • Pages – Simple maintenance of data within , such as hold codes on invoices.
  • Programs – Calculation programs, such as those used in Workflow, which can be run from Web Services
  • GET/SET – A service which allows you to GET information from any table and SET information in any table (subject to system security and Business Logic rules).
