Roles

Web Services are licensed using Roles.  These may be found within System > Setup> Module Maintenance..

This function contains all the different types of roles associated which includes more than just the web services, it is worth noting that all the web service roles contain the letters ‘ESB’ so can easily be identified. If the role does not exist it will never appear on the user record under the ‘groups’ section, to import a role from Development please see the next section

The Users column will indicate the number of users who may call the service – typically these will be named users.

 

Each role needs to be set up to give access to specific service functions
