Licensing

If you encounter any messages regarding licensing whilst using Web Services, check the following:

Additional Licensing

User Defined Services require an additional element of licensing.

In Module/ Maintenance, the Role %SYESB must be present

 

Within this, the functions %WXSYESB003/004/005 and 007 should be assigned:

 

Once you have checked that these are in place.  Under the Module/ Maintenance Extras Tab, there are four entries that define which user-defined services can be used:

X-ESBDA controls how many datasets can be created within user defined services.
X-ESBPA controls the number of pages that can be built
X-ESBPR controls how many user defined programs
X-ESBTABLES controls how many GET-SET services may be defined.
