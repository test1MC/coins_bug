Semantic Layer - Building a Dataset

Navigate to Collections and click the hyperlink for the required collection.

Click the Datasets Tab

The Main Tab will display any mandatory fields that have been added automatically by the system. These fields are those required by the system as other fields will be dependent upon them.

To add new fields, click . This will display a list of fields that are available from within the Collection for you to include in your dataset.

To build your Dataset select the fields required from the fields available list using right facing arrows to select. Once a field has been moved to the selected column, it can be highlighted and removed again using the left facing arrows.

You can use the filter to find the fields you require.

To change the order of the field in the selected column, highlight a field and use the up and down selectors to reposition the field in the list.

Continue selecting and re-ordering fields until you have all the fields you require and click .

You will be returned to the Main Tab, which will show all the fields currently selected.
