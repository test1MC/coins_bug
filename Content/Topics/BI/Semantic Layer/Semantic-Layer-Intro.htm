Semantic Layer - Introduction

The Semantic Layer provides a means by which users may create Datasets for use in reporting and analytics without the need to write data queries.

The Semantic Layer Datasets are compiled code, which brings performance benefits over the traditional OA Designer datasets. Additional licensing is required to facilitate the code compile - OpenEdge Query Results. Please check that your set up includes this.

 

The Semantic Layer is designed to allow cross-company reporting by default. Therefore, if you wish to filter data by specific companies in your reports, you should always include the company number field in your datasets. This is different to OA Designer Datasets where best practice is to write the dataset queries to restrict data to the currently logged in company.
