Enter and Fix the Gross Profit

Users can determine whether to keep the Profit value fixed. If the Gross profit is set to Fixed then the land value adjusts to keep the Gross Profit at the target (fixed) amount or percentage.

If the Gross Profit is not set to fixed then the profit adjusts to keep the land value fixed.

 

Select the Summary Tab.

Select the Hyperlink into Gross profit

Field

Comments
Profit: The current cost amount as a percentage of the overall appraisal income.
Fix Profit: Whether to keep the Profit value fixed. To fix the profit, this box should be ticked.
Calculation method: % Income,
% Build Cost,
% Total Cost
