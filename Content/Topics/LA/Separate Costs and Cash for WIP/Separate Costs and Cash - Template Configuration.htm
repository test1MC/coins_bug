Separate Costs and Cash - Template Configuration

New fields are available at the top level of Template Configuration. to mark the appraisal as using separate cost flow and cash flow, and to configure the percentage profit contingency recovery and release.

Template Header

Fields

Field
Description

Separate Cost and Cash Flows

If this is ed, appraisals based on this template will have separate cost flow and cash flow tabs.

You can only Separate Cost and Cash Flows if Detailed Cash Flow is also ed.

Use Profit Contingency?

A box to indicate that the profit contingency calculation is to be enabled.

Percentage

The percentage of the calculated profit figure to be held back as contingency. (Only available if the above box has been ed).

Costs Sections for Profit Calculation

A list of top level cost sections to be deducted from plot revenue to obtain the profit figure for the contingency calculation. (Only available if the above box has been ed).

On the template sections for templates flagged as “Separate Cost/Cash”, the Detailed Cash Flow layout has been renamed Detailed Cost Flow.

If Detailed Cost Flow at this Level is ed for the section, three new fields allow you to specify the payment profile for the template section as three percentages which must always add up to 100%. The methods and offsets will apply to the Cost Flow, and the Payment Profile will then be used to map these costs to the Cash Flow.
