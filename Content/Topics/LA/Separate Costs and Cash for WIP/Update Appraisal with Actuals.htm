Update Appraisal with Actuals

The Update Appraisal with Actuals process has been updated as follows:

  • The Cost Flow Detail figures are updated with the actual costs to date, and the Remaining Costs are spread in the Cost Flow in the same way as these figures were previously updated to the Cash Flow Detail (if the Links Configuration Cash Flow Source was set to Costs).

  • The Cash Flow Detail figures are updated with the actual cash movements to date from the Cash Profiles figures (as previously available if the Links Configuration had the Cash Flow Source set as Cash Profiles). Note: this will only be done if Cash Profiles is being used; this is determined by the Cash Profiles parameter CPROFILE.

    The Cash Flow Detail figures for future months are then calculated from the Cost Flow detail figures using the Payment Profile percentages set up against each template section.

    The cash flow figures are adjusted to absorb any difference in the actual cash flows to date and those that would have been forecast based on the actual costs to date in the first two forecast months.
