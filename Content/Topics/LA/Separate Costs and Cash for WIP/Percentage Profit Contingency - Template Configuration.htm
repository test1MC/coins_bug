Percentage Profit Contingency - Template Configuration

New fields are available on the top-level template configuration to configure the percentage profit contingency recovery and release.

Field
Description

Use Profit Contingency?

A box to indicate that the profit contingency calculation is to be enabled.

Percentage

The percentage of the calculated profit figure to be held back as contingency. (Only available if the above box has been ed).

Costs Sections for Profit Calculation

A list of top level cost sections to be deducted from plot revenue to obtain the profit figure for the contingency calculation. (Only available if the above box has been ed).
