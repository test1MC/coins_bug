Regional House Types - Library Regions

Library Regions is a new function that allows you to define a list of regions, which are used when recording regional costs and rates. Each region relates to a company; when you add a new development or appraisal, only regions that belong to the company you are logged in to will be shown.

Module User Profiles

A new option in User Profiles allows you to specify the default region for a user. This region is the default when the user creates a new development.
