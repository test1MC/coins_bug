Regional House Types - Appraisal

If the template has been configured to ‘Use Regional Rates and Library Costs’, the appraisal main tab displays the region the appraisal belongs to. When you add a new appraisal, the region is set to the current region selected on the development.

Appraisal Units

When adding a unit to an appraisal, the default costs pulled down from the library take into account whether regional rates are being used and the selected appraisal region.

In addition, the functionality which highlights if the build cost has been overridden from the standard library value takes into account the regional library cost.

Appraisal Plots

When adding a plot to an appraisal the default costs pulled down from the library take into account whether regional rates are being used and the selected appraisal region.

Appraisal Costs

When adding a new appraisal the default rates for a cost section take into account whether regional rates are being used and the selected appraisal region.

In addition, the functionality which highlights if the rate has been overridden from the template rate takes into account the regional rates.

Update Library Build Cost

The function to update the unit/plot library costs with the current library cost takes into account the regional library costs.

Update Appraisal Rates from Template

The function to update the appraisal rates with the current rates from the template takes into account the regional library costs.

Bids

When adding a bid appraisal, the bid details tab displays the region the appraisal belongs to.

Cost Breakdown Report

On the Cost Breakdown Report, the highlighting of rates which have been amended from the standard rate takes into account the regional library costs.
