Regional House Types - Developments

If the template has been configured to ‘Use Regional Rates and Library Costs’, a new field on the Development Main tab allows you to specify which region the development belongs to. When you add a new development, this defaults to the region specified on your module user profile, and only shows regions related to the company you are logged in to.

Set Rates

The set rates function takes into account whether regional rates are being used. If so, the rate shown and updated is the regional rate for the region currently specified on the development.

Update Development Rates from Template

The ‘Update Development Rates from Template’ function takes into account whether regional rates are being used. If so, the rates to be updated are the regional rates.
