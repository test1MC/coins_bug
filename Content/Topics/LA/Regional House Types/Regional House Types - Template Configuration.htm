Regional House Types - Template Configuration

A new box on the Template Configuration screen allows you to determine whether the regional library cost and template rates functionality is enabled. This allows existing users of to enable this functionality for new developments and appraisals whilst maintaining the integrity of existing appraisals.

If the new Use Regional Rates and Library Costs box is ed, when you maintain the details for a template cost section, the rate fields are hidden as they are no longer applicable.

The Set Rates tab has been enhanced to allow allows you to update the rates for each combination of region and section. A new sortable column at the start of the browse shows the region name. A new named filter allows you to select a region and show the rates for only that region.
