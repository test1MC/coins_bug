Create a new Appraisal

The Appraisals screen shows details of all the appraisals or bids that have been created for a site. You can create as many appraisals for the site as you need.

 

In the Main Workbench, Select the By Development Tab and select the hyper-link for development to work with.

On the next screen, select the Appraisals Tab.

Click

Complete the following details:

Field

Description
New Appraisal Number The appraisal (or bid) code.

This number will automatically default to the next free four-digit number, but this can be overridden.
Do Not Check Out A new appraisal will not be checked out to you if this option is selected.

Tick this if, for example, you are creating a copy of an appraisal and do not need to update it.
Narrative The appraisal narrative text.
The narrative field is designed to hold any additional important information about an appraisal. This is particularly useful for quick reference when more than one appraisal exists for a site.

Click

You are taken into the new Development at Consolidate Appraisal Level.

Click

The new appraisal is now availa ble on the Appraisals tab.
