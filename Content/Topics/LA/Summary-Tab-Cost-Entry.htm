Summary Tab Cost Entry

The Summary tab is a single-screen overview of the costs and units attached to the appraisal. This is also the starting point for entering the initial details of a bid appraisal

This section will show you how to maintain and manage Non Plot Costs.

Non Plot Costs can be recorded at three levels – Cost Section, Cost Sub Section and Detailed Cost:

Costs at Cost Section and Cost Sub Section level can be entered and maintained via the Summary Tab.

Select the Summary Tab.

Select a Hyperlink into the Cost Section

Enter costs using one of the following Cost Details sections:

Field

Content

% Income: The current cost amount as a percentage of the overall appraisal income.
% Build Cost: Update cost by inputting as a percentage of the appraisal build cost?
% Land Value: Update cost by inputting as a percentage of Land Value?
Per Unit SqFt: Update cost by inputting as an amount per square foot?
Cost / Unit: The current cost amount as an amount per unit.
Total Costs: The current total cost.

Click

 

Costs will be generated automatically by the System and may already appear in the Summary Costs.

 

Costs entered at this level will automatically go against the Unallocated Costs Sub Section.

Repeat the above steps until all Costs have been entered at Summary Level.
