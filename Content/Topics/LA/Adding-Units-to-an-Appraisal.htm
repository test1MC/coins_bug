Adding Units to an Appraisal

The Units tab shows a summary of the house types included in the appraisal. Adding Units in this way will allow the user to monitor how a House Type performs.

Select the Units Tab.

Click

 

Enter the following details

 

Field

Description
Code The house type.

House types allow you to record information about the house types you build, and the specifications that belong to each house type. The specifications in turn determine which cost heads are used on plots of that house type.
Unlabelled field The house type name.
Market Mix Information about the mix style of this unit, such as number of bedrooms and the style of the property.
Mix Type Indicates whether the unit is a house or apartment.
Dwelling Type The dwelling type for the house type.
Unit Count The number of units of this type that have been added to the appraisal.
Sellable Whether or not this is a sellable unit type.
SqFt The gross Floor Area of each unit of this type.
Frontage The frontage measurement for each unit of this type.
Density The density rating for units of this type.
Density Rating The density rating to be used when apportioning costs by unit density to this unit type.
Notional Area The notional area for the unit. This can be used to apportion cost recoveries.
Library Build Cost The library build cost of each unit of this type.
Unlabelled field The library build cost of each unit of this type.
Cost Section The cost section on the appraisal cost template against which the library build cost of this unit is to be held.
Selling Price The selling price of each unit of this type.
Unlabelled field The selling price as an amount per square foot.
Create Duplicate Unit Type If the house type already exists within the sector/appraisal this allows you to create another set of mix details for the same house type.
Plot Numbers The plot numbers to be used for this unit type when switching the appraisal into plot specific mode.

This should be a comma-separated list of the alpha-numeric codes to be used for plot numbering.
Notes Any notes and comments for this house type.

Click

Repeat the above steps until all of the Units have been entered.
