Maintaining an Appraisal

Once an Appraisal has been created it can be updated and maintained in its own right. Subsequent Appraisals can be created from new or by copying an existing Appraisal.

By copying an existing Appraisal you are able to maintain “what if” appraisals to monitor their outcome.

 

Click and enter the following detail:

Base Information

Field

Description
Area Net The net size of the land in both imperial and metric measurements.

Cashflow Appraisal Upd Whether or not this is the appraisal to be made available for this site in the Cashflow and Profit Forecasting module.

ROCE Start The month and year that the ROCE (Return on Capital Employed) calculation should begin.
This is usually the same as the first cost.
Land Value How is the Land Value to be funded for this appraisal?
Show Units The number of show units on this site.
Exchange Percentage The percentage of the land price paid at the point of exchange.
This defaults from the setting on the development.
Disable Auto Calculation of Costs Determines whether the automatic recalculation of costs is prevented when changing details in the appraisal.
This is useful when you have updated costs from information in the VAP budget or modules and you do want these costs to be fixed in the appraisal and not recalculated.

 

Schedule Information

Field

Description
Land Completion The month and year of land completion.
Planning Permission The month and year that planning permission was granted.
Construction Start The month and year of construction start.
Build Rate The expected monthly build rate of the site.

This can be calculated using the "Calculate" option.
Construction End The month and year of construction end.
Build Duration The default duration of the construction period (in months).
Calculate This option allows the build rate or month of either construction start or end to be automatically calculated (once units have been added to the site).
First Cost The month and year of site first costs.
Last Cost The month and year of site last costs.
First Sale The month and year of the first sales completion.
Selling Rate The expected monthly selling rate of the site.

This can be calculated using the "Calculate" option.
Last Sale The month and year of the last sales completion.
Sales Duration The default duration of the sales period (in months).
Calculate This option allows the selling rate or month of either the first or last sale to be automatically calculated (once units have been added to the site).

 

Sector Information

Field

Description
Use Sectors Whether or not sectors are in use on this appraisal.
Sectors allow an appraisal to be broken down into different phases or types of development. To add sectors to an appraisal, this field and select the sectors to be included from the list below (if configured). When sectors are added, any existing data at the consolidated appraisal level will be deleted.
Appraisal Sectors The list of sectors which are used on this appraisal.
