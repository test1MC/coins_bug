Create a New Development

WIthin the Workbench, the By Development tab shows a list of developments and a summary of the number of appraisals for that development. You can dig deeper to see a full list of the appraisals.

Adding a Development via the By Development Tab will allow a user to create a Detailed Appraisal. If a User wishes to create Bid (High Level) Appraisals then they should use the By Bid Development Tab.

Navigate to > Main Workbench. Select the By Development Tab

Click

Field

Description
Development The code for this site.
Name Enter site name.
Template

The template to use for this development.

The template defines the calculations that will be used on the development.

The template that is selected by default when you create a new appraisal, and whether you can select a different template, is configured in > Setup > Module Configuration. Templates are configured using > Setup > Template Configuration.

 

Click . You are taken into the new Development.

The Main screen is used to enter the main details for the development. The default values entered here will be copied onto each new appraisal that is created for this site.

Select the Main Tab and Click .

Complete the main details for this development.

Field

Description
Area Gross

The gross area of the land.

You can enter the area in either imperial or metric measures; calculates the other value. The units of measure to use for land area are defined in > Setup > Module Configuration.
Area Net Leave blank as will calculate from Sector Net Areas entered later.
Site Address: The site address of the development.
Postcode The site postcode of the development.
Appraisal Sectors

The sectors to be included by default on all new appraisals created for this site. If you are not sure whether you will use a sector, leave it ticked; only untick sectors if you are sure you will not use them on this development.

 

If you do not intend to use sectors then DO NOT any Sectors.
