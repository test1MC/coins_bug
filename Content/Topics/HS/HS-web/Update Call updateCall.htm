8.2 Update Call (updateCall)

 

A sample XML message to update the details relating to a call into is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<login>

<user>xxxxxx</user>

<password>xxxxxx</password>

<CID>1</CID>

</login>

<call object="hswshcm1" method="updateCall" id="test1234">

<Complaint reference="102672">

<DateLogged>2008-04-17</DateLogged>

<TimeLogged>11:55</TimeLogged>

<LoggedBy>xxxxxx</LoggedBy>

<CallType>PHONE</CallType>

<Employee>xxxxxx</Employee>

<RaisedBy>2925</RaisedBy>

<Issues>

<Issue reference="102882">

   <DateRaised>2008-04-17</DateRaised>

     <TimeRaised>10:50</TimeRaised>

  <LoggedBy>xxxxxx</LoggedBy>

<Category>BATHFIT</Category>

<Link></Link>

<Description>Test issue for the test call changed</Description>

<Subcontractor>lon001</Subcontractor>

<Supplier></Supplier>

<Abstract>Amended Bathroom</Abstract>

<Location>LOUNGE</Location>

<SubLocation></SubLocation>

<Appliance></Appliance>

<NonFaultIssue>N</NonFaultIssue>

</Issue>

<Issue reference="102883">

<DateRaised>2008-04-15</DateRaised>

<TimeRaised>08:52</TimeRaised>

<LoggedBy>xxxxxx</LoggedBy>

<Status>NEW</Status>

<Category>BATHFIT</Category>

<Link></Link>

<Employee>xxxxxx</Employee>

<Description>Test issue 2 for the test call</Description>

<Subcontractor>lon001</Subcontractor>

<Supplier></Supplier>

<Abstract>Test issue 2</Abstract>

<Location>PLOT</Location>

<SubLocation></SubLocation>

<Appliance></Appliance>

<NonFaultIssue>N</NonFaultIssue>

</Issue>

</Issues>

</Complaint>

</call>

</env:Body>

</env:Envelope>

 

If the field is not contained in the XML message then the existing value held in will be retained.

 

If you want to update the issue attached to the call record you will need to supply the unique reference of the issue, in addition to the unique call reference.

 

If the update of the prospect was successful this would return a XML message with the following format:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<call object="hswshcm1" method="updateCall" id=" test1234">

<ComplaintCompany>001</ComplaintCompany>

<ComplaintReference>102672</ComplaintReference>

<returnvalue></returnvalue>

</call>

</env:Body>

</env:Envelope>
