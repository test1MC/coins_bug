Purchaser Status Lookup Data

 

A sample XML message to send a query to to access purchaser status lookup information is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<login>

<user>xxxxxx</user>

<password>xxxxxx</password>

</login>

<call object="hswssys1" method="dbquery"

pcQuery="FOR EACH co_table WHERE co_table.kco = 1 AND co_table.cta_rtype = 'HSPURSTAT'" pcFields="kco,cta_code,cta_desc" />

</env:Body>

</env:Envelope>

 

The above request would retrieve all purchaser status lookup codes and descriptions in company 1.

This would return a XML message with the following format:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<call object="hswssys1" method="dbquery"

pcQuery="FOR EACH co_table WHERE co_table.kco = 1 AND co_table.cta_rtype = 'HSPURSTAT'" pcFields="kco,cta_code,cta_desc">

<record co_tableRowid="0x00091f25">

<kco>1</kco>

<cta_code>FTB</cta_code>

<cta_desc>First Time Buyer</cta_desc>

</record>

<record co_tableRowid="0x00b3fbe4">

<kco>1</kco>

<cta_code>INVEST</cta_code>

<cta_desc>Investor</cta_desc>

</record>

<returnvalue></returnvalue>

</call>

</env:Body>

</env:Envelope>
