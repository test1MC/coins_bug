Prospect Object (hswshvi1)

 

The prospect object allows the adding and updating of a prospect within .  The object hswshvi1 allows access to the following:

 

  • Adding of a new prospect record into
  • Updating of an existing prospect record
  • Retrieval of the current information held against a prospect
  • Adding of an event to a prospect’s history
  • Adding of a development to the prospects required development list
  • Updating of a development on the prospects required development list
  • Deleting of a development to the prospects required development list
