Development Data

 

A sample XML message to send a query to to access development information is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<login>

<user>xxxxxx</user>

<password>xxxxxx</password>

</login>

<call object="hswssys1" method="dbquery" id="dbenq001" pcQuery="FOR EACH vp_site WHERE vp_site.kco = 1,FIRST jc_job OF vp_site" pcFields="kco,vsi_name,job_num,jcl_loc,kco,vsi_add__1,vsi_add__2,vsi_add__3,vsi_add__4,vsi_pcode,vsi_salesinfo,vsi_directions,vsi_phone,vsi_fax,vsi_email,vsi_url,vsi_northing,vsi_easting,vsi_offopen,vsi_sales,vsi_status,vsi_summary,vsi_sprice,vsi_eprice,RW_vsi_caption__1^1,vsi_lastrev" />

</env:Body>

</env:Envelope>

 

The above message would retrieve development information for all developments in company 1.  As mentioned before the query is made against the hswssys1 object with the dbquery method.

The pcQuery argument specifies the query to run in progress syntax.

The pcFields argument lists the fields to return in the output XML.  The list of fields available on the jc_job and vp_site objects is available through the OA database enquiry.

Note in the above example we have requested the value of a development caption field.  The field RW_vsi_caption__1^1 would return the value of the 1st caption in the caption group with a caption group code of 1.
