Fetch Prospect (fetchProspect)

 

A sample XML message to fetch the details relating to a prospect into is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<login>

<user>xxxxxx</user>

<password>xxxxxx</password>

<CID>1</CID>

</login>

<call object="hswshvi1" method="fetchProspect" id="fsffdfds">

<Prospect reference="487">

</Prospect>

</call>

</env:Body>

</env:Envelope>
