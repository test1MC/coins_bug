Part Exchange Data

 

A sample XML message to send a query to to access part exchange information is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<login>

<user>xxxxxx</user>

<password>xxxxxx</password>

</login>

<call object="hswssys1" method="dbquery"

pcQuery="FOR EACH hs_partex WHERE hs_partex.kco = 1" pcFields="kco,hpx_code,hpx_add__1,hpx_add__2,hpx_add__3,hpx_add__4,hpx_pcode,hpx_location,hpx_desc,RW_hpx_caption__4^1,RW_hpx_caption__5^1,job_num,vwb_code" />

</env:Body>

</env:Envelope>

 

The above request would retrieve all part exchange properties in company 1.  Note in the above example we have requested the value of a PX caption field.  The field RW_hpx_caption__4^1 would return the value of the 4th caption in the caption group with a caption group code of 1.
