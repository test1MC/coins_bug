Update Required Development (updateDevelopment)

 

A sample XML message to update an existing required development on a prospect’s  record is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<login>

<user>xxxxxx</user>

<password>xxxxxx</password>

<CID>1</CID>

</login>

<call object="hswshvi1" method="updateDevelopment" id="xxxx">

<Prospect reference="487">

<Development reference="00044207">

<Contract>2006</Contract>

<Plot1>004</Plot1>

<Plot2>005</Plot2>

<Plot3></Plot3>

<HouseType1>A3</HouseType1>

<HouseType2>A</HouseType2>

<HouseType3></HouseType3>

<Comments>Amended comments</Comments>

<NoLongerInterested>no</NoLongerInterested>

</Development>

</Prospect>

</call>

</env:Body>

</env:Envelope>

 

If the update was successful the following message will be returned:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<call object="hswshvi1" method="updateDevelopment" id="xxxx">

<ProspectCompany>001</ProspectCompany>

<ProspectReference>000487</ProspectReference>

<DevelopmentReference>00044207</DevelopmentReference>

<returnvalue></returnvalue>

</call>

</env:Body>

</env:Envelope>
