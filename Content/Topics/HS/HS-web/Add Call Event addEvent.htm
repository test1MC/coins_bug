Add Call Event (addEvent)

 

A sample XML message to add a new event to the call’s history is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<login>

  <user>xxxxxx</user>

  <password>xxxxxx</password>

  <CID>1</CID>

</login>

<call object="hswshcm1" method="addEvent" id="test1234">

<Complaint reference="102670">

<ComplaintEvents>

<Event reference="">

  <Date>2008-04-16</Date>

  <UserCode>ISSUED</UserCode>

  <Comments>Test add event</Comments>

</Event>

</ComplaintEvents>

</Complaint>

</call>

</env:Body>

</env:Envelope>

 

If the add of the call event was successful this would return a XML message with the following format:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<call object="hswshcm1" method="addEvent" id="test1234">

<ComplaintCompany>001</ComplaintCompany>

<ComplaintReference>102670</ComplaintReference>

<EventReference>00048258</EventReference>

<returnvalue></returnvalue>

</call>

</env:Body>

</env:Envelope>

 

The EventReference returned is the unique reference which identifies the event within the selected company.
