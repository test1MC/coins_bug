Delete Required Development (deleteDevelopment)

 

A sample XML message to delete an existing required development on a prospect’s  record is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<login>

<user>xxxxxx</user>

<password>xxxxxx</password>

<CID>1</CID>

</login>

<call object="hswshvi1" method="deleteDevelopment">

<Prospect reference="487">

<Development reference="00044207">

</Development>

</Prospect>

</call>

</env:Body>

</env:Envelope>

 

If the delete was successful the following message will be returned:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<call object="hswshvi1" method="deleteDevelopment">

<ProspectCompany>001</ProspectCompany>

<ProspectReference>000487</ProspectReference>

<DevelopmentReference>00044207</DevelopmentReference>

<returnvalue></returnvalue>

</call>

</env:Body>

</env:Envelope>
