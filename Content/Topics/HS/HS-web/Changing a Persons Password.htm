Changing a Person’s Password

 

As part of the House Sales connection interface you can store a password for a person within the HS module.  This can be used then if you want to allow access to a ‘Client Area’ on your own web site.  Note a person must have a password to be able to login, blank passwords are not allowed and login verification will fail.

 

The password can be updated via the Prospects Workbench person details screen.  See below:

 

 

The web service provided to allow this to be validated also allows the date and time they last logged in to be stored in .  This is displayed next to the password.
