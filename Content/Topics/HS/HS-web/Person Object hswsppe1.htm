Person Object (hswsppe1)

 

The person object (hswsppe1) allows access to the following information:

  • Update of personal details (e.g. name, address, telephone numbers, password details etc).
  • Validating of client login information (email and password).
  • Retrieval of the related links in for a person (e.g. prospects workbench link, their plots purchased etc).
  • Postcode addressing functionality in .

 

In each person has their own individual record (pp_person) which holds their personal details, such as name, address, telephone numbers, password details etc.   This pp_person record can then be linked to a prospect record in the prospect workbench or to one or more plots in the sales workbench.  Note a person can only have one prospect record in a single company, but could be a recorded as a prospect in multiple companies.

 

Each pp_person record in has a unique reference, which is unique to the database.
