2.2 Error Handling

Errors responses appear using standard SOAP fault syntax:

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<env:Fault>

<env:Code>

<env:Value>env:Sender</env:Value>

</env:Code>

<env:Reason>

<env:Text>Context record not found [SY704]</env:Text>

</env:Reason>

</env:Fault>

</env:Body>

</env:Envelope>

If any error occurs, all actions in the XML request are rolled back.
