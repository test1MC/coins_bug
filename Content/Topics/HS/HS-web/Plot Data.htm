Plot Data

 

A sample XML message to send a query to to access plot information is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<login>

<user>xxxxxx</user>

<password>xxxxxx</password>

</login>

<call object="hswssys1" method="dbquery"

pcQuery="FOR EACH vp_wbsdef WHERE vp_wbsdef.kco = 1 AND vp_wbsdef.vwb_sale = TRUE AND vp_wbsdef.job_num BEGINS '2006'" pcFields="kco,job_num,hpg_group,vwb_code,RO_vud_name,RO_vud_catDesc,RO_vud_beds,vwb_relprice,vwb_desc,vwb_floorarea,vwb_showonrel,vwb_ssdate,RW_vwb_caption__1^4,vwb_lastrev" />

</env:Body>

</env:Envelope>

 

The above request would retrieve all sellable plots on development 2006 in company 1.

Note in the above example we have requested the value of a plot caption field.  The field RW_vwb_caption__1^4 would return the value of the 1st caption in the caption group with a caption group code of 4.
