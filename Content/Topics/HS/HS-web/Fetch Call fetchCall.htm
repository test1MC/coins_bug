Fetch Call (fetchCall)

 

A sample XML message to fetch the details relating to a call into is shown below:

 

<?xml version="1.0" encoding="utf-8" ?>

<env:Envelope xmlns:env="http://www.w3.org/2003/05/soap-envelope">

<env:Body>

<login>

<user>xxxxxx</user>

<password>xxxxxx</password>

<CID>1</CID>

</login>

<call object="hswshcm1" method="fetchCall" id="test1234">

<Complaint reference="102672">

</Complaint>

</call>

</env:Body>

</env:Envelope>
