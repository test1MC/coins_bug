Home Builder Improvements

The following changes have been made to improve the user experience for home building clients:

  • The BQ Apply Prices ... options have been removed from the Prices Workbench.
  • The drop-down has been removed from both the Surveyors Plot and Buyers Workbenches.
  • A new box, Lock Budget, has been added to Update VAP Budget on the Development House Type Workbench. If this is ed, the selected budget will be locked at the end of the update process.
  • On the House Type Workbenches and the Surveyors Plot Workbench, additional statuses (Estimated, Free and Not applicable) are now shown.
  • Column sets have been added to the Standard House Types, Development House Types and Development Maintenance screens.
  • In Development House Types, a new Rebuild from Standard House Type option allows both specifications and for the development house type to be rebuilt based on the related standard house type.
  • The Invoice Register menu has been removed from the VAP Input menu, as it is redundant.
  • Two new boxes have been added to the Sales Staff details: Maintained in and Maintained in House Sales. Two new functions, Staff and House Sales Staff, allow menu security to be used to prevent staff of one type from being able to amend the records of staff of a different type. The existing Sales Staff screen continues to show all records, providing a means of updating all records centrally. A new column, Maintained By, shows 'Both', 'Customer Care' or 'Sales'.
