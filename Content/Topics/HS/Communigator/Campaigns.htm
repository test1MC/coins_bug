Creating a Campaign

The starting point for a new campaign in Communigator is a prospect mailshot. The campaign details are published onto the ESB queue; the ESB engine then delivers the messages to Communigator.

To create a campaign

  1. Create a new mail shot.
  2. Set the Destination Media to Communigator.New Mail Shot Screen

  3. Set the selection criteria for the campaign, and click .New Mail Shot – Main Tab

    displays a list of that match the selection criteria you entered.

    New Mail Shot – Selected

    Only with an email address will be included, and only details from the main contact will be used.

    Prospect Event

    You can select from the list of pre-defined events (these are defined in the Communigator parameter PROEVNEW). The selected event will be created on the ’s history so you can see in that the has been included in a new campaign in Communigator.

    Comments

    The comments for the event. These default to ‘Communigator Campaign <CampaignName> - Initiated’.
  7. When you submit the campaign details, the campaign and its contacts are processed and published onto the ESB message queue. These are then sent to Communigator and a campaign is automatically created.
