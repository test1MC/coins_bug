Consent for Direct Marketing - Migration

Data Consent Migrationis a new tool that allows you to migrate data from the existing hard-coded data consent fields to the new configurable consent fields. This needs to be run when the data consent categories have been configured, otherwise people’s exsiting consent will not be taken into account.

For each of the new configurable consent categories you can select the default value for the category. You can also set the value for the source of consent for the consent fields that have been updated.

Against each consent category you can select one of the following values:

  • Unknown (This is the default value).
  • Yes
  • No
  • Do Not Email - This sets the value of the consent category on each person record based on the current value of the ‘Do Not Email’ flag. If the consent category has been mapped to ‘Do Not Email’ then this option is selected by default.
  • Do Not Mail - This sets the value of the consent category on each person record based on the current value of the ‘Do Not Mail’ flag. If the consent category has been mapped to ‘Do Not Mail’ then this option is selected by default.
  • Do Not SMS - This sets the value of the consent category on each person record based on the current value of the ‘Do Not SMS’ flag. If the consent category has been mapped to ‘Do Not SMS’ then this option is selected by default.
  • Consent to Use Details - This sets the value of the consent category on each person record based on the current value of the ‘Consent to Use Details’ flag. If the consent category has been mapped to ‘Consent to Use Details’, then this option is selected by default.
  • Consent for Third Party Use - This sets the value of the consent category on each person record based on the current value of the ‘Consent for Third Party Use’ flag. If the consent category has been mapped to ‘Consent for Third Party Use’ then this option is selected by default.

The consent audit details text will show an entry like the following: “Data consent migration. Consent set to Yes. Source set to Z”.
