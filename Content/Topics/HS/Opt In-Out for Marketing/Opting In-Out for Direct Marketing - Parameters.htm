Consent for Direct Marketing - Parameters

The following new HS parameters affect the data protection fields.

Parameter

Description
DPCONCAT

Whether to use the user-defined data protection consent categories. If this is set to Yes, new consent category fields are shown on the person record.

DPNOTEEW

Whether to display an error or a warning if no data consent notes are entered.

The following new HS portal parameters affect how data protection fields are passed from property portals to (see 1 - Opting In-Out for Direct Marketing - Property Portals).

Parameter

Description

RMCONCAT

A list of codes for the default consent categories for Rightmove leads.

RMCONDPA

A list of codes for the consent categories for Rightmove leads if the DPA flag set.

RMCONSRC

The code for the data consent source for Rightmove leads.

ZLCONCAT

A list of codes for the default consent categories for Zoopla leads.

ZLCONSRC

The code for the data consent source for Zoopla leads.

SNHCONCAT

A list of codes for the default consent categories for Smart New Homes leads.

SNHCONSRC

The code for the data consent source for Smart New Homes leads.

Other Configuration

To allow consent documents to be uploaded to the person record, the person table (pp_person) must be configured in Document Management.
