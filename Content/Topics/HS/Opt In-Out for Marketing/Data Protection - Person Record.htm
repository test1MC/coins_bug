Consent for Direct Marketing - Person Record

If the new HS parameter DPCONCAT is set to Yes, when you update a person record any data consent categories that have been configured are displayed. These allow you to record whether or not consent has been given. The Source field against each category allows you to select one of the configured data consent sources.

You can also enter a note or comment when any of the consent values are updated. The new HS parameter DPNOTENEW determines whether you receive an error or a warning if you update the consent values and do not enter a note.

Each time the value of a consent category or its source is changed; an audit of the change is kept (see 1 - Opting In-Out for Direct Marketing - Auditing).

If the Person table (pp_person) is configured in Document Management, the Data Consent Document button is available.

This allows you to load any document that supports proof of the consent given. Clicking this button opens a desktop frame where you can upload a file. Once uploaded the file name will be shown next to the upload button. You can only upload one document per update.

When the person record is saved or updated person record, the uploaded document will then be loaded against the person record in Document Manager. The consent audit records will also contain a link to the document so it can be viewed from the new auditing tab (see 1 - Opting In-Out for Direct Marketing - Auditing).

Prospects Workbench

The changes to the People Workbench have also been applied to the Prospect Workbench Personal Details tab.

Prospect Summary

The Prospect Summary tab has been amended to reflect the new consent categories. A single field shows a list of the consent categories for which consent has been given.

Prospect Browses by Name

If the new data consent categories are in use, the export and print options in the Prospect Workbench name browses include the value of each consent category for the main contact and second contact.

Other Workbenches

Corresponding changes have been made to other workbenches where appropriate.
