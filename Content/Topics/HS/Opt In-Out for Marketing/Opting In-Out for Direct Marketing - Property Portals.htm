Consent for Direct Marketing - Property Portals

Rightmove

If the new data consent categories are in use, the Rightmove interfaces (FTP and web service) set the categories based on the new portal parameters RMCONCAT and RMCONDPA. The Rightmove feed contains a ‘DPA’ flag which indicates ‘Whether the user wishes for their data to be protected’.

  • If this is set, the consent categories on the PIW record will be set per the list of categories specified in the parameter RMCONDPA.
  • If this is not set, the categories will be set based on the parameter RMCONCAT.

For each consent category set, the source of the consent will be determined by the portal parameter RMCONSRC.

Zoopla

If the new data consent categories are in use, the data consent categories to be set will be based on the new portal parameter ZLCONCAT. For each consent category set, the source of the consent will be determined by the portal parameter ZLCONSRC.

Smart New Homes

If the new data consent categories are in use, the data consent categories to be set will be based on the new portal parameter SNHCONCAT. For each consent category set, the source of the consent will be determined by the portal parameter SNHCONSRC.
