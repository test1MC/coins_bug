Adding a Plot and a Person

 

  • Go to > Plot Add

 

  • Enter the person’s surname and first name, then click the lookup.
  • If this person already exists in the database displays a list; click to select the person from the list.
  • If there is no matching person, displays an input screen; enter the person’s details and click to continue.

  • Select the type of contact – this could be Purchaser, or one of the contact types already set up.

  • Enter the development code or use the lookup to select it.
  • Enter the plot number.
  • If you are adding a new plot, prompts for the address. Enter the address details and the date. (The date is typically the legal completion date, but depends on how is configured; this is used to calculate the warranty expiry date.)
  • Click
  • If you are adding a new plot, displays an alert. Click OK.
  • adds the person to the contacts list or the purchaser list for the plot.
