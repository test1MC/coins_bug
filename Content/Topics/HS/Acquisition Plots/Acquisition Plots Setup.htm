Acquisition Plots Setup

 

House Type

This feature requires a dummy house type code for new plots that are created using the Plot/Person Add function. If a suitable code does not already exist, set one up using House Sales > Plot Sales > Standard House Types.

Although requires you to enter the Market Mix and Floor Area, this information is not used when creating plots using Person/Plot Add, so you can enter any value.

 

Purchaser Status

In House Sales > Prospects > Setup > Purchaser/Prospect Status, add a status code for new purchasers that are added using the Plot/Person Add function.

 

Sales Classification

In House Sales > Plot Sales > Setup > Sales Classification, add a sales classification code for new plots that are created using the Plot/Person Add function.

The Expiry Basis and Warranty Period can be set on the sales classification if the HS parameter SCLSWARR = Yes; otherwise they are taken from the HS parameters WARRDATE and WARRPER. For acquisition plots, calculates the warranty expiry date from the date that is entered when the plot is added.
