House Sales Connection Requirements

 

In order to use the web services you will require the following:

 

  • The URL of the web service. This can be determined using the administration program to inspect the configuration. There will be two parameters WEBROOT and WEBSCRPT. The URL of the web service is these two values combined with /wousoap.p appended. A typical URL would look like:

    http://195.40.14.50/cgi-bin/wsdevrtb.cgi/wousoap.p

 

The IP address and cgi script will be different on a user’s own installation.

 

  • The character set that is in use in the database. This can be found in the same set of settings as the above URL. The setting to read is the internal code page. This will most likely be iso8859-1 but may be utf-8 or another code page.
  • A named user web service .
  • A House Sales Connection enquiry and/or update . You will need the update if you wish to write information back into using the HS connection web services.
  • A login account that has the web services box enabled. Note: You can only enable web services for an account using the web-based OA interface of ; it is not possible from the Plus interface.
  • If you plan to use non House Sales connection methods on the web services you should obtain documentation on the object in question from .
