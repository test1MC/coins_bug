Deleting Data

 

There is no specific method for deleting data – this is achieved using a combination of the other methods:

  • Use set<tablename>query or setRowId$ to select the row to delete.
  • Use the setBufferField method to set a special field called “rsp_action” to the value “D”.
  • Call commitCurrent to delete the row.

As with all operations these calls must be in the same XML message.
