Bespoke Choices Setup

Bespoke choices are choices that the purchaser requests rather than selecting from a pre-defined list.

To set up a bespoke choice

  1. Using the Specification Choices add-type selector, select Bespoke then click .
  2. Enter a code for the bespoke choice. Bespoke choices are associated with standard choice codes to allow consolidated reporting on standard and bespoke choices.
  3. Fill in the remaining fields and save the record. Bespoke choices have a more limited set of fields than standard choices.
  4. In Specification Items, create a specification item for bespoke choices.
  5. the Bespoke Choices box.
  6. In the Bespoke Choices Codes field, enter the codes for the bespoke choices you have set up.
  7. Fill in the remaining fields and save the record.
