HS Choices - Parameters

House Sales Parameters

The following new House Sales parameters affect the choices functionality.

Parameter

Description
CHOUSE

Y = Use customer choices functionality.

N = Use the and functionality.
CHOINCAT

The incentive category to use for non-cash incentive choices.

If this is blank, the option to flag a choice as a non-cash incentive is not available.
CHOCOFF Whether to show an error if a choice is selected after the construction cut-off. If this is set to N, a warning will be shown.
CHODMTN The DM document group for choice thumbnail images. See 1 - Specification Choices.
CHODMGRP The DM document group for choice images. See 1 - Specification Choices.
CHOQUOTE

The types of customer choice to be included on the quotation.

This allows one or more of the following types of choices to be selected:

  • Standard Specification
  • Free Choices
  • Chargeable Choices (Non-Bespoke)
  • Bespoke Choices to be Accepted
  • Bespoke Choices Accepted
  • Home Builder Selected
  • Previous Reservation
CHOSTATE

The types of customer choice to be included on statement.

This allows one or more of the following types of choices to be selected:

  • Standard Specification
  • Free Choices
  • Chargeable Choices (Non-Bespoke)
  • Bespoke Choices to be Accepted
  • Bespoke Choices Accepted
  • Home Builder Selected
  • Previous Reservation
CHOCHGCS

The pricing status for chargeable choices that have already been fitted when a reservation is cancelled:

  • Free of Charge
  • Chargeable Choice

CHOBQVAR

Whether to process choices in the workbench. See 1 - BoQ Variations.

CHOINCAN

Whether to show incomplete customer choices at cancellation.

If this is set to Y, an Incomplete Choices tab is shown when cancelling a reservation. This lists all free and chargeable choices on which the construction status is not complete. See 1 - Customer Choices - Cancelling a Sale.

CHOWFBC

The workflow to launch when a new bespoke choice is added.

CHOWFCO

The workflow to launch when a new choice is selected after the construction cut-off.

VAP Parameters

The following new VP parameter affects the choices functionality.

Parameter

Description
LOCHTYPE

Whether to update locations against a house type. See 1 - House Type Locations.

This allows you to set up specification areas and items so that they only show if they are linked to a location that is valid on the house type; see 1 - Specification Areas and 1 - Specification Items.
