House Type Locations

If the new VAP parameter LOCHTYPE is set to Y, you can maintain against a house type (standard or development specific) the list of locations that are valid for that house type. The locations are selected from the standard list of  Issue Locations.

You can set up specification areas and items so that they only show if they are linked to a location that is valid on the house type; see 1 - Specification Areas and 1 - Specification Items.
