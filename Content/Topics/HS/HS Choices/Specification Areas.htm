Specification Areas

Specification Areas is a new function that allows you to define the top-level areas of a specification. These could be locations, or specific choices such as paint or flooring.

You can set up areas so that they are not shown against plots when they are not appropriate.

Field
Description

Location

Select the location that corresponds to the area. If the new VP parameter LOCHTYPE is set to Y, the area will only be shown for a plot if the corresponding location is valid on the house type.

Minimum Bedrooms

The area will only be shown for a plot if the number of bedrooms on the plot is equal to or greater than this number.

Minimum Bathrooms

The area will only be shown for a plot if the number of bathrooms on the plot is equal to or greater than this number.
