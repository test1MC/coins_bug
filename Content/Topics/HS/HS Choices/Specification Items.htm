Specification Items

Specification Items is a new function that allows you to group the detailed choices within an area. For example, items such as units, worktops, sink and appliances could be set up to provide a logical grouping of the choices offered in a kitchen.

You can set up items so that they are not shown against plots when they are not appropriate. (You can also prevent whole areas being shown; see 1 - Specification Areas.)

Field
Description

Location

Select a location. If the new VP parameter LOCHTYPE is set to Y, the item will only be shown for a plot if the corresponding location is valid on the house type.

Minimum Bedrooms

The item will only be shown for a plot if the number of bedrooms on the plot is equal to or greater than this number.

Minimum Bathrooms

The item will only be shown for a plot if the number of bathrooms on the plot is equal to or greater than this number.

You can specify how the latest date an item can be chosen is calculated:

Field
Description

Cut Off Stage

Select a construction stage.

Cut Off Days

Enter the number of days. The item must be chosen at least this number of days before the construction stage is completed.

If a choice is recorded for the item after the cut-off date, displays a warning or an error (depending on the HS parameter CHOCOFF), and can launch a workflow (specified by the HS parameter CHOWFCO).
