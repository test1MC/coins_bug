Development Specifications

Because different may have different specifications, choices or prices, each can have -specific templates defined.

A new The Specifications tab in Maintenance has three sub-tabs:

  • The Development Templates tab allows development-specific specification templates to be defined. When adding a template, you can choose to copy an existing master template or development template.
  • The House Types tab allow you to override which specification template should be used at house type level. The Prices button allows you to override prices for the selected house type.
  • The Plots tab allows you to override which specification template should be used at plot level. The Prices button allows you to override prices for the selected plot.
