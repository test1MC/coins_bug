Customer Choices - Cancelling a Sale

When cancelling a reservation, if the new choices functionality is in use and the new HS parameter CHOINCAN is set to Y, the new Incomplete Choices tab is shown. This shows all the customer free and chargeable choices that have been selected on which the construction status is not complete.

When the reservation is cancelled, a copy of all the selected purchaser choices is stored against the cancellation.

Any chargeable choices that have been fitted and were selected by the purchaser are retained against the plot so they show as part of the specification of the plot. The pricing status is set to either ‘Free of Charge’ or ‘Chargeable Choice’ (as determined by the new HS parameter CHOCHGCS). The Selected By for these retained choices is set to Previous Reservation. Any other choices selected by the purchaser are removed from the plot. Standard specification choices and Home Builder selected choices are retained against the plot.

The A new Choices tab on the Cancellation Workbench shows any choices stored against the cancellation.

If the cancellation is subsequently undone, the choices are reinstated to the Sales Workbench.
