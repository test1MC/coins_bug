Standard Calendar

A standard Calendar can be set-up, it should reflect the GL periods for which you intend to create any CVR(s).

A Started By and CVR Completed By date can be set for each period.  The dates control a Status indicator on the Contract tab

The Calendar is accessed by selecting the CVR Calendar function in the navigation area.
