Contract List and Copy Standard Calendar

The Workbench lists any contract record that has been assigned a CVR Model in Contract Status.

Click on a contract number to manage any Contract and the contract CVR Calendar.

Workbench is accessed by selecting the CVR Workbench function in the navigation area.

 

Contract 0001 is listed because the SureStart House Builders v1.8 Model has been assigned to it

 

The standard Calendar can be copied to a contract CVR Calendar via the Options Menu

 

 

Each contract must have a Calendar

 

The standard Calendar has been copied to contract 0001

 

 

The contract Calendar can be customised if required (e.g. a period can be removed if a CVR is not required for that period).
