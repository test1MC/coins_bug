House Builders - Overview

The SureStart House Builders is an out-of-the-box Cost Value Reconciliation (CVR) Model developed for the module.  The CVR Model is fully integrated with the database which enables data to be derived from multiple sources including: Contract Status, VAP and House Sales.

The Model can be viewed on-screen or as a PDF report and consists of the following 8 Views (tabs):-

  • Summary
  • Plot Revenue
  • Sundry Revenue
  • Summary
  • Detail
  • Valuation (Time)
  • Valuation (Amount)
  • Valuation (Plot)

As well as deriving data directly from the database some of the Views listed above are enabled for manual input of key data and adjustments (including the facility to adjust cost, revenue and valuation).

Typically run on a monthly basis for nominated contracts as part of a financial/commercial month end process, the SureStart House Builders will streamline and therefore speed-up CVR production.
