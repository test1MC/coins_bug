Configuration Changes

Several existing House Sales maintenance options have been changed to allow you to record data required for NHBC submissions.

Guarantee Schemes

A new column has been added to Guarantee Schemes maintenance.

Field
Description

NHBC Warranty Provider Code

Allows existing codes to be mapped to one of the following valid NHBC Warranty Provider Codes.

  • ‘N’ – NHBC is the warranty provider
  • ‘P’ – Premier Guarantee is the warranty provider
  • ‘O’ – Other provider

This column only appears if the HS Interface parameter NHBCSURL has been populated.

Market Mix Codes

A new column has been added to Market Mix Codes maintenance.

Market Mix Codes Maintenance
Field
Description

NHBC House Type

Allows existing codes to be mapped to a valid NHBC House Type. This column only appears if the HS Interface parameter NHBCSURL has been populated.

Purchaser/Prospect Status

A new column has been added to Purchaser/Prospect Status maintenance.

Purchaser/Prospect Status Maintenance
Field
Description

NHBC Homeowner Type

Allows existing codes to be mapped to a valid NHBC Homeowner Type. This column only appears if the HS Interface parameter NHBCSURL has been populated.

Development Maintenance

New fields have been added to Development Maintenance.

A new NHBC Builders Site Ref (SNIN) field has been added to the ‘Detail’ section of the Details tab. This field is only visible if the HS Interface parameter NHBCSURL has been populated. Only sites that have this field populated will be included in the NHBC submission.

A new Housing Association Contact Email field has been added to the Sales tab. This is a contact email address for the organation that administers any social plots on the . It is one of the optional fields that can be submitted to NHBC.

Standard House Types

New fields have been added to the Details tab of the Standard House Types screen and to the Development House Types screen.

House Types – EPC Fields
  • EPC Energy Efficiency
  • EPC Potential Energy Efficiency
  • EPC Environmental Efficiency
  • EPC Potential Environmental Efficiency

These fields allow a decimal value to be stored for each EPC value, which can then be converted to an A-G rating based on the standard EPC bands:

Band

Rating Points

A

92-100 SAP points (Most efficient)

B

81-91 SAP points

C

69-80 SAP points

D

55-68 SAP points

E

39-54 SAP points

F

21-38 SAP points

G

1-20 SAP points (Least efficient)

Plots

A new Flat Number field has been added to the Details tab of the Plot Summary screen.

The following fields have also been added:

  • EPC Energy Efficiency
  • EPC Potential Energy Efficiency
  • EPC Environmental Efficiency
  • EPC Potential Environmental Efficiency

These are populated from the plot’s house type.

These fields are also available on the Plot tab of the Sales Workbench.
