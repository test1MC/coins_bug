NHBC Data Submission

Home builders who provide home construction warranty and insurance through NHBC need to submit data regarding their reservations, exchanges and completions every month as part of the Insurance Distribution Directive (IDD). NHBC Data Submission is a new function that submits this information the information required under IDD to NHBC using a web service.

NHBC Data Submission Selection Screen
Field
Description

Report Only Mode

This allows you to run the submission in report-only mode so you can check on the data to be sent before the submission is made.

Up to Date

Data that is changed after this date will not be submitted.

Standard report scheduling allows you to set up this process to run at specified times. Standard Development selection allows you to run the submission process for selected sites only.

When run, the report collates information for all relevant changes to selected sites and their plots since the last submission was made. Only sites with a populated NHBC Builders Site Ref/SNIN will be included, and only plots whose sales classification is listed in the new parameter NHBCSCLS will be included (or if the parameter is blank then all plots will be included). If no change has occurred on a plot since the previous submission, it will be excluded from both the report and the submission.

Any records with errors are shown together in a section at the top of the report.

Buildmark Policy Number

Whenever a plot is submitted, NHBC’s Buildmark Policy Number is returned and is stored against the plot (in the Certificate Number field on the Plots tab of the Sales Workbench).
