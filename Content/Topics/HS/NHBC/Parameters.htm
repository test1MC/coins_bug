Parameters

The following HS Interface parameters affect the NHBC data submission functionality.

Parameter

Description
NHBCBRNO The NHBC builder registration number for electronic submission of IDD data.
NHBCHFDS

Optional HUG (Home User Guide) fields to include in NHBC data submissions. You can select one or more of the following HUG fields for submission alongside the mandatory Insurance Distribution Directive (IDD) fields (by default none will be selected).

  • Site Manager Name
  • Site Manager Telephone Number
  • Snag Contact Email
  • HA Snag Contact Email
  • Site Address
  • Local Authority Name
  • Site Latitude
  • Site Longitude
  • Site Status
  • Plot Meter Readings
  • Plot EPC Ratings
NHBCPWRD The password to use for NHBC submissions.
NHBCSCLS A list of sales classifications of plots to be included in the NHBC submission. If the parameter is blank, all plots will be included.
NHBCSUBN Whether to submit data about non-NHBC plots (Y/N). If Y, information about plots marked as having a warranty provider other than NHBC will be included in the submission.
NHBCSURL The URL for the NHBC submission portal (https://online.nhbc.co.uk/hugadmin/).
NHBCTCHG Whether the NHBC submission functionality is in use. If this is set to Y, tracks changes to IDD data.
NHBCUSER The user name to use for NHBC submissions.
