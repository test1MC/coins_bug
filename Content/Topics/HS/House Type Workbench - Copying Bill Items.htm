Copying Items

Within the workbench you have the ability to copy the items, this is useful when you have one or more house types that have the same baseline resources that should be allocated.

Instead of having to recreate the house types you can copy all house type items between standard house types OR if you have specifically used a development house type on one site you are able to convert this to a standard house type.

You must select the source type and source house type, this will give you the ability to customise the copying by cost head/payment stages and then whether to include the estimating sections or variants if required.

Main Tab

 

When copying items between house types or standard house types, you can now choose whether to copy the supplier, price, and discount fields, and the 'Ready to Use' flag. By default these are copied.

The Copy Items screens on the Development House Type Workbench and the Standard House Type Workbench have four tickboxes:

Field

Description

Copy Supplier

Whether to copy the supplier to the new bill items. If this is not ticked, the supplier and office fields on the new items are blank.

Copy Price

Whether to copy the price and pricing status to the new bill items. If this is not ticked, the price on the new items is set to zero, and the pricing status is set to 'Unpriced'.

Copy Discount

Whether to copy the discount fields to the new bill items. If this is not ticked, the discount fields on the new items are set to zero.

Copy Ready to Use

Whether to copy the 'Ready to Use' flag to the new bill items.

 

Packages Selection Tab

The ability to copy items exists in all levels of the process. This means you are able to copy items from standard house type, development house type and plot specific levels. Once you click it will create a report based on the selection.
