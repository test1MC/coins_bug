Deleting (Bulk)

Over time, home builders using HB CRM will build up a database of many and people records. Under GDPR you should only retain records where there is a legitimate reason. There may be no reason to retain old records, so you can use Bulk Delete Prospects to delete and people records on which there has been no recent activity.

  1. Go to Bulk Delete Prospects.Bulk Delete Selection Screen

  2. Fill in the fields:
    Field
    Description

    Companies

    Enter a can-do list of to process, or leave blank to process only the current .

    Initial Event Date Range

    Enter a range of dates; only on which the first event recorded was in this date range will be selected. If this is left blank then no check on the initial event date will be done.

    Number of Months Since Last Activity

    Only on which the last event was more than this number of months in the past will be selected.

    Report Mode
    this box to produce a report showing which records will be deleted, without deleting any records. Clear the box to delete the records.

The report will show all matching the selection criteria. Only on which the associated people have no links to any other records in the associated will be included.

If report mode has not been selected, the records will be deleted. If the associated people also have no other links to any other record in any , the people information workbench records will also be deleted.
