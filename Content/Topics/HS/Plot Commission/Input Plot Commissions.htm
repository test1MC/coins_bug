Input Plot Commissions

 

Navigate to the House sales Module. Select Plot Sales > Plot Commission Input .

Select the Development that is to receive the Commissions.

The Exclude Affordable Housing Plots will be selected by default, remove if not applicable

Click

Commissions can be set up by individual plot using the Open button by each plot

or

Select multiple rows and click the multi-update button.

The system will open the input screen where you can input commission for up to four Sales Staff.

Field

Description
Sales Staff Select the sales staff person who is to be paid the commission.Select <Reservation Sales Advisor> to use the sales advisor from the reservation.
Commission Rate The percentage of the net sales price that is to be paid as commission.
Commission Value An additional lump sum that is to be paid as commission.  This is added on to the comission rate to give the total commission payable
Override % Paid on Exchange Select if you want to override the percentage of the commission that is to be paid on exchange of contracts.
Unlabelled field The percentage of the commission to be paid on exchange of contracts.
