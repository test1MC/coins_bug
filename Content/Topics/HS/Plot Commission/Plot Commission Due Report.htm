Plot Commission Due Report

 

Plot Commission Due Report allows you to report on the plot commissions that are due to be paid.

The plot commissions to be paid are entered using the function House Sales > Plot Sales > Plot Commission Input.

 

Navigate to the House sales Module. Select Plot Sales > Reports > Plot Commission Due Report.

Main Tab

Field

Description
Date The range of dates to report on.  Plots that have exchanged or legally completed in this range will be included in the report..

The selection of exchanged plots depends on the setting of H/S parameter COMEXCH%.
Sales Advisors Select the sales advisors you wish to report on.
Unlabelled field Select the sales advisors you wish to report on.
Range Select a can-do list of parameters you wish to copy.
Page Per Advisor Whether to start a new page of the report for each advisor.

 

Development Tab

Development Selection allows you to choose which developments to include.

Field

Description
Development Enter the code of the contract in the Contract Status module.
Group Select the contract group.
Type Select the contract type.
Location Select the contract location.
Manager Select the contract manager.
Customer Select the customer on the contract.
Phase Enter the phase code, or use the lookup to select it.
Site Status The current status of this development.
Status of Contract The first analysis set.
Active Whether the contract is active or inactive. If a user accesses an inactive contract, displays a warning message.
Complete Whether the contract is complete or not. does not allow transactions to be posted to a complete contract. This is also used in selecting contracts for reporting purposes.
Sales Status Select whether you wish to include inactive, active and completed developments.

 

Once you have made your selections, click to generate the report.
