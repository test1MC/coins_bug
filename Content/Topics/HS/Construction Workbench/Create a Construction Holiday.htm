Create a Construction Holiday

From the House Sales Module select Plot Sales .

Select Setup. 

Select Construction Holidays.

Select the Add button.

Enter both the Date of the Holiday and also a description for the holiday as below:

Select the Save Button.

 

Continue to enter all Construction Holidays.

 

Remember to use the Copy button for holidays with the same description, such as Christmas Holiday.

 

Are you using a 5 day Construction Week?  If so remember to set Construction Holiday stand down as a 5 day week.

Avoid standard Bank Holidays such as Easter and May Day.  These will push your programme out by a day.  Especially if you prefer a build Stage to end on a Friday or are using a 5 day construction week.
