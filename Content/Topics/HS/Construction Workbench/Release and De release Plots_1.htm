Release and De-release Plots for Build

From the House Sales Module select Plot Sales.

Select Construction Workbench. Navigate into the required Development.

 

Build Release Plots

Select the Plots Tab.

 

‘%’  Note – check that your forecast has been set to the Current Forecast (Client Terminology Specific) Forecast.

Select the Plots that are to be released using the Tick selector for the row:

From the Actions Dropdown select Build Release then select Apply Action .  Select Stage Released To . Select on the Next button.

 

Select the Stage that the selected plots are released to, comments will be populated by the System but the User can add additional comments if required:

Select on the Next button.

Plots can be released to any stage, build progress can only be recorded against a released stage:

 

Build Derelease Plots

Select the Plots Tab.

 

Check that your forecast has been set to the Current Forecast (Client Terminology Specific) Forecast.
