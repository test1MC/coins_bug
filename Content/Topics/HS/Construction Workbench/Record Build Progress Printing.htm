Record Build Progress – Progress Tick Sheet

Select the Progress Tick Sheet Tab.

 

Only Plots that have been Build Released will appear on the Progress Tick Sheet Tab.

 

Click on the Open folder.

 

Ensure that you are recording against the correct date, then enter the last stage that has been completed for all plots by ticking each stage:

Click on the Save button:

 

Parameters can be set to auto complete previous unticked build stages for Construction Types (e.g. excluding Timber frame).

Once a has been entered it will remain on the Tick Sheet input and Report unless deleted from a Plots history.

 

 

Return to the Plots tab and the Current Stage is recorded against each plot.

From the plots tab you can see if a Plot is on forecast by the code:
Red-behind Baseline forecast.
Amber-within 5 days of baseline forecast
Green-in line with Baseline forecast
