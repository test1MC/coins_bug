Lock/Unlock the Forecast

 

Lock Forecasts allows you to lock the sales forecasts or construction forecasts for selected developments, to prevent them being updated or copied over.

From the House Sales Module select Forecasting > Lock Forecasts.

Lock a Forecast

Set the Forecast that is to be locked using the Forecast filter at the bottom of the screen:

Select the Apply Filter button.

Select the Development Row.

Select the Lock Construction Forecast action

Click the Apply Action button.

 

Enter any comments:

Click   Your Construction Forecast has now been locked.

Unlock a Forecast

Set the Forecast that is to be unlocked using the Forecast filter at the bottom of the screen:

Select the Apply Filter button.

Select the Development Row.

Select the Lock Construction Forecast action

Click the Apply Action button.

Enter any comments:

Click   Your Construction Forecast has now been unlocked.
