Release and De-release Plots for Build

 

The Progress of a Build can be updated and reported through the Construction Workbench.  But before you are able to record progress you must release a Plot for Build.  Plots can be build released to any stage.

 

Build Release Plots

Select the Plots Tab.

Check that your forecast has been set to the Current Forecast (Client Terminology Specific) Forecast.

Select the Plots that are to be released using the Tick selector for the row:

From the Actions Dropdown select Build Release then select Apply Action . 

Select the Stage Released To using the drop-down list

Comments will be populated by the System but the User can add additional comments if required:

Plots can be released to any stage, build progress can only be recorded against a released stage:
