Maintain and Edit the Construction Programme Stages – Multi Update Stages

The User can also maintain multiple stages that have the same build duration.  Select multiple rows as below:

 

Select the Multi Update Button.

Enter either the Length in Weeks, Days or a combination of both. If the Stage is not applicable to your Build Programme then place a in the Excluded column.

Select the Save Button.

‘%’  Note – These days are cumulative.

‘%’  Note – The System works on 5 working days, so when using a combination of Length (Weeks) and Length (Days), 4 weeks and 3 days equates to 23 days.

Continue to update the length / exclude for the remaining build stages.

Select the Next button to return to the Construction Programme list.
