The Programme allows for parallel stages or concurrent activities such as Internal maintain multiple stages that have the same build duration.  Select multiple rows as below:

. Open each stage using the button and select the Parallel Dropdown. If this stage runs in parallel to the main set of critical path stages then select the set to which this belongs:

 

Set 1 will show as a second row of stages on the Gantt Chart.  Set 2 will show as a third row of stages on the Gantt Chart.
