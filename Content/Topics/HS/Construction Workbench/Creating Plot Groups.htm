Creating Plot Groups

Select the hyperlink into the development.

Select the Groups Tab

Click and enter the Plot Groups

Field Description
Group: The code for the construction stage.
Description: A description of the construction stage.

Select the Save Button.  Continue to enter all Plot Groups.

‘%’  Note – The System will continue to open new lines for the User to enter.  Once the User has finished adding Groups click undo.

 

If you set up a plot group with a blank code to represent generic plots, then groups for each Group of Apartments, Terraces etc; when you add a new plot, by default it will belong to the blank (generic) group, and you only need to change it if the plot is part of a block of apartments.
