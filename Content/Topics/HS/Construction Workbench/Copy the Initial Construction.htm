Copy the Initial Construction Forecast

From the House Sales Module select Plot Sales.

Select Construction Workbench.

 

Copying the Construction Forecast

Select the hyperlink into the Development.

 

The User is always taken to the FC01 – Current Forecast, remember to check the Forecast you are working on.

From the Options menu select Revised Budget.

From the Options dropdown select Copy Forecast:

 

From the Target Forecast dropdown menu select FC01 – Current Forecast and then select the Next button.

To confirm that the Current Forecast now exists select the Hyperlink into the Gantt Chart:

Your forecast has now been copied.

 

Comparing Baseline Build forecast (BD02 – Revised budget) to the Working Forecast (FC01 – Current Forecast)

The Working (FC01 – Current Forecast) is where the User will record actual build as well as unexpected delays or advancement in programme.  Changes to the baseline can be easily identified as the build progresses.

Select the hyperlink into the Development. Select the Plots Tab.

 

Select the Plots Tab.

Delays in the programme are benchmarked against the baseline forecast and indicates in Amber or read.
