Delay the Project start or Finish

The Project Start or Finish Date can be changed for the whole programme rather than the user changing each Plot programme start or finish date.

Delay the Project Start Date

Gantt Chart Functions are available at the top of the screen as indicated below:

Select the Matrix actions button, then select the Delay Project Start option:

Select how many days to delay and if you wish to retain any existing gaps or durations that may have already been entered into the programme:

 

Select OK to confirm the change:

 

The complete programme has moved by the number of days set.
