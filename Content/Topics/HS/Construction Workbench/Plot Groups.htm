Plot Groups

The Development Maintenance - Plot Groups tab allows you to maintain the plot groups on this development.  Plot groups allow you to group plots together for reporting purposes, for example, to group all the plots in a block of flats together.

Set up plot groups for the development, then indicate which group a plot belongs to using the Plot Group field when updating a plot.

Not all plots need to belong to a group; for example, if you are building houses and blocks of flats on the same development, the house plots might not belong to a group.

Plot Groups are Development specific.
