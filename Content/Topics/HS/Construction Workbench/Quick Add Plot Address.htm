Quick Add Plot Address

A Development can have a number of plots. Users will need to add an address for each plot within a Development. If there are many number of plots this can be extremely time consuming as you will need to add the plot address individually for each plot. There is a quick add address option within which allows you to add an address for multiple plots.

Using the filter option type in the development Code/Name and click Apply Filter

You will need to change the search option to Development Name to search for the Development name.
You will need to change the search option to Development to search for the Development Code

Once you have found the development click on the hyperlink to go into the development.

Once you navigated into the development click on the Plots tab

You will now have a list of all the plots within the development. Using the option, add in a tick to all the plots you would like to add the address for.

To Select multiple plots hold the CTRL button and put a ticket on the plots you want to add the address in for
If you want to select all the plots within the development click on the SHOW  ALL button. Holding down the Shift key on your keyboard mark a on the first plot. Scroll down to the last plot and tick the last plot. This will select all the plots in-between.

Once you have selected the plots you want to add/change the address for, select the “Quick add address” option using the drop-down arrow.

 

Once you have selected this click

Under the “Address Details” section type in the address and postcode which will be applied to all the selected plots.

Under the “Plot House Number” type in the house number for each plot.

Click to apply the changes.

All the plot addresses have now been changed. You can check by going into the Development > Clicking on the plot hyperlink and checking the plot address.
