Record Build Progress – Whiteboard

Select the Whiteboard Tab.

Whiteboard provides an overview of the forecast dates for each construction stage.  From here you can also quickly update the actual progress by ticking the completed stages.

If the actual progress was late (as compared with the selected forecast) then it will be coloured red.  If it was on time or ahead of schedule it will be coloured green.

You can also highlight stages in orange which are not complete but are due or should have been completed in a certain number of days.  To do this add the attribute &WarnDays=X to the function parameter.  X should be replaced by the number of days warning.  Note a negative of number of days will warn ahead.

 

Only Plots that have been Build Released will appear on the Progress Tick Sheet Tab.
