Assigning Plots to Plot Groups

From the House Sales Module select Plot Sales.

Select Development Maintenance.

Select the hyperlink into the Development.

Select the Plots Tab.

 

Select the Open folder   into the Plot.

 

‘%’  Note – If you have set up a plot group with a blank code to represent house plots, then groups for each set of flats; then when you add a new plot, by default it will belong to the blank (generic) group, and you only need to change it if the plot belongs to an actual Plot Group.

 

Field Content Comments Buttons
Plot Group: Select The plot group that this plot belongs to within the selected development.  You can use plot groups, for example, to indicate which block a flat belongs to.

Select the Save Button.  Continue to update all Plots.

 

Bulk Assigning Plots to a Group

Select the hyperlink into the Development.

Select the Plots Tab.

Select Multiple Plots as below:

 

Select the Multi Update button.  Enter the following information:

 

Field Content Comments Buttons
Plot Group: Select

 

The plot group that this plot belongs to within the selected development.  You can use plot groups, for example, to indicate which block a flat belongs to.

Select the Save Button.

 

‘%’  Note – You may be required to “un” fields, the System will prompt you.

‘%’  Note – Use the Multi Update function for two or more plots to save time.
