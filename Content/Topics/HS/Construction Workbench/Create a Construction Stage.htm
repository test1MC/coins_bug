Create a Construction Stage

From the House Sales Module select Plot Sales .

Select Setup. 

Select Construction Stages.

 

Select the Add button. Enter the following information:

Table 1

Field

Content

Comments

 
Stage: Lookup The code for the construction stage.

Full Description: Free text A description of the construction stage.  
Short Description: Free text A short description of the construction stage.

 
Build Stages: Lookup A list of build stages that are linked to this construction stage

Equiv Unit %: percentage The equivalent unit expressed as a percentage.  Note 100% would represent a fully completed unit, 50% a half completed unit.

 
Colour: HTML Code An HTML code to be used on reports and Gantt charts for the construction stage. All stages have the stage code printed on them in Black - the selection of Black as stage colour is, therefore, not recommended.

 

Select the Save Button.

Continue to enter all Construction Stages.

 

‘%’  Note – We have used a number code for each Stage.

‘%’  Note – When using a Number Stage you could leave gaps, e.g. 10 20 30, so that you can insert stages at a later date.

‘%’  Note – The Sequence indicates the order in which the Build Stages take place.

‘%’  Note – You can use the Insert button to manipulate the Sequence of your Build Stages.

‘%’  Note – You can move your Build Stage up using the Move Up button  or the Move Down button.

‘%’  Note – Timber Frames – Because Timber frames are typically built out of sequence, there is a parameter that can be set to allow out of sequence.  You may also want to set up stages that are unique to timber frames, for exaple Frame Complete and Exterior Covering Complete.

 

‘%’  Note – Hexadecimal codes are written using the format #aabbcc, where aa, bb and cc are hexadecimal (base 16; 0 to 255) values that represent the amount of red, green and blue in the colour. For example:

  • #00FF00 is pure green (0 red, 255 green, 0 blue)
  • #808080 is a mid grey (128 red, 128 green, 128 blue)
  • #310073 is a dark purple (49 red, 0 green, 115 blue)
