Sales Kanban

A new button on the Sales Workbench opens the Sales Kanban for the in a new tab or window.

Sales Workbench

You can also open it from a tile on the Sales Desktop.

The Sales Kanban board shows a column for each item set up in Sales Kanban. Plots are shown under the last column where the sales event has been recorded.

Sales Kanban Board

Each card shows details of the plot: the plot number, the house type (if not yet reserved) or the purchasers (if reserved or a later status), and the current sales status and the date it was changed.

Kanban Card

A tooltip shows more detail.

Kanban Card Tooltip

Icons on each card allow you to run various actions for the plot.

Icon

Action
Display the main summary page of the Sales Workbench for the plot.
Generate a mail merge letter. When you click this icon, you can select a Word mail merge letter to generate.
Send a text/SMS. When you click this icon, you can select a text message template. You can then amend the contents of the text message.
Send an email. When you click this icon, you can select an email message template.
Record sales event. When you click this icon, you can select a sales event to run.

You can drag and drop plots between columns. If a drop action has been configured, you are prompted to complete the drop action.

Drop Action (When Dropping a Plot on the Exchanged Column)

A blue edge to a card shows that you have changed it.

You can filter the board to show only plots whose description (number, house type or purchaser name) matches what you type. You can collapse columns by clicking the arrow button in the column header, to give you more space, and you can scroll up and down individual columns separately.

Sales Kanban with Collapsed Columns
