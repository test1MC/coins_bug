Sales Kanban Setup

Sales Kanban is a new function that allows you to set up the columns for the Sales Kanban view.

Sales Kanban Setup Browse

For each column, you specify:

  • A label for the column.
  • The plot status. This determines which plots are shown under the column. This can be a sales event or a system event, or RELEASED or UNRELEASED. If the status is an event, plots on which that event was the last event to be run are shown in the column; If the status is RELEASED or UNRELEASED, plots with that status are shown in the column.
  • Optionally, a drop action. This is the action that a user is prompted to run when they drag and drop a plot into the column. The drop actions are:
    • No action - users cannot drop a plot into the column.
    • Sales event - run the sales event associated with the column.
    • Email - send an email; you can specify a list of email templates that the user can choose from.
    • Letter - generate a mail-merge letter; you can specify a list of letter templates that the user can choose from.
    • Email/Letter - send an email if the purchaser has an email address, otherwise generate a letter.
Sales Kanban Update

When you have updated the Kanban columns, you need to run Initiale Sales Kanban. This is a new function that rebuilds the sales Kanban summary records, which store the column a plot is shown under; these are needed for performance reasons.
