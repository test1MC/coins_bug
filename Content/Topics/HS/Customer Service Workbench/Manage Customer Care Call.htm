Manage Customer Care Call Types

Call Types allows you to set up the codes you will use to identify different types of Customer Care call.

Adding Customer Care Call Types

Navigate to the Customer Care Module.

Select Customer Care Setup Folder

Select the Call Types .

Adding a Customer Care Call Type

Click . 

Enter the following information:

Field

Content

Comments

Code:

Numerical

Enter a numerical code for the Call Type.

Description:

Free text

Enter the full description of the Sub-location (these will be listed Alphabetically when adding a call).

SLA:

Numerical

The target Service Level Agreement response time in days for calls of this type.
When you enter a task, can fill in the default time and date for completion of the task, based on the SLA on the priority of the associated issue.

Max Calls:

Numerical

Maximum number of calls you allow of this type.

Retired:

Tick

Call Types are never removed or deleted as they may have already been used.  We therefore mark them as retired so that they no longer are available when adding a call.

Click

The system will prompt you to add more Call types, click when you are finished.

You are also able to assign a Target Service Level Agreement (SLA). The target Service Level Agreement response time in days for each call type. When you enter a task, can fill in the default time and date for completion of the task, based on the SLA on the priority of the associated issue.
