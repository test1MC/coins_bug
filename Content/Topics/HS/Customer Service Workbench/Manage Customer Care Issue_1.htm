Manage Customer Care Issue Priority Codes

Issue Priority Codes allows you to set up the codes you will use to classify customer care issues by priority, and to set the target response time for issues.

Adding Customer Care Issue Priority Codes

Navigate to the Customer Care Module.

Select Customer Care Setup Folder.

Select the Issue Priority Codes .

 

 

Click on the Add button.  Enter the following information:

Field

Content

Comments

Code:

Free text

Enter a short code.

Description:

Free text

Enter a description to explain what the code means.

SLA:

Numerical

The target Service Level Agreement response time in days for issues with this priority code.

 

Click

The system will prompt you to add more Priority Codes, click when you are finished.

The SLA will be highlighted on the Customer Services Scheduler and the User will also receive warnings if allocating a Task which falls outside the target SLA
