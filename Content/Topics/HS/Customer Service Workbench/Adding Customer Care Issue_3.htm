11.1.1 Adding a Customer Care Issue Sub-location

Click on the Add button.  Enter the following information:

Table 9

Field

Content

Comments

Buttons

Location:

Select

Select the Location from the drop down list.

Sub-location:

Free text

Enter a short code for the Sub-location e.g. UT-5 for Utility Room fifth Sub-location.

 

Description:

Free text

Enter the full description of the Sub-location.

 

 

Click the Save button.

 

The system will prompt you to add more Sub-locations, click on the Undo button when you are finished.
