5.1 Adding Customer Service Call Types

Navigate to the Customer Service Module.

Select Customer Service Setup Folder.

Select the Call Types .

5.1.1 Adding a Customer Service Call Type

Click on the Add button.  Enter the following information:

Table 3

Field

Content

Comments

Buttons

Code:

Numerical

Enter a numerical code for the Call Type.

Description:

Free text

Enter the full description of the Sub-location (these will be listed Alphabetically when adding a call).

 

SLA:

Numerical

The target Service Level Agreement response time in days for calls of this type

 

Max Calls:

Numerical

Maximum number of calls you allow of this type.

 

Retired:

Tick

Call Types are never removed or deleted as they may have already been used.  We therefore mark them as retired so that they no longer are available when adding a call.

   or 

 

Click the Save button.

 

The system will prompt you to add more Call types, click on the Undo button when you are finished.

 

‘%’  Note – You are also able to assign a Target Service Level Agreement (SLA). The target Service Level Agreement response time in days for each call type. When you enter a task, can fill in the default time and date for completion of the task, based on the SLA on the priority of the associated issue.
