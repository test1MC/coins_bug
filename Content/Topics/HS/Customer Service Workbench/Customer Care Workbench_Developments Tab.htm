Workbench - Developments Tab

The developments tab lists the developments you have access to in the House Sales database.  There is a link to the plots on each development; from each plot record you can open the Customer Care plot details.

The development list can be filtered to include/exclude inactive, active and completed developments. Select the Filter drop-down and select Active, Inactive or Complete from the list and then either press Return or select the Apply Filter button.

To view the plot’s summary details select the development number for the site; this will display the development’s plots.
