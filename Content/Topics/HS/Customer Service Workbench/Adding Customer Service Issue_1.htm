9.1 Adding Customer Service Issue Priority Codes

Navigate to the Customer Service Module.

Select Customer Service Setup Folder.

Select the Issue Priority Codes .

9.1.1 Adding a Customer Service Priority Code

Click on the Add button.  Enter the following information:

Table 7

Field

Content

Comments

Buttons

Code:

Free text

Enter a short code.

 

Description:

Free text

Enter a description to explain what the code means.

 

SLA:

Numerical

The target Service Level Agreement response time in days for issues with this priority code.

 

 

Click the Save button.

 

 

 

 

 

 

The system will prompt you to add more Priority Codes, click on the Undo button when you are finished.

‘%’  Note – The SLA will be highlighted on the Customer Services Scheduler and the User will also receive arnings if allocating a Task which falls outside the target SLA.
