12.1 Adding Customer Service Mail Merge Letters

Navigate to the Customer Service Module.

Select Customer Service Setup Folder.

Select the Letter Configuration .

You must Change the Selectors Letter Type to Customer care and Apply the filter.

 

 

‘%’  Note – Mail Merge letters are set up in the same area for a number of different letter types (Prospect, Purchaser, Customer Service etc).  So it is important you filter the page to Customer Care as below, else you will not be able to see or use your letters.
