Adding Timeslots

Navigate to the Module.

Select the Setup Folder.

Select Timeslots.

Adding Timeslots

Click on the button.  Enter the following information:

Table 15

Field

Content

Comments

Buttons

Time Slot:

Free text

Enter a description of the time slot, which is usually AM or PM

 

Start Time:

Time HH:MM

Enter the start time of the time slot.

 

 

End Time:

Time HH:MM

Enter the end time of the time slot.

 

 

Click the button.

The system will prompt you to add more Timeslots; click on the button when you are finished.
