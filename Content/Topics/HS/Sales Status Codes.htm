Sales Status Codes

 

Sales Status Codes allows you to set up and maintain a list of codes that can be used to identify the sales status of a plot or part-exchange. For example: Unreleased, Released, Reserved, Exchanged and Completed.

You also need to fill in the parameters with the codes that represent certain specific statuses.

 

Navigate to House Sales > Plot Sales > Setup > Sales Status Codes

Click

Field

Description
Code The code for the sales status.
Description Enter a brief description for the code.
