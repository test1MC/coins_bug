Configurable House Sales Receipts - Overview

The new configurable receipt print allows you to print details of cash, and post-dated (PDCs) received from HS purchasers through House Sales and .

The configured documents can be run from two places:

  • From the Options menu on the Sales Workbench. This allows you to print a receipt for a single plot. You can select the document to be used and a date range for the transactions to be included.
  • From a new function on the Financials > Cash menu in House Sales called Receipt Print.

To define the layout for printing receipts

  1. Define the layout of receipts using Document Designer.

  2. Use the document type Receipt Print (HS). See Configurable Documents in the online documentation for general information on how to configure documents for printing.

    The receipt has three BODY sections, which show:

    • BODY1 - HS cash receipts
    • BODY2 - HS receipts
    • BODY3 - CB PDCs
  3. On the Extras tab, the appropriate boxes to select which sections to include.

    Cash and transactions are distinguished from each other by their HS transaction type. If you are including these sections, enter comma-separated lists of transaction types that identify them.

  4. You can configure to include an event automatically in the sales history when the receipt is printed.  Enter the event code and any comments.
