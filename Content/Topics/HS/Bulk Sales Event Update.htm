Bulk Sales Event Update

 

Bulk Sales Event Update allows you to run a single sales event against multiple plots on a site, but from just one screen. The selection criteria allow you to narrow down the list of plots against which an event can be run.

This is usually run after importing development and plot details into a new company (kco).

Navigate to House Sales > Plot Sales > Bulk Sales Event Update

  • Select the development to be updated
  • Select the Event you want to update and the criteria of the plots to be modified. Depending on the event chosen, the list of plots against which it can be run will be modified so that only plots with an appropriate sales status for this event will be displayed.
  • If you only want to include plots with a specific sales status or sales classification, select the relevant statuses by ticking or unticking the selections
  • Click

The next screen will show all the relevant plots based on the selection criteria.

 

  • Select the plot/s to update and select Run Event

  • On the next screen you can enter the event details

  • Click to update the selected plots

Once the bulk update has completed, you can continue to update the plots individually, taking each plot through to the stages required.
