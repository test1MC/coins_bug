Construction Programme

The construction programmes are set up using House Sales > Plot Sales > Setup > Construction Programme.  New options allow you to specify up to two sets of parallel stages; these are typically for use with timber-frame construction – for example, brickwork and internal fit-out proceeding in parallel.

You can also specify that the duration of a stage cannot be changed on the Gantt Chart Update screen.

NOTE

These new options do not apply to the forecast matrix, only to the Gantt Chart Update screen.

Figure 1: Construction Programme Stages Tab

Field Descriptions

Parallel

This allows you to assign the stage to one of two extra sets of stages which run in parallel to the main set of critical path stages.  These show as additional rows on the Gantt chart.

Predecessor Stage

This allows you to choose which stage a parallel stage immediately follows. The start date for the current stage follows the start date for the predecessor stage. 

Duration Fixed

If this is ticked, the stage duration cannot be changed from the Gantt Chart Update screen.
