Reservation Document

A new document type, Reservation Form (HS), is available in Document Designer.

Document Designer Reservation Form

The following fields are available on the Extras tab:

Field
Description
Discount Header Form

The header form for the discount section.

Discount Body Form

Along with the discount header and footer forms this allows a section to be configured which lists each cash discount/incentive record.

The discount section is optional and is ignored if these fields are left blank.

Discount Footer Form

The footer form for the discount section.

Incentive Header Form

The header form for the incentive section.

Incentive Body Form

Along with the incentive header and footer forms this allows a section to be configured which lists each non-cash discount/incentive record.

The incentive section is optional and is ignored if these fields are left blank.

Incentive Footer Form

The footer form for the incentive section.

Extras Header Form

The header form for the extras section.

Extras Body Form

Along with the extras header and footer forms this allows a section to be configured which lists each extras record.

The extras section is optional and is ignored if these fields are left blank.

Extras Footer Form

The footer form for the extras section.

First Page

A box indicating this form is the first page.

When printing the reservation document, the user will only be able to select documents that have this box ed.

Additional Pages Documents

A list of additional reservation documents to be included with this document.

Typically a reservation agreement may consist of multiple pages each with a long form and no body (effectively each page is just a header form). Each page will therefore need to be configured as a separate document.

Event Code
An optional event code. This will create an event in the Sales Workbench History when the document is generated.
Event Comments
Comments for the event created in the Sales Workbench.
