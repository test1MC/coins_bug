Generate Reservation Document - People Workbench

When viewing or updating a People Workbench (PIW) record, new fields are available if configured using the HS parameter PIWXFLD.

People Workbench Form of ID Fields

Fields

Field
Description
Form of Name ID

The form of identification used as proof of the person’s name.

The drop-down list includes only forms of ID set up in Forms of ID that have Proof of Name ed.

Form of Address ID

The form of identification used as proof of the person’s address.

The drop-down list includes only forms of ID set up in Forms of ID that have Proof of Address ed.

The same fields are available on the following screens:

  • Prospect Workbench – Personal Details tab.
  • Pre-Release Reservation – Customer Details tab.
  • Plot Reservation Purchaser – Personal Details tab.
  • Cancellation Purchaser – Personal Details tab.
  • Part Exchange Purchaser – Personal Details tab.
  • Customer Care Contact – Details function.
  • Lettings Workbench – Landlord tab.
  • Lettings Workbench – Tenants tab.
  • Lettings Workbench – Agent tab.
  • Lettings Workbench – Guarantors tab.
  • Lettings Workbench – References tab.
