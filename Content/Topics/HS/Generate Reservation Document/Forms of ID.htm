Generate Reservation Document - Forms of ID

Forms of ID is a new function that allows you to maintain a list of valid forms of identification.

Forms of ID Maintenance Screen

Fields

Field
Description
Code

The code for the form of ID.

Description

A description of the form of ID.

Proof of Name

A box to indicate whether this form of ID can be used as proof of a person’s name.

Proof of Address
A box to indicate whether this form of ID can be used as proof of a person’s address.
Retired
A box to indicate whether this form of ID is dormant (not used).
