Printing and Remittance Advices

Once you have created and frozen the payment batch, you can print the and generate payments.  You must print remittance advices after processing the payments, and after printing the (or at the same time, if you are using combined and remittances). 

To print and remittance advices:

  1. Go to Print or Print /Remittances.
  2. In the Batch Number column, click the link for the batch that contains the you want to print.
  3. Select the configurable document form to use for printing the .
  4. On the Main tab, if you only want to print for a range of , enter the codes of the first and last for whom you want to print .
  5. On the Selection tab:
    • Select the bank account on which the will be drawn.
    • Enter the date to appear on the .
    • Check that the "next number" displays is correct; if not, enter the correct number.
  6. Select any Output Options.  For example, to print the directly on a printer, the Print box (and choose the printer if the one you want is not shown).
  7. Click .

    displays a message informing you of the number of and/or payments that will be included in the payment run. To continue, click OK. Otherwise, click Cancel to return to the selection fields.

    Depending on the output options you selected, either displays a "Task completed successfully" message to tell you that the print request has been sent, or displays a "Generating" page, and then displays the printable output (PDF file). 
  8. Next, if you are not using combined /remittances, print the remittances.
  9. You can print a Payment List to check that the payments are correct (use Payment List).
  10. Finally, you need to post the batch.
