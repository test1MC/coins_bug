Palettes

palettes allow you to choose different schemes for different companies. A new Palettes maintenance function allows you to specify the theme s, and a new parameter SY/PALETTE allows you to choose which palette to use for the company.

A new User Preference option allows users to select a high-contrast version of the palette. If the user chooses this option, the palette with “HI” at the end of the name is used, if it exists.
