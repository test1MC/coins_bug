Advanced Filter

The advanced filter is now shown in a popup frame.

The Advanced / Simple button toggles between the advanced and simple filter.

A Not Filtered / Filter Applied button shows whether the advanced filter has been applied; if so, a tooltip shows which columns have been filtered. Clicking the button will display the Advanced Filter.

On the Advanced Filter popup:

  • The Apply button applies the filter and hides the filter popup.
  • The Clear button removes the filter and hides the filter popup.
  • The Simple button switches to the simple filter.

New shortcut keys: Ctrl+Shift+F to display or hide the Advanced Filter; Ctrl+Shift+A to apply the Advanced Filter; Esc to hide the Advanced Filter popup.
