1 Introduction

Previously, a Contract Number is entered on Gang Daily Report (CDR) header, which meant the CDR could hold data for one Contract. An enhancement has been made so that the Contract Number is entered on CDR detail lines in order to allow you to enter Daily Reports for Gangs that worked on different Contracts during the day.

4.1.1.1 Gang Daily Reports Tab

Gang Daily Reports can be accessed in Project Management > [selected Working Project] > Communication Management, as shown in .

Both the Gang Daily Reports and Review and Approval browses previously used the Contract Number on the CDR Header to show CDRs entered for the Project for one Contract only. This has been changed so that the CDR appears on each Project referenced on either the CDR Header or any details.

-The Review and approval browse now uses a linking table instead of selecting CDRs by Contract Number on the CDR Header.

PT Gang Daily Reports Tab

4.1.1.2 Project Daily Report Summary

Gang Daily Reports are displayed as part of Project Management > [selected Working Project] > Communication Management > Daily Reports tab > [click Date to drill down to Project Daily Report details] (). You can then choose Gang Daily Reports to access Gang Daily Reports ().

All CDRs linked to the current Project appear in the browse. When drilled down to CDR Summary, detail lines display that are filtered by the current Project. You can clear the filter to see all CDR details.
